k Minimum Values for Set Cardinality

Posted on 2021-02-22 Edited on 2021-03-19

Problem

We study the problem of estimating the number of distinct elements in a stream \(S\). WLOG, we assume that elements in \(S\) come from a domain \([d] = \{1, 2,..., d \}\), and that each element in \([d]\) appears in the \(S\) at least once. Hence, the problem is equivalent to the one of estimating \(d\).

k Minimum Values

The algorithm of \(k\)-minimum values returns a \((1 \pm \epsilon)\) estimate \(\hat d\) of \(d\), with probability at least \(1 - \delta\).

The algorithm picks a pairwise independent hash function \(g : [d] \rightarrow [m]\) and uses a min heap \(h\) that keeps the \(k\) minimum hash values. When it stops, it outputs an estimate \[ \hat d \doteq k \cdot m / v, \] where \(v\) is the maximum element in \(h\). Observe that \(v\) is also the \(k\)-th minimum hash value of the set \[ \{ g(1), g(2), ..., g(d) \}. \]

Algorithm: \(k\)-minimum values.
1. Select a pair-wise independent hash function \(g: [d] \rightarrow [m]\), where \(m \ge d^2 / \delta\);
2. Initialize a set \(h \leftarrow \{ \}\);
3. For \(s \in S\) do:
4. \(\qquad\) If \(g(s) \in h\) then continue; 5. \(\qquad\) \(h \leftarrow h \cup \{ g(s) \}\);
6. \(\qquad\) If \(|h| > k\) then
7. \(\qquad\) \(\qquad\) remove the max element in \(h\);
8. Let \(v \leftarrow\) max element in \(h\).
9. Return \(\hat d \doteq k \cdot m / v\).

Theorem 1.

With probability at least \(1 - \delta\), there isn't hash collision.

Proof:

For \(i \neq j \in [d]\), as \(g\) is pair-wise independent, \(\Pr[ g(i) = g(j) ] = \frac{1}{m}\). Therefore, \[ \mathbb{E} \left[ \sum_{i \neq j \in [d]} \mathbf{1} \big[ g(i) = g(j) \big] \right] = \sum_{i \neq j \in [d] } \mathbb{E} \left[ \mathbf{1} \big[ g(i) = g(j) \big] \right] \le \frac{d^2}{m} = \delta, \] where \(\mathbf{1} \big[ g(i) = g(j) \big]\) is the indicator random variable of \(g(i) = g(j)\).

By Markov's inequality, \[ \Pr \left[ \sum_{ i \neq j \in [d] } \mathbf{1} \big[ g(i) = g(j) \big] \ge 1 \right] \le \delta. \]

\(\square\)

Theorem 2.

With probability at least \(1 / (k\epsilon^2)\), \[ \hat d < \frac{d}{1 - \epsilon}. \]

Proof: For \(i \in [d]\), define the indicator random variable of \(X_i = \mathbf{1}\big[ g(i) \le (1 - \epsilon) \cdot k / d \cdot m \big]\). Recall that \[ v \doteq \text{The } k\text{-th smallest element of } \{ g(1), g(2), ..., g(d) \}. \] The event that \[ \left\{ v \le (1 - \epsilon) \cdot k / d \cdot m \right\} \equiv \left\{ \frac{d}{1 - \epsilon} \le \frac{ k \cdot m }{ v } \right\} \] happens if only \(\sum_{i \in [d] } X_i \ge k\).

By definition of \(X_i\), \[ p \doteq \Pr \left[ X_i \right] = (1 - \epsilon) \cdot \frac{ k }{d}. \]

By linearity of expectation, we have \[ \mathbb{E} \left[ \sum_{i \in [d] } X_i \right] = d \cdot p = (1 - \epsilon) \cdot k. \]

As \(g\) is pair-wise independent, so are the \(X_i\)'s. It follows that \[ \begin{aligned} \sigma^2 =\mathbb{Var} \left[ \sum_{i \in [d] } X_i \right] < d \cdot p = (1 - \epsilon) \cdot k. \end{aligned} \]

By monotonicity of probability and by Chebyshev's inequality, \[ \begin{aligned} \Pr\left[ \sum_{i \in [d] } X_i \ge k \right] &\le \Pr\left[ \left| \sum_{i \in [d] } X_i - (1 - \epsilon) \cdot k \right| \ge \epsilon k \right] < \frac{ (1- \epsilon) \cdot k }{ \epsilon^2 k^2 } < \frac{1}{\epsilon^2 k}. \end{aligned} \]

\(\square\)

Theorem 3.

With probability at least \((1 + \epsilon) / (k\epsilon^2)\), \[ \hat d \ge \frac{d}{1 + \epsilon}. \]

Proof: For \(i \in [d]\), define the indicator random variable of \(Y_i = \left[ g(i) \le (1 + \epsilon) \cdot k / d \cdot m \right]\). Recall that \[ v \doteq \text{The } k\text{-th smallest element of } \{ g(1), g(2), ..., g(d) \}. \] The event that \[ \left\{ v \ > \ (1 + \epsilon) \cdot k / d \cdot m \right\} \equiv \left\{ \frac{d}{1 + \epsilon} > \frac{ k \cdot m }{ v } \right\} \equiv \left\{ \frac{d}{1 + \epsilon} > \hat d \right\} \] happens if only \(\sum_{i \in [d] } Y_i < k\).

By definition of \(Y_i\), \[ q \doteq \Pr \left[ Y_i \right] = (1 + \epsilon) \cdot \frac{ k }{d}. \]

By linearity of expectation, we have \[ \mathbb{E} \left[ \sum_{i \in [d] } Y_i \right] = d \cdot q = (1 + \epsilon) \cdot k. \]

As \(g\) is pair-wise independent, so are the \(X_i\)'s. It follows that \[ \begin{aligned} \sigma^2 =\mathbb{Var} \left[ \sum_{i \in [d] } Y_i \right] &= \sum_{i \in [d] } \mathbb{Var} \left[ Y_i \right] \\ &= d \cdot q \cdot ( 1 - q) \\ &= (1 + \epsilon) \cdot k \cdot (1 - q). \end{aligned} \]

By monotonicity of probability and by Chebyshev's inequality, \[ \begin{aligned} \Pr\left[ \sum_{i \in [d] } Y_i < k \right] &\le \Pr\left[ \left| \sum_{i \in [d] } Y_i - (1 + \epsilon) \cdot k \right| \ge \epsilon k \cdot \sqrt{ 1 - q } \right] \\ &\le \frac{ (1 + \epsilon) \cdot k }{ \epsilon^2 k^2 } \\ &\le \frac{1}{\epsilon^2 k}. \end{aligned} \]

\(\square\)

Reference

[1] G. Cormode and K. Yi, Small Summaries for Big Data, 1st ed. Cambridge University Press, 2020.

Beyond Chebyshev's Inequality

Posted on 2021-02-12 Edited on 2022-04-02

We consider the Count Sketch with single array.

Definitions

  1. Let \(S = (s_1, s_2, ..., s_n)\) be a stream of \(n\) elements from a domain \([d]\).
  2. \(A:\) an array of size \(m\).
  3. \(h: [d] \rightarrow [m]\) a hash function that assigns each value in \([d]\) an integer uniformly at random from \([m]\).
  4. \(g: [d] \rightarrow \{ - 1, 1 \}\) a hash function that assigns each value in \([d]\) a random sign.

Algorithm

For each element \(s \in S\), Count-Sketch updates \(A\) as follows:

Update (\(s\)):
\(\qquad\) \(A[ h(s) ] \leftarrow A[ h(s) ] + g[s]\)

After processing elements in \(S\), Count-Sketch answer query for an element's frequency as follows:

Query (\(v\)):
\(\qquad\) return \(\hat f[v] \doteq g[v] \cdot A[ h(v) ]\)

Error Analysis

For a fixed \(v \in [d]\), let \(f[v]\) be its true frequencies in \(S\) (i.e., the number of times it appears in \(S\)). Define indicator random variables \[ X_1, X_2, ..., X_n, \]

such that for \(i \in [n]\), define \[ X_i = \begin{cases} &1, &\text{ if } h(s_i) = h(v), &g(s_i) = g(v), \\ &0, &\text{ if } h(s_i) \neq h(v), & \\ &-1, &\text{ if } h(s_i) = h(v), &g(s_i) \neq g(v), \end{cases} \]

The \(X_i\)'s are not independent. For \(i \in [n]\), if \(s_i \neq v\), it holds that \[ \mathbb{E} [ X_i ] = 0. \]

For \(i, j \in [n], s_i \neq v, s_j \neq v\), we have \[ \mathbb{ E } [X_i X_j ] = \begin{cases} &0, &\text{ if } s_i \neq s_j, \\ &\frac{1}{m}&\text{ if } s_i = s_j. \end{cases} \]

The estimation error of \(\hat f[v]\) can be written as \[ \left| \sum_{ i \in [n], s_i \neq v } X_i \right|. \]

For given \(\epsilon > 0\), we want to bound. \[ \begin{aligned} \Pr \left[ \left| \sum_{ i \in [n], s_i \neq v } X_i \right| \ge \epsilon \right] \end{aligned} \]

To do this, we pick some parameter \(t > 0\) and define \(\mathcal{V}_{bad} \doteq \{ u \in [d], h(u) \ge t \}\), and consider \[ \begin{aligned} &\Pr \left[ \left| \sum_{ i \in [n], s_i \neq v } X_i \right| \ge \epsilon \right] \\ &\le \Pr \left[ \exists u \in \mathcal{V}_{bad} : h(u) = h(v) \right] + \Pr \left[ \left| \sum_{ i \in [n], s_i \neq v } X_i \right| \ge \epsilon \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right] \\ &\le \frac{n}{ t \cdot m} + \Pr \left[ \left| \sum_{ i \in [n], s_i \neq v } X_i \right| \ge \epsilon \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right]. \end{aligned} \]

We consider the second probability. \[ \begin{aligned} &\mathbb{Var} \left[ \left| \sum_{ i \in [n], s_i \neq v } X_i \right| \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right] \\ &= \sum_{ i \in [n], s_i \neq v} \mathbb{Var} \left[ X_i \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right] \\ &\quad + \sum_{ i \in [n], j \in [n], i \neq j, s_i \neq v, s_j \neq v } \mathbb{E} \left[ X_i X_j \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right] \end{aligned} \]

The first term equals to \(\frac{n}{m}\). For the second term, if \(s_i \neq s_j\), then by independence of \(X_i\) and \(X_j\), it holds that \[ \begin{aligned} &\mathbb{E} \left[ X_i X_j \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right] \\ &= \mathbb{E} \left[ X_i \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right] \cdot \mathbb{E} \left[ X_j \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right]=0. \end{aligned} \]

If \(s_i = s_j\), then \(X_i = X_j\). Hence, \[ \begin{aligned} &\sum_{ i \in [n], j \in [n], i \neq j, s_i \neq v, s_j \neq v } \mathbb{E} \left[ X_i X_j \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right] \\ &= \sum_{ i \in [n], j \in [n], i \neq j, s_i \neq v, s_j \neq v, s_i = s_j } \mathbb{E} \left[ X_i^2 \mid \nexists u \in \mathcal{V}_{bad} : h(u) = h(v) \right] \\ &= \sum_{ i \in [n], j \in [n], i \neq j, s_i \neq v, s_j \neq v, s_i = s_j, f[s_i] \le t } \frac{1}{m} \\ &\le t^2 \cdot \frac{n}{t} \cdot \frac{1}{m} \\ &= \frac{ t \cdot n}{m}. \end{aligned} \]

By Chebyshev's inequality, \[ \begin{aligned} \Pr \left[ \left| \sum_{ i \in [n], s_i \neq v } X_i \right| \ge \epsilon \right] &\le \frac{n}{ t \cdot m} + \frac{t \cdot n}{m \cdot \epsilon^2}. \end{aligned} \]

By setting \(t = \epsilon\), the probability is minimized to \[ \begin{aligned} \Pr \left[ \left| \sum_{ i \in [n], s_i \neq v } X_i \right| \ge \epsilon \right] &\le \frac{n}{m} \cdot \frac{2}{\epsilon}. \end{aligned} \]

Note that this error bound is exactly the same as Count-Min sketch with a single array of size \(m\).

Morris Counter

Posted on 2021-02-07 Edited on 2021-02-25

Problem

Given a stream \(S\) of \(n\) elements, a server that observes the elements in \(S\) one by one is required to estimate the cardinality \(n\) with less than \(\log n\) bits, with relative error \(\epsilon \in (0, 1)\) and failure probability \(\delta\).

Let \(w \in (1, 2)\) be some parameter to be determined.

Algorithm. Morris Counter
1. \(k \leftarrow 0; \quad X_0 \leftarrow 0\); (introduced only for analysis)

  1. For \(t \leftarrow 1\) to \(n\):

  2. \(\qquad\) \(X_t \leftarrow X_{t - 1}\); (introduced only for analysis)

  3. \(\qquad\) With probability \(p_t \doteq 1 / w^k\) do:

  4. \(\qquad\) \(\qquad\) \(X_t \leftarrow X_t + w^k\);

  5. \(\qquad\) \(\qquad\) \(k \leftarrow k + 1\);

  6. Output \(\frac{w^k - 1}{w - 1}\).

It can be verified that, when the algorithm stops, \(X_n = \frac{w^k - 1}{w - 1}\).

Theorem. \(\forall 0 \le t \le n\), \(\mathbb{E} [ X_t ] = t\).

Proof: Clearly, \(\mathbb{E} [ X_0 ] = 0\). For \(t \ge 1\), \[ \mathbb{E} [ X_t \mid X_{t - 1} ] = p_t \cdot \left( X_{t - 1} + \frac{1}{p_t} \right) + ( 1 - p_t ) \cdot X_{t - 1} = X_{t - 1} + 1. \]

Hence, by induction on \(t\), we have \[ \mathbb{E} [ X_t ] = \mathbb{E} [ \mathbb{E} [ X_t \mid X_{t - 1} ] ] = \mathbb{E} [ X_{t - 1} ] + 1 = t. \]

\(\square\)

Theorem. \(\forall t \in [n]\), \(\mathbb{Var } [ X_t ] = (w - 1) \cdot \frac{ t^2 }{ 2 }\).

Proof: For \(t \ge 1\), it holds that \(X_{t - 1} = \frac{ 1 / p_t - 1}{w - 1}\). Therefore,

\[ 1 / p_t = (w - 1) \cdot X_{t - 1} + 1 \] and \[ \begin{aligned} \mathbb{E} [ X_t^2 | X_{t - 1} ] &= p_t \cdot \left( X_{t - 1} + 1 / p_t \right)^2 + ( 1 - p_t ) \cdot (X_{t - 1})^2 \\ &= 1 / p_t + 2 \cdot X_{t - 1} + (X_{t - 1})^2 \\ &= (w + 1) \cdot X_{t - 1} + 1 + (X_{t - 1})^2 \end{aligned} \]

Therefore, since \(X_0 = 0\), \[ \begin{aligned} \mathbb{E} [ X_t^2 ] &= \mathbb{E} [ \mathbb{E} [ X_t^2 | X_{t - 1} ] ] \\ &= (w + 1) \cdot \mathbb{E} [ X_{t - 1} ] + 1 + \mathbb{E} [ X_{t - 1} ^2] \\ &= \mathbb{E} [ X_{t - 1}^2 ] + (w + 1) \cdot (t - 1) + 1 \\ &... \\ &= \mathbb{E} [ X_0^2 ] + (w + 1) \cdot \sum_{i = 0}^{t - 1} i + t \\ &= (w + 1) \cdot \frac{ t (t - 1) }{ 2 } + t \\ &\le (w - 1) \cdot \frac{ t^2 }{ 2 } + t^2, \end{aligned} \]

and \[ \mathbb{Var } [ X_t ] = \mathbb{E} [ X_t^2 ] - (\mathbb{E} [ X_t ])^2 \le (w - 1) \cdot \frac{ t^2 }{ 2 } . \]

Corollary. By Chebyshev's inequality, \[ \Pr[ | X_t - t | \ge \epsilon \cdot t ] \le (w - 1) \cdot \frac{ t^2 }{ 2 \epsilon^2 \cdot t^2} = \frac{w - 1}{ 2 \epsilon^2 }. \]

Corollary. If we set \(w = 1 + 2 \epsilon^2 \delta\), then the failure probability is at most \(\delta\).

Reference.

[1] G. Cormode and K. Yi, Small Summaries for Big Data, 1st ed. Cambridge University Press, 2020.
