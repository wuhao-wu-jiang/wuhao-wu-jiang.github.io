Working Set BST

Posted on 2020-03-23 Edited on 2020-03-29

Various implementations of Balanced Binary Search Tree (BST) guaranteed membership queries in O(log⁡n)O(\log n)O(logn) time, where nnn is the number of elements in some set SSS. The working set property is a stronger requirement than O(log⁡n)O(\log n)O(logn) time complexity.
