Gaussian Distribution

We illustrate how Gaussian distribution (a.k.a. normal distribution) is constructed and some of its basic properties.

Suppose we would like to have a distribution in R\mathbb{R}R, such that

  1. It is centered at the origin.
  2. Its density function decreases exponentially with respect to the square distance to the origin.

Remark: if we replace 2 with “a density function decreases exponentially with the distance to the origin”, we obtain Laplace distribution.

By definition, the density function p(x)p(x)p(x) should be inversely and exponentially proportional to x2x^2x2:

p(x)∝exp⁡(−x2)p(x) \propto \exp(-x^2 ) p(x)∝exp(−x2)

If we define MMM to be the value of

M≐∫−∞∞exp⁡(−x2) dx,M \doteq \int_{-\infty}^\infty \exp(-x^2) \ dx, M≐∫−∞∞​exp(−x2) dx,

then we can give the precise formula of p(x)p(x)p(x):

p(x)=1Mexp⁡(−x2)p(x) = \frac{1}{M} \exp(- x^2) p(x)=M1​exp(−x2)

It is easy to see that p(x)p(x)p(x) is a density function such that ∫−∞∞p(x) dx=1\int_{-\infty}^\infty p(x) \ dx = 1∫−∞∞​p(x) dx=1.

Closed Form

In general, it is not hard to construct a distribution. Let h(x)h(x)h(x) be any integrable function on a domain D\mathcal{D}D such that

M=∫Dh(x) dx<∞M = \int_\mathcal{D} h(x) \ dx < \infty M=∫D​h(x) dx<∞

Then we can view h(x)h(x)h(x) as almost an density function of some distribution. The only obstacle is that MMM might not be 111. However this is easy to overcome. We just scale the value of h(x)h(x)h(x) by a constant factor at each point on R\mathbb{R}R and set

p(x)=1Mh(x)p(x) = \frac{1}{M} h(x) p(x)=M1​h(x)

then we have immediately a density function. We don’t even need to know the exact value of MMM. What we need to guarantee is just that

The value of MMM exists and is finite.

And we can safely use MMM to denote this value. For example, we know that

∫0∞x0.2e−x dx≤∫01e−x dx+∫1∞xe−x dx<∞.\int_0^\infty x^{0.2} e^{-x} \ dx \le \int_0^1 e^{-x} \ dx + \int_1^\infty x e^{-x} \ dx < \infty. ∫0∞​x0.2e−x dx≤∫01​e−x dx+∫1∞​xe−x dx<∞.

We can construct a density function as above. It turns out that this is a special case of Gamma distribution.

For Gaussian distribution, we are lucky as we can compute a closed form of MMM. This is done by a trick of computing its square:

M2=(∫−∞∞exp⁡(−x2) dx)2=∫−∞∞exp⁡(−x2) dx ⋅∫−∞∞exp⁡(−y2) dy=∫−∞∞∫−∞∞exp⁡(−(x+y)2) dxdy.\begin{aligned} M^2 &= \big( \int_{-\infty}^\infty \exp( -x^2 ) \ dx \big)^2 \\ &= \int_{-\infty}^\infty \exp( -x^2 ) \ dx \ \cdot \int_{-\infty}^\infty \exp( -y^2 ) \ dy \\ &= \int_{-\infty}^\infty \int_{-\infty}^\infty \exp( -(x + y) ^2 ) \ dx dy. \end{aligned} M2​=(∫−∞∞​exp(−x2) dx)2=∫−∞∞​exp(−x2) dx ⋅∫−∞∞​exp(−y2) dy=∫−∞∞​∫−∞∞​exp(−(x+y)2) dxdy.​

Denote r=x2+y2r = \sqrt{ x^2 + y^2}r=x2+y2​ the distance of (x,y)(x, y)(x,y) to (0,0)(0, 0)(0,0). We are integrating the function exp⁡(−r2)\exp( - r^2)exp(−r2) over the plane R2\mathbb{R}^2R2. It is natural to switch to polar coordinate system (r,θ)(r, \theta)(r,θ), where θ\thetaθ is the angle of a point. Here is an informal but intuitive explanation. p(x)=1πexp⁡(−x2).p(x) = \frac{1}{ \sqrt \pi} \exp( - x^2 ). p(x)=π​1​exp(−x2).

Gaussian Mechanism

Our discussion in this section focus on a metric space (X,d)(\mathcal{X}, d)(X,d) and a function f:X→Rnf: \mathcal{X} \rightarrow \mathbb{R}^nf:X→Rn. The Gaussian mechanism adds Gaussian noise to the output of fff, such that for any neighboring pairs x,y∈Xx, y \in \mathcal{X}x,y∈X with d(x,y)=1d(x, y) = 1d(x,y)=1, it is hard to distinguish the outputs, i.e., their outputs have similar distribution after adding noises.

Definition. The sensitivity Δ\DeltaΔ of fff is defined as

Δ=max⁡x,y∈X,d(x,y)=1∣∣f(x)−f(y)∣∣ \Delta = \max_{x, y \in \mathcal{X}, d(x, y) = 1 } ||f(x) - f(y) || Δ=x,y∈X,d(x,y)=1max​∣∣f(x)−f(y)∣∣

where ∣∣⋅∣∣|| \cdot ||∣∣⋅∣∣ is the ℓ2\ell_2ℓ2​-norm of a vector in Rn\mathbb{R}^nRn.

Definition. Given fff, and a variance parameter σ2\sigma^2σ2, the Gaussian mechanism is a random variable defines as

g(x)=f(x)+X g(x) = f(x) + X g(x)=f(x)+X

where X=(X1,X2,...,Xn)∼N(0,σ2I)X = (X_1, X_2, ..., X_n) \sim N(0, \sigma^2 I)X=(X1​,X2​,...,Xn​)∼N(0,σ2I).

We are ready to state the main theorem of this section.

Theorem. For δ,ϵ∈(0,1)\delta, \epsilon \in (0, 1)δ,ϵ∈(0,1), if σ=Δlog⁡1δϵ\sigma = \frac{ \Delta \log \frac{1}{\delta} }{ \epsilon }σ=ϵΔlogδ1​​, then ∀x,y∈X\forall x, y \in \mathcal{X}∀x,y∈X, s.t., d(x,y)=1d(x, y) = 1d(x,y)=1, and for all measurable subset S⊂RnS \subset \mathbb{R}^nS⊂Rn, it holds

It is obvious that if we set σ→∞\sigma \rightarrow \inftyσ→∞, then the Gaussian distribution tends to be a uniform one over Rn\mathbb{R}^nRn. So we are interested in how small σ\sigmaσ can be.

We claim that σ=O(Δϵ)\sigma = O(\frac{\Delta}{\epsilon } )σ=O(ϵΔ​) suffices. This is based on three observations: 1) Most of the probability mass of a Gaussian distribution concentrates on a ball centered at its mean with radius O(σ)O(\sigma)O(σ); 2) the density ratio of any two points within the ball is bounded by a constant; 3) the distance between the distribution center of g(x)g(x)g(x) and g(y)g(y)g(y) is bounded by Δ\DeltaΔ.
