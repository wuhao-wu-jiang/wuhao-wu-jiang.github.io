Complex Integral

Posted on 2018-12-13 Edited on 2022-11-29

We establish complex integral step by step in this article. We first review the classical Riemann integral and see how this motivates complex integral.

Riemann Sums

Given an interval \([a, b]\) and a function \(f : [a, b] \rightarrow \R\), the Riemann integration divides \([a, b]\) into many small intervals, and computes the sum of the products of the length of each interval and value of \(f\) at an arbitrary chosen point on the interval. Such a sum is well-defined if it converges when the chosen partition is "refined" enough.

Definition. A finite sequence \(P = \{x_0, x_1, \ldots, x_n \}\) is a partition of \([a, b]\), if \[ a = x_0 < x_1 < \ldots < x_n = b. \] The length of \(P\), denoted as \(\Vert P \Vert\), is given by \[ \Vert P \Vert \doteq \max_{i \in [n] } \Delta_i \] where \(\Delta_i \doteq x_i - x_{i - 1}\) for each \(i \in [n]\).

Definition. A number \(J \in \R\) is the definite integral of \(f\) over \([a, b]\), if
\(\forall \epsilon > 0\), \(\exists \delta > 0\) s.t., for every partition \(P = \{x_0, x_1, \ldots, x_n \}\) of \([a, b]\) with \(\Vert P \Vert < \delta\), and \(\forall c_k \in [x_{k - 1}, x_k]\), it holds that
\[ \left| \sum_{k = 1}^n f(c_k) \Delta_k - J \right| < \epsilon \]

There is a dedicated notation for the limit of Riemann sum \[ \int_a^b f(x) dx = \lim_{\Vert P \Vert \rightarrow 0} \sum_{k = 1}^n f(c_k) \Delta_{k}. \]

Complex Integral

Line Integral

How do we extend integral in real line to complex plane. The most simplest extension is of a real to complex function \[ \gamma(t) = x(t) + i y(t), \] whose integral on an interval is defined by integrating its real part and imaginary parts separately.

Definition. Let \(\gamma(t) = x(t) + i y(t)\) be a function defined on \([a, b]\). If both \(x(x)\) and \(y(t)\) are Riemann integrable, then the integral of \(\gamma\) over \([a, b]\) is defined as \[ \int_a^b \gamma(t) \, dt \doteq \int_a^b x(t) dt + i\int_a^b y(t) dt, \]

Path Integral

Another natural idea is to consider an oriented path \(\gamma \subseteq \mathbb{C}\) with endpoints \(a\) and \(b\) on the complex plane and a complex function \(f : \gamma \rightarrow \mathbb{C}\).

Note we do not even require \(a \neq b\); but we must specify an orientation for \(\gamma\).

Next, we repeat the partition and product idea: let \(P = \{z_0, z_1, \ldots, z_n\} \subseteq \gamma\) be a partition of the curve such that \(z_0 = a\) and \(z_n = b\). A Riemann sum associated with \(P\) is just what it is in the real case: \[ \sum_{k = 1}^n f(d_k) \cdot \Delta_{k}, \] where \(\Delta_k \doteq z_k - z_{k - 1}\), and \(d_k\) is an arbitrary point on the arc \(\overgroup{z_{k - 1}, z_k}\).

Definition (Path Integral). Suppose there exists \(J \in \mathbb{C}\), s.t., \(\forall \epsilon > 0\), \(\exists\delta > 0\) s.t., for every partition \(P = \{z_0, z_1, \ldots, z_n\}\) of \(\gamma\) with \(\Vert P \Vert \doteq \max_{k \in [n]} \Vert \Delta_k \Vert < \delta\) and every choice of \(d_k\) in \(\overgroup{z_{k - 1}, z_k}\), it holds that \[ \left| \sum_{k = 1}^n f(d_k) \Delta_{k} - J \right| < \epsilon. \] Then we call \(J\) the integral of \(f\) over \(\gamma\), and write it as \[ \int_\gamma f(x) dx \doteq J \doteq \lim_{\Vert P \Vert \rightarrow 0} \sum_{k = 1}^n f(d_k) \Delta_{k}. \]

Properties

Linearity. 1. Constant Multiple: \(\int_\gamma c f(x) dx = c \int_\gamma f(x) dx\).

  1. Sum: \(\int_\gamma f(x) + g(x) dx = \int_\gamma f(x) dx + \int_\gamma g(x) dx\).

Proof. For any partition \(P\) of \(\gamma\) and for every choice of the points \(d_k\),
\[ \begin{aligned} \sum_{k \in [n]} c f( d_k ) \Delta_k &= c \sum_{k \in [n]} f( d_k ) \Delta_k, \\ \sum_{k \in [n]} (f( d_k ) + g( z_k ) )\Delta_k &= \sum_{k \in [n]} f( d_k ) \Delta_k + \sum_{k \in [n]} g( z_k ) \Delta_k. \end{aligned} \] Taking limit of both side, we get the desired answer.

Remark: given two sequence \(\{ a_1, a_2, \ldots, \}\) and \(\{b_1, b_2, \ldots, \}\), if \(a_i = b_i\) for arbitrary \(i \ge 0\) and \(\lim_i b_i = S\), then \(\lim_i a_i\) exits and \(\lim_i a_i = S\).

\(\square\)

Existence

We provide a sufficient condition under which the path integral exists.

Definition (Parameterized Smooth Curve). We say that \(\gamma\) is a parameterized smooth curve, if there exists a function \(r (t): [a, b] \rightarrow \mathbb{C}\) which maps a closed interval \([a, b] \subseteq \mathbb{R}\) to the complex plane \(\mathbb{C}\), s.t., \(r'(t)\) exists and is continuous on \([a, b]\), and \(r'(t) \neq 0\) for \(t \in [a, b]\).

Theorem. Suppose that \(\gamma\) is a parameterized smooth curve and \(f: \gamma \rightarrow \mathbb{C}\) is a continuous function. Then the limit \(\lim_{\Vert P \Vert \rightarrow 0} \sum_{k = 1}^n f(d_k) \Delta_{k}\) exists and is given by \[ \lim_{\Vert P \Vert \rightarrow 0} \sum_{k = 1}^n f(d_k) \Delta_{k} = \int_a^b f \big( r(t) \big) \, r'(t) \, d t. \]

Proof. We first prove that \(f (r (t)) \cdot r'(t)\) is integrable, so the RHS of the equation is well-defined. Then we show that \(\sum_{k = 1}^n f(d_k) \Delta_{k}\) converges to the RHS as \(\Vert P \Vert \rightarrow 0\). Hence the limit \(\lim_{\Vert P \Vert \rightarrow 0} \sum_{k = 1}^n f(d_k) \Delta_{k}\) exists.

Integrability of \(f (r (t)) \cdot r'(t)\):

  • Observe that \(f \circ r (t) = f(r(t)): [a, b] \rightarrow \mathbb{C}\) is continuous.

  • By assumption \(r'(t) : [a, b] \rightarrow \mathbb{C}\) is continuous.

  • Therefore, \(f (r (t)) \cdot r'(t) : [a, b] \rightarrow \mathbb{C}\) is continuous.

  • Both the real and imaginary parts of \(f (r (t)) \cdot r'(t)\) are continuous, and hence integrable.

  • By definition of line integral, \[ \int_a^b f \big( r(t) \big) \, r'(t) \, d t \doteq \int_a^b \mathbb{Re} \Big( f \big( r(t) \big) \, r'(t) \Big) \, d t + i \cdot \int_a^b \mathbb{Im} \Big( f \big( r(t) \big) \, r'(t) \Big) \, d t \] is well-defined.

Proving The Equation:

Claim One. For each \(z \in \gamma\), there is a unique point \(t \in [a, b]\) for which \(r(t) = z\).

Proof of the Claim One. \(\square\)

We continue to prove the equation. Let \(P = \{z_0, z_1, \ldots, z_n\}\) be a partition of \(\gamma\). By Claim One, there exists unique points \(Q = \{ a = t_0 < t_1< \ldots < t_n = b \}\), s.t., \(r(k) = z_k\), for \(k \in [0 .\, .\, n]\).

\(\square\)

To compute such integral, we can choose a description \(\gamma(t) = x(t) + i y(t) : [a, b] \rightarrow \mathbb{C}\) of curve \(\gamma\). We partition \(\gamma\) by partitioning the interval \([a, b]\) in usual way \(P_t = \{ a = t_0 < t_1< \ldots < t_n = b \}\). Then \(\gamma(P_t) \doteq \{ \gamma(t_0), \gamma(t_1), \ldots, \gamma(t_n) \}\) is a partition of \(\gamma\).

If \(\gamma\) is continuous, it is uniformly continuous on \([a, b]\). Therefore, as \(\Vert P_t\Vert \rightarrow 0\), \(\Vert \gamma(P_t)\Vert \rightarrow 0\). If \(\int_\gamma f(z) dz\) does exist, then

\[ \int_\gamma f(z) dz = \lim_{\Vert P_t\Vert \rightarrow 0} \sum_{j = 1}^n f \big( \gamma( e_j ) \big) \big(\gamma(t_j) - \gamma(t_{j - 1} ) \big) \] for arbitrary choice \(e_j \in [t_{j - 1}, t_j]\).

When the derivative of \(\gamma\) exists and is continuous, it is uniformly continuous on the closed bounded interval \([a, b]\). We now can rewrite the Riemann sum as
\[ \begin{aligned} &\sum_{j = 1}^n f \big( \gamma( e_j ) \big) \frac{\gamma(t_j) - \gamma(t_{j - 1} ) }{t_j - t_{j - 1} }(t_j - t_{j - 1} ) \\ &= \sum_{j = 1}^n f \big( \gamma( e_j ) \big) \frac{x(t_j) - x(t_{j - 1} )+ i (y(t_j) - y_(t_{j - 1} ) ) }{t_j - t_{j - 1} }(t_j - t_{j - 1} ) \end{aligned} \]

By mean value theorem, as \(x'(t)\) and \(y'(t)\) exist, there are some \(e_j' \in [t_{j - 1}, t_j]\) and \(e_j'' \in [t_{j - 1}, t_j]\), such that

\[ \frac{x(t_j) - x(t_{j - 1} ) }{t_j - t_{j - 1} } = x'(e_j'), \qquad \frac{y(t_j) - y(t_{j - 1} ) }{t_j - t_{j - 1} } = x'(e_j''). \]

On the other hand, as \(x'(t)\) and \(y'(t)\) are uniformly continuous, for arbitrary \(\epsilon > 0\), there is a \(\delta\) such that if \(\Vert P_t\Vert < \delta\), \(|x'(e_j') - x'(e_j)| < \epsilon\) and \(|y'(e_j'') - y'(e_j)| < \epsilon\). It follows that
\[ \begin{aligned} \frac{x(t_j) - x(t_{j - 1} )+ i (y(t_j) - y_(t_{j - 1} ) ) }{t_j - t_{j - 1} } &= x'(e_j) + i y'(e_j) \pm \epsilon(1 + i)\\ \sum_{j = 1}^n f \big( \gamma( e_j ) \big) \frac{\gamma(t_j) - \gamma(t_{j - 1} ) }{t_j - t_{j - 1} }(t_j - t_{j - 1} ) &= \sum_{j = 1}^n f \big( \gamma( e_j ) \big) \gamma'(e_j) (t_j - t_{j - 1} ) \\ &\quad \pm \epsilon (1 + i) \cdot \sum_{j = 1}^n f \big( \gamma( e_j ) \big) (t_j - t_{j - 1} ), \end{aligned} \]

Taking the limit of \(\Vert P_t\Vert \rightarrow 0\), we get
\[ \int_\gamma f(z) dz = \int_a^b f(\gamma(t) )\gamma'(t) dt, \]

Remark 1: Can we relax the condition that \(\gamma'(t)\) is continuous and that \([a, b]\) is a closed interval?

Remark 2: We have also showed that the integration is independent of the choice of \(\gamma\), i.e., any representation of \(\gamma\) that is continuously differentiable gives the same integration value.

Anti-Derivative

Now suppose that \(F(z) = u(x, y) + i v(x, y)\) and \(F'(z) = f(z)\).

Theorem. \(\int_\gamma f(z) dz = F(\gamma(b) ) - F(\gamma(a) )\).

First, \(F'(\gamma(t) ) = \frac{\partial u}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial u}{\partial y} \frac{\partial y}{\partial t} + i ( \frac{\partial v}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial v}{\partial y} \frac{\partial y}{\partial t} )\). Therefore,
\[ \int_\gamma f(z) dz = \int_a^b \left(\frac{\partial u}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial u}{\partial y} \frac{\partial y}{\partial t} \right) \, dt + i \int_a^b \left( \frac{\partial v}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial v}{\partial y} \frac{\partial y}{\partial t} \right) \, dt \]

But
\[ \begin{aligned} &u(x(b), y(b) ) - u(x(a), y(a) ) \\ &= \sum_{j = 1}^n [u(x(t_j), y(t_j) ) - u(x(t_{j - 1} ), y(t_{j - 1} ) ) ] \\ &= \sum_{j = 1}^n [u(x(t_j), y(t_j) ) - u(x(t_j), y(t_{j - 1} ) ) + u(x(t_j), y(t_{j - 1} ) ) - u(x(t_{j - 1} ), y(t_{j - 1} ) ) ] \\ &= \sum_{j = 1}^n \frac{u(x(t_j), y(t_j) ) - u(x(t_j), y(t_{j - 1} ) ) }{y(t_j) - y(t_{j - 1} ) } \frac{y(t_j) - y(t_{j - 1} ) }{ t_j - t_{j - 1} }(t_j - t_{j - 1} )\\ &\quad +\sum_{j = 1}^n \frac{u(x(t_j), y(t_{j - 1} ) ) - u(x(t_{j - 1} ), y(t_{j - 1} ) ) }{x(t_j) - x(t_{j - 1} ) } \frac{x(t_j) - x(t_{j - 1} ) }{t_j - t_{j - 1} } (t_j - t_{j - 1} ) \\ &= \sum_{j = 1}^n [u_y' (x(t_j), y^*_j) y'(e_j') + u_x' (x^*_j, y(t_{j - 1} ) ) x'(e_j'') ] (t_j - t_{j - 1} )\\ \end{aligned} \]

where \(y^*_j \in [ y( t_{j - 1} ), y( t_j ) ]\), \(x^*_j \in [ x( t_{j - 1} ), x( t_j ) ]\), \(e_j', e_j'' \in [ t_{j - 1}, t_j ]\). If \(u'_x, u'_y, x'(t), y'(t)\) are continuous, then as \(\Vert P_t\Vert \rightarrow 0\), \[ \begin{aligned} &\sum_{j = 1}^n [u_y' ( x( t_j ), y( t_j ) ) y'( t_j ) + u_x' ( x( t_j ), y( t_j ) ) x'( t_j ) ] ( t_j - t_{j - 1} ) \\ &= \sum_{j = 1}^n \frac{\partial u}{\partial t} ( t_j - t_{j - 1} ) \rightarrow u(x(b), y(b) ) - u(x(a), y(a) ). \end{aligned} \]

An immediate consequence is that
\[ F(\gamma(b) ) - F(\gamma(a) ) = \int_\gamma F'(\gamma(t) ) dt \]

On the other hand, since \(F(z)\) is differential, \[ \lim_{h \rightarrow 0} \frac{F(z + h) - F(z) }{h} = \frac{\partial u}{\partial x} + i \frac{\partial v}{\partial x} = \lim_{ih \rightarrow 0} \frac{F(z + ih) - F(z) }{ih} = -i \frac{\partial u}{\partial y} + \frac{\partial v}{\partial y}. \]

It holds that \(\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}\) and \(\frac{\partial v}{\partial x} = -\frac{\partial u}{\partial y}\). Therefore,
\[ \begin{aligned} F'(\gamma(t) ) &= \frac{\partial u}{\partial x} \frac{\partial x}{\partial t} - \frac{\partial v}{\partial x} \frac{\partial y}{\partial t} + i \left( \frac{\partial v}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial u}{\partial x} \frac{\partial y}{\partial t} \right) \\ &= \left( \frac{\partial u}{\partial x} + i \frac{\partial v}{\partial x} \right) \left( \frac{\partial x}{\partial t} + i \frac{\partial y}{\partial t} \right) \\ &= F'(\gamma(t) ) \gamma'(t) \\ &= f(\gamma(t) ) \gamma'(t). \end{aligned} \]

and
\[ F(\gamma(b) ) - F(\gamma(a) ) = \int_\gamma f(\gamma(t) ) \gamma'(t) dt \]

Remark: That \(u_x', u_y'\) are continuous is a strong condition. What we need in the proof is just that *

\[ \begin{aligned} u(x(t_j), y(t_j) ) &= u(x(t_{j - 1} ), y(t_{j - 1} ) ) \\ &+ u_x'(x(t_{j - 1} ), y(t_{j - 1} ) ) (x(t_j) - x(t_{j - 1} ) ) \\ &+ u_y'(x(t_{j - 1} ), y(t_{j - 1} ) ) (y(t_j) - y(t_{j - 1} ) ) \\ &+ o(x(t_j) - x(t_{j - 1} ), y(t_j) - y(t_{j - 1} ) ) \end{aligned} \]

or

\[ \begin{aligned} u(x(t_{j - 1} ) + \delta_x, y(t_{j - 1} ) + \delta_y) &= u(x(t_{j - 1} ), y(t_{j - 1} ) ) \\ &+ u_x'(x(t_{j - 1} ), y(t_{j - 1} ) ) \delta_x \\ &+ u_y'(x(t_{j - 1} ), y(t_{j - 1} ) ) \delta_y \\ &+ o(\delta_x, \delta_y) \end{aligned} \]

i.e., \(u\) is differentiable at every point in the domain.

Question: Is is possible that \(u\) is differentiable but its partial derivatives are not continuous? The answer is NO. First we consider an one dimension counter example:
\[ f(x) = \begin{cases} x^2 \sin \frac{1}{x} \quad \forall x \neq 0, \\ 0, \quad if \quad x = 0 \end{cases} \] To show \(f(x)\) is differential at \(0\), we use the definition of derivative:
\[ f'(0) = \lim_{h \rightarrow } \frac{f(h) - f(0) }{h - 0} = \lim_{h \rightarrow 0} h \sin \frac{1}{h} = 0 \] On the other hand, \(f(x)\) is always differentiable from \(0\):
\[ f'(x) = 2x \sin \frac{1}{x} + x^2 ( -1 / x^2) cos \frac{1}{x} = 2x \sin \frac{1}{x} -cos \frac{1}{x} \] Hence, \(f(x)\) is differentiable everywhere, but \(f'(x)\) is not continuous at \(0\).

The above example can be easily extended to two dimension case:
\[ f(x, y) = \begin{cases} (x^2 + y^2) \sin \frac{1}{ \sqrt{x^2 + y^2} } \quad \forall (x, y) \neq (0, 0) \\ 0 \qquad \qquad \qquad \qquad \quad if \quad (x, y) = (0, 0) \end{cases} \]

By definition, \(f(x, y)\) is differential at \((0, 0)\),
\[ \lim_{\Vert (x, y)\Vert \rightarrow 0} \frac{\Vert f(x, y) - f(0) - (0,0) \cdot (x, y)\Vert }{\Vert (x, y)\Vert } \le \lim_{\Vert (x, y)\Vert \rightarrow 0} \frac{\Vert x^2 + y^2\Vert }{\Vert (x, y)\Vert } = \lim_{\Vert (x, y)\Vert \rightarrow 0} \Vert (x, y)\Vert = 0 \]

However, it partial derivative from \((0, 0)\) \[ \begin{aligned} \frac{\partial f}{\partial x} &= 2x \sin \frac{1}{ \sqrt{x^2 + y^2} } + (x^2 + y^2) \left( -\frac{ 2x }{ \sqrt{ x^2 + y^2 }^{3} } \right) \cos \frac{1}{ \sqrt{x + y} } \\ &= 2x \sin \frac{1}{ \sqrt{x^2 + y^2} } - \frac{ 2x }{ \sqrt{ x^2 + y^2 } } \cos \frac{1}{ \sqrt{x^2 + y^2 } } \end{aligned} \] and
\[ \frac{\partial f}{\partial x}(0, 0) = \lim_{h \rightarrow} \frac{h^2 \sin \frac{1}{|h|} }{h} = 0 \]

Therefore, \(\partial f / \partial x\) is not continuous.
