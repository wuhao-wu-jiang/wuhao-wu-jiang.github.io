Integration of Real-valued Functions

Motivation

Given an interval $[a, b]$ and some function $f: [a, b] \rightarrow \R$, we want to create some rule, that computes the area under the curve $y = f(x)$ between $x = a$ and $x = b$.

This rule, which assigns real numbers to functions, can be itself represented as a function $\varphi_{[a, b]} : \mathcal{F} \big( [a, b] \big) \rightarrow \R$.
Here, $\mathcal{F} \big( [a, b] \big)$, the domain of $\varphi$, refers to a collection of functions defined on $[a, b]$ such that the area under the curve $y = f(x)$ between $x = a$ and $x = b$ is well-defined according to the rule.

The integral should adhere to our geometric understanding of areas and satisfy various properties.
First and foremost, the integral should accurately compute the area of rectangles.

Property (Area of Rectangle). If $\exist c \in \R$, s.t., $f(x) = c$ for each $x \in [a, b]$, then $f \in \mathcal{F} \big( [a, b] \big)$ and $\int_a^b f \, dx = c \cdot (b - a)$. .

  2. If $f_1, f_2 \in \mathcal{F} \big( [a, b] \big)$, then $f_1 + f_2 \in \mathcal{F} \big( [a, b] \big)$, and
