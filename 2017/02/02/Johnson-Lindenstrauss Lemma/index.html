Johnson-Lindenstrauss Lemma

Posted on 2017-02-02 Edited on 2024-03-08

Problem

Let U⊆Rd\mathcal{U} \subseteq \mathbb{R}^dU⊆Rd be a set of nnn points.
We want to find a linear transformation A∈Rk×dA \in \mathbb{R}^{k \times d}A∈Rk×d that embeds the points in U\mathcal{U}U to Rk\mathbb{R}^kRk while preserving the pairwise Euclidean distances.

If k≥dk \ge dk≥d, the problem is trivial.
There is an isomorphic mapping between Rd\mathbb{R}^dRd and a ddd-dimensional subspace in Rk\mathbb{R}^kRk.
Let u1,…,udu_1, \ldots, u_du1​,…,ud​ be a subset of standard basis vectors in Rk\mathbb{R}^kRk, and e1,…,ede_1, \ldots, e_de1​,…,ed​ be standard basis vectors in Rd\mathbb{R}^dRd.
Then one possible isomorphic mapping is given by

A=u1e1T+…+udedT. A = u_1 e_1^T + \ldots + u_d e_d^T. A=u1​e1T​+…+ud​edT​.

If k<dk < dk<d, then such an isomorphic embedding is impossible.
However, it is feasible, if we aim to preserve the pair-wise distance up to a multiplicative error ϵ∈(0,1)\epsilon \in (0, 1)ϵ∈(0,1) with high probability, s.t., ∀x,y∈U\forall x, y \in \mathcal{U}∀x,y∈U,

∥Ax−Ay∥2∈(1±ϵ)∥x−y∥2, \Vert A x - A y \Vert_2 \in (1 \pm \epsilon) \Vert x - y \Vert_2, ∥Ax−Ay∥2​∈(1±ϵ)∥x−y∥2​,

where ∥⋅∥2\Vert \cdot \Vert_2∥⋅∥2​ is the ℓ2\ell_2ℓ2​ distance.

The smaller kkk is, the less space we need to store the embedded vectors

AU≐{Ax:x∈U}⊆Rk×n. A \mathcal{U} \doteq \{ Ax : x \in \mathcal{U} \} \subseteq \mathbb{R}^{k \times n}. AU≐{Ax:x∈U}⊆Rk×n.

It remains to study how small kkk can be?

Theorem. For each set U⊆Rd\mathcal{U} \subseteq \mathbb{R}^dU⊆Rd of nnn points, there is a matrix A∈Rk×dA \in \mathbb{R}^{k \times d}A∈Rk×d, such that k=O(log⁡nϵ2)k = O\left( \frac{ \log n }{ \epsilon^2} \right)k=O(ϵ2logn​) and

∀x,y∈U,∥Ax−Ay∥2∈(1±ϵ)∥x−y∥2, \forall x, y \in \mathcal{U}, \quad \Vert A x - A y \Vert_2 \in (1 \pm \epsilon) \Vert x - y \Vert_2, ∀x,y∈U,∥Ax−Ay∥2​∈(1±ϵ)∥x−y∥2​,

Further, there is an algorithm which returns such a matrix AAA with success probability at least 1−1n1 - \frac{1}{n}1−n1​

The value of kkk depends only on nnn but not ddd.

Column Perspective

On the high level, we try to squeeze ddd (in expectation) orthonormal vectors into a smaller space Rk\mathbb{R}^kRk.
We do this by generating a set of ddd random vectors u1,u2,…,ud∈Rku_1, u_2, \ldots, u_d \in \mathbb{R}^ku1​,u2​,…,ud​∈Rk, such that:

Let e1,…,ed∈Rde_1, \ldots, e_d \in \mathbb{R}^de1​,…,ed​∈Rd be unit vectors in Rd\mathbb{R}^dRd and eie_iei​ (i∈[d]i \in [d]i∈[d]) has the iii-th entry equal to 1. Define
