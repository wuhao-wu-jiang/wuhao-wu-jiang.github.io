Catalan Number

Posted on 2020-04-14

Let SSS be the set of binary strings with length 2n2n2n and equal number of 000 and 111's. That is, if s∈Ss \in Ss∈S, then sss contains nnn zeros and nnn ones. Define

σ(s)= number of 0 in sλ(s)= number of 1 in s\sigma(s) = \text{ number of 0 in } s \\ \lambda(s) = \text{ number of 1 in } s σ(s)= number of 0 in sλ(s)= number of 1 in s

Then

S={s:s∈(0,1)2n∧σ(s)=λ(s)}S = \{ s : s \in (0, 1)^{2n} \wedge \sigma(s) = \lambda(s) \} S={s:s∈(0,1)2n∧σ(s)=λ(s)}

Denote s[1:i]s[1:i]s[1:i] the initial segment of sss with length. Of interest is the subset P⊂SP \subset SP⊂S that contains strings whose initial segments has more (or equal) number of 000 than 111:

P≐{s:σ(s[1:i])≥λ(s[1:i]),∀i∈[2n]}⊂SP \doteq \{ s : \sigma(s[1:i]) \ge \lambda(s[1:i]), \forall i \in [2n] \} \subset S P≐{s:σ(s[1:i])≥λ(s[1:i]),∀i∈[2n]}⊂S

The cardinality of PPP is given by

∣P∣=1n+1(2nn)|P| = \frac{1}{n + 1} \binom{2n}{n} ∣P∣=n+11​(n2n​)

which is known as Catalan number.

Let S∖PS \setminus PS∖P be the set of strings that do not satisfy the requirement and T={s:s∈(0,1)2n∧σ(s)=n−1∧λ(s)=n+1}T = \{ s : s \in (0, 1)^{2n} \wedge \sigma(s) = n - 1 \wedge \lambda(s) = n + 1 \}T={s:s∈(0,1)2n∧σ(s)=n−1∧λ(s)=n+1}

The proof relies on the following theorem.

Theorem. There is a bijection fff between S∖PS \setminus PS∖P and TTT.

Before we prove it, we see how it gives Catalan number:

∣P∣=∣S∣−∣S∖P∣=(2nn)−∣T∣=(2nn)−(2nn−1)=(2nn)−(2nn)nn+1=1n+1(2nn)\begin{aligned} |P| &= |S| - |S\setminus P| \\ &= \binom{2n}{n} - |T| \\ &= \binom{2n}{n} - \binom{2n}{n - 1} \\ &= \binom{2n}{n} - \binom{2n}{n} \frac{n}{n + 1} \\ &= \frac{1}{n + 1} \binom{2n}{n} \end{aligned} ∣P∣​=∣S∣−∣S∖P∣=(n2n​)−∣T∣=(n2n​)−(n−12n​)=(n2n​)−(n2n​)n+1n​=n+11​(n2n​)​

Proof of Theorem.

⟶\longrightarrow⟶:

If s∈S∖Ps \in S \setminus Ps∈S∖P, then there exists some i∈[1,n−1]i \in [1, n - 1]i∈[1,n−1], such that σ(s[1:i])<λ(s[1:i])\sigma(s[1:i]) < \lambda(s[1:i])σ(s[1:i])<λ(s[1:i]). Then we can select the smallest such iii. Note the iii must be an odd number. Hence we can write i=2j+1i = 2j + 1i=2j+1, σ(s[1:i])=j\sigma(s[1:i]) = jσ(s[1:i])=j and λ(s[1:i])=j+1\lambda(s[1:i]) = j + 1λ(s[1:i])=j+1.

Further, as s∈Ss \in Ss∈S, we have σ(s)=λ(s)=n\sigma(s) = \lambda(s) = nσ(s)=λ(s)=n. Therefore,

σ(s[i+1:n])=n−jλ(s[i+1:n])=n−j−1\sigma(s[i + 1:n]) = n - j \\ \lambda(s[i + 1:n]) = n - j - 1 σ(s[i+1:n])=n−jλ(s[i+1:n])=n−j−1

To construct an s′∈Ts' \in Ts′∈T from sss, we can flipped every bit in s[i+1:n]s[i + 1:n]s[i+1:n], i.e.,

s′=(s[1:i],  s[i+1:n]‾)s' = (s[1:i], \ \ \overline{s[i + 1:n] } ) s′=(s[1:i],  s[i+1:n]​)

For sanity check, the number of zeros in s′s's′ is j+n−j−1=n−1j + n -j - 1 = n - 1j+n−j−1=n−1 and the number of ones is j+1+n−j=n+1j + 1 + n - j = n + 1j+1+n−j=n+1.

It is easy to see that this is a injection.

⟵\longleftarrow⟵:

If s′∈Ts' \in Ts′∈T, then there must exist some i∈[1,n−1]i \in [1, n - 1]i∈[1,n−1], such that σ(s′[1:i])<λ(s′[1:i])\sigma(s'[1:i]) < \lambda(s'[1:i])σ(s′[1:i])<λ(s′[1:i]). Further, as s′∈Ts' \in Ts′∈T, we have σ(s′)=n−1\sigma(s') = n - 1σ(s′)=n−1 and λ(s′)=n+1\lambda(s') = n + 1λ(s′)=n+1. Therefore,

σ(s′[i+1:n])=n−1−jλ(s′[i+1:n])=n+1−j−1=n−j\sigma(s'[i + 1:n]) = n - 1 - j\\ \lambda(s'[i + 1:n]) = n + 1 - j - 1 = n - j σ(s′[i+1:n])=n−1−jλ(s′[i+1:n])=n+1−j−1=n−j

To construct an s∈S∖Ps \in S \setminus Ps∈S∖P from s′s's′, we can flipped every bit in s′[i+1:n]s'[i + 1:n]s′[i+1:n], i.e.,

s=(s′[1:i],  s′[i+1:n]‾)s = (s'[1:i], \ \ \overline{s'[i + 1:n] } ) s=(s′[1:i],  s′[i+1:n]​)

For sanity check, the number of zeros in sss is j+n−j=nj + n - j = nj+n−j=n and the number of ones is j+1+n−1−j=nj + 1 + n - 1 - j = nj+1+n−1−j=n.

It is easy to see that this is a injection.
