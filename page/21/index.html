The Volume of Hamming Ball

Posted on 2021-04-02 Edited on 2022-12-22

Given a finite set \(\Sigma\) with size \(q = |\Sigma|\), and a integer \(n \in \mathbb{N}_+\), a code \(C \subseteq \Sigma^n\) is a set of strings that are far apart. The set \(\Sigma\) is called the alphabet of the code, and \(n\) is called the block length of the code. Strings in \(C\) are called codewords.

For \(x, y \in \Sigma^n\), define their Hamming distance as \[ \Delta(x, y) \doteq | \{ i \in [n] : x_i \neq y_i \} |. \]

The distance \(\Delta(C)\) (or \(d\) for a shorthand) of \(C\), is defined as the minimum distance between two different codes words in \(C\): \[ \Delta(C) = d \doteq \min_{x \neq y \in C} \Delta(x, y). \]

If we store a code \(x \in C\), and up to \((d - 1) / 2\) bits of \(x\) are corrupted, then we can still recover \(x\). Denote the ball centered at \(x\) with radius \(r \in \mathbb{R}_+\) as \[ \mathcal{B}(x, r) \doteq \{ y \in \Sigma^n : \Delta(y, x) \le r \}. \]

Fact. For \(x \in \Sigma^n\), \(r \in \mathbb{R}_+\), the size (volume) of the ball \(\mathcal{B}(x, r)\) is independent of \(x\) and is given by \[ \mathbb{Vol}(r) \doteq | \mathcal{B}(x, r) | = \sum_{i = 0}^{ \lfloor r \rfloor } \binom{n}{i} (q - 1)^i. \]

If the code \(C\) has distance \(\Delta(C) = d\), then the balls \(\{ \mathcal{B}(x, (d - 1) / 2) : x \in C \}\) should be disjoint. Therefore, we can upper bound the size \(|C|\) in terms of the size of \(\Sigma^n\), and the volume of the balls.

Theorem (Hamming Bound). Let \(C\) be a code with distance \(d \ge 3\). Then \[ |C| \le \frac{q^n }{ \mathbb{Vol} \Big( (d - 1) / 2 \Big) }. \]

We are going to prove an estimate of the volume.

Theorem (Estimate of Volume). Let \(r \in \mathbb{R}_+\). When \(r \le n - n / q\), \[ q^{ n h_q ( r / n) - O( \log n) } \le \mathbb{Vol} (r) \le q^{ n h_q ( \lfloor r \rfloor / n)}, \] where \[ \begin{aligned} h_q (\epsilon) &\doteq (1 - \epsilon) \log_q \left( \frac{1}{1 - \epsilon} \right) + (q - 1) \frac{\epsilon}{q - 1} \log_q \left( \frac{q - 1}{ \epsilon } \right) \\ &= (1 - \epsilon) \log_q \left( \frac{1}{1 - \epsilon} \right) + \epsilon \log_q \left( \frac{q - 1}{ \epsilon } \right). \end{aligned} \]

When \(q = 2\), we have the following corollary.

Corollary. Let \(r \in \mathbb{R}_+\). When \(r \le n / 2\), \[ 2^{ n \cdot h_2 ( r / n) - O( \log n) } \le \sum_{i = 0}^{ \lfloor r \rfloor } \binom{n}{i} \le 2^{ n \cdot h_2 ( \lfloor r \rfloor / n)}, \] where \(h_2( \cdot )\) is the Shannon's entropy function.

Observe that by concavity of \(\log_q (\cdot)\), we have \[ h_q (\epsilon) \le \log_q \left( (1 - \epsilon) \frac{1}{1 - \epsilon} + \epsilon \frac{q - 1}{ \epsilon } \right) = 1. \]

The inequality holds with equality when \(h_q(\epsilon)\) is maximized, i.e., when \(1 - \epsilon = \epsilon / (q - 1)\), i.e., \(1 - \epsilon = 1 / q\).

Rate of the Code

Based on the theorems of Hamming bound, and the estimate of the volume, we have \[ \begin{aligned} |C| \le \frac{q^n }{ \mathbb{Vol} \Big( (d - 1) / 2 \Big) } &\Longrightarrow n h_q( \lparen d / 2 - 1 / 2 \rparen / n) + \log_q |C| \le n + O(\log n) \\ &\Longleftrightarrow h_q( \lparen d / 2 - 1 / 2 \rparen / n) + \frac{ \log_q |C| }{ n } \le 1 + O\left(\frac{ \log_q n }{ n } \right) \\ &\Longrightarrow h_q( d / 2 n) + \frac{ \log_q |C| }{ n } \le 1 + O\left(\frac{ \log_q n }{ n } \right). \end{aligned} \]

The last \(\Longleftrightarrow\) holds, since \[ \begin{aligned} q^{ n \cdot h_q( \lparen d - 1 \rparen / (2n) ) } &= (q - 1)^r \left(\frac{n}{r} \right)^r \left(\frac{n}{n - r} \right)^{n - r} & \quad r \doteq (d - 1) / (2) \\ &\ge (q - 1)^{r'} \left(\frac{n}{r'} \right)^{r'} \left(\frac{n}{n - r'} \right)^{n - r'} q^{ - 1 / (2) } \left(\frac{n}{r'} \right)^{-1 / (2) } \left(\frac{n - r'}{n - r} \right)^{n - r'} & \quad r' \doteq d / 2 \\ &= q^{ n \cdot h_q( d / (2n) ) } \left(\frac{n}{n - r'} \right)^{n - r'} q^{ - 1 / 2 } \left(\frac{n}{r'} \right)^{-1 / 2 } \left(\frac{n - r'}{n - r} \right)^{n - r'} \\ \Longrightarrow n \cdot h_q( d / (2n) ) &\le n \cdot h_q( \lparen d - 1 \rparen / (2n) ) + \log_q \left( q^{ 1 / 2 } \left(\frac{n}{r'} \right)^{ 1 / 2 } \left(\frac{n - r}{n - r'} \right)^{n - r'} \right), \end{aligned} \]

and \[ \log_q \left( q^{ 1 / 2 } \left(\frac{n}{r'} \right)^{ 1 / 2 } \left(\frac{n - r}{n - r'} \right)^{n - r'} \right) = O \left( 1 + \log_q n + \log_q \left( 1 + \frac{1 / 2}{n - r'} \right)^{n - r'} \right) = O \left( \log_q n \right) . \]

Define the rate of the code \(R = \frac{ \log_q |C| }{ n }\), and relative distance of the code as \(\delta = \frac{d}{n}\), we get

\[ R + h_q( \delta / 2) \le 1 + O\left( \frac{\log_q n }{n} \right). \]

In what's follows, we prove the inequalities for volume estimation. WLOG, we assume that \(r\) is an integer.

Upper Bound

Theorem. For \(0 \le r \le n \cdot (1 - 1 / q)\), it holds that \(\mathbb{Vol} (r) \le q^{n h_q(r / n)}\).

Algebraic Approach

\[ \begin{aligned} (q - 1)^{ r } \left( \frac{n}{r} \right)^{ r } \left( \frac{ n }{ n - r } \right)^{n - r } &= (q - 1)^{ r } \left( \frac{n}{r} \right)^{ r } \left( \frac{ n }{ n - r } \right)^{n - r } \left( \frac{r}{n} + \frac{n - r}{n} \right)^n \\ &\ge (q - 1)^{ r } \left( \frac{n}{r} \right)^{ r } \left( \frac{ n }{ n - r } \right)^{n - r } \sum_{i = 0}^{ r} \binom{ n }{i} \left( \frac{r}{n} \right)^i \left( \frac{n - r }{n } \right)^{n - i} \\ &\ge \sum_{i = 0}^{ r} \binom{ n }{i} (q - 1)^{ i } \left( \left( \frac{n}{r} \right)^{ r - i} \left( \frac{n - r }{n } \right)^{r - i} (q - 1)^{r - i} \right) \\ &\ge \sum_{i = 0}^{ r} \binom{ n }{i} (q - 1)^{ i } \\ &= \mathbb{Vol}(r), \end{aligned} \]

where the last inequality follows from that \[ \frac{n}{r} \cdot \frac{n - r}{ n } \cdot (q - 1) = \Big( \frac{n}{r} - 1 \Big) \cdot (q - 1)\ge 1, \] as \(r / n \le 1 - 1 / q = q / (q - 1)\).

Finally, let \(\epsilon \doteq \frac{r}{n}\), we have \[ \begin{aligned} \log \mathbb{Vol}(r) &\le r \cdot \log (q - 1) + r \cdot \log \frac{1}{\epsilon} + (n - r) \cdot \log \frac{1}{1 - \epsilon} \\ &= n \left( \epsilon \cdot \log (q - 1) + \epsilon \cdot \log \frac{1}{\epsilon} + (1 - \epsilon) \cdot \log \frac{1}{1 - \epsilon} \right) \\ &= n \Big( \epsilon \cdot \log (q - 1) + \mathbb{H}_2(\epsilon) \Big) \\ &= (\log q) \cdot n \cdot h_q (\epsilon), \end{aligned} \] which proves the claim.

Entropy

We rely on sub-additivity of entropy.

Let \(X\) and \(Y\) be discrete random variables, then
1. \(H( X, Y ) \le H(X) + H(Y)\).
2. \(H(X | Y) \le H(X)\).

Proof. Consider the ball \(\mathcal{B}(1^n, r)\), and a uniform random point \(X = (X_1, ..., X_n)\) in the ball \(\mathcal{B}(1^n, r)\). Its entropy is \[ H(X) = \log \mathbb{Vol}(r). \]

We write \(Y = X_1\). By sub-additivity of entropy and by symmetry, \[ \begin{aligned} H(X) &\le H(X_1) + H(X_2) + ... + H(X_n) &= n H(Y). \end{aligned} \]

Let \(\epsilon \doteq \Pr[ Y \neq 1]\) be the probability that \(Y\) is not \(1\). Further, by symmetry, \(\Pr[Y = i]\) has equal probability for each \(i \neq 1\). Hence, \[ \begin{aligned} H(Y) &= \Pr[ Y = 1] \cdot \log \left( \frac{1}{\Pr[ Y = 1]}\right) + \sum_{i \neq 1} \Pr[ Y = i ] \cdot \log \left( \frac{1}{\Pr[ Y = i]}\right) \\ % &= (1 - \epsilon) \cdot \log \left( \frac{1}{1 - \epsilon} \right) + \epsilon \cdot \sum_{i \neq 1} \frac{ \Pr[ Y = i ] }{ \epsilon } \cdot \log \left( \frac{1}{\Pr[ Y = i]}\right) \\ % &\le (1 - \epsilon) \cdot \log \left( \frac{1}{1 - \epsilon} \right) + \epsilon \cdot \log \left( \sum_{i \neq 1} \frac{ \Pr[ Y = i ] }{ \epsilon } \frac{1}{\Pr[ Y = i]}\right) \\ &= (1 - \epsilon) \cdot \log \left( \frac{1}{1 - \epsilon} \right) + \epsilon \cdot \log \left( \frac{ q - 1 }{ \epsilon } \right) \\ \end{aligned} \] where then entropy of \(Y\) is maximized when \(Y\) is uniform of the elements that are not 1. Further, it can be verified that \(\epsilon < r / n\). As \(h_q (\epsilon)\) increases for \(\epsilon < 1 - 1 / q\), hence \[ H(Y) \le (1 - r / n) \cdot \log \left( \frac{1}{1 - r / n} \right) + \frac{r}{n} \cdot \log \left( \frac{ q - 1 }{ r / n } \right). \]

\(\square\)

Lower Bound

Theorem. For \(r \in \mathbb{R}_+\), it holds that \(\mathbb{Vol} (r) \ge q^{ n h_q ( r / n) - O( \log n) }\).

We rely on the Stirling's approximation.

Fact. For \(n \in \mathbb{N}_+\), \(\sqrt{2 \pi n} \left(\frac{n}{e} \right)^n \le n! \le e \sqrt{n} \left(\frac{n}{e} \right)^n\).

Proof. First, assume that \(r \in \mathbb{N}_+\). It holds that \[ \begin{aligned} \mathbb{Vol} (r) &\ge \binom{n}{r} (q - 1)^r \\ &\ge \frac{ \sqrt{2 \pi n } \left(\frac{n}{e} \right)^n }{ e \sqrt{r} \left(\frac{r}{e} \right)^r e \sqrt{n - r} \left(\frac{n - r}{e} \right)^{n - r} } (q - 1)^r \\ &= (q - 1)^r \left(\frac{n}{r} \right)^r \left(\frac{n}{n - r} \right)^{n - r} \sqrt{ \frac{2 \pi n }{e^4 r (n - r)} } \\ &= q^{n h_q ( r / n) - O(\log n) } \end{aligned} \] Assume that \(r \in \mathbb{R}_+\). Let \(r' = \lfloor r \rfloor\). Then \[ \begin{aligned} \mathbb{Vol} (r) \ge \mathbb{Vol} ( r' ) \ge q^{n h_q ( r' / n) - O(\log n) }. \end{aligned} \] But \[ \begin{aligned} \frac{ q^{n h_q ( r / n)} }{ q^{n h_q ( r' / n)} } \le (q - 1) \cdot \left(\frac{n}{r'} \right) \cdot \left(\frac{n - r}{n - r'} \right)^{n - r'} \in q^{ O \left( \log_q n \right) } . \end{aligned} \]

\(\square\)

Hence, the upper bound is rather tight.

Reference

[1] Anup Rao, Coding Theory Lecture 3: The Volume Bound and Finite Fields

Sparse Recovery

Posted on 2021-03-31 Edited on 2021-04-08

Model

The focus of this article is recovering a sparse vector \(x \in \mathbb{R}^n\) in the streaming setting. Initially, \(x = 0\). Then we see a sequence of updates \((i, \Delta)\) each causing the change \(x_i \leftarrow x_i + \Delta\). There are three popular studied models

  1. insertion only: \(\Delta = 1\) for each update.
  2. strict turnstile: \(\Delta\) can be negative, but it is guaranteed that \(x \ge 0\) all the time.
  3. general turnstile: \(\Delta\) can be negative, and entries of \(x\) can be negative.

Let \(\text{supp}(x) \subset [n]\) be the set of nonzero indices of \(x\). The vector \(x\) is called \(k\) sparse if \(\text{supp}(x) \le k\).

Goal. Given an integer \(k\), design a data structure such that, if \(x\) is \(k\) sparse, the structure returns \(x\) exactly; otherwise, the return can be arbitrary.

As we can't actually store real numbers, we use the following assumption:

The updates are integers and the entries of \(x\) are bounded by \(\pm M\).

With this restriction, it suffices to solve this problem in a finite field \(\mathbb{F}_p\) where \(p \ge \max \{ n, M \}\). As opposed to the naive solution that keeps all entries of \(x\) and uses \(O(n \log M)\) bits of memory, we show that the memory can be reduced to \(O( k \log (nM) )\) bits.

Algorithm

The problem is easy to solve in the insertion only model. We can use a hash set that keeps the first \(k\) occurrence of the indices. If the we detect the \(k + 1\) new index, we can just report error.

The scenario is a little more complicated for the other two models. E.g., let \(n = 4\) and \(k = 1\). Consider the following stream \[ (1, 1), (2, 1), (3, -1), (4, 1), (1, -1), (2, -1), (3, 1). \]

The vector \(x\) is finally \(1\)-sparse and the only non-zero index is revealed at the end of the stream.

Solution 1

Encode

In the first approach, we use a matrix \(H \in \mathbb{F}_p^{m \times n}\) (\(m\) to be determined) and encode \(x\) as \(\mathbb{Enc}(x) = H x\). If the following property holds, we are guaranteed to recover \(x\) if it is \(k\) sparse.

For every \(k\) sparse \(x, x' \in \mathbb{F}_p^n\), if \(x \neq x'\), then \(Hx \neq H x'\).

If \(H x \neq Hx'\), then \(H (x - x') \neq 0\). Observe that \(x - x'\) is a \(2k\) sparse vector. This is equivalent to that \(H z \neq 0\) for every nonzero \(2k\) sparse vector \(z \in \mathbb{F}_p^n\).

Such \(H\) does exists, with \(m = 2k\). Let \(a_1, a_2, ..., a_n \in \mathbb{F}_p\) be different integers. Then \(H\) is given as \[ H = \begin{bmatrix} 1 &1 &... &1 &... &1 \\ a_1 &a_2 &... &a_i &... &a_n \\ a_1^2 &a_2^2 &... &a_i^2 &... &a_n^2 \\ ...\\ a_1^{2k - 1} &a_2^{2k - 1} &... &a_i^{2k - 1} &... &a_n^{2k - 1} \end{bmatrix} \]

Let \(S = \text{supp}(z) \subset [n]\) be a set of size \(2k\). Denote \(z_S\) be the vector by keeping only the nonzero dimensions of \(z\) and \(H_S\) be an \(2k \times 2k\) matrix by keeping only columns indexed by \(S\). Then \(H z = H_S z_S\). We show that the rows of \(H_S\) are linearly independently, hence its columns. It follows that if \(z_S \neq 0 \Longrightarrow H_S z_S \neq 0\).

It is left to show that \(\forall c \in \mathbb{F}_p^{2k}\), \(c \neq 0 \rightarrow H_S^T c \neq 0\). Define the polynomial in \(\mathbb{F}_p\) \[ f(x) = c_{2k} x^{2k - 1} + c_{2k - 1} x^{2k - 1} + ... + c_1. \]

Then \(H_S^T c = ( f(a_i ) )_{i \in [S] }\). As \(f(x)\) has at most \(2k - 1\) distinct roots, and the \((a_i)_{i \in [S] }\) are different, there exists some \(i \in [S]\) such that \(f(a_i) \neq 0\).

Decode

Given \(y = \mathbb{Enc}(x) = H x\), how can we recover \(x\)? We can loop over every subset \(S \subset [n ]\) of size \(2k\) and compute \(x' = H_S^{-1} y\). If \(x'\) is \(k\) sparse, by the property of \(H\), it is the unique \(k\)-sparse vector such that \(H_S x' = y\).

Remark: The proposed method is exponential respect to \(k\). However, there is an algorithm (called syndrome decoding) to perform the recovery in \(O(k^2 poly(\log p))\) time.

Space and Time

Space. The algorithm uses \(O(k \log (nM))\) bits.

This is because we don't store \(H\). For the finite field \(\mathbb{F}_p\), there exists at least one primitive element whose powers give all nonzero elements. We can pick such a primitive element \(a\), and set \(a_1 = a\), \(a_2 = a^2\), ..., \(a_n = a^n\). Storing \(y = Hx\) requires only \(O(k \log (nM))\) bits.

Time. Processing each update \((i, \Delta)\) in the stream takes \(O(k)\) time.

Solution 2

We first show how to recover an \(1\) sparse vector when negative updates are allowed, then use it as a black box to recover \(k\) sparse vector.

1 sparse recovery

We show an algorithm that successes with probability \(n / p\).

First, we pick a non-zero integer \(z\) uniformly at random from \(\mathbb{F}_p\). Let \[ H = \begin{bmatrix} 1 &1 &1 &... &1 \\ 1 &2 &3 &... &n \\ z &z^2 &z^3 &... &z^n \end{bmatrix}. \]

The algorithm consists of 3 parts.

Initialize:
\(\qquad\) \(w \leftarrow \{ 0 \}^3\)
\(\qquad\) Sample \(z\) uniformly from \(1, ..., p - 1\).

Updates: \((i, \Delta)\)
\(\qquad\) \(w \leftarrow w + \Delta \cdot H_i\), where \(H_i\) is the \(i\)th column of \(H\).

Output:
\(\qquad\) If \(w_1 = 0\) or \(w_3 \neq w_1 \cdot z^{w_2 / w_1}\):
\(\qquad\qquad\) Return \(x\) is not \(1\) sparse;
\(\qquad\) Else:
\(\qquad\qquad\) Return \(j \doteq w_2 / w_1\)

Failure Analysis

Theorem. The algorithm fails with probability \(n / p\).

Observe that \[ w_1 = \sum_{i \in [n] } x_i, \qquad w_2 = \sum_{i \in [n] } i \cdot x_i, \qquad w_3 = \sum_{i \in [n] } z^i \cdot x_i. \]

If \(w_1 = 0\), either \(x = 0\) or there is cancellation. Conditioned on \(w_1 \neq 0\), if \(x\) is \(1\) sparse, then \[ w_1 = x_j, \qquad w_2 = j \cdot x_j, \qquad w_3 = z^j \cdot x_j. \]

Hence, \(j = w_2 / w_1\) is exactly the non-zero index, and \(w_3 = w_1 \cdot z^{j}\).

If \(x\) is not \(1\) sparse, then \[ w_3 - w_1 \cdot z^{j} \] is a polynomial of degree at most \(n\). It has at most \(n\) distinct roots with one of them being \(0\). The probability that \(z\) being a root is at most \((n - 1) / (p - 1) \le n / p\).

Space and Time

Space. The algorithm uses \(O(\log (nM))\) bits.

Time. Processing each update \((i, \Delta)\) takes \(O(1)\) time.

\(k\) sparse recovery

To apply the \(1\) sparse recovery data structure, we partition \(x\) into \(2k\) subsets randomly. In particular, we use a pairwise hash function that maps each \(i \in [n]\) to \([2k]\) uniformly at random. For \(\ell \in [2k]\), let \(S_\ell\) be the set of indices mapped to \(\ell\): \[ S_\ell \doteq \{ i \in [n]: h(i) = \ell \}. \]

Then we run \(1\) sparse recovery on each partition. If some partition contains only one nonzero index, then we recover it successfully.

Theorem. If \(x\) is \(k\) sparse, an nonzero dimension \(i\) is recovered with probability \(1 / 2\).

Proof. It suffices to prove that \(i\) is the only nonzero index in \(S_{h (i) }\). For \(j \in \text{supp}(x)\), define the indicator variable \(X_j\) which equals \(1\) if \(h(j) = h(i)\) and equals 0 otherwise. By Markov's inequality and linearity of expectation, \[ \Pr\left[ \sum_{ i \neq j \in \text{supp}(x) } X_j \ge 1 \right] \le \mathbb{E} \left[ \sum_{ i \neq j \in \text{supp}(x) } X_j \right] = \sum_{ i \neq j \in \text{supp}(x) } \mathbb{E} \left[ X_j \right] \le \frac{k - 1}{2k} \le \frac{1}{2}. \]

Remark. The Markov's inequality is rather tight here. Even if we have a fully independent hash function \(h\), the probability that none of the nonzero indices collides with \(i\) is given by \[ \left( 1 - \frac{1}{2k} \right)^{k - 1} \lessapprox \exp \left( - \frac{k - 1}{2k} \right) \approx \exp \left( - \frac{1}{2} \right) \approx 0.60653. \]

\(\square\)

Corollary. Running \(\log (k / \delta)\) independent copies of the above structure reduce the failure probability to \(\delta\).

Space and Time

Space. The algorithm uses \(O(k (\log (k / \delta)) \log (nM))\) bits.

Time. Processing each update \((i, \Delta)\) takes \(O(\log (k / \delta))\) time.

Reference

Norm of Random Variables

Posted on 2021-03-27 Edited on 2024-03-11

Fix probability space \((\Omega, \mathcal{F}, \Pr)\), for real number \(p > 0\), let \(L_p\) denote the vector space of real value random variable for which \(\mathbb{E} |X|^p < \infty\).

For \(p \ge 1\), the \(p\) norm \(\Vert X \Vert_p\) of a random variable \(X \in L_p\) is defined as \[ \Vert X \Vert_p = \left( \mathbb{E} \big[ |X|^p \big] \right)^{1 / p}. \]

Norm

\(\Vert X \Vert_p\) is a norm.

Proof. This can be proven directly by Minkowski’s inequality. For completeness, we give the details here. Let \(X, Y \in L_p\), and let \(a = \Vert X \Vert_p\) and \(b = \Vert Y \Vert_p\). We want to prove that \[ \Vert X + Y \Vert_p \le a + b \Longleftrightarrow \left\Vert \frac{a}{ a + b} \cdot \frac{X}{a} + \frac{a}{ a + b} \cdot \frac{Y}{a} \right\Vert_p \le 1. \]

It suffices to assume that \(\Vert X \Vert_p = \Vert Y \Vert_p = 1\) and prove that

For \(t \in [0, 1]\), \(\Vert t X + (1 - t) Y \Vert_p \le 1\).

This is much easier to prove as we can take the power of \(p\) of both side and show that: \[ \begin{aligned} \Vert t X + (1 - t) Y \Vert_p^p &= \mathbb{E}[ |t X + (1 - t) Y |^p ] \\ &\le \mathbb{E}[ | tX |^p + | (1 - t) Y|^p ] \\ &= \mathbb{E}[ | tX |^p ] + \mathbb{E} [ | ( 1 - t) Y|^p ] \\ &= t^p \Vert X \Vert_p^p + ( 1 - t)^p \Vert Y \Vert_p^p \\ &\le t + (1 - t) \\ &= 1. \end{aligned} \]

\(\square\)

Properties

Theorem. For \(0 \le p \le q\), \(\big( \mathbb{E} [ |X|^p ] \big)^{1 / p} \le \big( \mathbb{E} [ |X|^q ] \big)^{1 / q}\).

Corollary For \(1 \le p \le q\), \(\Vert X \Vert_p \le \Vert X \Vert_q\).

Proof. By convexity of the function \(y = |x|^{q / p}\), we have \[ \begin{aligned} \Big| \mathbb{E} \big[ |X|^p \big] \Big|^{q / p} \le \mathbb{E} \left[ \Big| |X|^p \Big|^{q / p} \right] = \mathbb{E} \left[ |X|^q \right] \end{aligned} \] Raising both hand sides to the power of \(1 / q\) finish the proof.

\(\square\)

Let \(X_1, ..., X_n \in L_p\) be independent random variables and let \(r_1, ... ,r_n\) be independent Rademacher random variables. Then \[ \left\Vert \sum_{i \in [n] } X_i - \mathbb{E} \left[ \sum_{i \in [n] } X_i \right] \right\Vert_p \le 2 \cdot \left\Vert \sum_{i \in [n] } r_i X_i \right\Vert_p \]

Proof. For \(i \in [n]\), let \(Y_i\) be independent copy of \(X_i\). Then \[ \begin{aligned} \left\Vert \sum_{i \in [n] } X_i - \mathbb{E} \left[ \sum_{i \in [n] } X_i \right] \right\Vert_p &= \left\Vert \sum_{i \in [n] } X_i - \mathbb{E}_Y \left[ \sum_{i \in [n] } Y_i \right] \right\Vert_p \\ &= \left( \mathbb{E}_X \left[ \left| \sum_{i \in [n] } X_i - \mathbb{E}_Y \left[ \sum_{i \in [n] } Y_i \right] \right|^p \right] \right)^{1 / p} \\ &\le \left( \mathbb{E}_X \mathbb{E}_Y \left[ \left| \sum_{i \in [n] } X_i - \sum_{i \in [n] } Y_i \right|^p \right] \right)^{1 / p} &( \text{Jensen}) \\ &= \left\Vert \sum_{i \in [n] } (X_i - Y_i) \right\Vert_p \end{aligned} \]

Let \(Z = (Z_1, ..., Z_n)\) be a sequence of independent random variables, such that either \(Z_i = X_i - Y_i\) or \(Z_i = Y_i - X_i\). Let \(\Gamma = \{ Z \}\) be the set of \(2^n\) such sequences. As \(X_i - Y_i\) and \(Y_i - X_i\) have the same distributions, we have for \(Z \in \Gamma\), \[ \mathbb{E}_Z \left[ \left| \sum_{i \in [n] } Z_i \right|^p \right] = \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in [n]} X_i - Y_i \right|^p \right]. \]

Therefore, \[ \frac{1}{ |\Gamma| } \sum_{ Z \in \Gamma } \mathbb{E}_Z \left[ \left| \sum_{i \in [n] } Z_i \right|^p \right] = \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in [n]} X_i - Y_i \right|^p \right]. \]

If \(R = (r_1, ..., r_n)\) are Rademacher random variables, then \((r_1 (X_1 - Y_1), ..., r_n(X_n - Y_n)\) gives a uniformly sequence in \(\Gamma\). It holds that \[ \mathbb{E}_R \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in [n]} r_i (X_i - Y_i) \right|^p \right] = \frac{1}{ |\Gamma| } \sum_{ Z \in \Gamma } \mathbb{E}_Z \left[ \left| \sum_{i \in [n] } Z_i \right|^p \right] = \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in [n]} X_i - Y_i \right|^p \right]. \]

Hence, \[ \begin{aligned} \left\Vert \sum_{i \in [n] } X_i - \mathbb{E} \left[ \sum_{i \in [n] } X_i \right] \right\Vert_p &\le \left\Vert \sum_{i \in [n] } (X_i - Y_i) \right\Vert_p \\ &= \left\Vert \sum_{i \in [n] } r_i(X_i - Y_i) \right\Vert_p \\ &\le 2 \left\Vert \sum_{i \in [n] } r_i X_i \right\Vert_p \end{aligned} \]

\(\square\)

Remark: We can simply the argument as follows. For a fixed sequence \((r_1,..., r_n) \in \{-1, 1 \}^n\), the sequence \((r_1(X_1 - Y_1), ..., r_n(X_n - Y_n) )\) has the same distribution as \((X_1 - Y_n, ..., X_n - Y_n)\). Averaging over all such possible sequences gives that \[ \left\Vert \sum_{i \in [n] } (X_i - Y_i) \right\Vert_p = \left\Vert \sum_{i \in [n] } r_i(X_i - Y_i) \right\Vert_p. \]

Let \(X_1, ..., X_n\) be independent and mean zero random variables and let \(Y_1, ..., Y_n\) be independent copies of them. Then for any \((a_{i, j} )\) and for \(p \ge 1\), it holds that \[ \begin{aligned} \left\Vert \sum_{i \neq j} a_{i,j} X_i X_j \right\Vert_p \le 4\left\Vert \sum_{i \neq j} a_{i,j} X_i Y_j \right\Vert_p \end{aligned} \]

Proof. Let \(B_1, ..., B_n\) be independent Bernoulli random variables each of expectation \(1 / 2\). Then \[ \begin{aligned} \left\Vert \sum_{i \neq j} a_{i,j} X_i X_j \right\Vert_p &= \left( \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j \right|^p \right]\right)^{1 / p} \\ &= \left( \mathbb{E}_X \left[ \left| 4 \mathbb{E}_B \left[ \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right] \right|^p \right]\right)^{1 / p} \\ &\le \left( \mathbb{E}_X \left[ \mathbb{E}_B \left[ 4 \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right] \right]\right)^{1 / p} & (\text{Jensen} ) \\ &= 4 \left( \mathbb{E}_X \mathbb{E}_B \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right]\right)^{1 / p} \\ &= 4 \left( \mathbb{E}_B \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right]\right)^{1 / p} \end{aligned} \]

Question to ponder: Under what condition can we exchange the order of expectation (integration)?

But \[ \mathbb{E}_B \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right] \le \max_B \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right]. \]

Let \[ B^* = (b^*_1, ..., b^*_n) = \arg \max_{B} \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right] \]

Define \(S \doteq \{ i : b^*_i = 1 \}\). Then \[ \begin{aligned} \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j b^*_i (1 - b^*_j) \right|^p \right] &= \mathbb{E}_X \left[ \left| \sum_{i \in S, j \notin S } a_{i,j} X_i X_j \right|^p \right] \\ &= \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in S, j \notin S } a_{i,j} X_i Y_j \right|^p \right] \end{aligned} \]

Let \(Z_1, ..., Z_n\) and \(Z_1', ..., Z_n'\) both be independent copies of \(X_1, ..., X_n\). Then

\[ \begin{aligned} \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in S, j \notin S } a_{i,j} X_i Y_j \right|^p \right] &= \mathbb{E}_{X, Y} \left[ \left| \mathbb{E}_{Z, Z'} \left( \sum_{i \in S} a_{i,j} X_i \cdot \left( \sum_{j \notin S} Y_j + \sum_{j \in S} Z_j' \right) + \sum_{i \notin S} a_{i,j} Z_i \cdot \left( \sum_{j \notin S} Y_j + \sum_{j \in S} Z_j' \right) \right) \right|^p \right] & ( \mathbb{E}[Z_i ] = \mathbb{E} [ Z_j' ] = 0 ) \\ &\le \mathbb{E}_{X, Y} \mathbb{E}_{Z, Z'} \left[ \left| \sum_{i \in S} a_{i,j} X_i \cdot \left( \sum_{j \notin S} Y_j + \sum_{j \in S} Z_j' \right) + \sum_{i \notin S} a_{i,j} Z_i \cdot \left( \sum_{j \notin S} Y_j + \sum_{j \in S} Z_j' \right) \right|^p \right] & (\text{Jensen} ) \\ &=\mathbb{E}_{X, Y} \left[ \left| \sum_{i , j } a_{i,j} X_i Y_j \right|^p \right]. \end{aligned} \]

\(\square\)
