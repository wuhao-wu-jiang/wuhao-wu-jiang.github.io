PAC Learning Basic

Posted on 2020-11-12 Edited on 2021-11-09

We discuss a toy model of probably approximately correct (PAC) learning [1].
The setting focuses on a ddd-dimension Boolean hypercube {0,1}d\{0, 1\}^d{0,1}d and a set of functions H={h:{0,1}d→{0,1}}\mathcal{H} = \{ h : \{0, 1\}^d \rightarrow \{0, 1 \} \}H={h:{0,1}d→{0,1}}.
A function h∈Hh \in \mathcal{H}h∈H is called a hypothesis or a concept, which assigns each vertex in the hypercube a label 0 or 1.

The goal of PAC learning is to learn an unknown hypothesis, denoted as h∗∈Hh^* \in \mathcal{H}h∗∈H, from a dataset, which consists of nnn points X1,X2,…,Xn∈{0,1}nX_1, X_2, \ldots, X_n \in \{0, 1\}^nX1​,X2​,…,Xn​∈{0,1}n sampled independently from an unknown distribution DDD (over the hypercube), as well as their labels Y1=h∗(X1),Y2=h∗(X2),…,Yn=h∗(Xn)Y_1 = h^*(X_1), Y_2 = h^*(X_2), \ldots, Y_n = h^*(X_n)Y1​=h∗(X1​),Y2​=h∗(X2​),…,Yn​=h∗(Xn​).

We can state the algorithm for PAC learning in just one sentence.

Output any hypothesis h′∈Hh' \in \mathcal{H}h′∈H that is consistent with the sampled dataset, i.e.,

h′(Xi)=Yi,∀i∈[n] h'(X_i) = Y_i, \qquad \forall i \in [n] h′(Xi​)=Yi​,∀i∈[n]

Such an hypothesis always exists, as h∗h^*h∗ satisfies this constraint.
We have the classic PAC learning theorem.

Theorem. Let X∈{0,1}nX \in \{0, 1\}^nX∈{0,1}n be a random point sampled from the unknown distribution, then for a given δ∈(0,1)\delta \in (0, 1)δ∈(0,1),

Pr⁡[h′(X)≠h∗(X)]≥log⁡∣H∣+log⁡1δ2n \Pr[ h'(X) \neq h^*(X)] \ge \sqrt{ \log |\mathcal{H}| + \log \frac{ 1 }{ \delta } \over 2n} Pr[h′(X)=h∗(X)]≥2nlog∣H∣+logδ1​​​

with probability at most δ\deltaδ.

The PAC learning theorem states that the probability that h′h'h′ mislabels a point is bounded by log⁡∣H∣+log⁡(1/δ)2n\sqrt{ \log |\mathcal{H}| + \log ( 1 / \delta ) \over 2n}2nlog∣H∣+log(1/δ)​​.

Caveat. In both theorem, we see a log⁡H\log \mathcal{H}logH term in the failure probability, which is due to the use of union bound and can be roughly considered as the complexity of the hypothesis family. It is tempting to think that this could be avoid as follows.

For the outputted h′h'h′, Z1,…,ZnZ_1, \ldots, Z_nZ1​,…,Zn​ are i.i.d samples. Therefore, the probability of all of them being 000 is give by

The argument is wrong. For the outputted h′h'h′, the Z1,…,ZnZ_1, \ldots, Z_nZ1​,…,Zn​ are not i.i.d samples, as h′h'h′ is selected according to Z1,…,ZnZ_1, \ldots, Z_nZ1​,…,Zn​ and depends on them.

To make it more concrete, suppose that there is an hhh, such that μh=log⁡(1/δ)n\mu_h = \frac{ \log ( 1 / \delta ) } {n}μh​=nlog(1/δ)​. Consider the following experiment

  1. Sample nnn points independently the distribution DDD.
  2. Pick an h′h'h′ that makes no prediction error on the samples.

If we repeat the experiment NNN times for some positive integer NNN, then the fixed hhh makes no mistake in roughly (1−δ)N(1 - \delta) N(1−δ)N of the experiments and constitutes an candidate of h′h'h′. However, on the other roughly δN\delta NδN experiments, where hhh makes prediction error, it is no longer considered as the candidate of h′h'h′. The selection procedure of h′h'h′ ignores bad event of hhh automatically.

We could also investigate the following example. Let h∗=h0h^* = h_0h∗=h0​. There are also other two hypothesis h1h_1h1​ and h2h_2h2​. Let SSS be the sample space. It holds that

∣h2(x)−h∗(x)∣={2ϵ,∀x∈S∖Sh20,  ∀x∈Sh2 |h_2(x) - h^*(x) | = \begin{cases} 2 \epsilon, \forall x \in S \setminus S_{h_2} \\ 0,\ \ \forall x \in S_{h_2} \end{cases} ∣h2​(x)−h∗(x)∣={2ϵ,∀x∈S∖Sh2​​0,  ∀x∈Sh2​​​

Moreover, Pr⁡[Sh1]=Pr⁡[Sh2]=δ\Pr[S_{h_1}] = \Pr[S_{h_2} ] = \deltaPr[Sh1​​]=Pr[Sh2​​]=δ. How, if we get a sample in Sh1S_{h_1}Sh1​​, then we can output h′h'h′ as h0h_0h0​ or h1h_1h1​, since both of them make no error when x∈Sh1x \in S_{h_1}x∈Sh1​​. Similarly, if x∈Sh2x \in S_{h_2}x∈Sh2​​, we can output either h0h_0h0​ or h2h_2h2​. We call Sh1S_{h_1}Sh1​​ and Sh2S_{h_2}Sh2​​ the bad areas. When xxx is in the bad area of any hypothesis, we can’t distinguish this hypothesis from the true hypothesis. That is why we need to use union bound to bound the total probabilities of these bad areas.

Gaussian Distribution

We show how Gaussian distribution (a.k.a. normal distribution) is constructed and list some of its basic properties.

Suppose we would like to have a distribution in R\mathbb{R}R, such that

  1. It is centered at the origin.
  2. Its density function decreases exponentially with respect to the square distance to the origin.

Remark: if we replace 2 with “a density function decreases exponentially with the distance to the origin”, we obtain Laplace distribution.

The density function p(x)p(x)p(x) should be inversely and exponentially proportional to x2x^2x2:

p(x)∝exp⁡(−x2) p(x) \propto \exp(-x^2 ) p(x)∝exp(−x2)

Denote MMM the value of

M≐∫−∞∞exp⁡(−x2) dx. M \doteq \int_{-\infty}^\infty \exp(-x^2) \ dx. M≐∫−∞∞​exp(−x2) dx.

It is easy to check that MMM is finite.
Then the close form of p(x)p(x)p(x) is

p(x)=1Mexp⁡(−x2). p(x) = \frac{1}{M} \exp(- x^2). p(x)=M1​exp(−x2).

It is easy to verity that p(x)p(x)p(x) is a valid density function: ∫−∞∞p(x) dx=1\int_{-\infty}^\infty p(x) \ dx = 1∫−∞∞​p(x) dx=1.

Background

Definition. Given fff and a variance parameter σ2\sigma^2σ2, the output of the Gaussian mechanism is a random variable defines as
