Coupling Lemma

Posted on 2021-11-01 Edited on 2021-11-02

Let \(\mu\) and \(\nu\) be two probability measures over a finite set \(\Omega\). A probability measure \(\omega\) over \(\Omega \times \Omega\) is a coupling of \((\mu, \nu)\) if its marginals are \(\mu\) and \(\nu\); that is

  1. \(\forall x \in \Omega\), \(\sum_{y \in \Omega} \omega(x, y) = \mu(x)\),
  2. \(\forall y \in \Omega\), \(\sum_{x \in \Omega} \omega(x, y) = \nu(x)\).

Lemma 1. For any coupling \(\omega\) of \((\mu, \nu)\), if the random variable \((X, Y)\) is distributed according to \(\omega\), then \[ \Pr[ X \neq Y ] \ge \Vert \mu - \nu \Vert_{TV}. \] 2. There exists a coupling \(\omega^*\) of \((\mu, \nu)\) for which \[ \Pr[ X \neq Y ] = \Vert \mu - \nu \Vert_{TV}. \]

Proof. First, observe that for every measurable set \(S \subset \Omega\), it holds that \[ \begin{aligned} \mu(S) - \nu(S) = \Pr [ X \in S ] - \Pr [ Y \in S ] \le \Pr[ X \in S, Y \notin S ] \le \Pr[ X \neq Y ]. \end{aligned} \] It follows that \[ \Vert \mu - \nu \Vert_{TV} = \max_{ \text{ measurable } S \subset \Omega} \big( \mu(S) - \nu(S) \big) \le \Pr[ X \neq Y ]. \]

Next, we show how to construct \(\omega^*\). For each \(z \in \Omega\), define \[ \omega^*(z, z) = \min \{ \mu(z), \nu(z) \}. \] Clearly it holds that \[ \begin{aligned} \Pr[ X \neq Y ] = 1 - \sum_{ z \in \Omega } \min \{ \mu(z), \nu(z) \} = \Vert \mu - \nu \Vert_{TV}. \end{aligned} \]

It is left to assign the probability mass \(\Vert \mu - \nu \Vert_{TV}\) to \((x, y) \in \Omega \times \Omega\), for \(x \neq y\).

Case 1 \(\big( \Vert \mu - \nu \Vert_{TV} = 0 \big)\). Then we can set \(\omega^*(x, y) = 0\) for all \(x \neq y\). Further, it holds that \(\mu(z) = \nu(z)\) for all \(z \in \Omega\). Therefore, \[ \begin{aligned} \sum_{ y \in \Omega } w^* (x, y) = w^*(x, x) = \min \{ \mu(x), \nu(x) \} = \mu( x ), \\ \sum_{ x \in \Omega } w^* (x, y) = w^*(y, y) = \min \{ \mu(y), \nu(y) \} = \nu( y )\,. \end{aligned} \]

Case 2 \(\big( \Vert \mu - \nu \Vert_{TV} \neq 0 \big)\). Denote \(S_+ \doteq \{ z \in \Omega: \mu(z) \ge \nu(z) \}\). Define two probability mass functions \[ \lambda_1 \doteq \frac{ \mu(z) - \nu(z) }{ \Vert \mu - \nu \Vert_{TV} } \cdot \mathbf{1}[z \in S_+],\, \lambda_2 \doteq \frac{ \nu(z) - \mu(z) }{ \Vert \mu - \nu \Vert_{TV} } \cdot \mathbf{1}[z \notin S_+]. \]

Then \(\lambda_1 ( \Omega ) = \lambda_1 (S_+) = \lambda_2 (\Omega) = \lambda_2 ( \bar S_+ ) = 1\).

Now, for \(x, y \in \Omega, x \neq y\), define \[ \omega^*(x, y) = \Vert \mu - \nu \Vert_{TV} \cdot \lambda_1 (x) \cdot \lambda_2 (y) . \]

Also note that for all \(z \in \Omega\), \(\Vert \mu - \nu \Vert_{TV} \cdot \lambda_1 (z) \cdot \lambda_2 (z) = 0\). Therefore, \[ \begin{aligned} \sum_{ y \in \Omega } w^* (x, y) &= w^*(x, x) + \sum_{ x \neq y } w^* (x, y) \\ &= \min \{ \mu(x), \nu(x) \} + \sum_{ y \in \Omega } \Vert \mu - \nu \Vert_{TV} \cdot \lambda_1 (x) \cdot \lambda_2 (y) \\ &= \min \{ \mu(x), \nu(x) \} + \big( \mu(x) - \nu(x) \big) \cdot \mathbf{1}[x \in S_+] \\ &= \mu(x). \end{aligned} \]

Similarly, we can verify that for each \(y \in \Omega\), \[ \sum_{y \in \Omega} \omega^*(x, y) = \nu(y). \]

\(\blacksquare\)

Reference

[1] Constantinos Daskalakis, 6.896 Probability and Computation, Lecture 3, 2011.
