Fano’s Inequality

Posted on 2021-07-14 Edited on 2021-07-25

Suppose that Alice wants to transmit a random variable XXX to Bob via a noisy channel.
The noisy channel is modelled by a stochastic mapping fff, and Bob receives a randomized version of XXX, denoted as Y=f(X)Y = f(X)Y=f(X).
Bob post-processes YYY by another stochastic mapping ggg to produce an estimate X^\hat XX^ of XXX.
The mapping ggg depends only on YYY and therefore X^\hat XX^ and XXX are conditionally independent given YYY.
The random variables XXX, YYY and X^\hat XX^ forms a Markov chain, denoted as

X→Y→X^. X \rightarrow Y \rightarrow \hat X. X→Y→X^.

Denote pe=Pr⁡[X^≠X]p_e = \Pr[ \hat X \neq X]pe​=Pr[X^=X] the probability of estimation error.
If the channel is noiseless, then H(X∣Y)=0H(X \mid Y ) = 0H(X∣Y)=0, and the identity mapping g(Y)≐Yg(Y) \doteq Yg(Y)≐Y has error probability pe=0p_e = 0pe​=0.
As the noisy of the channel increases, H(X∣Y)H(X \mid Y)H(X∣Y) increases, so as pep_epe​.
H(X∣Y)H(X \mid Y)H(X∣Y) can be viewed as the information lost incurred by the channel.
The information lost is small, only if when pep_epe​ is small, which is established by the following theorem.

Theorem (Fano’s Inequality). Suppose that the random variables X→Y→X^X \rightarrow Y \rightarrow \hat XX→Y→X^ forms a Markov chain.
Let pe≐Pr⁡[X≠X^]p_e \doteq \Pr[ X \neq \hat X]pe​≐Pr[X=X^].
The information lost of H(X∣Y)H(X \mid Y)H(X∣Y) is bounded by:

H(X∣Y)≤H(X∣X^)≤H2(pe)+pelog⁡(∣Ω∣−1), H(X \mid Y) \le H ( X \mid \hat X) \le H_2(p_e) + p_e \log ( | \Omega | - 1), H(X∣Y)≤H(X∣X^)≤H2​(pe​)+pe​log(∣Ω∣−1),

where H2(p)≐p⋅(log⁡1p)+(1−p)⋅log⁡11−pH_2(p) \doteq p \cdot \big( \log \frac{1}{p} \big) + (1 - p) \cdot \log \frac{1}{1 - p}H2​(p)≐p⋅(logp1​)+(1−p)⋅log1−p1​.

Remark. Note that

lim⁡pe→0H2(p2)+pelog⁡(∣Ω∣−1)=0, \lim_{p_e \rightarrow 0 } H_2 (p_2) + p_e \log ( | \Omega | - 1 ) = 0, pe​→0lim​H2​(p2​)+pe​log(∣Ω∣−1)=0,

which complies with our discussion before.

Proof. Let EEE be an indicator random variable for the event X≠X^X \neq \hat XX=X^.

Via chain rule for entropy, we expand H(E,X∣X^)H(E, X \mid \hat X)H(E,X∣X^) in two different ways

H(E,X∣X^)=H(X∣X^)+H(E∣X,X^)=H(E∣X^)+H(X∣E,X^). \begin{aligned} H(E, X \mid \hat X) &= H( X \mid \hat X ) + H( E \mid X, \hat X) \\ &= H( E \mid \hat X ) + H( X \mid E, \hat X). \end{aligned} H(E,X∣X^)​=H(X∣X^)+H(E∣X,X^)=H(E∣X^)+H(X∣E,X^).​

We see that

  • H(E∣X,X^)=0H(E \mid X, \hat X) = 0H(E∣X,X^)=0, since EEE is a function of XXX and X^\hat XX^, the conditional entropy.

  • H(E∣X^)≤H(E)=H2(pe)H( E \mid \hat X ) \le H(E) = H_2 (p_e)H(E∣X^)≤H(E)=H2​(pe​), since conditioning reduces entropy and EEE is a binary random variable.

  • H(X∣E,X^)≤pelog⁡(∣Ω∣−1)H(X \mid E, \hat X) \le p_e \log ( | \Omega | - 1)H(X∣E,X^)≤pe​log(∣Ω∣−1), which is obtained as follows:

    H(X∣E,X^)=peH(X∣X^,E=1)+(1−pe)H(X∣X^,E=0)≤pelog⁡(∣Ω∣−1). \begin{aligned} H(X \mid E, \hat X) &= p_e H( X \mid \hat X, E = 1) + (1 - p_e) H(X \mid \hat X, E = 0) \\ &\le p_e \log ( | \Omega | - 1). \end{aligned} H(X∣E,X^)​=pe​H(X∣X^,E=1)+(1−pe​)H(X∣X^,E=0)≤pe​log(∣Ω∣−1).​

    The last inequality holds, since given X^\hat XX^,

    1. when E=0E = 0E=0, there is no randomness about XXX; and

    2. when E=1E = 1E=1, XXX has ∣Ω−1∣|\Omega - 1|∣Ω−1∣ possible outcomes and its randomness is less than the one of uniform distribution over the possible outcomes.

Combining these results, we get

H(E,X∣X^)=H(X∣X^)≤H2(p2)+pelog⁡(∣Ω∣−1). \begin{aligned} H(E, X \mid \hat X) = H( X \mid \hat X ) \le H_2 ( p_2 ) + p_e \log ( | \Omega | - 1). \end{aligned} H(E,X∣X^)=H(X∣X^)≤H2​(p2​)+pe​log(∣Ω∣−1).​

It is left to prove that

H(X∣Y)≤H(X∣X^). H(X \mid Y) \le H( X \mid \hat X). H(X∣Y)≤H(X∣X^).

Via the data-processing inequality for the Markov chain X→Y→X^X \rightarrow Y \rightarrow \hat XX→Y→X^:

I(X;Y)≥I(X;X^), I(X ; Y) \ge I(X ; \hat X), I(X;Y)≥I(X;X^),

which implies

H(X∣X^)=H(X)−I(X;X^)≥H(X)−I(X;Y)=H(X∣Y). H( X \mid \hat X) = H(X) - I(X ; \hat X) \ge H(X) - I(X ; Y) = H(X \mid Y). H(X∣X^)=H(X)−I(X;X^)≥H(X)−I(X;Y)=H(X∣Y).

□\square□
