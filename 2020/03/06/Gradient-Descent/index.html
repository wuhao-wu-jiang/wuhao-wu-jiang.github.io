Gradient Descent

Posted on 2020-03-06 Edited on 2020-03-09

Gradient descent is a greedy search method for optimization. It approximates a function locally by a linear function. Its analysis varies depending on the what kinds of constraints you place on it.

We study a baby version of the method. The problem at hand is given by:

min⁡x∈Rnf(x)\min_{x \in \mathbb{R }^n} f(x) x∈Rnmin​f(x)

where fff is twice continuous differentiable and convex. The convexity of fff implies that it has a unique minimum value min⁡x∈Rnf(x)\min_{x \in \mathbb{R }^n} f(x)minx∈Rn​f(x). Assume that fff indeed achieves minimum at some point and denote the optimal point as x∗x^*x∗. Note that is some case, fff can not achieve minimum value at any point. Consider f(x)=exf(x) = e^xf(x)=ex. Then min⁡x∈Rex=0\min_{x \in \mathbb{R } } e^x = 0minx∈R​ex=0. fff can approach arbitrary close to 000 but never achieve that value.

Before we delve into the details, we need to define some frequently used notations.

  • The gradient ∇f(x):Rn→Rn\nabla f(x):\mathbb{R }^n \rightarrow \mathbb{R }^n∇f(x):Rn→Rn is the vector of partial derivatives of fff at point xxx:

∇f(x)=[∂f(x)∂x1,∂f(x)∂x2,..,∂f(x)∂xn]\nabla f(x) = \left[ \frac{\partial f (x) }{\partial x_1}, \frac{\partial f (x)}{\partial x_2}, .., \frac{\partial f (x) }{\partial x_n} \right] ∇f(x)=[∂x1​∂f(x)​,∂x2​∂f(x)​,..,∂xn​∂f(x)​]

  • The Hessian ∇2f(x):Rn→Rn×n\nabla^2 f(x) : \mathbb{R }^n \rightarrow \mathbb{R }^{n \times n}∇2f(x):Rn→Rn×n is the matrix of second order derivatives of fff at xxx:

    ∇f(x)=[∂2f(x)∂x1∂x1,∂2f(x)∂x1∂x2,...,∂2f(x)∂x1∂xn∂2f(x)∂x2∂x1,∂2f(x)∂x2∂x2,...,∂2f(x)∂x2∂xn...∂2f(x)∂xn∂x1,∂2f(x)∂xn∂x2,...,∂2f(x)∂xn∂xn]\nabla f(x) = \left[ \begin{aligned} \frac{\partial^2 f (x)}{\partial x_1 \partial x_1}, \frac{\partial^2 f (x)}{\partial x_1 \partial x_2}, &..., \frac{\partial^2 f (x)}{\partial x_1 \partial x_n} \\ \frac{\partial^2 f (x)}{\partial x_2 \partial x_1}, \frac{\partial^2 f (x)}{\partial x_2 \partial x_2}, &..., \frac{\partial^2 f (x)}{\partial x_2 \partial x_n} \\ &...\\ \frac{\partial^2 f (x)}{\partial x_n \partial x_1}, \frac{\partial^2 f (x)}{\partial x_n \partial x_2}, &..., \frac{\partial^2 f (x)}{\partial x_n \partial x_n} \\ \end{aligned} \right] ∇f(x)=⎣⎢⎢⎢⎢⎢⎢⎢⎢⎢⎡​∂x1​∂x1​∂2f(x)​,∂x1​∂x2​∂2f(x)​,∂x2​∂x1​∂2f(x)​,∂x2​∂x2​∂2f(x)​,∂xn​∂x1​∂2f(x)​,∂xn​∂x2​∂2f(x)​,​...,∂x1​∂xn​∂2f(x)​...,∂x2​∂xn​∂2f(x)​......,∂xn​∂xn​∂2f(x)​​⎦⎥⎥⎥⎥⎥⎥⎥⎥⎥⎤​

    Since fff is twice continuous differentiable, the Hessian is symmetric.

The algorithm is itself very simple.

Algorithm

Gradient Descent

  1. Begin from some x0∈Rnx_0 \in \mathbb{R }^nx0​∈Rn.
  2. xt+1←xt−η∇f(xt)x^{t + 1} \leftarrow x^t - \eta \nabla f(x^t)xt+1←xt−η∇f(xt).

where η>0\eta > 0η>0 is called step size. Below is an example that show how fff decreases as the algorithm iterates.

Convergence

To analyze the convergence behavior, we need additional assumptions on fff. We analyze a baby version of gradient descent with strong assumptions:

  1. fff is β\betaβ smooth, i.e., vT∇2f(z)v≤β∥v∥2v^T \nabla^2 f(z) v \le \beta \| v \|^2vT∇2f(z)v≤β∥v∥2 for ∀z,v∈Rn\forall z, v \in \mathbb{R }^n∀z,v∈Rn.
  2. fff is α\alphaα strongly convex, i.e., vT∇2f(z)v≥α∥v∥2v^T \nabla^2 f(z) v \ge \alpha \| v \|^2vT∇2f(z)v≥α∥v∥2 for ∀z,v∈Rn\forall z, v \in \mathbb{R }^n∀z,v∈Rn.

The interpretation is that, β\betaβ is the upper bound of the largest possible eigenvalue of the Hessian at any point and α\alphaα is the lower bound. By definition we have β≥α\beta \ge \alphaβ≥α.

Now define

Δt=f(xt)−f(x∗)\Delta_t = f(x^t) - f(x^*) Δt​=f(xt)−f(x∗)

and κ=βα\kappa = \frac{\beta}{\alpha}κ=αβ​. The following theorem characterize the convergence speed of the algorithm:

Theorem. If we set η=1β\eta = \frac{1}{\beta}η=β1​, then it holds that

Δt≤Δ0(1−14κ)t.\Delta_t \le \Delta_0 \left( 1 - \frac{1}{4 \kappa } \right)^t. Δt​≤Δ0​(1−4κ1​)t.

By the theorem, given any ϵ>0\epsilon > 0ϵ>0, after

O(κlog⁡Δ0ϵ)O \left( \kappa \log \frac{\Delta_0}{\epsilon} \right) O(κlogϵΔ0​​)

iteration, the error Δt\Delta_tΔt​ will be smaller than ϵ\epsilonϵ.

Proof:

The proof consists of two parts. β\betaβ-Smooth and α\alphaα-Convexity play a role in each part separately.

β\betaβ-Smoothness.

The property guarantees that ϵt+1≤ϵt−12β∥f(xt)∥2\epsilon_{t + 1} \le \epsilon_t - \frac{1}{2 \beta} \| f(x_t) \|^2ϵt+1​≤ϵt​−2β1​∥f(xt​)∥2.

First, by Taylor series expansion, we have for x,y∈Rnx, y \in \mathbb{R }^nx,y∈Rn:

f(y)=f(x)+∇f(x)T(y−x)+12(y−x)T∇2f(z)(y−x)≤f(x)+∇f(x)T(y−x)+12β∥y−x∥2\begin{aligned} f(y) &= f(x) + \nabla f(x)^T (y - x) + \frac{1}{2} (y - x)^T \nabla^2 f(z) (y - x) \\ &\le f(x) + \nabla f(x)^T (y - x) + \frac{1}{2} \beta \|y - x \|^2 \end{aligned} f(y)​=f(x)+∇f(x)T(y−x)+21​(y−x)T∇2f(z)(y−x)≤f(x)+∇f(x)T(y−x)+21​β∥y−x∥2​

where z=λx+(1−λ)yz = \lambda x + (1 - \lambda) yz=λx+(1−λ)y for λ∈[0,1]\lambda \in [0, 1]λ∈[0,1]. The second inequality comes from the β\betaβ-smoothness.

Replacing (y−x)(y - x)(y−x) as −η∇f(x)-\eta \nabla f(x)−η∇f(x), we get

f(y)≤f(x)−η∥∇f(x)∥2+12βη2∥∇f(x)∥2≤f(x)+(12βη2−η)∥∇f(x)∥2\begin{aligned} f(y) &\le f( x ) - \eta \| \nabla f(x) \|^2 + \frac{1}{2} \beta \eta^2 \| \nabla f( x ) \|^2 \\ &\le f( x ) + (\frac{1}{2} \beta \eta^2 -\eta) \| \nabla f(x ) \|^2 \\ \end{aligned} f(y)​≤f(x)−η∥∇f(x)∥2+21​βη2∥∇f(x)∥2≤f(x)+(21​βη2−η)∥∇f(x)∥2​

The second coefficient (12βη2−η)(\frac{1}{2} \beta \eta^2 -\eta)(21​βη2−η) is minimized when βη−1=0\beta \eta - 1 = 0βη−1=0, i.e., η=1β\eta = \frac{1}{\beta}η=β1​. It follows

f(y)≤f(x)+(12βη2−η)∥∇f(x)∥2≤f(x)−12β∥∇f(x)∥2\begin{aligned} f(y ) &\le f( x ) + (\frac{1}{2} \beta \eta^2 -\eta) \| \nabla f(x ) \|^2 \\ &\le f( x ) - \frac{1}{2 \beta} \| \nabla f(x ) \|^2 \\ \end{aligned} f(y)​≤f(x)+(21​βη2−η)∥∇f(x)∥2≤f(x)−2β1​∥∇f(x)∥2​

Finally, substituting yyy with xt+1x^{t + 1}xt+1 and xxx with xtx^txt, it becomes

f(xt+1)≤f(xt)−12β∥∇f(xt)∥2⟷Δt+1≤Δt−12β∥∇f(xt)∥2\begin{aligned} f(x^{t + 1}) &\le f(x^t) - \frac{1}{2 \beta } \| \nabla f(x^t) \|^2 \\ &\longleftrightarrow \\ \Delta_{t + 1} &\le \Delta_t - \frac{1}{2 \beta } \| \nabla f(x^t) \|^2 \end{aligned} f(xt+1)Δt+1​​≤f(xt)−2β1​∥∇f(xt)∥2⟷≤Δt​−2β1​∥∇f(xt)∥2​

α\alphaα Convexity.

We might still have a problem, if when xtx^txt approaches x∗x^*x∗, ∥∇f(xt)∥\| \nabla f(x^t) \|∥∇f(xt)∥ decreases too fast. That where α\alphaα convexity comes into play. We will show that

∥∇f(xt)∥2≥Δt⋅12α\| \nabla f(x^t) \|^2 \ge \Delta_t \cdot\frac{1}{2} \alpha ∥∇f(xt)∥2≥Δt​⋅21​α

  1. Proof 1. By α\alphaα convexity, we have for all y∈Rny \in \mathbb{R}^ny∈Rn,

    f(y)≥f(x)+∇f(x)T(y−x)+α2∥y−x∥2f(y) \ge f(x) + \nabla f(x)^T (y - x) + \frac{\alpha}{2} \| y - x \|^2 f(y)≥f(x)+∇f(x)T(y−x)+2α​∥y−x∥2

    Taking minimum of both sides,

    min⁡y∈Rnf(y)≥min⁡y∈Rn(f(x)+∇f(x)T(y−x)+α2∥y−x∥2)\min_{y \in \R^n} f(y) \ge \min_{y \in \R^n} \left( f(x) + \nabla f(x)^T (y - x) + \frac{\alpha}{2} \| y - x \|^2\right) y∈Rnmin​f(y)≥y∈Rnmin​(f(x)+∇f(x)T(y−x)+2α​∥y−x∥2)

    The left hand side equals to f(x∗)f(x^*)f(x∗) while the right hand side minimized to f(x)−12α∥∇f(x)∥2f(x) - \frac{1}{2 \alpha} \| \nabla f(x) \|^2f(x)−2α1​∥∇f(x)∥2 when y−x=−1α∇f(x)y - x = -\frac{1}{\alpha} \nabla f(x)y−x=−α1​∇f(x).

    Substituting xxx with xtx^txt, we get

    ∥∇f(xt)∥2≥2α(f(xt)−f(x∗))=2αΔt\| \nabla f(x^t) \|^2 \ge 2 \alpha \left( f(x^t) - f(x^*) \right) = 2\alpha \Delta_t ∥∇f(xt)∥2≥2α(f(xt)−f(x∗))=2αΔt​

  2. Proof 2. By convexity, we have

    ∇f(x)T(x−x∗)≥f(x)−f(x∗)\begin{aligned} \nabla f(x)^T (x - x^*) &\ge f(x) - f(x^*) \end{aligned} ∇f(x)T(x−x∗)​≥f(x)−f(x∗)​

    Then by Taylor expansion and α\alphaα convexity,

    f(x)−f(x∗)≥∇f(x∗)T(x−x∗)+12α∥x−x∗∥2=12α∥x−x∗∥2\begin{aligned} f(x) - f(x^*) &\ge \nabla f(x^*)^T (x - x^*) + \frac{1}{2} \alpha \| x - x^* \|^2\\ &=\frac{1}{2} \alpha \| x - x^* \|^2 \end{aligned} f(x)−f(x∗)​≥∇f(x∗)T(x−x∗)+21​α∥x−x∗∥2=21​α∥x−x∗∥2​

    Concatenating the inequalities, we get

    ∥∇f(x)∥∥x−x∗∥≥∇f(x)T(x−x∗)≥f(x)−f(x∗)≥12α∥x−x∗∥2\| \nabla f(x) \| \| x - x^* \| \ge \nabla f(x)^T (x - x^*) \ge f(x) - f(x^*) \ge \frac{1}{2} \alpha \| x - x^* \|^2 ∥∇f(x)∥∥x−x∗∥≥∇f(x)T(x−x∗)≥f(x)−f(x∗)≥21​α∥x−x∗∥2

    where the first inequality is Cauchy-Schwartz inequality. Multiplying the inequalities we get

    ∥∇f(x)∥2∥x−x∗∥2≥(f(x)−f(x∗))⋅12α∥x−x∗∥2\| \nabla f(x) \|^2 \| x - x^* \|^2 \ge (f(x) - f(x^*)) \cdot\frac{1}{2} \alpha \| x - x^* \|^2 ∥∇f(x)∥2∥x−x∗∥2≥(f(x)−f(x∗))⋅21​α∥x−x∗∥2

    which implies that

    ∥∇f(x)∥2≥(f(x)−f(x∗))⋅12α\| \nabla f(x) \|^2 \ge (f(x) - f(x^*)) \cdot\frac{1}{2} \alpha ∥∇f(x)∥2≥(f(x)−f(x∗))⋅21​α

    Replacing xxx with xtx^txt and f(x)−f(x∗)f(x) - f(x^*)f(x)−f(x∗) with Δt\Delta_tΔt​, we have

    ∥∇f(xt)∥2≥Δt⋅12α\| \nabla f(x^t) \|^2 \ge \Delta_t \cdot\frac{1}{2} \alpha ∥∇f(xt)∥2≥Δt​⋅21​α

In combination, we conclude that

Δt+1≤Δt(1−α4β)\begin{aligned} \Delta_{t + 1} &\le \Delta_t \left( 1- \frac{\alpha }{4 \beta} \right) \end{aligned} Δt+1​​≤Δt​(1−4βα​)​

which finishes the proof.
