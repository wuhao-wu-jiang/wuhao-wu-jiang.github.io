Hoeffding Inequality

We first prove a bound of

Pr⁡[∪k=1nMk]≤nδ\Pr[ \cup_{k = 1}^n M_k ] \le n \delta Pr[∪k=1n​Mk​]≤nδ

by Hoeffding inequality.

Hoeffding Inequality. For a fixed integer sss, let Z1,Z2,...,ZsZ_1, Z_2, ..., Z_sZ1​,Z2​,...,Zs​ be i.i.d. copies of a random variable ZZZ on [0,1][0, 1][0,1]. For any r>0r > 0r>0

Pr⁡[μ−μs≥r]≤exp⁡(−2sr2). \Pr[ \mu - \mu_s \ge r] \le \exp(- 2 s r^2 ). \\ \ Pr[μ−μs​≥r]≤exp(−2sr2). 

Applying Hoeffding inequality with s=ks = ks=k, r=log⁡1δ2kr = \sqrt{ \frac{ \log \frac{1}{\delta} }{2 k} }r=2klogδ1​​​, we get

Pr⁡[Mk]≤δ.\Pr[ M_k ] \le \delta. Pr[Mk​]≤δ.

By sub-additivity of probability,

Pr⁡[∪k=1nMk]≤∑k=1nPr⁡[Mk]=nδ.\Pr[ \cup_{k = 1}^n M_k ] \le \sum_{k = 1}^n \Pr[M_k] = n \delta. Pr[∪k=1n​Mk​]≤k=1∑n​Pr[Mk​]=nδ.

To make the bound meaningful, we require that δ≤1n\delta \le \frac{1}{n}δ≤n1​.

Beyond The Naive Union Bound

In this section, we avoid using union bound over nnn events and improve the Pr⁡[∪k=1nMk]\Pr[ \cup_{k = 1}^n M_k ]Pr[∪k=1n​Mk​] to

Pr⁡[∪k=1nMk]≤(log⁡1αn+1)⋅δα,\Pr[ \cup_{k = 1}^n M_k ] \le ( \log_\frac{1}{\alpha} n + 1 ) \cdot \delta^\alpha, Pr[∪k=1n​Mk​]≤(logα1​​n+1)⋅δα,

where α\alphaα is any real number that satisfies 0<α<10 < \alpha < 10<α<1.

As a sanity check, suppose δ=1n\delta = \frac{1}{n}δ=n1​ and α=12\alpha = \frac{1}{2}α=21​. The two bounds we have are

  1. Pr⁡[∪k=1nMk]≤nδ=1,\Pr[ \cup_{k = 1}^n M_k ] \le n \delta = 1, Pr[∪k=1n​Mk​]≤nδ=1,

    which is trivial.

  2. (log⁡1αn+1)⋅δα=(log⁡2n+1)⋅1n\begin{aligned} ( \log_\frac{1}{\alpha} n + 1 ) \cdot \delta^\alpha &= (\log_2 n + 1) \cdot {1 \over \sqrt n} \end{aligned} (logα1​​n+1)⋅δα​=(log2​n+1)⋅n​1​​

The proof relies on a strengthened version of Hoeffding inequality.

Fact [1]. For a fixed integer sss, let Z1,Z2,...,ZsZ_1, Z_2, ..., Z_sZ1​,Z2​,...,Zs​ be i.i.d. copies of a random variable ZZZ on [0,1][0, 1][0,1]. For any r>0r > 0r>0

Pr⁡[∃t≤s:t⋅μ−t⋅μt≥r]≤exp⁡(−2r2s). \Pr[ \exists t \le s : \quad t \cdot \mu - t \cdot \mu_t \ge r] \le \exp(- \frac{2 r^2}{s} ). \\ \ Pr[∃t≤s:t⋅μ−t⋅μt​≥r]≤exp(−s2r2​). 

Proof. We divide the MkM_kMk​’s into ⌊log⁡1αn⌋+1\lfloor \log_\frac{1}{\alpha} n \rfloor + 1⌊logα1​​n⌋+1 groups. For integer 0≤j≤⌊log⁡1αn⌋0 \le j \le \lfloor \log_\frac{1}{\alpha} n \rfloor0≤j≤⌊logα1​​n⌋, define

Bj≐∪n⋅αj+1<k≤n⋅αjMk={∃n⋅αj+1<k≤n⋅αj,μk+log⁡1δ2k≤μ}={∃n⋅αj+1<k≤n⋅αj,k⋅μk+klog⁡1δ2≤k⋅μ}⊂{∃n⋅αj+1<k≤n⋅αj,k⋅μk+n⋅αj+1log⁡1δ2≤k⋅μ}\begin{aligned} B_j &\doteq \cup_{n \cdot \alpha^{j + 1} < k \le n \cdot \alpha^j } M_k \\ &= \left\{ \exists \quad n \cdot \alpha^{j + 1} < k \le n \cdot \alpha^j, \mu_k + \sqrt{ \frac{ \log \frac{1}{\delta} }{2 k} } \le \mu \right\} \\ &= \left\{ \exists \quad n \cdot \alpha^{j + 1} < k \le n \cdot \alpha^j, k \cdot \mu_k + \sqrt{ \frac{ k \log \frac{1}{\delta} }{2} } \le k \cdot \mu \right\} \\ &\subset \left\{ \exists \quad n \cdot \alpha^{j + 1} < k \le n \cdot \alpha^j, k \cdot \mu_k + \sqrt{ \frac{ n \cdot \alpha^{j + 1} \log \frac{1}{\delta} }{2} } \le k \cdot \mu \right\} \\ \end{aligned} Bj​​≐∪n⋅αj+1<k≤n⋅αj​Mk​=⎩⎪⎨⎪⎧​∃n⋅αj+1<k≤n⋅αj,μk​+2klogδ1​​​≤μ⎭⎪⎬⎪⎫​=⎩⎪⎨⎪⎧​∃n⋅αj+1<k≤n⋅αj,k⋅μk​+2klogδ1​​​≤k⋅μ⎭⎪⎬⎪⎫​⊂⎩⎪⎨⎪⎧​∃n⋅αj+1<k≤n⋅αj,k⋅μk​+2n⋅αj+1logδ1​​​≤k⋅μ⎭⎪⎬⎪⎫​​

Applying Fact [1] with s=⌊n⋅αj⌋<n⋅αjs = \lfloor n \cdot \alpha^j \rfloor < n \cdot \alpha^js=⌊n⋅αj⌋<n⋅αj, r=n⋅αj+1log⁡1δ2r = \sqrt{ \frac{ n \cdot \alpha^{j + 1} \log \frac{1}{\delta} }{2 } }r=2n⋅αj+1logδ1​​​, and by monotonicity of probability, we get

Pr⁡[Bj]≤exp⁡(−n⋅αj+1log⁡1δ22n⋅αj)=δα.\Pr[B_j] \le \exp( - \frac{ n \cdot \alpha^{j + 1} \log \frac{1}{\delta} }{2} \frac{ 2 }{ n \cdot \alpha^j } ) = \delta^{ \alpha }. Pr[Bj​]≤exp(−2n⋅αj+1logδ1​​n⋅αj2​)=δα.

Summing over all possible values of jjj, we get

Pr⁡[∪j=0⌊log⁡1αn⌋Bj]≤∑j=0⌊log⁡1αn⌋Pr⁡[Bj]≤(log⁡1αn+1)⋅δα.\Pr[ \cup_{j = 0}^{ \lfloor \log_\frac{1}{\alpha} n \rfloor } B_j] \le \sum_{j = 0}^{ \lfloor \log_\frac{1}{\alpha} n \rfloor } \Pr[B_j] \le( \log_\frac{1}{\alpha} n + 1 ) \cdot \delta^\alpha.
