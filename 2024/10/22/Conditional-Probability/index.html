Let $(\Omega, \mathcal{F}, \mathbb{P})$ be a probability space.

In general, the conditional probability is defined as a random variable measurable over a sub $\sigma$-field over $\mathcal{F}$.
We present first the definition, and then use examples to illustrate it.

Definition (Conditional Probability). Let $\mathcal{G}$ be a $\sigma$-field in $\mathcal{F}$, and $A \in \mathcal{F}$.
The conditional probability $\mathbb{P}[A \mid \mathcal{G}]$ is a random variable with two properties:

  1. $\mathbb{P}[ A \mid \mathcal{G} ]$ is measurable $\mathcal{G}$ and integrable.
  2. $\mathbb{P}[A \mid \mathcal{G}]$ satisfies the functional equation

The following provides the definition of conditional probability with respect to a random variable.

Definition. The conditional probability of $A \in \mathcal{F}$ given a random variable $X$ is defined as $\mathbb{P}[ A \mid \sigma(X) ]$ and is denoted $\mathbb{P}[A \mid X]$, where $\sigma(X)$ is the $\sigma$-field generated by $X$.

Remark. The $\sigma$-field $\sigma(X)$ consists of the sets $[\omega: X(\omega) \in H]$ for $H \in \mathcal{R}^1$.

Remark. From the experiment corresponding to the $\sigma$-field $\sigma(X)$, one learns which of the set $[\omega’ : X(\omega’) = x]$ contains $\omega$ and hence learns the value of $X(\omega)$.

In the examples above, when interpreting the conditional probability as a function, we don’t need to know the exact value of $\omega$ to determine the function output; knowing which set in the $\sigma$-field contains $\omega$ is sufficient.
Requiring that $X(\omega) = \mathbb{P}[A \mid \mathcal{G}]_{\omega}$ to be measurable $\mathcal{G}$ essentially means that, if we can determine whether $\omega$ belongs to each set $H \in \mathcal{G}$, we can determine the function’s output.
Specifically, knowing which set $X^{-1}(a), \forall a \in \mathbb{R}$ contains $\omega$ allows us to identify the value of $X$.
