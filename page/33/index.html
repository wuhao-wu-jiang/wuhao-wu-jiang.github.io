Characterizations of Beta-Smoothness

Posted on 2020-03-05 Edited on 2020-03-06

Definition

A differential function \(f: \mathbb{R}^n \rightarrow \mathbb{R}\) is \(\beta\)-smooth if \[ \|\nabla f(y) - \nabla f(x) \| \le \beta \|y - x\| \] where \(\|\cdot \|\) is \(\ell_2\) norm.

Theorem 1.

A twice continuously differential function \(f : \mathbb{R}^n \rightarrow \mathbb{R}\) is \(\beta\)-smooth if and only if \[ v^T \nabla^2 f(z) v \le \beta \|v\|^2 \] for any \(z, v \in \mathbb{R}^n\).

Proof.

  1. (ONLY IF). \(v^T \nabla^2 f(z) v \le \beta \|v\|^2\) is equivalent to that the maximum eigenvalue of \(\nabla^2 f(z)\) is at most \(\beta\). Let \(v\) be the eigenvector of \(\nabla^2 f(z)\). By mean value theorem \[ \frac{\partial f}{\partial x_i} (z + v) - \frac{\partial f}{\partial x_i} (z) = \sum_{j = 1}^n \left( \frac{\partial f}{\partial x_i \partial x_j} (c) \right) v_j \]

    For some \(c\) between \(z\) and \(z + v\). As \(f\) is twice continuously differentiable, for any given \(\epsilon > 0\), \(\exists v\) that is small enough, such that \[ \left| \frac{\partial f}{\partial x_i} (z + v) - \frac{\partial f}{\partial x_i} (z) \right| \in \left| \sum_{j = 1}^n \left( \frac{\partial f}{\partial x_i \partial x_j} (z) \right) v_j \right| \pm \frac{\epsilon}{n} \]

    Hence, \[ | \nabla f(z + v) - \nabla f(z) | \in |\nabla^2 f(z) v| + \epsilon \]

    By equivalence of \(\ell_1\)-norm and \(\ell_2\) norm, we get for \(v\) small enough, \[ \| \nabla f(z + v) - \nabla f(z) \| \in \| \nabla^2 f(z) v \| + \epsilon \]

    As \(\epsilon\) can be arbitrary small, it concludes that \[ \| \nabla^2 f(z) v \| \le \beta \|v \| \]

    which is equivalent to that the eigenvalues of \(\nabla^2 f(z)\) is at most \(\beta\).

  2. (IF). We prove a stronger result that \[ \| \nabla f(y) - \nabla f(x) \| \cdot \| \nabla f(y) - \nabla f(x) \| \le \beta \left( \nabla f(y) - \nabla f(x) \right)^T (y - x) \] By Cauchy-Schwartz inequality, we have \[ \left( \nabla f(y) - \nabla f(x) \right)^T (y - x) \le \| \nabla f(y) - \nabla f(x) \| \cdot \| y - x \| \] which will finish our proof.

    By Taylor series expansion, we have \[ \begin{aligned} f(y) & = f(x) + \nabla f(x)^T (y - x) + \frac{1}{2} (y - x)^T \nabla^2 f(z) (y - x) \\ &\le f(x) + \nabla f(x)^T (y - x) + \frac{1}{2} \beta \| y - x \|^2 \end{aligned} \]

    for some \(z\) between \(x\) and \(y\). The inequality holds since \(v^T \nabla^2 f(z) v \le \beta \|v\|^2\) for any \(v \in \mathbb{R}^n\).

    By symmetry, we get \[ \begin{aligned} f(x) &\le f(y) + \nabla f(y)^T (x - y) + \frac{1}{2} \beta \| y - x \|^2 \end{aligned} \]

    Summing up the two inequalities, we obtain \[ \left( \nabla f(y) - \nabla f(x) \right)^T (y - x)^T \le \beta \| y - x \|^2 \]

\(\square\)

Theorem 2.

If \(f\) is \(\beta\)-smooth, then for \(x, y \in \mathbb{R}^n\), we have \[ f(y) \le f(x) + \nabla f(x)^T (y - x) + \frac{1}{2} \beta \| y - x \|^2 \]

Proof. By fundamental theorem of calculus, \[ \begin{aligned} f(y) - f(x) - \nabla f(x)^T (y - x) &= \int_{0}^1 \nabla f(x + t(y - x))^T(y - x) \ dt - \nabla f(x)^T (y - x) \\ &= \int_{0}^1 \left( \nabla f(x + t(y - x)) - \nabla f(x) \right)^T(y - x) \ dt \\ &\le \int_{0}^1 \| \nabla f(x + t(y - x)) - \nabla f(x) \| \cdot \| y - x \| \ dt \\ &\le \int_{0}^1 \beta t \| y - x \| \cdot \| y - x \| \ dt \\ &\le \frac{1}{2} \beta \| y - x \|^2 \end{aligned} \]

\(\square\)

Counting Triangle

Posted on 2020-03-01 Edited on 2022-02-04

Problem

  1. Let \(G = \left< V, E \right>\) be an undirected graph with \(|V| = n\) vertices and \(|E| = m\) edges.

  2. The number of triangle in \(G\) is denoted as \(\Delta\).

  3. For each \(v \in V\), let \(N(v) \doteq \{ u : (v, u) \in E \}\) be the set of neighbors of \(v\).

  4. The degree of \(v\) is \(d_v \doteq | N(v) |\).

  5. Define \(\lambda(v)\) be the number of triangles whose vertices contains \(v\).

  6. Similarly, for each \(e = (u, v) \in E\), define \(\lambda(e)\) the number of triangles whose edges contains \(e\).

Exact Algorithm

Theorem. The number of triangles in \(G\) can be computed in \(O \big ( m^{1.5} \big)\) time.

Proof.

WLOG, we assume that the vertices in \(V\) are labelled from \(1\) to \(n\) in increasing order according to their degrees, such that \[ d_1 \le d_2 \le ... \le d_n. \]

Each triangle in \(G\) can be represented by a unique triple \((u, v, w) \in [n]^3\) such that \(d_u \le d_v \le d_w\). To count the number of such triples, we can execute the following algorithm.

For each \(u \in [n]\) do
\(\quad\) For \(v \in N(u)\) of \(u\) s.t. \(d_v \ge d_u\) do
\(\quad\quad\) For \(w \in N(u)\) of \(u\) s.t. \(d_w \ge d_v\) do
\(\qquad\quad\) Check whether \((u, v, w)\) is a triangle.

It is left to prove that the algorithm has complexity \(O(m^{1.5})\).

Lemma. For each \((u, v) \in E\), such that \(d_u \le d_v\), the number of \(w \in N(u)\) such that \(d_w \ge d_v\) is bounded by \(O(\sqrt{m})\).

  • The claim is trivial if \(d_u \le \sqrt{m}\).

  • Otherwise, suppose that \(d_u \ge \sqrt{m}\). But the number of \(w \in V\) such that \(d_w \ge d_u \ge \sqrt{m}\) is bounded by \(O( m / \sqrt{ m }) = O(\sqrt{ m })\).

\(\square\)

Sampling Algorithms

Subgraph Sampling

  1. Sample each edge independently with some probability \(p\).
  2. Count the number of triangles, \(\tilde \Delta\), in the sampled subgraph.
  3. \(X \leftarrow \big( 1 / p^3 \big) \cdot \tilde \Delta\).

Properties

Theorem (Unbiased Estimator). \(X\) is an unbiased estimator of \(\Delta\).

Proof. For each triangle in \(G\), define the indicator random variable \(Z\) for its inclusion in teh sampled subgraph. Then \(\mathbb{E}[Z] = p^3\). Let \(S\) be the collection of such random variable. By linearity of expectation. \[ \mathbb{E} \left[ \sum_{Z \in S } Z \right] = \sum_{Z \in S } \mathbb{E} \left[ Z \right] = p^3 \cdot \Delta. \]

It follows that \((1 / p^3) \sum_{Z \in S} Z\) is an unbiased estimator of \(\Delta\).

\(\square\)

Let \(\Delta^N\) be the number of triangles that share an edge.

Theorem (Variance). \(X\) has variance \[ (1 / p^6) \cdot \left( \Delta \cdot \big( p^3 - p^6 \big) + 2 \cdot \Delta^N \cdot \big( p^5 - p^6 \big) \right). \]

Proof. \[ \begin{aligned} \mathbb{Var} \left[ \frac{1}{p^3} \sum_{Z \in S } Z \right] &= \frac{1}{p^6} \cdot \left( \sum_{Z \in S } \mathbb{Var} \left[ Z \right] + \sum_{Z, Z' \in S } \mathbb{Cov} \left[ Z, Z' \right] \right) \\ &= \frac{1}{p^6} \cdot \left( \sum_{Z \in S } \mathbb{Var} \left[ Z \right] + \sum_{Z, Z' \in S \wedge Z \text{ shares an edge with } Z' } \mathbb{Cov} \left[ Z, Z' \right] \right) \\ &= \frac{1}{p^6} \cdot \left( \sum_{Z \in S } \big( p^3 - p^6 \big) + \sum_{Z, Z' \in S \wedge Z \text{ shares an edge with } Z' } \big( p^5 - p^6 \big) \right) \\ &= \frac{1}{p^6} \cdot \left( \Delta \cdot \big( p^3 - p^6 \big) + 2 \cdot \Delta^N \cdot \big( p^5 - p^6 \big) \right). \end{aligned} \] \(\square\)

Vertex Sampling

  1. Sample a vertex \(v\) uniformly at random from \(V\).
  2. Compute \(\lambda(v)\).
  3. \(X \leftarrow \big( n / 3 \big) \cdot \lambda(v)\).

Properties

Theorem (Unbiased Estimator). \(X\) is an unbiased estimator of \(\Delta\).

Proof. \[ \mathbb{E}[X] = \frac{n}{3} \cdot \sum_{v \in V} \frac{1}{n} \cdot \lambda(v) = \frac{1}{3} \cdot \sum_{v \in V} \lambda(v) = \Delta. \] \(\square\)

Theorem (Variance). \(X\) has variance \(\left( (n / 9) \cdot \sum_{v \in V} \lambda(v)^2 \right) - \Delta^2\).

Proof. \[ \begin{aligned} \mathbb{Var} \left[ X \right] = \mathbb{E}[X^2] - \mathbb{E}[X]^2 = \left( \frac{n}{9} \cdot \sum_{v \in V} \lambda(v)^2 \right) - \Delta^2. \end{aligned} \]
\(\square\)

Edge Sampling

  1. Sample an edge \(e\) uniformly at random from \(E\).
  2. Compute \(\lambda(e)\).
  3. \(X \leftarrow \big( m / 3 \big) \cdot \lambda(e)\).

Properties

Theorem (Unbiased Estimator). \(X\) is an unbiased estimator of \(\Delta\).

Proof. \[ \mathbb{E}[X] = \frac{m}{3} \cdot \sum_{e \in E} \frac{1}{m} \cdot \lambda(e) = \frac{1}{3} \cdot \sum_{e \in E} \lambda(e) = \Delta. \] \(\square\)

Theorem (Variance). \(X\) has variance \(\left( (m / 9) \cdot \sum_{e \in E} \lambda(v)^2 \right) - \Delta^2\).

Proof. \[ \begin{aligned} \mathbb{Var} \left[ X \right] = \mathbb{E}[X^2] - \mathbb{E}[X]^2 = \left( \frac{m}{9} \cdot \sum_{e \in E} \lambda(e)^2 \right) - \Delta^2. \end{aligned} \]
\(\square\)

Triangle Sampling

  1. Sample an edge \(e = (u, v)\) uniformly at random from \(E\).
  2. Sample a vertex uniformly at random from \(\begin{cases} N(u), & \text{w.p.} & d_u / (d_u + d_v) , \\ N(v), & \text{w.p.} & d_v / (d_u + d_v) . \end{cases}\)
  3. \(I \leftarrow \begin{cases} 1, & \text{if} (u, v, w) \text{ constitutes a triangle}, \\ 0, & \text{otherwise}. \end{cases}\)
  4. \(X \leftarrow \big( m / 6 \big) \cdot \big( d_u + d_v \big) \cdot I\).

Properties

Theorem (Unbiased Estimator). \(X\) is an unbiased estimator of \(\Delta\).

Proof. Conditioned on the sampled edge \(e = (u, v)\), \[ \mathbb{E}[ I \mid e = (u, v) ] = \frac{ 2 \cdot |N(u) \cap N(v) | }{d_u + d_v} = 2 \cdot \frac{ \lambda(e) }{d_u + d_v}. \] Hence, \((1 / 2) \cdot (d_u + d_v) \cdot I\) is an unbiased estimator of \(\lambda(e)\). It follows that \[ \begin{aligned} \mathbb{E}[(m / 6) \cdot (d_u + d_v) \cdot I] &= \frac{m}{3} \cdot \mathbb{E}[(1 / 2) \cdot (d_u + d_v) \cdot I] \\ &= \frac{m}{3} \cdot \sum_{e \in E} \frac{1}{m} \cdot \mathbb{E}[(1 / 2) \cdot (d_u + d_v) \cdot I \mid e = (u, v) ] \\ &= \frac{1}{3} \sum_{e \in E} \lambda(e) = \Delta. \end{aligned} \] \(\square\)

Theorem (Variance). \(X\) has variance \((m / 18) \cdot \sum_{e = (u, v) \in E} (d_u + d_v) \cdot \lambda(e) - \Delta^2\).

Proof. \[ \begin{aligned} \mathbb{Var} \left[ X \right] &= \mathbb{E}[X^2] - \mathbb{E}[X]^2 \\ &= \frac{m^2}{36} \cdot \sum_{e = (u, v) \in E} \frac{1}{m} \cdot (d_u + d_v)^2 \cdot \frac{ 2 \cdot \lambda(e) }{d_u + d_v} - \Delta^2 \\ &= \frac{m}{18} \cdot \sum_{e = (u, v) \in E} (d_u + d_v) \cdot \lambda(e) - \Delta^2. \end{aligned} \]
\(\square\)

Wedge Sampling

A wedge is a triple \((u, v, w) \in V^3\) such that \((u, v), (v, w) \in E\). If a wedge is closed, i.e., \((v, w) \in E\), then it constitutes a triangle. Note that each triangle contains \(3\) wedges. Denote \(W\) the number of wedges in \(G\).

  1. Sample a wedge \((u, v, w)\) uniformly at random.
  2. \(I \leftarrow \begin{cases} 1, & \text{if} (u, v, w) \text{ constitutes a triangle}, \\ 0, & \text{otherwise}. \end{cases}\)
  3. \(X \leftarrow \big( W / 3 \big) \cdot I\).

Properties

Theorem (Unbiased Estimator). \(X\) is an unbiased estimator of \(\Delta\).

Proof. \[ \mathbb{E}[ I ] = \frac{ 3 \cdot \Delta }{ W }. \] Therefore, \(X = (W / 3 ) \cdot I\) is an unbiased estimator of \(\Delta\).

\(\square\)

Theorem (Variance). \(X\) has variance \(W \cdot \Delta / 3 - \Delta^2\).

Proof. \[ \begin{aligned} \mathbb{Var}[ I ] = \frac{ 3 \cdot \Delta }{ W } - \left( \frac{ 3 \cdot \Delta }{ W } \right)^2 \end{aligned} \] Therefore, \[ \begin{aligned} \mathbb{Var}[ X ] = W \cdot \Delta / 3 - \Delta^2 \end{aligned} \]

\(\square\)

Primal-Dual Algorithm

Posted on 2020-02-27 Edited on 2020-02-28

Primal: \[ \begin{aligned} &\min &c^T x \\ &s.t. &Ax = b\\ & &x \ge 0 \end{aligned} \]

We assume that \(b \ge 0\). Otherwise we can flip the \(i\) constraints \(A^T_i x = b_i\) by \(- A^T_i x = -b_i\), where \(A^T_i\) is the \(i\)-th column of matrix \(A^T\), i.e, the \(i\)-th row of matrix \(A\).

Dual: \[ \begin{aligned} &\max &y^T b \\ &s.t. &y^TA \le c^T \\ \end{aligned} \]

Suppose that we have a feasible solution \(y\) for the dual (if \(c \ge 0\), then \(y = 0\) is a feasible solution). Weak duality states that \[ c^T x \ge y^T A x = y^T b \]

Denote \(S = \{ i : i \in [n], y^T A_i < c_i \}\) (the set of constraints that have slacks for increments) and \(\bar S = [n] \setminus S\) (the set of indexes of tight dual constraints), where \(A_i\) is the \(i\)-th columns of \(A\).

By weak duality, if we can find an \(x\) such that

  1. \(x_{S } = 0\), where \(x_{S} \doteq (x_i : i \in {S})\).
  2. \(x_{\bar S} \ge 0\) and \(A_{\bar S} x_{\bar S} = b\), where \(A_{\bar S}\) is the sub-matrix of \(A\) containing only columns in \({\bar S}\).

then both \(y\) and \(x\) are optimal. However, as \(y\) is not necessarily optimal, we have the following relaxed optimization problem, which is called restricted primal

Relaxation: \[ \begin{aligned} &\min &1^T \bar x \\ &s.t. &A_{\bar S} x_{\bar S} + \bar x = b \\ & &x_{\bar S} \ge 0 \\ & &\bar x \ge 0 \end{aligned} \]

Note that \(x_{\bar S} = 0\) and \(\bar x = b\) is a feasible solution as we assume that \(b \ge 0\). Therefore, restricted primal is feasible. Further, it is bounded below by \(1^{\bar S} 0 = 0\). Indeed, if it achieves optimal solution value \(0\), it holds that \(\bar x = 0\) so that \(A_{\bar S} x_{\bar S} = b\).

Associated with the restricted primal is the restricted dual \[ \begin{aligned} &\max & \bar y^T b \\ &s.t. & \bar y^TA_{\bar S} \le 0 \\ & & \bar y^T \le 1^T \end{aligned} \]

By strong duality, it optimal solution \(\bar y^T b = 1^T \bar x\). If \(1^T \bar x= \bar y^T b > 0\), then we can improve the dual solution \(y\) by \[ y \leftarrow y + \epsilon \bar y \]

for some \(\epsilon\). To determine its value, first observe that \[ (y + \epsilon \bar y)^{T} A_{\bar S} = y^T A_{\bar S} + \epsilon \bar y^T A_{\bar S} \le y^T A_{\bar S} = c_{\bar S}^T \]

Hence it is left to guarantee that \[ (y + \epsilon \bar y)^T A_S = y^T A_S + \epsilon \bar y^T A_S \le c_S \]

By definition of \({S}\), we have \(y^T A_{S} < c_{S}^T\). If that \(\bar y^T A_i \le 0\) for \(\forall i \in {S}\), then \(\epsilon\) can be arbitrary large and the dual is unbounded. Otherwise,

\[ \epsilon = \min_{i \in S, \bar y^T A_i > 0} \frac{c_i - y^T A_i } { \bar y^T A_i } \]
