Maximum of Sub-Gaussian Random Variables

Posted on 2021-06-10 Edited on 2021-06-09

Definition. A random variable X∈RX \in \mathbb{R}X∈R is said to be sub-Gaussian with variance proxy σ2\sigma^2σ2 if E[X]=0\mathbb{E} [X] = 0E[X]=0 and its moment generating function satisfies

E[exp⁡(sX)]≤exp⁡(σ2s22),∀s∈R. \mathbb{E}[ \exp (sX) ] \le \exp \left( \frac{\sigma^2 s^2 }{2} \right), \qquad \forall s \in \mathbb{R}. E[exp(sX)]≤exp(2σ2s2​),∀s∈R.

In this case we write X∼subG(σ2)X \sim \text{subG}(\sigma^2)X∼subG(σ2).

Theorem. Let X1,…,XnX_1, \ldots, X_nX1​,…,Xn​ be nnn random variables such that Xi∼subG(σ2)X_i \sim \text{subG}(\sigma^2)Xi​∼subG(σ2). Then

E[max⁡i∈[n]Xi]≤σ2log⁡n,E[max⁡i∈[n]∣Xi∣]≤σ2log⁡(2n). \mathbb{E} \Big[ \max_{i \in [n] } X_i \Big] \le \sigma \sqrt{ 2 \log n }, \qquad \mathbb{E} \Big[ \max_{i \in [n] } | X_i | \Big] \le \sigma \sqrt{ 2 \log (2 n ) }. E[i∈[n]max​Xi​]≤σ2logn​,E[i∈[n]max​∣Xi​∣]≤σ2log(2n)​.

Proof. ∀s>0\forall s > 0∀s>0,

E[max⁡i∈[n]Xi]=1sE[max⁡i∈[n]sXi]=1sE[ln⁡(exp⁡(max⁡i∈[n]sXi))]≤1sln⁡(E[exp⁡(max⁡i∈[n]sXi))]Jensen=1sln⁡(E[max⁡i∈[n]exp⁡(sXi))]≤1sln⁡(∑i∈[n]E[exp⁡(sXi))]≤1sln⁡(∑i∈[n]exp⁡(σ2s22))=1sln⁡n+σ2s2≤2σ22ln⁡nSetting s=2ln⁡nσ2=σ2ln⁡n. \begin{aligned} \mathbb{E} \Big[ \max_{i \in [n] } X_i \Big] &= \frac{1}{s} \mathbb{E} \Big[ \max_{i \in [n] } s X_i \Big] \\ &= \frac{1}{s} \mathbb{E} \Big[ \ln \Big( \exp \Big( \max_{i \in [n] } s X_i \Big) \Big) \Big] \\ &\le \frac{1}{s} \ln \Big( \mathbb{E} \Big[ \exp \Big( \max_{i \in [n] } s X_i \Big) \Big) \Big] \qquad \text{Jensen} \\ &= \frac{1}{s} \ln \Big( \mathbb{E} \Big[ \max_{i \in [n] } \exp \Big( s X_i \Big) \Big) \Big] \\ &\le \frac{1}{s} \ln \Big( \sum_{i \in [n] } \mathbb{E} \Big[ \exp \Big( s X_i \Big) \Big) \Big] \\ &\le \frac{1}{s} \ln \Big( \sum_{i \in [n] } \exp \Big( \frac{\sigma^2 s^2}{2} \Big) \Big) \\ &= \frac{1}{s} \ln n + \frac{\sigma^2 s}{2} \\ &\le 2 \sqrt{ \frac{\sigma^2}{2} \ln n} \qquad \text{Setting } s = \sqrt{ \frac{2 \ln n}{\sigma^2} } \\ &= \sigma \sqrt{ 2 \ln n}. \end{aligned} E[i∈[n]max​Xi​]​=s1​E[i∈[n]max​sXi​]=s1​E[ln(exp(i∈[n]max​sXi​))]≤s1​ln(E[exp(i∈[n]max​sXi​))]Jensen=s1​ln(E[i∈[n]max​exp(sXi​))]≤s1​ln(i∈[n]∑​E[exp(sXi​))]≤s1​ln(i∈[n]∑​exp(2σ2s2​))=s1​lnn+2σ2s​≤22σ2​lnn​Setting s=σ22lnn​​=σ2lnn​.​

To prove the second inequality, we can set Xi+n=−XiX_{i + n} = - X_iXi+n​=−Xi​, for all i∈[n]i \in [n]i∈[n], so that

E[max⁡i∈[n]∣Xi∣]=E[max⁡i∈[2n]Xi]≤σ2log⁡(2n). \mathbb{E} \Big[ \max_{i \in [n] } | X_i | \Big] = \mathbb{E} \Big[ \max_{i \in [2n] } X_i \Big] \le \sigma \sqrt{ 2 \log (2n) }. E[i∈[n]max​∣Xi​∣]=E[i∈[2n]max​Xi​]≤σ2log(2n)​.
