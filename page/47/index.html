Convergence Tests

Posted on 2019-02-27 Edited on 2022-04-10

Ration Test

Given a series \(s_n = \sum_{k \in [n]} a_n\) with \(a_n \neq 0\), the Ratio Test states that if \((a_n)\) satisfies \[ \lim \left| \frac{a_{n + 1} }{ a_n } \right| = r \le 1 \] then \((s_n)\) converges absolutely.

Proof. The condition implies that \[ \exists N > 0, s.t., \exists r \le r' < 1, \forall n \ge N, a_{n + 1} \le r' a_n \]

It follows that for \(n > m \ge N\), \[ \begin{aligned} |s_n - s_m| &= | a_{m + 1} + a_{m + 1} + ... + a_n| \\ &\le |a_N| \sum_{k = 0}^{n - m - 1} (r')^{m - N + k} \\ &\le |a_N| \sum_{k = 0}^{ \infty } (r')^{m - N + k} \\ &\le \frac{|a_N| }{1 - (r') } (r')^{m - N} \end{aligned} \]

If \(m\) is sufficient large, then \(|s_n - s_m|\) is arbitrary small. It concludes that \((s_n)\) is Cauchy sequence.

\(\square\)

Abel's Test

If the series \(s_n = \sum_{k \in [n]} a_n\) converges and if \((b_n)\) is a sequence satisfying \[ b_1 \ge b_2 \ge b_3 \ge ... \ge 0 \] then the series \(\sum_{k \ge 1} a_k b_k\) converges.

Lemma: Partial Sum

Define \(s_0 = 0\), then \[ \begin{aligned} \sum_{k \in [n]} x_k y_k &= \sum_{k \in [n]} (s_k - s_{k - 1}) b_k \\ &= \sum_{k \in [n]} s_k b_k - \sum_{k \in [n-1]} s_k b_{k + 1} \\ &= s_n b_n + \sum_{k \in [n-1]} s_k (b_k - b_{k + 1}). \end{aligned} \]

Proof. \((b_n)\) is monotone and bounded, therefore \((b_n)\) converges. Since both \((s_n)\) and \((b_n)\) converges, \((s_n b_n)\) converges.

It remains to prove \(\sum_{k \in [n-1]} s_k (b_k - b_{k + 1})\) converges. We show it is a Cauchy sequence. We know the convergent sequence \((s_n)\) is bounded. Let \(| s_n | \le M\). Thus, for \(n > m\), \[ \left| \sum_{k = m}^{n - 1} s_k (b_k - b_{k + 1}) \right| \le M \sum_{k = m}^{n - 1} (b_k - b_{k + 1} ) = M (b_m - b_n) \]

The fact \((b_n)\) converges implies that \(b_m - b_n \rightarrow 0\) for large \(m\) and so, \(|\sum_{k = m}^{n - 1} s_k (b_k - b_{k + 1})| \rightarrow 0\).

\(\square\)

Dirichlet's Test

If the series \(s_n = \sum_{k \in [n]} a_n\) is bounded (but not necessary convergent), and if \((b_k)\) is a sequence satisfying \[ \begin{cases} b_1 \ge b_2 \ge b_3 \ge ... \ge 0 \quad and \\ \lim b_k = 0 \end{cases} \] then the series \(\sum_{k \ge 1} a_k b_k\) converges.

Proof. Let \(|s_n| \le M\). As \((b_n) \rightarrow 0\), \((s_n b_n) \rightarrow 0\). The proof for the convergence of \(\sum_{k \in [n-1]} s_k (b_k - b_{k + 1})\) is exactly the same as that of Abel's test.

\(\square\)

Reference

[1]. Stephen Abbott, Understanding Analysis, Second Edition.

The Hahn Decomposition

Posted on 2019-02-21 Edited on 2022-01-19

Let \((\Omega, \mathcal{F})​\) be a measurable space.

Definition (Additive Set Function). \(\mu: \mathcal{F} \rightarrow \mathbb{R}\) is an additive set function, if for all disjoint \(A_1, A_2, \ldots \in \mathcal{F}\), \[ \mu( \cup_{n \ge 1 } A_n ) = \sum_{n \ge 1} \mu (A_n). \]

Definition (Positive/Negative Set). A set \(A \in \mathcal{F}\) is positive (negative) if \(\mu (E) \ge 0\) (resp \(\mu (E) \le 0\)), for each measurable set \(E\) in \(A\).

The following theorem states that we can decompose \(\Omega\) into a positive set and a negative one.

Theorem (The Hahn Decomposition). There exists a partition \(\Omega = S^+ \cup S^-\) such that

  1. \(S^+\) is measurable and positive.
  2. \(S^-\) is measurable and negative.

Proof.

If \(\Omega\) itself is positive, then the proof is trivial.

Otherwise, \[ s^+ = \sup \{ \mu (E) : E \in \mathcal{F} \}. \]

Suppose that there is an \(S^+ \in \mathcal{F}\) such that \(\mu(S^+) = s^+\), then

  1. \(S^+\) should be positive: if not, there exists a measurable \(E \subset S^+\) such that \(\mu(E) < 0\). Then \(\mu(S^+ - E) = \mu(S^+) - \mu(E) > \mu(S^+)\), a contradiction.

  2. \(S^- \doteq \Omega - S^+\) should be negative: if not, there exists a measurable \(E \subset S^-\) such that \(\mu(E) > 0\). Then \(\mu(S^+ \cup E) = \mu(S^+) + \mu(E) > \mu(S^+)\), a contradiction.

It is left to prove that such \(S^+\) exists.

  1. Let \(A_n \in \mathcal{F}\), \(n \in \mathbb{N}^+\) be a sequence of measurable sets such that \(\mu(A_n) \uparrow s^+\).

  2. Let \(A = \cup_{n \ge 1} A_n\).

  3. For each \(\omega \in A\), define the sequence \(x_\omega = (x_\omega [1], x_\omega[2], \ldots )\), such that \[ x_\omega [n] = \begin{cases} 1, & \omega \in A_n, \\ 0, & \omega \in \bar A_n. \end{cases} \qquad \forall n \in \mathbb{N}^+. \]

  4. For each \(n \in \mathbb{N}^+\), and each \(y \in \{0, 1 \}^n\), define \[ B_{n, y} \doteq \{ \omega \in A : x_\omega = y \}. \]

    Then \(\mathcal{P}_n \doteq \{ B_{n,y} : y \in \{0, 1 \}^n \}\) constitutes a partition of \(A\). For each \(n \in \mathbb{N}^+\), the partition \(\mathcal{P}_{n + 1}\) refines the partition \(\mathcal{P}_n\), in the sense that each \(B_{n, y} \in \mathcal{P}_n\) can be decomposed into \[ B_{n, y} = B_{n + 1, y \circ 0 } \cup B_{n + 1, y \circ 1 }, \] where \(y \circ 0, y \circ 1 \in \{0, 1 \}^{n + 1}\) are the concatenation of \(y\) with \(0\) and \(1\), respectively. Note that \[ \begin{aligned} \mu( B_{n, y} ) &= \mu( B_{n + 1, y \circ 0 } ) + \mu( B_{n + 1, y \circ 1 } ) \\ &\le \max \big\{ 0, \mu( B_{n + 1, y \circ 0 } ) \big\} + \max \big\{ 0, \mu( B_{n + 1, y \circ 1 } ) \big\}. \end{aligned} \]

  5. For \(n \in \mathbb{N}^+\), define \[ C_n \doteq \underset{ B_{n, y} \in \mathcal{P}_n \, \wedge \, \mu( B_{n, y} ) \ge 0 }{\cup} \, B_{n, y}. \]

  6. For \(n \in \mathbb{N}^+\), define \[ D_n \doteq \cup_{k \ge n} C_k. \] It is easy to see that \(\mu(D_n) \ge \mu(A_n)\). Further,

    • \(D_1 \supset D_2 \supset D_3 \supset \ldots\) is a decreasing sequence.

    • \(\mu(D_n) \le \mu(D_{n + 1})\). Since if \(B_{n, y} \subset D_n\), it can be replaced by \(B_{n + 1, y \circ 0 } \cup B_{n + 1, y \circ 1 }\) in \(D_{n + 1}\), incurring no lose in the measure. But we can also drop \(B_{n + 1, y \circ 0 }\) or \(B_{n + 1, y \circ 1 }\) if one of them has negative measure.

Finally, define \(S^+\) by \[ S^+ = \cap_{n \ge 1} D_n = \lim_n D_n. \]

It holds that \(D_n \downarrow S^+\) and \(\mu(D_n) \uparrow \mu(S^+)\).

\(\square\)

Remark: Let \(R \subset \mathbb{R}\), and \(t^* = \sup R\). Then it is possible that \(t^* \notin R\). For example, let \(R = \{ 1 - 1 / n : n \in \mathbb{N}^+ \}\). We see \(t^* = 1 = \sup R\). But \(1 \notin R\).

In our analysis, we can an \(S^+ \in \mathcal{F}\) such that \[ \mu(S^+) = s^+ = \sup \{ \mu (E) : E \in \mathcal{F} \}, \] due to 1) \(\mathcal{F}\) is a \(\sigma\)-field, and 2) \(\mu\) is a signed measure.

There are other examples where the supremum is achieve.

Example. Let \(f: [0, 1] \rightarrow \mathbb{R}\) be a continuous function. Let \(R = \{ f(x) : x \in [0, 1] \}\), and \(t^* = \sup R\). It holds that \(t^* \in R\), since there exists \(x^* \in [0, 1]\), s.t., \(t^* = f(x^*)\). The claim is based on that 1) \([0, 1]\) is closed; 2) \(f\) is continuous.

Reference

[1]. H. Royden, P. Fitzpatrick Real Analysis
[2]. Billingsley. Patrick, Probability and measure

Strong Duality

Posted on 2019-01-19 Edited on 2021-07-15

We prove strong duality of linear program by Farkas Lemma.

Let \(A \in \mathbb{R}^{m \times n}\), \(b \in \mathbb{R}^m\), and \(c, x \in \mathbb{R}^n\). Consider the primal program defined as \[ \begin{array}{ll} \max &c^T x \\ \text{s.t.} &Ax = b \,, \\ &x \ge 0 . \end{array} \]

Let \(y \in \mathbb{R}^m\). The dual program of the primal is given by \[ \begin{array}{ll} \min &y^T b \\ \text{s.t.} &y^T A \ge c^T. \end{array} \]

Theorem. If both primal and dual are feasible, then \(\max = \min\).

Proof 1. Note that \[ c^T x \le (y^T A) x = y^T (A x) = y^T b. \] The first inequality follows from that \(y^T A \ge 0\) and that \(x \ge 0\). The last equality follows from that \(Ax = b\).

This is know as weak duality. It is left to prove the reverse direction.

Assume that the dual is feasible and \(\min = \alpha\). The goal is to prove that \(\max \ge \alpha\). It suffices to find an \(x \in \mathbb{R}^n\) satisfying

  1. \(x \ge 0\),
  2. \(Ax = b\) and,
  3. \(c^T x \ge \alpha\).

Consider the convex and closed set \[ P \doteq \{(z_1, z_2) \in \mathbb{R}^m \times \mathbb{R}: z_1 = A x,\, z_2 \le c^T x,\, x \in \mathbb{R}^n,\, x \ge 0\}. \]

Let \(\bar{ b } \doteq (b, \alpha)\). If \(\bar{ b } \in P\), we have done. Otherwise, we are going to prove that \(\exists y \in \mathbb{R}^m\), such that \[ y^T A \ge c^T ,\, \text{and } \, y^T b < \alpha. \]

Let \(\bar{ z } \in \mathbb{R}^m \times \mathbb{R}\) be the projection of \(\bar{ b }\) to \(P\), i.e., the point in \(P\) that minimizes its distance to \(\bar{ b }\). Denote \(v = \bar{ z } - \bar{ b } = (v_1, v_2) \in \mathbb{R}^m \times \mathbb{R}\). Consider the supporting plane \[ \ell(z) \doteq v^T (z - \bar{ z }). \]

We have \(\ell(\bar{ b }) < 0\) and \(\ell(z) \ge 0\) for all \(z \in P\). As \(0 \in P\) and \(2 \bar{ z } \in P\), it holds that \[ \ell(0) = v^T (-\bar{ z }) \ge 0, \quad \ell(2 \bar{ z }) = v^T (\bar{ z }) \ge 0. \]

It follows that \(v^T \bar{ z } = 0\). Hence \(v^T z \ge 0\) for all \(z \in P\) and \[ v^T \bar{ b } = v_1^T b + v_2 \cdot \alpha < 0. \] We will prove that \[ v_1^T A + v_2 \cdot c^T \ge 0. \]

If we can further prove that \(v_2 < 0\), then \(y = \frac{1}{v_2} \cdot v_1\) satisfies \(y^T A \ge c^T\) and \(y^T b < \alpha\), which leads to a contradiction.

Lemma: \(v^T z \ge 0\) \(\forall z \in P\) then \(v_1^T A + v_2 \cdot c^T \ge 0\).

Proof. For all \(x \ge 0\), it holds that \((Ax, c^T x) \in P\). Hence, \(v_1^T A x + v_2 \cdot c^T x \ge 0\), which implies that \[ v_1^T A + v_2 \cdot c^T \ge 0. \]

\(\square\)

Finally, we prove that \(v_2 < 0\). To see this, observe that \((0, -1) \in P\), since \(0 = A 0\), and \(-1 \le c^T 0\). Now \(v^T (0, -1) = -v_2 > 0\).

\(\blacksquare\)

Proof 2. We can convert the primal into an equivalent one: \[ \begin{array}{r} Ax & = b \,, \\ c^T x - z &= \alpha \,, \\ x,z & \ge 0 \,. \end{array} \] or in matrix form: \[ \begin{array}{r} \bar A \bar x &= \bar b \,,\\ \bar x & \ge 0 \,. \end{array} \]

where \[ \begin{array}{r} \bar A = \begin{bmatrix} A, & 0 \\ c^T, & -1 \end{bmatrix} ,\, \bar x = \begin{bmatrix} x \\ z \end{bmatrix} , \, \bar b = \begin{bmatrix} b \\ \alpha \end{bmatrix} . \end{array} \]

No feasible solution implies that \(\bar b\) lies outside the conic combination of columns of \(\bar A\). Let \(\bar{ z } \in \mathbb{R}^m \times \mathbb{R}\) be the projection of \(\bar{ b }\) to the cone formed by columns of \(\bar A\) and let \(v = \bar{ z} - \bar{ b } = (v_1, v_2) \in \mathbb{R}^m \times \mathbb{R}\). It holds that \(v^T \bar A \ge 0\). Then \[ \begin{array}{l} v^T \begin{bmatrix} A \\ c^T \end{bmatrix} = v_1^T A + v_2 \cdot c^T \ge 0 \,, \\ v^T \begin{bmatrix} 0 \\ -1 \end{bmatrix} = - v_2 \ge 0. \end{array} \]

It is left to verify that \(v_2 < 0\). Note that \[ v^T v = v^T (\bar{ z } - \bar{ b }) = - v^T \bar{ b } = - v_1^T b - v_2 \cdot \alpha > 0. \] If \(v_2 = 0\), then \(v_1^T A \ge 0\) and \(v_1^T b < 0\). Replacing \(y\) with \(v_1\), and scaling it by positive real number, we can obtain a feasible dual solution such that \(y^T b\) is arbitrary small. However, this contradicts the weak duality \[ c^T x \le y^T b. \]

See the following example in 2 dimensions.

In this example, \(A[:, 1] = (-2, 1)\), \(A[:, 2] = (-1, 2)\), \(c = (1, 1)\), \(\bar A [:, 1] = (-2, 1, 1), \bar A[:, 2] = (-1, 2, 1)\). \(\{ Ax = b, x \ge 0 \}\) is feasible implies that \(b\) is inside the cone spanned by \(A[:, 1], A[:, 2]\). \(\{ \bar A \bar x = \bar b, x \ge 0 \}\) implies that \(\bar b = (b, \alpha)\) is outside the cone formed by \(\bar A[:, 1], \bar A[:, 2], [0, 0, -1]\). The \(z\)-dimension of vector \(y\) must be negative.
