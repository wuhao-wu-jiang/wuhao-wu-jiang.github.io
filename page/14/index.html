Fubini's Theorem

Posted on 2021-06-28 Edited on 2023-02-04

Let \((X, \mathcal{F}_1, \mu_1)\) and \((Y, \mathcal{F}_2, \mu_2)\) be two \(\sigma\)-finite measure space.

Produce Measure Space

Consider the product space \(\Omega = X \times Y\).

Definition. A measurable rectangle is a product \(A \times B\) for which \(A \in \mathcal{F}_1\) and \(B \in \mathcal{F}_2\).

Denote the collection of measurable rectangles as \[ \mathcal{S} \doteq \begin{Bmatrix} A \times B, A \in \mathcal{F}_1, B \in \mathcal{F}_2 \end{Bmatrix}. \]

We would like to construct a measure \(\mu\) on \(\sigma(\mathcal{S})\), the \(\sigma\)-field generated by \(\mathcal{S}\), thus

  1. For each \(A \times B \in \mathcal{S}\), it's natural to define \(\mu (A \times B) = \mu_1(A) \cdot \mu_2(B)\).

  2. We prove that \(\mathcal{S}\) is a semialgebra.

  3. We show that \(\mu\) is a measure on \(\mathcal{S}\).

  4. Finally, it follows by the Caratheodory’s Extension Theorem, \(\mu\) has a unique extension on \(\sigma(\mathcal{S})\).

Lemma 1. \(\mathcal{S}\) is a semialgebra.

Proof. * \(\mathcal{S}\) is close under intersection: if \(A \times B, C \times D \in \mathcal{S}\), then \[ (A \times B) \cap (C \times D) = (A \cap C) \times (B \cap D). \] * The complement of element in \(\mathcal{S}\) is a finite disjoint union of elements in \(\mathcal{S}\): if \(A \times B \in \mathcal{S}\), since \[ \Omega = (A \times B) + ( \bar A \times B ) + ( A \times \bar B ) + ( \bar A \times \bar B ), \] it holds that \[ \overline{ A \times B } = ( \bar A \times B ) + ( A \times \bar B ) + ( \bar A \times \bar B ). \]

\(\square\)

Lemma 2. \(\mu\) is a measure on \(\mathcal{S}\).

Proof. It suffices to prove \(\sigma\)-additivity of \(\mu\). Suppose \(A \times B = +_{i \ge 1} (A_i \times B_i)\) is a countable disjoint union of elements in \(\mathcal{S}\).

Consider an \(x \in A\). For each \(y \in B\), we have \((x, y) \in A \times B\). As the sets \(\begin{Bmatrix} A_i \times B_i \end{Bmatrix}\) consists of a partition of \(A \times B\), it follows that \((x, y)\) belongs to exactly one \(A_i \times B_i\). Denote \[ I(x) \doteq \begin{Bmatrix} i : \exists y \in B,\, \text{s.t.},\, (x, y) \in A_i \times B_i \end{Bmatrix} = \begin{Bmatrix} i : x \in A_i \end{Bmatrix}. \] Then \(B = +_{i \in I(x) } B_i\).

As \(\mu_2\) is a measure on \(Y\), it is countably additive: \[ \mu_2 (B) = \sum_{i \in I(x) } \mu(B_i ) \implies \mathbb{I}_A (x) \mu_2(B) = \sum_{i \ge 1} \mathbb{I}_{ A_i } (x) \cdot \mu_2 (B). \]

By monotone convergence theorem, \[ \int_X \mathbb{I}_A(x) \mu_2(B) \, d\, \mu_1 = \sum_{i \ge 1} \int_X \mathbb{I}_{ A_i } (x) \cdot \mu_2 (B) d\, \mu_1, \]

which is \[ \mu_1 (A) \cdot \mu_2 (B) = \sum_{i \ge 1} \mu_1 (A_i) \cdot \mu_2 (B_i). \]

\(\square\)

Theorem. \(\mu\) has a unique extension \(\bar \mu\) on \(\sigma( \mathcal{S} )\), the \(\sigma\)-field generated by \(\mathcal{S}\).

Proof. Note that \(\mathcal{S}\) is a semialgebra and \(\mu\) is a \(\sigma\)-finite measure. Applying Caratheodory’s Extension Theorem directly finishes the proof.

\(\square\)

Fubini's Theorem

Let \(\begin{Bmatrix} a_i : i \in \mathbb{N} \end{Bmatrix}\) be a countably sequence of numbers. Then the series converges to the same limit under permutation, if

  1. \(a_i \ge 0, \forall i \in \mathbb{N}\). Note that \(\sum_{i \ge 1} a_i\) can equal \(\infty\).

  2. \(\sum_{i \ge 1} | a_i | < \infty\).

Integration is a generalization of discrete sum. Given a measure space \((\Omega, \sigma( \mathcal{S} ), \bar \mu)\), and a measure function \(f\), the integral of \(f\) exists, if

  1. \(f \ge 0\);
  2. Or \(\int |f| d\, \bar \mu < \infty\).

In case that \((\Omega, \sigma( \mathcal{S} ), \bar \mu)\) is a produce measure space, the Fubini's theorem states that if the double integral exists, it can be computed by iterated integrals.

Theorem (Fubini's Theorem). If \(f \ge 0\) or \(\int |f| d\, \bar \mu < \infty\), then there exists \(A \in \mathcal{F}_1\), \(B \in \mathcal{F}_2\), s.t., \(\mu_1 (A) = \mu_2 (B) = 0\), and \[ \int_{X - A} \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f(x, y) d\, \bar \mu = \int_{Y - B} \int_X f(x, y) d\, \mu_1 \, d\, \mu_2. \]

Remark. For the case concerning \(f \ge 0\), we have \(A = B = \emptyset\), and the Theorem is called Tonelli’s theorem.

Comparison with Discrete Sum

Let \(X = Y = \mathbb{N}\), and \(\mu_1 = \mu_2\) be counting measures. Let \(\begin{Bmatrix} a_{i,j} : i, j \in \mathbb{N} \end{Bmatrix}\) be a sequency of numbers and define \(f(i, j) = a_{i,j}\). Denote \[ E_1 \doteq \begin{Bmatrix} (i, j) \in \mathbb{N} \times \mathbb{N} : a_{i,j} \ge 0 \end{Bmatrix}, \\ E_2 \doteq \begin{Bmatrix} (i, j) \in \mathbb{N} \times \mathbb{N} : a_{i,j} < 0 \end{Bmatrix}. \]

By definition of integration, \(\int_\Omega f(x, y) \, d\, \bar \mu\) exists if

  1. Either \(f \ge 0\). In this case \[ \int_\Omega f(x, y) \, d\, \bar \mu = \sup \begin{Bmatrix} \sum_{ ( i,j ) \in S } a_{i,j} : \begin{array}{l} S \subseteq \mathbb{N} \times \mathbb{N} \\ |S| < \infty \end{array} \end{Bmatrix}, \]

  2. Or \(\int_\Omega | f(x, y) | \, d\, \bar \mu < \infty\). In this case, \[ \begin{aligned} &\int_\Omega f(x, y) \, d\, \bar \mu = \int_\Omega f_+ \, d\, \bar \mu - \int_\Omega f_- \, d\, \bar \mu \\ &= \sup \begin{Bmatrix} \sum_{ ( i,j ) \in S } a_{i,j}: \, \begin{matrix} S \subseteq E_1 \\ |S| < \infty \end{matrix} \end{Bmatrix} - \sup \begin{Bmatrix} \sum_{ ( i,j ) \in S } a_{i,j}: \, \begin{matrix} S \subseteq E_2 \\ |S| < \infty \end{matrix} \end{Bmatrix} . \end{aligned} \]

In the former case, the summation of elements in a chosen set \(S\) grows in one direction. While in the later, the summation converges absolutely.

When \(\int_\Omega f(x, y) \, d\, \bar \mu\) exists, the following iterated integrals exist: \[ \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 = \sum_{i \ge 1} \sum_{j \ge 1} a_{i,j}, \\ \int_Y \int_X f(x, y) d\, \mu_1 \, d\, \mu_2 = \sum_{j \ge 1} \sum_{i \ge 1} a_{i,j}. \]

Further, their values are equal.

\(\square\)

Proof of Fubini's Theorem

By symmetry, it suffices to prove that \[ \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f(x, y) d\, \bar \mu. \]

Sketch of the Proof

Before proving that the iterated integral equals the double integral, we need to make sure that the first iterated integral is well-defined. Specifically,

  • When \(x\) is fixed, the function \(f_x : Y \rightarrow \mathbb{R}\) defined by \(f_x(y) \doteq f(x, y)\) is \(\mathcal{F}_2\) measurable, and the integral \(\int_Y f(x, y) \, d\, \mu_2\) exists.

  • There is \(\mathcal{F}_1\) measurable function \(g : X \rightarrow \mathbb{R}\) such that \(g(x) = \int_Y f(x, y) \, d\, \mu_2\) almost everywhere.

We will first establish the theorem for indicator functions \(\mathbb{I}_E\) for sets \(E \in \sigma( \mathcal{S} )\), which serve as basic building blocks for extending the theorem for more general functions.

Lemma. Given \(E \in \sigma( \mathcal{S} )\), let \(\mathbb{I}_E\) be the indicator for \((x, y) \in E\). Then

  1. For each \(x \in X\), the function \(f_x(y) \doteq \mathbb{I}_E (x, y)\) is \(\mathcal{F}_2\) measurable.

  2. The function \(g(x) \doteq \int_Y \mathbb{I}_E (x, y) d\, \mu_2\) is \(\mathcal{F}_1\) measurable.

  3. \[ \int_X \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \]

The proof of the lemma is deferred to the end of the article.

Proof of the Theorem

For now, we apply it to prove Theorem for the following three increasingly more general cases.

Case One. \(f\) is a simple function on \((X \times Y, \sigma(\mathcal{S}))\).

Case Two. \(f \ge 0\) and is \(\sigma(\mathcal{S})\) measurable.

Case Three. \(f\) is measurable and integrable.

Remark: this follows exactly the same order with which we define Lebesgue integrals.

Proof.

Case 1. Suppose that \(f(x, y) = \sum_{i \in [n]} c_i \cdot \mathbb{I}_{E_i} (x, y)\) is a simple function on \((X \times Y, \sigma(\mathcal{S}))\).

  1. For fixed \(x \in X\), according the lemma, the functions \(f_x^{ (i) } (y) \doteq \mathbb{I}_{E_i} (x, y), i \in [n]\) are \(\mathcal{F}_2\) measurable. It follows that function \(f_x (y) = \sum_{i \in [n]} c_i \cdot f_x^{ (i) } (y)\) is \(\mathcal{F}_2\) measurable.

  2. Since \(g^{(i)} (x) \doteq \int_Y \mathbb{I}_{E_i} (x, y) d\, \mu_2\) is \(\mathcal{F}_1\) measurable, so is the function \[ g(x) = \int_Y f_x(y) d\, \mu_2 = \sum_{i \in [n]} c_i \cdot \int_Y \mathbb{I}_{E_i} (x, y) d\, \mu_2 \]

  3. Since \(\int_X \int_Y \mathbb{I}_{E_i} (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_{E_i} (x, y) d\, \bar \mu\), \[ \begin{aligned} \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \sum_{i \in [n]} c_i \cdot \int_X \int_Y \mathbb{I}_{E_i} (x, y) d\, \mu_2 \, d\, \mu_1 \\ &= \sum_{i \in [n]} c_i \cdot \int_{\Omega } \mathbb{I}_{E_i} (x, y) d\, \bar \mu \\ &= \int_{\Omega } \sum_{i \in [n]} c_i \cdot\mathbb{I}_{E_i} (x, y) d\, \bar \mu \\ &= \int_{\Omega } f(x, y) d\, \bar \mu. \end{aligned} \]

Case Two. Suppose that \(f \ge 0\) and is \(\sigma(\mathcal{S})\) measurable. Then there exists simple, nonnegative functions \(f^{ (n) } \uparrow f\).

  1. For fixed \(x \in X\). The functions \(f_x^{ (n) } (y) \doteq f^{ (n) } (x, y) \uparrow f_x(y)\). Since they are \(\mathcal{F}_2\) measurable, so is \(f_x (y) \doteq f(x, y)\).

  2. Let \(g^{(n)} (x) \doteq \int_Y f^{ (n) } (x, y) d\, \mu_2\). Since \(0 \le f_x^{ (n) } (y) \uparrow f_x(y)\), by monotone convergence theorem: \[ g(x) = \int_Y f_x(y) d\, \mu_2 = \lim_n \int_Y f^{ (n) } (x, y) d\, \mu_2 = \lim_n g^{(n)} (x) \] It follows that \(g^{(n)} (x) \uparrow g(x)\). Since they are \(\mathcal{F}_1\) measurable, so is \(g(x)\).

  3. Since \(\int_Y f^{ (n) } (x, y) d\, \mu_2 \uparrow \int_Y f(x, y) d\, \mu_2 \,\), \(\int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \,\), and \(f^{(n)} (x, y) \uparrow f(x, y)\), by monotone convergence theorem, \[ \begin{aligned} \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \lim_n \int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 \\ &= \lim_n \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \\ &= \int_{\Omega } \lim_n f^{ (n) } (x, y) d\, \bar \mu \\ &= \int_{\Omega } f (x, y) d\, \bar \mu. \end{aligned} \]

Case Three. Suppose that \(f(x, y)\) is a measurable and integrable function on \(X \times Y\), i.e., \(\int_{\Omega } |f(x, y)| d\, \bar \mu < \infty\). Write \(f(x, y) = f^+(x, y) - f^-(x, y)\), such that \(f^+(x, y), f^-(x, y) \ge 0\) are measurable and integrable.

  1. For a fixed \(x \in X\), since both \(f^+_x(y) \doteq f^+(x, y), f^-_x(y) \doteq f^-(x, y)\) are \(\mathcal{F}_2\) measurable, so is the function \(f_x (y) \doteq f(x, y) = f^+(x, y) - f^-(x, y)\) is \(\mathcal{F}_2\).

  2. Since \(f^+(x, y), f^-(x, y) \ge 0\) are measurable and integrable, we have
    \[ \begin{aligned} \int_X \int_Y f^+(x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^+(x, y) d\, \bar \mu < \infty, \\ \int_X \int_Y f^-(x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^-(x, y) d\, \bar \mu < \infty. \end{aligned} \] It follows that \(\exists A \subseteq \mathcal{F}_1\), s.t., \(\mu_1(A) = 0\) and \(g^+(x) \doteq \int_Y f^+(x, y) d\, \mu_2\) and \(g^-(x) \doteq \int_Y f^-(x, y) d\, \mu_2\) are finite on \(X \setminus A\). Define \[ g(x) = \begin{cases} \int_Y f_x(y) d\, \mu_2 = g^+(x) - g^-(x), & \text{if } x \in X \setminus A \\ 0, & \text{if } x \in A \end{cases}. \] Then \(g(x)\) is \(\mathcal{F}_1\) measurable.

  3. We have \[ \begin{aligned} \int_{X \setminus A} \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \int_{X \setminus A} g^+(x) \, d\, \mu_1 - \int_{X \setminus A} g^-(x) d\, \mu_2 \, d\, \mu_1 \\ &= \int_X g^+(x) \, d\, \mu_1 - \int_X g^-(x) d\, \mu_2 \, d\, \mu_1 \\ &= \int_{\Omega } f^+(x, y) d\, \bar \mu - \int_{\Omega } f^-(x, y) d\, \bar \mu \\ &= \int_{\Omega } f^+(x, y) - f^-(x, y) d\, \bar \mu \\ &= \int_{\Omega } f (x, y) d\, \bar \mu. \end{aligned} \]

\(\square\)

Proof of the Lemma

Lemma. Given \(E \in \sigma( \mathcal{S} )\), let \(\mathbb{I}_E\) be the indicator for \((x, y) \in E\). Then

  1. For each \(x \in X\), the function \(f_x(y) \doteq \mathbb{I}_E (x, y)\) is \(\mathcal{F}_2\) measurable.

  2. The function \(g(x) \doteq \int_Y \mathbb{I}_E (x, y) d\, \mu_2\) is \(\mathcal{F}_1\) measurable.

  3. \[ \int_X \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \]

There are various ways to prove this lemma. We present two proofs here.

Proof One

Since \(X\) and \(Y\) are both \(\sigma\)-finite, there exists \(A_k \in \mathcal{F}_1, \mu_1(A_k) < \infty, \forall n \in \mathbb{N}^+\), s.t., \(A_k \uparrow X\), and \(B_k \in \mathcal{F}_2, \mu_2(B_k) < \infty, \forall n \in \mathbb{N}^+\), s.t., \(B_k \uparrow Y\). It follows that \(\Omega_k \doteq A_k \times B_k \uparrow \Omega\), and that \(\mu(\Omega_k) < \infty\).

Define \[ \mathcal{L}_k \doteq \{ E \in \sigma(\mathcal{S}) : E \cap \Omega_k \, \text{ satisfies the conditions in the Lemma } \}. \]

Claim. If \(\mathcal{L}_k = \sigma(\mathcal{S})\), then the lemma holds.

Proof of the Claim. For each \(E \in \sigma(\mathcal{S})\),

  1. For fixed \(x \in X\). The functions \(f_x^{ (k) } (y) \doteq \mathbb{I}_{E \cap \Omega_k} (x, y) \uparrow f_x(y)\). Since they are \(\mathcal{F}_2\) measurable, so is \(f_x (y) \doteq f(x, y)\).

  2. Let \(g^{(k)} (x) \doteq \int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2\) are increasing. Since \(0 \le f_x^{ (n) } (y) \uparrow f_x(y)\), by monotone convergence theorem: by monotone convergence theorem: \[ g(x) = \int_Y f_x(y) d\, \mu_2 = \lim_k \int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2 = \lim_k g^{(k)} (x) \] It follows that \(g^{(k)} (x) \uparrow g(x)\). Since they are \(\mathcal{F}_1\) measurable, so is \(g(x)\).

  3. Since \(\int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2 \uparrow \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \,\), \(\int_X \int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \bar \mu \,\), and \(\mathbb{I}_{E \cap \Omega_k} (x, y) \uparrow \mathbb{I}_E (x, y)\), by monotone convergence theorem, \[ \begin{aligned} \int_X \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \, d\, \mu_1 &= \lim_k \int_X \int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2 \, d\, \mu_1 \\ &= \lim_k \int_{\Omega } \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \bar \mu \\ &= \int_{\Omega } \lim_k \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \bar \mu \\ &= \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \end{aligned} \]

\(\square\)

It is left to prove that \(\mathcal{L}_k = \sigma(\mathcal{S})\). The proof relies on the following fact.

Fact. If \(\tilde{S}\) is an algebra, then the monotone class \(\mathcal{M}(\tilde{S})\) generated by \(\tilde{S}\) equals \(\sigma(\tilde{S})\).

The proof consists of two steps:

  • \(\mathcal{L}_k\) contains the algebra \(\tilde{S}\) generated by \(S\).

  • \(\mathcal{L}_k\) is a monotone class.

In what follows, to ease the notations, we write \(\mathcal{L}_k\) as \(\mathcal{L}\) and assume that \(\mu(\Omega) < \infty\).

\(\mathcal{L}\) contains the algebra \(\tilde{S}\) generated by \(S\).

Let \(E = +_i \big( A_i \times B_i \big)\) for a finite sequence of disjoint rectangles \(A_i \times B_i\).

  1. For each \(x \in X\), the function \(f_x(y) \doteq \mathbb{I}_E (x, y)\) is a simple function on \(Y\) and there \(\mathcal{F}_2\) measurable.

  2. The function $$ g(x) = _Y _E (x, y) d, _2 = i {A_i} (x) _2(B_i)

    $$ is a simple function on \(X\) and there \(\mathcal{F}_1\) measurable.

  3. \[ \int_X g(x) \, d\, \mu_1 = \mu_1 (A) \cdot \mu_2 (B) = \bar \mu (E) = \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \]

\(\mathcal{L}\) is a monotone class.

Case One \(E_n \in \mathcal{L}, n \in \mathbb{N}^+\), s.t., \(E_n \uparrow E\)

  1. For each \(x \in X\), we have \(f_x^{ (n) } (y) = \mathbb{I}_{E_n} (x, y) \uparrow f_x(y)\). Since the \(f_x^{ (n) } (y)\) \(\mathcal{F}_2\) measurable, so is the function \(f_x(y)\).

  2. Let \(g^{(n)} (x) \doteq \int_Y f^{ (n) } (x, y) d\, \mu_2\). Since \(0 \le f_x^{ (n) } (y) \uparrow f_x(y)\), by monotone convergence theorem: \[ g(x) = \int_Y f_x(y) d\, \mu_2 = \lim_n \int_Y f_x^{ (n) } (y) d\, \mu_2 = \lim_n g^{(n)} (x) \] It follows that \(g^{(n)} (x) \uparrow g(x)\). Since they are \(\mathcal{F}_1\) measurable, so is \(g(x)\).

  3. Since \(\int_Y f^{ (n) } (x, y) d\, \mu_2 \uparrow \int_Y f(x, y) d\, \mu_2 \,\), \(\int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \,\), and \(f^{(n)} (x, y) \uparrow f(x, y)\), by monotone convergence theorem, \[ \begin{aligned} \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \lim_n \int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 \\ &= \lim_n \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \\ &= \int_{\Omega } \lim_n f^{ (n) } (x, y) d\, \bar \mu \\ &= \int_{\Omega } f (x, y) d\, \bar \mu. \end{aligned} \]

Case Two \(E_n \in \mathcal{L}, n \in \mathbb{N}^+\), s.t., \(E_n \downarrow E\)

  1. For each \(x \in X\), we have \(f_x^{ (n) } (y) = \mathbb{I}_{E_n} (x, y) \downarrow f_x(y)\). Since the \(f_x^{ (n) } (y)\) are \(\mathcal{F}_2\) measurable, so is the function \(f_x(y)\).

  2. Since \(f_x^{ (n) } (y) = \mathbb{I}_{E_n} (x, y) \downarrow f_x(y)\) and \(f_x^{ (n) } (y) \le 1\), \(0 \le 1 - f_x^{ (n) } (y) \uparrow 1 - f_x(y)\). By monotone convergence theorem: \[ \begin{aligned} \mu_2(Y) - \lim_n g^{(n)} (x) &= \lim_n \int_Y \left( 1 - f_x^{ (n) } (y) \right) d\, \mu_2 \\ &= \int_Y \left( 1 - f_x(y) \right) d\, \mu_2 \\ &= \mu_2(Y) - g(x). \end{aligned} \] Since \(\mu_2(Y) < \infty\), \(\lim_n g^{(n)} (x) = g(x)\). Since they are \(\mathcal{F}_1\) measurable, so is \(g(x)\).

  3. Since \(0 \le \mu_2(Y) - \int_Y f^{ (n) } (x, y) d\, \mu_2 \uparrow \mu_2(Y) - \int_Y f(x, y) d\, \mu_2 \,\), \(\int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \,\), and \(0 \le 1 - f^{(n)} (x, y) \uparrow 1 - f(x, y)\), by monotone convergence theorem, \[ \begin{aligned} \mu(\Omega) - \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \int_X \left( \mu_2(Y) - \int_Y f(x, y) d\, \mu_2 \right) \, d\, \mu_1 \\ &= \lim_n \int_X \left( \mu_2(Y) - \int_Y f^{ (n) } (x, y) d\, \mu_2 \right) \, d\, \mu_1 \\ &= \mu(\Omega) - \lim_n \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \\ &= \lim_n \int_{\Omega } \left( 1 - f^{ (n) } (x, y) \right) d\, \bar \mu \\ &= \int_{\Omega } \lim_n \left(1 - f^{ (n) } (x, y) \right) d\, \bar \mu \\ &= \mu(\Omega) - \int_{\Omega } f (x, y) d\, \bar \mu. \end{aligned} \] Observing that \(\mu(\Omega) < \infty\) finishes the proof.

\(\square\)

Proof Two

Lemma. Given \(E \in \sigma( \mathcal{S} )\), let \(f(x, y) = \mathbb{I}_E (x, y)\) be the indicator for \((x, y) \in E\). Then

  1. For each \(x \in X\), the function \(f_x(y) \doteq \mathbb{I}_E (x, y)\) is \(\mathcal{F}_2\) measurable.

  2. The function \(g(x) \doteq \int_Y \mathbb{I}_E (x, y) d\, \mu_2\) is \(\mathcal{F}_1\) measurable.

  3. \[ \int_X \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \]

Proof of (1).

Definition. Let \(E_x \doteq \begin{Bmatrix} y : (x, y) \in E \end{Bmatrix}\) be the cross-section at \(x\).

It suffices to show that

Claim. For each \(x \in X\) and \(E \in \sigma(\mathcal{S})\), \(E_x \in \mathcal{F}_2\).

Proof of the Claim. Fix an \(x \in X\). Define the function \(T_x : Y \rightarrow X \times Y\) by \(T_x (y) \doteq (x, y).\) Then for each \(E = A \times B \in \mathcal{S}\), \[ T_x^{-1} (E) = E_x = \begin{cases} B, &\text{ if } x \in A, \\ \emptyset, & \text{ otherwise }. \end{cases} \]

Therefore \(T_x^{-1} (E) = E_x \in \mathcal{F}_2\). Since \(\mathcal{S}\) generates \(\sigma(\mathcal{S})\), we conclude that \(T_x\) is measurable \(\mathcal{F}_2 / \sigma(\mathcal{S})\).

\(\square\)

Proof of (2) and (3).

Define the set \(\mathcal{L} \subseteq \sigma(\mathcal{S})\) as the collection of \(E\) for which the condition (2) and (3) hold.

  1. Clearly it holds that \(\mathcal{S} \subseteq \mathcal{L}\);
  2. We show that \(\mathcal{L}\) is a \(\lambda\) system.

Since \(\mathcal{S}\) is a \(\pi\) system, via Dynkin's \(\pi-\lambda\) theorem, \(\mathcal{L}\) is a \(\sigma\)-algebra, and hence \(\mathcal{L} = \sigma(\mathcal{S})\).

We will now check that \(\mathcal{L}\) is a \(\lambda\)-system. With the same technique applied in previous proof, we can suppose without loss of generality that \(\mu(\Omega) < \infty\).

  1. \(\Omega \in \mathcal{L}\).

  2. If \(E \in \mathcal{L}\), then \(\bar E \in \mathcal{L}\). First, observe that \(\bar E_x = \Omega_x - E_x = B - E_x \in \mathcal{F}_2\). Since \(\mu_2(B) < \infty\), \[ g_{\bar E} (x) = \mu_2 ( \bar E_x ) = \mu_2 (\Omega_x - E_x) = \mu_2 (B) - \mu_2 (E_x). \] Since \(g_E(x) = \mu_2 (E_x)\) is \(\mathcal{F}_1\) measurable, so is \(g_{\bar E}(x)\). Integrating over \(x\) gives \[ \begin{aligned} \int_X \mu_2 ( \bar E_x ) \, d\, \mu_1 &= \int_X \mu_2 (B) \, d\, \mu_1 - \int_X \mu_2 (E_x) \, d\, \mu_1 \\ &= \mu (\Omega) - \mu(E). \end{aligned} \]

    Since \(\mu(\Omega) < \infty\), we have \(\mu(\bar E) = \mu (\Omega) - \mu(E) = \int_X \mu_2 ( \bar E_x ) \, d\, \mu_1\).

  3. If \(E_n \in \mathcal{L}, n \in \mathbb{N}^+\) is a disjoint sequence of sets, then \(\cup_{n \ge 1} E_n \in \mathcal{L}\). Define \(g_{E_n} (x) \doteq \mu_2 ( (E_n)_x )\). Then \(g(x) \doteq \sum_{n \ge 1} g_{E_n} (x)\) is \(\mathcal{F}_1\) measurable, since \(g = \sup_{n \ge 1} \sum_{k \in [n]} g_{E_k}\). By monotone convergence theorem and that \(\mu\) is a measure, it holds that \[ \int_X g \, d\, \mu_1 = \int_X \sum_{n \ge 1} g_{E_n} \, d\, \mu_1 = \sum_{n \ge 1} \int_X g_{E_n} \, d\, \mu_1 = \sum_{n \ge 1} \mu (E_n) = \mu (E). \] Therefore, \(E \in \mathcal{L}\).

\(\square\)

Remarks

Given a function \(f(x, y)\), there is an easier way to prove that it is measurable \(\mathcal{F}_2 / \mathcal{B}\) when \(x\) is fixed, without approximating it by simple functions. However, to prove the measurability of \(\int_Y f(x, y) \, d \mu_2\) (as a function of \(x\)), we still need to write it as the limit of simple functions.

Lemma. Let \(f: X \times Y \rightarrow \mathbb{R}\) be measurable \(\sigma(\mathcal{S}) / \mathcal{B}\). For each \(x \in X\), the function \(f_x : Y \rightarrow \mathbb{R}\) defined by \(f_x(y) = f(x, y)\) is measurable \(\mathcal{F}_2 / \mathcal{B}\).

Proof. Note that \(f_x = f \circ T_x\). Since \(T_x\) is measurable \(\mathcal{F}_2 / \sigma(\mathcal{S})\) and \(f\) is measurable \(\sigma(\mathcal{S}) / \mathcal{B}\), it follows that \(f_x\) is measurable \(\mathcal{F}_2 / \mathcal{B}\).

Or, specifically, for each Borel set \(W \in \mathcal{B}\), since \(f\) is measurable \(\sigma(\mathcal{S}) / \mathcal{B}\), \[ E = f^{-1} (W) = \{ (x, y) : f(x, y) \in W \} \in \sigma (\mathcal{S}). \]

Then \[ f_x^{-1} (W) = \{ y : f(x, y) \in W \} = E_x \in \mathcal{F}_2. \] \(\square\)

For completeness, we include a proof for the following fact. Observe that the technique applied here is similar to the one used in the proof for \(\pi\)-\(\lambda\) theorem.

Fact. If \(\tilde{S}\) is an algebra, then the monotone class \(\mathcal{M}(\tilde{S})\) generated by \(\tilde{S}\) equals \(\sigma(\tilde{S})\).

Proof. It is trivial to see \(\mathcal{M}(\tilde{S}) \subseteq \sigma(\tilde{S})\).

We prove the other direction by showing that \(\mathcal{M}(\tilde{S})\) is an algebra. The proof consists of two steps:

  1. For each \(E \in \tilde{S}\), we prove that \[ \mathcal{R}(E) \doteq \left\{ F \in \mathcal{M}(\tilde{S}) : E \cup F, F - E, E - F \in \mathcal{M}(\tilde{S}) \right\} = \mathcal{M}(\tilde{S}). \] Clearly \(\tilde{S} \subseteq \mathcal{R}(E)\). We prove that \(\mathcal{R}(E)\) is a monotone class. Suppose that \(F_n \rightarrow F\) is an increasing or decreasing sequence. Then \(E \cup F_n, F_n - E, E - F_n\) are also increasing or decreasing sequences. Therefore, \(E \cup F, F - E, E - F \in \mathcal{R}(E)\).

  2. For each \(E \in \mathcal{M}(\tilde{S})\), we prove that \[ \mathcal{R}(E) \doteq \left\{ F \in \mathcal{M}(\tilde{S}) : E \cup F, F - E, E - F \in \mathcal{M}(\tilde{S}) \right\} = \mathcal{M}(\tilde{S}). \] If \(F \in \tilde{S}\), then the previous claim \(E \in \mathcal{R}(F)\). Therefore, \(F \in \mathcal{R}(E)\). Following similar vein, we can also prove that \(\mathcal{R}(E)\) is a monotone class.

\(\square\)

Jensen's Inequality

Posted on 2021-06-25 Edited on 2022-04-30

Let \((\Omega, \mathcal{F}, \mu)\) be a probability space. Let \(f\) be a random variable, i.e., a measurable map from \(\Omega\) to \(\mathcal{R}\), the Borel sets in \(\mathbb{R}\). Suppose that \(\varphi\) is a convex function defined on the interval \(I = [a, b]\), and that \(f(\Omega) \subset I\).
Suppose that \(f\) and \(\varphi(f)\) are integrable.

Theorem. (Jensen's Inequality) \[ \varphi \left( \int f \, d\, \mu \right) \le \int \varphi (f) \, d\, \mu. \]

Proof. Since \(f(\Omega) \subset I\), by monotonicity of integration, it holds that \[ x^* \doteq \int f \, d\, \mu \in I. \]

Case 1. Suppose that \(x^* \in (a, b)\). As \(\varphi\) is convex, there exits a linear function \(h(x) = c x + d\) such that

  • It passes \((x^*, \varphi( x^*) )\), i.e., \(h(x^*) = c x^* + d = \varphi( x^*)\).
  • \(\varphi (x) \ge h(x)\) for all \(x \in I\).

Hence, by monotonicity of integration, \[ \int \varphi (f) \, d\, \mu \ge \int (c f + d) \, d\, \mu = c x^* + d = \varphi (x^*). \]

Case 2. \(x^* = a\) (or \(b\)). Then \[ \int \big( \varphi (f) - a \big) \, d\, \mu = 0. \]

It follows that \(f( \omega) = a\) a.e. \(\omega \in \Omega\). Clearly \[ \varphi \left( \int f \, d\, \mu \right) = \varphi (a) = \int \varphi (a) \, d\, \mu. \]

\(\square\)

Example. Let \(\varphi(x) = - \ln x, \forall x > 0\). Let \(x_1, \ldots, x_n \in (0, \infty)\). Denote \(\bar x = (1 / n) \sum_{i} x_i\). \[ \varphi( \bar x ) \le \frac{1}{n} \sum_i (- \ln x_i ). \]

Instead invoking the theorem directly, we apply the same technique to prove this again. The claim is trivial if \(x_1 = \ldots = x_n\). Otherwise, consider the tangent line of \(\varphi(x)\) at \(\bar x\):
\[ h(x) = - \frac{1}{ \bar x } \cdot (x - \bar x) - \ln \bar x. \]

It holds that \(\varphi(x_i) \ge h(x_i)\), for all \(i \in [n]\). Therefore, \[ \frac{1}{n} \sum_{i} h(x_i) \le \frac{1}{n} \sum_i (- \ln x_i ). \]

On the other hand, \[ \frac{1}{n} \sum_{i} h(x_i) = - \frac{1}{ \bar x } \cdot \Big( \frac{1}{n} \sum_{i} x_i - \bar x \Big) - \ln \bar x = - \ln \bar x. \]

Lebesgue Integral, Revisited.

Posted on 2021-06-19 Edited on 2022-02-08

Let \((\Omega, \mathcal{F}, \mu)\) be measure space. Further, suppose that \(\mu\) is \(\sigma\)-finite, i.e., there is a countable sequence \(\{ S_k \}\) in \(\mathcal{F}\), such that

  1. \(S_k \uparrow \Omega\).
  2. \(\mu (S_k) < \infty\).

We study the integral, \(\int f \, d\, \mu\), of a measurable function \(f\). In what follows, we omit \(d\, \mu\) and abbreviate \(\int f \, d\, \mu\) as \(\int f\).

Definition

Simple Function

Definition. For each \(A \in \mathcal{F}\), define the indicator function \(\mathbf{1}_A : \Omega \rightarrow \mathbb{R}\) as \[ \mathbf{1}_A (\omega) = \begin{cases} 1 \quad \omega \in A\, \\ 0 \quad \omega \notin A \end{cases} \qquad \forall \omega \in \Omega. \]

Definition. \(\varphi : \Omega \rightarrow \mathbb{R}\) is said to be a simple function if \(\varphi = \sum_{i \in [n] } a_i \mathbf{1}_{ A_i }\), where \(A_i \in \mathcal{F}\) are mutually disjoint.

We can assume that the \(\{ A_i \}\) is a partition of \(\Omega\). Otherwise, let \(A_{n + 1} = \Omega - A_1 - \ldots - A_n\) and \(a_{n + 1} = 0\). Then \(A_1, \ldots, A_{n + 1}\) is a partition of \(\Omega\), and \[ \varphi = \sum_{ i \in [n + 1] } a_i \mathbf{1}_{ A_i }. \]

The definition of integral of measurable functions consists of two steps.

Nonnegative Function

Bending the notation slightly, we say that \(f\) is nonnegative, denoted as \(f \ge 0\), if for all \(\omega \in \Omega\), \(f(\omega) \ge 0\).

Definition \(\diamondsuit\). Let \(f \ge 0\) be measurable. Define \[ \int f \doteq \sup \left\{ \sum_{i \in [n] } a_i \mu(A_i): \begin{array}{l} \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \text{ is simple}, \\ 0 \le \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \le f \end{array} \right\}. \] It is possible that \(\int f = \infty\). We say that \(f\) is integrable if \(\int f < \infty\).

For each nonnegative simple function \(\varphi = \sum_{i \in [ n ] } a_i \mathbf{1}_{A_i}\), define \(\varphi_k \doteq \mathbf{1}_{S_k} \cdot \varphi = \sum_{i \in [n] } a_i \mathbf{1}_{A_i \cap S_k}\). Note that \(\varphi_k\) has finite support and \(\varphi_k \rightarrow \varphi\). By continuity of \(\mu\), \[ \sum_{i \in [n] } a_i \mu(A_i) = \lim_k \sum_{i \in [n] } a_i \mu(A_i \cap S_k). \]

Hence, the integral of \(f\) has an equivalent definition.

Definition \(\heartsuit\). Let \(f \ge 0\) be measurable. Define \[ \int f \doteq \sup \left\{ \sum_{i \in [n] } a_i \mu(A_i): \begin{array}{ll} \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \text{ is simple}, \\ \mathbb{Supp} \Big( \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \Big) < \infty, \\ 0 \le \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \le f \end{array} \right\}. \]

General Function

A general function \(f : \Omega \rightarrow \mathbb{R}\) is said to be integrable if \(\int |f| < \infty\). Let \(f_+ = \max \{ f, 0\}\) and \(f_- = \max \{ - f, 0 \}\). Then \(f = f_+ - f_-\).

Definition \(\clubsuit\). Let \(f\) be measurable. Define \[ \int f \doteq \int f_+ - \int f_-. \]

Integration over a subset

Let \(f: \Omega \rightarrow \mathbb{R}\) be measurable and \(E \in \mathcal{F}\). Then \[ \int_E f \doteq \int f \cdot \mathbf{1}_{E}. \]

Remark: note that if \(f\) is measurable and \(\mathbf{1}_{E}\) is measurable, then \(f \cdot \mathbf{1}_{E}\) is measurable.

Basic Properties

  1. Monotonicity. If \(f \ge g\) then \(\int f \le \int g\).

  2. Linearity. The integral is linear on the vector space of integral functions; in particular, \(\forall \alpha, \beta \in \mathbb{R}\), \(\int \alpha \cdot f + \beta \cdot g = \alpha \cdot \int f + \beta \cdot \int g\).

Monotonicity follows directly from the definition. Establishing linearity is more fiddly and we leave it to the end of the article. We proceed by assuming these two properties of integration.

Properties for Limits

Suppose that \(f_n \rightarrow f\) a.e., when can we argue that \[ \int f_n \rightarrow \int f, \] i.e., when the order of the limiting and the integration operations can be exchanged?

Intuitively, there are two kinds of convergence: the monotone ones and the bounded ones.

Monotone Convergence

Theorem (Monotone Convergence). If \(f_n \ge 0\) and \(f_n \uparrow f\), then \[ \int f = \lim_{n \ge 1} \int f_n. \]

Corollary 1. The integral is \(\sigma\)-additive for nonnegative functions.

Corollary 2. If \(f_n \ge g\), where \(g\) is integrable, and \(f_n \uparrow f\), then \[ \int f = \lim_{n \ge 1} \int f_n. \]

Proof. As \(f \ge f_n\), by monotonicity of integration, \(\int f \ge \int f_n\). It follows that \[ \int f \ge \lim_{n} \int f_n. \]

It is left to prove the reverse direction of the inequality. Let \(\varphi \doteq \sum_{i \in [k] } a_i \mathbf{1}_{ A_i }\) be a simple function such that \(\varphi \le f\). Let \(c \in (0, 1)\), and define \[ E_n \doteq \{ \omega \in \Omega : f_n(\omega) \ge c \cdot \varphi (\omega) \}. \]

Exercise. Prove that \(E_n \in \mathcal{F}\).

Then

  1. \(c \cdot \varphi \cdot 1_{E_n} \le f_n\) is also a simple function.

  2. \(\lim_n E_n = \Omega\), since \(f_n \uparrow f\).

By the definition of \(\int f_n\), we have \[ \int c \cdot \varphi \cdot 1_{E_n} = \sum_{i \in [k] } c \cdot a_i \cdot \mu \big( A_i \cap E_n \big) \le \int f_n. \]

Taking limit of both side, and by continuity of \(\mu\), we get \[ \small \lim_n \sum_{i \in [k] } c \cdot a_i \cdot \mu \big( A_i \cap E_n \big) = c \cdot \sum_{i \in [k] } a_i \cdot \mu \big( A_k \big) = c \cdot \int \varphi \le \lim_n \int f_n. \]

Taking supremum over \(\varphi \le f\) and letting \(c \rightarrow 1\), we get \[ \int f \le \lim_n \int f_n. \]

\(\square\)

Fatou's Lemma

Theorem (Fatou's Lemma). If \(f_n \ge 0\), then \[ \int \liminf_n f_n \le \liminf_n \int f_n. \]

Corollary. If \(f_n \le 0\) and \(g\) is integrable, then \[ \limsup_n \int f_n \le \int \limsup_n f_n . \]

Corollary. If \(f_n \ge g\) and \(g\) is integrable, then \[ \int \liminf_n f_n \le \liminf_n \int f_n. \]

Corollary. If \(f_n \le g\) and \(g\) is integrable, then \[ \limsup_n \int f_n \le \int \limsup_n f_n . \]

Proof. For all \(n \ge 1\), define \(h_n = \inf_{k \ge n} f_k\). Since \(h_n \le f_n\), by monotonicity of integration, \(\int h_n \le \int f_n\). Taking \(\liminf\) of both side, we get \[ \liminf_n \int h_n \le \liminf_n \int f_n. \]

Then \(h_n \uparrow \liminf_n f_n\), by monotonicity of integration, \(\int h_n\) is an increasing sequence. It holds that \(\liminf_n \int h_n = \lim_n \int h_n\). It is left to prove \[ \int \lim_n h_n \le \lim_n \int h_n. \]

This follows exactly from the Monotone Convergence theorem, or can be proven directly by using the same technique.

\(\square\)

Bounded Convergence

Theorem (Bounded Convergence). Let \(E \in \mathcal{F}\) with \(\mu(E) < \infty\). Let \(f_n\) be a sequence of functions supported on \(E\) with \(|f_n| \le M\). Suppose that \(f_n \rightarrow f\) point-wise. Then \(\int f_n \rightarrow \int f\).

Proof. The proof relies on the lemma.

Lemma. If \(f_n \rightarrow f\) point-wise, then \(f_n \rightarrow f\) in measure.

Suppose \(f_n \rightarrow f\) in measure. Let \(\epsilon > 0\). Define \(A_n \doteq \{ \omega \in \Omega : |f_n - f| > \epsilon \} \subset E\). Then by linearity and monotonicity of integration, \[ \begin{aligned} \left| \int f_n - \int f \right| &= \left| \int_{A_n} (f_n - f) + \int_{E - A_n} (f_n - f) \right| \\ &\le \int_{A_n} | f_n - f | + \int_{E - A_n} | f_n - f | \\ &\le 2 \cdot M \cdot \mu (A_n) + \epsilon \cdot \mu(E). \end{aligned} \] Since \(\mu(A_n) \rightarrow 0\) and \(\epsilon\) is arbitrary, this shows \(\int f_n \rightarrow \int f\).

\(\square\)

Modulus of Integrability

Theorem (Modulus of integrability). Let \(f \ge 0\) be integrable. Then for any \(\epsilon > 0\) there is a \(\delta > 0\) such that \(m(E) < \delta \implies \int_E f < \epsilon\).

Proof. Define \(f_n = \min( f, n) \cdot \mathbb{1}_{S_n}\). Then \(f_n \uparrow f\) and by monotone convergence theorem, when \(n\) is large enough, \[ \int f - \int f_n \le \epsilon. \]

On the other hand, by monotonicity of integration, when \(\delta < \epsilon / n\), \[ \int_E f_n \le \int_E n \le n \delta < \epsilon. \]

Applying linearity and monotonicity of integration again, \[ \int_E f = \int_E (f - f_n) + \int_E f_n \le \int (f - f_n) + n \delta = 2 \epsilon. \]

\(\square\)

Dominated Convergence

Theorem (Dominated Convergence). If \(f_n \rightarrow f \, a.e.\), \(|f_n| \le g\) for all \(n\), and \(g\) is integrable, then \(\int f_n \rightarrow \int f\).

Proof 1.

Step 1. \(\exists E \in \mathcal{F}\) with \(\mu(E) < \infty\), such that \(\int_{\Omega - E} g \le \epsilon\).

Define \(g_k = g \cdot \mathbb{1}_{S_k}\). Then \(g_k \uparrow g\). Using monotone convergence theorem and linearity of integration, there exists \(K \ge 0\), \[ \int g - \int g_K = \int_{\Omega - S_K} g \le \epsilon. \]

Then we let \(E \leftarrow S_k\).

Step 2.
Since \(f_n \rightarrow f\) a.e., \(f_n \rightarrow f\) in measure. Define \(A_n \doteq \{ \omega \in E : |f_n - f| > \epsilon \} \subset E\). Then \(\mu(A_n) \rightarrow 0\).

Finally, by linearity and monotonicity of integration, \[\small \begin{array}{ll} \left| \int f_n - \int f \right| &\le \int_{A_n} | f_n - f | + \int_{E - A_n} | f_n - f | + \int_{\Omega - E} | f_n - f | \\ &\le 2 \int_{A_n} g + \epsilon \cdot \mu(E) + 2 \cdot \epsilon. \end{array} \]

As \(\mu(A_n) \rightarrow 0\), \(\int_{A_n} g \rightarrow 0\). Since \(\epsilon\) is arbitrary, this shows \(\int f_n \rightarrow \int f\).

\(\square\)

Proof 2. There is a concise proof by Fatous' lemma.

Let \(h_n = 2g - |f - f_n| \ge 0\). Using Fatous' lemma, \[ \int \lim_n \left( 2 g - |f - f_n | \right) \le \liminf_n \int 2 g - |f - f_n| . \]

As \(g\) is integrable, we can subtract \(\int g\) from both side and get \[ \limsup_n \int |f - f_n| \le 0. \]

Therefore, \[ \lim_n \int |f - f_n | = 0. \]

\(\square\)

Proof 3. Since \(|f_n| \le g\), we have \(- g \le f_n \le g\). As \(f_n - (-g) = f_n + g \ge 0\), via Fatous' lemma, we get \[ \limsup \int f_n = \int \limsup f_n = \int \lim f_n = \int \liminf f_n \le \liminf \int f_n. \]

\(\square\)

Linearity

It is difficult to prove linearity by the definitions of \(\diamondsuit\) or \(\heartsuit\). Hence, we introduce some equivalent definitions for integration, starting from simple functions. These definitions also provide deeper intuitions.

Simple Functions

First, we prove that definition \(\diamondsuit\) has an equivalent one for nonnegative simple functions.

Theorem. Let \(\varphi \ge 0\) be a simple function and \(\sum_{j \in [m] } b_j \mathbf{1}_{B_j}\) be one of its possible representations. Then it holds that \[ \small \int \varphi \doteq \sup \left\{ \sum_{i \in [n] } a_i \mu(A_i): \begin{array}{l} \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \text{ is simple}, \\ 0 \le \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \le \varphi \end{array} \right\} = \sum_{j \in [m] } b_j \mu (B_j). \]

Proof. By the definition of \(\varphi\), clearly it holds that \[ \sum_{j \in [m] } b_j \mu (B_j) \le \int \varphi. \]

To prove the reverse inequality, let \(\psi \doteq \sum_{i \in [n] } a_i \mathbf{1}_{A_i}\) be a simple function, where \(A_i \in \mathcal{F}\) are mutually disjoint. Note that the \(\{ B_j \}\) are also mutually disjoint. If \(\psi \le \varphi\), it holds that for all \(i, j\), either \(a_i \le b_j\) or \(A_i \cap B_j = \emptyset\). Using that \(\mathcal{F}\) is a \(\sigma\)-field and that \(\mu\) is a measure on \(\mathcal{F}\), \[ \begin{aligned} \sum_{i \in [n] } a_i \mu (A_i) &= \sum_{i \in [n], j \in [m] } a_i \mu (A_i \cap B_j ) \\ &\le \sum_{i \in [n], j \in [m] } b_j \mu (A_i \cap B_j ) = \sum_{j \in [m] } b_j \mu (B_j). \end{aligned} \]

Taking the supremum over all possible \(\psi \le \varphi\) finishes the proof.

\(\square\)

Combined with definition \(\clubsuit\), we get

Corollary. Let \(\varphi\) be a simple function and \(\sum_{j \in [m] } b_j \mathbf{1}_{B_j}\) be one of its possible representations. \[ \small \int \varphi = \sum_{j \in [m] } b_j \mu (B_j). \]

We are ready to prove the linearity of integration for simple functions.

Linearity. The integral is linear on the vector space of simple functions.

Proof. If \(\varphi\) is a simple function, clearly \(\int c \cdot \varphi = c \cdot \int \varphi\) for all \(c \in \mathbb{R}\). Let \(\varphi = \sum_{i \in [n] } a_i \mathbf{1}_{ A_i }\) and \(\psi = \sum_{j \in [m] } b_j \mathbf{1}_{B_j}\). It is left to verify that \(\int \varphi + \psi = \int \varphi + \int \psi\). As both the \(\{ A_i \in \mathcal{F} : i \in [n] \}\) and \(\{ B_j \in \mathcal{F} : j \in [m] \}\) are mutually disjoint, so are \(\{ A_i \cap B_j \in \mathcal{F} : i \in [n], j \in [m] \}\). Then \[ \begin{aligned} \varphi + \psi &= \sum_{i \in [n], j \in [m] } (a_i + b_j) \cdot \mathbf{1}_{ A_i \cap B_j}\, \\ \int \varphi + \psi &= \sum_{i \in [n], j \in [m] } (a_i + b_j) \cdot \mu( A_i \cap B_j ). \end{aligned} \] By rearranging the term on the right hand side, and by finite additivity of \(\mu\), we get the desired result.

\(\square\)

Bounded Functions

Let \(E \in \mathcal{F}\) be bounded with \(\mu(E) \le \infty\). Let \(f\) be a function whose support \(\mathbb{Supp} (f) \subset E\) and suppose that \(f\) is bounded on \(E\), i.e., \(\exists M > 0\), s.t., \(\forall \omega \in E, | f (\omega) | \le M\). It holds that
\[ \int_E f = \int f = \sup_{ \varphi \le f } \int \varphi \]

where \(\varphi\) ranges over all simple functions.

Lemma. Suppose \(\mathbb{Supp}(f) \subset E\) and \(f\) is bounded on \(E\). Then \(f\) is measurable if and only if \[ \sup_{\varphi \le f } \int \varphi = \inf_{ \psi \ge f } \int \psi \] where \(\varphi\) and \(\psi\) ranges over all simple functions.

Proof. If \(\sup_{\varphi \le f } \int_E \varphi = \inf_{ \psi \ge f } \int_E \psi\), then we can find a sequence of \(\varphi_n, \psi_n\), such that \(\int_E (\varphi_n - \psi_n) \rightarrow 0\). Let \(\varphi = \sup_n \varphi_n\) and \(\psi = \inf_n \psi_n\). It must be the case that \(\varphi = \psi \, a.e.\). Otherwise, \(\exists \epsilon > 0\) and a set \(A \subset E\), such that \(\varphi - \psi > \epsilon\) on \(A\). As the integration for simple functions is monotone, \[ \int \varphi_n - \psi_n \ge \int \epsilon \cdot \mathbf{1}_A > \epsilon \mu(A) > 0. \]

If \(f\) is measurable, we need to prove that there are simple functions \(\varphi \le f \le \psi\) whose integrals are arbitrary close. For all \(\epsilon > 0\), we can find a finite collection of disjoint intervals \(\{ (a_i, b_i] \}\) of length less than \(\epsilon\), such that it covers \([-M, M]\). Define \(E_i = f^{-1} \big( (a_i, b_i] \big)\) and let \(\varphi \doteq \sum_i a_i \mathbf{1}_{ E_i }\) and \(\psi \doteq \sum_i b_i \mathbf{1}_{ E_j }\). Then \(\int_E \psi - \int_E \le \epsilon \cdot \mu(E)\). It concludes that \(\sup_{\varphi \le f } \int_E \varphi \le \inf_{ \psi \ge f } \int_E \psi\). The other direction of the inequality is trivial.

\(\square\)

Linearity. The integral on \(E \in \mathcal{F}\) with \(\mu(E) \le \infty\) is linear on the vector space of bounded and measurable functions.

Proof. Let \(f, g: E \rightarrow \mathbb{R}\) is measurable and bounded. Clearly \(\int c \cdot f = c \cdot \int f\) for all \(c \in \mathbb{R}\). It is left to verify that \(\int f + g = \int f + \int g\). First,
\[ \begin{aligned} \sup_{ \phi \le f + g} \int \phi &\ge \sup_{ \varphi \le f, \psi \le g} \int \varphi + \psi \\ &= \sup_{ \varphi \le f, \psi \le g} \left( \int \varphi + \int \psi \right) = \sup_{ \varphi \le f} \int \varphi + \sup_{ \psi \le g} \int \psi \,, \end{aligned} \]

where the second equality follows from that integral for simple functions are linear. Similarly, \[ \begin{aligned} \inf_{ \phi \le f + g} \int \phi &\le \inf_{ \varphi \le f, \psi \le g} \int \varphi + \psi \\ &= \inf_{ \varphi \le f, \psi \le g} \left( \int \varphi + \int \psi \right) = \inf_{ \varphi \le f} \int \varphi + \inf_{ \psi \le g} \int \psi \,. \end{aligned} \]

\(\square\)

Remark: In the proof we use the fact that for all subset \(A, B \subset \mathbb{R}\), it holds that \(\sup_{a \in A, b \in B} (a + b) = \sup_{a \in A} a + \sup_{b \in B} b\).

Nonnegative Functions

Let \(f : \Omega \rightarrow \mathbb{R}_{\ge 0}\) be a nonnegative function. Then \(\int f = \sup_{0 \le h \le f} \int h\), where \(h\) ranges over all nonnegative bounded functions with finite support. Formally, \[ \int f \doteq \sup \left\{ \int h : 0 \le h \le f,\, \begin{array}{ll} &h \text{ is bounded},\\ &\mu\big( \mathbb{Supp}(h) \big) < \infty \end{array} \right\} \]

It is easy to see that this definition is consistent with definition \(\heartsuit\).

Linearity. The integral is linear on the vector space of positive functions.

Proof 1. Let \(f, g: \Omega \rightarrow \mathbb{R}_{\ge 0}\) be nonnegative functions. Clearly \(\int c \cdot f = c \cdot \int f\) for all \(c \in \mathbb{R}\). It is left to verify that \(\int f + g = \int f + \int g\). First,
\[ \begin{aligned} \sup_{ h \le f + g} \int h &\ge \sup_{ h_1 \le f, h_2 \le g} \int h_1 + h_2 \\ &= \sup_{ h_1 \le f, h_2 \le g} \left( \int h_1 + \int h_2 \right) = \sup_{ h_1 \le f} \int h_1 + \sup_{ h_2 \le g} \int h_2 \,, \end{aligned} \]

where the second equality follows from that integral for bounded functions with finite support are linear. To prove the reverse direction, for each bounded function \(0 \le h \le f + g\), let \(h_1 = \min \{ f, h\}\) and \(h_2 = h - h_1\). Clearly \(0 \le h_1 \le f\). On the other hand, \(0 \le h_2 = h - h_1 \le h - \min \{ f, h \} = \max \{ h - f, 0 \} \le g\). Hence, \[ \int h = \int h_1 + \int h_2 \le \int f + \int g. \]

Taking supremum over \(h\) finishes the proof.

\(\square\)

Proof 2. There is another proof. If we examine the proof of monotone convergence theorem，we can see that it does not rely on linearity of integral. It turns out that we can rely on it to prove linearity of nonnegative functions. Let \(\varphi_n \uparrow f\) and \(\psi_n \uparrow g\) be two sequences of nonnegative simple functions. Using monotone convergence theorem, we get \[ \begin{aligned} \lim \int \varphi_n = \int f, \qquad \lim \int \psi_n = \int g, \qquad \lim \int \varphi_n + \psi_n = \int f + g. \end{aligned} \]

Then using only linearity of integration for simple function, we get \[ \lim \int \varphi_n + \psi_n = \lim \int \varphi_n + \lim \int \psi_n. \]

\(\square\)

General Functions

Linearity. The integral is linear on the vector space of integrable general functions.

Proof. Let \(f, g: \Omega \rightarrow \mathbb{R}\) be integrable functions.

For each \(c \in \mathbb{R}_+\), via the definition of integration,
\[ \int c \cdot f = \int c f_+ - \int c f_- = c \left( \int f_+ - \int f_- \right) = c \int f. \] For each \(c \in \mathbb{R}_-\), via the definition of integration,
\[ \int c \cdot f = \int (-c) \cdot f_- - \int (-c) \cdot f_+ = (-c) \cdot \left( \int f_- - \int f_+ \right) = c \int f. \]

It is left to verify that \(\int f + g = \int f + \int g\). First, \[ \int |f + g| \le \int |f| + |g| = \int |f| + \int |g| < \infty. \]

It follows that \(f + g\) is integrable. Observe \[ (f + g)_+ - (f + g)_- = f_+ - f_- + g_+ - g_- \,, \\ (f + g)_- - (f + g)_+ = f_- - f_+ + g_- - g_+ \,. \]

Suppose that \(|f| < \infty\), and \(|g| < \infty\), then clearly it holds that \[ (f + g)_+ + f_- + g_- = f_+ + g_+ + (f + g)_-. \]

Suppose that \(f + g = \infty\). Then \(0 \le (f + g)_- \le \infty\). And since \(f + g\) is well defined, we must have \(f(\omega) \neq -\infty\) and \(g(\omega) \neq -\infty\). Therefore, \(0 \le f_- \, , \, g_- \le \infty\). We can obtain \[ (f + g)_+ + f_- + g_- = f_+ + g_+ + (f + g)_-. \]

Suppose that \(f + g = -\infty\). Then \(0 \le (f + g)_+ \le \infty\). And since \(f + g\) is well defined, we must have \(f(\omega) \neq \infty\) and \(g(\omega) \neq \infty\). Therefore, \(0 \le f_+ \, , \, g_+ \le \infty\). We can obtain \[ (f + g)_- + f_+ + g_+ = f_- + g_- + (f + g)_+. \]

In all cases, we have \[ (f + g)_+ + f_- + g_- = f_+ + g_+ + (f + g)_-. \]

Integrating for both side, \[ \int (f + g)_+ + \int f_- + \int g_- = \int f_+ + \int g_+ + \int (f + g)_-. \]

Rearranging terms finishes the proof.

\(\square\)
