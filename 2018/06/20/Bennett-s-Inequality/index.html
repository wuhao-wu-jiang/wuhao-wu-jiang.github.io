Bennett's Inequality

Posted on 2018-06-20 Edited on 2021-06-15

Suppose \(X_1, \ldots, X_n\) are independent random variables for which \(|X_i| \le M\) and \(\mathbb{E} [ X_i ] = \mu_i\) and \(\mathbb{E} [ X_i^2 ] \le v_i^2\) for each \(i\), for nonnegative constants \(M\) and \(v_i^2\). Let \(V \doteq \sum_{i \in [n] } v_i^2\). Then for all \(t \ge 0\), \[ \begin{aligned} \Pr \left[ \sum_{i \in [n ] } (X_i - \mu_i) \ge t \right] &\le \left( \frac{ \exp \left( Mt / V \right) }{ \left( 1 + Mt / V \right)^{ 1 + Mt / V } } \right)^{V / M^2} \\ &=\exp \left( - \frac{ t^2 }{ 2 V } \Phi \left( \frac{M t}{V} \right) \right), \end{aligned} \] where \(\Phi\) is a function defined on \([-1, \infty)\) by \[ \Phi (x) \doteq \frac{ (1 + x) \ln (1 + x) - x }{ x^2 / 2 } \quad \text{for } t \neq 0, \text{ and } \Phi(0) = 1. \]

Remark: Let's perform a sanity check before proving it. Suppose that \(X_1, \ldots, X_n\) are i.i.d. Bernoulli random variables each with mean \(\mu\). Then \(X_i \le 1\), \(\mathbb{E}[ X_i^2 ] = \mu\) and \(V = n \mu\). Define \(t = \epsilon \cdot n \mu\). The Bennett's inequality states that \[ \begin{aligned} \Pr \left[ \sum_{i \in [n ] } X_i - n \mu \ge \epsilon \cdot n \mu \right] &\le \left( \frac{ \exp \left( \epsilon \right) }{ \left( 1 + \epsilon \right)^{ 1 + \epsilon } } \right)^{ n \mu }, \end{aligned} \]

which is just the standard Chernoff Bound.

Proof. For all \(\lambda \ge 0\), \[ \begin{aligned} \mathbb{E} [ \exp( \lambda X_i ) ] &= 1 + \lambda \mu_i + \sum_{k \ge 2} \frac{ \lambda^k \mathbb{E} [ X_i^k ] }{ k! } \\ &= 1 + \lambda \mu_i + \sum_{k \ge 2} \frac{ \lambda^k \mathbb{E} [ X_i^2 X_i^{k - 2} ] }{ k! } \\ &\le 1 + \lambda \mu_i + \frac{ \mathbb{E} [ X_i^2 ] }{ M^2 } \cdot \sum_{k \ge 2} \frac{ \lambda^k M^k }{ k! } \quad \text{ since } |X_i^2 X_i^{k - 2}| \le X_i^2 M^{k - 2}\\ &\le 1 + \lambda \mu_i + \frac{ v_i^2 }{ M^2 } \cdot \left( e^{\lambda M} - 1 - \lambda M \right) \\ &\le \exp \left( \lambda \mu_i + \frac{ v_i^2 }{ M^2 } \cdot \left( e^{\lambda M} - 1 - \lambda M \right) \right) \end{aligned} \]

Therefore, \[ \begin{aligned} \mathbb{E} [ \exp( \lambda (X_i - \mu_i) ) ] &\le \exp \left( \frac{ v_i^2 }{ M^2 } \cdot \left( e^{\lambda M} - 1 - \lambda M \right) \right). \end{aligned} \]

Via Markov's inequality, \[ \begin{aligned} \Pr \left[ \sum_{i \in [n ] } (X_i - \mu_i) \ge t \right] &\le \frac{ \mathbb{E} \left[ \exp \left( \lambda \sum_{i \in [n ] } (X_i - \mu_i) \right) \right] }{ \exp( \lambda t) } \\ &\le \exp \left( -\lambda t + \frac{ V }{ M^2 } \cdot \left( e^{\lambda M} - 1 - \lambda M \right) \right) \\ &= \exp \left( -\lambda \Big( t + \frac{V}{M} \Big) + \frac{ V }{ M^2 } \cdot \left( e^{\lambda M} - 1 \right) \right) \end{aligned} \]

Let \(f(\lambda) = -\lambda t + \frac{ V }{ M^2 } \cdot \left( e^{\lambda M} - 1 - \lambda M \right)\). Letting \[ \begin{aligned} f'(\lambda) =& - \Big( t + \frac{V}{M} \Big) + \frac{V}{M} e^{\lambda M} = 0. \end{aligned} \]

We get \(\lambda = \frac{1}{M} \ln \left( \frac{M t}{V} + 1 \right)\). Hence, \[ \begin{aligned} \Pr \left[ \sum_{i \in [n ] } (X_i - \mu_i) \ge t \right] &\le \exp \left( - \frac{1}{M} \Big( t + \frac{V}{M} \Big) \ln \left( \frac{M t}{V} + 1 \right) + \frac{ V }{ M^2 } \cdot \frac{M t}{V} \right) \\ &\le \exp \left( \left\{ \frac{M t}{V} - \Big( 1 + \frac{M t}{V} \Big) \ln \left( 1 + \frac{M t}{V} \right) \right\} \cdot \frac{V}{M^2}\right). \end{aligned} \]

\(\square\).

Reference

[1] David Pollard. Chapter 2, MiniEmpirical, 2015. Manuscript
