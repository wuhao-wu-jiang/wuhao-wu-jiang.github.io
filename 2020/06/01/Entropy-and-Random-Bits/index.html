Entropy

Definition

The entropy of a random variable XXX is defined as

H[X]=E[log⁡21Pr⁡[X]]\mathbf{H}[X] = \mathbb{E} \left[ \log_2 \frac{1}{\Pr[X] } \right] H[X]=E[log2​Pr[X]1​]

Here we adopt the convention that 0log⁡0=00 \log 0 = 00log0=0, as lim⁡x→0+xlog⁡x=lim⁡x→∞1xlog⁡1x=0\lim_{x \rightarrow 0^+} x \log x = \lim_{x \rightarrow \infty} \frac{1 }{ x } \log \frac{1 }{ x } = 0limx→0+​xlogx=limx→∞​x1​logx1​=0.

Properties

The entropy is oblivious to the specific values XXX takes and is only sensitive to the probabilities with which XXX takes these values.
Further, the entropy of a random variable may not reflect the whether a random variable is concentrated at its mean or not.
