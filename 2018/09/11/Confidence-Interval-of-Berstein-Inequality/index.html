Confidence Interval of Berstein Inequality

Posted on 2018-09-11 Edited on 2019-12-02

(Berstein inequality).
Let X1,...,XnX_1, ..., X_nX1​,...,Xn​ be random variables with range [0,1][0, 1][0,1] and

∑t=1nVar[Xt∣Xt−1,...,X1]=σ2\sum_{t = 1}^n Var[X_t \mid X_{t - 1}, ..., X_1] = \sigma^2 t=1∑n​Var[Xt​∣Xt−1​,...,X1​]=σ2

Let Sn=X1+...+XnS_n = X_1 + ... + X_nSn​=X1​+...+Xn​. Then for all a≥0a \ge 0a≥0

Pr⁡[Sn≥E[Sn]+an]≤exp⁡(−n2a2/2σ2+na/2)\Pr[S_n \ge E[S_n] + an] \le \exp\left(- \frac{n^2 a^2 / 2}{\sigma^2 + n a / 2} \right) Pr[Sn​≥E[Sn​]+an]≤exp(−σ2+na/2n2a2/2​)

Suppose we would like to calculate the confidence interval with probability δ\deltaδ, then we need to solve the following equation:

exp⁡(−n2a2/2σ2+na/2)=δ\exp\left( -\frac{n^2 a^2 / 2}{\sigma^2 + n a / 2} \right) = \delta exp(−σ2+na/2n2a2/2​)=δ

After transformation, we get

n2a2−naln⁡1δ−2σ2ln⁡1δ=0n^2 a^2 - n a \ln \frac{1}{\delta} - 2 \sigma^2 \ln \frac{1}{\delta} = 0 n2a2−nalnδ1​−2σ2lnδ1​=0

a=nln⁡1δ+n2ln⁡21δ+8σ2n2ln⁡1δ2n2=ln⁡1δ+ln⁡21δ+8σ2ln⁡1δ2na = \frac{n \ln \frac{1}{\delta} + \sqrt{n^2 \ln^2 \frac{1}{\delta} + 8 \sigma^2 n^2 \ln \frac{1}{\delta}} }{2n^2 } = \frac{\ln \frac{1}{\delta} + \sqrt{\ln^2 \frac{1}{\delta} + 8 \sigma^2 \ln \frac{1}{\delta}} }{2n } a=2n2nlnδ1​+n2ln2δ1​+8σ2n2lnδ1​​​=2nlnδ1​+ln2δ1​+8σ2lnδ1​​​
