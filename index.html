Azuma-Hoeffding Inequality

Posted on 2021-06-02 Edited on 2021-06-03

Azuma-Hoeffding Inequality

Let \(X_0, \ldots, X_n\) be a martingale such that \[ A_k \le X_k - X_{k - 1} \le B_k \] for some random variables \(A_k\) and \(B_k\) that may be functions of \(X_0, X_1, \ldots X_{k - 1}\). If \(B_k - A_k \le c_k\) for some constants \(c_k\), then, for all \(t \ge 0\) and any \(\eta > 0\), \[ \Pr[ | X_t - X_0 | \ge \eta ] \le 2 \exp \left( - \frac{ 2 \eta^2 }{ \sum_{ k \in [t] } c_k^2 }\right) \]

Hoeffding's Lemma

Let \(X\) be any real-valued random variable with mean \(0\), such that \({\displaystyle a\leq X\leq b}\) almost surely, i.e. with probability one. Then, for all \({\displaystyle \lambda \in \mathbb {R} }\), \[ {\displaystyle \mathbb {E} \left[ e^{\lambda X} \right] \le \exp {\Big (} {\frac{\lambda^{2}(b-a)^{2}} {8}} {\Big ) }. } \]

Proof. We prove a version with a weaker factor of \(\frac{1}{2}\) but simpler analysis. The proof for the stronger version is included in the appendix.

First, consider a special case of Rademacher random variable \(\sigma\) that equals \(-1\) and \(1\) with equal probability. For all \(\lambda \in \mathbb{R}\), \[ \begin{aligned} \mathbb{E} [ e^{ \lambda \sigma} ] &= \frac{ e^{ - \lambda } + e^{ \lambda } }{ 2 } &= \sum_{ i \ge 0 } \frac{ \lambda^{ 2i } }{ (2i)! } &\le \sum_{ i \ge 0 } \frac{ ( \lambda^2 / 2)^i }{ i ! } &= e^{ \lambda^2 / 2}. \end{aligned} \]

Next, let \(Y\) be identical copy of \(X\). By Jensen's inequality, \[ \begin{aligned} \mathbb{E} [ e^{ \lambda X }] &= \mathbb{E} [ e^{ \lambda ( X - \mathbb{E}[ X ] ) }] &= \mathbb{E}_{X} [ e^{ \lambda ( X - \mathbb{E}_{Y} [ Y ] ) }] &\le \mathbb{E}_{ X, Y } [ e^{ \lambda ( X - Y ) }] \end{aligned} \]

Since \(X\) and \(Y\) has identical distribution, the random variable \(\sigma( X - Y)\) has the same distribution as \(X - Y\). Hence, \[ \mathbb{E}_{ X, Y } [ e^{ \lambda ( X - Y ) }] = \mathbb{E}_{ X, Y } \Big[ \mathbb{E}_\sigma [ e^{ \lambda ( X - Y ) \sigma } \mid X, Y] \Big]. \]

Finally, by previous argument, \[ \begin{aligned} \mathbb{E}_{\sigma } [ e^{ \lambda ( X - Y ) \sigma } \mid X, Y ] \le e^{ \lambda^2 ( X - Y )^2 / 2 } \le e^{ \lambda^2 (b - a)^2 / 2 }. \end{aligned} \]

It concludes that \(\mathbb{E}_{ X, Y } [ e^{ \lambda ( X - Y ) }] \le e^{ \lambda^2 (b - a)^2 / 2 }\).

\(\square\)

Proof of the Theorem

Define \(Z_k = X_k - X_{k - 1}\) for each \(k \in [n]\). Since \(X_1, \ldots, X_n\) is a martingale, \[ \mathbb{E}[ Z_k \mid X_0, \ldots, X_{k - 1} ] = 0. \]

Via Hoeffding's Lemma, for each \(k \in [n]\), and for all \(\lambda \in \mathbb{R}\), it holds that \[ \mathbb{E} \Big[ e^{\lambda Z_t } \Big] \le e^{ \lambda^2 c_k^2 / 8 }. \]

It follows that \[ \begin{aligned} \mathbb{E} \Big[ e^{ \lambda \sum_{k \in [t]} Z_k } \Big] &=\mathbb{E} \Big[ \mathbb{E} \Big[ e^{ \lambda \sum_{k \in [t] } Z_k } \mid Z_1, \ldots, Z_{t - 1} \Big] \Big] \\ &=\mathbb{E} \Big[ \mathbb{E} \Big[ e^{\lambda Z_t } \Big] \cdot \mathbb{E} \Big[ e^{ \lambda \sum_{k \in [t - 1] } Z_k } \mid Z_1, \ldots, Z_{t - 1} \Big] \Big] \\ &\le e^{ \lambda^2 c_t^2 / 8 } \cdot \mathbb{E} \Big[ \mathbb{E} \Big[ e^{ \lambda \sum_{k \in [t - 1] } Z_k } \mid Z_1, \ldots, Z_{t - 1} \Big] \Big] \end{aligned} \]

By induction, it holds that \[ \mathbb{E} \Big[ e^{ \lambda \sum_{k \in [t]} Z_k } \Big] \le e^{ \lambda^2 \sum_{k \in [t] } c_k^2 / 8 }. \]

For all \(\lambda > 0\), via Markov's inequality, \[ \begin{aligned} \Pr \Big[ \sum_{k \in [t] } Z_k \ge \eta \Big] &= \Pr \Big[ e^{ \lambda \sum_{k \in [t] } Z_k } \ge e^{ \lambda \eta } \Big] \\ &\le \frac{ \mathbb{E} \Big[ e^{ \lambda \sum_{k \in [t]} Z_k } \Big] }{ \mathbb{E}[ e^{ \lambda \eta } ] } \\ &\le e^{ \lambda^2 \sum_{k \in [t] } c_k^2 / 8 - \lambda \eta }. \end{aligned} \]

When \(\lambda = \frac{4 \eta}{ \sum_{k \in [t] } c_k^2 }\), the probability is minimized to \[ \exp \Big\{ \frac{2 \eta^2 }{ \sum_{k \in [t] } c_k^2 } - \frac{4 \eta^2 }{ \sum_{k \in [t] } c_k^2 } \Big\} = \exp \Big\{ - \frac{2 \eta^2 }{ \sum_{k \in [t] } c_k^2 } \Big\}. \]

\(\square\)

Appendix

Proof of Hoeffding's Lemma.

Suppose that \(b \neq a\) (otherwise, \(X\) is a constant). Define

  1. \(\theta = - \frac{a}{b - a} = 1 - \frac{b}{b - a} \in (0, 1)\);
  2. \(c = (b - a) > 0\).

By Jensen's inequality, \[ \begin{aligned} \mathbb{E} [ e^{\lambda X } ] &= \mathbb{E} \Big[ \exp \Big( \frac{b - X}{ b - a} \cdot \lambda a + \frac{X - a}{b - a} \cdot \lambda b \Big) \Big] \\ &\le \mathbb{E} \Big[ \frac{b - X}{ b - a} \cdot \exp \Big( \lambda a \Big) + \frac{X - a}{b - a} \cdot \exp \Big( \lambda b \Big) \Big] \\ &= (1 - \theta) \cdot \exp \Big( \lambda a \Big) + \theta \cdot \exp \Big( \lambda b \Big) \\ &= \Big( 1 - \theta + \theta e^{\lambda (b - a)} \Big) \exp \Big( \lambda a \Big) \\ &= \Big( 1 - \theta + \theta e^{ (\lambda c) } \Big) e^ { - \theta (\lambda c) } \end{aligned} \]

Let \(f(z) \doteq - \theta z + \ln \Big( 1 - \theta + \theta e^{ z} \Big)\). The function is well-defined, as \(1 - \theta + \theta e^{ z} > 0\) for \(z \in \mathbb{R}\). Observe that \[ \exp \Big \{ f(\lambda c) \Big \} = \Big( 1 - \theta + \theta e^{\lambda c } \Big) \exp \Big( - \lambda \theta c \Big). \]

Further, \[ f'(z) = - \theta + \frac{ \theta e^{ z} }{ 1 - \theta + \theta e^{ z} }, \]

and \[ f''(z) = \frac{ \theta e^{ z} ( 1 - \theta + \theta e^{ z} ) - ( \theta e^z )^2 }{ ( 1 - \theta + \theta e^{ z} )^2 } = \frac{ ( 1 - \theta ) \theta e^{ z} }{ ( 1 - \theta + \theta e^{ z} )^2 } \le \frac{1}{4}. \]

Finally, by Taylor's expansion, there exists some \(z^* \in [0, \lambda c]\), such that \[ \begin{aligned} f(\lambda c) &= f(0) + f' (0) \cdot (\lambda c) + \frac{ f''(z^*) }{ 2 } (\lambda c)^2 \\ &= \frac{ f''(z^*) }{ 2 } (\lambda c)^2 \\ &\le \frac{1}{8 } (\lambda c)^2 \\ &= \frac{ \lambda^2 (b - a)^2 }{8 }. \end{aligned} \]

The picture above shows that the bound might not be tight.

\(\square\)

AMS Sketch For Inner Product Estimation

Posted on 2021-05-21 Edited on 2021-05-22

Problem

Let \(\vec x = (x_1, ..., x_n)\), \(\vec y = (y_1, ..., y_n) \in \mathbb{R}^n\). Given \(\delta \in (0, 1)\), construct a data structure with space usage \(O \left( \frac{1}{\epsilon^2} \log \frac{1}{\delta} \right)\), such that with failure probability at most \(\delta \in (0, 1)\),

  • it returns an estimate of the inner product \(\left< \vec x, \vec y \right>\) with error \(O( \epsilon \cdot \Vert \vec x \Vert_2 \cdot \Vert \vec y \Vert_2 )\).

Remark: If we have \(\vec x = \vec y\), we get an estimator of \(\Vert \vec x \Vert_2^2\) with \(\epsilon\)-relative error.

Prototype

Let \(\vec \sigma = (\sigma_1, ..., \sigma_n) \in \{ - 1, 1 \}^n\) a sequence of \(n\) independent Rademachers.

Define \[ X \doteq \left< \vec x, \vec \sigma \right>, \qquad Y \doteq \left< \vec y, \vec \sigma \right>. \]

Theorem. \(X \cdot Y\) is an unbiased estimator of \(\left< \vec x, \vec y \right>\).

Proof. Observe \(\forall i, j \in [n]\), if entries of \(\sigma\) are pairwise independent, then \[ \mathbb{E} [ \sigma_i \sigma_j ] = \begin{cases} 1, & i = j \\ 0, & i \neq j \end{cases}. \]

Hence, \[ \begin{aligned} \mathbb{E} [ X \cdot Y ] &= \mathbb{E} \left[ \left( \sum_{i \in [n] } \sigma_i x_i \right) \left( \sum_{j \in [n] } \sigma_j y_j \right) \right] \\ &= \sum_{i, j \in [n] } \mathbb{E} [ \sigma_i \sigma_j ]x_i y_j \\ &= \sum_{i \in [n] } x_i y_i. \end{aligned} \]

\(\square\)

Theorem. \(\mathbb{Var}[ X \cdot Y ] \le \Vert \vec x \Vert_2^2 \cdot \Vert \vec y \Vert_2^2\).

Proof. If entries of \(\sigma\) are four-wise independent, then \[ \begin{aligned} \mathbb{Var} [ X \cdot Y ] &= \mathbb{Var} \left[ \left( \sum_{i \in [n] } \sigma_i x_i \right) \left( \sum_{j \in [n] } \sigma_j y_j \right) \right] \\ &= \mathbb{Var} \left[ \sum_{i \neq j \in [n] } \sigma_i \sigma_j x_i y_j \right] \\ &= \sum_{i \neq j } \mathbb{Var} \left[ \sigma_i \sigma_j x_i y_j \right] + \sum_{i \neq j, k \neq \ell, (i, j) \neq (k, \ell) } \mathbb{Cov} [ \sigma_i \sigma_j x_i y_j, \sigma_k \sigma_\ell x_k y_\ell ]\\ &= \sum_{i \neq j } x_i^2 y_j^2 \\ &\le \Vert \vec x \Vert_2^2 \cdot \Vert \vec y \Vert_2^2 \end{aligned} \]

\(\square\)

If we have \(m\) independent copies of \(X\) and \(Y\), denoted as \(X_1, ..., X_m\), and \(Y_1, ..., Y_m\), then we can compute the average \((1 / m) \cdot \sum_{i \in [m] } X_i \cdot Y_i\), whose variance is bounded by \((1 / m) \cdot \Vert \vec x \Vert_2^2 \cdot \Vert \vec y \Vert_2^2\). If we set \(m = 1 / \epsilon^2\), then it becomes \(\epsilon^2 \cdot \Vert \vec x \Vert_2^2 \cdot \Vert \vec y \Vert_2^2\) and by Chebyshev's inequality \(|(1 / m) \cdot \sum_{i \in [m] } X_i \cdot Y_i - \left< \vec x , \vec y \right> |\) is bounded by \(\epsilon \cdot \Vert \vec x \Vert_2 \cdot \Vert \vec y \Vert_2\) with constant probability.

By Chernoff bound, we can boost the probability to \(1 - \delta\), by repeating the experiment \(O(\log \frac{1}{\delta})\) times and taking the median of the estimation.

Computing \((1 / m) \cdot \sum_{i \in [m] } X_i \cdot Y_i\) takes \(O( 1 / \epsilon^2 )\) time. Repeating this \(O(\log \frac{1}{ \delta})\) times and taking the median results in time complexity \(O( \frac{\log (1 / \delta) }{\epsilon^2} )\).

On the other hand, we construct \(O ( \epsilon^2 \cdot \log \frac{1}{\delta} )\) copies of \(X\), which takes time \[ O ( n \cdot \epsilon^2 \cdot \log \frac{1}{\delta} ). \]

Algorithm

In this section, we show how to reduce the construction time to \[ O ( n \cdot \log \frac{1}{\delta} ). \]

Given \(m \in \mathbb{N}^+\), let \(h : [n] \rightarrow [m]\) be a pairwise independent hash function.

Let \(\vec X = (X_1, ..., X_m), \vec Y = (Y_1, ..., Y_m) \in \mathbb{R}^m\) be random vectors, such that \(\forall t \in [m]\), \[ X_t = \sum_{ i \in [n] : h(i) = t } \sigma_i x_i, \\ Y_t = \sum_{ i \in [n] : h(i) = t } \sigma_i y_i. \]

Theorem. \(\left< \vec X, \vec Y \right>\) is an unbiased estimator of \(\left< \vec x, \vec y \right>\).

Proof.

If entries of \(\sigma\) are pairwise independent, then \[ \begin{aligned} \mathbb{E} \left[ \left< \vec X, \vec Y \right> \right] &= \sum_{ t \in [m] } \mathbb{E}[ X_t Y_t ] \\ &= \sum_{ t \in [m] } \mathbb{E} \left[ \sum_{ i, j \in [n] : h(i) = h(j) = t } \sigma_i \sigma_j x_i y_j \right] \\ &= \sum_{ t \in [m] } \mathbb{E} \left[ \sum_{ i \in [n] } \frac{1}{m} x_i y_i + \sum_{ i \neq j } \frac{1}{m^2} \sigma_i \sigma_j x_i y_j\right] \\ &= \sum_{ i \in [n] } x_i y_i \end{aligned} \]

\(\square\)

Theorem. \(\mathbb{Var} \left[ \left< \vec X, \vec Y \right> \right] \le \frac{1}{m} \cdot \Vert \vec x \Vert_2^2 \cdot \Vert \vec y \Vert_2^2\).

Proof. If entries of \(\sigma\) are four-wise independent, then \[ \begin{aligned} \mathbb{Var} \left[ \left< \vec X, \vec Y \right> \right] &= \mathbb{Var} \left[ \sum_{ t \in [m] } X_t Y_t \right] \\ &= \mathbb{Var} \left[ \sum_{ i, j \in [n] : h(i) = h(j) } \sigma_i \sigma_j x_i y_j \right] \\ &= \mathbb{Var} \left[ \sum_{ i \neq j, h(i) = h(j) } \sigma_i \sigma_j x_i y_j \right] \\ &= \sum_{i \neq j } \mathbb{Var} \left[ 1_{ [ h(i) = h (j) ] } \cdot \sigma_i \sigma_j x_i y_j \right] \\ &\quad+ \sum_{i \neq j, k \neq \ell, (i, j) \neq (k, \ell) } \mathbb{Cov} [ 1_{ [ h(i) = h (j) ] } \cdot \sigma_i \sigma_j x_i y_j, 1_{ [ h(k) = h (\ell ) ] } \cdot \sigma_k \sigma_\ell x_k y_\ell ] \\ &= \sum_{i \neq j } \mathbb{E} \left[ \left( 1_{ [ h(i) = h (j) ] } \cdot \sigma_i \sigma_j x_i y_j \right)^2 \right] \\ &= \sum_{i \neq j } \frac{1}{m} \cdot x_i^2 y_j^2 \\ &\le \frac{1}{m} \cdot \Vert \vec x \Vert_2^2 \cdot \Vert \vec y \Vert_2^2 \end{aligned} \]

\(\square\)

By setting \(m = 1/ \epsilon^2\), repeating the procedure \(O(\log \frac{1}{\delta})\) times and taking the median as our estimation, we can obtained the desired result.

Reference

[1] G. Cormode and K. Yi, Small Summaries for Big Data, 1st ed. Cambridge University Press, 2020.

L0 Sampling

Posted on 2021-04-16 Edited on 2021-05-15

Problem

Let \(x \in \mathbb{R}^n\) be a vector initialized with all zeros. There is a stream \(S\) of updates, each of which is a pair \((t, \Delta) \in [n] \times \mathbb{R}\), causing \(x_t \leftarrow x_t + \Delta\).

Define the support \(\mathbb{Supp}(x)\) to be the set of all non-zero indexes of \(x\): \[ \mathbb{Supp}(x) \doteq \{ i \in [n] : x_i \neq 0 \}. \]

\(\ell_0\) sampling: output an index uniformly at random from the support \(\mathbb{Supp}(x)\).

We assume that at any time, \(\forall t \in [n], x_t \le M\) for some \(M \in \mathbb{R}\).

Naive Solutions

We can uses an array to keep \(x\) explicitly. But this has size \(O( n \cdot \log M)\) bits.

Goal: design an algorithm for \(\ell_0\) sampling with space less than \(O( n \cdot \log M)\) bits.

If there are only insertion updates, i.e., \(\forall (t, \Delta) \in S\), it holds that \(\Delta > 0\), then we can use a hash set to keep the non-zero elements and their frequencies, which has size \(O( |\mathbb{Supp}(x) | \cdot \log M)\). If \(x\) is dense, i.e., \(|\mathbb{Supp}(x)|\) is large, this is still infeasible.

Sampling Game

We discuss one sampling game that illustrates the key ideas used in \(\ell_0\)-sampling.

In this game, we want to sample one integer uniformly at random from \([k]\). Instead of sampling it directly, we choose some \(p \in (0, 1]\) and perform the following two steps:

  1. First, we sample each \(i \in [k]\) uniformly and independent at random with probability \(p\). Denote \(B\) the set of sampled elements.
  2. If \(B \neq \emptyset\), then output one element from \(B\) uniformly at random.

Property. \(\mathbb{E} [ |B| ] = kp\).

Property. \(\Pr[ |B| > 0 ] = (1 - p)^k \le \exp(-kp)\).

Property. By Bernstein's Inequality, \(\forall \delta \in (0, 1)\), with probability at least \(1 - \delta\), we have \[ |B| - \mathbb{E} [ |B| ] \le \sqrt{2 \cdot k \cdot p \cdot (1 - p) \cdot \ln \frac{1}{\delta} } + \frac{2 \ln \frac{1}{\delta} }{3}. \]

Theorem. Each element in \([k]\) is outputted with equal probability \[ \frac{1 - \left(1 - p \right)^k }{ k }. \]

Remark: when \(p = 1\), this reduces to sampling an element uniformly at random from \([k]\).

Proof. The probability that \(B = \emptyset\) is given by \((1 - p)^k\). Conditioned on that \(B \neq \emptyset\), by symmetry, each element is sampled with equal probability.

\(\square\)

Remark: We can also have a proof by algebra. For evert \(i \in [k]\), the probability that it is sampled is given by \[ \begin{aligned} \sum_{t = 1}^k \frac{1}{t} \cdot \Pr [ i \in B \mid |B| = t] \cdot \Pr[ |B| = t] &= \sum_{t = 1}^k \frac{1}{t} \frac{ \binom{k - 1}{t - 1} }{ \binom{k}{t} } \binom{k}{t} p^t (1 - p)^{k - t} \\ &= \sum_{t = 1}^k \frac{1}{t} \binom{k - 1}{t - 1} p^t (1 - p)^{k - t} \\ &= \frac{1}{k} \sum_{t = 1}^k \binom{k}{t} p^t (1 - p)^{k - t} \\ &= \frac{1}{k} \left( 1 - \binom{k}{0} p^0 (1 - p)^{k - 0} \right) \\ &= \frac{1 - \left(1 - p \right)^k }{ k }. \end{aligned} \]

Sampling With Known Support Size

We use the technique developed in the Sampling Game section, to avoid using \(O( | \mathbb{Supp}(x) | \log M)\) bits. We have seen that \(p\) controls the tradeoff between the size of \(B\), and the successful probability of sampling an element (\(B \neq \emptyset\)).

We begin with the assumption that the size of \(\mathbb{Supp}(x)\) is known.

Insertion Only

As before, we consider insertion only model first.

Claim. There is an algorithm such that 1. with probability \(1 - \delta\), it returns an element from \(\mathbb{Supp}(x)\) uniformly at random;
2. with probability \(\delta\), it returns nothing.

Further, the space usage is \(O \left( \ln \frac{1}{\delta} \right)\) with probability at least \(1 - \delta\).

For simplicity, we assume a perfect hash function \(h\) that maps elements in \([n]\) to \(\{0, 1\}\) independently with probability \(p\) (to be determined). For each update \((t, \Delta)\) in the stream, we process it only if \(h(t) = 1\). As we are dealing with insertion only model, we can use a hashset to maintain the elements with hash value 1.

Let \(k = | \mathbb{Supp}(x) |\). As analysed before, with probability \((1 - p)^k\), the algorithm fails. To make the failure probability as small as \(\delta\), we have that \((1 - p)^k = \delta\), i.e., \[ p = 1 - \delta^{ \frac{1}{k} } \le -\ln \delta^{ \frac{1}{k} } = \frac{1}{k} \ln \frac{1}{\delta}. \]

The last inequality holds since the function \(y = - \ln x\) is convex, and \(y = 1 - x\) is the tangent line of \(y = - \ln x\) at the point \((1, 0)\). By previous analysis, with probability at least \(1 - \delta\), the size of \(B\) is bounded by \[ \left( kp + \sqrt{2 k p (1 - p) \ln \frac{1}{\delta} } + \frac{2}{3} \ln \frac{1}{\delta} \right) \in O \left( \ln \frac{1}{\delta} \right). \]

Handling Deletion

When deletions are possible, we can't use a hashset to keep the sampled updates. E.g., suppose we have a sequence \((1, 6), (1, -6), (2, 5), (3, 7), (4, 9)\), and that \(h(1) = h(2) = 1, h(3) = h(4) = 0\). We need a hashset of size 2 to keep both the frequencies of element 1 and element 2. Even through the frequency of element \(1\) is 0 finally, we don't this before we see the entire string.

In general, \(h\) maps \(np\) elements in \([n]\) to \(1\). If we use a hashset, its size can grow to \(O(n p)\).

However, there is data structure that can handle deletions while using less than \(O(np)\) space.

Sparse Recovery. Given an integer \(d\) at the beginning of the stream, if \(\mathbb{Supp}(x) \le d\) when the stream stops, there is a data structure that returns

  1. indices in \(\mathbb{Supp}(x)\);
  2. and their values,

using \(O(d (\log n + \log M) ) \subset O( d \log (nM) )\) bits. It processes each update \((t, \Delta)\) in the stream in \(O(d)\) time and recovers \(x\) in \(O(d^2 polylog(nM))\) time. Its recovery can be wrong if \(\mathbb{Supp}(x) > d\).

It suffices to use such a data structure for the sampled set \(B\) with \(d \in O( \ln \frac{1}{\delta} )\). We have seen that if we sampled each element in \(\mathbb{Supp}(x)\) with probability \(p = 1 - \delta^{1 / k}\), then with probability at least \(1 - \delta\), we obtain a sampled set \(B\) with size at most \(d\). Then we return one element from \(B\) uniformly at random.

Analyzing the probability of sampling an element is more subtle than the previous section. In previous section, each element is under-sampled with probability \(\frac{1 - \left(1 - p \right)^k }{ k }\). Here, conditioned on \(|B|\le d\), each element is sampled equally likely. However, \(|B| > d\), the recovery of sampled elements fails, some element can be returned more frequently and some can be returned less frequently (the recovery behavior is un-defined). As the failure probability is bounded by \(\delta\), each element is returned with probability \[ \left[ \frac{1 - \left(1 - p \right)^k }{ k } \pm \delta \right] \subseteq \left[ 1 - \left( 1 + \frac{ 1 }{ k } \right) \cdot \delta, 1 + \left( 1 - \frac{ 1 }{ k } \right) \cdot \delta \right]. \]

Sampling With UnKnown Support Size

Previously, we want to sample each element in \([n]\) with probability \(1 - \delta^{ \frac{1}{k} } \le \ln \delta^{ \frac{1}{k} } = \frac{1}{k} \ln \frac{1}{\delta}\), where \(k = | \mathbb{Supp}(x) |\). When this information is unavailable, we can guess by trying the values \[ 1 / 2, \quad 1 / 4, \quad 1 / 8, \quad 1 / 16, ... \]

We don't need to guess the value of \(\frac{1}{k} \ln \frac{1}{\delta}\) exactly. In order that the sparse recovery data structure works, any guess within a constant factor suffices.

For simplicity, assume that \(n\) is some power of \(2\). Now, instead of using one sampling set \(B\), we use \(\log n + 2\) of them \(B_1, B_2, B_3, ...\), each associated with a sparse recovery structure with parameter \(d \in \Theta( \ln \frac{1}{\delta} )\). Assume that we have a perfect hash function \(h: [n] \rightarrow [\log n + 2]\) such that it holds independently for every \(t \in [n]\), \[ \begin{aligned} &\Pr[ h(t) = i ] = \frac{1}{2^i}, \forall i \in [\log n + 1], \\ &\Pr[ h(t) = \log n + 2] = \frac{1}{ 2^{\log n + 1} }. \end{aligned} \]

The distribution is well defined, as \(\sum_{i \in [\log n + 1] } 1 / 2^i + 1 / 2^{ \log n + 1} = 1 - 1 / 2^{ \log n + 1} + 1 / 2^{ \log n + 1} = 1\).

If \(k = 1\), some \(B_i\) must recovers \(x\) successfully.

Otherwise, \(\exists i \in [\log n]\), such that \(\frac{1}{2^i} \ge \frac{1}{k} \ln \frac{1}{\delta} > \frac{1}{2^{i + 1} }\). We study the following good events:

Event \(E_1:\) for all \(j \ge i\), it holds that \(|\{ t \in \mathbb{Supp}(x) : h(t) = j \}| \le d\).

Under \(E_1\), the sparse recovery structure associated with \(B_j\) recovers \(B_j\) successfully for all \(j \ge i\).

Event \(E_2:\) \(|\{ t \in \mathbb{Supp}(x) : h(t) \ge i \}| > 0\).

Under \(E_2\), there exists at least one element that is mapped to \(B_j\) for some \(j \ge i\).

Under these two good events, we sample one element as follows:

For \(j \leftarrow \log n + 2\) downto \(1\) do
\(\qquad\) recover \(B_j\);
\(\qquad\) if \(B_j \neq \emptyset\) then
\(\qquad\qquad\) return one element from \(B_j\) uniformly at random.

Analysis the probability of \(E_1\):

For \(j \ge i\), it holds that \(\mathbb{E} [ |B_j| ] = k / 2^j\). Suppose that \(d = 2 \cdot C_d \cdot \ln \frac{1}{\delta}\) for some constant \(C_d\) (to be determined). Since \(\frac{1}{k} \ln \frac{1}{\delta} \ge 1 / 2^{i + 1}\), we have that \[ \frac{d}{\mathbb{E} [ |B_j| ]} \ge \frac{ 2^j }{ k } \cdot 2 \cdot C_d \cdot \ln \frac{1}{\delta} \ge C_d \cdot 2^{j - i}. \]

By Chernoff bound, \[ \begin{aligned} \Pr \left[ |B_j| \ge d \right] &\le \left( \frac{e^{ \left( {d} / { \mathbb{E} [ |B_j| ] } \right) - 1 } }{ \left( {d} / { \mathbb{E} [ |B_j| ] } \right)^{ \left( {d} / { \mathbb{E} [ |B_j| ] } \right) } } \right)^{ \mathbb{E} [ |B_j| ]} \\ &\le \left( \frac{e}{C_d \cdot 2^{j - i} } \right)^d \left( \frac{1}{e} \right)^{ \mathbb{E} [ |B_j| ]}. \end{aligned} \]

By union bound, when \(C_d \ge e^2\), it holds that \[ \begin{aligned} \Pr \left[ \exists j \ge i : |B_j| \ge d \right] &\le \sum_{j \ge i} \left( \frac{e}{C_d \cdot 2^{j - i} } \right)^d \\ &\le \left( \frac{e}{C_d } \right)^d \sum_{j = 0}^\infty \left( \frac{1}{2^{d} } \right)^j \\ &\le \left( \frac{e}{C_d } \right)^d \frac{1}{1 - \frac{1}{2^{d} } } \\ &\le 2 \cdot \delta^{2 e^2}. \end{aligned} \]

\(\square\)

Analysis the probability of \(E_2\):

For each \(t \in \mathbb{Supp}(x)\), we have \[ \Pr[ h(t) < i ] = \left( 1 - \frac{1}{2^i} \right). \]

Therefore, \[ \Pr[ h(t) < i : \forall t \in \mathbb{Supp}(x) ] = \left( 1 - \frac{1}{2^i} \right)^k \le \left( 1 - \frac{1}{k} \ln \frac{1}{\delta} \right)^k \le \delta. \]

\(\square\)
