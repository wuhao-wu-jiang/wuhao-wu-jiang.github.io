Counting Triangle

Posted on 2020-03-01 Edited on 2022-02-04

Problem

  1. Let G=<V,E>G = \left< V, E \right>G=⟨V,E⟩ be an undirected graph with ∣V∣=n|V| = n∣V∣=n vertices and ∣E∣=m|E| = m∣E∣=m edges.

  2. The number of triangle in GGG is denoted as Δ\DeltaΔ.

  3. For each v∈Vv \in Vv∈V, let N(v)≐{u:(v,u)∈E}N(v) \doteq \{ u : (v, u) \in E \}N(v)≐{u:(v,u)∈E} be the set of neighbors of vvv.

  4. The degree of vvv is dv≐∣N(v)∣d_v \doteq | N(v) |dv​≐∣N(v)∣.

  5. Define λ(v)\lambda(v)λ(v) be the number of triangles whose vertices contains vvv.

  6. Similarly, for each e=(u,v)∈Ee = (u, v) \in Ee=(u,v)∈E, define λ(e)\lambda(e)λ(e) the number of triangles whose edges contains eee.

Exact Algorithm

Theorem. The number of triangles in GGG can be computed in O(m1.5)O \big ( m^{1.5} \big)O(m1.5) time.

Proof.

WLOG, we assume that the vertices in VVV are labelled from 111 to nnn in increasing order according to their degrees, such that

d1≤d2≤...≤dn. d_1 \le d_2 \le ... \le d_n. d1​≤d2​≤...≤dn​.

Each triangle in GGG can be represented by a unique triple (u,v,w)∈[n]3(u, v, w) \in [n]^3(u,v,w)∈[n]3 such that du≤dv≤dwd_u \le d_v \le d_wdu​≤dv​≤dw​.
To count the number of such triples, we can execute the following algorithm.

For each u∈[n]u \in [n]u∈[n] do
\quad For v∈N(u)v \in N(u)v∈N(u) of uuu s.t. dv≥dud_v \ge d_udv​≥du​ do
\quad\quad For w∈N(u)w \in N(u)w∈N(u) of uuu s.t. dw≥dvd_w \ge d_vdw​≥dv​ do
\qquad\quad Check whether (u,v,w)(u, v, w)(u,v,w) is a triangle.

It is left to prove that the algorithm has complexity O(m1.5)O(m^{1.5})O(m1.5).

Lemma. For each (u,v)∈E(u, v) \in E(u,v)∈E, such that du≤dvd_u \le d_vdu​≤dv​, the number of w∈N(u)w \in N(u)w∈N(u) such that dw≥dvd_w \ge d_vdw​≥dv​ is bounded by O(m)O(\sqrt{m})O(m​).

  • The claim is trivial if du≤md_u \le \sqrt{m}du​≤m​.

  • Otherwise, suppose that du≥md_u \ge \sqrt{m}du​≥m​.
    But the number of w∈Vw \in Vw∈V such that dw≥du≥md_w \ge d_u \ge \sqrt{m}dw​≥du​≥m​ is bounded by O(m/m)=O(m)O( m / \sqrt{ m }) = O(\sqrt{ m })O(m/m​)=O(m​).

□\square□

Wedge Sampling

If a wedge is closed, i.e., (v,w)∈E(v, w) \in E(v,w)∈E, then it constitutes a triangle.
Note that each triangle contains 333 wedges.
Denote WWW the number of wedges in GGG.
