Problem

Let d∈N+d \in \N^+d∈N+ be a positive integer and f=(0,…,0)∈Rdf = (0, \ldots, 0) \in \R^df=(0,…,0)∈Rd be an all-zero vector.

  1. f←(0,…,0)∈Rdf \leftarrow (0, \ldots, 0) \in \R^df←(0,…,0)∈Rd;

  3. \qquad f[x]←f[x]+wf[x] \leftarrow f[x] + wf[x]←f[x]+w;

We are interested in answering the following query.

Rank Query
Input： x∈[d]x \in [d]x∈[d]
Returen: r[x]←∑i=1x−1f[i]r[x] \leftarrow \sum_{i = 1}^{x - 1} f[i]r[x]←∑i=1x−1​f[i].

We formally define the problem of approximate rank estimation.

Problem. Let ϵ\epsilonϵ be an error parameter.
Design an algorithm, such that
for a rank query with a given element x∈[d]x \in [d]x∈[d],
the summary will return an approximate rank r^[x]\hat r [x]r^[x], such that

r[x]−ϵ⋅W≤r^[x]≤r[x]+ϵ⋅W. r[x] − \epsilon \cdot W \le \hat r [x] \le r[x] + \epsilon \cdot W. r[x]−ϵ⋅W≤r^[x]≤r[x]+ϵ⋅W.

For each node vvv, denote L[v]L[v]L[v] and R[v]R[v]R[v] its left and right children respectively.

QQQ-Digest

In this section, we show how to reduce the space usage to O(1ϵlog⁡d)O(\frac{1}{\epsilon} \log d)O(ϵ1​logd).

In the previous section, when the algorithm perform a rank query for x∈[d]x \in [d]x∈[d], it walks down a path (v1,…,vlog⁡d)(v_1, \ldots, v_{\log d} )(v1​,…,vlogd​), where v1v_1v1​ is the root of the tree, and vlog⁡dv_{\log d}vlogd​ is the leaf node that contains xxx.

There are log⁡d\log dlogd nodes in the path in total.
If each node introduces an error of O(ϵW/log⁡d)O(\epsilon W / \log d)O(ϵW/logd) to the estimation of r[x]r[x]r[x], then the final estimation has error bounded by O(ϵW)O(\epsilon W)O(ϵW).

This motivates to associate each node vvv in the tree with an error counter c[v]c[v]c[v], which is interpreted as the maximum error it can introduce to the rank estimation to a given element xxx.
It is initialized with 000 and is guaranteed to be bounded by some threshold CCC (except the leaf node).

ϵ⋅W2log⁡d≤C≤ϵ⋅Wlog⁡d. \frac{\epsilon \cdot W}{2 \log d} \le C \le \frac{\epsilon \cdot W}{\log d}. 2logdϵ⋅W​≤C≤logdϵ⋅W​.

Space Usage

A node is called “saturated” if c[v]=Cc[v] = Cc[v]=C.
By the invariant of CCC, there are at most W/C=O((1/ϵ)log⁡d)W / C = O((1 / \epsilon ) \log d)W/C=O((1/ϵ)logd) “saturated” nodes.
By the algorithm, each node vvv in the tree with nonzero c[v]c[v]c[v], either “saturated”, or has a “saturated” parent.
However, each “saturated” can have at most two children.
Therefore, the total number of nodes with nonzero c[v]c[v]c[v] is bounded by O((1/ϵ)log⁡d)O((1 / \epsilon ) \log d)O((1/ϵ)logd).

Amplificatoin by Sampling

Posted on 2021-07-05 Edited on 2021-07-06

Definition. Two distribution \(\mu\) and \(\mu'\) are \((\epsilon, \delta)\) close, denoted as \(\mu \simeq_{(\epsilon, \delta) } \mu'\), if for all measurable set \(A\), we have \[ \begin{aligned} \mu(A) &\le e^\epsilon \mu'(A) + \delta, \\ \mu'(A) &\le e^\epsilon \mu(A) + \delta. \end{aligned} \]

Theorem. Let \(q \in (0, 1)\) and let \(\mu\) and \(\mu'\) be two distributions such that \(\mu \simeq_{(\epsilon, \delta)} \mu'\). For \(\nu = (1 - q) \mu + q \mu'\), it holds that \[ \nu \simeq_{ (\epsilon', q \delta) } \mu, \] where \(\epsilon' = \ln ( q (e^\epsilon - 1) + 1 )\).

Proof. For each measurable set \(A\) \[ \begin{aligned} \nu(A) &= (1 - q) \mu(A) + q \mu' (A) \\ &\le (1 - q) \mu(A) + q (e^\epsilon \mu (A) + \delta) \\ &= ( q (e^\epsilon - 1) + 1 ) \mu(A) + q \delta. \end{aligned} \]

Next we prove the other direction. Denote \(t = e^{\epsilon} - 1\). Then \[ e^{\epsilon'} = 1 + qt, \]

and \[ \begin{aligned} \nu(A) &= (1 - q) \mu(A) + q \mu' (A) \\ &\ge (1 - q) \mu(A) + q e^{-\epsilon} (\mu (A) - \delta) \\ &= (1 - q) \mu(A) + \frac{q}{1 + t} (\mu (A) - \delta) \\ &= \mu(A) - \frac{qt}{1 + t} \mu (A) - \frac{q}{1 + t} \delta \\ &= \left( 1 - \frac{qt}{1 + t} \right) \mu (A) - \frac{1}{1 + t} q \delta \\ &\ge \left( 1 - \frac{qt}{1 + qt} \right) \mu (A) - \frac{1}{1 + qt} q \delta \\ &= \frac{1}{1 + qt} \left( \mu (A) - q \delta \right). \end{aligned} \]

\(\square\)
