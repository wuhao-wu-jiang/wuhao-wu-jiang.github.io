Beyond Chebyshev's Inequality

Posted on 2021-02-12 Edited on 2021-02-15

We consider the Count Sketch with single array.

  3. h:[d]→[m]h: [d] \rightarrow [m]h:[d]→[m] a hash function that assigns each value in [d][d][d] an integer uniformly at random from [m][m][m].
  4. g:[d]→{−1,1}g: [d] \rightarrow \{ - 1, 1 \}g:[d]→{−1,1} a hash function that assigns each value in [d][d][d] a random sign.

Algorithm

For each element s∈Ss \in Ss∈S, Count-Sketch updates AAA as follows:

\qquad A[h(s)]←A[h(s)]+g[s]A[ h(s) ] \leftarrow A[ h(s) ] + g[s]A[h(s)]←A[h(s)]+g[s]

After processing elements in SSS, Count-Sketch answer query for an element’s frequency as follows:

Query (vvv):
\qquad return f^[v]≐g[v]⋅A[h(v)]\hat f[v] \doteq g[v] \cdot A[ h(v) ]f^​[v]≐g[v]⋅A[h(v)]
