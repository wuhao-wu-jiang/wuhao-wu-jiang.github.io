Problem

In general, a sketch is a concise representation of fff. Suppose that we limit our space usage to kkk words (i.e., ∥f^∥0=k\Vert \hat f \Vert_0 = k∥f^​∥0​=k). Then the sketch f^\hat ff^​ that minimizes the ℓ2\ell_2ℓ2​ error

In practice, however, SSS is not given offline. As we don’t know HHH, we can only compute an f^\hat ff^​ with kkk words that is as good as possible. The question is, how far ∥f−f^∥2\Vert f - \hat f \Vert_2∥f−f^​∥2​ can be from the minimum possible value, namely err2k(f)err_2^k(f)err2k​(f)?

Algorithm

One possible solution, proposed by Cormode and Muthukrishnan [1], is to construct f^\hat ff^​ vis Count-Sketch. In particular,

  1. It uses d=O(log⁡n)d = O(\log n)d=O(logn) hash tables.
  2. Each hash table has size w=3kϵ2w = \frac{3k}{\epsilon^2}w=ϵ23k​.
