Lebesgue Integral, Revisited.

Posted on 2021-06-19 Edited on 2022-02-07

Let (Ω,F,μ)(\Omega, \mathcal{F}, \mu)(Ω,F,μ) be measure space.
Further, suppose that μ\muμ is σ\sigmaσ-finite, i.e., there is a countable sequence {Sk}\{ S_k \}{Sk​} in F\mathcal{F}F, such that

The definition of integral of measurable functions consists of two steps.

Nonnegative Function

Bending the notation slightly, we say that fff is nonnegative, denoted as f≥0f \ge 0f≥0, if for all ω∈Ω\omega \in \Omegaω∈Ω, f(ω)≥0f(\omega) \ge 0f(ω)≥0.

Definition ♢\diamondsuit♢. Let f≥0f \ge 0f≥0 be measurable. Define

∫f≐sup⁡{∑i∈[n]aiμ(Ai):∑i∈[n]ai1Ai is simple,0≤∑i∈[n]ai1Ai≤f}. \int f \doteq \sup \left\{ \sum_{i \in [n] } a_i \mu(A_i): \begin{array}{l} \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \text{ is simple}, \\ 0 \le \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \le f \end{array} \right\}. ∫f≐sup⎩⎪⎨⎪⎧​i∈[n]∑​ai​μ(Ai​):∑i∈[n]​ai​1Ai​​ is simple,0≤∑i∈[n]​ai​1Ai​​≤f​⎭⎪⎬⎪⎫​.

For each nonnegative simple function φ=∑i∈[n]ai1Ai\varphi = \sum_{i \in [ n ] } a_i \mathbf{1}_{A_i}φ=∑i∈[n]​ai​1Ai​​,

General Function

A general function f:Ω→Rf : \Omega \rightarrow \mathbb{R}f:Ω→R is said to be integrable if ∫∣f∣<∞\int |f| < \infty∫∣f∣<∞. Let f+=max⁡{f,0}f_+ = \max \{ f, 0\}f+​=max{f,0} and f−=max⁡{−f,0}f_- = \max \{ - f, 0 \}f−​=max{−f,0}. Then f=f+−f−f = f_+ - f_-f=f+​−f−​.

Definition ♣\clubsuit♣. Let fff be measurable. Define

∫f≐∫f+−∫f−. \int f \doteq \int f_+ - \int f_-. ∫f≐∫f+​−∫f−​.

Integration over a subset

Let f:Ω→Rf: \Omega \rightarrow \mathbb{R}f:Ω→R be measurable and E∈FE \in \mathcal{F}E∈F. Then

∫Ef≐∫f⋅1E. \int_E f \doteq \int f \cdot \mathbf{1}_{E}. ∫E​f≐∫f⋅1E​.

Remark: note that if fff is measurable and 1E\mathbf{1}_{E}1E​ is measurable, then f⋅1Ef \cdot \mathbf{1}_{E}f⋅1E​ is measurable.

Basic Properties

  1. Monotonicity. If f≥gf \ge gf≥g then ∫f≤∫g\int f \le \int g∫f≤∫g.

  2. Linearity. The integral is linear on the vector space of integral functions; in particular, ∀α,β∈R\forall \alpha, \beta \in \mathbb{R}∀α,β∈R, ∫α⋅f+β⋅g=α⋅∫f+β⋅∫g\int \alpha \cdot f + \beta \cdot g = \alpha \cdot \int f + \beta \cdot \int g∫α⋅f+β⋅g=α⋅∫f+β⋅∫g.

Monotonicity follows directly from the definition.
Establishing linearity is more fiddly and we leave it to the end of the article.
We proceed by assuming these two properties of integration.

Properties for Limits

Suppose that fn→ff_n \rightarrow ffn​→f a.e., when can we argue that

∫fn→∫f, \int f_n \rightarrow \int f, ∫fn​→∫f,

i.e., when the order of the limiting and the integration operations can be exchanged?

Intuitively, there are two kinds of convergence: the monotone ones and the bounded ones.

Monotone Convergence

Theorem (Monotone Convergence). If fn≥0f_n \ge 0fn​≥0 and fn↑ff_n \uparrow ffn​↑f, then

∫f=lim⁡n≥1∫fn. \int f = \lim_{n \ge 1} \int f_n. ∫f=n≥1lim​∫fn​.

Corollary 1. The integral is σ\sigmaσ-additive for nonnegative functions.

Corollary 2. If fn≥gf_n \ge gfn​≥g, where ggg is integrable, and fn↑ff_n \uparrow ffn​↑f, then

∫f=lim⁡n≥1∫fn. \int f = \lim_{n \ge 1} \int f_n. ∫f=n≥1lim​∫fn​.

Proof. As f≥fnf \ge f_nf≥fn​, by monotonicity of integration,
∫f≥∫fn\int f \ge \int f_n∫f≥∫fn​.
It follows that

∫f≥lim⁡n∫fn. \int f \ge \lim_{n} \int f_n. ∫f≥nlim​∫fn​.

It is left to prove the reverse direction of the inequality.
Let φ≐∑i∈[k]ai1Ai\varphi \doteq \sum_{i \in [k] } a_i \mathbf{1}_{ A_i }φ≐∑i∈[k]​ai​1Ai​​ be a simple function such that φ≤f\varphi \le fφ≤f.

  1. c⋅φ⋅1En≤fnc \cdot \varphi \cdot 1_{E_n} \le f_nc⋅φ⋅1En​​≤fn​ is also a simple function.

∫c⋅φ⋅1En=∑i∈[k]c⋅ai⋅μ(Ai∩En)≤∫fn. \int c \cdot \varphi \cdot 1_{E_n} = \sum_{i \in [k] } c \cdot a_i \cdot \mu \big( A_i \cap E_n \big) \le \int f_n. ∫c⋅φ⋅1En​​=i∈[k]∑​c⋅ai​⋅μ(Ai​∩En​)≤∫fn​.

Linearity

It is difficult to prove linearity by the definitions of ♢\diamondsuit♢ or ♡\heartsuit♡. Hence, we introduce some equivalent definitions for integration, starting from simple functions.
These definitions also provide deeper intuitions.

Simple Functions

First, we prove that definition ♢\diamondsuit♢ has an equivalent one for nonnegative simple functions.

Theorem. Let φ≥0\varphi \ge 0φ≥0 be a simple function and ∑j∈[m]bj1Bj\sum_{j \in [m] } b_j \mathbf{1}_{B_j}∑j∈[m]​bj​1Bj​​ be one of its possible representations. Then it holds that

∫φ≐sup⁡{∑i∈[n]aiμ(Ai):∑i∈[n]ai1Ai is simple,0≤∑i∈[n]ai1Ai≤φ}=∑j∈[m]bjμ(Bj). \small \int \varphi \doteq \sup \left\{ \sum_{i \in [n] } a_i \mu(A_i): \begin{array}{l} \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \text{ is simple}, \\ 0 \le \sum_{i \in [n] } a_i \mathbf{1}_{A_i} \le \varphi \end{array} \right\} = \sum_{j \in [m] } b_j \mu (B_j). ∫φ≐sup{i∈[n]∑​ai​μ(Ai​):∑i∈[n]​ai​1Ai​​ is simple,0≤∑i∈[n]​ai​1Ai​​≤φ​}=j∈[m]∑​bj​μ(Bj​).

Proof. By the definition of φ\varphiφ, clearly it holds that

∑j∈[m]bjμ(Bj)≤∫φ. \sum_{j \in [m] } b_j \mu (B_j) \le \int \varphi. j∈[m]∑​bj​μ(Bj​)≤∫φ.
