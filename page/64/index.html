Sampling Theorem

In 1994, Karger gave the following result:

The limitation is that either ϵ\epsilonϵ or p depends on c. When c is little, we could not reduce many edges.

Improvement:
In 1996, Benczur and Karger improved previous sampling method such that the dependence on c is removed.

New Result:

Given a graph G and a parameter ϵ\epsilonϵ, there is a new graph G’ such that

  1. G’ has O(nln⁡nϵ2)O({n \ln n \over \epsilon^2})O(ϵ2nlnn​) edges.

The intuition is that in the dense parts of the graph, each edge can be sample with smaller probability in order to preserve cut value while edge should be sample with higher probability where the graph is sparse.

K-Connected Spanning Subgraph

Posted on 2015-11-23 Edited on 2015-12-01

K-Connected Spanning Subgraph

Background

Consider that you are given a transportation network, which is a graph. Now you are asked to find out a sub-graph that the transportation capacity between any two sites in the sub-graph is above some threshold k.

This sub-graph could be viewed as kind of a backbone. Its importance is as follows: even if some lines in the transportation network break down, so long as the lines of this sub-graph operates, we can guarantee that the network is able to transport k capacity cargo between any two sites.

If we examine this problem deeply, we may ask the following questions:

  1. Does such a sub-graph exist?
  2. If such a sub-graph exist, how to find one with the minimum number of edges?

Solution

Existence could be answered right now. If the minimum cut of the graph is no less than k, then we could find such a sub-graph. A trivial case is the graph itself. But the second one is difficult. It is believed that this is a NP hard problem.

Approximation algorithms are proposed. One of them was given by Hiroshi Nagamochi and Toshihide ibaraki [A Linear-Time Algorithm for Finding a Sparse k-Connected Spanning Subgraph of a k-Connected Graph] in 1992.

We firstly formalize the notions. To simplify the problem, we focus on graphs that are undirected and consist of no multiple edges and self loops.

k-Connected Graph: A graph is k-Connected if the minimum cut of the graph is no less than k.
