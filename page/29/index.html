Gaussian Distribution

We show how Gaussian distribution (a.k.a. normal distribution) is constructed and list some of its basic properties.

Suppose we would like to have a distribution in R\mathbb{R}R, such that

  1. It is centered at the origin.
  2. Its density function decreases exponentially with respect to the square distance to the origin.

Remark: if we replace 2 with “a density function decreases exponentially with the distance to the origin”, we obtain Laplace distribution.

The density function p(x)p(x)p(x) should be inversely and exponentially proportional to x2x^2x2:

p(x)∝exp⁡(−x2) p(x) \propto \exp(-x^2 ) p(x)∝exp(−x2)

Denote MMM the value of

M≐∫−∞∞exp⁡(−x2) dx. M \doteq \int_{-\infty}^\infty \exp(-x^2) \ dx. M≐∫−∞∞​exp(−x2) dx.

It is easy to check that MMM is finite.
Then the close form of p(x)p(x)p(x) is

p(x)=1Mexp⁡(−x2). p(x) = \frac{1}{M} \exp(- x^2). p(x)=M1​exp(−x2).

It is easy to verity that p(x)p(x)p(x) is a valid density function: ∫−∞∞p(x) dx=1\int_{-\infty}^\infty p(x) \ dx = 1∫−∞∞​p(x) dx=1.

Closed Form

Theorem.
p(x)=1πexp⁡(−x2) p(x) = \frac{1}{ \sqrt \pi} \exp( - x^2 ) p(x)=π​1​exp(−x2)
is a density function.

In general, it is not hard to construct a distribution. Let h(x)h(x)h(x) be any integrable function on a domain D\mathcal{D}D such that

M=∫Dh(x) dx<∞ M = \int_\mathcal{D} h(x) \ dx < \infty M=∫D​h(x) dx<∞

Then we can view h(x)h(x)h(x) as almost an density function of some distribution.
The only obstacle is that MMM might not be 111.
However this is easy to fix.
We scale the value of h(x)h(x)h(x) by a constant factor at each point by setting

p(x)=1Mh(x). p(x) = \frac{1}{M} h(x). p(x)=M1​h(x).

In this case, we don’t even need to know the exact value of MMM.
We only need to guarantee that

The value of MMM exists and is finite.

Then we can safely rely on MMM as a normalized. For example, we know that

∫0∞x0.2e−x dx≤∫01e−x dx+∫1∞xe−x dx<∞. \int_0^\infty x^{0.2} e^{-x} \ dx \le \int_0^1 e^{-x} \ dx + \int_1^\infty x e^{-x} \ dx < \infty. ∫0∞​x0.2e−x dx≤∫01​e−x dx+∫1∞​xe−x dx<∞.

We can construct a density function as discussed above.
It turns out that this is a special case of Gamma distribution.

For Gaussian distribution, we are lucky as the closed form of MMM does exists, due to the following “squaring trick”:

M2=(∫−∞∞exp⁡(−x2) dx)2=∫−∞∞exp⁡(−x2) dx ⋅∫−∞∞exp⁡(−y2) dy=∫−∞∞∫−∞∞exp⁡(−(x+y)2) dxdy.\begin{aligned} M^2 &= \Big( \int_{-\infty}^\infty \exp( -x^2 ) \ dx \Big)^2 \\ &= \int_{-\infty}^\infty \exp( -x^2 ) \ dx \ \cdot \int_{-\infty}^\infty \exp( -y^2 ) \ dy \\ &= \int_{-\infty}^\infty \int_{-\infty}^\infty \exp( -(x + y) ^2 ) \ dx dy. \end{aligned} M2​=(∫−∞∞​exp(−x2) dx)2=∫−∞∞​exp(−x2) dx ⋅∫−∞∞​exp(−y2) dy=∫−∞∞​∫−∞∞​exp(−(x+y)2) dxdy.​

Denote r≐x2+y2r \doteq \sqrt{ x^2 + y^2}r≐x2+y2​ the distance of (x,y)(x, y)(x,y) to (0,0)(0, 0)(0,0).

Background

Let (X,d)(\mathcal{X}, d)(X,d) be a metric space, and fff be a function X→Rn\mathcal{X} \rightarrow \mathbb{R}^nX→Rn.
A Gaussian mechanism adds Gaussian noise to the output of fff, such that for all neighboring pairs x,y∈Xx, y \in \mathcal{X}x,y∈X with d(x,y)=1d(x, y) = 1d(x,y)=1, it is hard to distinguish the outputs, i.e., their outputs distributions are similar.

Δ=max⁡x,y∈X,d(x,y)=1∥f(x)−f(y)∥2, \Delta = \max_{x, y \in \mathcal{X}, d(x, y) = 1 } \Vert f(x) - f(y) \Vert_2, Δ=x,y∈X,d(x,y)=1max​∥f(x)−f(y)∥2​,

where ∥⋅∥2\Vert \cdot \Vert_2∥⋅∥2​ is the ℓ2\ell_2ℓ2​-norm of a vector in Rn\mathbb{R}^nRn.

Definition. Given fff and a variance parameter σ2\sigma^2σ2, the output of the Gaussian mechanism is a random variable defines as

g(x)=f(x)+X, g(x) = f(x) + X, g(x)=f(x)+X,

where X=(X1,X2,...,Xn)∼N(0,σ2I)X = (X_1, X_2, ..., X_n) \sim N(0, \sigma^2 I)X=(X1​,X2​,...,Xn​)∼N(0,σ2I).

Theorem. For δ,ϵ∈(0,1)\delta, \epsilon \in (0, 1)δ,ϵ∈(0,1), if σ2=Δ2ϵ2(2ln⁡1δ+ϵ)\sigma^2 = \frac{ \Delta^2 }{ \epsilon^2 } \Big( 2 \ln \frac{1}{\delta} + \epsilon \Big)σ2=ϵ2Δ2​(2lnδ1​+ϵ), then ∀x,y∈X\forall x, y \in \mathcal{X}∀x,y∈X, s.t., d(x,y)=1d(x, y) = 1d(x,y)=1, and for all measurable subset S⊂RnS \subset \mathbb{R}^nS⊂Rn, it holds

It is obvious that if we set σ→∞\sigma \rightarrow \inftyσ→∞, then the Gaussian distribution converge to a uniform one and it holds that Pr⁡[g(x)∈S]=Pr⁡[g(y)∈S]\Pr[ g(x) \in S] = \Pr[ g(y) \in S]Pr[g(x)∈S]=Pr[g(y)∈S].
Therefore, we are interested in how small σ\sigmaσ can be.

We claim that σ=O(Δϵ)\sigma = O(\frac{\Delta}{\epsilon } )σ=O(ϵΔ​) suffices.
This is based on three observations:

  1. Most of the probability mass of a Gaussian distribution concentrates on a ball centered at its mean with radius O(σ)O(\sigma)O(σ).
  2. The density ratio of any two points within the ball is bounded by a constant.
  3. The distance between the distribution center of g(x)g(x)g(x) and g(y)g(y)g(y) is bounded by Δ\DeltaΔ.

Proof. To ease the notations, we write g(y)=f(y)+Yg(y) = f(y) + Yg(y)=f(y)+Y to distinguish it from g(x)g(x)g(x).
By definition,
