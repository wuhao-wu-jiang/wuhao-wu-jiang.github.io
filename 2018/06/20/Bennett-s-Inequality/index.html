Bennett's Inequality

Remark: Let’s perform a sanity check before proving it. Suppose that X1,…,XnX_1, \ldots, X_nX1​,…,Xn​ are i.i.d. Bernoulli random variables each with mean μ\muμ. Then Xi≤1X_i \le 1Xi​≤1, E[Xi2]=μ\mathbb{E}[ X_i^2 ] = \muE[Xi2​]=μ and V=nμV = n \muV=nμ. Define t=ϵ⋅nμt = \epsilon \cdot n \mut=ϵ⋅nμ. The Bennett’s inequality states that

Via Markov’s inequality,

Pr⁡[∑i∈[n](Xi−μi)≥t]≤E[exp⁡(λ∑i∈[n](Xi−μi))]exp⁡(λt)≤exp⁡(−λt+VM2⋅(eλM−1−λM))=exp⁡(−λ(t+VM)+VM2⋅(eλM−1)) \begin{aligned} \Pr \left[ \sum_{i \in [n ] } (X_i - \mu_i) \ge t \right] &\le \frac{ \mathbb{E} \left[ \exp \left( \lambda \sum_{i \in [n ] } (X_i - \mu_i) \right) \right] }{ \exp( \lambda t) } \\ &\le \exp \left( -\lambda t + \frac{ V }{ M^2 } \cdot \left( e^{\lambda M} - 1 - \lambda M \right) \right) \\ &= \exp \left( -\lambda \Big( t + \frac{V}{M} \Big) + \frac{ V }{ M^2 } \cdot \left( e^{\lambda M} - 1 \right) \right) \end{aligned} Pr⎣⎢⎡​i∈[n]∑​(Xi​−μi​)≥t⎦⎥⎤​​≤exp(λt)E[exp(λ∑i∈[n]​(Xi​−μi​))]​≤exp(−λt+M2V​⋅(eλM−1−λM))=exp(−λ(t+MV​)+M2V​⋅(eλM−1))​
