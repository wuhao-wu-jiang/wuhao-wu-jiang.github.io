Gaussian Distribution

We show how Gaussian distribution (a.k.a. normal distribution) is constructed and list some of its basic properties.

Suppose we would like to have a distribution in R\mathbb{R}R, such that

  1. It is centered at the origin.
  2. Its density function decreases exponentially with respect to the square distance to the origin.

Remark: if we replace 2 with “a density function decreases exponentially with the distance to the origin”, we obtain Laplace distribution.

The density function p(x)p(x)p(x) should be inversely and exponentially proportional to x2x^2x2:

p(x)∝exp⁡(−x2) p(x) \propto \exp(-x^2 ) p(x)∝exp(−x2)

Denote MMM the value of

M≐∫−∞∞exp⁡(−x2) dx. M \doteq \int_{-\infty}^\infty \exp(-x^2) \ dx. M≐∫−∞∞​exp(−x2) dx.

It is easy to check that MMM is finite.
Then the close form of p(x)p(x)p(x) is

p(x)=1Mexp⁡(−x2). p(x) = \frac{1}{M} \exp(- x^2). p(x)=M1​exp(−x2).

It is easy to verity that p(x)p(x)p(x) is a valid density function: ∫−∞∞p(x) dx=1\int_{-\infty}^\infty p(x) \ dx = 1∫−∞∞​p(x) dx=1.

Closed Form

Theorem.
p(x)=1πexp⁡(−x2) p(x) = \frac{1}{ \sqrt \pi} \exp( - x^2 ) p(x)=π​1​exp(−x2)
is a density function.

In general, it is not hard to construct a distribution. Let h(x)h(x)h(x) be any integrable function on a domain D\mathcal{D}D such that

M=∫Dh(x) dx<∞ M = \int_\mathcal{D} h(x) \ dx < \infty M=∫D​h(x) dx<∞

Then we can view h(x)h(x)h(x) as almost an density function of some distribution.
The only obstacle is that MMM might not be 111.
However this is easy to fix.
We scale the value of h(x)h(x)h(x) by a constant factor at each point by setting

p(x)=1Mh(x). p(x) = \frac{1}{M} h(x). p(x)=M1​h(x).

In this case, we don’t even need to know the exact value of MMM.
We only need to guarantee that

The value of MMM exists and is finite.

Then we can safely rely on MMM as a normalized. For example, we know that

∫0∞x0.2e−x dx≤∫01e−x dx+∫1∞xe−x dx<∞. \int_0^\infty x^{0.2} e^{-x} \ dx \le \int_0^1 e^{-x} \ dx + \int_1^\infty x e^{-x} \ dx < \infty. ∫0∞​x0.2e−x dx≤∫01​e−x dx+∫1∞​xe−x dx<∞.

We can construct a density function as discussed above.
It turns out that this is a special case of Gamma distribution.

For Gaussian distribution, we are lucky as the closed form of MMM does exists, due to the following “squaring trick”:

M2=(∫−∞∞exp⁡(−x2) dx)2=∫−∞∞exp⁡(−x2) dx ⋅∫−∞∞exp⁡(−y2) dy=∫−∞∞∫−∞∞exp⁡(−(x+y)2) dxdy.\begin{aligned} M^2 &= \Big( \int_{-\infty}^\infty \exp( -x^2 ) \ dx \Big)^2 \\ &= \int_{-\infty}^\infty \exp( -x^2 ) \ dx \ \cdot \int_{-\infty}^\infty \exp( -y^2 ) \ dy \\ &= \int_{-\infty}^\infty \int_{-\infty}^\infty \exp( -(x + y) ^2 ) \ dx dy. \end{aligned} M2​=(∫−∞∞​exp(−x2) dx)2=∫−∞∞​exp(−x2) dx ⋅∫−∞∞​exp(−y2) dy=∫−∞∞​∫−∞∞​exp(−(x+y)2) dxdy.​

Denote r≐x2+y2r \doteq \sqrt{ x^2 + y^2}r≐x2+y2​ the distance of (x,y)(x, y)(x,y) to (0,0)(0, 0)(0,0).
We are integrating the function exp⁡(−r2)\exp( - r^2)exp(−r2) over the plane R2\mathbb{R}^2R2.
It is natural to switch to polar coordinate system (r,θ)(r, \theta)(r,θ), where θ\thetaθ is the angle of a point.
Here is an informal but intuitive explanation.
As the picture below demonstrates, if rrr increases by drdrdr and θ\thetaθ increases by dθd\thetadθ, the new area spanned by drdrdr and dθd \thetadθ is roughly rdrdθr dr d \thetardrdθ.
We can also calculate this algebraically

12[(r+dr)2−r2]dθ=rdrdθ+12(dr)2dθ≈rdrdθ.\frac{1}{2} [(r + dr)^2 - r^2] d\theta = r dr d \theta + \frac{1}{2} (dr)^2 d \theta \approx r dr d \theta. 21​[(r+dr)2−r2]dθ=rdrdθ+21​(dr)2dθ≈rdrdθ.

Finally,

M2=∫02π∫0∞exp⁡(−r2)rdrdθ=∫02π12∫0∞exp⁡(−r2)dr2dθ=∫02π12dθ=π.\begin{aligned} M^2 &= \int_{0}^{ 2 \pi} \int_{0}^\infty \exp( -r^2 ) r d r d \theta \\ &= \int_{0}^{ 2 \pi} \frac{1}{2} \int_{0}^\infty \exp( -r^2 ) d r^2 d \theta \\ &= \int_{0}^{ 2 \pi} \frac{1}{2} d \theta \\ &= \pi. \end{aligned} M2​=∫02π​∫0∞​exp(−r2)rdrdθ=∫02π​21​∫0∞​exp(−r2)dr2dθ=∫02π​21​dθ=π.​

Basic Properties

Expectation

As the distribution is symmetric to 000, it has expectation 0.

Variance

As its expectation is 0, the variance is given by

∫−∞∞x2πexp⁡(−x2) dx=−12π∫−∞∞x dexp⁡(−x2)=12π∫−∞∞exp⁡(−x2)dx=12ππ=12\begin{aligned} \int_{-\infty}^\infty \frac{ x^2 }{ \sqrt \pi } \exp( -x^2 ) \ dx &= -\frac{1}{2 \sqrt \pi } \int_{-\infty}^\infty x \ d \exp( -x^2 ) \\ &= \frac{1}{2 \sqrt \pi } \int_{-\infty}^\infty \exp( -x^2 ) d x\\ &= \frac{1}{2 \sqrt \pi } \sqrt \pi \\ &= \frac{1}{2} \end{aligned} ∫−∞∞​π​x2​exp(−x2) dx​=−2π​1​∫−∞∞​x dexp(−x2)=2π​1​∫−∞∞​exp(−x2)dx=2π​1​π​=21​​

General Gaussian Distribution

We have proved that if XXX is a random variable with density function p(x)=1πexp⁡(−x2)p(x) = \frac{1}{\sqrt \pi} \exp( -x^2)p(x)=π​1​exp(−x2), then

Var[X]=12. \mathbb{Var}[X] = \frac{1}{2}. Var[X]=21​.

Define another random variable Y≐2XY \doteq \sqrt{2} XY≐2​X, whose variance is

Var[X]=2Var[X]=1. \mathbb{Var}[X] = 2 \mathbb{Var}[X] = 1. Var[X]=2Var[X]=1.

To obtain its density function of YYY, we first scale the function p(x)p(x)p(x) horizontally by a factor of 2\sqrt 22​ to get

1πexp⁡(−x22). \frac{1}{ \sqrt{ \pi } } \exp \Big( -\frac{x^2}{2} \Big). π​1​exp(−2x2​).

Now the area under the curve f(x)=1πexp⁡(−x2/2)f(x) = \frac{1}{ \sqrt{ \pi } } \exp \big( - x^2 / {2} \big)f(x)=π​1​exp(−x2/2) is 2\sqrt 22​.
We normalize this area to 111 and obtain YYY’s density function as

p(y)=12πexp⁡(−y22) p(y) = \frac{1}{ \sqrt{2 \pi } } \exp \Big( -\frac{y^2}{2} \Big) p(y)=2π​1​exp(−2y2​)

In what follows, we use N(0,1)N(0, 1)N(0,1) to denote a Gaussian distribution with mean 0 and variance 111.
We can also scale XXX by a factor of 2σ\sqrt 2 \sigma2​σ for any σ>0\sigma > 0σ>0, to get Y=2σXY = \sqrt 2 \sigma XY=2​σX.
It have variance σ2\sigma^2σ2 and density function

p(y)=12πσ2exp⁡(−(y2σ)2)=12πσ2exp⁡(−y22σ2). p(y) = \frac{1}{ \sqrt{2 \pi \sigma^2} } \exp \Big( - \Big(\frac{y}{ \sqrt{2} \sigma } \Big)^2 \Big) = \frac{1}{ \sqrt{2 \pi \sigma^2 } } \exp \Big( -\frac{y^2}{ 2 \sigma^2 } \Big). p(y)=2πσ2​1​exp(−(2​σy​)2)=2πσ2​1​exp(−2σ2y2​).

Finally, we can shift the center of 000 by any real number μ∈R\mu \in \mathbb{R}μ∈R to obtain a new density function

p(y)=1π⋅2σ2exp⁡(−(y−μ)22σ2). p(y) = \frac{1}{ \sqrt{ \pi \cdot 2 \sigma^2 } } \exp\Big( -\frac{ (y - \mu)^2 }{ 2 \sigma^2 } \Big). p(y)=π⋅2σ2​1​exp(−2σ2(y−μ)2​).

The corresponding distribution is denoted as N(μ,σ2)N(\mu, \sigma^2)N(μ,σ2).

Advanced Properties

Sum of Normal Random Variables

If X∼N(μ1,σ12)X \sim N(\mu_1, \sigma_1^2)X∼N(μ1​,σ12​) and Y∼N(μ2,σ22)Y \sim N(\mu_2, \sigma_2^2)Y∼N(μ2​,σ22​), then for each a,b∈Ra, b \in \mathbb{R}a,b∈R,

aX+bY∼N(aμ1+bμ2,a2σ12+b2σ22). a X + b Y \sim N(a \mu_1 + b \mu_2 , a^2 \sigma_1^2 + b^2 \sigma_2^2). aX+bY∼N(aμ1​+bμ2​,a2σ12​+b2σ22​).

Let X′,Y′∼N(0,1)X', Y' \sim N(0, 1)X′,Y′∼N(0,1).
Then X=μ1+σ1X′X = \mu_1 + \sigma_1 X'X=μ1​+σ1​X′ and Y=μ2+σ2Y′Y = \mu_2 + \sigma_2 Y'Y=μ2​+σ2​Y′,
and

aX+bY=aμ1+bμ2+aσ1X′+bσ2Y′. a X + b Y = a \mu_1 + b \mu_2 + a \sigma_1 X' + b \sigma_2 Y'. aX+bY=aμ1​+bμ2​+aσ1​X′+bσ2​Y′.

It suffices to prove that if X′,Y′∼N(0,1)X', Y' \sim N(0, 1)X′,Y′∼N(0,1),

aσ1X′+bσ2Y′∼N(0,a2σ12+b2σ22),∀a,b∈R,∀σ1,σ2∈R≥0. a \sigma_1 X' + b \sigma_2 Y' \sim N(0, a^2 \sigma_1^2 + b^2 \sigma_2^2), \quad \forall a, b \in \mathbb{R}, \forall \sigma_1, \sigma_2 \in \mathbb{R}_{\ge 0}. aσ1​X′+bσ2​Y′∼N(0,a2σ12​+b2σ22​),∀a,b∈R,∀σ1​,σ2​∈R≥0​.

Hence, we can assume without lose of generality that X,Y∼N(0,1)X, Y \sim N(0, 1)X,Y∼N(0,1), and prove that aX+bY∼N(0,a2+b2)a X + bY \sim N(0, a^2 + b^2)aX+bY∼N(0,a2+b2) for each a,b∈Ra, b \in \mathbb{R}a,b∈R.

The demonstration here follows from [1]. The key observation is that if X,Y∼N(0,1)X, Y \sim N(0, 1)X,Y∼N(0,1), the joint distribution of XXX and YYY is rotation invariant with the origin.

The figure below shows the same distribution. Observe again that the probability mass concentrates at a small region centered at origin.

Therefore, for any t∈Rt \in \mathbb{R}t∈R, the set

{(x,y)∈R2:ax+by≤t} \{ (x, y) \in \mathbb{R}^2 : ax + by \le t \} {(x,y)∈R2:ax+by≤t}

has the same probability measure as the one

{(x,y)∈R2:x≤ta2+b2}. \Big\{ (x, y) \in \mathbb{R}^2 : x \le \frac{ t}{ \sqrt{a^2 + b^2} } \Big\}. {(x,y)∈R2:x≤a2+b2​t​}.

Observe that the latter can be obtained by rotating the former with respect to the origin (see the figure below).

Now, the probability of the latter set is given by

Pr⁡[X≤ta2+b2]=Pr⁡[a2+b2X≤t] \Pr \Big[ X \le \frac{ t}{ \sqrt{a^2 + b^2} } \Big] = \Pr\Big[ \sqrt{a^2 + b^2} X \le t \Big] Pr[X≤a2+b2​t​]=Pr[a2+b2​X≤t]

It concludes that X+YX + YX+Y has the same distribution as a2+b2X∼N(0,a2+b2)\sqrt{a^2 + b^2} X \sim N(0, a^2 + b^2)a2+b2​X∼N(0,a2+b2).

Concentration Inequalities

Bound 1

If X∼N(0,σ2)X \sim N(0, \sigma^2)X∼N(0,σ2), then ∀t>0\forall t > 0∀t>0,

Pr⁡[∣X∣≥t]≤exp⁡(−t22σ2) \Pr[ |X| \ge t] \le \exp \Big( -\frac{t^2}{ 2\sigma^2} \Big) Pr[∣X∣≥t]≤exp(−2σ2t2​)

Remark: it is possible to get a tighter bound by using more advanced techniques.

Intuition: in previous figures, it seems Gaussian distributions have a shape bump around its mean.
This is kind of mis-leading, because the x-axis and y-axis are scaled.
In the figure below, we plot N(0,1)N(0, 1)N(0,1), with ratio between xxx and yyy axes being approximate 1:11:11:1.
We see only a small bump around its mean.
Although the distribution span the entire range of R\mathbb{R}R, the probability mass is centered tightly around the origin.

Proof by Moment Generating Function

We prove a weaker version by a factor of 222. For all λ∈R\lambda \in \mathbb{R}λ∈R,

E[eλX]=12πσ∫eλxe−x22σ2 dx=eλ2σ2/22πσ∫e−(x−λσ2)22σ2 dx=eλ2σ2/2. \begin{aligned} \mathbb{E}[ e^{\lambda X} ] &= \frac{1}{\sqrt{2 \pi \sigma} }\int e^{\lambda x} e^{ - \frac{x^2}{2 \sigma^2 } } \ d x \\ &= \frac{ e^{\lambda^2 \sigma^2 / 2} }{\sqrt{2 \pi \sigma} }\int e^{ - \frac{ (x - \lambda \sigma^2)^2}{2 \sigma^2 } } \ d x \\ &= e^{\lambda^2 \sigma^2 / 2}. \end{aligned} E[eλX]​=2πσ​1​∫eλxe−2σ2x2​ dx=2πσ​eλ2σ2/2​∫e−2σ2(x−λσ2)2​ dx=eλ2σ2/2.​

Via Markov’s inequality, for all λ≥0\lambda \ge 0λ≥0, we have

Pr⁡[X≥t]≤E[eλX]eλt≤eλ2σ2/2−λt. \begin{aligned} \Pr[ X \ge t] &\le \frac{ \mathbb{E}[ e^{\lambda X} ] }{ e^{\lambda t} } &\le e^{\lambda^2 \sigma^2 / 2 - \lambda t}. \end{aligned} Pr[X≥t]​≤eλtE[eλX]​​≤eλ2σ2/2−λt.​

Setting λ=tσ2\lambda = \frac{t}{\sigma^2}λ=σ2t​, we get

Pr⁡[X≥t]≤e−t22σ2. \Pr[ X \ge t ] \le e^{ - \frac{t^2}{2 \sigma^2} }. Pr[X≥t]≤e−2σ2t2​.

It concludes that Pr⁡[∣X∣≥t]≤2e−t22σ2.\Pr[ |X| \ge t ] \le 2 e^{ - \frac{t^2}{2 \sigma^2} }.Pr[∣X∣≥t]≤2e−2σ2t2​.
