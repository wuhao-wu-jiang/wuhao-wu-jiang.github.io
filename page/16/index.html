Q-Digest

Posted on 2021-07-08 Edited on 2025-08-27

Problem

Let \(d \in \mathbb{N}^+\) be a positive integer and \(f = (0, \ldots, 0) \in \mathbb{R}^d\) be an all-zero vector. There is a sequence \(S\) of updates, each of which consists of a pair \((x, w) \in [d] \times \mathbb{R}^+\), increasing \(f[x]\) by \(w\).

Model Description
1. \(f \leftarrow (0, \ldots, 0) \in \mathbb{R}^d\);
2. For \((x, w) \in S\) do
3. \(\qquad\) \(f[x] \leftarrow f[x] + w\);

After witnessing the stream \(S\), we call \(f\) the frequency vector. We are interested in answering the following query.

Rank Query
Input： \(x \in [d]\)
Returen: \(r[x] \leftarrow \sum_{i = 1}^{x - 1} f[i]\).

We formally define the problem of approximate rank estimation.

Problem. Let \(\epsilon\) be an error parameter. Design an algorithm, such that for a rank query with a given element \(x \in [d]\), the summary will return an approximate rank \(\hat r [x]\), such that \[ r[x] − \epsilon \cdot W \le \hat r [x] \le r[x] + \epsilon \cdot W. \]

\(Q\)-Digest

The naive solution maintain the vector \(f \in \mathbb{R}^d\) explicitly. Assuming that each entry of \(f\) can always be stored in one word, the space usage of the is \(O(d)\).

Theorem. There exists an algorithm, \(Q\)-digest, that has space usage of \(O( \frac{1}{\epsilon} \log d)\) words, insertion time \(O(\log d)\) and query time of \(O(\log d)\).

We first study a prototype which requires \(O(d)\) words.

Prototype

The algorithm maintains an array of size \(d\) to keep all values of \(f\). Further, it construct a binary tree on the array, such that

  1. Each node \(v\) keeps the sum of frequencies of leaves in its subtree, denote as \(s[v]\).

By definition, if \(v\) is a leaf node, then \(s[v]\) equals the frequency of the node. For each node \(v\), denote \(L[v]\) and \(R[v]\) its left and right children respectively.

The figure below is an example with \(d = 8\) and \(f = (0, 1, 1, 0, 2, 1, 0, 1)\).

Insertion

To construct the tree, we can initialize a tree built on the domain \([d]\) with all counters \(s[v]\) equal to \(0\). Then on each pair of update \((x, w) \in S\), we invoke the following algorithm.

Update: \((x, w)\)
1. \(v \leftarrow\) root;
2. while \(v\) is not a leaf do
3. \(\qquad\) \(s[v] \leftarrow s[v] + w\);
4. \(\qquad\) if \(x\) is in the left tree of \(v\) then
5. \(\qquad\) \(\qquad\) \(v \leftarrow L[v]\)
6. \(\qquad\) else
7. \(\qquad\) \(\qquad\) \(v \leftarrow R[v]\)
8. \(s[v] \leftarrow s[v] + w\);

The update takes time \(O(\log d)\).

Query

The query algorithm is also very simple. Given an input \(x\), the algorithm walks down the path from the root to the leaf that contains \(x\). The rank of \(x\) equals the sum of of \(s[v]\) for the \(v\) such that

  1. it is a left child of some node on the path;
  2. it is not on the path.

Rank Query
Input： \(x \in [d]\) 1. \(v \leftarrow root\);
2. \(\hat r \leftarrow 0\);
3. while \(v\) is not a leaf node do
5. \(\qquad\) if \(x\) is in left subtree of \(v\) then
6. \(\qquad\) \(\qquad\) \(v \leftarrow L[v]\)
7. \(\qquad\)else
8. \(\qquad\) \(\qquad\) \(\hat r \leftarrow \hat r + s[ L[v] ]\);
9. \(\qquad\) \(\qquad\) \(v \leftarrow R[v]\)
10. return \(\hat r\)

The query algorithm has time complexity \(O(\log d)\).

\(Q\)-Digest

In this section, we show how to reduce the space usage to \(O(\frac{1}{\epsilon} \log d)\).

In the previous section, when the algorithm perform a rank query for \(x \in [d]\), it walks down a path \((v_1, \ldots, v_{\log d} )\), where \(v_1\) is the root of the tree, and \(v_{\log d}\) is the leaf node that contains \(x\).

There are \(\log d\) nodes in the path in total. If each node introduces an error of \(O(\epsilon W / \log d)\) to the estimation of \(r[x]\), then the final estimation has error bounded by \(O(\epsilon W)\).

This motivates to associate each node \(v\) in the tree with an error counter \(c[v]\), which is interpreted as the maximum error it can introduce to the rank estimation to a given element \(x\). It is initialized with \(0\) and is guaranteed to be bounded by some threshold \(C\) (except the leaf node). The threshold \(C\) is updated along with the insertion of elements into the tree. The algorithm maintains the invariant that \[ \frac{\epsilon \cdot W}{2 \log d} \le C \le \frac{\epsilon \cdot W}{\log d}. \]

Under this invariant, the estimation of \(r[x]\) is always bounded by \(\epsilon \cdot W\).

Insertion

The insertion algorithm is very simple. When an update \((x, w)\) arrives, the algorithm walks through the path from the root to the leaf that contains \(x\). For each internal node \(v\) in the path, the amount of additional error it can make is \(C - c[v]\). If this amount is greater than \(w\), then we just increase \(c[v]\) by \(w\), and stops the algorithm. Intuitively, we insert \(x\) into this node. On the other hand, if \(C - c[v]\) is less than \(w\), then we can still partial insert \(x\) into \(v\), detailed by the pseudo-code below. Doing so increase the rank estimation error only for the element greater than \(x\), but not for the elements less than or equal to \(x\). Finally, if the invariant of \(C\) does not hold, we need to update \(C\) and the tree.

Update: \((x, w)\)
1. \(W \leftarrow W + w\);
2. \(v \leftarrow\) root;
3. while \(w > 0\) and \(v\) is not a leaf do
4. \(\qquad\) \(s[v] \leftarrow s[v] + w\);
3. \(\qquad\) \(t \leftarrow \min\{ C - c[v], w\}\);
4. \(\qquad\) \(c[v] \leftarrow c[v] + t\);
5. \(\qquad\) \(w \leftarrow w - t\);
6. \(\qquad\) if \(x\) is in the left tree of \(v\) then
7. \(\qquad\) \(\qquad\) \(v \leftarrow L[v]\)
8. \(\qquad\) else
9. \(\qquad\) \(\qquad\) \(v \leftarrow R[v]\)
10. \(s[v] \leftarrow s[v] + w\);
11. \(c[v] \leftarrow c[v] + w\);
12. if \(C \le \frac{\epsilon \cdot W}{2 \log d}\) then
13. \(\qquad\) \(C \leftarrow \frac{\epsilon \cdot W}{\log d}\);
14. \(\qquad\) Compress(root, 0) // update the tree

Query

The query algorithm is the same as the one in the previous section, except that if algorithm encounters a node \(v\) with \(c[v] = 0\), then the algorithm can terminate earlier.

Maintenance

Finally, we discuss how to update the tree in a top-down approach. For a node \(v\), we try to increase \(c[v]\) to \(C\), by borrowing weights from the its descendent. There are two possible cases. First, if \(s[v] \le C\), then we can set \(c[v] = C\) and delete all descendants of \(v\). Otherwise, setting \(c[v] = C\) incurs a "debt" of \(d \doteq C - c[v]\) that \(v\)'s descendants need to pay. We assign as much debt as possible to the left child.

Compress \((v,d)\)
1. if \(c[v] = 0\) then return;
2. if v is a leaf node then
3. \(\qquad\) \(c[v] \leftarrow c[v] − d\)
4. else
5. \(\qquad\) if \(s[v] > C + d\) then
6. \(\qquad\) \(\qquad\) \(d \leftarrow d + C - c[v]\);
7. \(\qquad\) \(\qquad\) \(c[v] \leftarrow C\);
8. \(\qquad\) \(\qquad\) \(u \leftarrow L[v]\);
9. \(\qquad\) \(\qquad\) Compress \((L[v], \min(d , s[u] ) )\);
10. \(\qquad\) \(\qquad\) Compress \((R[v], \max(d − s[u] , 0 ) )\);
11. \(\qquad\) else
12. \(\qquad\) \(\qquad\) \(c[v] \leftarrow s[v] − d\);
13. \(\qquad\) \(\qquad\) Set weight of all descendants of \(v\) to \(0\);

By induction, we can show that \(d \le s[v]\).

Space Usage

A node is called "saturated" if \(c[v] = C\). By the invariant of \(C\), there are at most \(W / C = O((1 / \epsilon ) \log d)\) "saturated" nodes. By the algorithm, each node \(v\) in the tree with nonzero \(c[v]\), either "saturated", or has a "saturated" parent. However, each "saturated" can have at most two children. Therefore, the total number of nodes with nonzero \(c[v]\) is bounded by \(O((1 / \epsilon ) \log d)\).

Reference

[1] G. Cormode and K. Yi, Small Summaries for Big Data, 1st ed. Cambridge University Press, 2020.

Basic Entropy Inequalities

Posted on 2021-07-07 Edited on 2022-01-17

Let \(X\) and \(Y\) be two discrete random variables.

Relative Entropy

Given a distribution \(p\), if we encode it with a distribution \(q\) (with the same support), the expected coding length is roughly \[ \sum_i p_i \log \frac{1}{q_i} \]

The cost over the optimal coding is given by \[ \mathbf{D} (p || q) \doteq \sum_i p_i \log \frac{1}{q_i} - \sum_i p_i \log \frac{1}{p_i} = \sum_i p_i \log \frac{p_i}{q_i} \]

By definition, we know that this gap is always non-negative. We can prove it rigorously algebraically. We show two different approaches here.

  1. By that \(f(x) = x \log x\) is convex for \(x \ge 0\), we have \[ \begin{aligned} \mathbf{D}(p || q) &= \sum_i p_i \log \frac{p_i}{q_i} \\ &= \sum_i q_i \cdot \frac{p_i}{q_i} \log \frac{p_i}{q_i} \\ &\ge \left( \sum_i q_i \cdot \frac{p_i}{q_i} \right) \log \left( \sum_i q_i \cdot \frac{p_i}{q_i} \right) \\ &= 0 \end{aligned} \]
  2. By that \(f(x) = -\log x\) is convex (for \(x \ge 0\) ), we have \[ \begin{aligned} \mathbf{D}(p || q) &= \sum_i - p_i \log \frac{q_i}{p_i} \\ &\ge -\log \left( \sum_i p_i \cdot \frac{q_i}{p_i} \right) \\ &= 0 \end{aligned} \]

Theorem (Maximum Value). Let \(X\) be a random variable taking values in an alphabet \(\Omega\). Then \[ \mathbf{H}(X) \le \log | \Omega | \] The inequality is saturated if and only if \(X\) is a uniform random variable on \(\Omega\).

Proof. Let \(U\) be a uniform random variable on \(\Omega\). Then \[ \mathbf{D}( X \Vert U ) = \sum p(x) \log \frac{ p(x) }{ 1 / | \Omega | } = \log | \Omega | - \mathbf{H}(x) \ge 0. \]

\(\square\)

Joint Entropy

Definition. The joint entropy of \(X\) and \(Y\) is \[ \mathbf{H}(X, Y) = \sum_{x, y} p(x, y) \log \frac{1}{ p(x, y) } \]

If \(X\) and \(Y\) are independent, then \[ \mathbf{H}(X, Y) = \mathbf{H}(X) + \mathbf{H}(Y). \]

Conditional Entropy

Definition. The conditional entropy of \(X\) given \(Y = y\) is the entropy of the probability distribution \(p(X \mid Y = y)\) \[ \mathbf{H}(X \mid Y = y) = \sum_x p(x \mid y) \log \frac{1}{ p(x \mid y) } \]

Definition. The conditional entropy of \(X\) given \(Y\), is the average, over \(y\), the conditional entropy of \(X\) given \(y\) \[ \mathbf{H}(X \mid Y) = \sum_y p(y) \mathbf{H}(X \mid Y = y) = \sum_{x, y} p(x , y) \log \frac{1}{ p(x \mid y) }. \]

Theorem (Zero Conditional Entropy). Suppose that \(X\) and \(Y\) have the same support. \(\mathbf{H}(X \mid Y) = 0\) if and only if \(X\) is a function of \(Y\).

Proof. If \[ \mathbf{H}(X \mid Y) = \sum_y p(y) \mathbf{H}(X \mid Y = y) = 0. \]

Then either \(p(y) = 0\) or \(\mathbf{H}(X \mid Y = y) = 0\) for all \(y\). On the other hand, for each \(y\), if \(p(y) > 0\), then \[ \mathbf{H}(X \mid Y = y) = \sum_x p(x \mid y) \log \frac{1}{ p(x \mid y) } = 0, \] Then either \(p(x \mid y) = 0\) or \(\log \frac{1}{ p(x \mid y) } = 0\). It follows that there is a unique \(x\), for which \(p(x \mid y) = 1\).

Conversely, if \(X\) is a function of \(Y\), for each \(y\) such that \(p(y) > 0\), there is a unique \(x\) such that \(p(x \mid y) = 1\). Hence, \(\mathbf{H}(X \mid Y = y) = 0\) and \(H(X \mid Y) = 0\).

\(\square\)

Mutual Information

Definition. The mutual information between \(X\) and \(Y\) is \[ I(X; Y) = \mathbb{E}_{p(x, y) } \left[ \log \frac{ p(x, y)}{p(x) p(y) } \right] = \mathbf{D}( p(x, y) \Vert p(x) p(y) ). \]

The definition is symmetric and it holds that \(I(X; Y) = I(Y; X)\). If we know \(p(x,y)\), we use in expectation \(\mathbb{E}_{p(x, y) } \left[ \log \frac{1} { p(x, y)} \right]\) bits to describe them. If we know only \(p(x)\) and \(p(y)\), the mutual information measures the bits we waste.

\[ \small \begin{aligned} &\sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } \\ =& \sum p(x, y) \left( \log \frac{ 1 }{p(x)} + \log \frac{ 1 }{p(y)} - \log \frac{ 1 }{p(x, y)} \right) \\ =& \mathbf{H} ( X ) + \mathbf{H} ( Y ) - \mathbf{H} ( X, Y ) \end{aligned} \]

Further, mutual information measures the average reduction of randomness of \(X\) that results from learning the value of \(Y\). \[ \small \begin{aligned} \sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } &=\sum p(x, y) \log \frac{ p(x | y)}{p(x) } \\ &= \mathbf{H} ( X ) - \mathbf{H} ( X \mid Y). \end{aligned} \]

In this sense, the mutual information quantifies the dependence between \(X\) and \(Y\). It measures how much knowing one random variable reduces the uncertainty about the other random variable.

Similarly, we can prove that \[ \small \begin{aligned} \sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } &=\sum p(x, y) \log \frac{ p(x | y)}{ p(x) } \\ &=\mathbf{H} ( Y ) - \mathbf{H} ( X \mid Y). \end{aligned} \]

\(\mathbf{I}(X; Y) \ge 0\).

Proof. \[ \mathbf{I}(X; Y) = \mathbf{D}( p(x, y) \mid p(x) p(y) ) \ge 0 \]

Or \[ \begin{aligned} \sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } &= -\sum p(x, y) \log \frac{p(x) p(y) }{ p(x, y)} \\ &\ge - \log \sum p(x, y) \frac{p(x) p(y) }{ p(x, y)} \\ &=0 \end{aligned} \]

\(\square\)

Corollary. \(\mathbf{H} ( Y ) - \mathbf{H} ( X \mid Y) = \mathbf{H} ( X ) - \mathbf{H} ( X \mid Y) \ge 0\).

Theorem. The entropy \(\mathbf{H}(X)\) is concave in the probability density \(p(x)\).

Proof. Let \(\lambda_1, \lambda_2 \in [0, 1]\) such that \(\lambda_1 + \lambda_2 = 1\). Suppose that \(Z_1, Z_2\) are random variables with density functions \(p_1(z)\) and \(p_2(z)\) respectively. Define the random variable \[ Y = \begin{cases} 1, &\text{w.p } \lambda_1,\\ 2, &\text{w.p } \lambda_2, \end{cases} \] and let \(X = Z_Y\).

Then \[ \mathbf{H}(X) = \mathbf{H}( \lambda_1 p_1 + \lambda_2 p_2) \]

On the other hand, \[ \mathbf{H}(X \mid Y) = \lambda_1 \mathbf{H}( p_1 ) + \lambda_2 \mathbf{H}( p_2 ). \]

We finish the proof by observing that \(\mathbf{H}(X) \ge \mathbf{H}(X \mid Y)\).

\(\square\)

Definition. Let \(X\) be a Bernoulli random variable with mean \(p \in [0, 1]\). Denote \(\mathbf{H}(p) = \mathbf{H}(X) = p \log \frac{1}{p} + (1 - p) \log \frac{1}{1 - p}\).
Corollary. \(\mathbf{H}(p)\) is concave in \(p\).

Conditional Mutual Information

Definition. The conditional mutual information between \(X\) and \(Y\) given \(Z = z\) is the mutual information between the random variables \(X\) and \(Y\) in the joint ensemble \(\Pr (X, Y | Z = z)\), \[ I(X ; Y \mid Z = z) = \mathbf{H}(X \mid Z = z) - \mathbf{H}(X \mid Y, Z = z) \]

Definition. The conditional mutual information between \(X\) and \(Y\) given \(Z\) is average over \(z\) of \(I(X; Y \mid Z = z)\), \[ \begin{aligned} I(X; Y \mid Z) &= \mathbf{H}(X \mid Z) - \mathbf{H}(X \mid Y, Z) \\ &= \mathbf{H}(Y \mid Z) - \mathbf{H}(Y \mid X, Z) \\ &= \mathbf{H}(X \mid Z) + \mathbf{H}(Y \mid Z) - \mathbf{H}(X, Y \mid Z) \\ &= \mathbb{E}_{X, Y, Z} \log \frac{ p(X, Y \mid Z )}{ p(X \mid Z) \cdot p(Y \mid Z)}. \end{aligned} \]

Conditional Relative Entropy

Definition (Conditional Relative Entropy). \[ \begin{aligned} \mathbf{D}( p(y \mid x) || q(y \mid x) ) &\doteq \mathbb{E}_{p(x, y) } \left[ \log \frac{ p(Y \mid X) }{ q(Y \mid X) } \right]\\ &=\sum_{x} \sum_{ y } p(y \mid x) \log \frac{ p(y \mid x) }{ q(y \mid x) } \end{aligned} \]

Lemma. Relative entropy \(\mathbf{D}( p || q )\) is convex in the pair \((p, q)\): if there are two distribution pairs \((p_1, q_1)\) and \((p_2, q_2)\), then for \(\alpha_1, \alpha_2 \in [0, 1]\), s.t., \(\alpha_1 + \alpha_2 = 1\), it holds that \[ \mathbf{D}( \alpha_1 p_1 + \alpha_2 p_2 || \alpha_1 q_1 + \alpha_2 q_2 ) \le \alpha_1 \mathbf{D}( p_1 || q_1 ) + \alpha_2 \mathbf{D}( p_2 || q_2 ) \]

Proof. For any fix value \(x\), let \(s(x)\) be a shorthand for \(\alpha_1 p_1(x) + \alpha_2 p_2(x)\). It holds that \[ \small \begin{aligned} s(x) \log \frac{ s(x) }{ \alpha_1 q_1(x) + \alpha_2 q_2(x) } &=-s(x) \log \frac { \alpha_1 q_1(x) + \alpha_2 q_2(x) } { s(x) } \\ &= -s(x) \log \left( \frac { \alpha_1 p_1(x) } { s(x) } \frac { \alpha_1 q_1(x) } { \alpha_1 p_1(x) } + \frac { \alpha_2 p_2(x) } { s(x) } \frac { \alpha_2 q_2(x) } { \alpha_2 p_2(x) } \right) \\ &\le -\alpha_1 p_1(x) \log \frac { q_1(x) } { p_1(x) } - \alpha_2 p_2(x) \log \frac { q_2(x) } { p_2(x) } \end{aligned} \]

\(\square\)

Chain Rules

Chain Rule for Information Content

\(\log \frac{1}{ p(x, y) } = \log \frac{1}{p(x) } + \log \frac{1}{p(y | x) }\).

Chain Rule for Entropy

Let \(X_1, \ldots, X_n\) be random variables whose density function is \(p(x_1 \ldots, x_n)\).

\(\mathbf{H}(X_1, \ldots, X_n) = \sum_{i = 1}^n \mathbf{H}(X_i \mid X_1, \ldots, X_{i - 1})\)

Proof.
\[ \small \begin{aligned} \mathbf{H}(X_1, \ldots, X_n) &= \sum_{x_1 \ldots, x_n} p(x_1 \ldots, x_n) \log \frac{1}{\prod_{i = 1}^n p(x_i \mid x_1, \ldots, x_{i - 1}) } \\ &= \sum_{x_1 \ldots, x_n} p(x_1 \ldots, x_n) \sum_{i = 1}^n \log \frac{1}{ p(x_i \mid x_1, \ldots, x_{i - 1}) } \\ &=\sum_{i = 1}^n \mathbf{H}(X_i \mid X_1, \ldots, X_{i - 1}) \\ \end{aligned} \]

\(\square\)

Corollary. $ ( X_1, , X_n) _{i = 1}^n (X_i). $

\(\square\)

Chain Rule for Mutual Information

\(\mathbf{I}(Y; X_1, \ldots, X_n) = \sum_{i = 1}^n \mathbf{I}( Y; X_i \mid X_1, \ldots, X_{i - 1})\)

Proof 1.
\[ \small \begin{aligned} \mathbf{I}(X_1, \ldots, X_n; Y) &= \mathbf{H}(X_1, \ldots, X_n) - \mathbf{H}(X_1, \ldots, X_n \mid Y) \\ &= \sum_{i = 1}^n \mathbf{H}(X_i \mid X_1, \ldots, X_{i - 1}) - \sum_{i = 1}^n \mathbf{H}(X_i \mid X_1, \ldots, X_{i - 1}, Y) \\ &= \sum_{i = 1}^n \mathbf{I}(X_i ; Y\mid X_1, \ldots, X_{i - 1}) \end{aligned} \]

\(\square\)

Proof 2.
\[ \small \begin{aligned} \mathbf{I}( Y ; X_1, \ldots, X_n ) &= \mathbf{D} ( p(x_1, \ldots, x_n, y) \Vert p(x_1, \ldots, x_n) p(y)) \\ &= \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \log \frac{ p( x_1, \ldots, x_n, y) }{ p( x_1, \ldots, x_n ) p(y) } \\ &= \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \left( \log \frac{1}{ p(x_1, \ldots, x_n) } - \log \frac{1}{ p(x_1, \ldots, x_n \mid y ) } \right) \\ &= \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \log \frac{1}{ p(x_1) \cdot p(x_2 \mid x_1) \cdot \ldots \cdot p(x_n \mid x_1, \ldots, x_{n - 1} ) } \\ & - \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \log \frac{1}{ p(x_1 \mid y) \cdot p(x_2 \mid x_1, y) \cdot \ldots \cdot p(x_n \mid x_1, \ldots, x_{n - 1}, y ) } \\ &= \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \left( \sum_{i \in [n] } \log \frac{ p(x_i \mid x_1, \ldots, x_{i - 1}, y) }{ p(x_i \mid x_1, \ldots, x_{i - 1}) } \right) \\ &= \sum_{i = 1}^n \mathbf{I}( Y ; X_i \mid X_1, \ldots, X_{i - 1}) \end{aligned} \]

\(\square\)

Convexity of Relative Entropy. \[ \mathbf{D}( p(x, y) || q(x, y) ) = \mathbf{D}( p(x) || q(x) ) + \mathbf{D}( p(y \mid x) || q(y \mid x) ). \]

Proof. \[ \begin{aligned} \mathbf{D}( p(x, y) || q(x, y) ) &= \sum_{x, y} p(x, y) \log \frac{ p(x) p(y \mid x) }{q(x) q (y \mid x) } \\ &= \sum_{x, y} p(x, y) \log \frac{ p(x) }{q(x) } + \sum_{x, y} p(x, y) \log \frac{ p(y \mid x) }{ q (y \mid x) } \\ &= \mathbf{D}( p(x) || q(x) ) + \mathbf{D}( p(y \mid x) || q(y \mid x) ) \end{aligned} \]

\(\square\)

Amplificatoin by Sampling

Posted on 2021-07-05

Definition. Two distribution \(\mu\) and \(\mu'\) are \((\epsilon, \delta)\) close, denoted as \(\mu \simeq_{(\epsilon, \delta) } \mu'\), if for all measurable set \(A\), we have \[ \begin{aligned} \mu(A) &\le e^\epsilon \mu'(A) + \delta, \\ \mu'(A) &\le e^\epsilon \mu(A) + \delta. \end{aligned} \]

Theorem. Let \(q \in (0, 1)\) and let \(\mu\) and \(\mu'\) be two distributions such that \(\mu \simeq_{(\epsilon, \delta)} \mu'\). For \(\nu = (1 - q) \mu + q \mu'\), it holds that \[ \nu \simeq_{ (\epsilon', q \delta) } \mu, \] where \(\epsilon' = \ln ( q (e^\epsilon - 1) + 1 )\).

Proof. For each measurable set \(A\) \[ \begin{aligned} \nu(A) &= (1 - q) \mu(A) + q \mu' (A) \\ &\le (1 - q) \mu(A) + q (e^\epsilon \mu (A) + \delta) \\ &= ( q (e^\epsilon - 1) + 1 ) \mu(A) + q \delta. \end{aligned} \]

Next we prove the other direction. Denote \(t = e^{\epsilon} - 1\). Then \[ e^{\epsilon'} = 1 + qt, \]

and \[ \begin{aligned} \nu(A) &= (1 - q) \mu(A) + q \mu' (A) \\ &\ge (1 - q) \mu(A) + q e^{-\epsilon} (\mu (A) - \delta) \\ &= (1 - q) \mu(A) + \frac{q}{1 + t} (\mu (A) - \delta) \\ &= \mu(A) - \frac{qt}{1 + t} \mu (A) - \frac{q}{1 + t} \delta \\ &= \left( 1 - \frac{qt}{1 + t} \right) \mu (A) - \frac{1}{1 + t} q \delta \\ &\ge \left( 1 - \frac{qt}{1 + qt} \right) \mu (A) - \frac{1}{1 + qt} q \delta \\ &= \frac{1}{1 + qt} \left( \mu (A) - q \delta \right). \end{aligned} \]

\(\square\)
