Jensen's Inequality

Posted on 2021-06-25 Edited on 2021-09-22

Let \((\Omega, \mathcal{F}, \mu)\) be a probability space. Let \(f\) be a random variable, i.e., a measurable map from \(\Omega\) to \(\mathcal{R}\), the Borel sets in \(\mathbb{R}\). Suppose that \(\varphi\) is a convex function defined on the interval \(I = [a, b]\), and that \(f(\Omega) \subset I\).
Suppose that \(f\) and \(\varphi(f)\) are integrable.

Theorem. (Jensen's Inequality) \[ \varphi \left( \int f \, d\, \mu \right) \le \int \varphi (f) \, d\, \mu. \]

Proof. Since \(f(\Omega) \subset I\), by monotonicity of integration, it holds that \[ x^* \doteq \int f \, d\, \mu \in I. \]

Case 1. Suppose that \(x^* \in (a, b)\). As \(\varphi\) is convex, there exits a linear function \(h(x) = c x + d\) such that

  • It passes \((x^*, \varphi( x^*) )\), i.e., \(h(x^*) = c x^* + d = \varphi( x^*)\).
  • \(\varphi (x) \ge h(x)\) for all \(x \in I\).

Hence, by monotonicity of integration, \[ \int \varphi (f) \, d\, \mu \ge \int (c f + d) \, d\, \mu = c x^* + d = \varphi (x^*). \]

Case 2. \(x^* = a\) (or \(b\)). Then \[ \int \big( \varphi (f) - a \big) \, d\, \mu = 0. \]

It follows that \(f( \omega) = a\) a.e. \(\omega \in \Omega\). Clearly \[ \varphi \left( \int f \, d\, \mu \right) = \varphi (a) = \int \varphi (a) \, d\, \mu. \]

\(\square\)

Example. Let \(\varphi(x) = - \ln x, \forall x > 0\). Let \(x_1, \ldots, x_n \in (0, \infty)\). Denote \(\bar x = (1 / n) \sum_{i} x_i\). \[ \varphi( \bar x ) \le \frac{1}{n} \sum_i (- \ln x_i ). \]

Instead invoking the theorem directly, we apply the same technique to prove this again. The claim is trivial if \(x_1 = \ldots = x_n\). Otherwise, consider the tangent line of \(\varphi(x)\) at \(\bar x\):
\[ h(x) = - \frac{1}{ \bar x } \cdot (x - \bar x) - \ln \bar x. \]

It holds that \(\varphi(x_i) \ge h(x_i)\), for all \(i \in [n]\). Therefore, \[ \frac{1}{n} \sum_{i} h(x_i) \le \frac{1}{n} \sum_i (- \ln x_i ). \]

On the other hand, \[ \frac{1}{n} \sum_{i} h(x_i) = - \frac{1}{ \bar x } \cdot \Big( \frac{1}{n} \sum_{i} x_i - \bar x \Big) - \ln \bar x = - \ln \bar x. \]
