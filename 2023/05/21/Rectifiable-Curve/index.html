Rectifiable Curve

Posted on 2023-05-21 Edited on 2024-08-27

Curve

A curve can be conceptualized as the trace created by a moving pen on a piece of paper.
It is considered a one-dimensional object, which implies that a single number is sufficient to describe the position of a point on the curve.
More generally, in Rk\mathbb{R}^kRk, we define a curve rigorously thus.

Definition. A continuous mapping γ\gammaγ of an interval [a,b][a, b][a,b] into Rk\mathbb{R}^kRk is called a curve in Rk\mathbb{R}^kRk.

It is important to note that, under this definition, different curves may specify the same set of points, i.e. they have the same range.

Curve Length

Given a partition P={x0,…,xn}P = \{x_0 , \ldots , x_n \}P={x0​,…,xn​} of [a,b][a, b][a,b], define

where the i(th)i^{(th)}i(th) term in the sum is the Euclidean distance between γ(xi−1)\gamma(x_{i - 1})γ(xi−1​) and γ(xi)\gamma(x_i)γ(xi​).

Therefore, Λ(P,γ)\Lambda(P, \gamma)Λ(P,γ) represents the length of a polygonal path with vertices located at γ(x0),γ(x1),…,γ(xn)\gamma(x_0), \gamma(x_1), \ldots , \gamma(x_n)γ(x0​),γ(x1​),…,γ(xn​) in order.
As the partition becomes increasingly finer, the polygon approaches the range of γ\gammaγ more closely.
Hence, it is reasonable to define the length of γ\gammaγ as

Λ(γ)≐sup⁡Λ(P,γ), \Lambda(\gamma) \doteq \sup \Lambda(P, \gamma), Λ(γ)≐supΛ(P,γ),

where the supremum is taken over all partitions of [a,b][a, b][a,b].
If Λ(γ)<∞\Lambda(\gamma) < \inftyΛ(γ)<∞, we say that γ\gammaγ is rectifiable.
