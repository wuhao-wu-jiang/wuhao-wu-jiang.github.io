Azuma-Hoeffding Inequality

Posted on 2021-06-02 Edited on 2021-11-16

Azuma-Hoeffding Inequality

Let \(X_0, \ldots, X_n\) be a martingale such that \[ A_k \le X_k - X_{k - 1} \le B_k \] for some random variables \(A_k\) and \(B_k\) that may be functions of \(X_0, X_1, \ldots X_{k - 1}\). If \(B_k - A_k \le c_k\) for some constants \(c_k\), then, for all \(t \ge 0\) and any \(\eta > 0\), \[ \Pr[ | X_t - X_0 | \ge \eta ] \le 2 \exp \left( - \frac{ 2 \eta^2 }{ \sum_{ k \in [t] } c_k^2 }\right) \]

Hoeffding's Lemma

Let \(X\) be any real-valued random variable with mean \(0\), such that \({\displaystyle a\leq X\leq b}\) almost surely, i.e. with probability one. Then, for all \({\displaystyle \lambda \in \mathbb {R} }\), \[ {\displaystyle \mathbb {E} \left[ e^{\lambda X} \right] \le \exp {\Big (} {\frac{\lambda^{2}(b-a)^{2}} {8}} {\Big ) }. } \]

Proof. We prove a version with a weaker factor of \(\frac{1}{2}\) but simpler analysis. The proof for the stronger version is included in the appendix.

First, consider a special case of Rademacher random variable \(\sigma\) that equals \(-1\) and \(1\) with equal probability. For all \(\lambda \in \mathbb{R}\), \[ \begin{aligned} \mathbb{E} [ e^{ \lambda \sigma} ] &= \frac{ e^{ - \lambda } + e^{ \lambda } }{ 2 } &= \sum_{ i \ge 0 } \frac{ \lambda^{ 2i } }{ (2i)! } &\le \sum_{ i \ge 0 } \frac{ ( \lambda^2 / 2)^i }{ i ! } &= e^{ \lambda^2 / 2}. \end{aligned} \]

Next, let \(Y\) be identical copy of \(X\). By Jensen's inequality, \[ \begin{aligned} \mathbb{E} [ e^{ \lambda X }] &= \mathbb{E} [ e^{ \lambda ( X - \mathbb{E}[ X ] ) }] &= \mathbb{E}_{X} [ e^{ \lambda ( X - \mathbb{E}_{Y} [ Y ] ) }] &\le \mathbb{E}_{ X, Y } [ e^{ \lambda ( X - Y ) }] \end{aligned} \]

Since \(X\) and \(Y\) has identical distribution, the random variable \(\sigma( X - Y)\) has the same distribution as \(X - Y\). Hence, \[ \mathbb{E}_{ X, Y } [ e^{ \lambda ( X - Y ) }] = \mathbb{E}_{ X, Y } \left[ \mathbb{E}_\sigma [ e^{ \lambda ( X - Y ) \sigma } \middle\vert X, Y] \right]. \]

Finally, by previous argument, \[ \begin{aligned} \mathbb{E}_{\sigma } [ e^{ \lambda ( X - Y ) \sigma } \mid X, Y ] \le e^{ \lambda^2 ( X - Y )^2 / 2 } \le e^{ \lambda^2 (b - a)^2 / 2 }. \end{aligned} \]

It concludes that \(\mathbb{E}_{ X, Y } [ e^{ \lambda ( X - Y ) }] \le e^{ \lambda^2 (b - a)^2 / 2 }\).

\(\square\)

Proof of the Theorem

Define \(Z_k = X_k - X_{k - 1}\) for each \(k \in [n]\). Since \(X_1, \ldots, X_n\) is a martingale, \[ \mathbb{E}[ Z_k \mid X_0, \ldots, X_{k - 1} ] = 0. \]

Via Hoeffding's Lemma, for each \(k \in [n]\), and for all \(\lambda \in \mathbb{R}\), it holds that \[ \mathbb{E} \left[ e^{\lambda Z_t } \right] \le e^{ \lambda^2 c_k^2 / 8 }. \]

It follows that \[ \begin{aligned} \mathbb{E} \left[ e^{ \lambda \sum_{k \in [t]} Z_k } \right] &=\mathbb{E} \left[ \mathbb{E} \left[ e^{ \lambda \sum_{k \in [t] } Z_k } \middle\vert Z_1, \ldots, Z_{t - 1} \right] \right] \\ &=\mathbb{E} \left[ \mathbb{E} \left[ e^{\lambda Z_t } \right] \cdot \mathbb{E} \left[ e^{ \lambda \sum_{k \in [t - 1] } Z_k } \middle\vert Z_1, \ldots, Z_{t - 1} \right] \right] \\ &\le e^{ \lambda^2 c_t^2 / 8 } \cdot \mathbb{E} \left[ \mathbb{E} \left[ e^{ \lambda \sum_{k \in [t - 1] } Z_k } \middle\vert Z_1, \ldots, Z_{t - 1} \right] \right] \end{aligned} \]

By induction, it holds that \[ \mathbb{E} \left[ e^{ \lambda \sum_{k \in [t]} Z_k } \right] \le e^{ \lambda^2 \sum_{k \in [t] } c_k^2 / 8 }. \]

For all \(\lambda > 0\), via Markov's inequality, \[ \begin{aligned} \Pr \left[ \sum_{k \in [t] } Z_k \ge \eta \right] &= \Pr \left[ e^{ \lambda \sum_{k \in [t] } Z_k } \ge e^{ \lambda \eta } \right] \\ &\le \frac{ \mathbb{E} \left[ e^{ \lambda \sum_{k \in [t]} Z_k } \right] }{ \mathbb{E}[ e^{ \lambda \eta } ] } \\ &\le e^{ \lambda^2 \sum_{k \in [t] } c_k^2 / 8 - \lambda \eta }. \end{aligned} \]

When \(\lambda = \frac{4 \eta}{ \sum_{k \in [t] } c_k^2 }\), the probability is minimized to \[ \exp \left\{ \frac{2 \eta^2 }{ \sum_{k \in [t] } c_k^2 } - \frac{4 \eta^2 }{ \sum_{k \in [t] } c_k^2 } \right\} = \exp \left\{ - \frac{2 \eta^2 }{ \sum_{k \in [t] } c_k^2 } \right\}. \]

\(\square\)

Hoeffding's Lemma

Proof 1

Suppose that \(b \neq a\) (otherwise, \(X\) is a constant). Define

  1. \(\theta = - \frac{a}{b - a} = 1 - \frac{b}{b - a} \in (0, 1)\);
  2. \(c = (b - a) > 0\).

By Jensen's inequality, \[ \begin{aligned} \mathbb{E} [ e^{\lambda X } ] &= \mathbb{E} \left[ \exp \left( \frac{b - X}{ b - a} \cdot \lambda a + \frac{X - a}{b - a} \cdot \lambda b \right) \right] \\ &\le \mathbb{E} \left[ \frac{b - X}{ b - a} \cdot \exp \left( \lambda a \right) + \frac{X - a}{b - a} \cdot \exp \left( \lambda b \right) \right] \\ &= (1 - \theta) \cdot \exp \left( \lambda a \right) + \theta \cdot \exp \left( \lambda b \right) \\ &= \left( 1 - \theta + \theta e^{\lambda (b - a)} \right) \exp \left( \lambda a \right) \\ &= \left( 1 - \theta + \theta e^{ (\lambda c) } \right) e^ { - \theta (\lambda c) } \end{aligned} \]

Let \(f(z) \doteq - \theta z + \ln \left( 1 - \theta + \theta e^{ z} \right)\). The function is well-defined, as \(1 - \theta + \theta e^{ z} > 0\) for \(z \in \mathbb{R}\). Observe that \[ \exp \Big \{ f(\lambda c) \Big \} = \left( 1 - \theta + \theta e^{\lambda c } \right) \exp \left( - \lambda \theta c \right). \]

Further, \[ f'(z) = - \theta + \frac{ \theta e^{ z} }{ 1 - \theta + \theta e^{ z} }, \]

and \[ f''(z) = \frac{ \theta e^{ z} ( 1 - \theta + \theta e^{ z} ) - ( \theta e^z )^2 }{ ( 1 - \theta + \theta e^{ z} )^2 } = \frac{ ( 1 - \theta ) \theta e^{ z} }{ ( 1 - \theta + \theta e^{ z} )^2 } \le \frac{1}{4}. \]

Finally, by Taylor's expansion, there exists some \(z^* \in [0, \lambda c]\), such that \[ \begin{aligned} f(\lambda c) &= f(0) + f' (0) \cdot (\lambda c) + \frac{ f''(z^*) }{ 2 } (\lambda c)^2 \\ &= \frac{ f''(z^*) }{ 2 } (\lambda c)^2 \\ &\le \frac{1}{8 } (\lambda c)^2 \\ &= \frac{ \lambda^2 (b - a)^2 }{8 }. \end{aligned} \]

The picture above shows that the bound might not be tight.

\(\square\)

Proof 2

Preliminaries

The proof relies on the following lemma.

Lemma. Let \((\Omega, \mathcal{F}, \mathbb{P})\) be a probability space. Let \(Y\) be a nonnegative \(\mathcal{F}\)-measurable function, and define the measure \(\mu\) by \[ \mu(A) \doteq \int_A Y \, d\,\mathbb{P}, \qquad A \in \mathcal{F}. \] Then for nonnegative \(\mathcal{F}\)-measurable function \(Z\), it holds that \[ \int Z \, d\, \mu = \int Z \cdot Y \, d\, \mathbb{P}. \]

Proof of the Lemma. If \(Z\) is an indicator function \(1_A\) for some \(A \in \mathcal{F}\), then the lemma clearly holds. It follows that the lemma holds for nonnegative simple functions. For general nonnegative \(Z\), there is a sequence of nonnegative simple functions \(\{ Z_n \}\), such that \(Z_n \uparrow Z\). Via monotone convergence theorem, we obtain \[ \lim_n \int Z_n \, d\, \mu = \int Z \, d\, \mu, \\ \lim_n \int Z_n \cdot Y \, d\, \mathbb{P} = \int Z \cdot Y \, d\, \mathbb{P}. \]

\(\blacksquare\)

Formal Proof

Let \(X\) be a random variable such that \(a \le X \le b\). Define the function \(f: \mathbb{R}_{\ge 0} \rightarrow \mathbb{R}\) as \[ f(\lambda) = \mathbb{E}[\exp(\lambda X)],\, \] and let \[ h(\lambda) = \ln f(\lambda). \]

We will justify the interchange of differentiation and expectation, and show that \[ f'(\lambda) = \mathbb{E}\left[ X \exp ( \lambda X) \right],\, \\ f''(\lambda) = \mathbb{E}\left[ X^2 \exp ( \lambda X) \right]. \]

Therefore, \[ \begin{aligned} h'(\lambda) &= \frac{ f'(\lambda) }{ f(\lambda) } = \frac{ \mathbb{E}\left[ X \exp ( \lambda X) \right] }{ \mathbb{E}[\exp(\lambda X)] }\,, \\ h''(\lambda) &= \frac{ f''(\lambda) \cdot f(\lambda) - \left( f'(\lambda) \right)^2 }{ \left( f(\lambda) \right)^2 } \\ &= \frac{ \mathbb{E}[X^2 \exp(\lambda X)] \cdot \mathbb{E}[\exp(\lambda X)] - \left( \mathbb{E}[ X \exp(\lambda X)] \right)^2 }{ \left( \mathbb{E}[\exp(\lambda X)] \right)^2 } \\ &= \frac{ \mathbb{E}[X^2 \exp(\lambda X)] }{ \mathbb{E}[\exp(\lambda X)] } - \frac{ \left( \mathbb{E}[ X \exp(\lambda X)] \right)^2 }{ \left( \mathbb{E}[\exp(\lambda X)] \right)^2 }. \end{aligned} \]

Suppose \((\Omega, \mathcal{F}, \mathbb{P})\) is the probability space on which \(X\) is defined, i.e., \(X\) is a function from \(\mathcal{F}\) to \(\mathbb{R}\). Now we define a new measure \(\mathbb{P}_\lambda: \mathcal{F} \rightarrow \mathbb{R}\) as \[ \mathbb{P}_{\lambda}(A) \doteq \int_A \frac{\exp(\lambda X)}{ \mathbb{E}[\exp(\lambda X)] } \, d \mathbb{P}, \quad \forall A \in \mathcal{F}. \] It is easy to see that \(\mathbb{P}_{\lambda}\) is a new probability measure. The function \(\delta(\lambda) \doteq \frac{\exp(\lambda X)}{ \mathbb{E}[\exp(\lambda X)] }\) is called the density function of \(\mathbb{P}_{\lambda}\). We see that \[ \begin{aligned} \mathbb{E}_{\lambda} [X] \doteq \int X d { \mathbb{P}_{\lambda} } &= \int X \frac{\exp(\lambda X)}{ \mathbb{E}[\exp(\lambda X)] } \, d \mathbb{P} = \frac{ \mathbb{E}\left[ X \exp ( \lambda X) \right] }{ \mathbb{E}[\exp(\lambda X)] }\,, \\ \mathbb{E}_{\lambda} [X^2] \doteq \int X^2 d { \mathbb{P}_{\lambda} } &= \int X^2 \frac{\exp(\lambda X)}{ \mathbb{E}[\exp(\lambda X)] } \, d \mathbb{P} = \frac{ \mathbb{E}\left[ X^2 \exp ( \lambda X) \right] }{ \mathbb{E}[\exp(\lambda X)] }\,. \end{aligned} \]

It follows that \[ \begin{aligned} h''(\lambda) =\mathbb{Var}_{\lambda} [ X ] \doteq \mathbb{E}_{\lambda} [X^2] - \big( \mathbb{E}_{\lambda} [X] \big)^2. \end{aligned} \] Therefore, \[ \mathbb{Var}_{\lambda} [ X ] = \mathbb{Var}_{\lambda} \left[ X - \frac{a + b}{2} \right] = \mathbb{E}_{\lambda} \left[ \left( X - \frac{a + b}{2} \right)^2 \right] \le \frac{(b - a)^2}{4}. \]

Finally, via Taylor's expansion, \[ h(\lambda) = h(0) + \lambda \cdot h'(0) + \frac{\lambda^2}{2} h''(\lambda^*). \]

Since \(h(0) = 0\), \(h'(0) = \mathbb{E}[X] = 0\), and \(h''(\lambda^*) = \mathbb{Var}_{\lambda^*}[X] \le (b - a)^2 / 4\), we conclude that \[ h(\lambda) \le \frac{ (b - a)^2 }{ 8 }. \]

First Derivative of \(f\).

Denote \(M \doteq \max\{ |a|, |b| \}\). For each \(n \in \mathbb{N}_+\), \[ g(n) \doteq \frac{ f(\lambda + 1 / n) - f(\lambda) }{ 1 / n} = \mathbb{E}\left[ \frac{ \exp( (\lambda + 1 / n) \cdot X) - \exp(\lambda X) }{ 1 / n} \right] \] Via mean value theorem, there exists \(\lambda' \in [\lambda, \lambda + 1 / n]\), s.t., \[ \frac{ \exp( (\lambda + 1 / n) \cdot X) - \exp(\lambda X) }{ 1 / n} = X \cdot \exp ( \lambda' \cdot X) \le M \cdot \exp( (\lambda + 1) \cdot M). \] Since the RHS is integrable, via dominated convergence theorem, \[ f'(\lambda) = \lim_n g(n) = \mathbb{E}\left[ \lim_n \frac{ \exp( (\lambda + 1 / n) \cdot X) - \exp(\lambda X) }{ 1 / n} \right] = \mathbb{E}\left[ X \exp ( \lambda X) \right] . \]

Second Derivative of \(f\).

Similarly, for each \(n \in \mathbb{N}\), there exists \(\lambda' \in [\lambda, \lambda + 1 / n]\), s.t., \[ \frac{ X \cdot \exp( (\lambda + 1 / n) \cdot X) - X \cdot \exp(\lambda X) }{ 1 / n} = X^2 \cdot \exp ( \lambda' \cdot X) \le M^2 \cdot \exp( (\lambda + 1) \cdot M). \]

It follows via dominated convergence theorem that \[ \begin{aligned} f''(\lambda) &= \lim_n \mathbb{E}\left[ \frac{ X \cdot \exp( (\lambda + 1 / n) \cdot X) - X \cdot \exp(\lambda X) }{ 1 / n} \right] \\ &= \mathbb{E}\left[ \lim_n \frac{ X \cdot \exp( (\lambda + 1 / n) \cdot X) - X \cdot \exp(\lambda X) }{ 1 / n} \right] \\ &= \mathbb{E}\left[ X^2 \exp ( \lambda X) \right] . \end{aligned} \]

\(\square\)
