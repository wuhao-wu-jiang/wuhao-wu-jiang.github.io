Reed-Solomon Codes

Posted on 2021-03-20

Alice wants to send a sequence of \(n\) messages to Bob. Let \(m_1, ..., m_n\) be the messages sent and \(r_1, ..., r_n\) be the message received. Each message is from an alphabet \(\Sigma\) of size \(|\Sigma | = p\), where \(p\) is a prime.

The figure below shows the case of \(n = 2\), \(m_1 = 2, m_2 = 3, r_1 = 2, r_2 = 3\), and \(p = 7\).

Ideally, the messages received equal the ones sent, i.e., \(r_i = m_i\) for every \(i \in [n]\). In practice, the communication channel could be unreliable and cause the message to be lost or corrupted. The former is called erasure error and the later is called general error.

To make the communication resilient to both types of errors, we encode the messages by adding redundancy. In this note we discuss a coding scheme called "Reed-Solomon code" that is based on polynomials. In particular, we use the following property of polynomials.

Property. Let \(\mathbb{F}_p\) be the finite field with size \(p\). For every integer \(n\) such that \(2 \le n \le p\), and for \(\forall (x_1, y_1), ..., (x_n, y_n) \in \mathbb{F}_p^+\), there is a unique polynomial \(f(x)\) with degree at most \(n - 1\) such that \[ f(x_i ) = y_i \pmod p, \forall i \in [n]. \]

Erasure Error

We illustrate the key idea by an example.

Suppose that \(n = 2\), \(p = 7\) and that at most one of the every three packages sent is lost. Let the messages sent be \(m_1 = 2, m_2 = 3\).

We can view \(m_1\) as a point \((1, m_1)\) and \(m_2\) as a point \((2, m_2)\) in the space \(\mathbb{F}_p^2\). There is a unique line on which there two points sit: \[ f(x) = x + 1 \pmod p. \]

We can send the value of \(f(x)\) at \(x = 3\) as the third message, i.e., \(m_3 = f(3)\).

If one message is lost, two messages are still received. These messages determine two points in the space \(\mathbb{F}_p^2\) with which we can reconstruct the line (i.e., \(f(x)\)) perfectly. Finally, we recover the values of \(m_1\) and \(m_2\) by computing \(f(1)\) and \(f(2)\).

The following theorem generalizes the techniques discussed.

Theorem. \(\forall k \in \N^+\), assuming that \(p \ge n + k\), there is a coding scheme such that 1. encodes \(n\) messages into \(n + k\) messages; 2. is resilient to \(k\) erasures.

Proof. We view the messages as \(n\) points in \(\mathbb{F}_p^2:\) \[ \{ (1, m_1), ..., (n, m_n) \}. \]

There is a unique polynomial \(f(x)\) of degree at most \(n - 1\) on which these points sit. We can compute \(f(x)\) by Lagrange interpolation method.

Then we generate \(k\) additional messages \(m_{n + 1} = f(n + 1), ..., m_{n + k} = f(n + k)\), which represent points \[ \{ (n + 1, m_{n + 1} ), ..., (n + k, m_{n + k}) \}. \]

We transmit messages \(m_1, ..., m_{n + k}\). As long as \(n\) of the \(n + k\) messages are received, the receiver can reconstruct the unique polynomial \(f(x)\) and recovers \(m_1 = f(1), ..., m_n = f(n)\).

\(\square\)

General Error

For general error, we need twice the redundancy. We continue with the previous example.

Suppose that \(n = 2\), \(p = 7\) and that at most one of the every four packages sent is corrupted. Let the messages sent be \(m_1 = 2, m_2 = 3\). Let \(f(x) = x + 1 \pmod p\) be the line that passes \((1, m_1)\) and \((2, m_2)\) in \(\mathbb{F}_p^2\).

This time we generate two additional messages \(m_3 = f(3)\) and \(m_4 = f(4)\), as the figure below shows.

Suppose that exactly one of the four messages is corrupted after transmission. If we can identify the messages that are not corrupted, then we can reconstruct the function \(f(x)\) by these messages and then use \(f(x)\) to recover the corrupted message.

Observation. For any three messages, at least two of them are correct, which determine an unique line. If the message left is also correct, it should sit on this line. If not, one of the three messages is corrupted.

The figure below illustrates the observation.

Hence, we enumerate all possible three points and check whether they are colinear.

The following theorem generalizes the techniques discussed.

Theorem. \(\forall k \in \N^+\), assuming that \(p \ge n + 2k\), there is a coding scheme such that 1. encodes \(n\) messages into \(n + 2k\) messages; 2. is resilient to \(k\) corruptions.

Proof. As before, there is a unique polynomial \(f(x)\) of degree at most \(n - 1\) on which the following points sit: \[ \{ (1, m_1), ..., (n, m_n) \}. \]

Then we generate \(2k\) additional messages \(m_{n + 1} = f(n + 1), ..., m_{n + 2k} = f(n + 2k)\), which represent points \[ \{ (n + 1, m_{n + 1} ), ..., (n + k, m_{n + k}) \}. \]

We transmit messages \(m_1, ..., m_{n + 2k}\).

Suppose that the receiver receives all of them and at most \(k\) of the \(n + 2k\) messages are corrupted.

Conceptually, we can identify the \(n + k\) correct messages as follows:

  1. For \(S = \{ i_1, i_2, ..., i_{n + k} \} \subset [n + 2 k]\) do
  2. \(\qquad\) Pick arbitrary \(n\) indexes from \(S\).
  3. \(\qquad\) Let \(j_1, ..., j_n\) be the \(n\) indexes selected from \(S\).
  4. \(\qquad\) Compute a polynomial \(p(x)\) that visits \[ (j_1, m_{j_1}), ..., (j_n, m_{j_n} ). \]
  5. \(\qquad\) Check if it visits all points \[ (i_1, m_{i_1}), ..., (i_{n + k}, m_{i_{n + k} } ). \]
  6. \(\qquad\) If so, output \(S\).

We verify the correctness of the algorithm.

  1. If \(S\) is the set of correct messages, then we can construct \(f(x)\) by arbitrary \(n\) messages from \(S\).
  2. If not, then \(S\) contains at least \(n + k - k = n\) correct messages.
  • If the computed polynomial \(p(x)\) makes error on any of these \(n\) messages, \(S\) will not be outputted.
  • If \(p(x)\) doesn't make error on any of these \(n\) messages, then it is the unique polynomial determined by these messages and \(p(x) = f(x)\). Then it must make error on some message of \(S\). Hence, \(S\) will not be outputted.

\(\square\)

Reference
