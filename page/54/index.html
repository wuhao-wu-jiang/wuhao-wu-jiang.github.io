Farkas' Lemma

Posted on 2019-01-17 Edited on 2020-01-31

Given \(A \in R^{m \times n}\) and \(b \in R^n\), the Farkas Lemma states that exactly one of the following holds:

  1. \(\exists x \in R^n​ : Ax = b\), \(x \ge 0\),
  2. \(\exists y \in R^m : y^T A \le 0\), \(y^T b > 0\).

Geometrically, the first statements says that \(b\) is a non-negative linear combination of columns of \(A\). Denote the conic combination of columns of \(A\) as \(P = \{ Ax: \forall x \ge 0\} \subset R^m\). Note that it is a subset of the columns space of \(A\).

If \(b \notin P\), the second statement is equivalent to that

  1. \(\exists y \in R^m\), such that \(y^T b > 0\) and \(\forall z \in P\), we have \(y^T z \le 0\).

When (3) holds, clearly (2) holds, as columns of \(A\) belong to \(P\). On the other hand, every element in \(P\) is conic combination of columns of \(A\), then \(\forall z = Ax \in P\), then \(y^T z = y^T (A x) = (y^T A) x \le 0\) by the fact that \(y^T A \le 0 \wedge x \ge 0\).

The existence of \(y\) is justified as follows: as \(P\) is convex and closed, the Separation Theorem states that we can find a point \(z^* = A x^*\) in \(P\) that satisfies

\[ (b - z^*)^T(z - z^*) \le 0​ \]

Let \(y = (b - z^*) \in R^m\). Note that \(y > 0\) as \(b \notin P\) and \(P\) is closed. Consider the plane \(y^T (x - z^*) = 0\). It has the following properties:

  1. Points in \(P\) lies on the negative side of the plane, that is \(z \in P\), we have \(y^T (z - z^*) \le 0\).
  2. \(b\) is on the positive side of the plane. As \(y^T (b - z^*) = y^T y > 0\).
  3. It holds that \(y^T z^* = 0\), which implies that either \(z^* = 0\) or \(y\) is normal to \(z^*\). To verify this, we move along the line that pass through the vector \(z^*\) and note that for any \(k \ge 0\), we have \(kz^* \in P\), hence \(y^T (kz^* - z^*) \le 0\). Replace \(k\) with \(0\) and \(2\) respectively, we have the desired result.
  4. It concludes that \(z \in P\), \(y^T z \le 0\) and \(y^T b > 0\).

An equivalent form states that

  1. \(\exists x \in R^n : Ax \le b\)
  2. \(\exists y \in R^m : y^T A = 0\), \(y^T b > 0\), \(y \le 0\).

We can prove this by converting statement (2) into an equivalent one:

  1. \(\exists y \in R^m : y^T A = 0\), \(y^T b = 1\), \(y \le 0\).

and hence

  1. \(\exists y \in R^m : (-y)^T [A, b] = [0, -1]\), \(-y \ge 0\).

If this doesn't hold, by previous result, we have \(\exists (x, \lambda) \in R^{n + 1}\), s,t., \([A, b] (x, \lambda) = Ax + \lambda b \le 0\) and \([0, 1](x, -\lambda) = -\lambda > 0\), which implies that\(A \frac{x}{-\lambda} \le b\).

But this can also be verified via their geometrical meanings. We are not going to give all details here, just an example in \(R^2\):

Another equivalent form of Farkas Lemma is written as

  1. \(\exists x \in R^n : Ax \le b\), \(x \ge 0\).
  2. \(\exists y \in R^m : y^T A \le 0\), \(y^T b > 0\), \(y \le 0\).

Define \(P = \{ z \mid \exists x \in R^n : z \ge Ax, x \ge 0\} \subset R^m\).

  1. Statement 1 says \(b \in Z\).
  2. Otherwise, as \(P\) is convex and closed

    • \(z_1, z_2 \in P \rightarrow \exists x_1, x_2 \ge 0, z_1 \ge Ax_1, z_2 \ge Ax_2\), then \(\mu_1 z_1 + \mu_2 z_2 \ge A(\mu_1 x_1 + \mu_2 x_2), \forall \mu_1, \mu_2 \ge 0, \mu_1 + \mu_2 = 1\),

    • Closeness follows from \(\ge\) in the definition of \(P\). There is a unique point \(z^*\) that minimize \(||z - b||\) for \(z \in Z\). Let \(y = (b - z^*) \neq 0\). Then it holds \(y^T (b - z^*) = y^T y > 0\) and \(y^T(z - z^*) \le 0\) for any \(z \in Z\).

    • Similarly, \(z^* \in P \rightarrow \exists x \ge 0, z^* \ge Ax \rightarrow k \ge 0, k z^* \ge A (kx) \rightarrow kz \in P\). Then \(0 \in P\) and \(2z^* \in P\), which implies \(- y^T z^* \le 0\) and \(y^T z^* \le 0\). Hence \(y^T z^* = 0\) and \(y^T b = y^T y + y^T z^* > 0\).

    • Further, the fact \(z^* + e_i \in Z\) gives \(y^T e_i \le 0\) for arbitrary \(i\). Therefore \(y \le 0\). The \(y^T b < 0\) and \(y^T A \le 0\) follow from the same proof given above.

Complex Integral

Posted on 2018-12-13 Edited on 2024-08-27

We construct complex integral step by step in this article.

Riemann Sums

We first review the classical Riemann integral and see how this motivates complex integral. Given an interval \([a, b]\), the Riemann integral involves dividing an interval, denoted by \([a, b]\), into several smaller sub-intervals.

Definition. A finite sequence \(P = \{x_0, x_1, \ldots, x_n \}\) is a partition of \([a, b]\), if \[ a = x_0 < x_1 < \ldots < x_n = b. \] The length of \(P\), denoted as \(\Vert P \Vert\), is given by \[ \Vert P \Vert \doteq \max_{i \in [n] } \Delta_i \] where \(\Delta_i \doteq x_i - x_{i - 1}\) for each \(i \in [n]\).

Given a function \(f : [a, b] \rightarrow \R\), the Riemann integral computes the sum of the products of the length of each sub-interval and the value of \(f\) at an arbitrary chosen point on that sub-interval. Such a sum is well-defined if it converges when the chosen partition is "refined" enough.

Definition. A number \(J \in \R\) is the definite integral of \(f\) over \([a, b]\), if
\(\forall \epsilon > 0\), \(\exists \delta > 0\) s.t., for every partition \(P = \{x_0, x_1, \ldots, x_n \}\) of \([a, b]\) with \(\Vert P \Vert < \delta\), and \(\forall c_k \in [x_{k - 1}, x_k]\), it holds that
\[ \left| \sum_{k = 1}^n f(c_k) \Delta_k - J \right| < \epsilon \]

There is a dedicated notation for the limit of Riemann sum \[ \int_a^b f(x) dx = \lim_{\Vert P \Vert \rightarrow 0} \sum_{k = 1}^n f(c_k) \Delta_{k}. \]

Complex Integral

Complex integration is performed over curves, specifically over a set of curves that have favorable properties.

Smooth Curve

A point set \(\gamma\) in the \(\mathbb{C}\) is a ``smooth curve'', if it is either a smooth arc or a smooth closed curve, defined as thus.

Definition. A point set \(\gamma\) in \(\mathbb{C}\) is said to be a smooth arc if it is the range of some continuous complex-valued function \(z = z(t), t \in [a, b]\), s.t.,

  1. \(z(t)\) has a continuous derivative on \([a, b]\),
  2. \(z(t) \neq 0, \forall t \in [a, b]\),
  3. \(z(t)\) is one-to-one on \([a, b]\).

A point set \(\gamma\) is called a smooth closed curve if it is the range of some continuous function \(z = z(t), t \in [a, b]\), satisfying conditions (1) and (2) and the following:

  • \(z(t)\) is one-to-one on the half-open interval \([a, b)\), but \(z(b) = z(a)\) and \(z' (b) = z' (a)\).

Remark: in general, we refer to a function \(z = z(t)\), where \(t \in [a, b]\), as "admissible" if it satisfies the conditions (1), (2), and (3) / (\(\cdot\)).

To explain the definition, let us consider the scenario where a curve \(\gamma\) is being drawn on a piece of paper using a pen. The position of the pen at each time \(t\in [a, b]\) is given by the function \(z(t)\), which parameterizes the curve.

To ensure that the curve is smooth, we must adhere to certain guidelines:

  1. We must not lift the pen from the paper during the drawing process, which implies that \(z(\gamma_t)\) must be continuous.

  2. The velocity of the pen point must be well-defined and finite, and it must also vary continuously. Therefore, \(z'(t)\) must exist and be continuous on the interval \([a, b]\).

  3. To avoid sharp corners in the curve (which would require a sudden break in the drawing process), we require that \(z'(t)\) is never zero.

  4. We also require that no point on the curve is drawn more than once, except for the initial and terminal points, which may coincide. In other words, \(z(t)\) must be one-to-one, except for the endpoints \(a\) and \(b\).

In order to demonstrate that a collection of points \(\gamma\) in the complex plane \(\mathbb{C}\) forms a smooth curve, we must present an admissible parametrization function \(z(t)\) whose range is \(\gamma\). There can be infinite number of such parametrizations if \(\gamma\) is smooth.

Directed Smooth Curve

Contour

Definition. A contour is either a single point \(z_0\) or a finite sequence of directed smooth curves \((\gamma_1, \gamma_2,\ldots,\gamma_n)\) such that the terminal point of \(\gamma_k\) coincides with the initial point of \(\gamma_{k+1}\) for each \(k = 1, 2, \ldots, n − 1\). In this case one can write \(\Gamma = \gamma_1 + \gamma_2 + \cdots + \gamma_n.\)

The (undirected) point set underlying a contour is known as a piecewise smooth curve

is said to be a closed contour or a loop if its initial and terminal points coincide. A simple closed contour is a closed contour with no multiple points other than its initial-terminal point; in other words, if \(z = z(t), a \le t \le b\), is a parametrization of the closed contour, then \(z(t)\) is one-to-one on the half-open interval \([a, b)\)

Contour Integral

We are ready to define the definite integral of a complex-valued function \(f\) along a contour \(\Gamma\).

Directed Smooth Curve

We start by defining the integral along a directed smooth curve \(\gamma\) (with initial point \(\alpha\) and terminal point \(\beta\)), and then defining integral along a contour in terms of the integrals along its smooth components.

As in the construction of the Riemann integrals, we repeat the partition and product idea: let \(P = \{z_0, z_1, \ldots, z_n\} \subseteq \gamma\) be a partition of the curve such that \(z_0 = \alpha\) and \(z_n = \beta\). For each \(k \in [n]\), define \(\Delta_k \doteq z_k - z_{k - 1}\), and let mesh of partition \(P\) be the maximum length of the \(\Delta_k\): \[ \Vert P \Vert \doteq \max_{k \in [n]} \Vert \Delta_k \Vert. \]

A Riemann sum associated with \(P\) is just what it is in the real case: \[ \sum_{k = 1}^n f(d_k) \cdot \Delta_{k}, \] where \(d_k\) is an arbitrary point on the arc \(\widehat{z_{k - 1}, z_k}\).

Definition. A function \(f: \gamma \rightarrow \mathbb{C}\) is integrable over a directed smooth curve \(\gamma\), if \(\exists J \in \mathbb{C}\), s.t., \(\forall \epsilon > 0\), \(\exists\delta > 0\) s.t., for every partition \(P\) with \(\Vert P \Vert < \delta\) and every choice of \(d_k\) in \(\widehat{z_{k - 1}, z_k}\), it holds that \[ \left| \sum_{k = 1}^n f(d_k) \Delta_{k} - J \right| < \epsilon. \]

We call \(J\) the integral of \(f\) over \(\gamma\), and write it as \[ \int_\gamma f(x) dx \doteq J = \lim_{\Vert P \Vert \rightarrow 0} \sum_{k = 1}^n f(d_k) \Delta_{k}. \]

Properties

Before establishing conditions under which the path integral exists, we study its properties directly, via its definition.

Linearity. 1. Constant Multiple: \(\int_\gamma c f(x) dx = c \int_\gamma f(x) dx\).

  1. Sum: \(\int_\gamma f(x) + g(x) dx = \int_\gamma f(x) dx + \int_\gamma g(x) dx\).

Proof. For every partition \(P\) of \(\gamma\) and for every choice of the points \(d_k\),
\[ \begin{aligned} \sum_{k \in [n]} c f( d_k ) \Delta_k &= c \sum_{k \in [n]} f( d_k ) \Delta_k, \\ \sum_{k \in [n]} (f( d_k ) + g( z_k ) )\Delta_k &= \sum_{k \in [n]} f( d_k ) \Delta_k + \sum_{k \in [n]} g( z_k ) \Delta_k. \end{aligned} \] Taking limit of both side, we get the desired answer.

Remark: given two sequence \(\{ a_1, a_2, \ldots, \}\) and \(\{b_1, b_2, \ldots, \}\), if \(a_i = b_i\) for arbitrary \(i \ge 0\) and \(\lim_i b_i = S\), then \(\lim_i a_i\) exits and \(\lim_i a_i = S\).

\(\square\)

Existence

We provide a sufficient condition under which the path integral exists.

Lemma. Let \(\gamma\) be a directed smooth curve, and \(r: [a, b] \rightarrow \mathbb{C}\) be a parameterization of \(\gamma\). For each partition \(P = \{z_0, \ldots, z_n\}\) of \(\gamma\), define the inverse partition of \([a, b]\) as \[ r^{-1} (P) \doteq \{ r^{-1} (z_0), \ldots, r^{-1} (z_n) \}. \] Then \[ \Vert P \Vert \rightarrow 0 \qquad \text{implies} \qquad \Vert r^{-1}(P) \Vert \rightarrow 0. \]

Remark: When \(\gamma\) is a directed smooth arc, then \(r\) is a bijection and \(r^{-1}(z)\) is well-defined for each \(z \in \gamma\). But when \(\gamma\) is a directed smooth closed curve, then \(r\) is no longer a bijection, since \(r(a) = r(b)\). But there is still a unique partition \((t_0, \ldots, t_n) \subseteq [a, b]\), s.t., \(a = t_0 \le \cdots \le t_n = b\) and \(r(t_i) = z_i,\) for each \(i \in [0 \, . \, . \, n]\). In this case, \(r^{-1}(z_i) \doteq t_i,\) for each \(i \in [0 \, . \, . \, n]\). Therefore, \(r^{-1}(P)\) is well-defined.

Proof. Since \(r\) is continuous on \([a, b]\), it is uniformly continuous.

\(\square\)

Theorem. If \(f: \gamma \rightarrow \mathbb{C}\) is continuous on the directed smooth curve \(\gamma\), then \(f\) is integrable over \(\gamma\) and \[ \int_\gamma f (x) \, d x = \int_a^b f \big( r(t) \big) \, r'(t) \, d t, \] where \(r : [a, b] \rightarrow \mathbb{C}\) is an parameterization of \(\gamma\).

Proof. We first prove that \(f (r (t)) \cdot r'(t)\) is integrable, so the RHS of the equation is well-defined. Then we show that \(\sum_{k = 1}^n f(d_k) \Delta_{k}\) converges to the RHS as \(\Vert P \Vert \rightarrow 0\). Hence the limit \(\lim_{\Vert P \Vert \rightarrow 0} \sum_{k = 1}^n f(d_k) \Delta_{k}\) exists.

Integrability of \(f (r (t)) \cdot r'(t)\):

  • Observe that \(f \circ r (t) = f(r(t)): [a, b] \rightarrow \mathbb{C}\) is continuous.

  • By assumption \(r'(t) : [a, b] \rightarrow \mathbb{C}\) is continuous.

  • Therefore, \(f (r (t)) \cdot r'(t) : [a, b] \rightarrow \mathbb{C}\) is continuous.

  • Both the real and imaginary parts of \(f (r (t)) \cdot r'(t)\) are continuous, and hence integrable.

  • By definition of line integral, \[ \int_a^b f \big( r(t) \big) \, r'(t) \, d t \doteq \int_a^b \mathbb{Re} \Big( f \big( r(t) \big) \, r'(t) \Big) \, d t + i \cdot \int_a^b \mathbb{Im} \Big( f \big( r(t) \big) \, r'(t) \Big) \, d t \] is well-defined.

Proving The Equation:

Claim One. For each \(z \in \gamma\), there is a unique point \(t \in [a, b]\) for which \(r(t) = z\).

Proof of the Claim One. \(\square\)

We continue to prove the equation. Let \(P = \{z_0, z_1, \ldots, z_n\}\) be a partition of \(\gamma\). By Claim One, there exists unique points \(Q = \{ a = t_0 < t_1< \ldots < t_n = b \}\), s.t., \(r(k) = z_k\), for \(k \in [0 .\, .\, n]\).

\(\square\)

To compute such integral, we can choose a description $(t) = x(t) + i y(t) : \([a, b] \rightarrow \mathbb{C}\) of curve \(\gamma\). We partition \(\gamma\) by partitioning the interval \([a, b]\) in usual way \(P_t = \{ a = t_0 < t_1< \ldots < t_n = b \}\). Then \(\gamma(P_t) \doteq \{ \gamma(t_0), \gamma(t_1), \ldots, \gamma(t_n) \}\) is a partition of \(\gamma\).

If \(\gamma\) is continuous, it is uniformly continuous on \([a, b]\). Therefore, as \(\Vert P_t\Vert \rightarrow 0\), \(\Vert \gamma(P_t)\Vert \rightarrow 0\). If \(\int_\gamma f(z) dz\) does exist, then

\[ \int_\gamma f(z) dz = \lim_{\Vert P_t\Vert \rightarrow 0} \sum_{j = 1}^n f \big( \gamma( e_j ) \big) \big(\gamma(t_j) - \gamma(t_{j - 1} ) \big) \] for arbitrary choice \(e_j \in [t_{j - 1}, t_j]\).

When the derivative of \(\gamma\) exists and is continuous, it is uniformly continuous on the closed bounded interval \([a, b]\). We now can rewrite the Riemann sum as
\[ \begin{aligned} &\sum_{j = 1}^n f \big( \gamma( e_j ) \big) \frac{\gamma(t_j) - \gamma(t_{j - 1} ) }{t_j - t_{j - 1} }(t_j - t_{j - 1} ) \\ &= \sum_{j = 1}^n f \big( \gamma( e_j ) \big) \frac{x(t_j) - x(t_{j - 1} )+ i (y(t_j) - y_(t_{j - 1} ) ) }{t_j - t_{j - 1} }(t_j - t_{j - 1} ) \end{aligned} \]

By mean value theorem, as \(x'(t)\) and \(y'(t)\) exist, there are some \(e_j' \in [t_{j - 1}, t_j]\) and \(e_j'' \in [t_{j - 1}, t_j]\), such that

\[ \frac{x(t_j) - x(t_{j - 1} ) }{t_j - t_{j - 1} } = x'(e_j'), \qquad \frac{y(t_j) - y(t_{j - 1} ) }{t_j - t_{j - 1} } = x'(e_j''). \]

On the other hand, as \(x'(t)\) and \(y'(t)\) are uniformly continuous, for arbitrary \(\epsilon > 0\), there is a \(\delta\) such that if \(\Vert P_t\Vert < \delta\), \(|x'(e_j') - x'(e_j)| < \epsilon\) and \(|y'(e_j'') - y'(e_j)| < \epsilon\). It follows that
\[ \begin{aligned} \frac{x(t_j) - x(t_{j - 1} )+ i (y(t_j) - y_(t_{j - 1} ) ) }{t_j - t_{j - 1} } &= x'(e_j) + i y'(e_j) \pm \epsilon(1 + i)\\ \sum_{j = 1}^n f \big( \gamma( e_j ) \big) \frac{\gamma(t_j) - \gamma(t_{j - 1} ) }{t_j - t_{j - 1} }(t_j - t_{j - 1} ) &= \sum_{j = 1}^n f \big( \gamma( e_j ) \big) \gamma'(e_j) (t_j - t_{j - 1} ) \\ &\quad \pm \epsilon (1 + i) \cdot \sum_{j = 1}^n f \big( \gamma( e_j ) \big) (t_j - t_{j - 1} ), \end{aligned} \]

Taking the limit of \(\Vert P_t\Vert \rightarrow 0\), we get
\[ \int_\gamma f(z) dz = \int_a^b f(\gamma(t) )\gamma'(t) dt, \]

Remark 1: Can we relax the condition that \(\gamma'(t)\) is continuous and that \([a, b]\) is a closed interval?

Remark 2: We have also showed that the integration is independent of the choice of \(\gamma\), i.e., any representation of \(\gamma\) that is continuously differentiable gives the same integration value.

Anti-Derivative

Now suppose that \(F(z) = u(x, y) + i v(x, y)\) and \(F'(z) = f(z)\).

Theorem. \(\int_\gamma f(z) dz = F(\gamma(b) ) - F(\gamma(a) )\).

First, \(F'(\gamma(t) ) = \frac{\partial u}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial u}{\partial y} \frac{\partial y}{\partial t} + i ( \frac{\partial v}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial v}{\partial y} \frac{\partial y}{\partial t} )\). Therefore,
\[ \int_\gamma f(z) dz = \int_a^b \left(\frac{\partial u}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial u}{\partial y} \frac{\partial y}{\partial t} \right) \, dt + i \int_a^b \left( \frac{\partial v}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial v}{\partial y} \frac{\partial y}{\partial t} \right) \, dt \]

But
\[ \begin{aligned} &u(x(b), y(b) ) - u(x(a), y(a) ) \\ &= \sum_{j = 1}^n [u(x(t_j), y(t_j) ) - u(x(t_{j - 1} ), y(t_{j - 1} ) ) ] \\ &= \sum_{j = 1}^n [u(x(t_j), y(t_j) ) - u(x(t_j), y(t_{j - 1} ) ) + u(x(t_j), y(t_{j - 1} ) ) - u(x(t_{j - 1} ), y(t_{j - 1} ) ) ] \\ &= \sum_{j = 1}^n \frac{u(x(t_j), y(t_j) ) - u(x(t_j), y(t_{j - 1} ) ) }{y(t_j) - y(t_{j - 1} ) } \frac{y(t_j) - y(t_{j - 1} ) }{ t_j - t_{j - 1} }(t_j - t_{j - 1} )\\ &\quad +\sum_{j = 1}^n \frac{u(x(t_j), y(t_{j - 1} ) ) - u(x(t_{j - 1} ), y(t_{j - 1} ) ) }{x(t_j) - x(t_{j - 1} ) } \frac{x(t_j) - x(t_{j - 1} ) }{t_j - t_{j - 1} } (t_j - t_{j - 1} ) \\ &= \sum_{j = 1}^n [u_y' (x(t_j), y^*_j) y'(e_j') + u_x' (x^*_j, y(t_{j - 1} ) ) x'(e_j'') ] (t_j - t_{j - 1} )\\ \end{aligned} \]

where \(y^*_j \in [ y( t_{j - 1} ), y( t_j ) ]\), \(x^*_j \in [ x( t_{j - 1} ), x( t_j ) ]\), \(e_j', e_j'' \in [ t_{j - 1}, t_j ]\). If \(u'_x, u'_y, x'(t), y'(t)\) are continuous, then as \(\Vert P_t\Vert \rightarrow 0\), \[ \begin{aligned} &\sum_{j = 1}^n [u_y' ( x( t_j ), y( t_j ) ) y'( t_j ) + u_x' ( x( t_j ), y( t_j ) ) x'( t_j ) ] ( t_j - t_{j - 1} ) \\ &= \sum_{j = 1}^n \frac{\partial u}{\partial t} ( t_j - t_{j - 1} ) \rightarrow u(x(b), y(b) ) - u(x(a), y(a) ). \end{aligned} \]

An immediate consequence is that
\[ F(\gamma(b) ) - F(\gamma(a) ) = \int_\gamma F'(\gamma(t) ) dt \]

On the other hand, since \(F(z)\) is differential, \[ \lim_{h \rightarrow 0} \frac{F(z + h) - F(z) }{h} = \frac{\partial u}{\partial x} + i \frac{\partial v}{\partial x} = \lim_{ih \rightarrow 0} \frac{F(z + ih) - F(z) }{ih} = -i \frac{\partial u}{\partial y} + \frac{\partial v}{\partial y}. \]

It holds that \(\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}\) and \(\frac{\partial v}{\partial x} = -\frac{\partial u}{\partial y}\). Therefore,
\[ \begin{aligned} F'(\gamma(t) ) &= \frac{\partial u}{\partial x} \frac{\partial x}{\partial t} - \frac{\partial v}{\partial x} \frac{\partial y}{\partial t} + i \left( \frac{\partial v}{\partial x} \frac{\partial x}{\partial t} + \frac{\partial u}{\partial x} \frac{\partial y}{\partial t} \right) \\ &= \left( \frac{\partial u}{\partial x} + i \frac{\partial v}{\partial x} \right) \left( \frac{\partial x}{\partial t} + i \frac{\partial y}{\partial t} \right) \\ &= F'(\gamma(t) ) \gamma'(t) \\ &= f(\gamma(t) ) \gamma'(t). \end{aligned} \]

and
\[ F(\gamma(b) ) - F(\gamma(a) ) = \int_\gamma f(\gamma(t) ) \gamma'(t) dt \]

Remark: That \(u_x', u_y'\) are continuous is a strong condition. What we need in the proof is just that *

\[ \begin{aligned} u(x(t_j), y(t_j) ) &= u(x(t_{j - 1} ), y(t_{j - 1} ) ) \\ &+ u_x'(x(t_{j - 1} ), y(t_{j - 1} ) ) (x(t_j) - x(t_{j - 1} ) ) \\ &+ u_y'(x(t_{j - 1} ), y(t_{j - 1} ) ) (y(t_j) - y(t_{j - 1} ) ) \\ &+ o(x(t_j) - x(t_{j - 1} ), y(t_j) - y(t_{j - 1} ) ) \end{aligned} \]

or

\[ \begin{aligned} u(x(t_{j - 1} ) + \delta_x, y(t_{j - 1} ) + \delta_y) &= u(x(t_{j - 1} ), y(t_{j - 1} ) ) \\ &+ u_x'(x(t_{j - 1} ), y(t_{j - 1} ) ) \delta_x \\ &+ u_y'(x(t_{j - 1} ), y(t_{j - 1} ) ) \delta_y \\ &+ o(\delta_x, \delta_y) \end{aligned} \]

i.e., \(u\) is differentiable at every point in the domain.

Question: Is is possible that \(u\) is differentiable but its partial derivatives are not continuous? The answer is NO. First we consider an one dimension counter example:
\[ f(x) = \begin{cases} x^2 \sin \frac{1}{x} \quad \forall x \neq 0, \\ 0, \quad if \quad x = 0 \end{cases} \] To show \(f(x)\) is differential at \(0\), we use the definition of derivative:
\[ f'(0) = \lim_{h \rightarrow } \frac{f(h) - f(0) }{h - 0} = \lim_{h \rightarrow 0} h \sin \frac{1}{h} = 0 \] On the other hand, \(f(x)\) is always differentiable from \(0\):
\[ f'(x) = 2x \sin \frac{1}{x} + x^2 ( -1 / x^2) cos \frac{1}{x} = 2x \sin \frac{1}{x} -cos \frac{1}{x} \] Hence, \(f(x)\) is differentiable everywhere, but \(f'(x)\) is not continuous at \(0\).

The above example can be easily extended to two dimension case:
\[ f(x, y) = \begin{cases} (x^2 + y^2) \sin \frac{1}{ \sqrt{x^2 + y^2} } \quad \forall (x, y) \neq (0, 0) \\ 0 \qquad \qquad \qquad \qquad \quad if \quad (x, y) = (0, 0) \end{cases} \]

By definition, \(f(x, y)\) is differential at \((0, 0)\),
\[ \lim_{\Vert (x, y)\Vert \rightarrow 0} \frac{\Vert f(x, y) - f(0) - (0,0) \cdot (x, y)\Vert }{\Vert (x, y)\Vert } \le \lim_{\Vert (x, y)\Vert \rightarrow 0} \frac{\Vert x^2 + y^2\Vert }{\Vert (x, y)\Vert } = \lim_{\Vert (x, y)\Vert \rightarrow 0} \Vert (x, y)\Vert = 0 \]

However, it partial derivative from \((0, 0)\) \[ \begin{aligned} \frac{\partial f}{\partial x} &= 2x \sin \frac{1}{ \sqrt{x^2 + y^2} } + (x^2 + y^2) \left( -\frac{ 2x }{ \sqrt{ x^2 + y^2 }^{3} } \right) \cos \frac{1}{ \sqrt{x + y} } \\ &= 2x \sin \frac{1}{ \sqrt{x^2 + y^2} } - \frac{ 2x }{ \sqrt{ x^2 + y^2 } } \cos \frac{1}{ \sqrt{x^2 + y^2 } } \end{aligned} \] and
\[ \frac{\partial f}{\partial x}(0, 0) = \lim_{h \rightarrow} \frac{h^2 \sin \frac{1}{|h|} }{h} = 0 \]

Therefore, \(\partial f / \partial x\) is not continuous.

Line Integral

How do we extend integral in real line to \(\mathbb{C}\). The most simplest extension is of a real to complex function \[ \gamma(t) = x(t) + i y(t), \] whose integral on an interval is defined by integrating its real part and imaginary parts separately.

Definition. Let \(\gamma(t) = x(t) + i y(t)\) be a function defined on \([a, b]\). If both \(x(x)\) and \(y(t)\) are Riemann integrable, then the integral of \(\gamma\) over \([a, b]\) is defined as \[ \int_a^b \gamma(t) \, dt \doteq \int_a^b x(t) dt + i\int_a^b y(t) dt, \]

Separation Plane Theorem

Posted on 2018-12-01 Edited on 2022-04-29

Separation Theorem. Given a convex and closed set \(S \subset \mathbb{R}^d\) and a point \(p \notin S\), there is a separation plane between \(S\) and \(p\).

Proof. Define \(\delta = \min \{ \Vert s - p \Vert : s \in S \}\), where \(\Vert \cdot \Vert\) is the \(\ell_2\) norm.

Lemma. There is a point \(s^\star \in S\) that achieves \(\Vert s^\star - p \Vert = \delta\).

Proof:

  1. Pick arbitrary point \(s \in S\) and let \(r = \Vert s - p \Vert\).
  2. Define \(S' = S \cap \{\Vert x - p \Vert \le r \}\). Observe that \(S'\) is nonempty, bounded, and closed.
  3. Define a function \(f: S' \rightarrow R\) as \(f(s) = \Vert s - p \Vert\), which is a continuous.
  4. Since \(S'\) is bounded and closed, there exists a point \(s^\star\) that minimizes \(f(s)\).
  5. We claim that \(f(s^\star) = \delta\). Otherwise, there is a point \(s'' \in S\), s.t., \(f(s^\star) > \Vert s'' - p\Vert > \delta\). But this implies \(s'' \in S'\), a contradicting \(s^\star = \min_{s \in S'} f(s)\).

\(\square\)

Lemma. \(s^\star\) is unique.

Proof: The uniqueness results from the convexity of \(S\). If not, let \(s''\) be another optimal point. Now \(\Vert q - s'\Vert = \delta\) and \(\Vert q - s''\Vert = \delta\). The middle point between \(s'\) and \(s''\) is clearly closer to \(p\) than both \(s'\) and \(s''\), as
\[ \Vert p - (s' + s'') / 2\Vert = \Vert (p - s') / 2 + (p - s'') / 2\Vert \le 1/2 \Vert p - s'\Vert + 1/2 \Vert p - s''\Vert = \delta \] The inequality is tight if \(s' \neq s''\). Therefore, we use \(s^\star\) to indicate the unique point.

\(\square\)

Lemma. \((p - s^\star)^T(x - s^\star) = 0\) is a separation plane between \(S\) and \(p\).

Proof.

First, observe that \((p - s^\star)^T(p - s^\star) = \delta^2 > 0\) since \(p \notin S\) and \(S\) is closed.

Further, for each \(s \in S​\), let * \(a = s^\star - p\), and * \(b = s - s^\star\).

We claim that \(\left< b, a \right> \ge 0\).

For all \(\lambda \in (0, 1)​\), by convexity, \(\lambda \cdot (s - s^\star) + s^\star \in S​\). By the definition of \(s^\star​\), \[ \Vert \lambda(s - s^\star) + s^\star - p\Vert ^2 \ge \Vert p - s^\star\Vert ^2 = \Vert a \Vert ^2. \] On the other hand, \[ \Vert \lambda(s - s^\star) + s^\star - p\Vert ^2 = \lambda^2 \Vert b \Vert ^2 + 2 \lambda \left< b, a \right> + \Vert a \Vert ^2 \] Therefore, \(\lambda^2 \Vert b \Vert ^2 + 2 \lambda \left< b, a \right> \ge 0\). As \(\lambda\rightarrow 0\), we obtained the desired result.

\(\square\)

Corollary. Given a convex and closed set \(S \subset \mathbb{R}^d\) and a point \(p \in \partial S\) (the boundary of \(S\)), there is supporting hyperplane passing through \(p\) and containing \(S\) in one of its half-spaces.

Proof. Since \(p \in \partial S\), there is a countable, bounded sequence of points \(p_k\) such that \(p_k \notin S\) and \(p_k \rightarrow p\).

Let \(s_k\) be the projection of \(p_k\) to \(S\). Clearly \(\Vert s_k - p_k \Vert \le \Vert p - p_k \Vert\). By triangle inequality, \[ \Vert s_k - p \Vert \le \Vert s_k - p_k \Vert + \Vert p_k - p \Vert \le 2 \Vert p - p_k \Vert \rightarrow 0. \] It follows that \(s_k \rightarrow p\).

For each \(p_k\), we can find a supporting hyperplane \(f_k (x) = \left< a_k, x - p_k \right>\), such that 1. \(\Vert a_k \Vert \neq 0\) and, 2. \(\left< a_k, s - s_k \right> \ge 0\) for all \(s \in S\), where \(s_k\) is the projection of \(p_k\) to \(S\).

Replacing \(a_k\) by \(a_k / \Vert a_k \Vert\), we can assume without lose of generality that \(\Vert a_k \Vert = 1\).

The sequence \(\{ a_k \}\) is therefore bounded. There is a converging subsequence \(\{ a_\ell \}\) of \(\{ a_k \}\) such that \(a_\ell \rightarrow a\).
Finally, \[ \lim \left< a_\ell, s - s_\ell \right> = \left< a, s - p\right> \ge 0. \]

\(\blacksquare\)
