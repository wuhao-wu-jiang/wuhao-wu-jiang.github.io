Δ≐max⁡x∼x′∣q(x)−q(x′)∣ \Delta \doteq \max_{x \sim x' } | q(x) - q(x') | Δ≐x∼x′max​∣q(x)−q(x′)∣

The Laplace mechanism MMM of qqq adds random noise N∼Lap(Δε)N \sim \mathbb{Lap}(\frac{\Delta}{\varepsilon})N∼Lap(εΔ​) to the output of qqq:

where Lap(Δε)\mathbb{Lap}(\frac{\Delta}{\varepsilon})Lap(εΔ​) is the Laplacian distribution with parameter Δε\frac{\Delta}{\varepsilon}εΔ​.
It guarantees that if x∼x′x \sim x'x∼x′, the output distributions of M(x)M(x)M(x) and M(x′)M(x')M(x′) are indistinguishable in the sense that for all S⊂RS \subset \mathbb{R}S⊂R, s.t. SSS is measurable, it holds that

Pr⁡[M(x)∈S]∈[e−ε,eε]⋅Pr⁡[M(x′)∈S] \Pr[ M(x) \in S] \in [ e^{ - \varepsilon }, e^{ \varepsilon } ] \cdot \Pr[ M(x') \in S] Pr[M(x)∈S]∈[e−ε,eε]⋅Pr[M(x′)∈S]

The value of Δ\DeltaΔ could be large and is independent of an x∈Xx \in \mathcal{X}x∈X. We would like to investigate the possibility of adding a smaller noise than Lap(Δε)\mathbb{Lap}(\frac{\Delta}{\varepsilon})Lap(εΔ​).

For a fixed xxx, we can define the local sensitivity with respect to xxx as

δ(x)≐max⁡x′∼x∣q(x)−q(x′)∣. \delta(x) \doteq \max_{x' \sim x} | q(x) - q(x') |. δ(x)≐x′∼xmax​∣q(x)−q(x′)∣.

Here the maximum is over the neighbors of a fixed xxx. The value between Δ\DeltaΔ and δ(x)\delta(x)δ(x) is quite different.

Example. Let

  • X≐[0,100]3\mathcal{X} \doteq [0, 100]^3X≐[0,100]3.
  • d(x,x′)≐# different values between x and x′d(x, x')\doteq \#\text{ different values between } x \text{ and } x'd(x,x′)≐# different values between x and x′.
  • q(x)≐min⁡1≤i<j≤3∣xi−xj∣q(x) \doteq \min_{1 \le i < j \le 3} |x_i - x_j|q(x)≐min1≤i<j≤3​∣xi​−xj​∣, where xix_ixi​ is the iii-dimension of xxx.

Then for the point x=(0,0,0)∈[0,100]3x = (0, 0, 0) \in [0, 100]^3x=(0,0,0)∈[0,100]3, if x′∼xx' \sim xx′∼x, x′x'x′ contains at least two zeros in some of its dimensions. Therefore, q(x′)=0q(x') = 0q(x′)=0 and δ(x)=0\delta(x) = 0δ(x)=0. But for x=(0,0,100)x = (0, 0, 100)x=(0,0,100), it has a neighbor x′=(0,50,100)x' = (0, 50, 100)x′=(0,50,100). Now q(x)=0q(x) = 0q(x)=0 but q(x′)=50q(x') = 50q(x′)=50. So Δ≥δ(x)≥50\Delta \ge \delta(x) \ge 50Δ≥δ(x)≥50.

We are interested in whether adding smaller noise N∼Lap(δ(x)ε)N \sim \mathbb{Lap}(\frac{\delta(x) }{\varepsilon})N∼Lap(εδ(x)​) to the output of q(x)q(x)q(x) would preserve privacy:

M(x)=q(x)+N. M(x) = q(x) + N. M(x)=q(x)+N.

However, this approach is not private.
In the previous example, (0,0,0)(0, 0, 0)(0,0,0) and (0,0,100)(0, 0, 100)(0,0,100) are neighbors.
But for the former we add noise N∼Lap(0)N \sim \mathbb{Lap}(0)N∼Lap(0) and for the latter we add noise N∼Lap(50ε)N \sim \mathbb{Lap}(\frac{50}{\varepsilon})N∼Lap(ε50​).
So the output distributions of them are very different and this would violate privacy protection.

Perhaps, we should propose some fixed parameter β>0\beta > 0β>0 (which could be much smaller than Δ\DeltaΔ), and add a noise N∼Lap(βε)N \sim \mathbb{Lap}(\frac{\beta}{\varepsilon})N∼Lap(εβ​) for whatever x∈Xx \in \mathcal{X}x∈X.
For a pair of neighboring x,x′x, x'x,x′, when ∣q(x)−q(x′)∣≤β|q(x) - q(x')| \le \beta∣q(x)−q(x′)∣≤β, then the distributions of q(x)+Nq(x) + Nq(x)+N and q(x′)+Nq(x') + Nq(x′)+N are indeed (ε,0)(\varepsilon, 0)(ε,0) indistinguishable.
The only concern is for the case where ∣q(x)−q(x′)∣>β|q(x) - q(x')| > \beta∣q(x)−q(x′)∣>β and adding a noise N∼Lap(βε)N \sim \mathbb{Lap}(\frac{\beta}{\varepsilon})N∼Lap(εβ​) fails to protect the privacy.

One simple remedy is simply to refuse to output q(x)+Nq(x) + Nq(x)+N (or q(x)+N′q(x) + N'q(x)+N′). This should be done in a differentially private manner. Let

{y∈X:δ(y)>β}\{ y \in \mathcal{X} : \delta(y) > \beta \} {y∈X:δ(y)>β}

be the set of points with local sensitivity greater than β\betaβ. For a fixed xxx, define d(x,{y:δ(y)>β})d(x, \{ y : \delta(y) > \beta \})d(x,{y:δ(y)>β}) be its distance to the set. It follows that for x∼x′x \sim x'x∼x′,

∣d(x,{y∈X:δ(y)>β})−d(x′,{y∈X:δ(y)>β})∣≤1.|d(x, \{ y \in \mathcal{X} : \delta(y) > \beta \}) - d(x', \{ y \in \mathcal{X} : \delta(y) > \beta \})| \le 1. ∣d(x,{y∈X:δ(y)>β})−d(x′,{y∈X:δ(y)>β})∣≤1.

This motivates the following propose-test-release algorithm.

  1. Propose a bound β\betaβ on local sensitivity.
  2. Compute d^≐d(x,{y∈X:δ(y)>β})+Lap(1/ε)\hat d \doteq d(x, \{ y \in \mathcal{X} : \delta(y) > \beta \}) + \mathbb{Lap}(1 / \varepsilon)d^≐d(x,{y∈X:δ(y)>β})+Lap(1/ε).
  3. If d^≤1εln⁡1γ\hat d \le \frac{1}{\varepsilon} \ln \frac{1}{\gamma}d^≤ε1​lnγ1​ then
    \qquad output ⊥\bot⊥.
  4. Else
    \qquad output q(x)+Lap(β/ε)q(x) + \mathbb{Lap}(\beta / \varepsilon)q(x)+Lap(β/ε).

Proof.

First observe that for two neighboring x,x′x, x'x,x′, their probabilities of outputting ⊥\bot⊥ are similar.
