Basic Entropy Inequalities

Posted on 2021-07-07 Edited on 2022-01-17

Let \(X\) and \(Y\) be two discrete random variables.

Relative Entropy

Given a distribution \(p\), if we encode it with a distribution \(q\) (with the same support), the expected coding length is roughly \[ \sum_i p_i \log \frac{1}{q_i} \]

The cost over the optimal coding is given by \[ \mathbf{D} (p || q) \doteq \sum_i p_i \log \frac{1}{q_i} - \sum_i p_i \log \frac{1}{p_i} = \sum_i p_i \log \frac{p_i}{q_i} \]

By definition, we know that this gap is always non-negative. We can prove it rigorously algebraically. We show two different approaches here.

  1. By that \(f(x) = x \log x\) is convex for \(x \ge 0\), we have \[ \begin{aligned} \mathbf{D}(p || q) &= \sum_i p_i \log \frac{p_i}{q_i} \\ &= \sum_i q_i \cdot \frac{p_i}{q_i} \log \frac{p_i}{q_i} \\ &\ge \left( \sum_i q_i \cdot \frac{p_i}{q_i} \right) \log \left( \sum_i q_i \cdot \frac{p_i}{q_i} \right) \\ &= 0 \end{aligned} \]
  2. By that \(f(x) = -\log x\) is convex (for \(x \ge 0\) ), we have \[ \begin{aligned} \mathbf{D}(p || q) &= \sum_i - p_i \log \frac{q_i}{p_i} \\ &\ge -\log \left( \sum_i p_i \cdot \frac{q_i}{p_i} \right) \\ &= 0 \end{aligned} \]

Theorem (Maximum Value). Let \(X\) be a random variable taking values in an alphabet \(\Omega\). Then \[ \mathbf{H}(X) \le \log | \Omega | \] The inequality is saturated if and only if \(X\) is a uniform random variable on \(\Omega\).

Proof. Let \(U\) be a uniform random variable on \(\Omega\). Then \[ \mathbf{D}( X \Vert U ) = \sum p(x) \log \frac{ p(x) }{ 1 / | \Omega | } = \log | \Omega | - \mathbf{H}(x) \ge 0. \]

\(\square\)

Joint Entropy

Definition. The joint entropy of \(X\) and \(Y\) is \[ \mathbf{H}(X, Y) = \sum_{x, y} p(x, y) \log \frac{1}{ p(x, y) } \]

If \(X\) and \(Y\) are independent, then \[ \mathbf{H}(X, Y) = \mathbf{H}(X) + \mathbf{H}(Y). \]

Conditional Entropy

Definition. The conditional entropy of \(X\) given \(Y = y\) is the entropy of the probability distribution \(p(X \mid Y = y)\) \[ \mathbf{H}(X \mid Y = y) = \sum_x p(x \mid y) \log \frac{1}{ p(x \mid y) } \]

Definition. The conditional entropy of \(X\) given \(Y\), is the average, over \(y\), the conditional entropy of \(X\) given \(y\) \[ \mathbf{H}(X \mid Y) = \sum_y p(y) \mathbf{H}(X \mid Y = y) = \sum_{x, y} p(x , y) \log \frac{1}{ p(x \mid y) }. \]

Theorem (Zero Conditional Entropy). Suppose that \(X\) and \(Y\) have the same support. \(\mathbf{H}(X \mid Y) = 0\) if and only if \(X\) is a function of \(Y\).

Proof. If \[ \mathbf{H}(X \mid Y) = \sum_y p(y) \mathbf{H}(X \mid Y = y) = 0. \]

Then either \(p(y) = 0\) or \(\mathbf{H}(X \mid Y = y) = 0\) for all \(y\). On the other hand, for each \(y\), if \(p(y) > 0\), then \[ \mathbf{H}(X \mid Y = y) = \sum_x p(x \mid y) \log \frac{1}{ p(x \mid y) } = 0, \] Then either \(p(x \mid y) = 0\) or \(\log \frac{1}{ p(x \mid y) } = 0\). It follows that there is a unique \(x\), for which \(p(x \mid y) = 1\).

Conversely, if \(X\) is a function of \(Y\), for each \(y\) such that \(p(y) > 0\), there is a unique \(x\) such that \(p(x \mid y) = 1\). Hence, \(\mathbf{H}(X \mid Y = y) = 0\) and \(H(X \mid Y) = 0\).

\(\square\)

Mutual Information

Definition. The mutual information between \(X\) and \(Y\) is \[ I(X; Y) = \mathbb{E}_{p(x, y) } \left[ \log \frac{ p(x, y)}{p(x) p(y) } \right] = \mathbf{D}( p(x, y) \Vert p(x) p(y) ). \]

The definition is symmetric and it holds that \(I(X; Y) = I(Y; X)\). If we know \(p(x,y)\), we use in expectation \(\mathbb{E}_{p(x, y) } \left[ \log \frac{1} { p(x, y)} \right]\) bits to describe them. If we know only \(p(x)\) and \(p(y)\), the mutual information measures the bits we waste.

\[ \small \begin{aligned} &\sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } \\ =& \sum p(x, y) \left( \log \frac{ 1 }{p(x)} + \log \frac{ 1 }{p(y)} - \log \frac{ 1 }{p(x, y)} \right) \\ =& \mathbf{H} ( X ) + \mathbf{H} ( Y ) - \mathbf{H} ( X, Y ) \end{aligned} \]

Further, mutual information measures the average reduction of randomness of \(X\) that results from learning the value of \(Y\). \[ \small \begin{aligned} \sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } &=\sum p(x, y) \log \frac{ p(x | y)}{p(x) } \\ &= \mathbf{H} ( X ) - \mathbf{H} ( X \mid Y). \end{aligned} \]

In this sense, the mutual information quantifies the dependence between \(X\) and \(Y\). It measures how much knowing one random variable reduces the uncertainty about the other random variable.

Similarly, we can prove that \[ \small \begin{aligned} \sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } &=\sum p(x, y) \log \frac{ p(x | y)}{ p(x) } \\ &=\mathbf{H} ( Y ) - \mathbf{H} ( X \mid Y). \end{aligned} \]

\(\mathbf{I}(X; Y) \ge 0\).

Proof. \[ \mathbf{I}(X; Y) = \mathbf{D}( p(x, y) \mid p(x) p(y) ) \ge 0 \]

Or \[ \begin{aligned} \sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } &= -\sum p(x, y) \log \frac{p(x) p(y) }{ p(x, y)} \\ &\ge - \log \sum p(x, y) \frac{p(x) p(y) }{ p(x, y)} \\ &=0 \end{aligned} \]

\(\square\)

Corollary. \(\mathbf{H} ( Y ) - \mathbf{H} ( X \mid Y) = \mathbf{H} ( X ) - \mathbf{H} ( X \mid Y) \ge 0\).

Theorem. The entropy \(\mathbf{H}(X)\) is concave in the probability density \(p(x)\).

Proof. Let \(\lambda_1, \lambda_2 \in [0, 1]\) such that \(\lambda_1 + \lambda_2 = 1\). Suppose that \(Z_1, Z_2\) are random variables with density functions \(p_1(z)\) and \(p_2(z)\) respectively. Define the random variable \[ Y = \begin{cases} 1, &\text{w.p } \lambda_1,\\ 2, &\text{w.p } \lambda_2, \end{cases} \] and let \(X = Z_Y\).

Then \[ \mathbf{H}(X) = \mathbf{H}( \lambda_1 p_1 + \lambda_2 p_2) \]

On the other hand, \[ \mathbf{H}(X \mid Y) = \lambda_1 \mathbf{H}( p_1 ) + \lambda_2 \mathbf{H}( p_2 ). \]

We finish the proof by observing that \(\mathbf{H}(X) \ge \mathbf{H}(X \mid Y)\).

\(\square\)

Definition. Let \(X\) be a Bernoulli random variable with mean \(p \in [0, 1]\). Denote \(\mathbf{H}(p) = \mathbf{H}(X) = p \log \frac{1}{p} + (1 - p) \log \frac{1}{1 - p}\).
Corollary. \(\mathbf{H}(p)\) is concave in \(p\).

Conditional Mutual Information

Definition. The conditional mutual information between \(X\) and \(Y\) given \(Z = z\) is the mutual information between the random variables \(X\) and \(Y\) in the joint ensemble \(\Pr (X, Y | Z = z)\), \[ I(X ; Y \mid Z = z) = \mathbf{H}(X \mid Z = z) - \mathbf{H}(X \mid Y, Z = z) \]

Definition. The conditional mutual information between \(X\) and \(Y\) given \(Z\) is average over \(z\) of \(I(X; Y \mid Z = z)\), \[ \begin{aligned} I(X; Y \mid Z) &= \mathbf{H}(X \mid Z) - \mathbf{H}(X \mid Y, Z) \\ &= \mathbf{H}(Y \mid Z) - \mathbf{H}(Y \mid X, Z) \\ &= \mathbf{H}(X \mid Z) + \mathbf{H}(Y \mid Z) - \mathbf{H}(X, Y \mid Z) \\ &= \mathbb{E}_{X, Y, Z} \log \frac{ p(X, Y \mid Z )}{ p(X \mid Z) \cdot p(Y \mid Z)}. \end{aligned} \]

Conditional Relative Entropy

Definition (Conditional Relative Entropy). \[ \begin{aligned} \mathbf{D}( p(y \mid x) || q(y \mid x) ) &\doteq \mathbb{E}_{p(x, y) } \left[ \log \frac{ p(Y \mid X) }{ q(Y \mid X) } \right]\\ &=\sum_{x} \sum_{ y } p(y \mid x) \log \frac{ p(y \mid x) }{ q(y \mid x) } \end{aligned} \]

Lemma. Relative entropy \(\mathbf{D}( p || q )\) is convex in the pair \((p, q)\): if there are two distribution pairs \((p_1, q_1)\) and \((p_2, q_2)\), then for \(\alpha_1, \alpha_2 \in [0, 1]\), s.t., \(\alpha_1 + \alpha_2 = 1\), it holds that \[ \mathbf{D}( \alpha_1 p_1 + \alpha_2 p_2 || \alpha_1 q_1 + \alpha_2 q_2 ) \le \alpha_1 \mathbf{D}( p_1 || q_1 ) + \alpha_2 \mathbf{D}( p_2 || q_2 ) \]

Proof. For any fix value \(x\), let \(s(x)\) be a shorthand for \(\alpha_1 p_1(x) + \alpha_2 p_2(x)\). It holds that \[ \small \begin{aligned} s(x) \log \frac{ s(x) }{ \alpha_1 q_1(x) + \alpha_2 q_2(x) } &=-s(x) \log \frac { \alpha_1 q_1(x) + \alpha_2 q_2(x) } { s(x) } \\ &= -s(x) \log \left( \frac { \alpha_1 p_1(x) } { s(x) } \frac { \alpha_1 q_1(x) } { \alpha_1 p_1(x) } + \frac { \alpha_2 p_2(x) } { s(x) } \frac { \alpha_2 q_2(x) } { \alpha_2 p_2(x) } \right) \\ &\le -\alpha_1 p_1(x) \log \frac { q_1(x) } { p_1(x) } - \alpha_2 p_2(x) \log \frac { q_2(x) } { p_2(x) } \end{aligned} \]

\(\square\)

Chain Rules

Chain Rule for Information Content

\(\log \frac{1}{ p(x, y) } = \log \frac{1}{p(x) } + \log \frac{1}{p(y | x) }\).

Chain Rule for Entropy

Let \(X_1, \ldots, X_n\) be random variables whose density function is \(p(x_1 \ldots, x_n)\).

\(\mathbf{H}(X_1, \ldots, X_n) = \sum_{i = 1}^n \mathbf{H}(X_i \mid X_1, \ldots, X_{i - 1})\)

Proof.
\[ \small \begin{aligned} \mathbf{H}(X_1, \ldots, X_n) &= \sum_{x_1 \ldots, x_n} p(x_1 \ldots, x_n) \log \frac{1}{\prod_{i = 1}^n p(x_i \mid x_1, \ldots, x_{i - 1}) } \\ &= \sum_{x_1 \ldots, x_n} p(x_1 \ldots, x_n) \sum_{i = 1}^n \log \frac{1}{ p(x_i \mid x_1, \ldots, x_{i - 1}) } \\ &=\sum_{i = 1}^n \mathbf{H}(X_i \mid X_1, \ldots, X_{i - 1}) \\ \end{aligned} \]

\(\square\)

Corollary. $ ( X_1, , X_n) _{i = 1}^n (X_i). $

\(\square\)

Chain Rule for Mutual Information

\(\mathbf{I}(Y; X_1, \ldots, X_n) = \sum_{i = 1}^n \mathbf{I}( Y; X_i \mid X_1, \ldots, X_{i - 1})\)

Proof 1.
\[ \small \begin{aligned} \mathbf{I}(X_1, \ldots, X_n; Y) &= \mathbf{H}(X_1, \ldots, X_n) - \mathbf{H}(X_1, \ldots, X_n \mid Y) \\ &= \sum_{i = 1}^n \mathbf{H}(X_i \mid X_1, \ldots, X_{i - 1}) - \sum_{i = 1}^n \mathbf{H}(X_i \mid X_1, \ldots, X_{i - 1}, Y) \\ &= \sum_{i = 1}^n \mathbf{I}(X_i ; Y\mid X_1, \ldots, X_{i - 1}) \end{aligned} \]

\(\square\)

Proof 2.
\[ \small \begin{aligned} \mathbf{I}( Y ; X_1, \ldots, X_n ) &= \mathbf{D} ( p(x_1, \ldots, x_n, y) \Vert p(x_1, \ldots, x_n) p(y)) \\ &= \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \log \frac{ p( x_1, \ldots, x_n, y) }{ p( x_1, \ldots, x_n ) p(y) } \\ &= \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \left( \log \frac{1}{ p(x_1, \ldots, x_n) } - \log \frac{1}{ p(x_1, \ldots, x_n \mid y ) } \right) \\ &= \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \log \frac{1}{ p(x_1) \cdot p(x_2 \mid x_1) \cdot \ldots \cdot p(x_n \mid x_1, \ldots, x_{n - 1} ) } \\ & - \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \log \frac{1}{ p(x_1 \mid y) \cdot p(x_2 \mid x_1, y) \cdot \ldots \cdot p(x_n \mid x_1, \ldots, x_{n - 1}, y ) } \\ &= \sum_{x_1, \ldots, x_n, y } p(x_1, \ldots, x_n, y) \left( \sum_{i \in [n] } \log \frac{ p(x_i \mid x_1, \ldots, x_{i - 1}, y) }{ p(x_i \mid x_1, \ldots, x_{i - 1}) } \right) \\ &= \sum_{i = 1}^n \mathbf{I}( Y ; X_i \mid X_1, \ldots, X_{i - 1}) \end{aligned} \]

\(\square\)

Convexity of Relative Entropy. \[ \mathbf{D}( p(x, y) || q(x, y) ) = \mathbf{D}( p(x) || q(x) ) + \mathbf{D}( p(y \mid x) || q(y \mid x) ). \]

Proof. \[ \begin{aligned} \mathbf{D}( p(x, y) || q(x, y) ) &= \sum_{x, y} p(x, y) \log \frac{ p(x) p(y \mid x) }{q(x) q (y \mid x) } \\ &= \sum_{x, y} p(x, y) \log \frac{ p(x) }{q(x) } + \sum_{x, y} p(x, y) \log \frac{ p(y \mid x) }{ q (y \mid x) } \\ &= \mathbf{D}( p(x) || q(x) ) + \mathbf{D}( p(y \mid x) || q(y \mid x) ) \end{aligned} \]

\(\square\)

Amplificatoin by Sampling

Posted on 2021-07-05 Edited on 2021-07-06

Definition. Two distribution \(\mu\) and \(\mu'\) are \((\epsilon, \delta)\) close, denoted as \(\mu \simeq_{(\epsilon, \delta) } \mu'\), if for all measurable set \(A\), we have \[ \begin{aligned} \mu(A) &\le e^\epsilon \mu'(A) + \delta, \\ \mu'(A) &\le e^\epsilon \mu(A) + \delta. \end{aligned} \]

Theorem. Let \(q \in (0, 1)\) and let \(\mu\) and \(\mu'\) be two distributions such that \(\mu \simeq_{(\epsilon, \delta)} \mu'\). For \(\nu = (1 - q) \mu + q \mu'\), it holds that \[ \nu \simeq_{ (\epsilon', q \delta) } \mu, \] where \(\epsilon' = \ln ( q (e^\epsilon - 1) + 1 )\).

Proof. For each measurable set \(A\) \[ \begin{aligned} \nu(A) &= (1 - q) \mu(A) + q \mu' (A) \\ &\le (1 - q) \mu(A) + q (e^\epsilon \mu (A) + \delta) \\ &= ( q (e^\epsilon - 1) + 1 ) \mu(A) + q \delta. \end{aligned} \]

Next we prove the other direction. Denote \(t = e^{\epsilon} - 1\). Then \[ e^{\epsilon'} = 1 + qt, \]

and \[ \begin{aligned} \nu(A) &= (1 - q) \mu(A) + q \mu' (A) \\ &\ge (1 - q) \mu(A) + q e^{-\epsilon} (\mu (A) - \delta) \\ &= (1 - q) \mu(A) + \frac{q}{1 + t} (\mu (A) - \delta) \\ &= \mu(A) - \frac{qt}{1 + t} \mu (A) - \frac{q}{1 + t} \delta \\ &= \left( 1 - \frac{qt}{1 + t} \right) \mu (A) - \frac{1}{1 + t} q \delta \\ &\ge \left( 1 - \frac{qt}{1 + qt} \right) \mu (A) - \frac{1}{1 + qt} q \delta \\ &= \frac{1}{1 + qt} \left( \mu (A) - q \delta \right). \end{aligned} \]

\(\square\)

Fubini's Theorem

Posted on 2021-06-28 Edited on 2023-02-04

Let \((X, \mathcal{F}_1, \mu_1)\) and \((Y, \mathcal{F}_2, \mu_2)\) be two \(\sigma\)-finite measure space.

Produce Measure Space

Consider the product space \(\Omega = X \times Y\).

Definition. A measurable rectangle is a product \(A \times B\) for which \(A \in \mathcal{F}_1\) and \(B \in \mathcal{F}_2\).

Denote the collection of measurable rectangles as \[ \mathcal{S} \doteq \begin{Bmatrix} A \times B, A \in \mathcal{F}_1, B \in \mathcal{F}_2 \end{Bmatrix}. \]

We would like to construct a measure \(\mu\) on \(\sigma(\mathcal{S})\), the \(\sigma\)-field generated by \(\mathcal{S}\), thus

  1. For each \(A \times B \in \mathcal{S}\), it's natural to define \(\mu (A \times B) = \mu_1(A) \cdot \mu_2(B)\).

  2. We prove that \(\mathcal{S}\) is a semialgebra.

  3. We show that \(\mu\) is a measure on \(\mathcal{S}\).

  4. Finally, it follows by the Caratheodory’s Extension Theorem, \(\mu\) has a unique extension on \(\sigma(\mathcal{S})\).

Lemma 1. \(\mathcal{S}\) is a semialgebra.

Proof. * \(\mathcal{S}\) is close under intersection: if \(A \times B, C \times D \in \mathcal{S}\), then \[ (A \times B) \cap (C \times D) = (A \cap C) \times (B \cap D). \] * The complement of element in \(\mathcal{S}\) is a finite disjoint union of elements in \(\mathcal{S}\): if \(A \times B \in \mathcal{S}\), since \[ \Omega = (A \times B) + ( \bar A \times B ) + ( A \times \bar B ) + ( \bar A \times \bar B ), \] it holds that \[ \overline{ A \times B } = ( \bar A \times B ) + ( A \times \bar B ) + ( \bar A \times \bar B ). \]

\(\square\)

Lemma 2. \(\mu\) is a measure on \(\mathcal{S}\).

Proof. It suffices to prove \(\sigma\)-additivity of \(\mu\). Suppose \(A \times B = +_{i \ge 1} (A_i \times B_i)\) is a countable disjoint union of elements in \(\mathcal{S}\).

Consider an \(x \in A\). For each \(y \in B\), we have \((x, y) \in A \times B\). As the sets \(\begin{Bmatrix} A_i \times B_i \end{Bmatrix}\) consists of a partition of \(A \times B\), it follows that \((x, y)\) belongs to exactly one \(A_i \times B_i\). Denote \[ I(x) \doteq \begin{Bmatrix} i : \exists y \in B,\, \text{s.t.},\, (x, y) \in A_i \times B_i \end{Bmatrix} = \begin{Bmatrix} i : x \in A_i \end{Bmatrix}. \] Then \(B = +_{i \in I(x) } B_i\).

As \(\mu_2\) is a measure on \(Y\), it is countably additive: \[ \mu_2 (B) = \sum_{i \in I(x) } \mu(B_i ) \implies \mathbb{I}_A (x) \mu_2(B) = \sum_{i \ge 1} \mathbb{I}_{ A_i } (x) \cdot \mu_2 (B). \]

By monotone convergence theorem, \[ \int_X \mathbb{I}_A(x) \mu_2(B) \, d\, \mu_1 = \sum_{i \ge 1} \int_X \mathbb{I}_{ A_i } (x) \cdot \mu_2 (B) d\, \mu_1, \]

which is \[ \mu_1 (A) \cdot \mu_2 (B) = \sum_{i \ge 1} \mu_1 (A_i) \cdot \mu_2 (B_i). \]

\(\square\)

Theorem. \(\mu\) has a unique extension \(\bar \mu\) on \(\sigma( \mathcal{S} )\), the \(\sigma\)-field generated by \(\mathcal{S}\).

Proof. Note that \(\mathcal{S}\) is a semialgebra and \(\mu\) is a \(\sigma\)-finite measure. Applying Caratheodory’s Extension Theorem directly finishes the proof.

\(\square\)

Fubini's Theorem

Let \(\begin{Bmatrix} a_i : i \in \mathbb{N} \end{Bmatrix}\) be a countably sequence of numbers. Then the series converges to the same limit under permutation, if

  1. \(a_i \ge 0, \forall i \in \mathbb{N}\). Note that \(\sum_{i \ge 1} a_i\) can equal \(\infty\).
  2. \(\sum_{i \ge 1} | a_i | < \infty\).

Integration is a generalization of discrete sum. Given a measure space \((\Omega, \sigma( \mathcal{S} ), \bar \mu)\), and a measure function \(f\), the integral of \(f\) exists, if

  1. \(f \ge 0\);
  2. Or \(\int |f| d\, \bar \mu < \infty\).

In case that \((\Omega, \sigma( \mathcal{S} ), \bar \mu)\) is a produce measure space, the Fubini's theorem states that if the double integral exists, it can be computed by iterated integrals.

Theorem (Fubini's Theorem). If \(f \ge 0\) or \(\int |f| d\, \bar \mu < \infty\), then there exists \(A \in \mathcal{F}_1\), \(B \in \mathcal{F}_2\), s.t., \(\mu_1 (A) = \mu_2 (B) = 0\), and \[ \int_{X - A} \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f(x, y) d\, \bar \mu = \int_{Y - B} \int_X f(x, y) d\, \mu_1 \, d\, \mu_2. \]

Remark. For the case concerning \(f \ge 0\), we have \(A = B = \emptyset\), and the Theorem is called Tonelli’s theorem.

Comparison with Discrete Sum

Let \(X = Y = \mathbb{N}\), and \(\mu_1 = \mu_2\) be counting measures. Let \(\begin{Bmatrix} a_{i,j} : i, j \in \mathbb{N} \end{Bmatrix}\) be a sequency of numbers and define \(f(i, j) = a_{i,j}\). Denote \[ E_1 \doteq \begin{Bmatrix} (i, j) \in \mathbb{N} \times \mathbb{N} : a_{i,j} \ge 0 \end{Bmatrix}, \\ E_2 \doteq \begin{Bmatrix} (i, j) \in \mathbb{N} \times \mathbb{N} : a_{i,j} < 0 \end{Bmatrix}. \]

By definition of integration, \(\int_\Omega f(x, y) \, d\, \bar \mu\) exists if

  1. Either \(f \ge 0\). In this case \[ \int_\Omega f(x, y) \, d\, \bar \mu = \sup \begin{Bmatrix} \sum_{ ( i,j ) \in S } a_{i,j} : \begin{array}{l} S \subseteq \mathbb{N} \times \mathbb{N} \\ |S| < \infty \end{array} \end{Bmatrix}, \]

  2. Or \(\int_\Omega | f(x, y) | \, d\, \bar \mu < \infty\). In this case, \[ \begin{aligned} &\int_\Omega f(x, y) \, d\, \bar \mu = \int_\Omega f_+ \, d\, \bar \mu - \int_\Omega f_- \, d\, \bar \mu \\ &= \sup \begin{Bmatrix} \sum_{ ( i,j ) \in S } a_{i,j}: \, \begin{matrix} S \subseteq E_1 \\ |S| < \infty \end{matrix} \end{Bmatrix} - \sup \begin{Bmatrix} \sum_{ ( i,j ) \in S } a_{i,j}: \, \begin{matrix} S \subseteq E_2 \\ |S| < \infty \end{matrix} \end{Bmatrix} . \end{aligned} \]

In the former case, the summation of elements in a chosen set \(S\) grows in one direction. While in the later, the summation converges absolutely.

When \(\int_\Omega f(x, y) \, d\, \bar \mu\) exists, the following iterated integrals exist: \[ \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 = \sum_{i \ge 1} \sum_{j \ge 1} a_{i,j}, \\ \int_Y \int_X f(x, y) d\, \mu_1 \, d\, \mu_2 = \sum_{j \ge 1} \sum_{i \ge 1} a_{i,j}. \]

Further, their values are equal.

\(\square\)

Proof of Fubini's Theorem

By symmetry, it suffices to prove that \[ \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f(x, y) d\, \bar \mu. \]

Sketch of the Proof

Before proving that the iterated integral equals the double integral, we need to make sure that the first iterated integral is well-defined. Specifically,

  • When \(x\) is fixed, the function \(f_x : Y \rightarrow \mathbb{R}\) defined by \(f_x(y) \doteq f(x, y)\) is \(\mathcal{F}_2\) measurable, and the integral \(\int_Y f(x, y) \, d\, \mu_2\) exists.

  • There is \(\mathcal{F}_1\) measurable function \(g : X \rightarrow \mathbb{R}\) such that \(g(x) = \int_Y f(x, y) \, d\, \mu_2\) almost everywhere.

We will first establish the theorem for indicator functions \(\mathbb{I}_E\) for sets \(E \in \sigma( \mathcal{S} )\), which serve as basic building blocks for extending the theorem for more general functions.

Lemma. Given \(E \in \sigma( \mathcal{S} )\), let \(\mathbb{I}_E\) be the indicator for \((x, y) \in E\). Then

  1. For each \(x \in X\), the function \(f_x(y) \doteq \mathbb{I}_E (x, y)\) is \(\mathcal{F}_2\) measurable.

  2. The function \(g(x) \doteq \int_Y \mathbb{I}_E (x, y) d\, \mu_2\) is \(\mathcal{F}_1\) measurable.

  3. \[ \int_X \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \]

The proof of the lemma is deferred to the end of the article.

Proof of the Theorem

For now, we apply it to prove Theorem for the following three increasingly more general cases.

Case One. \(f\) is a simple function on \((X \times Y, \sigma(\mathcal{S}))\).

Case Two. \(f \ge 0\) and is \(\sigma(\mathcal{S})\) measurable.

Case Three. \(f\) is measurable and integrable.

Remark: this follows exactly the same order with which we define Lebesgue integrals.

Proof.

Case 1. Suppose that \(f(x, y) = \sum_{i \in [n]} c_i \cdot \mathbb{I}_{E_i} (x, y)\) is a simple function on \((X \times Y, \sigma(\mathcal{S}))\).

  1. For fixed \(x \in X\), according the lemma, the functions \(f_x^{ (i) } (y) \doteq \mathbb{I}_{E_i} (x, y), i \in [n]\) are \(\mathcal{F}_2\) measurable. It follows that function \(f_x (y) = \sum_{i \in [n]} c_i \cdot f_x^{ (i) } (y)\) is \(\mathcal{F}_2\) measurable.

  2. Since \(g^{(i)} (x) \doteq \int_Y \mathbb{I}_{E_i} (x, y) d\, \mu_2\) is \(\mathcal{F}_1\) measurable, so is the function \[ g(x) = \int_Y f_x(y) d\, \mu_2 = \sum_{i \in [n]} c_i \cdot \int_Y \mathbb{I}_{E_i} (x, y) d\, \mu_2 \]

  3. Since \(\int_X \int_Y \mathbb{I}_{E_i} (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_{E_i} (x, y) d\, \bar \mu\), \[ \begin{aligned} \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \sum_{i \in [n]} c_i \cdot \int_X \int_Y \mathbb{I}_{E_i} (x, y) d\, \mu_2 \, d\, \mu_1 \\ &= \sum_{i \in [n]} c_i \cdot \int_{\Omega } \mathbb{I}_{E_i} (x, y) d\, \bar \mu \\ &= \int_{\Omega } \sum_{i \in [n]} c_i \cdot\mathbb{I}_{E_i} (x, y) d\, \bar \mu \\ &= \int_{\Omega } f(x, y) d\, \bar \mu. \end{aligned} \]

Case Two. Suppose that \(f \ge 0\) and is \(\sigma(\mathcal{S})\) measurable. Then there exists simple, nonnegative functions \(f^{ (n) } \uparrow f\).

  1. For fixed \(x \in X\). The functions \(f_x^{ (n) } (y) \doteq f^{ (n) } (x, y) \uparrow f_x(y)\). Since they are \(\mathcal{F}_2\) measurable, so is \(f_x (y) \doteq f(x, y)\).

  2. Let \(g^{(n)} (x) \doteq \int_Y f^{ (n) } (x, y) d\, \mu_2\). Since \(0 \le f_x^{ (n) } (y) \uparrow f_x(y)\), by monotone convergence theorem: \[ g(x) = \int_Y f_x(y) d\, \mu_2 = \lim_n \int_Y f^{ (n) } (x, y) d\, \mu_2 = \lim_n g^{(n)} (x) \] It follows that \(g^{(n)} (x) \uparrow g(x)\). Since they are \(\mathcal{F}_1\) measurable, so is \(g(x)\).

  3. Since \(\int_Y f^{ (n) } (x, y) d\, \mu_2 \uparrow \int_Y f(x, y) d\, \mu_2 \,\), \(\int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \,\), and \(f^{(n)} (x, y) \uparrow f(x, y)\), by monotone convergence theorem, \[ \begin{aligned} \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \lim_n \int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 \\ &= \lim_n \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \\ &= \int_{\Omega } \lim_n f^{ (n) } (x, y) d\, \bar \mu \\ &= \int_{\Omega } f (x, y) d\, \bar \mu. \end{aligned} \]

Case Three. Suppose that \(f(x, y)\) is a measurable and integrable function on \(X \times Y\), i.e., \(\int_{\Omega } |f(x, y)| d\, \bar \mu < \infty\). Write \(f(x, y) = f^+(x, y) - f^-(x, y)\), such that \(f^+(x, y), f^-(x, y) \ge 0\) are measurable and integrable.

  1. For a fixed \(x \in X\), since both \(f^+_x(y) \doteq f^+(x, y), f^-_x(y) \doteq f^-(x, y)\) are \(\mathcal{F}_2\) measurable, so is the function \(f_x (y) \doteq f(x, y) = f^+(x, y) - f^-(x, y)\) is \(\mathcal{F}_2\).

  2. Since \(f^+(x, y), f^-(x, y) \ge 0\) are measurable and integrable, we have
    \[ \begin{aligned} \int_X \int_Y f^+(x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^+(x, y) d\, \bar \mu < \infty, \\ \int_X \int_Y f^-(x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^-(x, y) d\, \bar \mu < \infty. \end{aligned} \] It follows that \(\exists A \subseteq \mathcal{F}_1\), s.t., \(\mu_1(A) = 0\) and \(g^+(x) \doteq \int_Y f^+(x, y) d\, \mu_2\) and \(g^-(x) \doteq \int_Y f^-(x, y) d\, \mu_2\) are finite on \(X \setminus A\). Define \[ g(x) = \begin{cases} \int_Y f_x(y) d\, \mu_2 = g^+(x) - g^-(x), & \text{if } x \in X \setminus A \\ 0, & \text{if } x \in A \end{cases}. \] Then \(g(x)\) is \(\mathcal{F}_1\) measurable.

  3. We have \[ \begin{aligned} \int_{X \setminus A} \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \int_{X \setminus A} g^+(x) \, d\, \mu_1 - \int_{X \setminus A} g^-(x) d\, \mu_2 \, d\, \mu_1 \\ &= \int_X g^+(x) \, d\, \mu_1 - \int_X g^-(x) d\, \mu_2 \, d\, \mu_1 \\ &= \int_{\Omega } f^+(x, y) d\, \bar \mu - \int_{\Omega } f^-(x, y) d\, \bar \mu \\ &= \int_{\Omega } f^+(x, y) - f^-(x, y) d\, \bar \mu \\ &= \int_{\Omega } f (x, y) d\, \bar \mu. \end{aligned} \]

\(\square\)

Proof of the Lemma

Lemma. Given \(E \in \sigma( \mathcal{S} )\), let \(\mathbb{I}_E\) be the indicator for \((x, y) \in E\). Then

  1. For each \(x \in X\), the function \(f_x(y) \doteq \mathbb{I}_E (x, y)\) is \(\mathcal{F}_2\) measurable.

  2. The function \(g(x) \doteq \int_Y \mathbb{I}_E (x, y) d\, \mu_2\) is \(\mathcal{F}_1\) measurable.

  3. \[ \int_X \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \]

There are various ways to prove this lemma. We present two proofs here.

Proof One

Since \(X\) and \(Y\) are both \(\sigma\)-finite, there exists \(A_k \in \mathcal{F}_1, \mu_1(A_k) < \infty, \forall n \in \mathbb{N}^+\), s.t., \(A_k \uparrow X\), and \(B_k \in \mathcal{F}_2, \mu_2(B_k) < \infty, \forall n \in \mathbb{N}^+\), s.t., \(B_k \uparrow Y\). It follows that \(\Omega_k \doteq A_k \times B_k \uparrow \Omega\), and that \(\mu(\Omega_k) < \infty\).

Define \[ \mathcal{L}_k \doteq \{ E \in \sigma(\mathcal{S}) : E \cap \Omega_k \, \text{ satisfies the conditions in the Lemma } \}. \]

Claim. If \(\mathcal{L}_k = \sigma(\mathcal{S})\), then the lemma holds.

Proof of the Claim. For each \(E \in \sigma(\mathcal{S})\),

  1. For fixed \(x \in X\). The functions \(f_x^{ (k) } (y) \doteq \mathbb{I}_{E \cap \Omega_k} (x, y) \uparrow f_x(y)\). Since they are \(\mathcal{F}_2\) measurable, so is \(f_x (y) \doteq f(x, y)\).

  2. Let \(g^{(k)} (x) \doteq \int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2\) are increasing. Since \(0 \le f_x^{ (n) } (y) \uparrow f_x(y)\), by monotone convergence theorem: by monotone convergence theorem: \[ g(x) = \int_Y f_x(y) d\, \mu_2 = \lim_k \int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2 = \lim_k g^{(k)} (x) \] It follows that \(g^{(k)} (x) \uparrow g(x)\). Since they are \(\mathcal{F}_1\) measurable, so is \(g(x)\).

  3. Since \(\int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2 \uparrow \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \,\), \(\int_X \int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \bar \mu \,\), and \(\mathbb{I}_{E \cap \Omega_k} (x, y) \uparrow \mathbb{I}_E (x, y)\), by monotone convergence theorem, \[ \begin{aligned} \int_X \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \, d\, \mu_1 &= \lim_k \int_X \int_Y \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \mu_2 \, d\, \mu_1 \\ &= \lim_k \int_{\Omega } \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \bar \mu \\ &= \int_{\Omega } \lim_k \mathbb{I}_{E \cap \Omega_k} (x, y) d\, \bar \mu \\ &= \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \end{aligned} \]

\(\square\)

It is left to prove that \(\mathcal{L}_k = \sigma(\mathcal{S})\). The proof relies on the following fact.

Fact. If \(\tilde{S}\) is an algebra, then the monotone class \(\mathcal{M}(\tilde{S})\) generated by \(\tilde{S}\) equals \(\sigma(\tilde{S})\).

The proof consists of two steps:

  • \(\mathcal{L}_k\) contains the algebra \(\tilde{S}\) generated by \(S\).

  • \(\mathcal{L}_k\) is a monotone class.

In what follows, to ease the notations, we write \(\mathcal{L}_k\) as \(\mathcal{L}\) and assume that \(\mu(\Omega) < \infty\).

\(\mathcal{L}\) contains the algebra \(\tilde{S}\) generated by \(S\).

Let \(E = +_i \big( A_i \times B_i \big)\) for a finite sequence of disjoint rectangles \(A_i \times B_i\).

  1. For each \(x \in X\), the function \(f_x(y) \doteq \mathbb{I}_E (x, y)\) is a simple function on \(Y\) and there \(\mathcal{F}_2\) measurable.

  2. The function $$ g(x) = _Y _E (x, y) d, _2 = i {A_i} (x) _2(B_i)

    $$ is a simple function on \(X\) and there \(\mathcal{F}_1\) measurable.

  3. \[ \int_X g(x) \, d\, \mu_1 = \mu_1 (A) \cdot \mu_2 (B) = \bar \mu (E) = \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \]

\(\mathcal{L}\) is a monotone class.

Case One \(E_n \in \mathcal{L}, n \in \mathbb{N}^+\), s.t., \(E_n \uparrow E\)

  1. For each \(x \in X\), we have \(f_x^{ (n) } (y) = \mathbb{I}_{E_n} (x, y) \uparrow f_x(y)\). Since the \(f_x^{ (n) } (y)\) \(\mathcal{F}_2\) measurable, so is the function \(f_x(y)\).

  2. Let \(g^{(n)} (x) \doteq \int_Y f^{ (n) } (x, y) d\, \mu_2\). Since \(0 \le f_x^{ (n) } (y) \uparrow f_x(y)\), by monotone convergence theorem: \[ g(x) = \int_Y f_x(y) d\, \mu_2 = \lim_n \int_Y f_x^{ (n) } (y) d\, \mu_2 = \lim_n g^{(n)} (x) \] It follows that \(g^{(n)} (x) \uparrow g(x)\). Since they are \(\mathcal{F}_1\) measurable, so is \(g(x)\).

  3. Since \(\int_Y f^{ (n) } (x, y) d\, \mu_2 \uparrow \int_Y f(x, y) d\, \mu_2 \,\), \(\int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \,\), and \(f^{(n)} (x, y) \uparrow f(x, y)\), by monotone convergence theorem, \[ \begin{aligned} \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \lim_n \int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 \\ &= \lim_n \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \\ &= \int_{\Omega } \lim_n f^{ (n) } (x, y) d\, \bar \mu \\ &= \int_{\Omega } f (x, y) d\, \bar \mu. \end{aligned} \]

Case Two \(E_n \in \mathcal{L}, n \in \mathbb{N}^+\), s.t., \(E_n \downarrow E\)

  1. For each \(x \in X\), we have \(f_x^{ (n) } (y) = \mathbb{I}_{E_n} (x, y) \downarrow f_x(y)\). Since the \(f_x^{ (n) } (y)\) are \(\mathcal{F}_2\) measurable, so is the function \(f_x(y)\).

  2. Since \(f_x^{ (n) } (y) = \mathbb{I}_{E_n} (x, y) \downarrow f_x(y)\) and \(f_x^{ (n) } (y) \le 1\), \(0 \le 1 - f_x^{ (n) } (y) \uparrow 1 - f_x(y)\). By monotone convergence theorem: \[ \begin{aligned} \mu_2(Y) - \lim_n g^{(n)} (x) &= \lim_n \int_Y \left( 1 - f_x^{ (n) } (y) \right) d\, \mu_2 \\ &= \int_Y \left( 1 - f_x(y) \right) d\, \mu_2 \\ &= \mu_2(Y) - g(x). \end{aligned} \] Since \(\mu_2(Y) < \infty\), \(\lim_n g^{(n)} (x) = g(x)\). Since they are \(\mathcal{F}_1\) measurable, so is \(g(x)\).

  3. Since \(0 \le \mu_2(Y) - \int_Y f^{ (n) } (x, y) d\, \mu_2 \uparrow \mu_2(Y) - \int_Y f(x, y) d\, \mu_2 \,\), \(\int_X \int_Y f^{ (n) } (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \,\), and \(0 \le 1 - f^{(n)} (x, y) \uparrow 1 - f(x, y)\), by monotone convergence theorem, \[ \begin{aligned} \mu(\Omega) - \int_X \int_Y f(x, y) d\, \mu_2 \, d\, \mu_1 &= \int_X \left( \mu_2(Y) - \int_Y f(x, y) d\, \mu_2 \right) \, d\, \mu_1 \\ &= \lim_n \int_X \left( \mu_2(Y) - \int_Y f^{ (n) } (x, y) d\, \mu_2 \right) \, d\, \mu_1 \\ &= \mu(\Omega) - \lim_n \int_{\Omega } f^{ (n) } (x, y) d\, \bar \mu \\ &= \lim_n \int_{\Omega } \left( 1 - f^{ (n) } (x, y) \right) d\, \bar \mu \\ &= \int_{\Omega } \lim_n \left(1 - f^{ (n) } (x, y) \right) d\, \bar \mu \\ &= \mu(\Omega) - \int_{\Omega } f (x, y) d\, \bar \mu. \end{aligned} \] Observing that \(\mu(\Omega) < \infty\) finishes the proof.

\(\square\)

Proof Two

Lemma. Given \(E \in \sigma( \mathcal{S} )\), let \(f(x, y) = \mathbb{I}_E (x, y)\) be the indicator for \((x, y) \in E\). Then

  1. For each \(x \in X\), the function \(f_x(y) \doteq \mathbb{I}_E (x, y)\) is \(\mathcal{F}_2\) measurable.

  2. The function \(g(x) \doteq \int_Y \mathbb{I}_E (x, y) d\, \mu_2\) is \(\mathcal{F}_1\) measurable.

  3. \[ \int_X \int_Y \mathbb{I}_E (x, y) d\, \mu_2 \, d\, \mu_1 = \int_{\Omega } \mathbb{I}_E (x, y) d\, \bar \mu. \]

Proof of (1).

Definition. Let \(E_x \doteq \begin{Bmatrix} y : (x, y) \in E \end{Bmatrix}\) be the cross-section at \(x\).

It suffices to show that

Claim. For each \(x \in X\) and \(E \in \sigma(\mathcal{S})\), \(E_x \in \mathcal{F}_2\).

Proof of the Claim. Fix an \(x \in X\). Define the function \(T_x : Y \rightarrow X \times Y\) by \(T_x (y) \doteq (x, y).\) Then for each \(E = A \times B \in \mathcal{S}\), \[ T_x^{-1} (E) = E_x = \begin{cases} B, &\text{ if } x \in A, \\ \emptyset, & \text{ otherwise }. \end{cases} \]

Therefore \(T_x^{-1} (E) = E_x \in \mathcal{F}_2\). Since \(\mathcal{S}\) generates \(\sigma(\mathcal{S})\), we conclude that \(T_x\) is measurable \(\mathcal{F}_2 / \sigma(\mathcal{S})\).

\(\square\)

Proof of (2) and (3).

Define the set \(\mathcal{L} \subseteq \sigma(\mathcal{S})\) as the collection of \(E\) for which the condition (2) and (3) hold.

  1. Clearly it holds that \(\mathcal{S} \subseteq \mathcal{L}\);
  2. We show that \(\mathcal{L}\) is a \(\lambda\) system.

Since \(\mathcal{S}\) is a \(\pi\) system, via Dynkin's \(\pi-\lambda\) theorem, \(\mathcal{L}\) is a \(\sigma\)-algebra, and hence \(\mathcal{L} = \sigma(\mathcal{S})\).

We will now check that \(\mathcal{L}\) is a \(\lambda\)-system. With the same technique applied in previous proof, we can suppose without loss of generality that \(\mu(\Omega) < \infty\).

  1. \(\Omega \in \mathcal{L}\).

  2. If \(E \in \mathcal{L}\), then \(\bar E \in \mathcal{L}\). First, observe that \(\bar E_x = \Omega_x - E_x = B - E_x \in \mathcal{F}_2\). Since \(\mu_2(B) < \infty\), \[ g_{\bar E} (x) = \mu_2 ( \bar E_x ) = \mu_2 (\Omega_x - E_x) = \mu_2 (B) - \mu_2 (E_x). \] Since \(g_E(x) = \mu_2 (E_x)\) is \(\mathcal{F}_1\) measurable, so is \(g_{\bar E}(x)\). Integrating over \(x\) gives \[ \begin{aligned} \int_X \mu_2 ( \bar E_x ) \, d\, \mu_1 &= \int_X \mu_2 (B) \, d\, \mu_1 - \int_X \mu_2 (E_x) \, d\, \mu_1 \\ &= \mu (\Omega) - \mu(E). \end{aligned} \]

    Since \(\mu(\Omega) < \infty\), we have \(\mu(\bar E) = \mu (\Omega) - \mu(E) = \int_X \mu_2 ( \bar E_x ) \, d\, \mu_1\).

  3. If \(E_n \in \mathcal{L}, n \in \mathbb{N}^+\) is a disjoint sequence of sets, then \(\cup_{n \ge 1} E_n \in \mathcal{L}\). Define \(g_{E_n} (x) \doteq \mu_2 ( (E_n)_x )\). Then \(g(x) \doteq \sum_{n \ge 1} g_{E_n} (x)\) is \(\mathcal{F}_1\) measurable, since \(g = \sup_{n \ge 1} \sum_{k \in [n]} g_{E_k}\). By monotone convergence theorem and that \(\mu\) is a measure, it holds that \[ \int_X g \, d\, \mu_1 = \int_X \sum_{n \ge 1} g_{E_n} \, d\, \mu_1 = \sum_{n \ge 1} \int_X g_{E_n} \, d\, \mu_1 = \sum_{n \ge 1} \mu (E_n) = \mu (E). \] Therefore, \(E \in \mathcal{L}\).

\(\square\)

Remarks

Given a function \(f(x, y)\), there is an easier way to prove that it is measurable \(\mathcal{F}_2 / \mathcal{B}\) when \(x\) is fixed, without approximating it by simple functions. However, to prove the measurability of \(\int_Y f(x, y) \, d \mu_2\) (as a function of \(x\)), we still need to write it as the limit of simple functions.

Lemma. Let \(f: X \times Y \rightarrow \mathbb{R}\) be measurable \(\sigma(\mathcal{S}) / \mathcal{B}\). For each \(x \in X\), the function \(f_x : Y \rightarrow \mathbb{R}\) defined by \(f_x(y) = f(x, y)\) is measurable \(\mathcal{F}_2 / \mathcal{B}\).

Proof. Note that \(f_x = f \circ T_x\). Since \(T_x\) is measurable \(\mathcal{F}_2 / \sigma(\mathcal{S})\) and \(f\) is measurable \(\sigma(\mathcal{S}) / \mathcal{B}\), it follows that \(f_x\) is measurable \(\mathcal{F}_2 / \mathcal{B}\).

Or, specifically, for each Borel set \(W \in \mathcal{B}\), since \(f\) is measurable \(\sigma(\mathcal{S}) / \mathcal{B}\), \[ E = f^{-1} (W) = \{ (x, y) : f(x, y) \in W \} \in \sigma (\mathcal{S}). \]

Then \[ f_x^{-1} (W) = \{ y : f(x, y) \in W \} = E_x \in \mathcal{F}_2. \] \(\square\)

For completeness, we include a proof for the following fact. Observe that the technique applied here is similar to the one used in the proof for \(\pi\)-\(\lambda\) theorem.

Fact. If \(\tilde{S}\) is an algebra, then the monotone class \(\mathcal{M}(\tilde{S})\) generated by \(\tilde{S}\) equals \(\sigma(\tilde{S})\).

Proof. It is trivial to see \(\mathcal{M}(\tilde{S}) \subseteq \sigma(\tilde{S})\).

We prove the other direction by showing that \(\mathcal{M}(\tilde{S})\) is an algebra. The proof consists of two steps:

  1. For each \(E \in \tilde{S}\), we prove that \[ \mathcal{R}(E) \doteq \left\{ F \in \mathcal{M}(\tilde{S}) : E \cup F, F - E, E - F \in \mathcal{M}(\tilde{S}) \right\} = \mathcal{M}(\tilde{S}). \] Clearly \(\tilde{S} \subseteq \mathcal{R}(E)\). We prove that \(\mathcal{R}(E)\) is a monotone class. Suppose that \(F_n \rightarrow F\) is an increasing or decreasing sequence. Then \(E \cup F_n, F_n - E, E - F_n\) are also increasing or decreasing sequences. Therefore, \(E \cup F, F - E, E - F \in \mathcal{R}(E)\).

  2. For each \(E \in \mathcal{M}(\tilde{S})\), we prove that \[ \mathcal{R}(E) \doteq \left\{ F \in \mathcal{M}(\tilde{S}) : E \cup F, F - E, E - F \in \mathcal{M}(\tilde{S}) \right\} = \mathcal{M}(\tilde{S}). \] If \(F \in \tilde{S}\), then the previous claim \(E \in \mathcal{R}(F)\). Therefore, \(F \in \mathcal{R}(E)\). Following similar vein, we can also prove that \(\mathcal{R}(E)\) is a monotone class.

\(\square\)
