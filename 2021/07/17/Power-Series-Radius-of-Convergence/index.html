Power Series--Radius of Convergence

Posted on 2021-07-17 Edited on 2022-11-13

Theorem (Hadamard's Formula). Let \(f : \mathbb{C} \rightarrow \mathbb{C}\) be a power series defined as \(f(z) = \sum_{n \ge 1} a_n z^n\). The radius of convergence of \(f\), denoted as \(R \in [0, \infty]\), is given by \[ 1 / R = L \doteq \limsup_{k \ge 1} |a_k|^{1 / k}. \] where we adopt the convention that \(1 / 0 = \infty\) and \(1 / \infty = 0\).

It is guaranteed that 1. If \(|z| < R\), then \(f\) converges absolutely. 2. If \(|z| > R\), then \(f\) diverges.

Remark: The region \(|z| < R\) is called the disc of convergence of \(f\).

Proof.
If \(|z| < R\), then \(L \cdot |z| < 1\). Hence, \[ \exists \epsilon > 0, \, r < 1, \, \text{ s.t. } \, (L + \epsilon) \cdot |z| < r. \]

Since \(\sup_{k \ge n} | a_k |^{1 / k} \rightarrow L\), for large enough \(n\), we have \[ L + \epsilon \ge \sup_{k \ge n} | a_k |^{1 / k} \ge |a_n|^{1 / n}. \]

Therefore, \[ | a_n z^n | = \big( |a_n|^{1 / n} \cdot |z| \big)^n \le \big( (L + \epsilon) \cdot |z| \big)^n \le r^n. \]

It follows that \(f\) converges absolutely.

If \(|z| > R\), then \(L \cdot |z| > 1\). Hence, \[ \exists \epsilon > 0, \, r > 1, \, \text{ s.t. } \, (L - \epsilon) \cdot |z| > r. \]

Since \(\sup_{k \ge n} | a_k |^{1 / k}\) decreases to \(L\), for each \(n \in \mathbb{N}^+\), \[ \exists k \ge n, \, \text{ s.t. } \, |a_k|^{1 / k} \ge (L - \epsilon). \]

For such \(k\), \[ | a_k z^k | = \big( |a_k|^{1 / k} \cdot |z| \big)^k \ge \big( (L - \epsilon) \cdot |z| \big)^k \ge r^k. \]

Since the terms of \(f\) are unbounded, the series is consequently divergent.

\(\square\)

Theorem. Let \(f : \mathbb{C} \rightarrow \mathbb{C}\) be a power series defined as \(f(z) = \sum_{n \ge 1} a_n z^n\). Then it is holomorphic in its disc of convergence. Its derivative is given by \[ f'(z) = \sum_{n \ge 1} n a_n z_n^{n - 1}. \] Moreover, \(f'(z)\) has the same disc of convergence.

Proof. Let \(b_n = n a_n\). Define \[ g(z) = \sum_{n \ge 1} b_n z_n^{n - 1} = \sum_{n \ge 1} n a_n z_n^{n - 1}. \] Observe that \[ \begin{aligned} \limsup_n | b_n |^{ 1 / n} &= \limsup_n | n a_n |^{ 1 / n} \\ &= \lim_n | n |^{ 1 / n} \cdot \limsup_n | a_n |^{ 1 / n} \\ &= \limsup_n | a_n |^{ 1 / n}. \end{aligned} \]

It follows that \(g(z)\) has the same disc of convergence as \(f(z)\).

It is left to prove that \(g(z)\) is the derivative of \(f(z)\). For \(k \in \mathbb{N}^+\), write \(f(z) = p_k (z) + r_k(z)\), where \[ p_k(z) = \sum_{n \in [k] } a_n z^n,\quad r_k(z) = \sum_{n > k} a_n z^n. \]

Denote \(R\) the radius of convergence of \(f(z)\). and suppose \(0 \le |z_0| < r < R\). Then for \(h \in \mathbb{C}\), such that \(|z_0 + h| < r\), we have \[ \begin{aligned} \left| \frac{ f(z_0 + h) - f(z_0) }{ h } - g(z_0) \right| &= \left| \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } + \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } - g(z_0) \right| \\ &= \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } + \left( \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } - p_k'(z_0) \right) + p_k'(z_0) - g(z_0 ) \right| \\ &\le \left| \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } - p_k'(z_0) \right| + \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| + \left| p_k'(z_0) - g(z_0 ) \right| . \end{aligned}. \]

Claim. \(\forall \epsilon > 0\), \(\exists k \in \mathbb{N}^+\), s.t., \[ \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| \le \epsilon, \quad \left| p_k'(z_0) - g(z_0 ) \right| \le \epsilon. \] Further, for such integer \(k\), \(\exists h > 0\), s.t., \[ \left| \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } - p_k'(z_0) \right| \le \epsilon. \]

Case One \(\left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| \le \epsilon\): since \(a^n - b^n = (a - b) \left( a^{n - 1} + a^{n - 2} b^2 + \ldots + b^{n - 1} \right),\) and \(|z_0 + h| < r\), \(|z_0 | < r\), \[ \begin{aligned} \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| &= \left| \frac{ \sum_{n > k} a_n \Big[ (z_0 + h)^n - z ^n \Big] }{ h } \right| &\le \sum_{n > k} a_n n r^{n - 1}. \end{aligned} \]

Since \(g(r)\) converges absolutely, for large enough \(k\), \[ \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| \le \sum_{n > k} a_n n r^{n - 1} \le \epsilon. \]

Case Two \(\left| p_k'(z_0) - g(z_0 ) \right| \le \epsilon\): since \(g(z_0)\) converges absolutely, for large enough \(k\), \[ \left| p_k'(z_0) - g(z_0 ) \right| \le \epsilon. \]

Case Three \(\left| \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } - p_k'(z_0) \right| \le \epsilon\): following the definition of derivative, this is true for small enough \(h\).

\(\square\)

Reference

[1] L. V. Ahlfors, Complex analysis: an introduction to the theory of analytic functions of one complex variable, 3d ed. New York: McGraw-Hill, 1979.
