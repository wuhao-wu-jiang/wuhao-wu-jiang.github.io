The Number Of Empty Bins.

Posted on 2021-02-27

Problem

There are nnn balls and mmm bins, where m≫nm \gg nm≫n. Each ball goes into a bin uniformly and independently at random. What is the number of empty bins?

Analysis

  1. The probability of a particular bin being empty is (1−1/m)n(1 - 1 / m)^n(1−1/m)n.

    Proof: The probability of a particular ball not falling into this bin is 1−1/m1 - 1 / m1−1/m. Therefore,

    Pr⁡[none of the balls falls into this bin]=(1−1m)n. \Pr[ \text{none of the balls falls into this bin} ] = \left( 1 - \frac{1}{m} \right)^n. Pr[none of the balls falls into this bin]=(1−m1​)n.

  2. The expected of empty bins is m⋅(1−1/m)nm \cdot (1 - 1 / m)^nm⋅(1−1/m)n.

    Proof: For j∈[m]j \in [m]j∈[m], define the indicator random variable ZjZ_jZj​ that equal to 111 if the jjj-th bin is empty. Then by linearity of expectation,

    E[∑j∈[m]Zj]=∑j∈[m]E[Zj]=m⋅(1−1m)n. \mathbb{E} \left[ \sum_{j \in [m] } Z_j \right] = \sum_{j \in [m] } \mathbb{E} [ Z_j ] = m \cdot \left( 1 - \frac{1}{m} \right )^n. E⎣⎢⎡​j∈[m]∑​Zj​⎦⎥⎤​=j∈[m]∑​E[Zj​]=m⋅(1−m1​)n.

  3. With probability at least 1−δ1 - \delta1−δ (δ∈(0,1)\delta \in (0, 1)δ∈(0,1)), the number of empty bin is

    m⋅(1−1/m)n±n2ln⁡2δ. m \cdot (1 - 1 / m)^n \pm \sqrt{ \frac{n}{2} \ln \frac{2}{\delta} }. m⋅(1−1/m)n±2n​lnδ2​​.

    Proof: We need to use McDiarmid inequality.

    McDiarmid inequality ( [1] ). Consider an arbitrary set D\mathcal{D}D. The function f:Dn→Rf: \mathcal{D} ^n \rightarrow \mathbb{R}f:Dn→R has bounded difference if there exits nonnegative constants b1,...,bn∈Rb_1, ..., b_n \in \mathbb{R}b1​,...,bn​∈R such that for i∈[n]i \in [n]i∈[n],

    sup⁡x1,...,xn,xi′∈D∣f(x1,...,xn)−f(x1,...,xi−1,xi′,xi+1,...,xn)∣≤bi. \sup_{ x_1, ..., x_n , x_i' \in \mathcal{D} } | f(x_1, ..., x_n ) - f( x_1, ..., x_{i - 1}, x_i', x_{i + 1}, ..., x_n ) | \le b_i. x1​,...,xn​,xi′​∈Dsup​∣f(x1​,...,xn​)−f(x1​,...,xi−1​,xi′​,xi+1​,...,xn​)∣≤bi​.

    Pr⁡[Y−E[Y]≥ϵ]≤exp⁡(−2ϵ2∑i∈[n]bi2). \Pr [ Y - \mathbb{E} [ Y ] \ge \epsilon ] \le \exp \left( - \frac{2 \epsilon^2 }{ \sum_{i \in [n]} b_i^2 }\right). Pr[Y−E[Y]≥ϵ]≤exp(−∑i∈[n]​bi2​2ϵ2​).

    Further, the outcome of the iii-th ball can affect the number of empty bins by at most 111:

    sup⁡x1,...,xn,xi′∈D∣f(x1,...,xn)−f(x1,...,xi−1,xi′,xi+1,...,xn)∣≤1. \sup_{ x_1, ..., x_n , x_i' \in \mathcal{D} } | f(x_1, ..., x_n ) - f( x_1, ..., x_{i - 1}, x_i', x_{i + 1}, ..., x_n ) | \le 1. x1​,...,xn​,xi′​∈Dsup​∣f(x1​,...,xn​)−f(x1​,...,xi−1​,xi′​,xi+1​,...,xn​)∣≤1.
