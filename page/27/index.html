Jaccard Similarity

Posted on 2019-05-17 Edited on 2019-05-19

The Jaccard similarity measures the fraction of shared elements between set. In particular, given set AAA and BBB, the Jaccard similarity between them is defined as

J(A,B)=∣A∩B∣∣A∪B∣J(A, B) = \frac{|A \cap B|}{|A \cup B|} J(A,B)=∣A∪B∣∣A∩B∣​

Lemma One

X={1, if x∈ A∩B0, otherwiseX = \begin{cases} 1, \ if \ x \in \ A \cap B \\ 0, \ otherwise \end{cases} X={1, if x∈ A∩B0, otherwise​

Suppose that we have a sequence of i.i.d. copies of XXX, denoted X1,X2,...,XkX_1, X_2, ..., X_kX1​,X2​,...,Xk​, then the average μ=1k∑i=1kXi\mu = \frac{1}{k} \sum_{i = 1}^k X_iμ=k1​∑i=1k​Xi​ is also an unbiased estimator of J(A,B)J(A, B)J(A,B).   1. Whether the value of the required error is relative to J(A,B)J(A, B)J(A,B) or independent to J(A,B)J(A, B)J(A,B).
  2. What is the tolerance of failure probability.
