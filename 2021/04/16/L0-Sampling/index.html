L0 Sampling

Posted on 2021-04-16 Edited on 2021-04-17

Problem

Let \(x \in \R^n\) be a vector initialized with all zeros. There is a stream \(S\) of updates, each of which is a pair \((t, \Delta) \in [n] \times \R\), causing \(x_t \leftarrow x_t + \Delta\).

Define the support \(\supp(x)\) to be the set of all non-zero indexes of \(x\): \[ \supp(x) \doteq \{ i \in [n] : x_i \neq 0 \}. \]

\(\ell_0\) sampling: output an index uniformly at random from the support \(\supp(x)\).

We assume that at any time, \(\forall t \in [n], x_t \le M\) for some \(M \in \R\).

Naive Solutions

We can uses an array to keep \(x\) explicitly. But this has size \(O( n \cdot \log M)\) bits.

Goal: design an algorithm for \(\ell_0\) sampling with space less than \(O( n \cdot \log M)\) bits.

If there are only insertion updates, i.e., \(\forall (t, \Delta) \in S\), it holds that \(\Delta > 0\), then we can use a hash set to keep the non-zero elements and their frequencies, which has size \(O( |\supp(x) | \cdot \log M)\). If \(|\supp(x)|\) is large, this is still infeasible.

Sampling Game

We discuss one sampling game that illustrates the key ideas used in \(\ell_0\)-sampling.

In this game, we want to sample one integer uniformly at random from \([k]\). Instead of sampling it directly, we choose some \(p \in (0, 1]\) and perform the following two steps:

  1. First, we sample each \(i \in [k]\) uniformly and independent at random with probability \(p\). Denote \(B\) the set of sampled elements.
  2. If \(B \neq \empty\), then output one element from \(B\) uniformly at random.

Property. \(\E [ |B| ] = kp\).

Theorem. Each element in \([k]\) is outputted with equal probability \[ \frac{1 - \left(1 - p \right)^k }{ k }. \]

Remark: when \(p = 1\), this reduces to sampling an element uniformly at random from \([k]\).

Proof. The probability that \(B = \emptyset\) is given by \((1 - p)^k\). Conditioned on that \(B \neq \empty\), by symmetry, each element is sampled with equal probability.

\(\square\)

Remark: We can also have a proof by algebra. For evert \(i \in [k]\), the probability that it is sampled is given by \[ \begin{aligned} \sum_{t = 1}^k \frac{1}{t} \cdot \Pr [ i \in B \mid |B| = t] \cdot \Pr[ |B| = t] &= \sum_{t = 1}^k \frac{1}{t} \frac{ \binom{k - 1}{t - 1} }{ \binom{k}{t} } \binom{k}{t} p^t (1 - p)^{k - t} \\ &= \sum_{t = 1}^k \frac{1}{t} \binom{k - 1}{t - 1} p^t (1 - p)^{k - t} \\ &= \frac{1}{k} \sum_{t = 1}^k \binom{k}{t} p^t (1 - p)^{k - t} \\ &= \frac{1}{k} \left( 1 - p^0 (1 - p)^{k - 0} \right) \\ &= \frac{1 - \left(1 - p \right)^k }{ k }. \end{aligned} \]
