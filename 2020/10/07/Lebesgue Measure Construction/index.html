Lebesgue Measure Construction

Posted on 2020-10-07 Edited on 2021-09-02

Measure Space

Let \(\Omega\) be a set and \(2^\Omega\) be the power set of \(\Omega\).

\(\sigma\)-Algebra

Definition. A collection of subsets \(\mathcal{F} \subset 2^\Omega\) is a \(\sigma\)-algebra (or \(\sigma\)-field) if

  • It is closed under countable union, intersection and complement.

Measure

Definition. Given a collection of subsets \(\mathcal{F} \subset 2^\Omega\), a function \(\mu : \mathcal{F} \rightarrow \mathbb{R}\) is a measure if it satisfies

  1. It is non-negative: \(\forall A \in \mathcal{F}, \mu(A) \ge 0\).
  2. The empty set is mapped to 0: \(\mu(\emptyset) = 0\).
  3. \(\sigma\)-additivity: if \(A_i \in \mathcal{F}\) is a countable sequence of disjoint sets, s.t., \(\cup_{i \ge 1} A_i \in \mathcal{F}\), then
    \[ \mu(\cup_{i \ge 1} A_i) = \sum_{i \ge 1} \mu(A_i ). \]

Measure Space

Definition. A measure space \((\Omega, \mathcal{F}, \mu)\) consists of three part:

  1. A set \(\Omega\), on which it is defined;
  2. A \(\sigma\)-field \(\mathcal{F}\) that contains all measurable sets;
  3. A measure \(\mu\) defined on \(\mathcal{F}\).

In general, it is not easy to construct a measure space on \(\Omega\) directly. Our indirect construction consists of three steps:

  1. Consider a simple set class \(S\) and a measure \(\mu\) defined on \(S\).
  2. We extend the domain of \(\mu\) from \(S\) to a larger one \(\Omega=2^\Omega\).
  3. We choose a subset from \(2^\Omega\), such that it is a \(\sigma\)-field and \(\mu^*\) is a measure defined on it.

In what follows, we illustrate the idea via constructing measures on \(\mathbb{R}^d\).

Semialgebra

Definition. A class of subsets \(S \subset 2^\Omega\) is a semialgebra if

  1. It is closed under intersection: \(\forall A, B \in S\), \(A \cap B \in S\).
  2. If \(A \in S\), then \(\bar A\) is a finite disjoint union of sets in \(S\).

Examples. For \(d \in \mathbb{Z}^+\), the collection of half-open intervals in \(\mathbb{R}^d\) plus the empty set \[ S_d \doteq \{ \times_{i \in [d] } (a_i, b_i] : -\infty \le a_i \lt b_i \le \infty, \forall i \in [d] \} \cup \{ \emptyset \} \]

Algebra

Definition. A class of subsets \(\mathcal{F} \subset 2^\Omega\) is an algebra (or field) if

  • It is closed under union, intersection and complement.

Remark. By De Morgan's laws, \(\overline{A \cap B } = \bar A \cup \bar B\) and \(\overline{A \cup B} = \bar A \cap \bar B\). If we know a collection is closed under intersection (union) and complement, so is union (intersection).

Theorem If \(S \subset \Omega\) is a semialgebra, then \[ \widetilde{S} \doteq \{ \text{finite disjoint union of sets in } S \} \] is an algebra.

Proof.
1. \(\widetilde{S}\) is closed under intersection: Let \(A, B \in \widetilde{S}\) with \(A = +_i A_i\) and \(B = +_j B_j\), where \(+\) denotes disjoint union and \(A_i, B_j \in S\). Then \(A \cap B = +_{i, j} A_i \cap B_j\). As \(S\) is a semialgebra, it follows that \(A_i \cap B_j \in S\) and \(A \cap B \in \widetilde{S}\).

  1. \(\widetilde{S}\) is closed under complement. Let \(A \in \widetilde{S}\) with \(A = +_i A_i\), where \(A_i \in S\). Then \(\bar A = \cap_i \bar A_i\). As \(S\) is a semialgebra, \(\bar A_i\) is a finite disjoint union of sets in \(S\), which implies that \(\bar A_i \in \widetilde{S}\). Therefore \(\bar A\) is a finite intersection of sets in \(\widetilde{S}\). Since \(\widetilde{S}\) is closed under intersection, \(\bar A \in \widetilde{S}\).

\(\square\)

Measure Extension Theorems

From Semialgebra to Algebra

Theorem. Let \(S\) be a semialgebra and \(\mu\) be a measure on \(S\). Then \(\mu\) has a unique extension \(\bar \mu\) that is a measure on \(\widetilde S\), the algebra generated by \(S\).

Remark. The condition of \(\mu\) being a measure can be replaced by
1. Finite additivity. If \(A, A_i \in S\) with \(A = +_i A_i\), then \(\mu(A) = \sum_i \mu (A_i)\).
2. Subadditivity. If \(A, A_i \in S\) with \(A = +_{i \ge 1} A_i\), then \(\mu (A) \le \sum_{i \ge 1} \mu (A_i)\).

The subscript \(i \ge 1\) in the conditions indicates a countable union, while a plain subscript \(i\) or \(j\) indicates a finite union.

The relation between the condition stated in the Theorem and the relaxed ones is established as follows.

Exercise. A finitely additive nonnegative set function on an algebra \(\widetilde{S}\) is monotone and the following conditions are equivalent 1. \(\sigma\)-additivity. 2. Subadditivity.

The proof for the Theorem consists of two steps.

\(\blacktriangleright\) Defining \(\bar \mu\)

If \(\mu\) is finitely additive in \(S\), then it has a finitely additive extension \(\bar \mu\) in \(\widetilde{S}\).

  1. Extension. If \(A = +_i A_i \in \widetilde{S}\) with \(A_i \in S\), define \[ \bar \mu(A) = \sum_i \mu (A_i). \]

    We need to verify the value of \(\bar \mu(A)\) is unique. Suppose \(A\) has another decomposition: \(A = +_j B_j\) with \(B_j \in S\). Since \(S\) is a semialgebra, \(A_i \cap B_j \in S\). Using that \(\mu\) is a measure \[ \sum_i \mu (A_i) = \sum_{i, j} \mu(A_i \cap B_j) = \sum_j \mu(B_j). \]

    It follows that \(\bar \mu(A)\) does not change with the decomposition of \(A\) and hence is well-defined.

  2. Finite additivity. Let \(A = +_i A_i \in \widetilde{S}\) with \(A_i \in \widetilde{S}\) and suppose that \(A_i = +_j B_{i,j}\) with \(B_j \in S\). Then \(A = +_{i,j} B_{i,j}\). By definition of \(\bar \mu (A)\) and \(\bar \mu (A_i)\), \[ \bar \mu (A) = \sum_{i,j} \mu (B_{i,j}) = \sum_{i} \bar \mu ( A_i ). \]

We list two more properties about \(\bar \mu\) based on finite additivity.

  • Monotonicity. If \(A, B \in \widetilde{S}\) with \(A \subset B\), then \(\bar \mu (A) \le \bar \mu (B)\). As \(\widetilde{S}\) is an algebra, \(B - A \in \widetilde{S}\) and by finite additivity of \(\bar \mu\), \[ \bar \mu (A) \le \bar \mu (A) + \bar \mu (B - A) = \bar \mu (B). \]

  • Finite subadditivity. If \(A \subset \cup_{i \in [n] } A_i\) with \(A, A_i \in \widetilde{S}\), then \(\bar \mu(A) \le \sum_i \bar \mu (A_i)\). Let \(B_1 = A_1\) and \(B_i = A_i - A_{i - 1} - \ldots - A_1\) for \(i \ge 1\). As \(\widetilde{S}\) is an algebra, \(B_i \in \widetilde{S}\). By monotonicity and finite additivity of \(\bar \mu\), \[ \bar \mu (A) \le \bar \mu( +_i B_i ) = \sum_i \bar \mu (B_i) \le \sum_i \bar \mu (A_i). \]

\(\square\)

\(\blacktriangleright\) \(\sigma\)-Additivity of \(\bar \mu\)

If \(A, A_1, \ldots \in \widetilde S\) with \(A = +_{i \ge 1} A_i\), then \[\small \bar \mu (A) = \sum_{i \ge 1} \bar \mu (A_i). \]

Proof. Let \(A_i = +_j B_{i,j}\) with \(B_{i,j} \in S\). Then \(A = +_{i,j} B_{i,j}\) and \(\sum_{i \ge 1} \bar \mu(A_i) = \sum_{i \ge 1} \sum_j \mu ( B_{i,j} )\). WLOG, we assume that \(A_i\) are countable union of disjoint sets in \(S\).

Next, assume that \(A = +_k C_k\) with \(C_k \in S\). Using that \(\mu\) is a measure and \(S\) is closed under intersection, \[ \mu(C_k) = \sum_{i \ge 1} \mu (C_k \cap A_i). \]

Observing that nonnegative numbers can be summed in any order, \[ \small \begin{aligned} \bar \mu(A) = \sum_k \mu (C_k) = \sum_k \sum_{i \ge 1} \mu (C_k \cap A_i) = \sum_{i \ge 1} \sum_k \mu (C_k \cap A_i) = \sum_{i \ge 1} \mu (A_i). \end{aligned} \]

\(\square\)

From Algebra to \(\sigma\)-Algebra

Definition. \(\bar \mu\) is said to be \(\sigma\)-finite if there is a sequence of sets \(A_i \in \widetilde{S}\) such that \(\mu(A_i) < \infty\) and \(\cup_{i \ge 1} A_i = \Omega\).

Caratheodory’s Extension Theorem. Let \(\bar \mu\) be a \(\sigma\)-finite measure on an algebra \(\widetilde S\). Then \(\bar \mu\) has a unique extension \(\mu^*\) (that is a measure) to \(\sigma(\widetilde S)\), the smallest \(\sigma\)-field containing \(\widetilde S\).

Remark: Observe that \(\sigma(\widetilde S) = \sigma(S)\).

The proof is included at the end of the article.

Lebesgue-Stieltjes Measure

In this section, we construct concrete measures on \(\mathbb{R}^d\).

Definition. A function \(F: \mathbb{R} \rightarrow \mathbb{R}\) is a Stieltjes measure function if
1. \(F\) is non-decreasing.
2. \(F\) is right continuous, i.e., \(\lim_{y \downarrow x} F(y) = F(x)\).

Definition (Borel Sets). Let \(\mathcal{R}^d = \sigma (S_d)\). It is the smallest \(\sigma\)-field containing all open sets in \(\mathbb{R}^d\).

Theorem 3. Associated with each Stieltjes measure function \(F\) there is a unique measure \(\mu\) on \((\mathbb{R},\mathcal{R}^1)\) with \[ \mu \big( (a, b] \big) = F(b) - F(a). \] When \(F(x) = x\), the measure \(\mu\) is called Lebesgue measure.

Proof. We just to verity that \(\mu\) is a measure defined on \[ S_1 \doteq \{ (a, b] : -\infty \le a \lt b \le \infty \} \cup \{ \emptyset \}. \]

  1. \(\mu((a, b] ) \ge 0\), since \(F\) is non-decreasing.

  2. Finite additivity. If \((a, b] = +_i (a_i, b_i]\), \(\mu \big( (a, b] \big) = \sum_i \mu \big( ( a_i, b_i ] \big)\). We can rearrange the \(\{ (a_i, b_i] \}\), such that \(a = a_1\), \(b_i = a_{i + 1}\) for all \(1 \le i < n\), and \(b_n = b\). Hence, \[ \mu \big( (a, b] \big) = F(b) - F(a) = \sum_i ( F(b_i) - F(a_i) ) = \sum_i \mu \big( ( a_i, b_i ] \big). \]

  3. Finite subadditivity. If \((a, b] \subset \cup_{i \in [n] } (a_i, b_i]\), \(\mu \big( (a, b] \big) \le \sum_{i \in [ n ] } \mu \big( ( a_i, b_i ] \big)\). Recall that the definition of \(\bar \mu\) only requires finite additivity of \(\mu\). Working with \(\widetilde{S_1}\) and \(\bar \mu\) significantly simplifies the proof. As for all \(A \in S_1\), it holds that \(\mu(A) = \bar \mu(A)\). It suffices to prove that \[ \bar \mu \big( (a, b] \big) \le \sum_{i \in [ n ] } \bar \mu \big( (a_i, b_i] \big). \] But this follows exactly from finite subadditivity of \(\bar \mu\).

  4. \(\sigma\)-additivity. If \((a, b] = +_{i \ge 1} (a_i, b_i]\), then \(\mu \big( (a, b] \big) = \sum_{i \ge 1} \mu \big( ( a_i, b_i ] \big)\). Using finite subadditivity of \(\mu\), for all \(n \in \mathbb{Z}^+\), \[ \mu \big( (a, b] \big) \ge \sum_{i \in [n] } \mu \big( ( a_i, b_i ] \big). \] Letting \(n \rightarrow \infty\), we get \(\mu \big( (a, b] \big) \ge \sum_{i \ge 1} \mu \big( ( a_i, b_i ] \big)\).

    It is left to prove the other direction of the inequality. If \(\mu \big( ( a_i, b_i ] \big) = \infty\) for some \(i\), then the inequality is trivial. WLOG, assume that \(-\infty < a_i < b_i < \infty\) for all \(i\).

    First, suppose that both \(a\) and \(b\) are finite. For each \(\epsilon > 0\), as \(F\) is right continuous, there exists some \(\delta > 0\), such that \[ \begin{aligned} F(a + \delta) &\le F(a) + \epsilon\,,\qquad \text{ and } \\ \mu \Big( (a, b] \Big) &\le \mu \Big( (a + \delta, b] \Big) + \epsilon. \end{aligned} \] For each \(i\), there exists some \(\delta_i > 0\), such that \[ \begin{aligned} F(b_i + \delta_i) &\le F(b_i) + \epsilon / 2^i\,,\qquad \text{ and } \\ \mu \Big( (a_i, b_i + \delta_i] \Big) &\le \mu \Big( (a_i, b_i] \Big) + \epsilon / 2^i. \end{aligned} \] Since \(\{ (a_i, b_i + \delta_i) \}\) is an open cover of the closed interval \([a + \delta, b]\), there is a finite sub-cover \(\{ (s_i, t_i) \}\). It follows that \(\{ (s_i, t_i] \}\) covers \((a + \delta, b]\). Therefore, \[ \mu \Big( (a + \delta, b] \Big) \le \sum_i \mu \Big( (s_i, t_i] \Big) \le \sum_{i \ge 1} \mu \Big( (a_i, b_i + \delta_i] \Big) \] It follows that \[ \mu \Big( (a, b] \Big) \le \sum_{i \ge 1} \mu \Big( (a_i, b_i] \Big) + 2 \epsilon. \] Letting \(\epsilon \rightarrow 0\) finishes the proof.

    If \(a = -\infty\) or \(b = \infty\), the inequality \(\small \mu \Big( (a', b'] \Big) \le \sum_{i \ge 1} \mu \Big( (a_i, b_i] \Big)\) holds for all finite subset \((a', b'] \subset (a, b]\). Hence it holds for \((a, b]\).

\(\square\)

Caratheodory’s Extension Theorem

Caratheodory’s Extension Theorem. Let \(\bar \mu\) be a \(\sigma\)-finite measure on an algebra \(\widetilde S\). Then \(\bar \mu\) has a unique extension \(\mu^*\) (that is a measure) to \(\sigma(\widetilde S)\), the smallest \(\sigma\)-field containing \(\widetilde S\).

\(\spadesuit\) Existence

Define the function \(\mu^*: 2^\Omega \rightarrow \mathbb{R}_{\ge 0}\) as \(\forall A \subset \Omega\), \[ \mu^*(A) = \inf \left\{ \sum_{i \ge 1} \bar{ \mu }(A_i) : A \subset \cup_{i \ge 1} A_i \text{ and } A_i \in \widetilde{S}\,,\forall i \right\} \]

Intuitively, \(\mu^*\) is an upper bound on the measure of \(A\). The following properties follows immediately form the definition.

  1. \(\mu^*(\emptyset) \le \sum_{i = 1}^\infty \bar{ \mu }(\emptyset) = 0\).

  2. Subadditivity. If \(A \subset \cup_{i \ge 1} A_i\), then \(\mu^*(A) \le \sum_{i \ge 1} \mu^*(A_i)\). Given \(\epsilon > 0\), for each \(A_i\), there exists a countable sequence \(\{ B_{i, j} : j \in \mathbb{Z}^+ \}\) in \(\widetilde{S}\) such that it covers \(A_i\) and \[ \sum_{j \ge 1} \bar{ \mu }(B_{i, j}) \le \mu^*(A_i) + \epsilon / 2^i. \] The sequence \(\{ B_{i, j} : i, j \in \mathbb{Z}^+ \}\) is countable and covers \(A\). Further \[ \mu^*( A ) \le \sum_{i \ge 1} \sum_{j \ge 1} \bar{ \mu }(B_{i, j}) \le \sum_{i \ge 1} \mu^* (A_i) + \epsilon. \] The inequality holds for all \(\epsilon > 0\) hence it holds that \[ \mu^*( A ) \le \sum_{i \ge 1} \mu^* (A_i). \]

Now, \(\mu^*\) is almost a measure: it is an nonnegative set function and satisfies subadditivity. By previous analysis, we know that

Given a nonnegative, finitely additive set function defined on an algebra, the following conditions are equivalent * \(\sigma\)-additivity. * \(\sigma\)-subadditivity.

If we can prove that \(\mu^*\) is finitely additive, then it is a measure on the \(\sigma\)-algebra, \(2^\Omega\) (which is also an algebra). However, in general, finite additivity does not hold on \(2^\Omega\). Therefore, we restrict our discussion on a subset of \(2^\Omega\).

Measurable Set

Intuitively, the measurable sets are the ones on which finite additivity of \(\mu^*\) holds. Formally, we say that a set \(A \subset \Omega\) is called measurable if \[ \mu^* (B) = \mu^* (B \cap A) + \mu^* (B \cap \bar A),\qquad \forall B \subset \Omega. \]

Using subadditivity, to prove \(A\) is measurable, it is enough to show that \[ \mu^* (B) \ge \mu^* (B \cap A) + \mu^* (B \cap \bar A),\qquad \forall B \subset \Omega. \]

Definition. \(S^* =\) { the collection of all measurable sets }.

Theorem. \(S^*\) is a \(\sigma\)-field that contains \(\widetilde{S}\) and the restriction of \(\mu^*\) to \(S^*\) is a measure.

\(\blacktriangleright \quad S^*\) is a \(\sigma\)-field

  1. If \(A \in S^*\), then so is \(\bar A\).

    Proof. By definition.

  2. If \(A_1, A_2 \in S^*\), then \(A_1 \cup A_2 \in S^*\) and \(A_1 \cap A_2 \in S^*\).

    Proof. Observe that \[ A_1 \cup A_2 \subset (A_1 \cap \bar A_2) \cup (A_1 \cap A_2) \cup (\bar A_1 \cap A_2). \] For all \(B \subset \Omega\), it holds that \[ \begin{aligned} \mu^*(B) &= \mu^*(B \cap A_1) + \mu^* (B \cap \bar A_1) \\ &= \mu^*(B \cap A_1 \cap A_2 ) + \mu^*(B \cap A_1 \cap \bar A_2 ) \\ &\quad + \mu^* (B \cap \bar A_1 \cap A_2 ) + \mu^* (B \cap \bar A_1 \cap \bar A_2 ) \\ &\ge \mu^*(B \cap (A_1 \cup A_2) ) + + \mu^* (B \cap \bar A_1 \cap \bar A_2 ) \end{aligned} \] The last inequality follows form subadditivity of \(\mu^*\). Hence \(A_1 \cup A_2 \in S^*\). Finally, \(A_1 \cap A_2 = \overline{ \bar A_1 \cup \bar A_2} \in S^*\).

    \(\square\)

  3. If \(A_i \in S^*\), then \(A = \cup_{i \ge 1} A_i \in S^*\).

    Proof. We have proven that \(S^*\) is an algebra. Let \(B_1 = A_1\) and \(B_i = A_i - A_{i - 1} - \ldots - A_1\) for \(i \ge 2\). It holds that \(B_i \in S^*\) and \(\cup_i A_i = \cup B_i\). Replacing \(A_i\) with \(B_i\) we can assume without lose of generality that the \(A_i\) are disjoint. For all \(n \in \mathbb{Z}^+\), let \(C_n = +_{i \in [n] } A_i\). For all \(B \subset \Omega\) and \(n > 1\), \[ \begin{array}{ll} \mu^*(B) &= \mu^* (B \cap C_n ) + \mu^* (B \cap \bar C_n ) \\ &= \mu^* (B \cap C_n \cap A_n ) + \mu^* (B \cap C_n \cap \bar A_n ) + \mu^* (B \cap \bar C_n ) \\ &= \mu^* (B \cap A_n ) + \mu^* (B \cap C_{n - 1} ) + \mu^* (B \cap \bar C_n ) \\ \end{array} \] The last equality follows from that \(C_n \cap A_n = A_n\) and \(C_n \cap \bar A_n = C_{n - 1}\). By induction, and subadditivity of \(\mu^*\), we get \[ \begin{array}{ll} \mu^*(B) &= \sum_{i \in [n] } \mu^* (B \cap A_i ) + \mu^* (B \cap \bar C_n ) \\ &\ge \sum_{i \in [n] } \mu^* (B \cap A_i ) + \mu^* (B \cap \bar A ) \end{array} \] Letting \(n \rightarrow \infty\) finishes the proof.

    \(\square\)

\(\blacktriangleright \quad S^* \supset \widetilde{S}\)

Proof. For all \(B \subset \Omega\), and all \(\epsilon > 0\), there exists a sequence \(\{ B_i \}\) in \(\widetilde{S}\), such that \(B \subset \cup_i B_i\) and \[ \begin{array}{ll} \mu^* (B) + \epsilon \ge \sum_{i \ge 1} \bar \mu (B_i). \end{array} \] Let \(A \in \widetilde{S}\). Observe that \(B_i \cap A, B_i \cap \bar A \in \widetilde{S}\), and that \[ \begin{array}{ll} B \cap A \subset \cup_i (B_i \cap A)\,, \qquad B \cap \bar A \subset \cup_i (B_i \cap \bar A). \end{array} \] By the definition of \(\mu^*\) and finite additivity of \(\bar \mu\), it holds that \[ \small \begin{aligned} \mu^* (B \cap A) + \mu^* (B \cap \bar A) \le \sum_{i \ge 1} \bar \mu (B_i \cap A) + \sum_{i \ge 1} \bar \mu (B_i \cap \bar A) = \sum_{i \ge 1 } \bar \mu (B_i). \end{aligned} \]

Letting \(\epsilon \rightarrow 0\), it concludes that \[ \mu^*(B) \ge \mu^* (B \cap A) + \mu^* (B \cap \bar A). \]

\(\square\)

\(\blacktriangleright \quad \mu^*\) is a measure restricted to \(S^*\)

Sigma additivity of \(\mu^*\). Let \(A = +_{i \ge 1} A_i\), with \(A_i \in S^*\), then \[ \mu^*(A) = \sum_{i \ge 1 } \mu^*( A_i ). \]

Proof. As \(S^*\) is an algebra and \(\mu^*\) is \(\sigma\)-subadditive, it suffices to prove that \(\mu^*\) is finitely additive. Let \(A = +_{i \in [n] } A_i\). \[ \begin{aligned} \mu^*(A) &= \mu^*(A \cap A_n) + \mu^*(A \cap \bar A_n) \\ &= \mu^*(A_n) + \mu^*( +_{i \in [n - 1] } A_i ) \\ &\ldots\, \\ &= \sum_{ i \in [n] } \mu^* (A_i). \end{aligned} \]

\(\square\)

\(\spadesuit\) Uniqueness

\(\blacktriangleright\,\,\pi\)-system

Definition. \(\mathcal{P}\) is said to be a \(\pi\)-system if it is closed under intersection.

\(\blacktriangleright\,\,\lambda\)-system

Definition. \(\mathcal{L}\) is said to be a \(\lambda\)-system if (i) \(\Omega \in \mathcal{L}\). (ii) If \(A, B \in \mathcal{L}\) with \(A \subset B\), then \(B - A \in \mathcal{L}\). (iii) If \(A_n \in \mathcal{L}\) with \(A_n \uparrow A\), then \(A \in \mathcal{L}\).

Property

  • \(\mathcal{L}\) is closed under complement: if \(A \in \mathcal{L}\), since \(\Omega \in \mathcal{L}\), then \(\bar A = \Omega - A \in \mathcal{L}\).

Lemma. If a \(\lambda\)-system \(\mathcal{L}\) is closed under intersection, then it is a \(\sigma\)-field.

Proof. Since it is closed under complement and intersection, it is an algebra. It is a \(\sigma\)-field as it satisfies monotonicity.

\(\square\)

\(\blacktriangleright\,\,\) \(\pi-\lambda\) Lemma.

\(\pi-\lambda\) Lemma. If \(\mathcal{P}\) is a \(\pi\) system and \(\mathcal{L}\) is a \(\lambda\)-system that contains \(\mathcal{P}\), then \(\sigma(\mathcal{P}) \subset \mathcal{L}\).

Proof. Let \(\lambda(\mathcal{P})\) the smallest \(\lambda\)-system that contains \(\mathcal{P}\). We will show that \(\lambda(\mathcal{P})\) is a \(\sigma\)-field. Then \[ \sigma(\mathcal{P}) \subset \lambda(\mathcal{P}) \subset \mathcal{L}. \]

It suffices to prove that

  • \(\triangleright\) \(\lambda(\mathcal{P})\) is closed under intersection.

​For all \(A \in \lambda (\mathcal{P})\), define \[ \Pi_A \doteq \{ B : B \cap A \in \lambda (\mathcal{P}) \}. \]

We will prove that \(\Pi_A\) is a \(\lambda\)-system that contains \(\mathcal{P}\). It follows that \(\Pi_A \supset \lambda(\mathcal{P})\) and \(\lambda(\mathcal{P})\) is closed under intersection.

\(\Pi_A\) is a \(\lambda\)-system:

  1. \(\Omega \in \Pi_A\).
  2. If \(B, C \in \Pi_A\) with \(B \subset C\), then \(C - B \in \Pi_A\), since \((C - B) \cap A = C \cap A - B \cap A \in \lambda(\mathcal{P})\).
  3. If \(B_n \in \Pi_A\) with \(B_n \uparrow B\), then \(B \in \Pi_A\), since \((B_n \cap A) \uparrow (B \cap A) \in \lambda (\mathcal{P})\) .

\(\Pi_A\) contains \(\mathcal{P}\):

If \(B \in \mathcal{P}\), denote \(\Pi_B = \{ A : A \cap B \in \lambda (\mathcal{P}) \}\). Since \(\mathcal{P}\) is closed under intersection, \(\Pi_B \supset \mathcal{P}\). Since \(\Pi_B\) is a \(\lambda\)-system, \(\Pi_B \supset \lambda (\mathcal{P})\).

Hence, \(\forall A \in \lambda (\mathcal{P}), B \in \mathcal{P}\), we have \(A \cap B \in \lambda (\mathcal{P})\). Therefore \(\Phi_A \supset \mathcal{P}\).

\(\square\)

Theorem. Let \(\mathcal{P}\) be a \(\pi\)-system. If \(\nu_1\) and \(\nu_2\) are measures (on \(\sigma\)-fields \(\mathcal{F}_1\) and \(\mathcal{F}_2\)) that agree on \(\mathcal{P}\) and there is a sequence \(A_n \in \mathcal{P}\) with \(A_n \uparrow \Omega\) and \(\nu_i (A_n) < \infty\) for \(i = 1, 2\), then \(\nu_1\) and \(\nu_2\) agree on \(\sigma( \mathcal{P} )\).

Proof. Since \(\nu_i\) are measures on \(\sigma(\mathcal{P} )\), for all \(A \in \sigma(\mathcal{P})\), \[ \nu_i (A) = \lim_{n \rightarrow \infty} \nu_i (A \cap A_n). \] We prove that \(\nu_1 (A \cap A_n) = \nu_2 (A \cap A_n)\) holds for all \(n \in \mathbb{Z}^+\).

For a fixed \(n\), now consider the set
\[ \mathcal{L} = \{ B \in \sigma(\mathcal{P} ) : \nu_1 (B \cap A_n) = \nu_2 (B \cap A_n) \} \subset \sigma( \mathcal{P} ). \]

  • \(\triangleright\) \(\mathcal{P} \subset \mathcal{L}:\) this immediately follows from that \(\mathcal{P}\) is closed under intersection and that \(\nu_1\) and \(\nu_2\) agrees on \(\mathcal{P}\).

  • \(\triangleright\) \(\mathcal{L}\) is a \(\lambda\) system: (i) \(\Omega \in \mathcal{L}\), and (ii) if \(B, C \in \mathcal{L}\) with \(B \subset C\), then
    \[ \begin{aligned} \nu_1 ( (C - B) \cap A_n) &= \nu_1 ( C \cap A_n ) - \nu_1 (B \cap A_n) \\ &= \nu_2 ( C \cap A_n ) - \nu_2 (B \cap A_n) = \nu_2 ( (C - B) \cap A_n ), \end{aligned} \] and (iii) If \(B_j \uparrow B\) with \(B_j \in \mathcal{L}\), then \(B \in \mathcal{L}\). As \(\nu_i\) are measures, we have \[ \nu_1 (B \cap A_n) = \lim_{j} \nu_1 (B_j \cap A_n) = \lim_{j} \nu_2 (B_j \cap A_n) = \nu_2 (B \cap A_n). \]

Using the \(\pi-\lambda\) lemma, we get \(\mathcal{L} \supset \sigma( \mathcal{P} )\), which finishes the proof.

\(\square\)

[1]. R. Durrett, “Probability : theory and examples,” p. 432.
