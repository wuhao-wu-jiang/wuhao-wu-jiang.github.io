The Volume of Hamming Ball

Posted on 2021-04-02 Edited on 2022-12-22

Given a finite set \(\Sigma\) with size \(q = |\Sigma|\), and a integer \(n \in \mathbb{N}_+\), a code \(C \subseteq \Sigma^n\) is a set of strings that are far apart. The set \(\Sigma\) is called the alphabet of the code, and \(n\) is called the block length of the code. Strings in \(C\) are called codewords.

For \(x, y \in \Sigma^n\), define their Hamming distance as \[ \Delta(x, y) \doteq | \{ i \in [n] : x_i \neq y_i \} |. \]

The distance \(\Delta(C)\) (or \(d\) for a shorthand) of \(C\), is defined as the minimum distance between two different codes words in \(C\): \[ \Delta(C) = d \doteq \min_{x \neq y \in C} \Delta(x, y). \]

If we store a code \(x \in C\), and up to \((d - 1) / 2\) bits of \(x\) are corrupted, then we can still recover \(x\). Denote the ball centered at \(x\) with radius \(r \in \mathbb{R}_+\) as \[ \mathcal{B}(x, r) \doteq \{ y \in \Sigma^n : \Delta(y, x) \le r \}. \]

Fact. For \(x \in \Sigma^n\), \(r \in \mathbb{R}_+\), the size (volume) of the ball \(\mathcal{B}(x, r)\) is independent of \(x\) and is given by \[ \mathbb{Vol}(r) \doteq | \mathcal{B}(x, r) | = \sum_{i = 0}^{ \lfloor r \rfloor } \binom{n}{i} (q - 1)^i. \]

If the code \(C\) has distance \(\Delta(C) = d\), then the balls \(\{ \mathcal{B}(x, (d - 1) / 2) : x \in C \}\) should be disjoint. Therefore, we can upper bound the size \(|C|\) in terms of the size of \(\Sigma^n\), and the volume of the balls.

Theorem (Hamming Bound). Let \(C\) be a code with distance \(d \ge 3\). Then \[ |C| \le \frac{q^n }{ \mathbb{Vol} \Big( (d - 1) / 2 \Big) }. \]

We are going to prove an estimate of the volume.

Theorem (Estimate of Volume). Let \(r \in \mathbb{R}_+\). When \(r \le n - n / q\), \[ q^{ n h_q ( r / n) - O( \log n) } \le \mathbb{Vol} (r) \le q^{ n h_q ( \lfloor r \rfloor / n)}, \] where \[ \begin{aligned} h_q (\epsilon) &\doteq (1 - \epsilon) \log_q \left( \frac{1}{1 - \epsilon} \right) + (q - 1) \frac{\epsilon}{q - 1} \log_q \left( \frac{q - 1}{ \epsilon } \right) \\ &= (1 - \epsilon) \log_q \left( \frac{1}{1 - \epsilon} \right) + \epsilon \log_q \left( \frac{q - 1}{ \epsilon } \right). \end{aligned} \]

When \(q = 2\), we have the following corollary.

Corollary. Let \(r \in \mathbb{R}_+\). When \(r \le n / 2\), \[ 2^{ n \cdot h_2 ( r / n) - O( \log n) } \le \sum_{i = 0}^{ \lfloor r \rfloor } \binom{n}{i} \le 2^{ n \cdot h_2 ( \lfloor r \rfloor / n)}, \] where \(h_2( \cdot )\) is the Shannon's entropy function.

Observe that by concavity of \(\log_q (\cdot)\), we have \[ h_q (\epsilon) \le \log_q \left( (1 - \epsilon) \frac{1}{1 - \epsilon} + \epsilon \frac{q - 1}{ \epsilon } \right) = 1. \]

The inequality holds with equality when \(h_q(\epsilon)\) is maximized, i.e., when \(1 - \epsilon = \epsilon / (q - 1)\), i.e., \(1 - \epsilon = 1 / q\).

Rate of the Code

Based on the theorems of Hamming bound, and the estimate of the volume, we have \[ \begin{aligned} |C| \le \frac{q^n }{ \mathbb{Vol} \Big( (d - 1) / 2 \Big) } &\Longrightarrow n h_q( \lparen d / 2 - 1 / 2 \rparen / n) + \log_q |C| \le n + O(\log n) \\ &\Longleftrightarrow h_q( \lparen d / 2 - 1 / 2 \rparen / n) + \frac{ \log_q |C| }{ n } \le 1 + O\left(\frac{ \log_q n }{ n } \right) \\ &\Longrightarrow h_q( d / 2 n) + \frac{ \log_q |C| }{ n } \le 1 + O\left(\frac{ \log_q n }{ n } \right). \end{aligned} \]

The last \(\Longleftrightarrow\) holds, since \[ \begin{aligned} q^{ n \cdot h_q( \lparen d - 1 \rparen / (2n) ) } &= (q - 1)^r \left(\frac{n}{r} \right)^r \left(\frac{n}{n - r} \right)^{n - r} & \quad r \doteq (d - 1) / (2) \\ &\ge (q - 1)^{r'} \left(\frac{n}{r'} \right)^{r'} \left(\frac{n}{n - r'} \right)^{n - r'} q^{ - 1 / (2) } \left(\frac{n}{r'} \right)^{-1 / (2) } \left(\frac{n - r'}{n - r} \right)^{n - r'} & \quad r' \doteq d / 2 \\ &= q^{ n \cdot h_q( d / (2n) ) } \left(\frac{n}{n - r'} \right)^{n - r'} q^{ - 1 / 2 } \left(\frac{n}{r'} \right)^{-1 / 2 } \left(\frac{n - r'}{n - r} \right)^{n - r'} \\ \Longrightarrow n \cdot h_q( d / (2n) ) &\le n \cdot h_q( \lparen d - 1 \rparen / (2n) ) + \log_q \left( q^{ 1 / 2 } \left(\frac{n}{r'} \right)^{ 1 / 2 } \left(\frac{n - r}{n - r'} \right)^{n - r'} \right), \end{aligned} \]

and \[ \log_q \left( q^{ 1 / 2 } \left(\frac{n}{r'} \right)^{ 1 / 2 } \left(\frac{n - r}{n - r'} \right)^{n - r'} \right) = O \left( 1 + \log_q n + \log_q \left( 1 + \frac{1 / 2}{n - r'} \right)^{n - r'} \right) = O \left( \log_q n \right) . \]

Define the rate of the code \(R = \frac{ \log_q |C| }{ n }\), and relative distance of the code as \(\delta = \frac{d}{n}\), we get

\[ R + h_q( \delta / 2) \le 1 + O\left( \frac{\log_q n }{n} \right). \]

In what's follows, we prove the inequalities for volume estimation. WLOG, we assume that \(r\) is an integer.

Upper Bound

Theorem. For \(0 \le r \le n \cdot (1 - 1 / q)\), it holds that \(\mathbb{Vol} (r) \le q^{n h_q(r / n)}\).

Algebraic Approach

\[ \begin{aligned} (q - 1)^{ r } \left( \frac{n}{r} \right)^{ r } \left( \frac{ n }{ n - r } \right)^{n - r } &= (q - 1)^{ r } \left( \frac{n}{r} \right)^{ r } \left( \frac{ n }{ n - r } \right)^{n - r } \left( \frac{r}{n} + \frac{n - r}{n} \right)^n \\ &\ge (q - 1)^{ r } \left( \frac{n}{r} \right)^{ r } \left( \frac{ n }{ n - r } \right)^{n - r } \sum_{i = 0}^{ r} \binom{ n }{i} \left( \frac{r}{n} \right)^i \left( \frac{n - r }{n } \right)^{n - i} \\ &\ge \sum_{i = 0}^{ r} \binom{ n }{i} (q - 1)^{ i } \left( \left( \frac{n}{r} \right)^{ r - i} \left( \frac{n - r }{n } \right)^{r - i} (q - 1)^{r - i} \right) \\ &\ge \sum_{i = 0}^{ r} \binom{ n }{i} (q - 1)^{ i } \\ &= \mathbb{Vol}(r), \end{aligned} \]

where the last inequality follows from that \[ \frac{n}{r} \cdot \frac{n - r}{ n } \cdot (q - 1) = \Big( \frac{n}{r} - 1 \Big) \cdot (q - 1)\ge 1, \] as \(r / n \le 1 - 1 / q = q / (q - 1)\).

Finally, let \(\epsilon \doteq \frac{r}{n}\), we have \[ \begin{aligned} \log \mathbb{Vol}(r) &\le r \cdot \log (q - 1) + r \cdot \log \frac{1}{\epsilon} + (n - r) \cdot \log \frac{1}{1 - \epsilon} \\ &= n \left( \epsilon \cdot \log (q - 1) + \epsilon \cdot \log \frac{1}{\epsilon} + (1 - \epsilon) \cdot \log \frac{1}{1 - \epsilon} \right) \\ &= n \Big( \epsilon \cdot \log (q - 1) + \mathbb{H}_2(\epsilon) \Big) \\ &= (\log q) \cdot n \cdot h_q (\epsilon), \end{aligned} \] which proves the claim.

Entropy

We rely on sub-additivity of entropy.

Let \(X\) and \(Y\) be discrete random variables, then
1. \(H( X, Y ) \le H(X) + H(Y)\).
2. \(H(X | Y) \le H(X)\).

Proof. Consider the ball \(\mathcal{B}(1^n, r)\), and a uniform random point \(X = (X_1, ..., X_n)\) in the ball \(\mathcal{B}(1^n, r)\). Its entropy is \[ H(X) = \log \mathbb{Vol}(r). \]

We write \(Y = X_1\). By sub-additivity of entropy and by symmetry, \[ \begin{aligned} H(X) &\le H(X_1) + H(X_2) + ... + H(X_n) &= n H(Y). \end{aligned} \]

Let \(\epsilon \doteq \Pr[ Y \neq 1]\) be the probability that \(Y\) is not \(1\). Further, by symmetry, \(\Pr[Y = i]\) has equal probability for each \(i \neq 1\). Hence, \[ \begin{aligned} H(Y) &= \Pr[ Y = 1] \cdot \log \left( \frac{1}{\Pr[ Y = 1]}\right) + \sum_{i \neq 1} \Pr[ Y = i ] \cdot \log \left( \frac{1}{\Pr[ Y = i]}\right) \\ % &= (1 - \epsilon) \cdot \log \left( \frac{1}{1 - \epsilon} \right) + \epsilon \cdot \sum_{i \neq 1} \frac{ \Pr[ Y = i ] }{ \epsilon } \cdot \log \left( \frac{1}{\Pr[ Y = i]}\right) \\ % &\le (1 - \epsilon) \cdot \log \left( \frac{1}{1 - \epsilon} \right) + \epsilon \cdot \log \left( \sum_{i \neq 1} \frac{ \Pr[ Y = i ] }{ \epsilon } \frac{1}{\Pr[ Y = i]}\right) \\ &= (1 - \epsilon) \cdot \log \left( \frac{1}{1 - \epsilon} \right) + \epsilon \cdot \log \left( \frac{ q - 1 }{ \epsilon } \right) \\ \end{aligned} \] where then entropy of \(Y\) is maximized when \(Y\) is uniform of the elements that are not 1. Further, it can be verified that \(\epsilon < r / n\). As \(h_q (\epsilon)\) increases for \(\epsilon < 1 - 1 / q\), hence \[ H(Y) \le (1 - r / n) \cdot \log \left( \frac{1}{1 - r / n} \right) + \frac{r}{n} \cdot \log \left( \frac{ q - 1 }{ r / n } \right). \]

\(\square\)

Lower Bound

Theorem. For \(r \in \mathbb{R}_+\), it holds that \(\mathbb{Vol} (r) \ge q^{ n h_q ( r / n) - O( \log n) }\).

We rely on the Stirling's approximation.

Fact. For \(n \in \mathbb{N}_+\), \(\sqrt{2 \pi n} \left(\frac{n}{e} \right)^n \le n! \le e \sqrt{n} \left(\frac{n}{e} \right)^n\).

Proof. First, assume that \(r \in \mathbb{N}_+\). It holds that \[ \begin{aligned} \mathbb{Vol} (r) &\ge \binom{n}{r} (q - 1)^r \\ &\ge \frac{ \sqrt{2 \pi n } \left(\frac{n}{e} \right)^n }{ e \sqrt{r} \left(\frac{r}{e} \right)^r e \sqrt{n - r} \left(\frac{n - r}{e} \right)^{n - r} } (q - 1)^r \\ &= (q - 1)^r \left(\frac{n}{r} \right)^r \left(\frac{n}{n - r} \right)^{n - r} \sqrt{ \frac{2 \pi n }{e^4 r (n - r)} } \\ &= q^{n h_q ( r / n) - O(\log n) } \end{aligned} \] Assume that \(r \in \mathbb{R}_+\). Let \(r' = \lfloor r \rfloor\). Then \[ \begin{aligned} \mathbb{Vol} (r) \ge \mathbb{Vol} ( r' ) \ge q^{n h_q ( r' / n) - O(\log n) }. \end{aligned} \] But \[ \begin{aligned} \frac{ q^{n h_q ( r / n)} }{ q^{n h_q ( r' / n)} } \le (q - 1) \cdot \left(\frac{n}{r'} \right) \cdot \left(\frac{n - r}{n - r'} \right)^{n - r'} \in q^{ O \left( \log_q n \right) } . \end{aligned} \]

\(\square\)

Hence, the upper bound is rather tight.
