Clarkson Type Inequalities

Posted on 2022-03-30

Theorem. Let \(p \in (1, 2]\). Then for \(x, y \in \mathbb{R}\), it holds that \[ \frac{1}{2} |x - y|^p + \frac{1}{2} |x + y|^p \le |x|^p + |y|^p. \]

Proof. Let \(z = \big(a, b\big) \in \mathbb{R}^2\). Since \(p \le 2\), it holds that \[ \left( |a|^p + |b|^p \right)^{1 / p} = \Vert z \Vert_p \ge \Vert z \Vert_2 = \left( |a|^2 + |b|^2 \right)^{1 / 2} = \left( \frac{1}{2}|a + b|^2 + \frac{1}{2}|a - b|^2 \right)^{1 / 2}. \] Raising the power of both hand sides by \(p\), and via concavity of the function \(y = x^{p / 2}\), we get
\[ \begin{aligned} |a|^p + |b|^p &\ge \left( \frac{1}{2}|a + b|^2 + \frac{1}{2}|a - b|^2 \right)^{p / 2} \\ &\ge \frac{1}{2} \left( |a + b|^2 \right)^{p / 2} + \frac{1}{2} \left( |a - b|^2 \right)^{p / 2} \\ &\ge \frac{1}{2} |a + b|^p + \frac{1}{2} |a - b|^p. \end{aligned} \] \(\square\)

Remark: in general, for \(z \in \mathbb{R}^n\), and \(0 < q \le p\), we have \[ \Vert z \Vert_q = \left( \sum_{i \in [n]} |z_i|^q \right)^{1 / q} \ge \Vert z \Vert_p = \left( \sum_{i \in [n]} |z_i|^p \right)^{1 / p} \]

To prove this, let \(y_i = |z_i|^q\) for each \(i \in [n]\), and \(r = p / q \ge 1\). It suffices to show that \[ \left( \sum_{i \in [n]} y_i \right)^r \ge \sum_{i \in [n]} y_i^r. \]

Further, let \(x_j = y_j / \left( \sum_{i \in [n]} y_i \right)\) for each \(j \in [n]\). It reduces to prove that
\[ \sum_{j \in [n]} x_j^r \le \sum_{j \in [n]} x_j =1 . \]
