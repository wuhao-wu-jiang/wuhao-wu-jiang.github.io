Exchanging Summation Order

Posted on 2019-03-02 Edited on 2022-12-05

Case One

Theorem. Let \(x_n \ge 0\) for \(n \in \mathbb{N}^+\). Then \[ \sum_{n \ge 1} x_n = \sup \left\{ \sum_{n \in E} x_n : E \subseteq \mathbb{N}^+ \wedge E \text{ is finite } \right\}. \]

Proof. First, \[ \sum_{n \ge 1} x_n = \lim_{k \rightarrow \infty} \sum_{n \in [k]} x_n \le \sup \left\{ \sum_{n \in E} x_n : E \subseteq \mathbb{N}^+ \wedge E \text{ is finite } \right\}. \] Second, for each \(E \subseteq \mathbb{N}^+\) such that \(E\) is finite, there exists an \(k \in \mathbb{N}^+\) that is larger than all numbers in \(E\). Hence, \[ \sum_{n \ge 1} x_n \ge \sum_{n \in [k]} x_n \ge \sum_{n \in E} x_n. \] Taking supremum over the RHS finishes the proof.

\(\square\)

Theorem. Let \(x_n \ge 0\) for \(n \in \mathbb{N}^+\), \(\sigma\) be a permutation of \(\mathbb{N}^+\), i.e., an one-to-one mapping of \(\mathbb{N}^+\) onto itself, and \(y_n = x_{\sigma(n)}\) for \(n \in \mathbb{N}^+\). Then \[ \sum_{n \ge 1} x_n = \sum_{n \ge 1} y_n. \]

Proof. First, for each \(k \in \mathbb{N}^+\), there exits a \(k' \in \mathbb{N}^+\), such that \(\{ x_1, \ldots, x_k \} \subseteq \{ y_1, \ldots, y_{k'} \}\). Therefore, \[ \sum_{n \in [k]} x_n \le \sum_{n \in [k']} y_n \le \sum_{n \ge 1} y_n. \] Taking \(k \rightarrow \infty\) proves \(\sum_{n \ge 1} x_n \le \sum_{n \ge 1} y_n\).

Second, for each \(k \in \mathbb{N}^+\), let \(k' = \max_{i \in [k] } \sigma(i)\). We have \(\{ y_1, \ldots, y_k \} \subseteq \{ x_1, \ldots, x_{k'} \}\), and \[ \sum_{n \ge 1} x_n \ge \sum_{n \in [k']} x_n \ge \sum_{n \in [k]} y_n. \] Taking \(k \rightarrow \infty\) proves \(\sum_{n \ge 1} x_n \ge \sum_{n \ge 1} y_n\).

\(\square\)

Case Two

Consider a matrix \(A = (a_{i,j})\) with countable rows and columns.

Suppose that for each row \(i \in \mathbb{N}^+\), the following limit exits \[ x_i = \sum_{j \ge 1} a_{i, j}, \] and that the limit of the sum of \(x_i\) exits \[ s = \sum_{i \ge 1} x_i. \]

Suppose that for each column \(j \in \mathbb{N}^+\), the following limit exits \[ y_j = \sum_{i \ge 1} a_{i,j}. \]

Moreover, \(\forall \epsilon > 0\), \(\exists N \in \mathbb{N}^+\), s.t., if \(n > N\), for all \(m \in \mathbb{N}^+\), we have \[ \left| \sum_{j = 1}^m \sum_{i = 1}^n a_{i,j} - \sum_{j = 1}^m y_j \right| \le \epsilon. \] then the limit of \(\sum_{j \ge 1} y_j\) exits, and \[ \sum_{i \ge 1} x_i = \sum_{j \ge 1} y_j. \]

Proof: For each \(m \in \mathbb{N}^+\), \[ \begin{aligned} \left| s - \sum_{j = 1}^m y_j \right| &\le \left| s - \sum_{i = 1}^n x_i \right| + \left| \sum_{i = 1}^n x_i - \sum_{i = 1}^n \sum_{j = 1}^m a_{i,j} \right| + \left|\sum_{i = 1}^n \sum_{j = 1}^m a_{i,j} - \sum_{j = 1}^m y_j \right| \\ &= \left|s - \sum_{i = 1}^n x_i \right| + \left| \sum_{i = 1}^n \left( x_i - \sum_{j = 1}^m a_{i,j} \right) \right| + \left|\sum_{i = 1}^n \sum_{j = 1}^m a_{i,j} - \sum_{j = 1}^m y_j \right| \end{aligned} \]

First, we choose large enough \(n\), such that \[ \left|s - \sum_{i = 1}^n x_i \right| + \left|\sum_{i = 1}^n \sum_{j = 1}^m a_{i,j} - \sum_{j = 1}^m y_j \right| \le 2 \epsilon. \] Fix an \(n\), we can choose large enough \(m\), such that \[ \left| \sum_{i = 1}^n \left( x_i - \sum_{j = 1}^m a_{i,j} \right) \right| \le \epsilon, \] Therefore, \[ \left| s - \sum_{j = 1}^m y_j \right| \le 3 \epsilon. \]

\(\square\)
