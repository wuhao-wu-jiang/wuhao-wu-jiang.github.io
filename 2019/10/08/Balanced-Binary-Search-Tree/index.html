Balanced Binary Search Tree

Posted on 2019-10-08 Edited on 2019-10-10

It is the depth of the binary search that affects the performance of it. We say that a binary search tree with nnn nodes is balanced if its depth is of order O(log⁡n)O(\log n)O(logn) (hence we can perform find, insert and delete operations in time O(log⁡n)O(\log n)O(logn)).
