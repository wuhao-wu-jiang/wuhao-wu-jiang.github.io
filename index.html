Weighed Random Sampling

Posted on 2021-03-04 Edited on 2021-03-06

Problem

A stream of \(n\) elements indexed by \([n] \doteq \{ 1, 2, 3, ..., n \}\). Each element \(i\) is associated with a weight \(w_i\). The goal is to maintain a set \(S\) of \(m\) samples, such that

  1. \(S\) is initialized with \(\{1, 2, ..., m \}\);
  2. \(\forall t > m\), when element \(t\) arrives, \(\forall i \in [t]\), the \(i\)-th element is kept with probability \[ p_i \doteq \min \left\{ 1, \frac{w_i}{ z_t } \right\}, \] where \(z_t\) is the unique solution to the equation \(\sum_{i \in [t] } p_i = m\), i.e., \[ \left| \{ i \in [t] \wedge w_i \ge z_t \} \right| + \sum_{ i \in [t] \wedge w_i < z_t } \frac{w_i}{ z_t } = m. \]

Definition. \(L(t, x) \doteq \{ i \in [t ] : w_i \ge x \}\).
Definition. \(W(t, x) \doteq \sum_{ i \in [t] , w_i < x } w_i\).
Definition.
\[ \begin{aligned} f(t, x) \doteq \left| \{ i \in [t] \wedge w_i \ge x \} \right| + \sum_{ i \in [t] , w_i < x } \frac{w_i}{ x } = \left| L(t, x) \right| + \frac{1}{x} \cdot W(t, x). \end{aligned} \]

For given \(t \in [n]\), \(z_t\) is the solution to \(f(t, z_t) = m\).

Algorithm

Initially, we can set \(z_t = 0\) for \(t \in [m]\), so that \(\forall i \in [m], p_i = 1\) and element \(i\) is added to \(S\).

Next, we discuss how to compute \(z_{t + 1}\) based on \(z_t\) and how to maintain \(S\) when the \(t + 1\)-th element arrives. Define the sets

Step 1

We want to find the solution \(z_{t + 1}\) for the equation \(f(t + 1, x ) = m\).

Observe that \[ f(t + 1, z_t) = f(t, z_t ) + \min \left\{ \frac{w_{t + 1} }{ z_t}, 1 \right\} > m. \]

Hence, it holds that \(z_{t + 1} > z_t\).

W.L.O.G, assume that \(L(t + 1, z_t)\) contains \(k\) elements whose weights \(\{ x_1, x_2, .., x_k \}\) are sorted in non-decreasing order: \[ z_t \doteq x_0 \le x_1 \le x_2 \le ... \le x_k < x_{k + 1} \doteq \infty. \]

If we fix \(t + 1\), then for \(x > z_t\), \(f(t + 1, x)\) is an decreasing and piece-wise linear function on intervals \[ ( x_0 , x_1 ], ( x_1 , x_2 ], ... ( x_{k - 1} , x_k ] ,( x_k , x_{k + 1} ). \] Here if \(x_{j - 1} = x_j\) for some \(j \in [ k ]\), then we consider the interval \((x_{j -1}, x_j ]\) an empty set and ignore it. If \(z_{t + 1} \in (x_{j -1}, x_j ]\), it holds that \[ \begin{aligned} &f(t + 1, x_{ j - 1 } ) > m &\Longleftrightarrow & \quad | L(t + 1, x_{ j - 1 } ) | + \frac{ 1 }{ x_{ j - 1 } } \cdot W (t + 1, x_{ j - 1 } ) > m, \\ &f(t + 1, x_j ) \le m &\Longleftrightarrow & \quad | L(t + 1, x_j ) | + \frac{ 1 }{ x_{ j } } \cdot W (t + 1, x_{ j } ) \le m . \end{aligned} \]

When \(x \in(x_{j -1}, x_j ]\), \(| L (t + 1, x) |\) is a constant that equals to \(|L(t + 1, x_j )|\). We have \[ | L(t + 1, z_{t + 1} ) | + \frac{ 1 }{ z_{ t + 1 } } \cdot W(t + 1, z_{ t + 1 } )= | L(t + 1, x_j ) | + \frac{ 1 }{ z_{ t + 1 } } \cdot W(t + 1, x_{ j } ) = m. \]

Therefore, \[ z_{t + 1} = \frac{ W(t + 1, x_{ j } ) }{ m - \left| L(t + 1, x_j ) \right| } = \frac{ \sum_{ i \in [t + 1], w_i < x_j } w_i }{ m - \left| L(t + 1, x_j ) \right| }. \]

The following code shows the detailed process for the search of \(z_{t + 1}\). It maintains

  1. A set \(L\), such that at the beginning of the algorithm, \(L = L(t, z_t)\), and when the algorithm stops, \(L = L(t + 1, z_{t + 1} )\);

  2. The sum of weights \(W\), which equals to \(W(t, z_t)\) at the beginning of the algorithm, and equals to \(W(t + 1, z_{t + 1} )\) when the algorithm stops;

  3. A set \(C\), which contains elements \(L(t, z_t) \cup \{ t + 1 \} \setminus L(t + 1, z_{t + 1} )\).

Input: a) The weight \(w_{t + 1}\); b) The set \(L = L(t, z_t)\).
1. \(C \leftarrow \emptyset\);
2. If \(w_{t + 1} \ge z_t:\)
3. \(\qquad\) \(L \leftarrow L \cup \{ t + 1 \}\).
4. Else:
5. \(\qquad\) \(C \leftarrow C \cup \{ t + 1 \}\).
6. While \(|L| \neq \emptyset\) and \(W / \min_{i \in L } w_i + | L | > m:\)
7. \(\qquad\) \(W \leftarrow W + \min_{i \in L} w_i\);
8. \(\qquad\) move the element \(i \in L\) with minimum \(w_i\) from \(L\) to \(C\). 7. \(z_{t + 1} \leftarrow W / (m - |L| )\).

Step 2

Denote \(S_t\) the sampled set after \(t\) elements are processed. Define \[ D = [t] \setminus L( t, z_t ), \\ E = S_t \setminus L( t, z_t ). \] By definition, it holds that \(|E| = s - |L(t, z_t) | = W(t, z_t ) / { z_t } = \sum_{i \in D} w_i / z_t\).

When the element \(t + 1\) comes, let \(S' \doteq S_t \cup \{ t + 1 \}\) elements. To obtain a set \(S_{t + 1}\) of \(m\) samples, we need to discard one element in \(S'\). Suppose that we have computed \(z_{t + 1}\). The elements \(\in [t + 1]\) can be partitioned into three sets: \[ [ t + 1 ] = L(t + 1, z_{t + 1} ) \cup C \cup D. \]

Similarly, the set \(S'\) can be decomposed into three parts: \[ S' = L(t + 1, z_{t + 1} ) \cup C \cup E. \]

  1. If \(i \in L(t + 1, z_{t + 1} )\), it must stay in the sample set.

  2. If \(i \in C\), it should be removed with probability \(1 - w_i / z_{t + 1}\). Because if \(i \in C\), either \(i = t + 1\) or \(i \in L(t , z_t)\). In both case, \(i\) appears in \(S'\) with probability \(1\). On the other hand, we only keep it with probability \(w_i / z_{ t + 1} < 1\).

  3. If \(i \in D\), it is appears in \(S'\) with probability \(w_i / z_t\). Let \(p\) be the probability of discarding \(i\), if it appears in \(S'\). As the target probability to keep \(i\) should be \(w_i / z_{t + 1}\), it holds that \[ \frac{w_i}{ z_t } \cdot (1 - p) = \frac{ w_i }{ z_{t + 1} }. \] Therefore, if \(i \in E\), it should be discarded with probability \(p = 1 - z_t / z_{t + 1}\).

Observe that \[ \begin{aligned} |S'| - m &= f(t, z_t ) + 1 - f(t + 1, z_{ t + 1} ) \\ &= | L (t, z_t ) | + \frac{1}{z_t} \cdot \sum_{i \in [t], w_i < z_t} w_i + 1 - | L (t + 1, z_{t + 1} | - \frac{1}{z_t} \cdot \sum_{i \in [t + 1], w_i < z_{t + 1} } w_i \\ &= \sum_{i \in C} \left( 1 - \frac{w_i}{ z_{ t + 1} } \right) + \sum_{ i \in D} \frac{w_i }{ z_t } \left( 1 - \frac{z_t}{ z_{t + 1} } \right) \\ &= \sum_{i \in C} \left( 1 - \frac{w_i}{ z_{ t + 1} } \right) + |E| \cdot \left( 1 - \frac{z_t}{ z_{t + 1} } \right) \\ &= 1. \end{aligned} \]

To decide which element from \(C \cup E\) to discard, we can perform the follows:

  1. Sample a real number \(r \in [0, 1]\) uniformly at random.
  2. For \(i \in C\) do:
  3. \(\qquad\) if \(r < 1 - w_i / z_{t + 1}\):
  4. \(\qquad\) \(\qquad\) discard \(i\) and return;
  5. \(\qquad\) else:
  6. \(\qquad\) \(\qquad\) \(r \leftarrow 1 - w_i / z_{t + 1}\);
  7. Discard one element from \(E\) uniformly at random;

Reference

[1] G. Cormode and K. Yi, Small Summaries for Big Data, 1st ed. Cambridge University Press, 2020.

The Number Of Empty Bins.

Posted on 2021-02-27

Problem

There are \(n\) balls and \(m\) bins, where \(m \gg n\). Each ball goes into a bin uniformly and independently at random. What is the number of empty bins?

Analysis

  1. The probability of a particular bin being empty is \((1 - 1 / m)^n\).

    Proof: The probability of a particular ball not falling into this bin is \(1 - 1 / m\). Therefore, \[ \Pr[ \text{none of the balls falls into this bin} ] = \left( 1 - \frac{1}{m} \right)^n. \]

  2. The expected of empty bins is \(m \cdot (1 - 1 / m)^n\).

    Proof: For \(j \in [m]\), define the indicator random variable \(Z_j\) that equal to \(1\) if the \(j\)-th bin is empty. Then by linearity of expectation, \[ \mathbb{E} \left[ \sum_{j \in [m] } Z_j \right] = \sum_{j \in [m] } \mathbb{E} [ Z_j ] = m \cdot \left( 1 - \frac{1}{m} \right )^n. \]

  3. With probability at least \(1 - \delta\) (\(\delta \in (0, 1)\)), the number of empty bin is \[ m \cdot (1 - 1 / m)^n \pm \sqrt{ \frac{n}{2} \ln \frac{2}{\delta} }. \]

    Proof: We need to use McDiarmid inequality.

    McDiarmid inequality ( [1] ). Consider an arbitrary set \(\mathcal{D}\). The function \(f: \mathcal{D} ^n \rightarrow \mathbb{R}\) has bounded difference if there exits nonnegative constants \(b_1, ..., b_n \in \mathbb{R}\) such that for \(i \in [n]\),
    \[ \sup_{ x_1, ..., x_n , x_i' \in \mathcal{D} } | f(x_1, ..., x_n ) - f( x_1, ..., x_{i - 1}, x_i', x_{i + 1}, ..., x_n ) | \le b_i. \] Consider a set of independent random variables \(X_1, ..., X_n\) sampled from $ $. Denote \(Y \doteq f(X_1, ..., X_n)\). It holds that \[ \Pr [ Y - \mathbb{E} [ Y ] \ge \epsilon ] \le \exp \left( - \frac{2 \epsilon^2 }{ \sum_{i \in [n]} b_i^2 }\right). \] The analogous inequality is valid for \(\Pr [ \mathbb{E} [Y] - Y \ge \epsilon ]\).

    To apply the theorem, let \(\mathcal{D} = [m]\). For \(i \in [n]\), define the random variable \(X_i \in \mathcal{D}\) to be the index of the bin that the \(i\)-th ball falls into. Let \(Y \doteq f (X_1, ..., X_n)\) be the number of empty bins, which is a random variable that depends on \(X_1, ..., X_n\). We have proven that \[ \mathbb{E} [ Y ] = m \cdot \left( 1 - \frac{1}{m} \right)^n. \]

    Further, the outcome of the \(i\)-th ball can affect the number of empty bins by at most \(1\): \[ \sup_{ x_1, ..., x_n , x_i' \in \mathcal{D} } | f(x_1, ..., x_n ) - f( x_1, ..., x_{i - 1}, x_i', x_{i + 1}, ..., x_n ) | \le 1. \]

    Hence, by McDiarmid inequality, \[ \Pr [ | Y - \mathbb{E} [ Y] | \ge \epsilon ] \le 2 \cdot \exp \left( - \frac{2 \epsilon^2 }{ n } \right). \]

    Equating this probability to \(\delta\), we get \[ \epsilon = \sqrt{ \frac{n}{2} \ln \frac{2}{\delta} }. \]

Reference

[1] C. McDiarmid, “Concentration,” in Probabilistic Methods for Algorithmic Discrete Mathematics, M. Habib, C. McDiarmid, J. Ramirez-Alfonsin, and B. Reed, Eds. Berlin, Heidelberg: Springer Berlin Heidelberg, 1998, pp. 195–248.

k Minimum Values for Set Cardinality

Posted on 2021-02-22

Problem

We study the problem of estimating the number of distinct elements in a stream \(S\). W.L.O.G, we assume that elements in \(S\) come from a domain \([d] = \{1, 2,..., d \}\), and that each element in \([d]\) appears in the \(S\) at least once. Hence, the problem is equivalent to the one of estimating the value \(d\).

k Minimum Values

The algorithm of \(k\)-minimum values returns a \((1 \pm \epsilon)\) estimate \(\hat d\) of \(d\), with probability at least \(1 - \delta\).

The algorithm picks a hash function \(g : [d] \rightarrow [m]\) and uses a min heap \(h\) that keeps the \(k\) minimum hash values. When it stops, it outputs an estimate \[ \hat d \doteq k \cdot m / v, \] where \(v\) is the maximum element in \(h\). Observe that \(v\) is also the \(k\)-th minimum hash value of the set \[ g( [d] ) \doteq \{ g(1), g(2), ..., g(d) \}. \]

Algorithm: \(k\)-minimum values.
1. Select a pair-wise independent hash function \(g: [d] \rightarrow [m]\), where \(m \ge d^2 / \delta\);
2. Initialize a min heap \(h \leftarrow \{ \}\);
3. For \(s \in S\) do:
4. \(\qquad\) If \(g(s) \notin h:\)
5. \(\qquad\) \(\qquad\) \(h \leftarrow h \cup \{ g(s) \}\);
6. \(\qquad\) \(\qquad\) If \(|h| > k:\)
7. \(\qquad\) \(\qquad\) \(\qquad\) remove the max element in \(h\);
8. Let \(v \leftarrow\) max element in \(h\).
9. Return \(\hat d \doteq k \cdot m / v\).

Theorem 1.

With probability at least \(1 - \delta\), there isn't hash collision.

Proof:

For \(i, j \in [d], i \neq j\), as \(g\) is pair-wise independent, \(\Pr[ g(i) = g(j) ] = \frac{1}{m}\). Therefore, \[ \mathbb{E} \left[ \sum_{i, j \in [d], i \neq j} [ g(i) = g(j) ] \right] = \sum_{i, j \in [d], i \neq j} \mathbb{E} \left[ [ g(i) = g(j) ] \right] \le \frac{d^2}{m} = \delta, \] where \([ g(i) = g(j) ]\) is the indicator random variable of \(g(i) = g(j)\).

By Markov's inequality, \[ \Pr \left[ \sum_{i, j \in [d], i \neq j} [ g(i) = g(j) ] \ge 1 \right] \le \delta. \]

\(\square\)

Theorem 2.

With probability at least \(1 / (k\epsilon^2)\), \[ \hat d < \frac{d}{1 - \epsilon}. \]

Proof: For \(i \in [d]\), define the indicator random variable of \(X_i = \left[ g(i) \le (1 - \epsilon) \cdot k / d \cdot m \right]\). Recall that \[ v \doteq \text{The } k\text{-th smallest element of } \{ g(1), g(2), ..., g(d) \}. \] The event that \[ \left\{ v \le (1 - \epsilon) \cdot k / d \cdot m \right\} \equiv \left\{ \frac{d}{1 - \epsilon} \le \frac{ k \cdot m }{ v } \right\} \] happens if only \(\sum_{i \in [d] } X_i \ge k\).

By definition of \(X_i\), \[ p \doteq \Pr \left[ X_i \right] = (1 - \epsilon) \cdot \frac{ k }{d}. \]

By linearity of expectation, we have \[ \mathbb{E} \left[ \sum_{i \in [d] } X_i \right] = d \cdot p = (1 - \epsilon) \cdot k. \]

As \(g\) is pair-wise independent, so are the \(X_i\)'s. It follows that \[ \begin{aligned} \sigma^2 =\mathbb{Var} \left[ \sum_{i \in [d] } X_i \right] < d \cdot p = (1 - \epsilon) \cdot k. \end{aligned} \]

By monotonicity of probability and by Chebyshev's inequality, \[ \begin{aligned} \Pr\left[ \sum_{i \in [d] } X_i \ge k \right] &\le \Pr\left[ \left| \sum_{i \in [d] } X_i - (1 - \epsilon) \cdot k \right| \ge \epsilon k \right] < \frac{ (1- \epsilon) \cdot k }{ \epsilon^2 k^2 } < \frac{1}{\epsilon^2 k}. \end{aligned} \]

\(\square\)

Theorem 3.

With probability at least \((1 + \epsilon) / (k\epsilon^2)\), \[ \hat d \ge \frac{d}{1 + \epsilon}. \]

Proof: For \(i \in [d]\), define the indicator random variable of \(Y_i = \left[ g(i) \le (1 + \epsilon) \cdot k / d \cdot m \right]\). Recall that \[ v \doteq \text{The } k\text{-th smallest element of } \{ g(1), g(2), ..., g(d) \}. \] The event that \[ \left\{ v \ > \ (1 + \epsilon) \cdot k / d \cdot m \right\} \equiv \left\{ \frac{d}{1 + \epsilon} > \frac{ k \cdot m }{ v } \right\} \equiv \left\{ \frac{d}{1 + \epsilon} > \hat d \right\} \] happens if only \(\sum_{i \in [d] } Y_i < k\).

By definition of \(Y_i\), \[ q \doteq \Pr \left[ Y_i \right] = (1 + \epsilon) \cdot \frac{ k }{d}. \]

By linearity of expectation, we have \[ \mathbb{E} \left[ \sum_{i \in [d] } Y_i \right] = d \cdot q = (1 + \epsilon) \cdot k. \]

As \(g\) is pair-wise independent, so are the \(X_i\)'s. It follows that \[ \begin{aligned} \sigma^2 =\mathbb{Var} \left[ \sum_{i \in [d] } Y_i \right] &= \sum_{i \in [d] } \mathbb{Var} \left[ Y_i \right] \\ &= d \cdot q \cdot ( 1 - q) \\ &= (1 + \epsilon) \cdot k \cdot (1 - q). \end{aligned} \]

By monotonicity of probability and by Chebyshev's inequality, \[ \begin{aligned} \Pr\left[ \sum_{i \in [d] } Y_i < k \right] &\le \Pr\left[ \left| \sum_{i \in [d] } Y_i - (1 + \epsilon) \cdot k \right| \ge \epsilon k \cdot \sqrt{ 1 - q } \right] \\ &\le \frac{ (1 + \epsilon) \cdot k }{ \epsilon^2 k^2 } \\ &\le \frac{1}{\epsilon^2 k}. \end{aligned} \]

\(\square\)

Reference

[1] G. Cormode and K. Yi, Small Summaries for Big Data, 1st ed. Cambridge University Press, 2020.
