Pairwise Independent Hashing

Posted on 2018-07-18 Edited on 2023-07-16

Let \(U \in \N^+\), and \(S = \{ x_1, x_2, \ldots, x_m \} \subseteq \mathbb{Z}_U\) be a set of distinct element.

Consider a function \(h: \mathbb{Z}_U \rightarrow \mathbb{Z}_n\), where \(n \le U\). We call \(h\) a hash function. In the case where \(n < U\), there must be some \(i \in \mathbb{Z}_n\), s.t., \(|h^{-1}(i)| > 1\). We are interested in the number of total duplicate value in \[ h(S) \doteq \{ h(x_1), h(x_2), \ldots, h(x_m) \}. \]

If both \(S\) and \(h\) are fixed, there is nothing uncertain. There are two ways to add some randomness to our analysis

  1. \(h\) is fixed, and \(S\) follows some distribution on \(\mathbb{Z}_U\). E.g., let \(U = 2^{20}\) and \(h(x) = x \ \& \ 1023\) that returns the last \(10\) bits of \(x\) (in binary form). If \(S\) is sampled uniformly at random from \(\mathbb{Z}_U\), then on average for each \(x_i \in S\), there are \(\frac{k - 1}{1024}\) elements that have the same value as \(x_i\) under \(h\).

  2. The assumption that \(S\) follows some distribution is too strong. Instead we assume that given \(S\), we select a function \(h\) uniformly, from a set of functions \(\mathcal{H}\). Now we are interested in the probability \[ \Pr[ h(x_i) = h(x_j) ], \quad i \neq j. \]

The finite field \(\mathbb{F}_p\)

Let \(p > U\) be a prime number and \[ \mathcal{H} \doteq \left\{ h(x) \doteq \left( ax + b \mod p \right) \mod n \, \middle| \, a, b \in \mathbb{F}_p \right\}. \]

Theorem. If \(h\) is sampled uniformly at random from \(\mathcal{H}\), then \[ \Pr[ h(x_i) = h(x_j) ] \le \frac{1}{n}, \quad x_i \neq x_j. \]

Lemma 1. Given \(x_i, x_j, s, t \in \mathbb{F}_p\), s.t., \(x_i \neq x_j\), there is a unique pair \((a,b) \in \mathbb{F}_p \times \mathbb{F}_p\) \[ a x_i + b = s, \\ a x_j + b = t, \] where the equations hold in the field \(\mathbb{F}_p\).

Proof. Since \(x_i \neq x_j\), the following matrix is invertible \[ \begin{bmatrix} x_i , 1 \\ x_j , 1 \end{bmatrix} \in \mathbb{F}_p^{2 \times 2}. \]

Therefore,

\[ \begin{bmatrix} a \\ b \end{bmatrix} = \begin{bmatrix} x_i, 1 \\ x_j, 1 \end{bmatrix}^{-1} \begin{bmatrix} s \\ t \end{bmatrix} \]

\(\square\)

Proof of the Theorem.

For \(x_i \neq x_j\), \[ \begin{aligned} \Pr[h(x_i) = h(x_j)] &= \sum_{s, t \in \mathbb{F}_p, s \equiv t \pmod n} \Pr[ h(x_i) = s \wedge h(x_j) = t] \\ &= \sum_{s, t \in \mathbb{F}_p, s \equiv t \pmod n} \frac{1}{ p^2 } \\ &= \sum_{s \in \mathbb{F}_p} \left( \sum_{t \in \mathbb{F}_p, t \equiv s \pmod n} \frac{1}{ p^2 } \right) \\ &\le \sum_{s \in \mathbb{F}_p} \frac{p}{ n } \cdot \frac{1}{ p^2 } \\ &= \frac{1}{ n }. \end{aligned} \]

\(\square\)

Lemma 3. The expected number of pair \(x_i, x_j\), s.t., \(x_i \neq x_j \wedge h(x_i) = h(x_j)\) is given by \[ \mathbb{E} \left[ \sum_{x_i \neq x_j} \mathbb{I}_{h(x_i) = h(x_j)} \right] = \sum_{x_i \neq x_j} \Pr[h(x_i = x_j)] = \binom{m}{2} \frac{1}{n} = \frac{m (k - 1)}{ 2n } \]

When \(m^2 \le n\), then \(\mathbb{E} \left[ \sum_{x_i \neq x_j} \mathbb{I}_{h(x_i) = h(x_j)} \right] \le \frac{1}{2}\). By Markov inequality, \[ \Pr[\exists \, \, \text{collision}] \le \frac{1}{ 2 }. \]

Lemma 4. With probability at most \(1/2\), some bin contains more than \(\frac{m}{n} + \sqrt{2m}\) balls.

For a fixed bin, let \(Y\) denote the number of ball in this bin. Further, define \[ Y_i = \begin{cases} 1, \qquad \text{if the } i \text{-th ball falls into the bin} \\ 0, \qquad \text{otherwise} \end{cases} \] Then \(Y = \sum_i Y_i\), and \[ \mathbb{E} [Y] = \frac{m}{n} \\ \] By pair-wise independence, we have \[ Var[Y] = \sum_i Var[Y_i] = m \frac{1}{n} \left(1 - \frac{1}{n} \right) \] Therefore, by Chebyshev inequality, we have \[ \Pr \left[ |Y - \mathbb{E}[Y]| \ge \sqrt{2n} \sqrt{ m \frac{1}{n} \left(1 - \frac{1}{n} \right)} \right] = \Pr[|Y - \mathbb{E}[Y]| \ge \sqrt{2n} \cdot \sigma] \le \frac{Var[Y]}{2n\sigma^2} = \frac{1}{2n}. \] Finally, applying union bound over \(n\) bins, we get \[ \Pr[\text{some bin gets more than } \sqrt{2m} \text{ balls} ] \le \frac{1}{2}. \]

The finite field \(\mathbb{F}_{2^k}\)

We consider an extension such that each element \(x_i \in S\) is now a vector \(\in \mathbb{F}_2^k\).

We would like to have some operations \(\otimes\) and \(\oplus\), such that if we sample a hash function \(h : \mathbb{F}_2^k \rightarrow \mathbb{F}_2^k\) uniformly at random from the following family \[ \mathcal{H} = \left\{ h(x) \doteq a \otimes x \oplus b \, \middle| \, a, b \in \mathbb{F}_2^k \right\}, \]

then \(h(x_i)\) and \(h(x_j)\) are pairwise independent for \(x_i \neq x_j\).

In particular, when \(x_i \neq x_j\), it holds that \[ \begin{align*} \Pr[ h(x_i) = s \, \wedge \, h(x_j) = t] &= \Pr[ h(x_i) = s ] \cdot \Pr[ h(x_j) = t ] \\ &= \left( \frac{1}{2^k} \right)^2, &\quad \forall s, t \in \mathbb{F}_2^k. \end{align*} \]

We would like the \(a, b \in \mathbb{F}_2^k\) to be determined by \[ \begin{bmatrix} a \\ b \end{bmatrix} = \begin{bmatrix} x_i, 1 \\ x_j, 1 \end{bmatrix}^{-1} \begin{bmatrix} s \\ t \end{bmatrix}. \]

In order that this holds, the set of vectors \(\mathbb{F}_2^m\), when augmented with operations \(\otimes\) and \(\oplus\), should constitute a field.

Fact. For each \(k \in \mathbb{N}^+\), there is a polynomial \(q_k\) with coefficients in \(\mathbb{F}_2\) and of degree \(k\) that is irreducible (cannot be factor).

We now view \(x_i = (x_i[0], \ldots, x_i[k - 1])\) as a polynomial \[ p_{x_i}(z) \doteq \left( x_i[0] + x_i[1] \cdot z + \cdots + x_i[k - 1] \cdot z^{k - 1} \right) \mod q_k. \]

The \(\oplus\) is defined to be the addition of polynomials. Hence, \[ x_i \oplus x_j = (x_i[0] + x_j[0], \ldots, x_i[k - 1] + x_j[k - 1]), \] since \[ \begin{aligned} &p_{x_i}(z) + p_{x_j}(z) \\ &= \left( ( x_i[0] + x_j[0] ) + ( x_i[1] + x_j[1] ) \cdot z + \cdots + ( x_i[k - 1] + x_j[k - 1] ) \cdot z^{k - 1} \right) \mod q_k. \end{aligned} \]

Moreover, we can consider the \(\otimes\) as polynomial multiplications \(\pmod q_k\), which results in a polynomial of degree at most \(k\).

Finally, since \(q_k\) is irreducible, if \(x_i \neq 0\), there exists polynomial \(c\) and \(d\), such that \(c \pmod q_k \equiv c\) and \[ 1 = c \cdot x_i + d \cdot q_k. \]

Hence, \(c\) can be viewed as the inverse of \(x_i\) and division is well-defined. It concludes that \((\mathbb{F}_2^k, \otimes, \oplus)\) constitute a field.
