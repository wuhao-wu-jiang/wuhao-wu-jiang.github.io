The Poisson Approximation

Posted on 2021-03-22

Poisson Distribution

A discrete random variable \(X\) follows Poisson (\(ˈ\)pwɑːsɒn) distribution, with a parameter \(\lambda\), if its probability mass function is given by \[ \Pr[ X = k ] = e^{ - \lambda } \frac{ \lambda^k }{ k! }, \qquad \forall k \in \mathbb{N}. \]

We can verify that the definition is valid: \[ \sum_{ k \in \mathbb{N} } \Pr[ X = k ] = e^{ - \lambda } \sum_{ k \in \mathbb{N} } \frac{ \lambda^k }{ k! } = e^{ - \lambda } e^{ \lambda } = 1. \]

We use \(\text{Pois}(\lambda)\) to denote a Poisson distribution with parameter \(\lambda\).

Mean

Property. \(\mathbb{E}[X] = \lambda\).

Proof. \[ \sum_{ k \in \mathbb{N} } k\Pr[ X = k ] = e^{ - \lambda } \lambda \sum_{ 1 \le k \in \mathbb{N} } \frac{ \lambda^{k - 1} }{ (k - 1)! } = e^{ - \lambda } \lambda \sum_{ k \in \mathbb{N} } \frac{ \lambda^k }{ k! } = \lambda. \]

\(\square\)

Variance

Property. \(\mathbb{Var}[X] = \lambda\).

Proof. \[ \sum_{ k \in \mathbb{N} } k(k - 1)\Pr[ X = k ] = e^{ - \lambda } \lambda^2 \sum_{ 2 \le k \in \mathbb{N} } \frac{ \lambda^{k - 2} }{ (k - 2)! } = e^{ - \lambda } \lambda^2 \sum_{ k \in \mathbb{N} } \frac{ \lambda^k }{ k! } = \lambda^2. \]

Hence, \[ \begin{aligned} \mathbb{Var}[X] &= \sum_{ k \in \mathbb{N} } k^2 \Pr[ X = k ] - (\mathbb{E}[X] )^2 \\ &= \sum_{ k \in \mathbb{N} } ( k (k - 1) + k ) \Pr[ X = k ] - (\mathbb{E}[X] )^2 \\ &= \lambda^2 + \lambda - \lambda^2 \\ &= \lambda. \end{aligned} \]

\(\square\)

Sum of Two Independent Poisson Variable

Property. If \(X_1 \sim \text{Pois}(\lambda_1)\) and \(X_2 \sim \text{Pois}(\lambda_2)\), then \(X_1 + X_2 \sim \text{Pois} (\lambda_1 + \lambda_2)\).

Proof. For every \(k \in \mathbb{N}\), \[ \begin{aligned} \Pr[X_1 + X_2 = k] &= \sum_{a, b \in \mathbb{N}, a + b = k} \left( e^{- \lambda_1 } \frac{ \lambda_1^{a} }{ a! } \right) \left( e^{-\lambda_2 } \frac{ \lambda_2^b }{ b! } \right) \\ &= e^{- (\lambda_1 + \lambda_2) } \frac{1}{k!} \sum_{a, b \in \mathbb{N}, a + b = k} \frac{ k! }{ a! b! } \lambda_1^{a} \lambda_2^b \\ &= e^{- (\lambda_1 + \lambda_2) } \frac{ (\lambda_1 + \lambda_2)^k }{k!}. \end{aligned} \]

\(\square\)

Corollary. If \(X_i \sim \text{Pois}(\lambda_i)\) for every \(i \in [n]\), then \(\sum_{i \in [n] } X_i \sim \text{Pois} \left( \sum_{i \in [n] } \lambda_i \right)\).

Poisson Approximation

Binomial Distribution

The Poisson distribution can be used to approximate the limit of the binomial distribution, under some condition.

Poisson Limit Theorem. Suppose that \(\forall n \in \mathbb{N}^+\), the random variable \(Y_n\) follows binomial distribution \(\text{B}(n, p_n)\) and that \(n p_n = \lambda\). Let \(X \sim \text{Pois}(\lambda)\), then \[ \lim_n \Pr[ Y_n = k ] = \Pr[ X = k ]. \]

Proof. \[ \begin{aligned} \Pr[ Y_n = k ] &= \binom{n}{k } p_n^k (1 - p_n)^{n - k} \\ &= \frac{ n (n - 1) ... (n - k + 1) }{ k !} p_n^k ( 1 - p_n )^{n - k} \\ &= \frac{ n^k + O(n^{k - 1} ) }{ k !} \frac{ \lambda^k }{ n^k } \left( 1 - \frac{ \lambda }{ n } \right)^{n - k} \\ &= \frac{ \lambda^k \left( 1 + O( n^{-1} ) \right) }{ k !} \left( 1 - \frac{ \lambda }{ n } \right)^{n - k} \end{aligned} \]

For fixed \(k\), taking \(n \rightarrow \infty\), we have \[ \frac{ \lambda^k \left( 1 + O( n^{-1} ) \right) }{ k !} \left( 1 - \frac{ \lambda }{ n } \right)^{n - k} \rightarrow \frac{\lambda^k}{ k !} e^{-\lambda }. \]

Balls and Bins

We study the problem of throwing \(m\) balls into \(n\) bins randomly. For every \(i \in [n]\), let \(Y_i\) denote the number of balls in the \(i\)th bin, which follows binomial distribution \(\text{B}(m, 1 / m)\). We have shown that \(Y_i\) can be approximated by \(X_i \sim \text{Pois}(m / n)\). In this section, we show a stronger result that the joint distribution of \((Y_1, ..., Y_n)\) can be approximated by the joint distribution of Poisson random variables.

Theorem. \(\forall \lambda > 0\), let \(X_i \sim \text{Pois}(\lambda)\) be independent random variables. Then the joint distribution of \((X_1, ..., X_n)\) conditioned on \(\sum_{i \in [n]} X_i = m\) is the same as \((Y_1, ..., Y_n)\).

Remark: the theorem holds for \(\forall \lambda > 0\) and does not require that \(\lambda = m / n\).

Proof. Let \(X = \sum_{i \in [n] } X_i\). We have proven that \(X \sim \text{Pois}( n \lambda)\). Hence, \(\forall k_1, ..., k_n \ge 0\), s.t. \(\sum_{i \in [n] } k_i = m\), we have \[ \begin{aligned} \Pr[ X_1 = k_1, X_2 = k_2, ..., X_n = k_n \mid X = m ] &= \left( \prod_{i \in [n] } e^{-\lambda} \frac{ \lambda^{k_i} }{ k_i! } \right) / \left( e^{ - n\lambda} \frac{ (n\lambda)^m }{ m! } \right) \\ &= \frac{ e^{ - n \lambda} \lambda^{ \sum_{i \in [n] } k_i} }{ e^{ - n\lambda} (n\lambda)^m } \frac{ m! }{ \prod_{i \in [n] } k_i ! } \\ &= \binom{ m! }{ k_1; k_2; ...; k_n } \frac{1}{ n^m }. \end{aligned} \] \(\square\)

Theorem [1]. \(\forall i \in [n]\), \(X_i \sim \text{Pois}(m / n)\). Let \(f(y_1, ..., y_n)\) be a nonnegative function. Then \[ \mathbb{E}[ f(Y_1, ..., Y_n) ] \le e \sqrt{m} \mathbb{E}[ f(X_1, ..., X_n) ]. \]

Proof. Let \(X = \sum_{i \in [n] } X_i\). Then \(X \sim \text{Pois}(m)\). As \(f\) is nonnegative, it holds that \[ \begin{aligned} \mathbb{E}[ f(X_1, ..., X_n) ] &= \sum_{ k \in \mathbb{N} } \mathbb{E}[ f(X_1, ..., X_n) \mid X = k] \cdot \Pr[ X = k ] \\ &\ge \mathbb{E}[ f(X_1, ..., X_n) \mid X = m] \cdot \Pr[ X = m ] \\ &= \mathbb{E}[ f(Y_1, ..., Y_n) ] \cdot e^{ - m } \frac{ m^m }{ m! } \\ &\ge \mathbb{E}[ f(Y_1, ..., Y_n) ] \cdot e^{ - m } (m)^m \left( \frac{e}{m} \right)^m \frac{1}{ e \sqrt{m} } \\ &= \mathbb{E}[ f(Y_1, ..., Y_n) ] \cdot \frac{1}{ e \sqrt{m} }, \end{aligned} \] and the theorem is proven. The last inequality follows from Stirling's formula that \(m! \le e \sqrt{m} (m / e)^m\).

\(\square\)

Corollary. If some event happens with probability \(p\) with \(X_1, ..., X_n\), then it happens with probability at most \(e\sqrt{m} \cdot p\) with \(Y_1, ..., Y_n\).

Proof. Let \(f\) be the indicator variable of the event that takes 1 if the event occurs and 0 otherwise. Then by the previous theorem, \[ \Pr[ f(Y_1, ..., Y_n) ] \le e \sqrt{m} \Pr[ f(X_1, ..., X_n) ]. \]

\(\square\)

The analysis makes some analysis simpler since the \(X_i\)'s are independent random variables.

In some special cases, the theorem can be strengthened.

Theorem [1]. Let \(f(y_1, ..., y_n)\) be a nonnegative function such that \(\mathbb{E} [ f(Y_1, ..., Y_n) ]\) is either monotonically increasing or decreasing in \(m\). Then \[ \mathbb{E}[ f(Y_1, ..., Y_n) ] \le 2 \mathbb{E}[ f(X_1, ..., X_n) ]. \]

Proof. Refer to [1]. Omitted.

Reference

[1] M. Mitzenmacher and E. Upfal, “Probability and Computing,” p. 490.
