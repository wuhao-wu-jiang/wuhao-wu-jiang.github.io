Consistent Hashing

Posted on 2018-08-27

Suppose that we would like to implement a distributed file system. There are \(m\) files and \(n\) servers to store the file. We want to spread the work load to the servers so that each one keeps approximate \(\frac{m}{n}\) files. One way to achieve this is by hashing. Let \(p \gg m\) be a prime number. Then \[ h(x) = ax + b \mod p \mod n \] gives us the desired property if \(a \in \{1, 2, ..., p -1\}\) and \(b \in \{0, 1, 2, ..., p - 1\}\) are selected uniformly at random.

The problem is that what if \(n\) changes constantly?

Consider mapping the \(n\) servers uniformly and independently to real numbers in \([0, 1]\). Let's divide \([0, 1]\) into \(n\) equally sized intervals \([0, 1/n]\), \([1/n, 2/n]\), ..., \([(n - 1)/n, 1]\). Now, for an interval that a particular server belongs, the probability that no server maps to the \(2 \log n\) intervals to the left of this interval is given by

\[ (1 - \frac{2 log n}{n})^{n - 1} \le e^{-\frac{2 log n}{n} (n - 1)} = \frac{1}{n^2} e^{\frac{2 log n}{n}} \le \frac{e}{n^2} \]

By union bound, this happens to some server is bounded by \(\frac{e}{n}\).

What is the probability that the interval between some pair of servers is \(O(1 / n^2)\). If we divide \([0, 1]\) into \(\Theta(n^2)\) equally size intervals, by Birthday's Paradox, there is high probability that some interval contains two servers.

Birthday Paradox

Posted on 2018-08-26 Edited on 2022-12-07

Suppose that we throw \(m\) balls into \(n\) bins uniformly at random, where \(n\) is a fixed number. What is the probability that some pair of balls are placed in the same bin?

Define \(X_{i, j}\) be the indicator variable that the \(i\)-th ball and the \(j\)-th ball go to the same bin. Its expectation is given by

\[ \mathbb{E}\left[ X_{i,j} \right] = 1\cdot \Pr\left[ X_{i,j} = 1\right] + 0 \cdot \Pr\left[X_{i,j} = 0\right] = \frac{1}{n}. \]

By linearity of expectation, we have \[ \mathbb{E}\left[\sum_{1 \le i < j \le m} X_{i,j}\right] = \sum_{1 \le i < j \le m} \mathbb{E}\left[X_{i,j}\right] = \frac{m (m - 1)}{2n}. \]

When \(m \approx \sqrt{2n}\), in expectation, there is a pair of ball going to the same bin. However, this estimate is loose.

Define \(p(m, n)\) the exact probability that no ball goes to the same bin. Then \[ p(m, n) = \prod_{i = 0}^m \frac{n - i}{n} = \prod_{i = 0}^m \left(1 - \frac{i}{n}\right). \]

By inequality \(1 - \frac{i}{n} \le e^{- \frac{i}{n} }\), we have a upper bound for the above probability: \[ p(m, n) \le e^{- \frac{m (m - 1)}{2n} }. \]

Therefore, the probability that there is at least one collision is lower bounded by \[ 1 - e^{- \frac{m (m - 1)}{2n} }. \]

If \(m = \sqrt n\), we get \(e^{- \frac{m (m - 1)}{2n} } \approx e^{-\frac{1}{2} } \approx 0.6\). If \(m = 2 \sqrt n\), we get \(e^{- \frac{m (m - 1)}{2n} } \approx e^{-2} \approx 0.14\).

In the case \(n = 365\), we say that if there are \(\sqrt n \approx 19\) people, \(40\%\) of times some pair of them share the same birthday. If there are \(36\) people, the probability is over \(80\%\).

Remark: let's play some math to bound \(p(m, n)\) asymptotically. First, it can be proven that \[ e^x \le 1 + 2 x, \qquad \forall x \in [0, 1]. \]

Hence, \[ p(m, n) \le e^{- \frac{m (m - 1)}{2n} } \le \exp \left( -\frac{m^2}{2n} \right) \cdot \left( 1 + \frac{m}{n} \right). \]

On the other hand, it can be proven that \[ \ln (1 - x) % \ge - x - \frac{x^2}{2} \left( \sum_{i = 0}^\infty (\frac{1}{2})^i \right) \ge -x - x^2, \quad \forall x \in [0, 1 / 2]. \]

So, when \(m < n / 2\), \[ \begin{aligned} p(m, n) &= \prod_{i = 0}^m \left(1 - \frac{i}{n} \right) \\ &\ge \exp \left( - \sum_{i = 0}^m \frac{i}{n} - \sum_{i = 0}^m \frac{i^2}{n^2} \right) \\ &\ge \exp \left( - \frac{m (m - 1)}{2n} \right) \left( 1 - \sum_{i = 0}^m \frac{i^2}{n^2} \right) \\ &\ge \exp \left( - \frac{m^2}{2n} \right) \left( 1 - O \left( \frac{m^3}{n^2} \right) \right). \end{aligned} \]

Combined, we get \[ \exp\left( - \frac{m^2}{2n} \right) \left( 1 - O \left( \frac{m^3}{n^2} \right) \right) \le p(m, n) \le \exp\left( - \frac{m^2}{2n} \right) \left( 1 + \frac{m}{n} \right). \]

If \(m < \sqrt n\), then \(m^3 / n^2 < m / n\). In this case, \[ p(m, n) = \exp\left( - \frac{m^2}{2n} \right) \left( 1 \pm \Theta \left( \frac{m}{n} \right) \right). \]

Replacing \(m = \sqrt{2 n\ln 2}\), then \[ p(m, n) = \frac{1}{2} + \Theta \left( \frac{1}{\sqrt{n}} \right). \]

Reference.

[1]. Ryan O'Donnell, Lecture 3, CS Theory Toolkit, CARNEGIE MELLON UNIVERSITY

Projection of A Random Unit Vector on Its First Dimension

Posted on 2018-08-24 Edited on 2020-05-30

Let \(x = <x_1, x_2, ..., x_d>\) be an \(d\) dimension random unit vector, i.e., a vector generated uniformly at random from the sphere of the unit ball. What is the probability that \(|x_1| \ge \frac{1}{n}\) for some given \(n \ge 2\)?

When \(d = 1\), then \(x_1 = 1\) or \(x_1 = -1\). Therefore \(P[|x_1| \ge \frac{1}{n}] = 1\).

When \(d = 2\),

\[ P[|x_1| \ge \frac{1}{n}] = \frac{4}{2 \pi} \int_{1/n}^1 \sqrt{1 - t^2} d_t = \frac{2}{\pi} \arccos \frac{1}{n} \]

By \[ \begin{aligned} \arcsin(z) &= z + \left({\frac {1}{2} } \right) {\frac {z^{3} } {3} } +\left({\frac {1\cdot 3}{2\cdot 4} } \right){\frac {z^{5} } {5} } +\left({\frac {1\cdot 3\cdot 5}{2\cdot 4\cdot 6} } \right){\frac {z^{7} } {7} } +\cdots \\\\ &=\sum _{n=0}^{\infty }{\frac {(2n-1)!!}{(2n)!!} } \cdot {\frac {z^{2n+1} } {2n+1} } \\\\ &=\sum _{n=0}^{\infty }{\frac { {\binom {2n}{n} } z^{2n+1} } {4^{n}(2n+1)} } \qquad |z|\leq 1 \end{aligned} \]

We get \[ \begin{aligned} \arccos \frac{1}{n} &= \frac{\pi}{2} - \arcsin \frac{1}{n} \\\\ &= \frac{\pi}{2} - (\frac{1}{n} + \left({\frac {1}{2} } \right) {\frac {\frac{1}{n}^{3} } {3} } +\left({\frac {1\cdot 3}{2\cdot 4} } \right){\frac {\frac{1}{n}^{5} } {5} } +\left({\frac {1\cdot 3\cdot 5}{2\cdot 4\cdot 6} } \right){\frac {\frac{1}{n}^{7} } {7} } +\cdots ) \\\\ &\ge \frac{\pi}{2} - \frac{1}{n} - \frac{1}{2 n^3}\left( \sum_{i = 0}^\infty 1 / 2^i \right) \qquad since \ n \ge 2 \\\\ &\ge \frac{\pi}{2} - \frac{5}{4n} \end{aligned} \]

It follows that \[ P[|x_1| \ge \frac{1}{n}] = \frac{2}{\pi} \arccos \frac{1}{n} \ge 1 - \frac{1}{n} \]

For general values of \(d\), we would like to give an easier estimation by upper bounding the following integration

\[ \begin{aligned} \begin{matrix} \int_{x_1 = 0}^{1 / n} & \underbrace{\int_{x_2 = 0}^{1} \int_{x_3 = 0}^{1} ... \int_{x_d = 0}^{1} } & 1 \ d_{x_d} d_{x_{d - 1} } ... d_{x_1} \le \frac{1}{n} &\underbrace{\int_{x_2 = 0}^{1} \int_{x_3 = 0}^{1} ... \int_{x_d = 0}^{1} } & 1 \ d_{x_d} d_{x_{d - 1} } ... d_{x_2} \\\\ & x_2^2 + x_3^2 + ... + x_d^2 \le 1 - x_1^2 & & x_2^2 + x_3^2 + ... + x_d^2 = 1 & \end{matrix} \end{aligned} \]

Using the result that the surface area of a \(d - 1\) dimension sphere is given by

\[ {\frac {2\pi ^{\frac {d}{2} } }{\Gamma \left({\frac {d}{2} } \right)} } \]

We get \[ \begin{aligned} P[|x_1| \ge \frac{1}{n}] &= 1 - \frac{1}{n} \frac{ {\frac {2\pi ^{\frac {d}{2} } }{\Gamma \left({\frac {d}{2} } \right)} } }{ {\frac {2\pi ^{\frac {d + 1}{2} } }{\Gamma \left({\frac {d + 1}{2} } \right)} } } \\ &= 1 - \frac{1}{n} \frac{ \Gamma \left({\frac {d + 1}{2} } \right)}{ \Gamma \left({\frac {d}{2} } \right) \pi^{\frac {1}{2} } } \\ &\ge 1 - \sqrt{ \frac{d}{2\pi} } \frac{e}{n} \end{aligned} \]

where by Stirling’s formula \[ \begin{aligned} \frac{ \Gamma \left({\frac {d + 1}{2} } \right)}{ \Gamma \left({\frac {d}{2} } \right)} &\le (\frac{d + 1}{2})^{\frac {d + 1}{2} - 1 / 2} e^{1 / (12 (\frac {d + 1}{2}))} / (\frac{d}{2})^{\frac{d}{2} - 1 / 2} \\ &= (1 + \frac{1}{d})^{d/2} e^{1 / (6(d + 1))} (\frac{d}{2})^{1 / 2} \\ &\le e (\frac{d}{2})^{1 / 2} \end{aligned} \]
