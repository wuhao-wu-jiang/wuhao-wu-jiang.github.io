Example. Let

  • $\mathcal{X} \doteq [0, 100]^3$.
  • $d(x, x’)\doteq #\text{ different values between } x \text{ and } x’$.

  • $q(x) \doteq \min_{1 \le i < j \le 3} |x_i - x_j|$, where $x_i$ is the $i$-dimension of $x$.

    Then for the point $x = (0, 0, 0) \in [0, 100]^3$, if $x’ \sim x$, $x’$ contains at least two zeros in some of its dimensions. Therefore, $q(x’) = 0$ and $\delta(x) = 0$. But for $x = (0, 0, 100)$, it has a neighbor $x’ = (0, 50, 100)$. Now $q(x) = 0$ but $q(x’) = 50$. So $\Delta \ge \delta(x) \ge 50$.

We are interested in whether adding smaller noise $N \sim \mathbb{Lap}(\frac{\delta(x) }{\varepsilon})$ to the output of $q(x)$ would preserve privacy:

However, this approach is not private.
In the previous example, $(0, 0, 0)$ and $(0, 0, 100)$ are neighbors.
But for the former we add noise $N \sim \mathbb{Lap}(0)$ and for the latter we add noise $N \sim \mathbb{Lap}(\frac{50}{\varepsilon})$.
So the output distributions of them are very different and this would violate privacy protection.

Perhaps, we should propose some fixed parameter $\beta > 0$ (which could be much smaller than $\Delta$), and add a noise $N \sim \mathbb{Lap}(\frac{\beta}{\varepsilon})$ for whatever $x \in \mathcal{X}$.
For a pair of neighboring $x, x’$, when $|q(x) - q(x’)| \le \beta$, then the distributions of $q(x) + N$ and $q(x’) + N$ are indeed $(\varepsilon, 0)$ indistinguishable.
The only concern is for the case where $|q(x) - q(x’)| > \beta$ and adding a noise $N \sim \mathbb{Lap}(\frac{\beta}{\varepsilon})$ fails to protect the privacy.

This motivates the following propose-test-release algorithm.

  1. Propose a bound $\beta$ on local sensitivity.
  2. Compute $\hat d \doteq d(x, { y \in \mathcal{X} : \delta(y) > \beta }) + \mathbb{Lap}(1 / \varepsilon)$.
  3. If $\hat d \le \frac{1}{\varepsilon} \ln \frac{1}{\gamma}$ then
    $\qquad$ output $\bot$.
  4. Else
    $\qquad$ output $q(x) + \mathbb{Lap}(\beta / \varepsilon)$.

Theorem. The algorithm is $(2\varepsilon, \gamma / 2)$ differentially private.
