Report Noisy Max

Posted on 2022-03-06

Let d∈N+d \in \mathbb{N}^+d∈N+ be a positive integer, and u∈Ndu \in \mathbb{N}^du∈Nd be a vector.

Definition. Two vectors u,v⊂Ndu, v \subset \mathbb{N}^du,v⊂Nd are neighboring, denoted as u∼vu \sim vu∼v, if ∥u−v∥∞=1\Vert u - v \Vert_\infty = 1∥u−v∥∞​=1.

We are interested in the following research problem.

Problem. Design an ϵ\epsilonϵ-differentially private algorithm to report

arg⁡max⁡i∈[d]ui. \arg \max_{i \in [d]} u_i. argi∈[d]max​ui​.

Laplacian Mechanism

Since ∥u−v∥∞=1\Vert u - v \Vert_\infty = 1∥u−v∥∞​=1, it is possible that ∥u−v∥1=d\Vert u - v \Vert_1 = d∥u−v∥1​=d.
A naive approach would be adding independent Laplacian noise Xi∼Lap(ϵ/d)X_i \sim \mathcal{Lap}( \epsilon / d)Xi​∼Lap(ϵ/d) to each coordinate of uuu, and then return

M(u)≐arg⁡max⁡i∈[d](ui+Xi). \mathcal{M}(u) \doteq \arg \max_{i \in [d]} \big(u_i + X_i \big). M(u)≐argi∈[d]max​(ui​+Xi​).

Theorem (Privacy Guarantee). The algorithm is ϵ\epsilonϵ-differentially private.

Proof.

This follows directly from Laplacian mechanism.

□\square□

Theorem (Utility Guarantee). With probability at least 1−β1 - \beta1−β, it holds that

uM(u)≥(max⁡i∈[d]ui)−dϵln⁡dβ. u_{ \mathcal{M}(u) } \ge \left( \max_{i \in [d]} u_i \right) - \frac{d}{\epsilon} \ln \frac{d}{\beta}. uM(u)​≥(i∈[d]max​ui​)−ϵd​lnβd​.

Proof.
For each i∈[d]i \in [d]i∈[d],

Pr⁡[∣Xi∣≥dϵln⁡dβ]≤βd. \Pr \left[ |X_i| \ge \frac{d}{\epsilon} \ln \frac{d}{\beta} \right] \le \frac{\beta}{d}. Pr[∣Xi​∣≥ϵd​lnβd​]≤dβ​.

Via union bound,

Pr⁡[∃i∈[d]:∣Xi∣≥dϵln⁡dβ]≤β. \Pr \left[ \exists i \in [d] : |X_i| \ge \frac{d}{\epsilon} \ln \frac{d}{\beta} \right] \le \beta. Pr[∃i∈[d]:∣Xi​∣≥ϵd​lnβd​]≤β.

□\square□

Report Noisy Max

It is possible to add Laplacian noise Xi∼Lap(ϵ/2)X_i \sim \mathcal{Lap}(\epsilon / 2)Xi​∼Lap(ϵ/2) to each coordinate of uuu, and then return

M(u)≐arg⁡max⁡i∈[d](ui+Xi). \mathcal{M}(u) \doteq \arg \max_{i \in [d]} \big(u_i + X_i \big). M(u)≐argi∈[d]max​(ui​+Xi​).

Theorem (Privacy Guarantee). The algorithm is ϵ\epsilonϵ-differentially private.

Proof.
Fix a u∈Ndu \in \mathbb{N}^du∈Nd.
We prove that for each i∈[d]i \in [d]i∈[d], and each v∼uv \sim uv∼u, it holds that

e−ϵ⋅Pr⁡[M(v)=i]≤Pr⁡[M(u)=i]≤eϵ⋅Pr⁡[M(v)=i]. e^{-\epsilon} \cdot \Pr[ \mathcal{M}(v) = i] \le \Pr[ \mathcal{M}(u) = i ] \le e^\epsilon \cdot \Pr[ \mathcal{M}(v) = i]. e−ϵ⋅Pr[M(v)=i]≤Pr[M(u)=i]≤eϵ⋅Pr[M(v)=i].

By symmetry, we prove the claim for just i=ni = ni=n.

Let Xj∼Lap(ϵ/2),j∈[n]X_j \sim \mathcal{Lap}(\epsilon / 2), j \in [n]Xj​∼Lap(ϵ/2),j∈[n], and Zu=max⁡j∈[n−1](uj+Xj)Z_u = \max_{j \in [n - 1]} \big( u_j + X_j \big)Zu​=maxj∈[n−1]​(uj​+Xj​).
The event M(u)=n\mathcal{M}(u) = nM(u)=n happens if and only if

Zu<un+Xn. Z_u < u_n + X_n. Zu​<un​+Xn​.

Denote X⃗n−1≐(X1,…,Xn−1)\vec X_{n - 1} \doteq (X_1, \ldots, X_{n - 1})Xn−1​≐(X1​,…,Xn−1​).
Fix a sequence s⃗=(s1,…,sn−1)∈R\vec s = (s_1, \ldots, s_{n - 1} ) \in \mathbb{R}s=(s1​,…,sn−1​)∈R.
Conditioned on X⃗n−1=s⃗\vec X_{n - 1} = \vec sXn−1​=s, the probability density of Pr⁡[M(u)=n]\Pr\big[ \mathcal{M}(u) = n \big]Pr[M(u)=n] is given by

p[M(u)=n]=p[Zu<un+Xn]=∫p(x)⋅1[Zu<un+x] dx, \begin{aligned} p\big[ \mathcal{M}(u) = n \big] = p\big[ Z_u < u_n + X_n \big] = \int p(x) \cdot \mathbf{1}\big[ Z_u < u_n + x \big] \, dx, \end{aligned} p[M(u)=n]=p[Zu​<un​+Xn​]=∫p(x)⋅1[Zu​<un​+x]dx,​

where p(x)p(x)p(x) are probability densities of XnX_nXn​.

Similarly, let Yj∼Lap(ϵ/2),j∈[n]Y_j \sim \mathcal{Lap}(\epsilon / 2), j \in [n]Yj​∼Lap(ϵ/2),j∈[n] and Zv=max⁡j∈[n−1](vj+Yj)Z_v = \max_{j \in [n - 1]} \big( v_j + Y_j \big)Zv​=maxj∈[n−1]​(vj​+Yj​).
The conditional probability density of Pr⁡[M(v)=n]\Pr\big[ \mathcal{M}(v) = n \big]Pr[M(v)=n], when Y⃗n−1=s⃗\vec Y_{n - 1} = \vec sYn−1​=s, is given by

p[M(v)=n]=p[Zv<vn+Yn]=∫p(y)⋅1[Zv<vn+y] dy. \begin{aligned} p\big[ \mathcal{M}(v) = n \big] = p\big[ Z_v < v_n + Y_n \big] = \int p(y) \cdot \mathbf{1}\big[ Z_v < v_n + y \big] \, dy. \end{aligned} p[M(v)=n]=p[Zv​<vn​+Yn​]=∫p(y)⋅1[Zv​<vn​+y]dy.​

Suppose that the following equations holds:

un+x−Zu=vn+y−Zv. \begin{aligned} u_n + x - Z_u &= v_n + y - Z_v. \end{aligned} un​+x−Zu​​=vn​+y−Zv​.​

Then

1[Zu<un+x]=1[Zv<vn+y]. \mathbf{1}\big[ Z_u < u_n + x \big] = \mathbf{1}\big[ Z_v < v_n + y \big]. 1[Zu​<un​+x]=1[Zv​<vn​+y].

But this implies that

∣y−x∣=∣un−Zu−(vn−Zv)∣=∣un−vn+Zv−Zu∣≤2. \begin{aligned} |y - x| = \big| u_n - Z_u - \big( v_n - Z_v \big) \big| = \big| u_n - v_n + Z_v - Z_u \big| \le 2. \end{aligned} ∣y−x∣=∣∣​un​−Zu​−(vn​−Zv​)∣∣​=∣∣​un​−vn​+Zv​−Zu​∣∣​≤2.​

The inequality follows as conditioned on X⃗n−1=Y⃗n−1=s⃗\vec X_{n - 1} = \vec Y_{n - 1} = \vec sXn−1​=Yn−1​=s,

∣un−vn∣≤∥u−v∥∞≤1,∣Zv−Zu∣=∣max⁡j∈[n−1](vj+sj)−max⁡j∈[n−1](uj+sj)∣,≤max⁡j∈[n−1]∣vj−uj∣≤∥u−v∥∞≤1. \begin{aligned} | u_n - v_n | &\le \Vert u - v \Vert_\infty \le 1, \\ | Z_v - Z_u | &= \left| \max_{j \in [n - 1]} \big( v_j + s_j \big) - \max_{j \in [n - 1]} \big( u_j + s_j \big)\right|, \\ &\le \max_{j \in [n - 1]} \left| v_j - u_j \right| \le \Vert u - v \Vert_\infty \le 1. \end{aligned} ∣un​−vn​∣∣Zv​−Zu​∣​≤∥u−v∥∞​≤1,=∣∣∣∣​j∈[n−1]max​(vj​+sj​)−j∈[n−1]max​(uj​+sj​)∣∣∣∣​,≤j∈[n−1]max​∣vj​−uj​∣≤∥u−v∥∞​≤1.​

Therefore,

p[M(u)=n]p[M(v)=n]=∫p(x)⋅1[Zu<un+x] dx∫p(y)⋅1[Zv<vn+y] dy≤exp⁡(2⋅ϵ2)≤exp⁡(ϵ). \begin{aligned} \frac{p\big[ \mathcal{M}(u) = n \big] }{p\big[ \mathcal{M}(v) = n \big] } = \frac{ \int p(x) \cdot \mathbf{1}\big[ Z_u < u_n + x \big] \, dx } { \int p(y) \cdot \mathbf{1}\big[ Z_v < v_n + y \big] \, dy } \le \exp \Big( 2 \cdot \frac{\epsilon}{2}\Big) \le \exp(\epsilon). \end{aligned} p[M(v)=n]p[M(u)=n]​=∫p(y)⋅1[Zv​<vn​+y]dy∫p(x)⋅1[Zu​<un​+x]dx​≤exp(2⋅2ϵ​)≤exp(ϵ).​

Integrating over X⃗n−1=s⃗\vec X_{n - 1} = \vec sXn−1​=s (Y⃗n−1=s⃗\vec Y_{n - 1} = \vec sYn−1​=s) for all possible s⃗\vec ss finishes the proof.

□\square□

Theorem (Utility Guarantee). With probability at least 1−β1 - \beta1−β, it holds that

uM(u)≥(max⁡i∈[d]ui)−1ϵln⁡dβ. u_{ \mathcal{M}(u) } \ge \left( \max_{i \in [d]} u_i \right) - \frac{1}{\epsilon} \ln \frac{d}{\beta}. uM(u)​≥(i∈[d]max​ui​)−ϵ1​lnβd​.

Proof.
For each i∈[d]i \in [d]i∈[d],

Pr⁡[∣Xi∣≥1ϵln⁡dβ]≤βd. \Pr \left[ |X_i| \ge \frac{1}{\epsilon} \ln \frac{d}{\beta} \right] \le \frac{\beta}{d}. Pr[∣Xi​∣≥ϵ1​lnβd​]≤dβ​.

Via union bound,

Pr⁡[∃i∈[d]:∣Xi∣≥1ϵln⁡dβ]≤β. \Pr \left[ \exists i \in [d] : |X_i| \ge \frac{1}{\epsilon} \ln \frac{d}{\beta} \right] \le \beta. Pr[∃i∈[d]:∣Xi​∣≥ϵ1​lnβd​]≤β.

□\square□

Exponential Mechanism

The exponential mechanism is defined by its output distribution:

Pr⁡[M(u)=i]∝exp⁡(ϵ⋅ui/2),∀i∈[d]. \Pr[\mathcal{M}(u) = i] \propto \exp \left( \epsilon \cdot u_i / 2 \right), \quad \forall i \in [d]. Pr[M(u)=i]∝exp(ϵ⋅ui​/2),∀i∈[d].

Normalizing these probabilities give

Pr⁡[M(u)=i]=exp⁡(ϵ⋅ui/2)∑j∈[d]exp⁡(ϵ⋅uj/2),∀i∈[d]. \Pr[\mathcal{M}(u) = i] = \frac{ \exp \left( \epsilon \cdot u_i / 2 \right) }{ \sum_{j \in [d]} \exp \left( \epsilon \cdot u_j / 2 \right) }, \quad \forall i \in [d]. Pr[M(u)=i]=∑j∈[d]​exp(ϵ⋅uj​/2)exp(ϵ⋅ui​/2)​,∀i∈[d].

Theorem (Privacy Guarantee). The algorithm is ϵ\epsilonϵ-differentially private.

Proof.
Fix a u∈Ndu \in \mathbb{N}^du∈Nd.
We prove that for each i∈[d]i \in [d]i∈[d], and each v∼uv \sim uv∼u, it holds that

e−ϵ⋅Pr⁡[M(v)=i]≤Pr⁡[M(u)=i]≤eϵ⋅Pr⁡[M(v)=i]. e^{-\epsilon} \cdot \Pr[ \mathcal{M}(v) = i] \le \Pr[ \mathcal{M}(u) = i ] \le e^\epsilon \cdot \Pr[ \mathcal{M}(v) = i]. e−ϵ⋅Pr[M(v)=i]≤Pr[M(u)=i]≤eϵ⋅Pr[M(v)=i].

By symmetry, we prove the claim for just i=ni = ni=n.

Pr⁡[M(u)=n]Pr⁡[M(v)=n]=exp⁡(ϵ⋅un/2)exp⁡(ϵ⋅vn/2)⋅∑j∈[d]exp⁡(ϵ⋅vj/2)∑j∈[d]exp⁡(ϵ⋅uj/2)≤exp⁡(ϵ/2)⋅exp⁡(ϵ/2)=exp⁡(ϵ). \begin{aligned} \frac{\Pr[ \mathcal{M}(u) = n]}{\Pr[ \mathcal{M}(v) = n]} &= \frac{ \exp \left( \epsilon \cdot u_n / 2 \right) }{ \exp \left( \epsilon \cdot v_n / 2 \right) } \cdot \frac{ \sum_{j \in [d]} \exp \left( \epsilon \cdot v_j / 2 \right) }{ \sum_{j \in [d]} \exp \left( \epsilon \cdot u_j / 2 \right) } \\ &\le \exp( \epsilon / 2 ) \cdot \exp( \epsilon / 2 ) \\ &= \exp( \epsilon ). \end{aligned} Pr[M(v)=n]Pr[M(u)=n]​​=exp(ϵ⋅vn​/2)exp(ϵ⋅un​/2)​⋅∑j∈[d]​exp(ϵ⋅uj​/2)∑j∈[d]​exp(ϵ⋅vj​/2)​≤exp(ϵ/2)⋅exp(ϵ/2)=exp(ϵ).​

The inequality follows since

ui−vi≤∥u−v∥∞≤1,  ⟹  exp⁡(ϵ(ui−vi)/2)≤exp⁡(ϵ/2), ∀i∈[d]. \begin{aligned} u_i - v_i \le \Vert u - v \Vert_\infty \le 1, \implies \exp( \epsilon (u_i - v_i) / 2) \le \exp( \epsilon / 2), \, \forall i \in [d]. \end{aligned} ui​−vi​≤∥u−v∥∞​≤1,⟹exp(ϵ(ui​−vi​)/2)≤exp(ϵ/2),∀i∈[d].​

□\square□

Theorem (Utility Guarantee). With probability at least 1−β1 - \beta1−β, it holds that

uM(u)≥(max⁡i∈[d]ui)−2ϵ⋅ln⁡∣S∣∣SOPT∣⋅β. u_{ \mathcal{M}(u) } \ge \left( \max_{i \in [d]} u_i \right) - \frac{2}{\epsilon} \cdot \ln \frac{|S|}{|S_{OPT}| \cdot \beta}. uM(u)​≥(i∈[d]max​ui​)−ϵ2​⋅ln∣SOPT​∣⋅β∣S∣​.

Proof.
For each t∈Rt \in \mathbb{R}t∈R, the un-normalized probability mass assigned to the set S≐{i∈[d]:ui≤t}S \doteq \{ i \in [d] : u_i \le t \}S≐{i∈[d]:ui​≤t} is at most ∣S∣⋅exp⁡(ϵ⋅t/2)|S| \cdot \exp \left( \epsilon \cdot t / 2\right)∣S∣⋅exp(ϵ⋅t/2).

On the other hand, the un-normalized probability mass assigned to the set SOPT⊆[d]S_{OPT} \subseteq [d]SOPT​⊆[d] consisting of optimal solutions is ∣SOPT∣⋅exp⁡(ϵ⋅(max⁡i∈[d]ui)/2)|S_{OPT}| \cdot \exp \left( \epsilon \cdot \left( \max_{i \in [d]} u_i \right) / 2\right)∣SOPT​∣⋅exp(ϵ⋅(maxi∈[d]​ui​)/2).

Pr⁡[uM(u)≤t]≤∣S∣⋅exp⁡(ϵ⋅t/2)∣SOPT∣⋅exp⁡(ϵ⋅(max⁡i∈[d]ui)/2)≤∣S∣∣SOPT∣exp⁡(ϵ(t−max⁡i∈[d]ui)/2). \begin{aligned} \Pr \left[ u_{ \mathcal{M}(u) } \le t \right] &\le \frac{ |S| \cdot \exp \left( \epsilon \cdot t / 2\right) } { |S_{OPT}| \cdot \exp \left( \epsilon \cdot \left( \max_{i \in [d]} u_i \right) / 2 \right) } \\ &\le \frac{|S|}{|S_{OPT}|} \exp \left( \epsilon \left( t - \max_{i \in [d]} u_i \right) / 2 \right). \end{aligned} Pr[uM(u)​≤t]​≤∣SOPT​∣⋅exp(ϵ⋅(maxi∈[d]​ui​)/2)∣S∣⋅exp(ϵ⋅t/2)​≤∣SOPT​∣∣S∣​exp(ϵ(t−i∈[d]max​ui​)/2).​

To bound the last term by β\betaβ, we can set

t=(max⁡i∈[d]ui)−2ϵ⋅ln⁡∣S∣∣SOPT∣⋅β. t = \left( \max_{i \in [d]} u_i \right) - \frac{2}{\epsilon} \cdot \ln \frac{|S|}{|S_{OPT}| \cdot \beta}. t=(i∈[d]max​ui​)−ϵ2​⋅ln∣SOPT​∣⋅β∣S∣​.

□\square□
