Report Noisy Max

Posted on 2022-03-06

Let \(d \in \mathbb{N}^+\) be a positive integer, and \(u \in \mathbb{N}^d\) be a vector.

Definition. Two vectors \(u, v \subset \mathbb{N}^d\) are neighboring, denoted as \(u \sim v\), if \(\Vert u - v \Vert_\infty = 1\).

We are interested in the following research problem.

Problem. Design an \(\epsilon\)-differentially private algorithm to report \[ \arg \max_{i \in [d] } u_i. \]

Laplacian Mechanism

Since \(\Vert u - v \Vert_\infty = 1\), it is possible that \(\Vert u - v \Vert_1 = d\). A naive approach would be adding independent Laplacian noise \(X_i \sim \mathcal{Lap}( \epsilon / d)\) to each coordinate of \(u\), and then return \[ \mathcal{M}(u) \doteq \arg \max_{i \in [d] } \big( u_i + X_i \big). \]

Theorem (Privacy Guarantee). The algorithm is \(\epsilon\)-differentially private.

Proof.

This follows directly from Laplacian mechanism.

\(\square\)

Theorem (Utility Guarantee). With probability at least \(1 - \beta\), it holds that \[ u_{ \mathcal{M}(u) } \ge \left( \max_{i \in [d] } u_i \right) - \frac{d}{\epsilon} \ln \frac{d}{\beta}. \] Proof. For each \(i \in [d]\), \[ \Pr \left[ |X_i| \ge \frac{d}{\epsilon} \ln \frac{d}{\beta} \right] \le \frac{\beta}{d}. \] Via union bound, \[ \Pr \left[ \exists i \in [d] : |X_i| \ge \frac{d}{\epsilon} \ln \frac{d}{\beta} \right] \le \beta. \] \(\square\)

Report Noisy Max

It is possible to add Laplacian noise \(X_i \sim \mathcal{Lap}(\epsilon / 2)\) to each coordinate of \(u\), and then return \[ \mathcal{M}(u) \doteq \arg \max_{i \in [d] } \big( u_i + X_i \big). \]

Theorem (Privacy Guarantee). The algorithm is \(\epsilon\)-differentially private.

Proof. Fix a \(u \in \mathbb{N}^d\). We prove that for each \(i \in [d]\), and each \(v \sim u\), it holds that \[ e^{-\epsilon} \cdot \Pr[ \mathcal{M}(v) = i] \le \Pr[ \mathcal{M}(u) = i ] \le e^\epsilon \cdot \Pr[ \mathcal{M}(v) = i]. \] By symmetry, we prove the claim for just \(i = n\).

Let \(X_j \sim \mathcal{Lap}(\epsilon / 2), j \in [n]\), and \(Z_u = \max_{j \in [n - 1] } \big( u_j + X_j \big)\). The event \(\mathcal{M}(u) = n\) happens if and only if
\[ Z_u < u_n + X_n. \]

Denote \(\vec X_{n - 1} \doteq (X_1, \ldots, X_{n - 1})\). Fix a sequence \(\vec s = (s_1, \ldots, s_{n - 1} ) \in \mathbb{R}\). Conditioned on \(\vec X_{n - 1} = \vec s\), the probability density of \(\Pr\big[ \mathcal{M}(u) = n \big]\) is given by \[ \begin{aligned} p\big[ \mathcal{M}(u) = n \big] = p\big[ Z_u < u_n + X_n \big] = \int p(x) \cdot \mathbf{1}\big[ Z_u < u_n + x \big] \, dx, \end{aligned} \] where \(p(x)\) are probability densities of \(X_n\).

Similarly, let \(Y_j \sim \mathcal{Lap}(\epsilon / 2), j \in [n]\) and \(Z_v = \max_{j \in [n - 1] } \big( v_j + Y_j \big)\). The conditional probability density of \(\Pr\big[ \mathcal{M}(v) = n \big]\), when \(\vec Y_{n - 1} = \vec s\), is given by \[ \begin{aligned} p\big[ \mathcal{M}(v) = n \big] = p\big[ Z_v < v_n + Y_n \big] = \int p(y) \cdot \mathbf{1}\big[ Z_v < v_n + y \big] \, dy. \end{aligned} \]

Suppose that the following equations holds: \[ \begin{aligned} u_n + x - Z_u &= v_n + y - Z_v. \end{aligned} \]

Then \[ \mathbf{1}\big[ Z_u < u_n + x \big] = \mathbf{1}\big[ Z_v < v_n + y \big]. \]

But this implies that \[ \begin{aligned} |y - x| = \big| u_n - Z_u - \big( v_n - Z_v \big) \big| = \big| u_n - v_n + Z_v - Z_u \big| \le 2. \end{aligned} \]

The inequality follows as conditioned on \(\vec X_{n - 1} = \vec Y_{n - 1} = \vec s\), \[ \begin{aligned} | u_n - v_n | &\le \Vert u - v \Vert_\infty \le 1, \\ | Z_v - Z_u | &= \left| \max_{j \in [n - 1] } \big( v_j + s_j \big) - \max_{j \in [n - 1] } \big( u_j + s_j \big)\right|, \\ &\le \max_{j \in [n - 1] } \left| v_j - u_j \right| \le \Vert u - v \Vert_\infty \le 1. \end{aligned} \]

Therefore, \[ \begin{aligned} \frac{p\big[ \mathcal{M}(u) = n \big] }{p\big[ \mathcal{M}(v) = n \big] } = \frac{ \int p(x) \cdot \mathbf{1}\big[ Z_u < u_n + x \big] \, dx } { \int p(y) \cdot \mathbf{1}\big[ Z_v < v_n + y \big] \, dy } \le \exp \Big( 2 \cdot \frac{\epsilon}{2}\Big) \le \exp(\epsilon). \end{aligned} \]

Integrating over \(\vec X_{n - 1} = \vec s\) (\(\vec Y_{n - 1} = \vec s\)) for all possible \(\vec s\) finishes the proof.

\(\square\)

Theorem (Utility Guarantee). With probability at least \(1 - \beta\), it holds that \[ u_{ \mathcal{M}(u) } \ge \left( \max_{i \in [d] } u_i \right) - \frac{1}{\epsilon} \ln \frac{d}{\beta}. \] Proof. For each \(i \in [d]\), \[ \Pr \left[ |X_i| \ge \frac{1}{\epsilon} \ln \frac{d}{\beta} \right] \le \frac{\beta}{d}. \] Via union bound, \[ \Pr \left[ \exists i \in [d] : |X_i| \ge \frac{1}{\epsilon} \ln \frac{d}{\beta} \right] \le \beta. \] \(\square\)

Exponential Mechanism

The exponential mechanism is defined by its output distribution:
\[ \Pr[\mathcal{M}(u) = i] \propto \exp \left( \epsilon \cdot u_i / 2 \right), \quad \forall i \in [d]. \]

Normalizing these probabilities give \[ \Pr[\mathcal{M}(u) = i] = \frac{ \exp \left( \epsilon \cdot u_i / 2 \right) }{ \sum_{j \in [d] } \exp \left( \epsilon \cdot u_j / 2 \right) }, \quad \forall i \in [d]. \]

Theorem (Privacy Guarantee). The algorithm is \(\epsilon\)-differentially private.

Proof. Fix a \(u \in \mathbb{N}^d\). We prove that for each \(i \in [d]\), and each \(v \sim u\), it holds that \[ e^{-\epsilon} \cdot \Pr[ \mathcal{M}(v) = i] \le \Pr[ \mathcal{M}(u) = i ] \le e^\epsilon \cdot \Pr[ \mathcal{M}(v) = i]. \] By symmetry, we prove the claim for just \(i = n\).

\[ \begin{aligned} \frac{\Pr[ \mathcal{M}(u) = n] }{\Pr[ \mathcal{M}(v) = n] } &= \frac{ \exp \left( \epsilon \cdot u_n / 2 \right) }{ \exp \left( \epsilon \cdot v_n / 2 \right) } \cdot \frac{ \sum_{j \in [d] } \exp \left( \epsilon \cdot v_j / 2 \right) }{ \sum_{j \in [d] } \exp \left( \epsilon \cdot u_j / 2 \right) } \\ &\le \exp( \epsilon / 2 ) \cdot \exp( \epsilon / 2 ) \\ &= \exp( \epsilon ). \end{aligned} \]

The inequality follows since \[ \begin{aligned} u_i - v_i \le \Vert u - v \Vert_\infty \le 1, \implies \exp( \epsilon (u_i - v_i) / 2) \le \exp( \epsilon / 2), \, \forall i \in [d]. \end{aligned} \]

\(\square\)

Theorem (Utility Guarantee). With probability at least \(1 - \beta\), it holds that \[ u_{ \mathcal{M}(u) } \ge \left( \max_{i \in [d] } u_i \right) - \frac{2}{\epsilon} \cdot \ln \frac{|S|}{|S_{OPT}| \cdot \beta}. \] Proof. For each \(t \in \mathbb{R}\), the un-normalized probability mass assigned to the set \(S \doteq \{ i \in [d] : u_i \le t \}\) is at most \(|S| \cdot \exp \left( \epsilon \cdot t / 2\right)\).

On the other hand, the un-normalized probability mass assigned to the set \(S_{OPT} \subseteq [d]\) consisting of optimal solutions is \(|S_{OPT}| \cdot \exp \left( \epsilon \cdot \left( \max_{i \in [d] } u_i \right) / 2\right)\).

\[ \begin{aligned} \Pr \left[ u_{ \mathcal{M}(u) } \le t \right] &\le \frac{ |S| \cdot \exp \left( \epsilon \cdot t / 2\right) } { |S_{OPT}| \cdot \exp \left( \epsilon \cdot \left( \max_{i \in [d] } u_i \right) / 2 \right) } \\ &\le \frac{|S|}{|S_{OPT}|} \exp \left( \epsilon \left( t - \max_{i \in [d] } u_i \right) / 2 \right). \end{aligned} \]

To bound the last term by \(\beta\), we can set \[ t = \left( \max_{i \in [d] } u_i \right) - \frac{2}{\epsilon} \cdot \ln \frac{|S|}{|S_{OPT}| \cdot \beta}. \]

\(\square\)
