Karger Algorithm I

Posted on 2018-07-21 Edited on 2022-02-17

Algorithm

Let G=(V,E)G = (V, E)G=(V,E) be an undirected connected graph with nnn vertices, mmm edges, and min-cut size c∗c^*c∗.
David R. Karger [1] proposed a simple randomized algorithm for estimating the value of c∗c^*c∗, as described thus.

Karger Algorithm.

  1. repeat until two nodes are left in the graph:
  2. \qquad Select an edge e=(u,v)e = (u,v)e=(u,v) from the graph uniformly at random.
  3. \qquad Merge its endpoints uuu and vvv into new vertex www.
  4. \qquad Remove self loops created by the merge.
  5. \qquad Connect the edges incident to uuu or vvv to new vertex www.
  6. Return the number of edges, denoted as ccc, between the remaining two vertices.

Each iteration in the algorithm is called a contraction.
