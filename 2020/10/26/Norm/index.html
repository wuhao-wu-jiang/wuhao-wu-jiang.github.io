A norm ∣⋅∣:Rn→R| \cdot |:\mathbb{R}^n \rightarrow \mathbb{R}∣⋅∣:Rn→R is a function that satisfies the following properties:

  1. Positive definite: ∣x∣=0→x=0⃗|x| = 0 \rightarrow x = \vec 0∣x∣=0→x=0,
  2. Nonnegative:∣x∣≥0|x| \ge 0∣x∣≥0 for any x∈Rnx \in \mathbb{R}^nx∈Rn,
  3. Absolutely homogeneous: ∣kx∣=∣k∣∣x∣|k x| = |k| |x|∣kx∣=∣k∣∣x∣, for any x∈Rn,k∈Rx \in \mathbb{R}^n, k \in \mathbb{R}x∈Rn,k∈R,
  4. Subadditive (triangle inequality): ∣x+y∣≤∣x∣+∣y∣|x + y| \le |x| + |y|∣x+y∣≤∣x∣+∣y∣, for any x∈Rn,y∈Rnx \in \mathbb{R}^n, y \in \mathbb{R}^nx∈Rn,y∈Rn.

  1. The region {x∈Rn:∣x∣≤1}\{ x \in \mathbb{R}^n: |x| \le 1\}{x∈Rn:∣x∣≤1} is convex.

Proof: The proof is straightforward.

Application

The idea discussed above significantly simplifies the proof of Minkowski inequality.

The ppp-norm (p≥1p \ge 1p≥1) on Rn\mathbb{R}^nRn is given as

∣x∣p=(∑i=1n∣xi∣p)1p| x |_p = \big( \sum_{i = 1}^n |x_i|^p \big)^\frac{1}{p} ∣x∣p​=(i=1∑n​∣xi​∣p)p1​

Clearly, ppp-norm is positive definite, nonnegative, absolutely homogeneous. It is left to verify triangle inequality, which is known as Minkowski inequality for the case of ppp-norm. This is equivalent to show that

{x∈Rn:∣x∣p≤1}\{ x \in \mathbb{R}^n : |x |_p \le 1 \} {x∈Rn:∣x∣p​≤1}

is convex. The trick here is that ∣x∣p≤1|x|_p \le 1∣x∣p​≤1 is equivalent to (∣x∣p)p≤1(|x|_p)^p \le 1(∣x∣p​)p≤1. Hence,

{x∈Rn:∣x∣p≤1}={x∈Rn:∑i=1n∣xi∣p≤1}\{ x \in \mathbb{R}^n : |x |_p \le 1 \} = \{ x \in \mathbb{R}^n : \sum_{i = 1}^n |x_i|^p \le 1 \} {x∈Rn:∣x∣p​≤1}={x∈Rn:i=1∑n​∣xi​∣p≤1}

The convexity of this set follows directly from the point-wise convexity of the function ∣⋅∣p|\cdot |^p∣⋅∣p.
