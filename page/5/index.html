Norm of Random Variables

Posted on 2021-03-27 Edited on 2021-03-31

Fix probability space \((\Omega, \mathcal{F}, \Pr)\), for real number \(p > 0\), let \(L_p\) denote the vector space of real value random variable for which \(\mathbb{E} |X|^p < \infty\).

For \(p \ge 1\), the \(p\) norm \(||X||_p\) of a random variable \(X \in L_p\) is defined as \[ || X ||_p = \left( \mathbb{E} \big[ |X|^p \big] \right)^{1 / p}. \]

Norm

\(||X||_p\) is a norm.

Proof. This can be proven directly by Minkowski’s inequality. For completeness, we give the details here. Let \(X, Y \in L_p\), and let \(a = ||X||_p\) and \(b = ||Y||_p\). We want to prove that \[ ||X + Y||_p \le a + b \Longleftrightarrow \left\Vert \frac{a}{ a + b} \cdot \frac{X}{a} + \frac{a}{ a + b} \cdot \frac{Y}{a} \right\Vert_p \le 1. \]

It suffices to assume that \(||X||_p = ||Y||_p = 1\) and prove that

For \(t \in [0, 1]\), \(|| t X + (1 - t) Y||_p \le 1\).

This is much easier to prove as we can take the power of \(p\) of both side and show that: \[ \begin{aligned} || t X + (1 - t) Y ||_p^p &= \mathbb{E}[ |t X + (1 - t) Y |^p ] \\ &\le \mathbb{E}[ | tX |^p + | (1 - t) Y|^p ] \\ &= \mathbb{E}[ | tX |^p ] + \mathbb{E} [ | ( 1 - t) Y|^p ] \\ &= t^p ||X||_p^p + ( 1 - t)^p ||Y||_p^p \\ &\le t + (1 - t) \\ &= 1. \end{aligned} \]

\(\square\)

Properties

For \(1 \le p \le q\), \(||X|_p \le ||X||_q\).

Proof. Let \(t = q / p \ge 1\). By convexity of the function \(y = x^t\), we have \[ \begin{aligned} ||X||_p^t = (\mathbb{E} [ |X|^p ])^t \le \mathbb{E}[ (|X|^p)^t ] = \mathbb{E} [ |X|^q ]. \end{aligned} \] Raising both sides to the power of \(1 / q\) finish the proof.

\(\square\)

Let \(X_1, ..., X_n \in L_p\) be independent random variables and let \(r_1, ... ,r_n\) be independent Rademacher random variables. Then \[ \left\Vert \sum_{i \in [n] } X_i - \mathbb{E} \left[ \sum_{i \in [n] } X_i \right] \right\Vert_p \le 2 \cdot \left\Vert \sum_{i \in [n] } r_i X_i \right\Vert_p \]

Proof. For \(i \in [n]\), let \(Y_i\) be independent copy of \(X_i\). Then \[ \begin{aligned} \left\Vert \sum_{i \in [n] } X_i - \mathbb{E} \left[ \sum_{i \in [n] } X_i \right] \right\Vert_p &= \left\Vert \sum_{i \in [n] } X_i - \mathbb{E}_Y \left[ \sum_{i \in [n] } Y_i \right] \right\Vert_p \\ &= \left( \mathbb{E}_X \left[ \left| \sum_{i \in [n] } X_i - \mathbb{E}_Y \left[ \sum_{i \in [n] } Y_i \right] \right|^p \right] \right)^{1 / p} \\ &\le \left( \mathbb{E}_X \mathbb{E}_Y \left[ \left| \sum_{i \in [n] } X_i - \sum_{i \in [n] } Y_i \right|^p \right] \right)^{1 / p} &( \text{Jensen}) \\ &= \left\Vert \sum_{i \in [n] } (X_i - Y_i) \right\Vert_p \end{aligned} \]

Let \(Z = (Z_1, ..., Z_n)\) be a sequence of independent random variables, such that either \(Z_i = X_i - Y_i\) or \(Z_i = Y_i - X_i\). Let \(\Gamma = \{ Z \}\) be the set of \(2^n\) such sequences. As \(X_i - Y_i\) and \(Y_i - X_i\) have the same distributions, we have for \(Z \in \Gamma\), \[ \mathbb{E}_Z \left[ \left| \sum_{i \in [n] } Z_i \right|^p \right] = \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in [n]} X_i - Y_i \right|^p \right]. \]

Therefore, \[ \frac{1}{ |\Gamma| } \sum_{ Z \in \Gamma } \mathbb{E}_Z \left[ \left| \sum_{i \in [n] } Z_i \right|^p \right] = \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in [n]} X_i - Y_i \right|^p \right]. \]

If \(R = (r_1, ..., r_n)\) are Rademacher random variables, then \((r_1 (X_1 - Y_1), ..., r_n(X_n - Y_n)\) gives a uniformly sequence in \(\Gamma\). It holds that \[ \mathbb{E}_R \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in [n]} r_i (X_i - Y_i) \right|^p \right] = \frac{1}{ |\Gamma| } \sum_{ Z \in \Gamma } \mathbb{E}_Z \left[ \left| \sum_{i \in [n] } Z_i \right|^p \right] = \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in [n]} X_i - Y_i \right|^p \right]. \]

Hence, \[ \begin{aligned} \left\Vert \sum_{i \in [n] } X_i - \mathbb{E} \left[ \sum_{i \in [n] } X_i \right] \right\Vert_p &\le \left\Vert \sum_{i \in [n] } (X_i - Y_i) \right\Vert_p \\ &= \left\Vert \sum_{i \in [n] } r_i(X_i - Y_i) \right\Vert_p \\ &\le 2 \left\Vert \sum_{i \in [n] } r_i X_i \right\Vert_p \end{aligned} \]

\(\square\)

Remark: We can simply the argument as follows. For a fixed sequence \((r_1,..., r_n) \in \{-1, 1 \}^n\), the sequence \((r_1(X_1 - Y_1), ..., r_n(X_n - Y_n) )\) has the same distribution as \((X_1 - Y_n, ..., X_n - Y_n)\). Averaging over all such possible sequences gives that \[ \left\Vert \sum_{i \in [n] } (X_i - Y_i) \right\Vert_p = \left\Vert \sum_{i \in [n] } r_i(X_i - Y_i) \right\Vert_p. \]

Let \(X_1, ..., X_n\) be independent and mean zero random variables and let \(Y_1, ..., Y_n\) be independent copies of them. Then for any \((a_{i, j} )\) and for \(p \ge 1\), it holds that \[ \begin{aligned} \left\Vert \sum_{i \neq j} a_{i,j} X_i X_j \right\Vert_p \le 4\left\Vert \sum_{i \neq j} a_{i,j} X_i Y_j \right\Vert_p \end{aligned} \]

Proof. Let \(B_1, ..., B_n\) be independent Bernoulli random variables each of expectation \(1 / 2\). Then \[ \begin{aligned} \left\Vert \sum_{i \neq j} a_{i,j} X_i X_j \right\Vert_p &= \left( \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j \right|^p \right]\right)^{1 / p} \\ &= \left( \mathbb{E}_X \left[ \left| 4 \mathbb{E}_B \left[ \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right] \right|^p \right]\right)^{1 / p} \\ &\le \left( \mathbb{E}_X \left[ \mathbb{E}_B \left[ 4 \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right] \right]\right)^{1 / p} & (\text{Jensen} ) \\ &= 4 \left( \mathbb{E}_X \mathbb{E}_B \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right]\right)^{1 / p} \\ &= 4 \left( \mathbb{E}_B \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right]\right)^{1 / p} \end{aligned} \]

Question to ponder: Under what condition can we exchange the order of expectation (integration)?

But \[ \mathbb{E}_B \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right] \le \max_B \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right]. \]

Let \[ B^* = (b^*_1, ..., b^*_n) = \arg \max_{B} \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j B_i (1 - B_j) \right|^p \right] \]

Define \(S \doteq \{ i : b^*_i = 1 \}\). Then \[ \begin{aligned} \mathbb{E}_X \left[ \left| \sum_{i \neq j} a_{i,j} X_i X_j b^*_i (1 - b^*_j) \right|^p \right] &= \mathbb{E}_X \left[ \left| \sum_{i \in S, j \notin S } a_{i,j} X_i X_j \right|^p \right] \\ &= \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in S, j \notin S } a_{i,j} X_i Y_j \right|^p \right] \end{aligned} \]

Let \(Z_1, ..., Z_n\) and \(Z_1', ..., Z_n'\) both be independent copies of \(X_1, ..., X_n\). Then

\[ \begin{aligned} \mathbb{E}_{X, Y} \left[ \left| \sum_{i \in S, j \notin S } a_{i,j} X_i Y_j \right|^p \right] &= \mathbb{E}_{X, Y} \left[ \left| \mathbb{E}_{Z, Z'} \left( \sum_{i \in S} a_{i,j} X_i \cdot \left( \sum_{j \notin S} Y_j + \sum_{j \in S} Z_j' \right) + \sum_{i \notin S} a_{i,j} Z_i \cdot \left( \sum_{j \notin S} Y_j + \sum_{j \in S} Z_j' \right) \right) \right|^p \right] & ( \mathbb{E}[Z_i ] = \mathbb{E} [ Z_j' ] = 0 ) \\ &\le \mathbb{E}_{X, Y} \mathbb{E}_{Z, Z'} \left[ \left| \sum_{i \in S} a_{i,j} X_i \cdot \left( \sum_{j \notin S} Y_j + \sum_{j \in S} Z_j' \right) + \sum_{i \notin S} a_{i,j} Z_i \cdot \left( \sum_{j \notin S} Y_j + \sum_{j \in S} Z_j' \right) \right|^p \right] & (\text{Jensen} ) \\ &=\mathbb{E}_{X, Y} \left[ \left| \sum_{i , j } a_{i,j} X_i Y_j \right|^p \right]. \end{aligned} \]

\(\square\)

Fast Fourier Transform In Finite Field

Posted on 2021-03-24

We discuss the fast Fourier transform in a finite field \(\mathbb{F}\) with size \(p\).

For \(w \in \mathbb{F}\), and \(n \in \mathbb{F}_+\), define \[ M_{w, n} \doteq \begin{bmatrix} 1 &w &w^2 &... &w^{n - 1} \\ 1 &w^2 &w^4 &... &w^{2(n - 1)} \\ ... \\ 1 &w^{n} &w^{2n} &... &w^{n(n - 1)} \end{bmatrix}. \]

If the following two conditions hold:
1. \(n\) is some power of \(2\) and; 2. \(w^n = 1\) in the finite field.

Then \(\forall \vec x \doteq (x_1, x_2, ..., x_n) \in \mathbb{F}^n\), we can compute \[ M_{w, n} \vec x = \begin{bmatrix} 1 &w &w^2 &... &w^{n - 1} \\ 1 &w^2 &w^4 &... &w^{2(n - 1)} \\ ... \\ 1 &w^{n} &w^{2n} &... &w^{n(n - 1)} \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \\ ... \\ x_n \end{bmatrix} \] in \(O(n \log n)\) time.

Proof. If \(n = 2\), then the claim holds trivially.

Otherwise, let \(d = n / 2 = 2^{k - 1}\) and \(v = w^2\). The odd columns of the first \(d\) rows of the matrix \(M_{w, n}\) constitutes \[ \begin{bmatrix} 1 &w^2 &w^4 &... &w^{n - 2} \\ 1 &w^4 &w^8 &... &w^{2(n - 2)} \\ & & &... \\ 1 &w^{2d} &w^{4d} &... &w^{d(n - 2)} \end{bmatrix} = \begin{bmatrix} 1 &v &v^2 &... &v^{d - 1} \\ 1 &v^2 &v^4 &... &v^{2(d - 1)} \\ & & &... \\ 1 &v^{d} &v^{2d} &... &v^{d(d - 1)} \end{bmatrix} = M_{v, d}. \]

By assumption, for even numbers \(i = 2, 4, ..., (n - 2)\), we have \(i \cdot d = 1\) in the finite field. Hence, the odd columns of the last \(d\) rows of \(M_{w, n}\) is \[ \begin{bmatrix} 1 &w^{2(d + 1)} &w^{4(d + 1)} &... &w^{(n - 2)(d + 1)} \\ 1 &w^{2(d + 2)} &w^{4(d + 2)} &... &w^{(n - 2)(d + 2)} \\ & & &... \\ 1 &w^{2(d + d)} &w^{4(d + d)} &... &w^{(n - 2)(d + d)} \end{bmatrix} = \begin{bmatrix} 1 &w^2 &w^4 &... &w^{n - 2} \\ 1 &w^4 &w^8 &... &w^{2(n - 2)} \\ & & &... \\ 1 &w^{2d} &w^{4d} &... &w^{d(n - 2)} \end{bmatrix} = M_{v, d}. \]

Let \(\vec x_o, \vec x_e \in \mathbb{F}^d\) be the vectors obtained by the odd and even dimensions of \(\vec x\) respectively. By our construction,

  1. \(d\) is some power of \(2\) and;
  2. \(v^d = w^n = 1\).

Hence, by induction hypothesis we can compute \(M_{v, d} \vec x_o\) in \(O(d \log d) = O( n \log n)\) time.

Next, consider the even columns of the first \(d\) rows of the matrix \(M_{w, n}\) constitutes \[ \begin{aligned} \begin{bmatrix} w &w^3 &w^5 &... &w^{n - 1} \\ w^2 &w^6 &w^{10} &... &w^{2(n - 1)} \\ & & &... \\ w^d &w^{3d} &w^{5d} &... &w^{d(n - 1)} \end{bmatrix} &= \begin{bmatrix} w &... & \\ &w^2 &... \\ & & &... \\ & & & & w^d \\ \end{bmatrix} \begin{bmatrix} 1 &w^2 &w^4 &... &w^{n - 2} \\ 1 &w^4 &w^8 &... &w^{2(n - 2)} \\ & & &... \\ 1 &w^{2d} &w^{4d} &... &w^{d(n - 2)} \end{bmatrix} \\ &= \begin{bmatrix} w &... & \\ &w^2 &... \\ & & &... \\ & & & & w^d \\ \end{bmatrix} M_{v, d} \end{aligned} \]

Similarly, the even columns of the last \(d\) rows of the matrix \(M_{w, n}\) constitutes \[ \begin{aligned} \begin{bmatrix} w^{d + 1} &... & \\ &w^{d + 2} &... \\ & & &... \\ & & & & w^{d + d} \\ \end{bmatrix} M_{v, d} = (-1) \begin{bmatrix} w &... & \\ &w^2 &... \\ & & &... \\ & & & & w^d \\ \end{bmatrix} M_{v, d}. \end{aligned} \]

The equality comes from that \(w^d = w^{n / 2} = -1\). By induction hypothesis, we can compute \(M_{v, d} \vec x_e\) in \(O(n \log n)\) time.

Finally, the first \(d\) dimensions of \(M_{w, n} \vec x\) equals \[ M_{v, d} \vec x_o + \begin{bmatrix} w &... & \\ &w^2 &... \\ & & &... \\ & & & & w^d \\ \end{bmatrix} M_{v, d} \vec x_e, \] which can be computed in \(O(n \log n) + O(n \log n) + O(n) = O(n \log n)\) time.

The last \(d\) dimensions of \(M_{w, n} \vec x\) equals \[ M_{v, d} \vec x_o - \begin{bmatrix} w &... & \\ &w^2 &... \\ & & &... \\ & & & & w^d \\ \end{bmatrix} M_{v, d} \vec x_e, \] which can also be computed in \(O(n \log n) + O(n \log n) + O(n) = O(n \log n)\) time.

\(\square\)

The Poisson Approximation

Posted on 2021-03-22

Poisson Distribution

A discrete random variable \(X\) follows Poisson (\(ˈ\)pwɑːsɒn) distribution, with a parameter \(\lambda\), if its probability mass function is given by \[ \Pr[ X = k ] = e^{ - \lambda } \frac{ \lambda^k }{ k! }, \qquad \forall k \in \mathbb{N}. \]

We can verify that the definition is valid: \[ \sum_{ k \in \mathbb{N} } \Pr[ X = k ] = e^{ - \lambda } \sum_{ k \in \mathbb{N} } \frac{ \lambda^k }{ k! } = e^{ - \lambda } e^{ \lambda } = 1. \]

We use \(\text{Pois}(\lambda)\) to denote a Poisson distribution with parameter \(\lambda\).

Mean

Property. \(\mathbb{E}[X] = \lambda\).

Proof. \[ \sum_{ k \in \mathbb{N} } k\Pr[ X = k ] = e^{ - \lambda } \lambda \sum_{ 1 \le k \in \mathbb{N} } \frac{ \lambda^{k - 1} }{ (k - 1)! } = e^{ - \lambda } \lambda \sum_{ k \in \mathbb{N} } \frac{ \lambda^k }{ k! } = \lambda. \]

\(\square\)

Variance

Property. \(\mathbb{Var}[X] = \lambda\).

Proof. \[ \sum_{ k \in \mathbb{N} } k(k - 1)\Pr[ X = k ] = e^{ - \lambda } \lambda^2 \sum_{ 2 \le k \in \mathbb{N} } \frac{ \lambda^{k - 2} }{ (k - 2)! } = e^{ - \lambda } \lambda^2 \sum_{ k \in \mathbb{N} } \frac{ \lambda^k }{ k! } = \lambda^2. \]

Hence, \[ \begin{aligned} \mathbb{Var}[X] &= \sum_{ k \in \mathbb{N} } k^2 \Pr[ X = k ] - (\mathbb{E}[X] )^2 \\ &= \sum_{ k \in \mathbb{N} } ( k (k - 1) + k ) \Pr[ X = k ] - (\mathbb{E}[X] )^2 \\ &= \lambda^2 + \lambda - \lambda^2 \\ &= \lambda. \end{aligned} \]

\(\square\)

Sum of Two Independent Poisson Variable

Property. If \(X_1 \sim \text{Pois}(\lambda_1)\) and \(X_2 \sim \text{Pois}(\lambda_2)\), then \(X_1 + X_2 \sim \text{Pois} (\lambda_1 + \lambda_2)\).

Proof. For every \(k \in \mathbb{N}\), \[ \begin{aligned} \Pr[X_1 + X_2 = k] &= \sum_{a, b \in \mathbb{N}, a + b = k} \left( e^{- \lambda_1 } \frac{ \lambda_1^{a} }{ a! } \right) \left( e^{-\lambda_2 } \frac{ \lambda_2^b }{ b! } \right) \\ &= e^{- (\lambda_1 + \lambda_2) } \frac{1}{k!} \sum_{a, b \in \mathbb{N}, a + b = k} \frac{ k! }{ a! b! } \lambda_1^{a} \lambda_2^b \\ &= e^{- (\lambda_1 + \lambda_2) } \frac{ (\lambda_1 + \lambda_2)^k }{k!}. \end{aligned} \]

\(\square\)

Corollary. If \(X_i \sim \text{Pois}(\lambda_i)\) for every \(i \in [n]\), then \(\sum_{i \in [n] } X_i \sim \text{Pois} \left( \sum_{i \in [n] } \lambda_i \right)\).

Poisson Approximation

Binomial Distribution

The Poisson distribution can be used to approximate the limit of the binomial distribution, under some condition.

Poisson Limit Theorem. Suppose that \(\forall n \in \mathbb{N}^+\), the random variable \(Y_n\) follows binomial distribution \(\text{B}(n, p_n)\) and that \(n p_n = \lambda\). Let \(X \sim \text{Pois}(\lambda)\), then \[ \lim_n \Pr[ Y_n = k ] = \Pr[ X = k ]. \]

Proof. \[ \begin{aligned} \Pr[ Y_n = k ] &= \binom{n}{k } p_n^k (1 - p_n)^{n - k} \\ &= \frac{ n (n - 1) ... (n - k + 1) }{ k !} p_n^k ( 1 - p_n )^{n - k} \\ &= \frac{ n^k + O(n^{k - 1} ) }{ k !} \frac{ \lambda^k }{ n^k } \left( 1 - \frac{ \lambda }{ n } \right)^{n - k} \\ &= \frac{ \lambda^k \left( 1 + O( n^{-1} ) \right) }{ k !} \left( 1 - \frac{ \lambda }{ n } \right)^{n - k} \end{aligned} \]

For fixed \(k\), taking \(n \rightarrow \infty\), we have \[ \frac{ \lambda^k \left( 1 + O( n^{-1} ) \right) }{ k !} \left( 1 - \frac{ \lambda }{ n } \right)^{n - k} \rightarrow \frac{\lambda^k}{ k !} e^{-\lambda }. \]

Balls and Bins

We study the problem of throwing \(m\) balls into \(n\) bins randomly. For every \(i \in [n]\), let \(Y_i\) denote the number of balls in the \(i\)th bin, which follows binomial distribution \(\text{B}(m, 1 / m)\). We have shown that \(Y_i\) can be approximated by \(X_i \sim \text{Pois}(m / n)\). In this section, we show a stronger result that the joint distribution of \((Y_1, ..., Y_n)\) can be approximated by the joint distribution of Poisson random variables.

Theorem. \(\forall \lambda > 0\), let \(X_i \sim \text{Pois}(\lambda)\) be independent random variables. Then the joint distribution of \((X_1, ..., X_n)\) conditioned on \(\sum_{i \in [n]} X_i = m\) is the same as \((Y_1, ..., Y_n)\).

Remark: the theorem holds for \(\forall \lambda > 0\) and does not require that \(\lambda = m / n\).

Proof. Let \(X = \sum_{i \in [n] } X_i\). We have proven that \(X \sim \text{Pois}( n \lambda)\). Hence, \(\forall k_1, ..., k_n \ge 0\), s.t. \(\sum_{i \in [n] } k_i = m\), we have \[ \begin{aligned} \Pr[ X_1 = k_1, X_2 = k_2, ..., X_n = k_n \mid X = m ] &= \left( \prod_{i \in [n] } e^{-\lambda} \frac{ \lambda^{k_i} }{ k_i! } \right) / \left( e^{ - n\lambda} \frac{ (n\lambda)^m }{ m! } \right) \\ &= \frac{ e^{ - n \lambda} \lambda^{ \sum_{i \in [n] } k_i} }{ e^{ - n\lambda} (n\lambda)^m } \frac{ m! }{ \prod_{i \in [n] } k_i ! } \\ &= \binom{ m! }{ k_1; k_2; ...; k_n } \frac{1}{ n^m }. \end{aligned} \] \(\square\)

Theorem [1]. \(\forall i \in [n]\), \(X_i \sim \text{Pois}(m / n)\). Let \(f(y_1, ..., y_n)\) be a nonnegative function. Then \[ \mathbb{E}[ f(Y_1, ..., Y_n) ] \le e \sqrt{m} \mathbb{E}[ f(X_1, ..., X_n) ]. \]

Proof. Let \(X = \sum_{i \in [n] } X_i\). Then \(X \sim \text{Pois}(m)\). As \(f\) is nonnegative, it holds that \[ \begin{aligned} \mathbb{E}[ f(X_1, ..., X_n) ] &= \sum_{ k \in \mathbb{N} } \mathbb{E}[ f(X_1, ..., X_n) \mid X = k] \cdot \Pr[ X = k ] \\ &\ge \mathbb{E}[ f(X_1, ..., X_n) \mid X = m] \cdot \Pr[ X = m ] \\ &= \mathbb{E}[ f(Y_1, ..., Y_n) ] \cdot e^{ - m } \frac{ m^m }{ m! } \\ &\ge \mathbb{E}[ f(Y_1, ..., Y_n) ] \cdot e^{ - m } (m)^m \left( \frac{e}{m} \right)^m \frac{1}{ e \sqrt{m} } \\ &= \mathbb{E}[ f(Y_1, ..., Y_n) ] \cdot \frac{1}{ e \sqrt{m} }, \end{aligned} \] and the theorem is proven. The last inequality follows from Stirling's formula that \(m! \le e \sqrt{m} (m / e)^m\).

\(\square\)

Corollary. If some event happens with probability \(p\) with \(X_1, ..., X_n\), then it happens with probability at most \(e\sqrt{m} \cdot p\) with \(Y_1, ..., Y_n\).

Proof. Let \(f\) be the indicator variable of the event that takes 1 if the event occurs and 0 otherwise. Then by the previous theorem, \[ \Pr[ f(Y_1, ..., Y_n) ] \le e \sqrt{m} \Pr[ f(X_1, ..., X_n) ]. \]

\(\square\)

The analysis makes some analysis simpler since the \(X_i\)'s are independent random variables.

In some special cases, the theorem can be strengthened.

Theorem [1]. Let \(f(y_1, ..., y_n)\) be a nonnegative function such that \(\mathbb{E} [ f(Y_1, ..., Y_n) ]\) is either monotonically increasing or decreasing in \(m\). Then \[ \mathbb{E}[ f(Y_1, ..., Y_n) ] \le 2 \mathbb{E}[ f(X_1, ..., X_n) ]. \]

Proof. Refer to [1]. Omitted.

Reference

[1] M. Mitzenmacher and E. Upfal, “Probability and Computing,” p. 490.
