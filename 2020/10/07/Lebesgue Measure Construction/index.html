Measure Space

Let Ω\OmegaΩ be a set and 2Ω2^\Omega2Ω be the power set of Ω\OmegaΩ.

σ\sigmaσ-Algebra

Definition. A collection of subsets F⊆2Ω\mathcal{F} \subseteq 2^\OmegaF⊆2Ω is a σ\sigmaσ-algebra (or σ\sigmaσ-field) if

  • It is closed under countable union, intersection and complement.

Measure

Definition. Given a collection of subsets F⊆2Ω\mathcal{F} \subseteq 2^\OmegaF⊆2Ω, a function μ:F→R\mu : \mathcal{F} \rightarrow \mathbb{R}μ:F→R is a measure if it satisfies

  1. It is non-negative: ∀A∈F,μ(A)≥0\forall A \in \mathcal{F}, \mu(A) \ge 0∀A∈F,μ(A)≥0.

  2. The empty set is mapped to 0: μ(∅)=0\mu(\emptyset) = 0μ(∅)=0.

  3. σ\sigmaσ-additivity: if Ai∈FA_i \in \mathcal{F}Ai​∈F is a countable sequence of disjoint sets, s.t., ∪i≥1Ai∈F\cup_{i \ge 1} A_i \in \mathcal{F}∪i≥1​Ai​∈F, then

Measure Space

Definition. A measure space (Ω,F,μ)(\Omega, \mathcal{F}, \mu)(Ω,F,μ) consists of three part:

  1. A set Ω\OmegaΩ, on which it is defined;

  2. A σ\sigmaσ-field F\mathcal{F}F that contains all measurable sets;

  3. A measure μ\muμ defined on F\mathcal{F}F.

In general, it is not easy to construct a measure space on Ω\OmegaΩ directly. Our indirect construction consists of three steps:

  1. Consider a simple set class SSS and a measure μ\muμ defined on SSS.
  2. We extend the domain of μ\muμ from SSS to a larger one Ω=2Ω\Omega=2^\OmegaΩ=2Ω.
  3. We choose a subset from 2Ω2^\Omega2Ω, such that it is a σ\sigmaσ-field and μ∗\mu^*μ∗ is a measure defined on it.

In what follows, we illustrate the idea via constructing measures on Rd\mathbb{R}^dRd.

Remark. By De Morgan’s laws, A∩B‾=Aˉ∪Bˉ\overline{A \cap B } = \bar A \cup \bar BA∩B=Aˉ∪Bˉ and A∪B‾=Aˉ∩Bˉ\overline{A \cup B} = \bar A \cap \bar BA∪B=Aˉ∩Bˉ.
