Norm and Unit Ball

A norm ∥⋅∥:Rn→R\Vert \cdot \Vert:\mathbb{R}^n \rightarrow \mathbb{R}∥⋅∥:Rn→R is a function that satisfies the following properties:

  1. Positive definite: ∥x∥=0→x=0⃗\Vert x \Vert = 0 \rightarrow x = \vec 0∥x∥=0→x=0,

  2. Nonnegative:∥x∥≥0\Vert x \Vert \ge 0∥x∥≥0 for any x∈Rnx \in \mathbb{R}^nx∈Rn,

  3. Absolutely homogeneous: ∥kx∥=∣k∣⋅∥x∥\Vert k x\Vert = |k| \cdot \Vert x \Vert∥kx∥=∣k∣⋅∥x∥, for any x∈Rn,k∈Rx \in \mathbb{R}^n, k \in \mathbb{R}x∈Rn,k∈R,

  4. Subadditive (triangle inequality): ∥x+y∥≤∥x∥+∥y∥\Vert x + y \Vert \le \Vert x \Vert + \Vert y \Vert∥x+y∥≤∥x∥+∥y∥, for any x∈Rn,y∈Rnx \in \mathbb{R}^n, y \in \mathbb{R}^nx∈Rn,y∈Rn.

Theorem. Given properties 1-3, property 4 is equivalent to

  1. The region {x∈Rn:∥x∥≤1}\{ x \in \mathbb{R}^n: \Vert x \Vert \le 1\}{x∈Rn:∥x∥≤1} is convex.

Proof. The proof is straightforward.

5→4:5 \rightarrow 4:5→4: If x=0⃗x = \vec 0x=0 or y⃗=0\vec y = 0y​=0, then 4 holds trivially. Otherwise, suppose that x,y≠0⃗x, y \neq \vec 0x,y=0. Then

∥x+y∥≤∥x∥+∥y∥⟺∥∥x∥∥x∥+∥y∥x∥x∥+∥x∥∥x∥+∥y∥y∥y∥∥≤1. \begin{aligned} \Vert x + y \Vert \le \Vert x \Vert + \Vert y \Vert \Longleftrightarrow \left\Vert \frac{ \Vert x \Vert }{ \Vert x \Vert + \Vert y \Vert } \frac{ x }{ \Vert x \Vert } + \frac{ \Vert x \Vert }{ \Vert x \Vert + \Vert y \Vert } \frac{ y }{ \Vert y \Vert } \right\Vert \le 1. \end{aligned} ∥x+y∥≤∥x∥+∥y∥⟺∥∥∥∥∥​∥x∥+∥y∥∥x∥​∥x∥x​+∥x∥+∥y∥∥x∥​∥y∥y​∥∥∥∥∥​≤1.​

4→5:4 \rightarrow 5:4→5: Let x,y∈Rn,∥x∥≤1,∥y∥≤1x, y \in \mathbb{R}^n, \Vert x \Vert \le 1, \Vert y \Vert \le 1x,y∈Rn,∥x∥≤1,∥y∥≤1 and a,b≥0a, b \ge 0a,b≥0, a+b=1a + b = 1a+b=1. Then

Application

The idea discussed above significantly simplifies the proof of Minkowski inequality.

The ppp-norm (p≥1p \ge 1p≥1) on Rn\mathbb{R}^nRn is given as

∥x∥p=(∑i∈[n]∣xi∣p)1p \Vert x \Vert _p = \left( \sum_{i \in [n]} |x_i|^p \right)^\frac{1}{p} ∥x∥p​=⎝⎛​i∈[n]∑​∣xi​∣p⎠⎞​p1​

Clearly, ppp-norm is positive definite, nonnegative, absolutely homogeneous. It is left to verify triangle inequality, which is known as Minkowski inequality for the case of ppp-norm. This is equivalent to show that

{x∈Rn:∥x∥p≤1} \{ x \in \mathbb{R}^n : \Vert x \Vert_p \le 1 \} {x∈Rn:∥x∥p​≤1}

is convex. The trick here is that ∥x∥p≤1\Vert x \Vert_p \le 1∥x∥p​≤1 is equivalent to (∥x∥p)p≤1(\Vert x \Vert_p)^p \le 1(∥x∥p​)p≤1. Hence,

{x∈Rn:∥x∥p≤1}={x∈Rn:∑i∈[n]∣xi∣p≤1}. \{ x \in \mathbb{R}^n : \Vert x \Vert_p \le 1 \} = \left\{ x \in \mathbb{R}^n : \sum_{i \in [n]} |x_i|^p \le 1 \right\}. {x∈Rn:∥x∥p​≤1}=⎩⎪⎨⎪⎧​x∈Rn:i∈[n]∑​∣xi​∣p≤1⎭⎪⎬⎪⎫​.

The convexity of this set follows directly from the point-wise convexity of the function f(t)=tpf(t) = t^pf(t)=tp.

□\square□
