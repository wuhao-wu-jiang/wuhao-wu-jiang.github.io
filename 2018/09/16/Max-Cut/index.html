Max Cut

Posted on 2018-09-16 Edited on 2022-07-13

Given an undirected graph G=(V,E)G = (V, E)G=(V,E), the maximum cut problem asks for a subset S⊂VS \subset VS⊂V that maximizes the number of edges which cross SSS, i.e.,

S=arg⁡max⁡S′⊂Sδ(S), S = \arg \max_{S' \subset S} \delta(S), S=argS′⊂Smax​δ(S),

where δ(S)=∣{ (i,j) ∣ (i,j)∈E∧i∈S,j∉S}∣\delta(S) = |\{ \ (i, j) \ \mid \ (i, j) \in E \wedge i \in S, j \notin S\}|δ(S)=∣{ (i,j) ∣ (i,j)∈E∧i∈S,j∈/S}∣.

Programming Formulation

WLOG, we assume that i∈Si \in Si∈S if xi=1x_i = 1xi​=1.
The value of δ(S)\delta(S)δ(S) can be written as

δ(S)=∑(i,j)∈E1−xixj2 \delta(S) = \sum_{(i,j) \in E} \frac{1 - x_i x_j}{2} δ(S)=(i,j)∈E∑​21−xi​xj​​

Its correctness can be verified as follows:

  1. if i,j∈Vi, j \in Vi,j∈V belongs to the same partition, i.e, i,j∈Si, j \in Si,j∈S or i,j∈V−Si, j \in V - Si,j∈V−S, then xixj=1x_i x_j = 1xi​xj​=1 and (1−xixj)/2=0(1 - x_i x_j) / 2 = 0(1−xi​xj​)/2=0
  2. Otherwise, xixj=−1x_i x_j = -1xi​xj​=−1 and (1−xixj)/2=1(1 - x_i x_j) / {2} = 1(1−xi​xj​)/2=1.
