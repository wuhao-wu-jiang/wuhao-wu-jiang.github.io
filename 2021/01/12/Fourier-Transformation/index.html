Fourier Transformation

Posted on 2021-01-12 Edited on 2021-02-04

Let f:[−π,π]:→Rf: [-\pi, \pi]: \rightarrow \mathbb{R}f:[−π,π]:→R be a real function. As fff itself can be complicated, we want to decompose it into a linear combination of simpler functions.

Example. A typical family of simple functions are polynomials:

{1,t,t2,t3,...}. \{ 1, t, t^2, t^3, ... \}. {1,t,t2,t3,...}.

Under proper condition, e.g., if fff is infinitely differentiable, it can be indeed decomposed into a linear combination of polynomials, such as:

f=f(0)+f′(0)t+f′′(t)2!t2+... f = f(0) + f'(0) t + \frac{f''(t)}{2!} t^2 + ... f=f(0)+f′(0)t+2!f′′(t)​t2+...

Problem

Although f:[−π,π]:→Rf: [-\pi, \pi]: \rightarrow \mathbb{R}f:[−π,π]:→R is a function with output on R\mathbb{R}R, we view it as a function [−π,π]:→C[-\pi, \pi]: \rightarrow \mathbb{C}[−π,π]:→C to take advantage of complex exponential. Now, we approximate fff with a set of sinusoids:

The goal is to find the closest function to fff in S(Γ)S(\Gamma)S(Γ).

The key issues with the approximation are

  1. How do we measure the closeness between two functions?

  2. Under what condition, can fff be rewrite as a linear combination of functions in Γ\GammaΓ?

  3. What is the approximation error, between fff and the linear combination?
