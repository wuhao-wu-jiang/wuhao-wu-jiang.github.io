Poisson Tail Bound

Posted on 2021-06-13 Edited on 2021-06-16

We discuss deviation bounds for Poisson random variables. A random variable \(X\) follows Poisson distribution with mean \(\mu\), if for all \(n \ge 0\), \[ \Pr[ X = n] = \exp( -\mu ) \cdot \frac{ \mu^n }{ n! }. \]

The bounds share similar forms to the Chernoff bound for independent Bernoulli random variables.

Chernoff Bound. Let \(X_1, \ldots, X_n\) be independent 0-1 random variables such that \(\Pr[ X_i = 1] = \mu_i\). Let \(X = \sum_{i \in [n] } X_i\) and \(\mu = \mathbb{E} [ X_i ]\). Then 1. For all \(\delta > 0\), \[ \Pr[ X \ge (1 + \delta) \mu ] \le \left( \frac{ e^\delta }{ (1 + \delta )^{1 + \delta} } \right)^\mu; \] 2. For all \(\delta \in (0, 1)\),
\[ \Pr[ X \le (1 - \delta) \mu ] \le \left( \frac{ e^{- \delta } }{ (1 - \delta )^{1 - \delta} } \right)^\mu. \]

Moment Generating Function

Let \(X\) be a Poisson random variable with mean \(\mu\). For all \(s > 0\), \[ \begin{aligned} \mathbb{E} [ \exp( s X ) ] = e^{ ( e^s - 1 ) \mu }. \end{aligned} \]

Proof. For all \(s > 0\), \[ \begin{aligned} \mathbb{E} [ \exp( s X ) ] = \sum_n e^{ sn } e^{- \mu } \frac{ \mu^n }{ n! } = e^{- \mu } \sum_n \frac{ (e^s \cdot \mu)^n }{ n! } = e^{- \mu + e^s \mu } = e^{ ( e^s - 1 ) \mu }. \end{aligned} \]

\(\square\)

Upper Bound

Let \(X\) be a Poisson random variable with mean \(\mu\). Then for all \(\eta \ge 0\), \[ \Pr[ X \ge \mu + \eta ] \le \left( \frac{ \exp \Big( \frac{\eta}{ \mu} \Big) }{ \Big( 1 + \frac{ \eta }{ \mu } \Big)^{ 1 + \frac{ \eta }{ \mu } } } \right)^\mu \le \exp \left( - \frac{ \eta^2 }{ 2 \mu + \eta } \right). \]

Proof. Via Markov's inequality, for all \(s > 0, \eta \ge 0\), \[ \small \begin{aligned} \Pr[ X \ge \mu + \eta ] &= \Pr \Big[ \exp( s X ) \ge \exp \big( s ( \mu + \eta) \big) \Big] \\ &\le \frac{ \mathbb{E} [ \exp( s X ) ] }{ \exp \big( s ( \mu + \eta) \big) } \\ &= \exp \Big( ( e^s - 1 ) \mu - \big( s ( \mu + \eta) \big) \Big). \end{aligned} \]

Define \(f(s) = ( e^s - 1 ) \mu - \big( s ( \mu + \eta) \big)\). Letting \[ f'(s) = \mu e^s - (\mu + \eta) = 0, \]

we get \(s = \ln \frac{\mu + \eta}{ \mu}\) and \[ \small \begin{aligned} \Pr[ X \ge \mu + \eta ] &\le \exp \left( \Big( \frac{\mu + \eta}{ \mu} - 1 \Big) \cdot \mu - ( \mu + \eta) \cdot \ln \frac{\mu + \eta}{ \mu} \right) \\ &= \exp \left( - \mu \left\{ - \frac{\eta}{ \mu} + \Big( 1 + \frac{ \eta }{ \mu } \Big) \cdot \ln \Big( 1 + \frac{ \eta }{ \mu } \Big) \right\} \right) \\ &= \left( \frac{ \exp \Big( \frac{\eta}{ \mu} \Big) }{ \Big( 1 + \frac{ \eta }{ \mu } \Big)^{ 1 + \frac{ \eta }{ \mu } } } \right)^\mu. \end{aligned} \]

Next, we prove the inequality of \[ \left( \frac{ \exp \Big( \frac{\eta}{ \mu} \Big) }{ \Big( 1 + \frac{ \eta }{ \mu } \Big)^{ 1 + \frac{ \eta }{ \mu } } } \right)^\mu \le \exp \left( - \frac{ \eta^2 }{ 2 \mu + \eta } \right). \]

Let \(t = \frac{\eta}{\mu} \ge 0\). It suffices to prove that \[ \begin{aligned} (1 + t) \ln (1 + t) - t &\ge \frac{ t^2 }{ 2 + t }. \end{aligned} \]

The inequality follows from that for \(t \ge 0\), \[ \begin{aligned} \ln (1 + t) \ge \frac{t}{1 + t / 2} = \frac{2t}{2 + t}. \end{aligned} \]

The inequality clearly holds for \(t = 0\). Define \(g(x) = \frac{1}{1 + x}\), which is convex. Via Hermite-Hadamard inequality, when \(t > 0\), \[ \frac{\ln (1 + t) }{ t } = \frac{1}{t} \int_0^t g(x) \ dx \ge \frac{1}{ 1 + \frac{1}{t} \int_0^t x \ d x } = \frac{1}{ 1 + t / 2}. \]

Remark: A tighter bound shows that \[ (1 + t) \ln (1 + t) - t \ge \frac{t^2}{ 2 + (2 / 3) t}, \quad \forall t \ge 0. \]

which gives \[ \left( \frac{ \exp \Big( \frac{\eta}{ \mu} \Big) }{ \Big( 1 + \frac{ \eta }{ \mu } \Big)^{ 1 + \frac{ \eta }{ \mu } } } \right)^\mu \le \exp \left( - \frac{ \eta^2 }{ 2 \mu + (2 / 3) \eta } \right). \]

To prove it, define \[ f(t) \doteq (1 + t) \ln (1 + t) - t. \]

As \(f(0）= 0\), \[ \small \begin{aligned} f(t) = \int_0^1 f'( tx ) \, d\, (tx) = \int_0^1 \int_0^x f''( ty ) \, d\, (ty) \, d\, (tx) = t^2 \int_0^1 \int_0^x \frac{1}{1 + ty} \, d\, y \, d\, x \end{aligned}. \]

The function \(g(x, y) = \frac{1}{1 + ty}\) is convex on the region \[ \{ (x, y) : x \in [0, 1], y \in [0, x] \}. \]

It has area \(1 / 2\). By Jensen's inequality, \[ \frac{1}{1 / 2} \int_0^1 \int_0^x \frac{1}{1 + ty} \, d\, y \, d\, x \ge \frac{1}{1 + t \cdot \frac{1}{1 / 2} \cdot \int_0^1 \int_0^x y \, d\, y \, d\, x } = \frac{1}{1 + t / 3}. \]

Remark: A looser bound can be obtained as follows. Define \[ f(t) \doteq (1 + t) \ln (1 + t) - t. \] Then \(f'(t) = \ln (1 + t)\) and \(f''(t) = 1 / (1 + t)\). Using Taylor's expansion, there exists some \(c \in [0, t]\), such that \[ f(t) = f(0) + f'(0) \cdot t + f''(c) \cdot \frac{t^2}{2} = \frac{1}{1 + c} \cdot \frac{t^2}{2}. \]

The bound we obtained before asserts that \(c \le t / 2\). This is due to that \(f(t)\) is convex and is increasing for \(t \ge 0\). Even if we don't know this, we can lower bound \(f(t)\) by \[ f(t) = \frac{1}{1 + c} \cdot \frac{t^2}{2} \ge \frac{1}{1 + t} \cdot \frac{t^2}{2}, \]

which gives \[ \left( \frac{ \exp \Big( \frac{\eta}{ \mu} \Big) }{ \Big( 1 + \frac{ \eta }{ \mu } \Big)^{ 1 + \frac{ \eta }{ \mu } } } \right)^\mu \le \exp \left( - \frac{ \eta^2 }{ 2 (\mu + \eta) } \right). \]

\(\square\)

Lower Bound

Let \(X\) be a Poisson random variable with mean \(\mu\). Then for all \(0 \le \eta \le \mu\), \[ \Pr[ X \le \mu - \eta ] = \left( \frac{ \exp \Big( \frac{\eta}{ \mu} \Big) }{ \Big( 1 - \frac{ \eta }{ \mu } \Big)^{ 1 - \frac{ \eta }{ \mu } } } \right)^\mu \le \exp \left( - \frac{ \eta^2 }{ 2 \mu } \right). \]

Proof. Via Markov's inequality, for all \(s > 0, 0 \le \eta \le \mu\), \[ \small \begin{aligned} \Pr[ X \le \mu - \eta ] &= \Pr \Big[ \exp( - s X ) \ge \exp \big( -s ( \mu - \eta) \big) \Big] \\ &\le \frac{ \mathbb{E} [ \exp( - s X ) ] }{ \exp \big( - s ( \mu - \eta) \big) } \\ &= \exp \Big( ( e^{-s} - 1 ) \mu + \big( s ( \mu - \eta) \big) \Big). \end{aligned} \]

Define \(f(s) = ( e^{-s} - 1 ) \mu + \big( s ( \mu - \eta) \big)\). Letting \[ f'(s) = - \mu e^{-s} + (\mu - \eta) = 0, \]

we get \(s = - \ln \frac{\mu - \eta}{ \mu}\) and \[ \small \begin{aligned} \Pr[ X \ge \mu - \eta ] &\le \exp \left( \Big( \frac{\mu - \eta}{ \mu} - 1 \Big) \cdot \mu - ( \mu - \eta) \cdot \ln \frac{\mu - \eta}{ \mu} \right) \\ &= \exp \left( - \mu \left\{ \frac{\eta}{ \mu} + \Big( 1 - \frac{ \eta }{ \mu } \Big) \cdot \ln \Big( 1 - \frac{ \eta }{ \mu } \Big) \right\} \right) \\ &= \left( \frac{ \exp \Big( - \frac{\eta}{ \mu} \Big) }{ \Big( 1 - \frac{ \eta }{ \mu } \Big)^{ 1 - \frac{ \eta }{ \mu } } } \right)^\mu. \end{aligned} \]

Next, we prove the inequality of \[ \left( \frac{ \exp \Big( - \frac{\eta}{ \mu} \Big) }{ \Big( 1 - \frac{ \eta }{ \mu } \Big)^{ 1 - \frac{ \eta }{ \mu } } } \right)^\mu \le \exp \left( - \frac{ \eta^2 }{ 2 \mu } \right). \]

Let \(t = \frac{\eta}{\mu} \in [0, 1)\). It suffices to prove that \[ (1 - t) \ln (1 - t) + t \ge \frac{ t^2 }{ 2 }. \]

Let \(f(t) = (1 - t) \ln (1 - t) + t\). Then \[ \begin{aligned} f'(t) &= -\ln (1 - t), \\ f''(t) &= \frac{1}{1 - t}. \end{aligned} \]

Using Taylor's expansion, there exists some \(c \in [0, t]\), such that \[ f(t) = f(0) + f'(0) \cdot t + f''(c) \cdot \frac{t^2}{2} = \frac{1}{1 - c} \cdot \frac{t^2}{2}. \]

The proof follows by observing that \(\frac{1}{1 - c} \ge 1\).

\(\square\)
