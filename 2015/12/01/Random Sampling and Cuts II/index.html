Sampling Theorem

In 1994, Karger gave the following result:

[Sampling Karger] [Using randomized sparsification to approximate minimum cuts] Given a graph G and a collection of random variables Xe(Xe≤M)X_e (X_e \le M)Xe​(Xe​≤M) associated with all edges e, with probability 1−O(n−d)1 - O(n^{-d})1−O(n−d) that every cut in the instance G(Xe)G(X_e)G(Xe​) generated by putting weight XeX_eXe​ on each e has value with (1−ϵ,1+ϵ)(1 - \epsilon, 1 + \epsilon)(1−ϵ,1+ϵ) times its expected value, where

ϵ=2(d+2)Mln⁡nc′\epsilon = \sqrt{2(d + 2)M\ln n \over c'} ϵ=c′2(d+2)Mlnn​​

and c’ is G’s minimum expected value.

As an application, if we fix ϵ\epsilonϵ and let p=Θ(ln⁡nϵ2c)p = \Theta({\ln n \over \epsilon^2 c})p=Θ(ϵ2clnn​), where c is the minimum cut of G, then G’ approximate all its original graph cuts by a factor p with high probability.

The limitation is that either ϵ\epsilonϵ or p depends on c. When c is little, we could not reduce many edges.

Improvement:
In 1996, Benczur and Karger improved previous sampling method such that the dependence on c is removed.

New Result:

Given a graph G and a parameter ϵ\epsilonϵ, there is a new graph G’ such that

  1. G’ has O(nln⁡nϵ2)O({n \ln n \over \epsilon^2})O(ϵ2nlnn​) edges.
  2. every cut in G’ has value (1±ϵ)(1 \pm \epsilon)(1±ϵ) times its original cut value.

The intuition is that in the dense parts of the graph, each edge can be sample with smaller probability in order to preserve cut value while edge should be sample with higher probability where the graph is sparse.

Before moving on we first formalize the notion of a dense region. We characterize a dense region by Strong Connectivity.

k-Connected

A graph G is k connected if its minimum cut value is no less than k.

Vertex-Induced Subgraph

G′=<V′,E′>G'=<V',E'>G′=<V′,E′> is a vertex induced subgraph of G=<V,E>G =<V,E>G=<V,E> if V′V'V′ is a subset of VVV and E′E'E′ is all edges of EEE whose endpoints are both in V′V'V′.

k-Strong Component

A k-Strong Component is a maximal k-connected vertex-induced subgraph.

Strong Connectivity

The strong connectivity cec_ece​ of an edge e is the maximum value of k such that a k-Strong component contains e.

Number of edges

Theorem 1

If an undirected connected graph has k(n - 1) edges, then it must has a k strong component.

Proof: We prove it by induction. When k = 1, then the graph itself is a k-strong component. When n = 2, the theorem clearly holds.

Now we consider the general case. Suppose that the graph does not contain a k-strong component. Then there exist a cut C=<A,B>C = <A, B>C=<A,B> in the graph such that cut(A,B)<kcut(A,B) < kcut(A,B)<k. Now both A and B have less than n nodes. By induction, A has less then k(|A| - 1) edges inside and B has less then k(|B| - 1) edges inside. The total number of edges is less than k(|A| - 1) + k + k(|B| - 1) = k(|A|+|B| - 1) = k(n - 1). A contradiction.
