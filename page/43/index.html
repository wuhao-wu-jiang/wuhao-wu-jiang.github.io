Priority Sampling

Posted on 2020-01-12 Edited on 2021-02-24

Suppose there are a set of \(n\) items with nonnegative weights \(w_1, w_2, ..., w_n\). We want to maintain a subset of samples \(S \subset [n]\) in order to answer queries like given arbitrary \(I \subset [n]\) , what is \(\sum_{i \in I} w_i\)?

A trivial solution is to set \(S = [n]\) and we can answer the query exactly. This is an overkill if our goal is to only answer the query approximately.

Alternatively, we can sample each item \(i \in [n]\) independently with probability \(p_i\) (to be determined), and maintain a random variable \[ \begin{aligned} X_i = \begin{cases} \frac{w_i}{p_i} & i \text{ is chosen} \\ 0 & \text{ else } \end{cases} \end{aligned} \]

Note that \[ \mathbb{E} \left[ X_i \right] = w_i \]

And therefore, \[ \mathbb{E} \left[\sum_{i \in I} X_i \right] = \sum_{i \in I} \mathbb{E}[X_i] = \sum_{i \in I} w_i \]

Threshold Sampling

One possible implementation of this sampling scheme is called threshold sampling.

First, a threshold \(T\) is chosen. Then, for each item \(i \in [n]\), if \(w_i \ge T\), then the algorithm record a value \(X_i = w_i\) deterministically. Otherwise, the algorithm record a value \(X_i = T\) with probability \(\frac{w_i}{T}\). If \(X_i\) is not recorded, it has default value \(0\). In both cases, \(\mathbb{E} [ X_i ] = w_i\).

Mathematically, the two cases can be combined into one. Let \(u_i\) be a random number generated between \((0, 1)\). Then

\[ \begin{aligned} X_i = \begin{cases} \max \{ w_i, T \} & u_i < \frac{w_i}{T} \\ 0 & \text{ else } \end{cases} \end{aligned} \]

Observe that if \(w_i \ge T\), then \(w_i / T \ge 1\) and \(X_i \equiv \max \{ w_i, T \} = w_i\). The expectation of \(X_i\) is given algebraically by \[ \begin{aligned} \mathbb{E} [ X_i ] = \max \{ w_i, T \} \cdot \Pr \left[ u_i \le \frac{w_i}{T} \right] = \max \{ w_i, T \} \cdot \min \{ 1, w_i / T \} = w_i. \end{aligned} \]

It follows by linearity of expectation that \[ \mathbb{E} \left[ \sum_{i \in I} X_i \right] = \sum_{i \in I} w_i. \]

Space Usage

Denote \(S\) the set of items with \(X_i \neq 0\) and let \(p_i\) be the probability of \(i\) being sampled. Then the expected size of \(S\) is given by \[ \sum_{i = 1}^n p_i = \sum_{i = 1}^n \min \left\{ 1 , \frac{w_i}{T} \right\} \]

The problem with threshold sampling is that it is hard to control the memory usage of the sample set \(S\). If the \(n\) items are known offline, and we would like \(|S| \approx k\) for some integer \(k \le n\), then we can search for some \(T\) such that \[ \sum_{i = 1}^n p_i = k \]

Then \(|S| = k\) in expectation.

The question is, is it possible to force \(|S|= k\) exactly? Further, what if the items come online (one by one)?

Priority Sampling

Priority sampling achieves this. First recall that in threshold sampling we decide whether to keep \(i\) by checking whether a uniform sampled real number \(u_i\) is less than \(\frac{w_i}{T}\): \[ u_i < \frac{w_i}{T} \]

This is equivalent to \[ T < \frac{w_i}{u_i} \]

Define the priority of an item \(i \in [n]\) as \(q_i = \frac{w_i}{u_i}\). Note that \(q_i\) is a random variable. In threshold sampling, item \(i\) is kept with weight \(\max \{ w_i, T \}\) if \[ T < q_i \]

Now, instead of using a fixed value of \(T\), the Priority sampling sets \[ T = (k + 1) \text{-th largest } \{ q_i \} \]

By the definition of \(T\), only the \(k\) items with largest priorities \(q_i\)'s are kept in the sample set \(S\): \[ S = \{ k \text{ items with largest } q_i \text{ 's } \} \]

It can be seen that \(S\) and \(T\) can be easily maintained by a binary heap when the items come in stream fashion.

The challenge is not to analyze the behaviors of the r.v.'s: \[ \begin{aligned} X_i = \begin{cases} \max \{ w_i, T \} & T < \frac{w_i}{u_i} \equiv u_i < \frac{w_i}{T} \\ 0 & \text{ else } \end{cases} \end{aligned} \]

since now \(T\) is itself a random variable.

Theorem

\[ \mathbb{E}[X_i] = w_i. \]

Proof. There is an elegant proof based on conditional expectation. For \(i \in [n]\), define the random variable \(T_i\) as the \(k\)-th largest priority of elements other than \(i\). If \(i\)-th is sampled, then \(q_i \ge T_i\) and \(T_i\) becomes the \(k+1\)-th largest priority. Therefore, \[ \mathbb{E}_{X_i} [ X_i \mid T_i ] = \max \{ X_i, T_i \} \cdot \min \{1, w_i / T_i \} = w_i. \]

Finally, \(\mathbb{E}_{X_i} [X_i ] = \mathbb{E}_{T_i} [ \mathbb{E}_{X_i} [ X_i \mid T_i] ] = w_i\).

Theorem

For \(I \subset [n]\), \[ \begin{aligned} \mathbb{E}\left[ \prod_{i \in I} X_i \right] = \begin{cases} \prod_{i \in I} w_i & \text{ if } |I| \le k \\ 0 & else \end{cases} \end{aligned} \]

Proof: Clearly, when \(|I| > k\), then \(\exists i \in I\), such that \(q_i\) is not one of the \(k\) largest priorities. Hence \(\prod_{i \in I} X_i = 0\) for sure.

If \(|I| \le k\), define \(T_I\) to be the \(k - |I| + 1\)-th largest priority of elements in \([n] \setminus I\). Observe that \(\prod_{i \in I} X_i \neq 0\), if only the priorities of all \(i \in I\) are among the \(k\) largest ones. In such case, \(T_I\) becomes the \(k+1\)-th largest priority. Therefore, \[ \mathbb{E} \left[ \prod_{i \in I} X_i \mid T_I \right] = \prod_{i \in I} \mathbb{E} [ X_i \mid T_I ] = \prod_{i \in I} w_i, \]

and \(\mathbb{E}[ \prod_{i \in I} X_i ] = \mathbb{E} [ \mathbb{E} \left[ \prod_{i \in I} X_i \mid T_I \right] ] = \prod_{i \in I} w_i\).

Reference

  1. Nick Duffield, Carsten Lund, and Mikkel Thorup. Priority sampling for estimation fo arbitrary subset sums. Journal of ACM, 2007
  2. Chandra Chekuri, Lecture note, CS 598: Algorithms for Big Data

Newton's Method

Posted on 2020-01-02

Newton's method is an algorithm for finding root of a function \(f(x)\) numerically. If \(f\) is differentially at a point \(x\), then we can use this local information at \(x\) to approximate \(f\): \[ \bar f(y) = f(x) + f'(x) (y - x) \]

This is the tangent line at \(x\). We use its intersection with the \(x\)-axis as the guess of root: \[ \bar f(y) = 0 = f(x) + f'(x) (y - x) \]

If \(f'(x) \neq 0\), then \(y = x - \frac{f(x)}{f'(x)}\). Intuitively, \(f(x)\) is the height and \(f'(x)\) is the slope, hence \(\frac{f(x)}{f'(x)} = x - y\) gives the distance between \(x\) and \(y\). Note that if \(f'(x) = 0\), the tangent line does not intersect with \(x\)-axis.

With a starting point \(x_1\), the Newton's Method updates its value iteratively: \[ x_{n + 1} = x_n - \frac{f(x_n) }{ f'(x_n) } \]

Let \(z\) be the root of \(f\) (\(f(z) = 0\)). Then under proper condition, \(x_n\) converges to \(z\) quadratically, that is, \(|x_n - z| \le O(2^{-n})\).

Theorem.

Let \(f \in C^2\) in some interval \([z \pm c]\) and \(f'(x) \neq 0\), \(f''(x)\) is bounded for \(x \in [z \pm c]\). If \(x_0 \in [z \pm c]\), then Newton's Method achieves quadratic convergence.

Proof: By Taylor expansion, \(\forall x \in [z \pm c]\), \(\exists y\) that lies between \(x\) and \(z\), such that \[ 0 = f(z) = f(x) + f'(x) (x - z) + \frac{f''(y)}{2} (z - x)^2 \]

Hence \[ x - \frac{f(x)}{f'(x)} - z = \frac{f''(y)}{2 f'(x)} (z - x)^2 \]

Define \(\epsilon_n = (z - x_n)\), and replacing \(x\) with \(x_n\), we have \[ |\epsilon_{n + 1}| = |x_{n + 1} - z| = |x_n - \frac{f(x_n)}{f'(x_n)} - z| = |\frac{f''(y)}{2 f'(x_n)}| (z - x_n)^2 = |\frac{f''(y)}{2 f'(x_n)}| \epsilon_n^2 \]

Since \(f\)'s second derivative is continuous, \(f''(y)\) is bounded on the closed interval \([z \pm c]\). Further, as \(f'(x) \neq 0\), then \(\min_{x [z \pm c]} f'(x) > 0\) as \(f'(x)\) is continuous on the closed interval \([z \pm c]\). Therefore, \[ C \doteq \max |\frac{f''(y)}{2 f'(x_n)}| \] is bounded. Now \[ |\epsilon_{n + 1}| \le |C| \epsilon_n^2 \]

If for some \(n\) such that \(\epsilon_n < 1\), then the number of precise digits doubles for \(\epsilon_{n + 1}\).

Online Frequent Directions

Posted on 2020-01-01 Edited on 2024-08-27

Problem

One scenario is to approximate \(A \in \mathbb{R}^{n \times d}\) in an online fashion, i.e., the rows of \(A\) comes on by one (no missing values). Denote the \(i\)-th row of \(A\) as \(a_i\). We need to maintain a smaller matrix \(B \in \mathbb{R}^{\ell \times d}\) (\(\ell \ll n\)), termed the matrix sketch of \(A\), trying to minimize \[ \begin{aligned} \max_{x \in \mathbb{R}^d, \Vert x \Vert_2 = 1} \Big| \Vert A x \Vert_2^2 - \Vert B x \Vert_2^2 \Big| &= \max_{x \in \mathbb{R}^d, \Vert x \Vert_2 = 1} \Big| x^T (A^T A - B^T B) x \Big| = \Vert A^T A - B^T B \Vert_2. \end{aligned} \]

Algorithm

We will present an algorithm, which achieves an error of \[ \Vert A^T A - B^T B \Vert_2 \le \min_{k \in [\ell]} \frac{\Vert A - A_{[k]} \Vert _F^2}{ \ell - k }, \] where \(\Vert A \Vert _F \doteq \sqrt{ \sum_{ i \in [n], j \in [d] } A_{i,j}^2 }\) is the Frobenius norm of \(A\).

Algorithm [1]
1. \(B \leftarrow\) empty matrix.
2. For \(i \in [n]\):
3. \(\qquad\) \(B \leftarrow [B ; a_i]\) matrix obtained by inserting \(a_i\) after the last row \(B\)
4. \(\qquad\) \(B\) has \(\ell\) rows:
5. \(\qquad\) \(\qquad\) \(U_i, \Sigma_i, V_i \leftarrow \text{SVD} (B)\), such that \(B = U_i \Sigma_i V_i^T\).
6. \(\qquad\) \(\qquad\) \(\overline{\Sigma_i} \leftarrow \sqrt{ \max \{ \Sigma_i^2 - \sigma_{i, \ell}^2 I, 0 \} }\)
7. \(\qquad\) \(\qquad\) \(B \leftarrow \overline{\Sigma_i} V_i^T\)

Here \(\sigma_{i, \ell}\) is the \(\ell\)-th largest singular value of \(\Sigma_i\). The line 6 throws away all singular values that are less than \(\sigma_{i, \ell}\), and then subtract the squares of all singular values that are greater by \(\sigma_{i, \ell}^2\).

For the convenience of discussion, at each iteration, if \(B\) has less than \(\ell\) rows after insertion, we still define \[ U_i, \Sigma_i, V_i \leftarrow \text{SVD} (B) , \, \text{ and } \, \overline{\Sigma_i} = \Sigma_i. \]

To analyze the error, first observe that, when the algorithm stops,
\[ \begin{aligned} A^T A - B^T B &= \sum_{i \in [n]} \left( A_i^T A_i + B_{i - 1}^T B_{i - 1} - B_i^T B_i \right). \end{aligned} \] Therefore, for an arbitrary unit vector \(x \in \R^d\), \[ \begin{aligned} x^T A^T A x - x ^T B^T B x &= \sum_{i \in [n]} \left( x^T A_i^T A_i x + x^T B_{i - 1}^T B_{i - 1} x - x^T B_i^T B_i x \right) \\ &= \sum_{i = 1}^j \left( \left\langle A_i, x \right\rangle ^2 + \Vert B_{i - 1} x \Vert_2^2 - \Vert B_{i} x \Vert_2^2 \right) \\ &= \sum_{i = 1}^n \left( \Vert \Sigma_i V_i^T v \Vert_2^2 - \Vert \overline{\Sigma_i} V_i^T v \Vert_2^2 \right) \\ &\le \sum_{i = 1}^n \sigma_{i, \ell }^2. \end{aligned} \] The last inequality holds since \(V_i^T x\) is a unit vector, and \(\Sigma_i^T \Sigma_i - \overline{\Sigma_i}^T \overline{\Sigma_i}\) is a diagonal matrix whose elements are bounded by \(\sigma_{i, \ell}^2\).

It is left to bound the size of \(\sum_{i = 1}^n \sigma_{i, \ell }^2\): \[ \begin{aligned} \ell \cdot \sum_{i = 1}^n \sigma_{i, \ell }^2 &\le \sum_{i = 1}^n \left( \Vert \Sigma_i \Vert^2_F - \Vert \overline{\Sigma_i} \Vert^2 _F \right) \\ &= \sum_{i = 1}^n \left( \Vert C_i \Vert_F^2 - \Vert B_{i} \Vert^2_F \right) \\ &= \sum_{i = 1}^n \left( \Vert a_i \Vert_2^2 + \Vert B_{i - 1} \Vert^2_F - \Vert B_{i} \Vert^2_F \right) \\ &= \Vert A \Vert^2_F - \Vert B \Vert^2_F. \end{aligned} \]

Finally, let \(u_1, \ldots, u_d \in \mathbb{R}^d\) be the right singular vectors of \(A\). Then for each \(k \in [\ell]\), it holds that \[ \begin{aligned} \Vert A \Vert^2_F - \Vert B \Vert^2_F &= \sum_{i = 1}^k \Vert A u_i \Vert_2^2 + \sum_{i = k + 1}^d \Vert A u_i \Vert_2^2 - \Vert B \Vert^2_F \\ &= \sum_{i = 1}^k \Vert A u_i \Vert_2^2 + \Vert A - A_{[k]} \Vert_2^2 - \Vert B \Vert^2_F \\ &\le \sum_{i = 1}^k \big( \Vert A u_i \Vert_2^2 - \Vert B u_i \Vert_2^2 \big) + \Vert A - A_{[k]} \Vert_2^2 & \qquad \Vert B \Vert^2_F = \Vert B V\Vert^2_F \ge \sum_{i = 1}^k \Vert B u_i \Vert_2^2 \\ &\le \sum_{i = 1}^k \big( \Vert A u_i \Vert_2^2 - \Vert B u_i \Vert_2^2 \big) + \Vert A - A_{[k]} \Vert_2^2 \\ &\le k \cdot \Vert A^T A - B^T B \Vert_2 + \Vert A - A_{[k]} \Vert_2^2. \end{aligned} \]

In conclusion, we have \[ \ell \cdot \Vert A^T A - B^T B \Vert_2 \le k \cdot \Vert A^T A - B^T B \Vert_2 + \Vert A - A_{[k]} \Vert_2^2, \] which proves the theorem.

Remark: In the proof we apply the fact that the squared Frobenius norm of a matrix \(M\) equals the sum of the squares of its singular values. Let \(M = \sum_{j = 1}^d \sigma_j u_j v_j^\top\), then \[ \begin{align} \Vert M \Vert_{\mathrm{F}}^2 = \Vert U \Sigma V^T \Vert_{\mathrm{F}}^2 = \Vert \Sigma \Vert_{\mathrm{F}}^2. \end{align} \] The last inequality holds since \(U\) and \(V^T\) are rotations which does not change the Frobenius norm.

In conclusion, \[ \begin{aligned} \frac{ \Vert A^T A - B^T B \Vert_2 }{ \Vert A \Vert^2_F} \le \frac{ \Vert A^T A - B^T B \Vert_2 }{ \Vert A \Vert^2_F - \Vert B \Vert^2_F} \le \frac{2}{k}. \end{aligned} \]

\(\square\)

A Counterpart For Heavy Hitters

We apply similar idea for the heavy hitter problem in streaming model.

Problem

There is a streaming \(S = \{ a_1, ..., a_m \}\) of \(m\) elements each from a finite domain \([d]\). Given a parameter \(k\), we want to find the elements in \([d]\) whose frequencies, i.e., the number of times they appear in \(S\), are at least \(k\).

Algorithm

The following algorithm will provide under-estimates of elements' frequencies, with space usage \(2k\) and error \(m / k\).

Processing Stream
1. \(B_0 \leftarrow\) an empty (element, frequency) list of capacity \(2k\).
2. For \(t \in [m]\):
3. \(\qquad\) \(C_t \leftarrow\) list obtained by inserting \(\{ a_t, 1 \}\) into \(B_{t - 1}\).
4. \(\qquad\) If \(C_t\) has size \(2k\):
5. \(\qquad\) \(\qquad\) \(B_t \leftarrow\) list obtained from \(C_t\) by merging frequencies.
6. \(\qquad\) \(\qquad\) // each element in \([d]\) now appears in \(B_t\) at most once.
6. \(\qquad\) \(\qquad\) \(\sigma_t \leftarrow\) the \(k\)-th largest frequency in \(B_t\).
7. \(\qquad\) \(\qquad\) Subtract the frequency of each element in \(B_t\) by \(\sigma_t\).
8. \(\qquad\) \(\qquad\) Remove elements with non-positive frequencies.
8. \(\qquad\) Else
9. \(\qquad\) \(\qquad\) \(\sigma_t \leftarrow 0;\)
9. \(\qquad\) \(\qquad\) \(B_t \leftarrow C_t\).

QUERY
Input: \(B_t\) for some \(t \in [m]\), \(v \in [d].\)
Output: frequency estimation of \(v\) in \(\{a_1, ...., a_t \}\)
1. \(\hat f_{B_t}[v] \leftarrow 0;\)
2. for \((v_i, f_i) \in B_t\) do
3. \(\qquad\) if \(v_i = v\) then
4. \(\qquad\) \(\qquad\) \(\hat f_{B_t} [ v ] \leftarrow \hat f_{B_t} [v] + f_i;\)
5. Return \(\hat f_{B_t} [v]\).

Observe that for \(v \in [d]\), if \(v\) does not appears in \(B_t\), then the algorithm returns 0.

Time

The amortized time for each update is \(O(1)\).

  1. Inserting \(\{a_i, 1\}\) into \(B_{i - 1}\) takes \(O(1)\) time.
  2. Merging \(C_i\), subtracting elements' frequencies by \(\sigma_i\) and removing non-positive elements take \(O(k)\) time. But this happens only every \(k / 2\) updates. Hence the amortized time is \(O(1)\).

Error

Theorem 1. \(\forall t \in [m], v \in [d]\), \(\hat f_{B_t} [v]\) is an underestimate of \(v\)'s frequency in \(\{a_1, ... a_t \}\).

Proof. Obviously true by how the algorithm works.

Theorem 2. \(\forall v \in [d], t \in [m]\), let \(f_t [v]\) be the number of times \(v\) appears in \(\{a_1, ..., a_t \}\). We have that \[ f_t[v] - \hat f_{B_t} [v] < \frac{t}{k}. \]

Proof. For each \(i \in \{0, 1, ..., m \}\), define \(\Vert B_i \Vert_1 = \left( \sum_{ (v_j, f_j) \in B_i } f_j \right)\) and for each \(i \in [m]\), let \(\Vert C_i \Vert_1 = \left( \sum_{ (v_j, f_j) \in C_i } f_j \right)\). Observe that \[ \begin{aligned} f_t[v]- \hat f_{B_t} [v] \le \sum_{i \in [t] } \sigma_i. \end{aligned} \]

On the other hand, \[ \begin{aligned} t - \Vert B_m \Vert_1 &= \sum_{i \in [t] } \left( 1 + \Vert B_{i - 1} \Vert_1 - \Vert B_i \Vert_1 \right) \\ &= \sum_{i \in [t] } \left( \Vert C_i \Vert_1 - \Vert B_i \Vert_1 \right) \\ &\ge k \cdot \sum_{i \in [t] } \sigma_i. \end{aligned} \]

Combined, we get \(f_t[v]- \hat f_{B_t} [v] \le \frac{t - \Vert B_t \Vert_1}{k}\).

Reference

[1]. Edo, Liberty. "Simple and deterministic matrix sketching." In Proceedings of the 19th ACM SIGKDD international conference on Knowledge discovery and data mining, pp. 581-588. ACM, 2013.
