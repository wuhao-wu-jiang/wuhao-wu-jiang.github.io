Twin Drive or Not?

Posted on 2020-10-27 Edited on 2020-11-05

In this article, we introduce differential privacy. We start with a story of information leakage.

Twin Drive or Not

Celestial Being is a private military organization with superior technology. It has four advanced machines. The most powerful one is the Gundam 00.

The Gundam 00 is about to engage the enemy.

Due to maintenance, Gundam 00 is not always equipped with two engines (known as twin drive system). With probability 0.5, it uses one drive. We use T=(1,1)T = (1, 1)T=(1,1) to denote the status of equipping with the twin drive system, and S=(1,0)S = (1, 0)S=(1,0) (or (0,1)(0, 1)(0,1)) for status of single drive. Let XXX be a random variable that indicates drive status. Therefore,

Pr⁡[X=T]=0.5,Pr⁡[X=S]=0.5.\Pr[ X = T ] = 0.5, \\ \Pr[ X = S ] = 0.5. Pr[X=T]=0.5,Pr[X=S]=0.5.

Raiser sword is one of Gundam 00 most powerful weapon. The energy level of the Raiser sword, denote as YYY, is a random variable in [0,100][0, 100][0,100], whose distribution, denoted as Pr⁡[⋅∣X]\Pr[\cdot \mid X]Pr[⋅∣X], depends on the drive status. Thus, Pr⁡[Y=y∣X=T]\Pr[Y = y \mid X = T]Pr[Y=y∣X=T] (or Pr⁡[Y=y∣X=S]\Pr[Y = y \mid X = S]Pr[Y=y∣X=S]) is the probability that YYY equals to yyy conditioned on X=TX = TX=T (X=SX = SX=S). Specifically,

Pr⁡[⋅∣T]∼B(100,0.81)Pr⁡[⋅∣S]∼B(100,0.09)\Pr[\cdot \mid T] \sim B(100, 0.81) \\ \Pr[\cdot \mid S] \sim B(100, 0.09) Pr[⋅∣T]∼B(100,0.81)Pr[⋅∣S]∼B(100,0.09)

where B(n,p)B(n, p)B(n,p) denotes a binomial distribution. The expected energy level is 818181 with twin drive system, compared to only 999 with single drive. This is called “squaring” phenomenon of twin drive.

Now suppose that you’re the enemy pilot of a mobile suit with just average performance. Before you start dog fighting with Gundam 00, you will be attacked by the 00’s long-range raiser sword (you have not seen 00 yet). Luckily, you survive the raiser sword attack. Now you need to make a decision. If 00 is equipped with twin drive, there is zero chance that you can win the dog fight. The best choice is to leave the battle. Otherwise, you can outperform 00 and you would like to engage it.

Since you have just been attacked by the raiser sword, you have an observation of its energy level yyy. By Bayes’ theorem,

Pr⁡[X=T∣Y=y]=Pr⁡[Y=y∣X=T]⋅Pr⁡[X=T]Pr⁡[Y=y∣X=T]⋅Pr⁡[X=T]+Pr⁡[Y=y∣X=S]⋅Pr⁡[X=S]Pr⁡[X=S∣Y=y]=Pr⁡[Y=y∣X=S]⋅Pr⁡[X=S]Pr⁡[Y=y∣X=T]⋅Pr⁡[X=T]+Pr⁡[Y=y∣X=S]⋅Pr⁡[X=S]\begin{aligned} \Pr[ X = T \mid Y = y] = \frac{ \Pr[Y = y \mid X = T] \cdot \Pr[X = T] }{ \Pr[Y = y \mid X = T] \cdot \Pr[X = T] + \Pr[Y = y \mid X = S] \cdot \Pr[X = S] } \\ \\ \Pr[ X = S \mid Y = y] = \frac{ \Pr[Y = y \mid X = S] \cdot \Pr[X = S] }{ \Pr[Y = y \mid X = T] \cdot \Pr[X = T] + \Pr[Y = y \mid X = S] \cdot \Pr[X = S] } \end{aligned} Pr[X=T∣Y=y]=Pr[Y=y∣X=T]⋅Pr[X=T]+Pr[Y=y∣X=S]⋅Pr[X=S]Pr[Y=y∣X=T]⋅Pr[X=T]​Pr[X=S∣Y=y]=Pr[Y=y∣X=T]⋅Pr[X=T]+Pr[Y=y∣X=S]⋅Pr[X=S]Pr[Y=y∣X=S]⋅Pr[X=S]​​

Substituting with Pr⁡[X=S]=Pr⁡[X=T]=0.5\Pr[X = S] = \Pr[X = T] = 0.5Pr[X=S]=Pr[X=T]=0.5, we get

Pr⁡[X=T∣Y=y]=Pr⁡[Y=y∣X=T]Pr⁡[Y=y∣X=T]+Pr⁡[Y=y∣X=S]Pr⁡[X=S∣Y=y]=Pr⁡[Y=y∣X=S]Pr⁡[Y=y∣X=T]+Pr⁡[Y=y∣X=S]\begin{aligned} \Pr[ X = T \mid Y = y] = \frac{ \Pr[Y = y \mid X = T] }{ \Pr[Y = y \mid X = T] + \Pr[Y = y \mid X = S] } \\ \\ \Pr[ X = S \mid Y = y] = \frac{ \Pr[Y = y \mid X = S] }{ \Pr[Y = y \mid X = T] + \Pr[Y = y \mid X = S] } \end{aligned} Pr[X=T∣Y=y]=Pr[Y=y∣X=T]+Pr[Y=y∣X=S]Pr[Y=y∣X=T]​Pr[X=S∣Y=y]=Pr[Y=y∣X=T]+Pr[Y=y∣X=S]Pr[Y=y∣X=S]​​

The distributions B(100,0.09)B(100, 0.09)B(100,0.09) and B(100,0.81)B(100, 0.81)B(100,0.81) are plotted together below.

Immediately, you can draw some conclusions based on single value of yyy, such as

  1. If y=10y = 10y=10, it is likely 00 has only one drive.
  2. If y=75y = 75y=75, it is likely 00 has twin drive.

Or you can conclude based on the range of yyy, such as

  1. If y∈[0,20]y \in [0, 20]y∈[0,20], it is likely 00 has single drive.
  2. If y∈[70,90]y \in [70, 90]y∈[70,90], it is likely 00 has twin drive.

Before the observation of yyy, as Pr⁡[X=S]=Pr⁡[X=T]=0.5\Pr[X = S] = \Pr[X = T] = 0.5Pr[X=S]=Pr[X=T]=0.5, you have only random guess over the engine status of 00. After the observation, you might be much more confident about your guess, even though the energy level of the raiser sword is a random variable.

Why does this happen? Because the two distributions are well-separated. They are far from each other. Further, the prior probabilities Pr⁡[X=S]\Pr[X = S]Pr[X=S] and Pr⁡[X=T]\Pr[X = T]Pr[X=T] play important roles. If you know Pr⁡[X=S]=10−10\Pr[X = S] = 10^{-10}Pr[X=S]=10−10, even if you observe an energy level of y=10y = 10y=10, you had better not engage Gundam 00. You know it could be just a trap!

Formal Definition

Database managers face similar scenarios in privacy protection. We know define the setting for differential privacy. We view a dataset DDD as a table of nnn rows, each of which comes from a domain X\mathcal{X}X. Hence D∈XnD \in \mathcal{X}^nD∈Xn. Instead of releasing the dataset directly, the manager runs a randomized algorithm A:Xn→YA: \mathcal{X}^n \rightarrow \mathcal{Y}A:Xn→Y on DDD, and outputs Y=A(D)∈YY = A(D) \in \mathcal{Y}Y=A(D)∈Y. Here Y\mathcal{Y}Y is called the co-domain of AAA and does not necessarily equal to Xn\mathcal{X}^nXn. Note that the output distribution of AAA may depend on the input DDD. Further, for simplicity of discussion, we believe a uniform prior distribution over datasets in Xn\mathcal{X}^nXn.

In the Gundam 00’s story, X={0,1}2\mathcal{X} = \{0, 1\}^2X={0,1}2, and Y=[0,100]\mathcal{Y} = [0, 100]Y=[0,100].

Suppose there is another dataset D′D'D′ that differs only one row from DDD. The algorithm AAA is said to be differentially private if a malicious user is unlikely to distinguish the input to AAA between DDD and D′D'D′, by merely observing AAA’s output. Simply put, the conditional distributions of Pr⁡[⋅∣X=D]\Pr[\cdot \mid X = D]Pr[⋅∣X=D] and Pr⁡[⋅∣X=D′]\Pr[\cdot \mid X = D']Pr[⋅∣X=D′] should be similar, where XXX is a variable that denotes the input dataset.

We observe in the previous section that, if the distributions are well separated, then we can infer the underlying input with high confidence when the output value takes specific values or lies in certain ranges.

There are many ways to characterize the closeness of two distributions. We introduce the one proposed in [1].

Algorithm AAA is called (ϵ,δ)(\epsilon, \delta)(ϵ,δ) differentially private, if for any pair of neighboring datasets DDD and D′D'D′ (the ones that differ in only one row), and for any (measurable) subset R⊂Y\mathcal{R} \subset \mathcal{Y}R⊂Y, it holds that

Pr⁡[Y∈R∣X=D]≤exp⁡(ϵ)⋅Pr⁡[Y∈R∣X=D′]+δ \Pr[Y \in \mathcal{R} \mid X = D ] \le \exp(\epsilon) \cdot \Pr[Y \in \mathcal{R} \mid X = D' ]+ \delta Pr[Y∈R∣X=D]≤exp(ϵ)⋅Pr[Y∈R∣X=D′]+δ

In other words, we can’t find a subset R\mathcal{R}R, with which we can distinguish DDD and D′D'D′ with high confidence. In the previous example, such R\mathcal{R}R exists. E.g., R=[70,90]R = [70, 90]R=[70,90]. If YYY lies in R\mathcal{R}R, we can infer that Gundam 00 is likely to have twin drive.

Hypothesis Testing

There is alternative view of (ϵ,0)(\epsilon, 0)(ϵ,0) differential privacy as hypothesis testing. Suppose that we know the underlying dataset is either DDD or D′D'D′ with equal probability. After observing the output of AAA, we need to decide which hypothesis of the following holds:

H0: the dataset if DH1: the dataset if D′H_0: \text{ the dataset if } D \\ H_1: \text{ the dataset if } D' H0​: the dataset if DH1​: the dataset if D′

Assume that we adopt a fixed strategy:

  1. First we choose a fixed subset R⊂Y\mathcal{R} \subset \mathcal{Y}R⊂Y,
  2. If Y∈RY \in \mathcal{R}Y∈R, we choose to accept H0H_0H0​,
  3. Otherwise, we accept H1H_1H1​.

There are two kinds of errors we can make. The type I error is the one when the true hypothesis is H0H_0H0​ and we accept H1H_1H1​. Conversely, type II error is the one when the true hypothesis is H1H_1H1​ and we accept H0H_0H0​. Let ppp and qqq be the probabilities we make type I and II errors respectively. The following theorem holds

Algorithm A is (ϵ,0)(\epsilon, 0)(ϵ,0) differentially private then

  1. 11+exp⁡(ϵ)≤q≤11+exp⁡(−ϵ)\frac{1}{1 + \exp(\epsilon) } \le q \le \frac{1}{1 + \exp(-\epsilon) }1+exp(ϵ)1​≤q≤1+exp(−ϵ)1​
  2. 11+exp⁡(ϵ)≤p≤11+exp⁡(−ϵ)\frac{1}{1 + \exp(\epsilon) } \le p \le \frac{1}{1 + \exp(-\epsilon) }1+exp(ϵ)1​≤p≤1+exp(−ϵ)1​

Proof. By definition,

q=Pr⁡[X=D∣Y∈Rˉ]=Pr⁡[Y∈Rˉ∣X=D]Pr⁡[X=D]Pr⁡[Y∈Rˉ∣X=D]Pr⁡[X=D]+Pr⁡[Y∈Rˉ∣X=D′]Pr⁡[X=D′]=Pr⁡[Y∈Rˉ∣X=D]Pr⁡[Y∈Rˉ∣X=D]+Pr⁡[Y∈Rˉ∣X=D′]\begin{aligned} q &= \Pr[ X = D \mid Y \in \bar{\mathcal{R}} ] \\ &= \frac{ \Pr[ Y \in \bar{\mathcal{R}} \mid X = D ] \Pr[X = D ] }{ \Pr[ Y \in \bar{\mathcal{R}} \mid X = D ] \Pr[X = D ] + \Pr[ Y \in \bar{\mathcal{R}} \mid X = D' ] \Pr[X = D' ] } \\ &= \frac{ \Pr[ Y \in \bar{\mathcal{R}} \mid X = D ] }{ \Pr[ Y \in \bar{\mathcal{R}} \mid X = D ] + \Pr[ Y \in \bar{\mathcal{R}} \mid X = D' ] } \end{aligned} q​=Pr[X=D∣Y∈Rˉ]=Pr[Y∈Rˉ∣X=D]Pr[X=D]+Pr[Y∈Rˉ∣X=D′]Pr[X=D′]Pr[Y∈Rˉ∣X=D]Pr[X=D]​=Pr[Y∈Rˉ∣X=D]+Pr[Y∈Rˉ∣X=D′]Pr[Y∈Rˉ∣X=D]​​

As AAA is (ϵ,0)(\epsilon, 0)(ϵ,0) differentially private, it holds that

Pr⁡[Y∈Rˉ∣X=D′]≤exp⁡(ϵ)⋅Pr⁡[Y∈Rˉ∣X=D]\Pr[ Y \in \bar{\mathcal{R}} \mid X = D' ] \le \exp(\epsilon) \cdot \Pr[ Y \in \bar{\mathcal{R}} \mid X = D ] Pr[Y∈Rˉ∣X=D′]≤exp(ϵ)⋅Pr[Y∈Rˉ∣X=D]

and

Pr⁡[Y∈Rˉ∣X=D]≤exp⁡(ϵ)⋅Pr⁡[Y∈Rˉ∣X=D′]\Pr[ Y \in \bar{\mathcal{R}} \mid X = D ] \le \exp(\epsilon) \cdot \Pr[ Y \in \bar{\mathcal{R}} \mid X = D' ] Pr[Y∈Rˉ∣X=D]≤exp(ϵ)⋅Pr[Y∈Rˉ∣X=D′]

therefore,

11+exp⁡(ϵ)≤q≤11+exp⁡(−ϵ)\frac{1}{1 + \exp(\epsilon) } \le q \le \frac{1}{1 + \exp(-\epsilon) } 1+exp(ϵ)1​≤q≤1+exp(−ϵ)1​

By symmetry, we also have

11+exp⁡(ϵ)≤p≤11+exp⁡(−ϵ)\frac{1}{1 + \exp(\epsilon) } \le p \le \frac{1}{1 + \exp(-\epsilon) } 1+exp(ϵ)1​≤p≤1+exp(−ϵ)1​
