Multiplicative Weight Updates

Posted on 2019-04-05 Edited on 2022-03-17

Problem

Let X1,X2,...,XTX_1, X_2, ..., X_TX1​,X2​,...,XT​ be a sequence of TTT binary values.
We predict at each t∈[T]t \in [T]t∈[T], the value of XtX_tXt​.

At each time t∈[T]t \in [T]t∈[T], each expert i∈[n]i \in [n]i∈[n] generates a prediction pitp_i^tpit​ of XtX_tXt​.
Based on the experts’ predictions, we make our own pAtp_A^tpAt​.
The goal to minimize the number of wrong predictions/mistakes.

Measurement of Error

The first question is how good can we do?

Define

  1. Lit:L_i^t:Lit​: the indicator for the expert iii making a mistake at round ttt.
  2. Li=∑t∈[T]Lit:L_i = \sum_{t \in [T] } L_i^t:Li​=∑t∈[T]​Lit​: the total number of mistakes expert iii makes.
  3. LA:L_A:LA​: the mistakes by our algorithm (denoted as AAA).

Ideally, we would like achieve as few mistakes as

∑t∈[T]min⁡i∈[n]Lit. \sum_{t \in [T] } \min_{i \in [n]} L_i^t. t∈[T]∑​i∈[n]min​Lit​.

However, this is impossible.
To see this, let {Xt,t∈[T]}\{ X_t , t \in [T] \}{Xt​,t∈[T]} be independent Bernoulli random variables with equal probability taking
111 and 000.
Whatever our strategy is, the expected number of mistakes is T/2T / 2T/2.
On the other hand, it is possible that ∑t∈[T]min⁡i∈[n]Lit=0\sum_{t \in [T] } \min_{i \in [n]} L_i^t = 0∑t∈[T]​mini∈[n]​Lit​=0.

Despite this negative result, there is hope to achieve as few mistakes as the best expert (up to some constant), that is

LA∈O(min⁡i∈[n]∑t∈[T]Lit)=O(min⁡i∈[n]Li). L_A \in O \left( \min_{i \in [n]} \sum_{t \in [T] } L_i^t \right) = O \left( \min_{i \in [n] } L_i \right). LA​∈O⎝⎛​i∈[n]min​t∈[T]∑​Lit​⎠⎞​=O(i∈[n]min​Li​).

Algorithms

Follow the Majority

We start with an easy case where an expert who always predicts correctly.
In such case, we can maintain a set of experts who do not make any mistake so far.
Each time we make a decision, we follow the majority of the set.

Theorem. If there is a perfect expert, the algorithm Follow the Majority achieves

LA≤min⁡i∈[n]Li+log⁡n. L_A \le \min_{i \in [n] } L_i + \log n. LA​≤i∈[n]min​Li​+logn.

  1. Let E0←[n]E_0 \leftarrow [n]E0​←[n] be the set of all experts.

  3. \qquad Let SSS be the subset of Et−1E_{t - 1}Et−1​ that predicts 0 at time ttt.

The key observation is that the set we maintain halves each time we make a mistake.
Halving the set can happen at most log⁡n\log nlogn times, as there are only nnn experts.

Proof. Let MMM be the number of mistakes the algorithm makes. Each time it makes a mistake, the size of EtE_{t}Et​ shrink by half. Hence,

Theorem. If there isn’t a perfect expert, the algorithm Follow the Majority With Reset achieves

LA≤(min⁡i∈[n]Li+1)⋅log⁡n. L_A \le \left( \min_{i \in [n] } L_i + 1 \right) \cdot \log n. LA​≤(i∈[n]min​Li​+1)⋅logn.

Follow the Weighted Majority

The previous algorithm forgets the historical performance of the experts each time it resets EtE_tEt​, which might contain helpful information.
To fix this, we assign each expert iii a weight witw_i^twit​, follow the weighted majority at time ttt.

Theorem. The algorithm Follow the Weighted Majority achieves

LA≤2.41⋅(min⁡i∈[n]Li+log⁡n). L_A \le 2.41 \cdot \left( \min_{i \in [n]} L_i + \log n \right). LA​≤2.41⋅(i∈[n]min​Li​+logn).

Multiplicative Weight Updates

We are in good shape so far.
Let’s extend the problem to more general settings.
In this cases, the experts interact with the environment in round-robin fashion.
At each round ttt,

  1. The expert iii performs action, termed aita_i^tait​.
  2. The environment returns the loss of performing aita_i^tait​, denoted as ℓit∈[0,1]\ell_i^t \in [0, 1]ℓit​∈[0,1].

Note that there is no restriction of the experts’ actions and their losses (as long as they are within [0,1][0, 1][0,1]).
We will present an algorithm, called Multiplicative Weight Update, which is almost as good as the best expert.

Theorem. The algorithm Multiplicative Weight Update achieves

LA≤min⁡i∈[n]Li+2Tln⁡n, L_A \le \min_{i \in [n] } L_i + 2 \sqrt{ T \ln n }, LA​≤i∈[n]min​Li​+2Tlnn​,

where Li≐∑t∈[T]ℓitL_i \doteq \sum_{t \in [T] } \ell_i^tLi​≐∑t∈[T]​ℓit​ is the loss of expert iii.

This implies that the average expected loss converges at a rate of 1/T\sqrt{1 / T}1/T​ to the best one.

  1. Set wi1←1,∀i∈[n]w_i^1 \leftarrow 1, \forall i \in [n]wi1​←1,∀i∈[n].

  3. \qquad Let Wt=∑i∈[n]witW^t = \sum_{i \in [n] } w_i^tWt=∑i∈[n]​wit​.
  4. \qquad Follow action aita_i^tait​ with probability wit/Wtw_i^t / W^twit​/Wt.
  5. \qquad For each expert i∈[n]i \in [n]i∈[n]:
  6. \qquad\qquad wit+1←wit⋅(1−ϵ⋅ℓit)w_i^{t + 1} \leftarrow w_i^t \cdot (1 - \epsilon \cdot \ell_i^t)wit+1​←wit​⋅(1−ϵ⋅ℓit​)
