Unique Factorization

Posted on 2019-12-02 Edited on 2019-12-03

A prime number is an positive integer greater than 1 and whose divisor is 1 and itself. We claim that

∀n∈N+,n>1,\forall n \in N^+, n > 1, ∀n∈N+,n>1,

nnn has a unique prime factorization, i.e., we can write nnn as

n=p1p2p3...pkn = p_1 p_2 p_3 ... p_k n=p1​p2​p3​...pk​

where k≥1k \ge 1k≥1 and the pip_ipi​’s are primes and

p1≤p2≤p3≤...≤pkp_1 \le p_2 \le p_3 \le ... \le p_k p1​≤p2​≤p3​≤...≤pk​

The claim uses Euclid’s Lemma, which states:

  • Lemma: If ppp is a prime and p∣abp | abp∣ab, then p∣ap | ap∣a or p∣bp | bp∣b.
  • Proof: if ppp is prime, and p∤ap \nmid ap∤a, then exists a−1∈Fpa^{-1} \in F_pa−1∈Fp​, s.t, a−1a≡1mod  pa^{-1} a \equiv 1 \mod pa−1a≡1modp. It concludes that b≡a−1ab≡0mod  pb \equiv a^{-1}a b \equiv 0 \mod pb≡a−1ab≡0modp. Similar results holds when p∤bp \nmid bp∤b.
  • Corollary: if p∣q1q2q3...qlp | q_1 q_2 q_3 ... q_lp∣q1​q2​q3​...ql​, then p∣qip | q_ip∣qi​ for some 1≤i≤l1 \le i \le l1≤i≤l.

Now, suppose nnn has another prime decomposition

n=q1q2...qln = q_1 q_2 ... q_l n=q1​q2​...ql​

Then by assumption, q1∣qiq_1 | q_iq1​∣qi​ for some 1≤i≤l1 \le i \le l1≤i≤l. But both q1q_1q1​ and qiq_iqi​ are prime. Hence q1=qiq_1 = q_iq1​=qi​. By cancellation q1q_1q1​ and qiq_iqi​ and induct on the rest products, we get the desired result.

Corollary: For integers nnn and mmm, if we write them as the prime decompositions:

n=p1d1p2d2...pkdkm=q1f1q2f2...qlfln = p_1^{d_1} p_2^{d_2} ... p_k^{d_k} \\ m = q_1^{f_1} q_2^{f_2} ... q_l^{f_l} n=p1d1​​p2d2​​...pkdk​​m=q1f1​​q2f2​​...qlfl​​

such that pi≠pjp_i \neq p_jpi​=pj​ for 1≤i<j≤k1 \le i < j \le k1≤i<j≤k and qi≠qjq_i \neq q_jqi​=qj​ for 1≤i<j≤l1 \le i < j \le l1≤i<j≤l, and

n∣mn \mid m n∣m

Then for any 1≤i≤k1 \le i \le k1≤i≤k, ∃1≤j≤l\exists 1 \le j \le l∃1≤j≤l, such that

pi=qjdi≤fjp_i = q_j \\ d_i \le f_j pi​=qj​di​≤fj​

Proof: By Euclid’s Lemma, and proper cancelation, we have the desired result.

Approximate Membership Problem

In the approximate membership problem, we have subset S⊂US \subset US⊂U, that supports two operations:

  1. update(xxx): S←S∪{x}S \leftarrow S \cup \{x\}S←S∪{x}
  2. query(xxx): return True if x∈Sx\in Sx∈S else False.

where UUU is the domain that elements xxx belongs to. If U=232U = 2^{32}U=232, then each representation of xxx takes 323232 bits. If U=264U = 2^{64}U=264, then each xxx needs 646464 bits. If UUU is the website address, the length of its elements might is unlimited. Suppose that the average length is about 151515 chars, and each char takes 888 bits, then the representation could occupy most 120120120 bits.

We can solve the membership problem exactly by dictionaries, such as balanced-binary search tree and hash set. This requires space O(nlog⁡U)O(n \log U)O(nlogU) bits. If we allows false positive, we can break this bound.

The idea is to have a bit array AAA with length O(kn)O(kn)O(kn) bits and a pairwise hash family HHH that maps the elements in UUU uniformly at random to each bit in the array. After picking a hash function uniformly at random from HHH, the operation on SSS is performed as follows:

Posted on 2019-11-02 Edited on 2021-03-11

Problem

A stream SSS (viewed as a multi-set) of nnn elements each from a domain U=[1,,2,...,u]U = [1, , 2, ..., u]U=[1,,2,...,u]. Estimate the number of times each element appears in the SSS.

If we use a balanced binary search, this can be done in O(nlog⁡min⁡{n,u})O(n \log \min\{n, u\})O(nlogmin{n,u}) time and Θ(min⁡{n,u})\Theta(\min\{n, u\})Θ(min{n,u}) space. If we use a hash table, this can be done in Θ(min⁡{n,u})\Theta(\min\{n, u\})Θ(min{n,u}) time and space.

Question to ponder: can we do it with sub-linear space?

By allowing errors, this is possible. Let a=<a1,a2,...,au>a = \left< a_1, a_2, ..., a_{u} \right>a=⟨a1​,a2​,...,au​⟩ be the frequency of elements in UUU and b=<b1,...,bu>b = \left< b_1, ..., b_{u} \right>b=⟨b1​,...,bu​⟩ be our estimation. Our goal is to minimize the ℓ∞\ell_\inftyℓ∞​ error:
