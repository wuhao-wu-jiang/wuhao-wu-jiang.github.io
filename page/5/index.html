The Number of Binary Trees

Posted on 2021-09-03

Definition. A binary tree is either empty, or consists of a vertex called the root, plus an ordered pair of binary trees called the left-subtree and right-subtree.

For each \(n \in \mathbb{N}\), let \(b_n\) be the number of binary trees with \(n\) vertices.

Theorem. For all \(n \in \mathbb{N}\), we have \[ b_n = \frac{1}{1 + n} \cdot \binom{2 n}{n}. \]

Proof. When \(n = 0\), \(b_0 = 1 = \frac{1}{1 + 1} \binom{2}{1}\).

For all \(n \ge 1\), by enumerating the possible numbers of vertices in the left-subtrees and right-subtrees, we see that, \[ b_n = \sum_{ k = 0 }^{n - 1} b_k \cdot b_{n - 1 - k}. \]

Define the generating function of \((b_0, b_1, \ldots )\) as \[ b(x) \doteq b_0 + b_1 x + b_2 x^2 + \ldots \]

It holds that \(b_n\) is the coefficient of \(x^{n - 1}\) in the product \(b(x) \cdot b(x)\). Hence it is the coefficient of \(x^n\) in the product \(x \cdot b(x) \cdot b(x)\). We can write the following equality: \[ b(x) = b_0 + x \cdot (b (x) )^2 = 1 + x \cdot (b (x) )^2. \]

There are two possible closed form solution for \(b(x)\): \[ b(x) = \frac{ 1 \pm \sqrt{1 -4 x} }{ 2 x }. \]

As \(b(0)= b_0 = 1\), and \((1 + \sqrt{1 - 4 \cdot 0} ) / (2 \cdot 0) = \infty\), we must have \[ b(x) = \frac{ 1 - \sqrt{ 1 - 4 x} }{ 2 x }. \]

The Taylor's expansion \(\sqrt{1 - 4 \cdot x}\) at \(x = 0\) is given by \[ \begin{aligned} \sqrt{1 - 4 \cdot x} &= 1 + \binom{1 / 2}{1} \cdot (4x) + \binom{1 / 2}{2} \cdot (4x)^2 + ... \\ &= \sum_{k \in \mathbb{N}} \binom{1 / 2}{ k } (-4)^k x^k. \end{aligned} \] where \[ \binom{1 / 2}{ k } \doteq \frac{ (1 / 2) \cdot (1 / 2 - 1) \cdot (1 /2 - 2) \ldots \cdot (1 / 2 - k + 1) }{ k! }. \]

We conclude that \[ b(x) = \sum_{k \in \mathbb{N} } \left( -\frac{1}{2} \right) \cdot \binom{1 / 2}{k + 1} (-4)^{k + 1} x^k. \] From this, we can also verify that \((1 - \sqrt{ 1 - 4 x} ) / ( 2 x ) = 1\).

Finally, for \(k \ge 1\), \[ \begin{aligned} &\left( -\frac{1}{2} \right) \cdot \binom{1 / 2}{k + 1} \cdot ( -4 )^{k + 1} \\ =& \left( -\frac{1}{2} \right) \cdot \frac{ (1 / 2 - k) \cdot (1 / 2 - k + 1) \cdot ... \cdot (- 1 / 2) \cdot (1 / 2) }{ (k + 1)! } \cdot ( -4 )^{k + 1} \\ =& \frac{ (1 / 2 - k) \cdot (1 / 2 - k + 1) \cdot ... \cdot (- 1 / 2) }{ (k + 1)! } \cdot ( -4 )^{k} \\ =& \frac{ (2k - 1) \cdot (2k - 3) \cdot ... \cdot 1 }{ (k + 1)! } \cdot 2^k \\ =& \frac{ (2k)! }{ (k + 1)! \cdot k! \cdot 2^k } \cdot 2^k \\ =& \frac{1}{k + 1} \binom{2k}{k}. \end{aligned} \]

\(\square\)

Remark: We reach an interesting equality that \[ \Big( (2k - 1) \cdot (2k - 3) \cdot \ldots \cdot 1 \Big) \cdot 2^k = (2k) \cdot (2k - 1) \cdot \ldots \cdot (k + 1). \]

Reference

[1] J. Matoušek and J. Nešetřil, Invitation to discrete mathematics, 2nd ed. Oxford ; New York: Oxford University Press, 2009.

Nonmeasurable Sets

Posted on 2021-09-02 Edited on 2021-09-22

Let \(\mu\) and \(\mu^*\) denote the Lebesgue measure and Lebesgue outer measure for sets of \(\mathbb{R}\) respectively. We prove that there exists a set \(E \subset [0, 1]\) that fails to be measurable.

Preliminary

Suppose that \(E \subset \mathbb{R}\) is lebesgue measurable and that \(\mu(E) < \infty\).

Lemma 1. \(\forall \epsilon > 0, \exists\) an open set \(G \supset E\), s.t., \(\mu(G \setminus E) < \epsilon\).

Proof. Via the definition of the Lebesgue measure \(\mu(\cdot)\), there exists a sequence of open intervals \(\{ I_n \}\), such that $ _{n } (I_n) (E) + . $

Define \(G \doteq \cup_{n \ge 1} I_n\). Then \(G\) is an open set and \(G \supset E\). Via monotonicity and countably sub-additivity of Lebesgue measure, \[ \mu(E) \le \mu(G) \le \sum_{n \ge 1} \mu (I_n) . \]

As \(E\) is measurable and \(\mu(E) < \infty\), $ ( G E) = (G) - (E) . $

\(\square\)

Let \(G_\delta\) denote the collection of Borel sets that are countable intersection of open sets.

Lemma 2. \(E = A \setminus Z\), where \(A\) is a \(G_\delta\) set and \(\mu(Z) = 0\).

Proof. For each \(k \in \mathbb{N}_+\), there exists an open set \(G_k\), s.t.,

  1. \(G_k \supset E\) and,
  2. \(\mu(G_k \setminus E) < 1 / k\).

It follows that

  1. \(A \doteq \cap_{k \ge 1} G_k \supset E\) and,
  2. \(Z \doteq A \setminus E\) has measure zero.

\(\square\)

We are ready to prove that Lebesgue measure is translation invariant.

Lemma 3. \(\forall x \in \mathbb{R}\), the set \(E + x \doteq \{ y + x : y \in E \}\) is measurable and \(\mu(E) = \mu (E + x)\).

Proof. As Lebesgue outer measure is translation invariant, it holds that \[ \mu(E) = \mu^*( E ) = \mu^* ( E + x ). \]

It is left to prove that \(E + x\) is Lebesgue measurable.

We can rewrite \(E = A \setminus Z\), where \(A = \cap_{k \ge 1} G_k\) is a countable intersection of open sets and \(\mu(Z) = 0\). Since \(\{ G_k \}\) are open, \(\{ G_k + x \}\) are also open sets. Further, \(Z + x\) also has outer measure \(0\) and is therefore measurable.

It follows that \(E + x\) is measurable as \[ E + x = \Big( \cap_{k \ge 1} ( G_k + x ) \Big) \setminus \Big( Z + x \Big). \]

\(\square\)

Nonmeasurable Set

For each \(x, y \in [0, 1]\), denote \(x \sim y\) if \(|x - y| \in \mathbb{Q}\). Observe that \(\sim\) is an equivalent relation, i.e., it is reflexive, symmetric, and transitive. For each \(x \in [0, 1]\), define the equivalent class of \(x\) as \[ [x] \doteq \{ y \in [0, 1] : |y - x | \in \mathbb{Q} \}. \]

It is easy to see that the collection of all possible equivalent classes forms a partition of \([0, 1]\). Via Axiom of Choice, we can select a representative number from each equivalent class. Denote the resulting set as \(V\).

Theorem. \(V\) is not measurable.

Proof. We list all rational numbers in \([-1, 1]\) as \(r_1, r_2, \ldots\).

For each \(y \in [0, 1]\), \(\exists\) \(x \in V\), s.t. \(y - x\) is a rational number between \([-1, 1]\). Therefore, \[ [0, 1] \subset \cup_{n \ge 1 } \big( V + r_n \big) \subset [-1, 2]. \]

Further, as \(V\) contains exactly one representative number from each equivalent class, the sets \(\{ V + r_n : n \in \mathbb{N}_+ \}\) are disjoint.

Assuming that \(V\) is measurable, the lemmas in the previous section imply that \(V + r_n\) is measurable and \(\mu(V + r_n) = \mu(V)\), for each \(n \in \mathbb{N}_+\).

Then \(\cup_{n \ge 1 } \big( V + r_n \big)\) is also measurable. Via \(\sigma\)-additivity of \(\mu\), it holds that \[ \begin{aligned} 1 &= \mu \Big( [0, 1] \Big) \le \mu \left( \cup_{n \ge 1 } \big( V + r_n \big) \right) \\ &= \sum_{n \ge 1} \mu(V + r_n) = \sum_{n \ge 1} \mu(V) \le \mu \Big( [-1, 2] \Big) \le 3, \end{aligned} \]

which is impossible since \(\sum_{n \ge 1} \mu(V)\) is either infinity or zero.

\(\square\)

[1] 实变函数论，周民强.

The Weak and Strong Laws of Large Numbers For Rademacher

Posted on 2021-09-01

Let \(\pmb X = (X_1, X_2, X_3, \ldots)\) be a countable sequence of independent Rademacher random variables. For each \(n \in \mathbb{N}_+\), denote \(S_n = \sum_{i \in [n] } X_i\).

Weak Law of Large Numbers. \(\forall \epsilon > 0\), it holds that \[ \lim_{n } \Pr \left[ \left| \frac{1}{n} \cdot S_n \right| \ge \epsilon \right] = 0. \]

Proof. Via Hoeffding's inequality, we have \[ \Pr \left[ \left| \frac{1}{n} \cdot S_n \right| \ge \epsilon \right] \le \exp \left( - \frac{n \cdot \epsilon^2 }{ 2 } \right). \] Letting \(n \rightarrow \infty\), the right hand side converges to \(0\).

\(\square\)

Strong Law of Large Numbers. Let \[ A \doteq \left\{ \pmb x \in \{-1, 1\}^{ \mathbb{N} } : \lim_n \left( \frac{1}{n} \sum_{i \in [n] } x_i \right) = 0 \right\} \] be the set consisting of those \(\pmb x\) whose partial sums converges to 0. Then \[ \Pr[ \pmb X \in A] = 1. \]

Proof. For each \(n \ge 2\), let \(\epsilon_n = 2 \cdot \sqrt{(\ln n) / n}\), and define the set \[ B_n \doteq \left\{ \pmb x \in \{-1, 1\}^{ \mathbb{N} } : \left| \frac{1}{n} \sum_{i \in [n] } x_i \right| \ge \epsilon_n \right\}. \] Via Hoeffding's inequality, we see that \[ \Pr[ X \in B_n ] \le \frac{1}{n^2}. \]

Let \(C_k \doteq \cup_{n \ge k} B_k\). For each \(\pmb x \in \bar A\), \(\exists \epsilon > 0\), s.t., \[ \pmb x \in \left( \cap_{k \ge 1} \cup_{n \ge k} \left\{ \pmb x \in \{-1, 1\}^{ \mathbb{N} } : \left| \frac{1}{n} \sum_{i \in [n] } x_i \right| \ge \epsilon \right\} \right) \subset \Big( \cap_{ k \ge 2 } C_k \Big). \]

It follows that \(\bar A \subset \Big( \cap_{k \ge 2} C_k \Big)= \lim_k C_k\). Further, for each \(k \ge 2\), \[ \Pr[ X \in C_k ] \le \sum_{n \ge k} \Pr[ X \in B_n] \le \frac{1}{k}. \]

Hence, \[ \Pr[ X \in \lim_k C_k ] = \lim_k \Pr [ X \in C_k ] = 0. \]

\(\square\)
