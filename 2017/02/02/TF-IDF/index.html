TF-IDF

Posted on 2017-02-02 Edited on 2018-07-02

TF-IDF stands for term frequency and inverse document frequency. It evaluates the importance of a word to a document in a corpus, which increases with the number of times it appears in the document but is offset by its frequency in the corpus. More precisely, for TF, we define

tf(term| document) = \frac{\mbox{the number of times "term" appears in "document"} }{\mbox{total number of words in "document"} }

For IDF, we define

idf(term) = 1 + \log(\frac{\mbox{the total number of documents} }{\mbox{the number of documents this term appears in} } )

Now suppose we have a query. Let Q={t1,t2,...,tn}Q = \{t_1, t_2, ..., t_n \}Q={t1​,t2​,...,tn​} the set of different term in the query. We convert QQQ into a vector,

V(Q)=[tf(t1∣Q)∗idf(t1),tf(t2∣Q)∗idf(t2),...,tf(tn∣Q)∗idf(tn)]V(Q) = [tf(t_1 | Q)* idf(t_1), tf(t_2|Q) * idf(t_2),..., tf(t_n|Q) * idf(t_n)] V(Q)=[tf(t1​∣Q)∗idf(t1​),tf(t2​∣Q)∗idf(t2​),...,tf(tn​∣Q)∗idf(tn​)]
