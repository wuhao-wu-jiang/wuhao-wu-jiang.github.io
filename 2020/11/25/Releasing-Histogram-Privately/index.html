Releasing Histogram Privately

Posted on 2020-11-25

Given a set of nnn elements SSS that belongs to some finite domain X={x1,x2,...,xm}\mathcal{X}= \{ x_1, x_2, ..., x_m \}X={x1​,x2​,...,xm​}, we want to release the (normalized) histogram in a differentially private manner. Define ∀xi∈X\forall x_i \in \mathcal{X}∀xi​∈X:

  2. fS(xi)≐FS(xi)∣S∣=FS(xi)nf_S(x_i) \doteq \frac{F_S(x_i) }{ |S| } = \frac{F_S(x_i) }{ n }fS​(xi​)≐∣S∣FS​(xi​)​=nFS​(xi​)​, the frequency of xix_ixi​ in SSS.

The (normalized) histogram of hSh_ShS​ is a frequency vector:

hS≐<fS(x1),fS(x2),...,fS(xm)>∈Rmh_S \doteq \left< f_S(x_1), f_S(x_2), ..., f_S(x_m) \right> \in \mathbb{R}^m hS​≐⟨fS​(x1​),fS​(x2​),...,fS​(xm​)⟩∈Rm

Consider another set S′S'S′ obtained by replacing an element xkx_kxk​ in SSS to another element xlx_lxl​, such that

  1. FS′(xk)=FS(xk)−1F_{S'} (x_k) = F_{S} (x_k) - 1FS′​(xk​)=FS​(xk​)−1;
  2. FS′(xl)=FS(xl)+1F_{S'} (x_l) = F_{S} (x_l) + 1FS′​(xl​)=FS​(xl​)+1;
  3. FS′(xi)=FS(xi),∀i∉{k,l}F_{S'} (x_i) = F_{S} (x_i), \forall i \notin \{k, l\}FS′​(xi​)=FS​(xi​),∀i∈/{k,l}.

Let hS′h_{S'}hS′​ be the histogram of S′S'S′. Our goal is to design some mechanism MMM that adds noise to the normalized so that the output distribution of M(hS)M(h_S)M(hS​) and M(hS′)M(h_{S'})M(hS′​) are indistinguishable.

Naive Laplacian

One way is to add Laplacian noise directly to each dimension of the histogram. For any vector v∈Rmv \in \mathbb{R}^mv∈Rm, MMM outputs a new perturbed vector:

M(v)≐v+LM(v) \doteq v + L M(v)≐v+L

where L≐<l1,l2,...,lm>L \doteq \left< l_1, l_2, ..., l_m\right>L≐⟨l1​,l2​,...,lm​⟩ and each lil_ili​ is sampled independently from Lap(2nϵ)Lap( \frac{2 }{ n \epsilon } )Lap(nϵ2​).

We claim this mechanism is (ϵ,δ)(\epsilon, \delta)(ϵ,δ)-differentially private.

  • Case 1. If SSS and S′S'S′ have the same set of supports, i.e., fS(xi)>0↔fS′(xi)>0f_S(x_i) > 0 \leftrightarrow f_{S'}(x_i) > 0fS​(xi​)>0↔fS′​(xi​)>0, then MMM is (ϵ,0)(\epsilon, 0)(ϵ,0) differentially private.
