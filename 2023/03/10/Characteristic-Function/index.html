Proof of the Theorem.
For each $n \in \mathbb{N}+$, define $E_n \doteq { x \in \mathbb{R} : \mu( { x } ) \ge 1 / n }$.
It is easy to see that $|E_n| \le n$.
It follows that the $E \doteq \cup{n \in \mathbb{N}_+} E_n$ is countable.

Consider the set $S = { (a, b] : a < b, \mu({a}) = \mu({b}) = 0 }$.
Obviously this is a $\pi$ system.
Further, it generates $\mathcal{B}$.
To see this, for each point $c \in E$, we can find

  1. a sequence $a_1, a_2, \ldots$ from $\bar{E}$ that increases to $c$,
  2. a sequence $b_1, b_2, \ldots$ from $\bar{E}$ that decreases to $c$.

Then $(ai, b_i] \in S$, and $\cap{i \ge 1} (a_i, b_i] = [c, c]$.

It follows that every interval $(a, b] \subseteq \mathbb{R}$ can be generated by intervals in $S$.

According to the lemma, $\mu$ and $\nu$ agrees on a $\pi$ system that generates $\mathcal{B}$.
