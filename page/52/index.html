Function Convergence

Posted on 2019-03-01 Edited on 2022-01-16

In this article we discuss several important convergence properties of a sequence functions, \(\{ f_n \}\), defined on the measure space \((X, a, \mu)\).

Definition (Pointwise Convergence) \(\{ f_n \}​\) converges to \(f​\) almost every (a.e.) if \(\{ f_n \}\) converges to \(f\) except on a set of measure zero, i.e, \(\exists E \subset X​\), s.t., \[ \mu(X - E) = 0 \, \text{ and }\, \forall x \in E, \lim_n f_n(x) = f(x)​. \]

Definition (Uniform Convergence) \(\{ f_n \}​\) converges to \(f​\) almost uniformly if \(\forall \epsilon > 0​\). \(\exists F \subset X​\), s.t, \(\mu(X - F) \le \epsilon​\) and \(\{ f_n \}​\) converges uniformly to \(f​\) on \(F​\).

The following two theorems discuss the relation between pointwise and almost uniform convergence.

Theorem 1.[Almost Uniform \(\rightarrow\) Pointwise] If \(\{ f_n \}\) converges to \(f\) almost uniformly, then \(\{ f_n \}\) converge to \(f\) almost everywhere.

Proof: By definition, \(\forall n \in \mathbb{N}^+, \exists F_n \subset X, s.t., \mu(X - F_n) \le 1 / n\) and \(\{ f_n \}\) converges uniformly to \(f\) on \(F_n\). But this immediately implies that \(\{ f_n \}\) converges pointwise on \(F_n\) and therefore it converges pointwise on \(\cup_{n = 1}^\infty F_n\).

We claim that \(\mu \Big( X - \big( \cup_{n = 1}^\infty F_n \big) \Big) = 0\), and hence \(\{ f_n \}\) converges to \(f\) almost everywhere. The claim holds as for each \(n \in \mathbb{N}^+\), \[ \mu(X - \cup_{n = 1}^\infty F_n) = \mu( \cap_{n = 1}^\infty (X - F_n)) \le \mu(X - F_n) = 1/n. \]

Taking \(n \rightarrow \infty\) finishes the proof.

\(\square\)

Theorem 2. [Pointwise \(\rightarrow\) Almost Uniform][Egorov] Let \(\{f_n\}\) be a sequence of measurable functions that converges pointwise to \(f\) almost everywhere and suppose \(\mu(X) \le \infty\), then \(\{ f_n \}\) converges to \(f\) almost uniformly.

Proof: Without lost of generality, we assume that \(\{ f_n \}\) converge to \(f\) everywhere on \(X\).

For each \(\epsilon > 0\), we need to find a set \(F\) such that

  1. \(\{ f_n \} \rightarrow f\) uniformly on \(F\) and,
  2. \(m(X - F) \le \epsilon\).

For each pair of \(k, m \in \mathbb{N}\), consider the set \(F_{k, m} = \cap_{n = m}^\infty \{ x : |f_n(x) - f(x)| \le 1 / k \}\). Now fix \(k\) and note that \(F_{k, m} \subset F_{k, m+1}\).

Next, \(\forall x \in E\), \(\lim_n f_n(x) = f(x)\). For fix \(x\) and \(k\), \(\exists m > 0\), s.t., for all \(n > m, |f_n(x) - f(x)| \le 1/k\). Hence \(x \in F_{k, m}\).

We conclude that \(F_{k, m}\) is a sequence of monotone increasing sets and \(\lim_{m} F_{k,m} = X\). Thus we have \(\lim_{m} F_{k,m}) = \mu(X) < \infty\). For arbitrary \(\epsilon > 0\), we can find an integer \(N_k\), s.t., \(\mu(X - F_{k, N_k}) \le \epsilon / 2^k\). Now for convenience we abbreviate \(F_{k, N_k}\) as \(F_k\).

Observe that \(\{ f_n \}\) converges uniformly on \(F = \cap_{k = 1}^\infty F_k\), as for any \(\lambda > 0, \exists k \in \mathbb{N}, s.t., 1 / k < \lambda\), and \(x \in F\) implies \(x \in F_k\). Therefore if \(n > N_K\), \(|f_n(x) - f(x)| \le 1/k \le \lambda​\).

It remains to prove \(\mu(X - F)\) is small enough. This is because of \[ \begin{aligned} \mu(E - F) &= \mu(E - \cap_{k = 1}^\infty F_k) \\ &= \mu(\cup_{k = 1}^\infty (E - F_k)) \\ &\le \sum_{k = 1}^\infty \mu(E - F_k) \\ &\le \sum_{k = 1}^\infty \epsilon / 2^k \\ &= \epsilon \end{aligned} \]

\(\square\)

There are another two frequently used definitions for convergence.

Definition (Converges in measure) \(\{ f_n \}\) converges to \(f\) in measure if \(\forall \epsilon > 0\), \(\forall \delta > 0\), \(\exists N \in \mathbb{N}\), s.t, for \(n > N\), \(\mu( \{ x : |f_n (x) - f(x)| \ge \epsilon \} ) < \delta, i.e,\),
\[ \lim_{n} \mu( \{ x : |f_n(x) - f(x)| \ge \epsilon \}) = 0. \]

Definition (Cauchy in measure) \(\{ f_n \}\) is Cauchy in measure if \(\forall \epsilon > 0\), \(\forall \delta > 0\), \(\exists N \in \mathbb{N}\), s.t, for \(n, m > N\), \(\mu( \{ x : |f_n (x) - f_m(x)| \ge \epsilon \} ) < \delta, i.e,\),
\[ \lim_{n, m} \mu( \{ x : |f_n(x) - f_m(x)| \ge \epsilon \}) = 0. \]

It is easy to see that if \(\{ f_n \}\) converge in measure, then it is Cauchy in measure. Later we will show that the converse is also true.

Remark: Converging in measure does not imply converge almost every where. To see this, consider a countable sequence of functions defined on \([0, 1]\).

  1. \(f_{1, 1} = \mathbf{1}_{[0, 1/2]}​\), \(f_{1, 2} = \mathbf{1}_{[1/2, 1]}​\)
  2. \(f_{2, 1} = \mathbf{1}_{[0, 1/4]}\), \(f_{2, 2} = \mathbf{1}_{[1/4, 1/2]}\), \(f_{2, 3} = \mathbf{1}_{[1/2, 3/4]}\), \(f_{2, 4} = \mathbf{1}_{[3/4, 1]}\)
  3. ....
  4. \(f_{n, 1} = \mathbf{1}_{ [0, 1/2^n] }\), ..., \(f_{n, i} = \mathbf{1}_{ [(i - 1) / 2^{n}, i /2^n] }\), ..., \(f_{n, n} = \mathbf{1}_{ [1 - 1/2^{n}, 1] }\)
  5. ...

We can list the functions as \(\{ f_{1, 1}, f_{1, 2}, f_{2, 1}, ...\}\) and relabel them concisely as \(\{g_1, g_2, g_3, ... \}\). Apparently, \(\{ g_n \}\) converges to \(g(x) \equiv 0\) in measure. But for any \(x \in [0, 1]\), there are infinite number of functions from the sequence \(\{ g_n \}\) such that \(g_n(x) = 1\) and so \(\{ g_n \} \nrightarrow g, a.e\).

However, we have the following theorem.

Theorem 3 If \(\{ f_n \}\) is Cauchy is Cauchy in measure, then it has a sub-sequence that converges to a measurable function \(f\) almost uniformly.

Key Observation: if we look at the previous example, \(f_{n, i}\) only deviates from \(g(x) \equiv 0\) by a small interval \([( i - 1) / 2^n, i / 2^n]\) with length \(1 / 2^n\). However, there are \(2^n\) functions with such deviation and the intervals of deviation do not overlap. This motivates filtering out a sub-sequence.

Proof: Since \(\{ f_n \}\) is Cauchy, \(\forall k > 0\), we can find an integer \(n_k > 0, s.t., \forall n, m \ge n_k, \mu( \{ x: |f_n(x) - f_m(x)| \ge 1 / 2^k \} ) \le 1/ 2^k\). We choose such a increasing sequence \((n_k)\) satisfying \[ n_1 < n_2 < n_3 < ... n_k < ... \] and the corresponding sub-sequence \(\{ f_{n_k} \}\) of \(\{ f_n \}\).

Let \[ \begin{aligned} F_k &= \{x : |f_n(x) - f_m(x)| \le 1 / 2^k, \forall n, m \ge n_k \} \\ E_n &= \cap_{k = n}^\infty F_k \end{aligned} \]

Then \(\{ f_{n_k} \}\) is uniformly Cauchy in \(E_n\) and therefore converge uniformly. It follows that there is a measurable function defined on

\[ F = \cup_{n = 1}^\infty E_n \]

such that \(\{ f_{n_k} \}\) converges uniformly on each \(E_n\). We can extend the domain of \(f\) to \(X\) by defining \(f(x) = 0\) for \(x \in X - F\). Note that \(f\) is measurable on \(X\).

Finally, \[ \mu(X - E_n) = \mu(X - \cap_{k = n}^\infty F_k) \le \sum_{k = n}^\infty \mu(x - F_k) \le 1 / 2^{n - 1}. \] It follows that \(\mu(F) = \mu(X)\) and \(( f_{n_k} )\) converge almost uniformly on \(F\).

\(\square\)

Corollary If \(\{ f_n \}\) is Cauchy is Cauchy in measure, then it converges to a measurable function \(f\) in measure.

Compact Set

Posted on 2019-02-28

Definition. Compact Set

A set \(S \subset R\) is compact if every sequence in \(S\) has a convergent sub-sequence whose limit is in \(S\).

Theorem. \(S\) is compact if and only if it is closed and bounded.

Proof:
1. \("\Rightarrow"\): Suppose \(S\) is not bounded. Then there is a sequence \((a_n)\) is \(S\), such that \(|a_n| \ge n\), \(\forall n \in N\). Now every sub-sequence in \((a_n)\) diverges. Therefore \(S\) must be bounded. Next we show that \(S\) contains all its limit points. Let \(a\) be a limit point of \(S\). Then there is a sequence \((a_n) \rightarrow a\). \((a_n)\) is a sub-sequence of itself and so \(a \in S\). We have proved that \(S\) is closed. 2. \("\Leftarrow"\): Assume \(S\) is closed and bounded. Consider a sequence \((a_n)\) in \(S\). By Weierstrass Theorem, there is a convergent sub-sequence \(( a_{n_k} )\). But since \(S\) is closed, \(\lim_k a_{n_k} \in S\).

Theorem. \(S\) is compact if and only if every open cover of \(S\) has a finite sub-cover.

Proof:
1. \("\Rightarrow"\): If \(S\) is compact, it is bounded. We can find a closed interval \(I_0\) that contains \(S\). Suppose \(S\) can not be covered by a finite sub-cover. We bisect \(I_0\) into two closed intervals of equal length. At least one of them can not be covered with finite sets. Denote this interval \(I_1\). We continue the process, resulting in a series of intervals \(I_1 \supset I_2 \supset I_3 \supset ...\). Define \(J_n = I_n \cap S \quad \forall n \in N\). The construction of \(I_n\) ensures \(J_n \neq \emptyset \quad \forall n \in N\) and the length of \(I_n\) approaches \(0\) as \(n \rightarrow \infty\). We claim that \(\cap_n J_n \neq \emptyset\). Indeed, there is an element \(a_n \in J_n\) for \(n \in N\). It not difficult to see \((a_n)\) is a Cauchy sequence since \(\forall m > n \ge N\), \(a_n, a_m \in I_N\) and \(|a_n - a_m| \le |I_N|\). Let \(a = \lim_n a_n\). We conclude that \(a \in J_n\) for \(n \in N\). This is because \(a_{n + 1}, a_{n + 2}..., \in J_n\) for all \(n \in N\). It follows that \(a \in \cap_n J_n\). Now \(a\) must be covered by some open set \(O_a\). On the other hand, the construction of \(I_n\) ensures that there is an integer \(N_a\) such that \(I_{N_a}\) is small enough such that \(I_{N_a} \subset O_a\). Contradicting the assumption that \(I_{N_a}\) cannot be covered with finite sets.

  1. \("\Leftarrow"\): Consider the collection of open balls \(\{ B(O, 1),\ B(O, 2),\ B(O, 3), ...\}\) centered at \(0\) with radii \(1, 2, 3, ...\). It covers \(S\) and has a finite sub-cover. It follows that \(S\) is bounded by the ball with largest radius in the finite sub-cover. To show \(S\) is closed, it suffices to prove it complement \(S^c\) is open. Let \(a \in S^c\). Take the collection of closed balls \(\{ \bar B(a, 1),\ \bar B(a, 1/2),\ \bar B(a, 1/3), ...\}\) centered at \(a\) with radii \(1, 1/2, 1/3, ..., 1/n, ...\). Their complements \(\{ \bar B(a, 1)^c,\ \bar B(a, 1/2)^c,\ \bar B(a, 1/3)^c, ...\}\) constitute an open cover of \(S\). By assumption we can find a finite subset of them that covers \(S\). Let \(\bar B(a, \epsilon)^c\) be the one with the smallest radius in the finite subset collection. We learn that \(B(a, \epsilon) \subset S^c\) and therefore \(S^c\) is open.

Reading Notes-Some basic theorems for real numbers

Posted on 2019-02-27 Edited on 2019-02-28

Axiom of completeness

Any nonempty set of real numbers that is bounded above has an least upper bound.

Monotone Convergence Theorem

Let \((a_n)\) be an increasing sequence that is bounded above, then it converges.

Proof:

Consider the set of point \(\{ a_n : n \in N \}\). By assumption, it is bounded. By Axiom of Completeness, the set \(\{ a_n \}\) has an least upper bound, denoted as \(a = \sup \{ a_n : n \in N \}\). Because \(a\) is a least upper bound, \(\forall \epsilon > 0\), there is an element \(a_N\), such that \(a_N \ge a - \epsilon\). But \((a_n)\) is an increasing sequence. Therefore, for any \(n \ge N\), we have \(a \ge a_n \ge a_N \ge a - \epsilon\), which implies that \((a_n) \rightarrow a\).

Corollary:

Let \((b_n)​\) be a decreasing sequence that is bounded below, then it converges.

Nested Interval Property

Let \(I_1 \supset I_2 \supset I_3 ...\) be a collection of closed intervals. Then \(\cap_{n} I_n \neq \emptyset\).

Proof 1 (By Axiom of Completeness):

Denote \(I_n = [a_n, b_n]= \{ x \in R : a_n \le x \le b_n \}\). By definition, we have \[ a_1 \le a_2 \le a_3 \le ... a_n \le ... \le b_n \le ... \le b_3 \le b_2 \le b_1 \] We are going to use Axiom of Completeness to produce a single real number \(a \in \cap_n I_n\). The set we consider is
\[ A \doteq \{ a_1, a_2, ... \} \] It is bounded above by any \(b_n​\) for \(n \ge 1​\). Therefore we are justified in the setting \(a = \sup A​\). Because \(a​\) is an upper bound, we have \(a_n \le a​\). The fact \(b_n​\) is an upper bound and that \(a​\) is the least upper bound implies \(a \le b_n​\). Putting together, \(a \in I_n​\) for \(n \ge 1​\). Hence, \(a \in \cap_n I_n​\).

Proof 2 (By Monotone Convergence Theorem):

\((a_n)\) is a monotone increasing sequence and bounded above. By Monotone Convergence Theorem, there exists a real number \(a = \lim_n a_n\). Moreover, it holds that \(b_m \ge a_n\) for any integer \(m\) and \(n\), which implies that \(b_m \ge \lim_n a_n = a\), as desired.

Remark:

We may not necessarily choose Axiom of Completeness as our starting point. For example, if we assume that Nested Interval Property is true and \(\lim 1/2^n \rightarrow 0\), it can be used to prove Axiom of Completeness.

Proof: Let \(A\) be an nonempty set that is bounded above, let \(b_1\) be an upper bound and \(a_1\) be an element in \(A\). Necessarily, \(a_1 < b_1\) and we can define the closed interval \(I_1 = [a_1, b_1]\).

The idea is to construct a sequence of closed intervals and use Nested Interval Property. Now we bisect \(I_1\) into two closed intervals of equal length let \(c_1\) be the midpoint. If \(c_1\) is an upper bound for \(A\), we define \(I_2 = [a_1, c_1]\). Otherwise, let \(I_2 = [c_1, b_1]\). In general, we construct \(I_n = [a_n, b_n]\) by taking the half from \(I_{n - 1}\) such that, \(a_n\) is no greater than any upper bound of \(A\) and \(b_n\) is no less than any element of \(A\).

By Nested Interval Property, there exists \(a \in \cap_n I_n\). Also we notice that the length of \(I_n \rightarrow 0\), which implies \(\lim a_n = \lim b_n = s\).

Because for all \(n \in N\), and any element \(a \in A\), we have \(a \le b_n\). Taking limit of the right hand side, we obtain \(a \le s\), which shows \(s\) is an upper bound for \(A\).

Moreover, for any upper bound \(l\) for \(A\), we have \(a_n \le l\). Again taking limit of the left hand side, we conclude \(s \le l\), which shows that \(s\) is the least upper bound.

Bolzano-Weierstrass Theorem

Let \((a_n)\) be a bounded sequence, then it has a convergent sub-sequence.

Proof:
Since \((a_n)\) is bounded there exists \(M > 0\) such that \(|a_n| \le M\) for all \(n \in N\) . Define \(I_0 = [-M, M]\). Then at least one of the intervals \([-M, 0]\) and \([0, M]\) contains infinite number of terms from the sequence \((a_n)\). Select such half interval and label it \(I_1\). Let \(a_{n_1}\) be some term from \((a_n)\) belonging to \(I_1\).

Next, we continue to bisect \(I_1\) into two closed intervals of equal length. Choose the half that contains infinite number of terms from \((a_n)\) and label it as \(I_2\). We can select an element \(a_{n_2} \in I_2\) from \((a_n)\) with \(n_2 \ge n_1\). In general, we construct \(I_k\) by taking the half from \(I_{k - 1}\) that contains infinite number of terms of \((a_n)\) and then choose \(a_{n_k} \in I_k\) so that \(n_k > n_{k - 1} > ... > n_2 > n_1\). The sets \(\{ I_n : n \in N \}\) forms a sequence of nested closed intervals \[ I_1 \supset I_2 \supset I_3 ... \] By Nested Interval Property, there is at least on point \(a \in \cap_n I_n\). On the other hand, the lengths of the intervals shrink to \(0\), which implies that \(a_{n_k} \rightarrow a\), as desired.

Cauchy Criterion

Let \((a_n)​\) be a sequence which satisfies \[ \forall \epsilon > 0, \exists N \in \mathbf{N}, s.t., \forall n \ge m \ge N \rightarrow |a_n - a_m| \le \epsilon \] Then \((a_n)​\) converges.

Proof:

Step 1: \((a_n)\) is bounded. To see this, let \(N_1\) be the integer that for \(n \ge N\), we have \(|a_n - a_{N_1} | \le 1\) and so \(|a_n| \le |a_{N_1} | + 1\). It can be verified \(\max \{ |a_1|, |a_2|, ..., |a_{N_1 - 1} |, |a_{N_1} | + 1 \}\) is an upper bound for the sequence \(( a_n )​\).

Step 2: Now, by Bolzano-Weierstrass Theorem, \((a_n)\) has an convergent sub-sequence \(( a_{n_k} ) \rightarrow a\). We show that \((a_n)\) converges to the same limit. Let \(\epsilon > 0\), because \((a_n)\) is Cauchy, \[ \exists N > 0,s.t., \forall n \ge m \ge N \rightarrow |a_n - a_m| \le \epsilon / 2 \] We also know \((a_{n_k}) \rightarrow a\), so choose a term \(a_{n_k}\), with \(n_k > N\) and \[ |a_{n_k} - a| \le \epsilon / 2 \] Putting together, \[ \forall n > N, |a_n - a| \le |a_n - a_{n_k} | + |a_{n_k} - a| \le \epsilon \] as desired.

Remark:

Cauchy Criterion can be used to prove Bolzano-Weierstrass Theorem. The key is to construct a sequence of nested closed intervals \(I_1 \supset I_2 \supset I_3 ...\) as before and obtain a sub-sequence \((a_{n_k})\). Then, instead of using Nested Interval Property, we observe that \((a_{n_k})\) is Cauchy as the length of \(I_k\) shrinks to zero. We know \((a_{n_k})\) converges by Cauchy Criterion.

Reference: Stephen Abbott, Understanding Analysis
