L0 Sampling

Posted on 2021-04-16 Edited on 2021-05-15

Problem

Let \(x \in \mathbb{R}^n\) be a vector initialized with all zeros. There is a stream \(S\) of updates, each of which is a pair \((t, \Delta) \in [n] \times \mathbb{R}\), causing \(x_t \leftarrow x_t + \Delta\).

Define the support \(\mathbb{Supp}(x)\) to be the set of all non-zero indexes of \(x\): \[ \mathbb{Supp}(x) \doteq \{ i \in [n] : x_i \neq 0 \}. \]

\(\ell_0\) sampling: output an index uniformly at random from the support \(\mathbb{Supp}(x)\).

We assume that at any time, \(\forall t \in [n], x_t \le M\) for some \(M \in \mathbb{R}\).

Naive Solutions

We can uses an array to keep \(x\) explicitly. But this has size \(O( n \cdot \log M)\) bits.

Goal: design an algorithm for \(\ell_0\) sampling with space less than \(O( n \cdot \log M)\) bits.

If there are only insertion updates, i.e., \(\forall (t, \Delta) \in S\), it holds that \(\Delta > 0\), then we can use a hash set to keep the non-zero elements and their frequencies, which has size \(O( |\mathbb{Supp}(x) | \cdot \log M)\). If \(x\) is dense, i.e., \(|\mathbb{Supp}(x)|\) is large, this is still infeasible.

Sampling Game

We discuss one sampling game that illustrates the key ideas used in \(\ell_0\)-sampling.

In this game, we want to sample one integer uniformly at random from \([k]\). Instead of sampling it directly, we choose some \(p \in (0, 1]\) and perform the following two steps:

  1. First, we sample each \(i \in [k]\) uniformly and independent at random with probability \(p\). Denote \(B\) the set of sampled elements.
  2. If \(B \neq \emptyset\), then output one element from \(B\) uniformly at random.

Property. \(\mathbb{E} [ |B| ] = kp\).

Property. \(\Pr[ |B| > 0 ] = (1 - p)^k \le \exp(-kp)\).

Property. By Bernstein's Inequality, \(\forall \delta \in (0, 1)\), with probability at least \(1 - \delta\), we have \[ |B| - \mathbb{E} [ |B| ] \le \sqrt{2 \cdot k \cdot p \cdot (1 - p) \cdot \ln \frac{1}{\delta} } + \frac{2 \ln \frac{1}{\delta} }{3}. \]

Theorem. Each element in \([k]\) is outputted with equal probability \[ \frac{1 - \left(1 - p \right)^k }{ k }. \]

Remark: when \(p = 1\), this reduces to sampling an element uniformly at random from \([k]\).

Proof. The probability that \(B = \emptyset\) is given by \((1 - p)^k\). Conditioned on that \(B \neq \emptyset\), by symmetry, each element is sampled with equal probability.

\(\square\)

Remark: We can also have a proof by algebra. For evert \(i \in [k]\), the probability that it is sampled is given by \[ \begin{aligned} \sum_{t = 1}^k \frac{1}{t} \cdot \Pr [ i \in B \mid |B| = t] \cdot \Pr[ |B| = t] &= \sum_{t = 1}^k \frac{1}{t} \frac{ \binom{k - 1}{t - 1} }{ \binom{k}{t} } \binom{k}{t} p^t (1 - p)^{k - t} \\ &= \sum_{t = 1}^k \frac{1}{t} \binom{k - 1}{t - 1} p^t (1 - p)^{k - t} \\ &= \frac{1}{k} \sum_{t = 1}^k \binom{k}{t} p^t (1 - p)^{k - t} \\ &= \frac{1}{k} \left( 1 - \binom{k}{0} p^0 (1 - p)^{k - 0} \right) \\ &= \frac{1 - \left(1 - p \right)^k }{ k }. \end{aligned} \]

Sampling With Known Support Size

We use the technique developed in the Sampling Game section, to avoid using \(O( | \mathbb{Supp}(x) | \log M)\) bits. We have seen that \(p\) controls the tradeoff between the size of \(B\), and the successful probability of sampling an element (\(B \neq \emptyset\)).

We begin with the assumption that the size of \(\mathbb{Supp}(x)\) is known.

Insertion Only

As before, we consider insertion only model first.

Claim. There is an algorithm such that 1. with probability \(1 - \delta\), it returns an element from \(\mathbb{Supp}(x)\) uniformly at random;
2. with probability \(\delta\), it returns nothing.

Further, the space usage is \(O \left( \ln \frac{1}{\delta} \right)\) with probability at least \(1 - \delta\).

For simplicity, we assume a perfect hash function \(h\) that maps elements in \([n]\) to \(\{0, 1\}\) independently with probability \(p\) (to be determined). For each update \((t, \Delta)\) in the stream, we process it only if \(h(t) = 1\). As we are dealing with insertion only model, we can use a hashset to maintain the elements with hash value 1.

Let \(k = | \mathbb{Supp}(x) |\). As analysed before, with probability \((1 - p)^k\), the algorithm fails. To make the failure probability as small as \(\delta\), we have that \((1 - p)^k = \delta\), i.e., \[ p = 1 - \delta^{ \frac{1}{k} } \le -\ln \delta^{ \frac{1}{k} } = \frac{1}{k} \ln \frac{1}{\delta}. \]

The last inequality holds since the function \(y = - \ln x\) is convex, and \(y = 1 - x\) is the tangent line of \(y = - \ln x\) at the point \((1, 0)\). By previous analysis, with probability at least \(1 - \delta\), the size of \(B\) is bounded by \[ \left( kp + \sqrt{2 k p (1 - p) \ln \frac{1}{\delta} } + \frac{2}{3} \ln \frac{1}{\delta} \right) \in O \left( \ln \frac{1}{\delta} \right). \]

Handling Deletion

When deletions are possible, we can't use a hashset to keep the sampled updates. E.g., suppose we have a sequence \((1, 6), (1, -6), (2, 5), (3, 7), (4, 9)\), and that \(h(1) = h(2) = 1, h(3) = h(4) = 0\). We need a hashset of size 2 to keep both the frequencies of element 1 and element 2. Even through the frequency of element \(1\) is 0 finally, we don't this before we see the entire string.

In general, \(h\) maps \(np\) elements in \([n]\) to \(1\). If we use a hashset, its size can grow to \(O(n p)\).

However, there is data structure that can handle deletions while using less than \(O(np)\) space.

Sparse Recovery. Given an integer \(d\) at the beginning of the stream, if \(\mathbb{Supp}(x) \le d\) when the stream stops, there is a data structure that returns

  1. indices in \(\mathbb{Supp}(x)\);
  2. and their values,

using \(O(d (\log n + \log M) ) \subset O( d \log (nM) )\) bits. It processes each update \((t, \Delta)\) in the stream in \(O(d)\) time and recovers \(x\) in \(O(d^2 polylog(nM))\) time. Its recovery can be wrong if \(\mathbb{Supp}(x) > d\).

It suffices to use such a data structure for the sampled set \(B\) with \(d \in O( \ln \frac{1}{\delta} )\). We have seen that if we sampled each element in \(\mathbb{Supp}(x)\) with probability \(p = 1 - \delta^{1 / k}\), then with probability at least \(1 - \delta\), we obtain a sampled set \(B\) with size at most \(d\). Then we return one element from \(B\) uniformly at random.

Analyzing the probability of sampling an element is more subtle than the previous section. In previous section, each element is under-sampled with probability \(\frac{1 - \left(1 - p \right)^k }{ k }\). Here, conditioned on \(|B|\le d\), each element is sampled equally likely. However, \(|B| > d\), the recovery of sampled elements fails, some element can be returned more frequently and some can be returned less frequently (the recovery behavior is un-defined). As the failure probability is bounded by \(\delta\), each element is returned with probability \[ \left[ \frac{1 - \left(1 - p \right)^k }{ k } \pm \delta \right] \subseteq \left[ 1 - \left( 1 + \frac{ 1 }{ k } \right) \cdot \delta, 1 + \left( 1 - \frac{ 1 }{ k } \right) \cdot \delta \right]. \]

Sampling With UnKnown Support Size

Previously, we want to sample each element in \([n]\) with probability \(1 - \delta^{ \frac{1}{k} } \le \ln \delta^{ \frac{1}{k} } = \frac{1}{k} \ln \frac{1}{\delta}\), where \(k = | \mathbb{Supp}(x) |\). When this information is unavailable, we can guess by trying the values \[ 1 / 2, \quad 1 / 4, \quad 1 / 8, \quad 1 / 16, ... \]

We don't need to guess the value of \(\frac{1}{k} \ln \frac{1}{\delta}\) exactly. In order that the sparse recovery data structure works, any guess within a constant factor suffices.

For simplicity, assume that \(n\) is some power of \(2\). Now, instead of using one sampling set \(B\), we use \(\log n + 2\) of them \(B_1, B_2, B_3, ...\), each associated with a sparse recovery structure with parameter \(d \in \Theta( \ln \frac{1}{\delta} )\). Assume that we have a perfect hash function \(h: [n] \rightarrow [\log n + 2]\) such that it holds independently for every \(t \in [n]\), \[ \begin{aligned} &\Pr[ h(t) = i ] = \frac{1}{2^i}, \forall i \in [\log n + 1], \\ &\Pr[ h(t) = \log n + 2] = \frac{1}{ 2^{\log n + 1} }. \end{aligned} \]

The distribution is well defined, as \(\sum_{i \in [\log n + 1] } 1 / 2^i + 1 / 2^{ \log n + 1} = 1 - 1 / 2^{ \log n + 1} + 1 / 2^{ \log n + 1} = 1\).

If \(k = 1\), some \(B_i\) must recovers \(x\) successfully.

Otherwise, \(\exists i \in [\log n]\), such that \(\frac{1}{2^i} \ge \frac{1}{k} \ln \frac{1}{\delta} > \frac{1}{2^{i + 1} }\). We study the following good events:

Event \(E_1:\) for all \(j \ge i\), it holds that \(|\{ t \in \mathbb{Supp}(x) : h(t) = j \}| \le d\).

Under \(E_1\), the sparse recovery structure associated with \(B_j\) recovers \(B_j\) successfully for all \(j \ge i\).

Event \(E_2:\) \(|\{ t \in \mathbb{Supp}(x) : h(t) \ge i \}| > 0\).

Under \(E_2\), there exists at least one element that is mapped to \(B_j\) for some \(j \ge i\).

Under these two good events, we sample one element as follows:

For \(j \leftarrow \log n + 2\) downto \(1\) do
\(\qquad\) recover \(B_j\);
\(\qquad\) if \(B_j \neq \emptyset\) then
\(\qquad\qquad\) return one element from \(B_j\) uniformly at random.

Analysis the probability of \(E_1\):

For \(j \ge i\), it holds that \(\mathbb{E} [ |B_j| ] = k / 2^j\). Suppose that \(d = 2 \cdot C_d \cdot \ln \frac{1}{\delta}\) for some constant \(C_d\) (to be determined). Since \(\frac{1}{k} \ln \frac{1}{\delta} \ge 1 / 2^{i + 1}\), we have that \[ \frac{d}{\mathbb{E} [ |B_j| ]} \ge \frac{ 2^j }{ k } \cdot 2 \cdot C_d \cdot \ln \frac{1}{\delta} \ge C_d \cdot 2^{j - i}. \]

By Chernoff bound, \[ \begin{aligned} \Pr \left[ |B_j| \ge d \right] &\le \left( \frac{e^{ \left( {d} / { \mathbb{E} [ |B_j| ] } \right) - 1 } }{ \left( {d} / { \mathbb{E} [ |B_j| ] } \right)^{ \left( {d} / { \mathbb{E} [ |B_j| ] } \right) } } \right)^{ \mathbb{E} [ |B_j| ]} \\ &\le \left( \frac{e}{C_d \cdot 2^{j - i} } \right)^d \left( \frac{1}{e} \right)^{ \mathbb{E} [ |B_j| ]}. \end{aligned} \]

By union bound, when \(C_d \ge e^2\), it holds that \[ \begin{aligned} \Pr \left[ \exists j \ge i : |B_j| \ge d \right] &\le \sum_{j \ge i} \left( \frac{e}{C_d \cdot 2^{j - i} } \right)^d \\ &\le \left( \frac{e}{C_d } \right)^d \sum_{j = 0}^\infty \left( \frac{1}{2^{d} } \right)^j \\ &\le \left( \frac{e}{C_d } \right)^d \frac{1}{1 - \frac{1}{2^{d} } } \\ &\le 2 \cdot \delta^{2 e^2}. \end{aligned} \]

\(\square\)

Analysis the probability of \(E_2\):

For each \(t \in \mathbb{Supp}(x)\), we have \[ \Pr[ h(t) < i ] = \left( 1 - \frac{1}{2^i} \right). \]

Therefore, \[ \Pr[ h(t) < i : \forall t \in \mathbb{Supp}(x) ] = \left( 1 - \frac{1}{2^i} \right)^k \le \left( 1 - \frac{1}{k} \ln \frac{1}{\delta} \right)^k \le \delta. \]

\(\square\)

AMS Sketch

Posted on 2021-04-14 Edited on 2021-05-14

Problem

There is a stream \(S\) of elements, each consists of a pair \((t, \Delta) \in [n] \times \mathbb{R}\). Let \(f \in \mathbb{R}^n\) be the frequency vector of \(S\), s.t., \(\forall i \in [n]\), \[ f_i = \sum_{ (t, \Delta) \in S \wedge t = i} \Delta. \]

Given \(\epsilon \in (0, 1)\), design a data structure that outputs an \((1 \pm \epsilon)\) estimate \(\Vert f \Vert_2^2\) with probability at least \(1 - \delta\), and uses space of \(O( \frac{1}{\epsilon^2 } \ln \frac{1}{\delta} )\).

Solution

Based on Count-Sketch.

Algorithm

Count-Sketch
1. \(k \leftarrow O \left( \ln \frac{1}{ \delta} \right); m \leftarrow \frac{3}{\epsilon^2}\).
2. \(h_1, h_2, ..., h_k: [n] \rightarrow [m]\), pairwise independent hash functions.
3. \(g_1, g_2, ..., g_k: [n] \rightarrow \{-1, 1\}\), four-wise independent hash functions.
4. \(M \leftarrow \{ 0 \}^{k \times m}\) be a \(k \times m\) matrix initialized with zeros.
4. For \((t, \Delta) \in S:\)
5. \(\qquad\) For \(i \in [k]:\)
6. \(\qquad\qquad\) \(M[i, h_i (t) ] \leftarrow M[i, h_i (t) ] + g_i(t) \cdot \Delta\)
\(\qquad\)

Estimation of \(\Vert f \Vert_2^2\)
1. For \(i \in [k]:\)
2. \(\qquad\) Let \(M_i\) be the \(i\)-th row of \(M\).
3. \(\qquad\) Let \(a_i \leftarrow \Vert M_i \Vert_2\).
4. Return the median of \((a_i^2)_{i \in [k] }\).

Analysis

Theorem 1. \(\forall i \in [k], \mathbb{E}[ a_i^2 ] = \Vert f \Vert_2^2\).

Proof. Consider a fixed \(i \in [k]\). \(\forall j \in [m]\), define the set \(h_i^{-1}[j]\) as elements in \([n]\) hashed to \(j\): \[ h_i^{-1}[j] \doteq \{ t \in [n] : h(t) = j \}. \]

Once \(h_i\) is fixed, it holds that \[ M[i, j] = \sum_{ t \in h_i^{-1}[j] } g_i (t) \cdot f_t. \]

By linearity of expectation and four-wise independence of \(g_i\), \[ \underset{g_i}{ \mathbb{E} }[ (M[i, j])^2 ] = \sum_{ t, t' \in h_i^{-1}[j] } \underset{g_i}{ \mathbb{E} } [ g_i (t) \cdot g_i (t) ] \cdot f_t \cdot f_{t'}. \]

It holds that \[ \underset{g_i}{ \mathbb{E} } [ g_i (t) \cdot g_i (t) ] = \begin{cases} 1, & \text{ if } t = t', \\ 0, & \text{ otherwise }. \end{cases} \]

Therefore, \(\underset{g_i}{ \mathbb{E} }[ (M[i, j])^2 ] = \sum_{ t \in h_i^{-1}[j] } f_t^2\).

Remark. This is a rather deep result, which says that whatever elements are hashed to bucket \(j\), \(\underset{g}{ \mathbb{E} } [ (M[i,j] )^2 ]\) equals the sum of squares of these elements.

By linearity of expectation, when \(h_i\) is fixed, \[ \underset{g}{ \mathbb{E} } [ a_i^2 ] = \sum_{j \in [m] } \underset{g}{ \mathbb{E} } [ (M[i,j])^2 ] = \sum_{j \in [m] } \sum_{ t \in h_i^{-1}[j] } f_t^2 = \Vert f \Vert_2^2. \]

It follows that \(\underset{h}{ \mathbb{E} } \left[ \underset{g}{ \mathbb{E} } [ a_i^2 ] \right] = \Vert f \Vert_2^2\).

Remark. The proof holds even for \(m = 1\), i.e., we can use only one bucket to construct an unbiased estimator of \(\Vert f \Vert_2^2\).

\(\square\)

Theorem 2. \(\mathbb{Var}[ a_i^2 ] = \frac{\Vert x \Vert_2^4 }{ m }\).

Proof.

\(\forall t \in [n]\), define \(X_t \doteq g_i(t) \cdot x_t\).

\(\forall t \in [n], j \in [m]\), define \(Y_{t, j} \doteq \mathbb{1}[ h_i[t] = j ] \cdot X_t\). Then \[ M[i,j] = \sum_{t \in [n] } Y_{t, j}, \] and hence

\[ \begin{aligned} \mathbb{Var}[a_i^2] &= \mathbb{Var} \left[ \sum_{j \in [m] } (M[i, j])^2 \right] \\ &= \mathbb{Var} \left[ \sum_{j \in [m] } \left( \sum_{t \in [n] } Y_{t, j} \right)^2 \right] \\ &= \mathbb{Var} \left[ \sum_{j \in [m] } \sum_{t, t' \in [n]} Y_{t, j} Y_{t', j} \right] \\ &= \mathbb{Var} \left[ \sum_{t, t' \in [n]} \sum_{j \in [m] } Y_{t, j} Y_{t', j} \right] \\ &= \mathbb{Var} \left[ \sum_{t, t' \in [n]} \left( \sum_{j \in [m] } \mathbb{1}[ h_i[t] = j ] \cdot \mathbb{1}[ h_i[t'] = j ] \right) \cdot X_t \cdot X_{t'} \right] \\ &= \mathbb{Var} \left[ \sum_{t, t' \in [n]} \mathbb{1}[ h_i[t] = h_i[t'] ] \cdot X_t \cdot X_{t'} \right] \\ &= \mathbb{Var} \left[ \sum_{t \in [n] } f_t^2 + \sum_{t \neq t' \in [n]} \mathbb{1}[ h_i[t] = h_i[t'] ] \cdot X_t \cdot X_{t'} \right] \\ &= \mathbb{Var} \left[ \sum_{t \neq t' \in [n]} \mathbb{1}[ h_i[t] = h_i[t'] ] \cdot X_t \cdot X_{t'} \right] \end{aligned} \]

For \(t \neq t' \in [n]\), define \(Z_{t, t'} = \mathbb{1}[ h_i[t] = h_i[t'] ] \cdot X_t \cdot X_{t'}\). Then \[ \mathbb{E}[ Z_{t, t'} ] = \mathbb{E}[ \mathbb{1}[ h_i[t] = h_i[t'] ] ] \cdot \mathbb{E}[ X_t ] \cdot \mathbb{E}[ x_{t'} ] = 0. \]

Consider \(t < t' \in [n]\), \(s < s' \in [n]\). If \((t, t') \neq (s, s')\), then either \(t \neq s\) or \(t' \neq s'\). Therefore, there must be an element that appears only once among the four. WLOG, suppose that \(t\) is a unique element. Then \[ Z_{t, t'} Z_{s, s'} = X_t \cdot \left( X_{t'} \cdot X_s \cdot X_{s'} \cdot \mathbb{1}[ h_i[t] = h_i[t'] ] \cdot \mathbb{1}[ h_i[s] = h_i[s'] ] \right). \]

As \(g_i\) is four-wise independent, and that \(t \neq t'\), \(t \neq s\) and \(t \neq s'\), it holds that \(g_i(t)\) is independent of \(\left( g_i(t') \cdot g_i(s) \cdot g_i(s') \cdot \mathbb{1}[ h_i[t] = h_i[t'] ] \cdot \mathbb{1}[ h_i[s] = h_i[s'] ] \right)\). Hence, \[ \mathbb{E}[ Z_{t, t'} Z_{s, s'}] = 0. \]

Therefore, \[ \begin{aligned} \mathbb{Var} \left[ \sum_{t \neq t' \in [n]} Z_{t, t'} \right] &= \mathbb{E} \left[ \left( \sum_{t \neq t' \in [n]} Z_{t, t'} \right)^2 \right] \\ &= \mathbb{E} \left[ \sum_{t \neq t' \in [n]} Z_{t, t'}^2 \right] \\ &= \sum_{t \neq t' \in [n]} \frac{ x_t^2 x_{t'}^2}{ m } \\ &= \frac{\Vert x \Vert_2^4 }{ m }. \end{aligned} \]

\(\square\)

\(\forall i \in [k], \Pr [ | a_i^2 - \Vert x \Vert_2^2 | \ge \epsilon \Vert x \Vert_2^2] \le \frac{1}{3}\).

Proof. By Chebyshev's inequality.

\(\square\)

\(\Pr \left[ \left| \{ i \in [k] : \Pr [ | a_i^2 - \Vert x \Vert_2^2 | \le \epsilon \Vert x \Vert_2^2] \} \right| > \frac{k}{2} \right] \ge 1 - \delta\).

Proof. By Chernoff bound.

\(\square\)

Reference

[1] G. Cormode and K. Yi, Small Summaries for Big Data, 1st ed. Cambridge University Press, 2020.

The Volume Bound

Posted on 2021-04-02 Edited on 2021-04-03

Given a finite set \(\Sigma\) and a integer \(n \in \mathbb{N}_+\), a code \(C \subset \Sigma^n\) is a set of strings that are far apart. The set \(\Sigma\) is called the alphabet of the code, and \(n\) is called the block length of the code. We write \(q = |\Sigma|\) as the size of the alphabet. Strings in \(C\) are called codewords.

For \(x, y \in \Sigma^n\), define their Hamming distance as \[ \Delta(x, y) = | \{ i \in [n] : x_i \neq y_i \} |. \]

The distance \(\Delta(C)\) of \(C\), is defined as the minimum distance between two different codes words in \(C\): \[ \Delta(C) \doteq \min_{x \neq y \in C} \Delta(x, y). \]

Suppose that we have a code \(C\) with distance \(d\). If we store a code \(x \in C\), and up to \((d - 1) / 2\) bits of \(x\) are corrupted, then we can still recover \(x\). Denote the ball centered at \(x\) with radius \(r \in \mathbb{R}_+\) as \[ B(x, r) \doteq \{ y \in \Sigma^n : \Delta(y, x) \le r \}. \]

The balls \(\{ B(x, (d - 1) / 2) : x \in C \}\) are non-overlapped.

We investigate the maximum possible size of \(C\).

Definition. Given \(q \in \mathbb{N}_+\), for \(r \in \mathbb{R}_+\), define the volume of the ball \(B(x, d)\) as \[ \mathbb{Vol}(r) = \sum_{i = 0}^{ \lfloor r \rfloor } \binom{n}{i} (q - 1)^i. \]

The following theorem upper bounds \(|C|\) in terms of the volume of the balls.

Theorem (Hamming Bound). Let \(C\) be a code with distance \(d \ge 3\). Then \[ |C| \le \frac{q^n }{ \mathbb{Vol}( (d - 1) / 2 ) }. \]

We are going to prove an estimate of the volume.

Estimate of Volume. Let \(r \in \mathbb{R}_+\) be a positive integer. When \(r \le n - n / q\), \[ q^{ n h_q ( \lfloor r \rfloor / n) - O( \log n) } \le \mathbb{Vol} (r) \le q^{ n h_q ( \lfloor r \rfloor / n)}, \] where \[ \begin{aligned} h_q (\epsilon) &\doteq (1 - \epsilon) \log_q \left( \frac{1}{1 - \epsilon} \right) + (q - 1) \frac{\epsilon}{q - 1} \log_q \left( \frac{q - 1}{ \epsilon } \right) \\ &= (1 - \epsilon) \log_q \left( \frac{1}{1 - \epsilon} \right) + \epsilon \log_q \left( \frac{q - 1}{ \epsilon } \right). \end{aligned} \]

Observe that by concavity of \(\log_q (\cdot)\), we have \[ h_q (\epsilon) \le \log_q \left( (1 - \epsilon) \frac{1}{1 - \epsilon} + \epsilon \frac{q - 1}{ \epsilon } \right) = 1. \]

The inequality holds with equality when \(h_q(\epsilon)\) is maximized when \(1 - \epsilon = \epsilon / (q - 1)\), i.e., \(1 - \epsilon = 1 / q\).

It immediately follows from the estimate that \[ n h_q( \lfloor d / 2 - 1 / 2 \rfloor / n) + \log_q |C| \le n + O(\log n) \\ \Longleftrightarrow h_q( \lfloor d / 2 - 1 / 2 \rfloor / n) + \frac{ \log_q |C| }{ n } \le 1 + O\left(\frac{ \log_q n }{ n } \right). \]

On the other hand, \[ \begin{aligned} h_q'(\epsilon) &= \frac{1}{\ln q} \left( \ln(1 - \epsilon) + (1 - \epsilon) \frac{1}{1 - \epsilon} + \ln (q - 1) + \ln \left( \frac{1}{\epsilon} \right) - \epsilon \frac{1}{\epsilon} \right) \\ &= \frac{1}{\ln q} \left( \ln \left( \frac{1 - \epsilon}{\epsilon} \right) + \ln (q - 1) \right). \end{aligned} \]

When \(\frac{1 - \epsilon}{\epsilon} = \frac{1}{\epsilon} - 1 \ge \frac{1}{q - 1}\), i.e., \(\epsilon \le 1 - 1 / q\), it holds that \(h_q' \ge 0\). Conditioned on that \(\lfloor d / 2 - 1 / 2 \rfloor \ge 1\), i.e., \(d \ge 3\), we have \(d / 2 \le \lfloor d / 2 - 1 / 2 \rfloor + 1\) and \[ h_q( \lfloor d / 2 - 1 / 2 \rfloor / n) \ge h_q( d / (2n) ) + h'_q(1 / n ) (1 / n) = h_q( d / (2n) ) + O( (\log_q n) / n). \]

Define the rate of the code \(R = \frac{ \log_q |C| }{ n }\), and relative distance of the code as \(\delta = \frac{2d + 1}{n}\), we get

\[ R + h_q( \delta / 2) \le 1 + O\left( \frac{\log_q n }{n} \right). \]

In what's follows, we prove the inequalities for volume estimation. WLOG, we assume that \(r\) is an integer.

Upper Bound

For \(r \in \mathbb{N}_+\), it holds that \(\mathbb{Vol} (r) \le q^{n h_q(r / n)}\).

Algebraic Approach

\[ \begin{aligned} \mathbb{Vol}(r) &\le (q - 1)^r \sum_{i = 0}^r \binom{n}{i} \\ &\le (q - 1)^r \sum_{i = 0}^r \binom{n - r}{i} \frac{n^r}{(n - r)^i} \\ &\le (q - 1)^r \left( \frac{n}{r} \right)^r \sum_{i = 0}^r \binom{n - r}{i} \left( \frac{r}{n - r} \right)^i \\ &\le (q - 1)^r \left( \frac{n}{r} \right)^r \sum_{i = 0}^{n - r} \binom{n - r}{i} \left( \frac{r}{n - r} \right)^i \\ &= (q - 1)^r \left( \frac{n}{r} \right)^r \left( 1 + \frac{r}{n - r} \right)^{n - r} \\ &= (q - 1)^{ n \frac{r}{n} } \left( \frac{n}{r} \right)^{ n \frac{r}{n} } \left( \frac{1}{1 - \frac{r}{n} } \right)^{n \left( 1 - \frac{r}{n} \right) } \end{aligned} \]

Let \(\epsilon \doteq \frac{r}{n}\), we have \[ \log_q \mathbb{Vol}(r) \le n \left( \epsilon \cdot \log_q \frac{q - 1}{\epsilon} + (1 - \epsilon) \cdot \log_q \frac{1}{1 - \epsilon} \right) = n h_q (\epsilon). \]

Entropy

We rely on sub-additivity of entropy.

Let \(X\) and \(Y\) be discrete random variables, then
1. \(H( X, Y ) \le H(X) + H(Y)\).
2. \(H(X | Y) \le H(X)\).

Proof. Consider the ball \(B(1^n, r)\), and a uniform random point \(X = (X_1, ..., X_n)\) in the ball \(B(1^n, r)\). Its entropy is \[ H(X) = \log \mathbb{Vol}(r). \]

We write \(Y = X_1\). By sub-additivity of entropy and by symmetry, \[ \begin{aligned} H(X) &\le H(X_1) + H(X_2) + ... + H(X_n) &= n H(Y). \end{aligned} \]

Let \(\epsilon \doteq \Pr[ Y \neq 1]\) be the probability that \(Y\) is not 1. Then by concavity of \(\log (\cdot)\), \[ \begin{aligned} H(Y) &= \Pr[ Y = 1] \cdot \log \left( \frac{1}{\Pr[ Y = 1]}\right) + \sum_{i \neq 1} \Pr[ Y = i ] \cdot \log \left( \frac{1}{\Pr[ Y = i]}\right) \\ &= (1 - \epsilon) \cdot \log \left( \frac{1}{1 - \epsilon} \right) + \epsilon \cdot \sum_{i \neq 1} \frac{ \Pr[ Y = i ] }{ \epsilon } \cdot \log \left( \frac{1}{\Pr[ Y = i]}\right) \\ &\le (1 - \epsilon) \cdot \log \left( \frac{1}{1 - \epsilon} \right) + \epsilon \cdot \log \left( \sum_{i \neq 1} \frac{ \Pr[ Y = i ] }{ \epsilon } \frac{1}{\Pr[ Y = i]}\right) \\ &= (1 - \epsilon) \cdot \log \left( \frac{1}{1 - \epsilon} \right) + \epsilon \cdot \log \left( \frac{ q - 1 }{ \epsilon } \right) \\ \end{aligned} \] where then entropy of \(Y\) is maximized when \(Y\) is uniform of the elements that are not 1. Further, it can be verified that \(\epsilon < r / n\). As \(h_q (\epsilon)\) increases for \(\epsilon < 1 - 1 / q\), hence \[ H(Y) \le (1 - r / n) \cdot \log \left( \frac{1}{1 - r / n} \right) + \frac{r}{n} \cdot \log \left( \frac{ q - 1 }{ r / n } \right). \]

\(\square\)

Supplementary Material

Definition. Let \(X\) be a discrete r.v., the entropy of \(X\) is given by \[ H(X) = \sum_{a} \Pr[ X = a ] \log \frac{1}{ \Pr[ X = a] } \]

Definition. Let \(X, Y\) be discrete r.v., the entropy of \(X\) conditioned \(Y\) is given by \[ \begin{aligned} H(X \mid Y) &= \mathbb{E}_{b \leftarrow Y} [ H(X \mid Y = b) ] \\ &= \sum_{b} \Pr[ Y = b] \sum_{a } \Pr[ X = a \mid Y = b] \log \frac{1}{ \Pr[ X = a \mid y = b] } \\ &= \sum_{b, a} \Pr[ Y = b, X = a] \log \frac{1}{ \Pr[ X = a \mid y = b] } \\ \end{aligned} \]

Chain Rule. \[ \begin{aligned} H(X \mid Y) + H(Y) &= \sum_{b, a} \Pr[ Y = b, X = a] \log \left( \frac{1}{ \Pr[ X = a \mid y = b] } \cdot \frac{1}{ \Pr[ Y = b ] } \right) \\ &= \sum_{b, a} \Pr[ Y = b, X = a] \log \left( \frac{1}{ \Pr[ X = a \mid y = b] } \cdot \frac{1}{ \Pr[ Y = b ] } \right) \\ &= H(X, Y) \end{aligned} \]

Sub-additivity. For convenience, we rewrite \(p(a, b) = \Pr[X = a, Y = a]\), \(p(a) = \Pr[ X = a]\), and \(p(b) = \Pr[ Y = b]\). First, \[ \begin{aligned} H(X, Y) - H(X) - H(Y) &= \sum_{a, b} p(a, b) \log \frac{1}{ p(a, b) } - \sum_{a, b} p(a, b) \log \frac{1}{ p(a)} - \sum_{a, b} \log \frac{1}{p(b)} \\ &= \sum_{a, b} p(a, b) \log \frac{ p(a) p(b) }{ p(a, b) } \\ &\le \log \sum_{a, b} p(a, b) \frac{ p(a) p(b) }{ p(a, b) } \\ &= \log \sum_{a, b} p(a) p(b) \\ &= 0. \end{aligned} \] Second, we have \[ H(X \mid Y) = H(X, Y) - H(Y) \le H(X). \]

Lower Bound

For \(r \in \mathbb{N}_+\), it holds that \(\mathbb{Vol} (r) \ge q^{ n h_q ( \lfloor r \rfloor / n) - O( \log n) }\).

We rely on the Stirling's approximation.

For \(n \in \mathbb{N}_+\), \(\sqrt{2 \pi n} \left(\frac{n}{e} \right)^n \le n! \le e \sqrt{n} \left(\frac{n}{e} \right)^n\).

Proof. \[ \begin{aligned} \mathbb{Vol} (r) &\ge \binom{n}{r} (q - 1)^r \\ &\ge \frac{ \sqrt{2 \pi n } \left(\frac{n}{e} \right)^n }{ e \sqrt{r} \left(\frac{r}{e} \right)^r e \sqrt{n - r} \left(\frac{n - r}{e} \right)^{n - r} } (q - 1)^r \\ &= (q - 1)^r \left(\frac{n}{r} \right)^r \left(\frac{n}{n - r} \right)^{n - r} \sqrt{ \frac{2 \pi n }{e^4 r (n - r)} } \\ &= q^{n h_q ( r / n) - O(\log n) } \end{aligned} \]

\(\square\)

Hence, the upper bound is rather tight.
