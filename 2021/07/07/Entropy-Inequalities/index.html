Entropy Inequalities

Posted on 2021-07-07 Edited on 2021-07-08

Let XXX and YYY be two discrete random variables.

Joint Entropy

Definition. The joint entropy of XXX and YYY is

H(X,Y)=∑x,yp(x,y)log⁡1p(x,y) H(X, Y) = \sum_{x, y} p(x, y) \log \frac{1}{ p(x, y) } H(X,Y)=x,y∑​p(x,y)logp(x,y)1​

If XXX and YYY are independent, then

H(X,Y)=H(X)+H(Y). H(X, Y) = H(X) + H(Y). H(X,Y)=H(X)+H(Y).

Conditional Entropy

Definition. The conditional entropy of XXX given Y=yY = yY=y is the entropy of the probability distribution p(X∣Y=y)p(X \mid Y = y)p(X∣Y=y)

H(X∣Y=y)=∑xp(x∣y)log⁡1p(x∣y) H(X \mid Y = y) = \sum_x p(x \mid y) \log \frac{1}{ p(x \mid y) } H(X∣Y=y)=x∑​p(x∣y)logp(x∣y)1​

Definition. The conditional entropy of XXX given YYY, is the average, over yyy, the conditional entropy of XXX given yyy

H(X∣Y)=∑yp(y)H(X∣Y=y)=∑x,yp(x,y)log⁡1p(x∣y). H(X \mid Y) = \sum_y p(y) H(X \mid Y = y) = \sum_{x, y} p(x , y) \log \frac{1}{ p(x \mid y) }. H(X∣Y)=y∑​p(y)H(X∣Y=y)=x,y∑​p(x,y)logp(x∣y)1​.

Relative Entropy

Given a distribution ppp, if we encode it with a distribution qqq (with the same support), the expected coding length is roughly

∑ipilog⁡1qi \sum_i p_i \log \frac{1}{q_i} i∑​pi​logqi​1​

The cost over the optimal coding is given by

D(p∣∣q)≐∑ipilog⁡1qi−∑ipilog⁡1pi=∑ipilog⁡piqi D(p || q) \doteq \sum_i p_i \log \frac{1}{q_i} - \sum_i p_i \log \frac{1}{p_i} = \sum_i p_i \log \frac{p_i}{q_i} D(p∣∣q)≐i∑​pi​logqi​1​−i∑​pi​logpi​1​=i∑​pi​logqi​pi​​

By definition, we know that this gap is always non-negative.
We can prove it rigorously algebraically. We show two different approaches here.

  1. By that f(x)=xlog⁡xf(x) = x \log xf(x)=xlogx is convex for x≥0x \ge 0x≥0, we have

    D(p∣∣q)=∑ipilog⁡piqi=∑iqi⋅piqilog⁡piqi≥(∑iqi⋅piqi)log⁡(∑iqi⋅piqi)=0\begin{aligned} D(p || q) &= \sum_i p_i \log \frac{p_i}{q_i} \\ &= \sum_i q_i \cdot \frac{p_i}{q_i} \log \frac{p_i}{q_i} \\ &\ge \left( \sum_i q_i \cdot \frac{p_i}{q_i} \right) \log \left( \sum_i q_i \cdot \frac{p_i}{q_i} \right) \\ &= 0 \end{aligned} D(p∣∣q)​=i∑​pi​logqi​pi​​=i∑​qi​⋅qi​pi​​logqi​pi​​≥(i∑​qi​⋅qi​pi​​)log(i∑​qi​⋅qi​pi​​)=0​

  2. By that f(x)=−log⁡xf(x) = -\log xf(x)=−logx is convex (for x≥0x \ge 0x≥0 ), we have

    D(p∣∣q)=∑i−pilog⁡qipi≥−log⁡(∑ipi⋅qipi)=0\begin{aligned} D(p || q) &= \sum_i - p_i \log \frac{q_i}{p_i} \\ &\ge -\log \left( \sum_i p_i \cdot \frac{q_i}{p_i} \right) \\ &= 0 \end{aligned} D(p∣∣q)​=i∑​−pi​logpi​qi​​≥−log(i∑​pi​⋅pi​qi​​)=0​

Mutual Information

Definition. The mutual information between XXX and YYY is

I(X;Y)=Ep(x,y)[log⁡p(x,y)p(x)p(y)]=D(p(x,y)∥p(x)p(y)). I(X; Y) = \mathbb{E}_{p(x, y) } \left[ \log \frac{ p(x, y)}{p(x) p(y) } \right] = D( p(x, y) \Vert p(x) p(y) ). I(X;Y)=Ep(x,y)​[logp(x)p(y)p(x,y)​]=D(p(x,y)∥p(x)p(y)).

The definition is symmetric and it holds that I(X;Y)=I(Y;X)I(X; Y) = I(Y; X)I(X;Y)=I(Y;X).
If we know p(x,y)p(x,y)p(x,y), we use in expectation Ep(x,y)[log⁡1p(x,y)]\mathbb{E}_{p(x, y) } \left[ \log \frac{1} { p(x, y)} \right]Ep(x,y)​[logp(x,y)1​] bits to describe them.
If we know only p(x)p(x)p(x) and p(y)p(y)p(y), the mutual information measures the bits we waste.

∑p(x,y)log⁡p(x,y)p(x)p(y)=∑p(x,y)(log⁡1p(x)+log⁡1p(y)−log⁡1p(x,y))=H(X)+H(Y)−H(X,Y) \small \begin{aligned} &\sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } \\ =& \sum p(x, y) \left( \log \frac{ 1 }{p(x)} + \log \frac{ 1 }{p(y)} - \log \frac{ 1 }{p(x, y)} \right) \\ =& \mathbf{H} ( X ) + \mathbf{H} ( Y ) - \mathbf{H} ( X, Y ) \end{aligned} ==​∑p(x,y)logp(x)p(y)p(x,y)​∑p(x,y)(logp(x)1​+logp(y)1​−logp(x,y)1​)H(X)+H(Y)−H(X,Y)​

Further, mutual information measures the average reduction of randomness of XXX that results from learning the value of YYY.

∑p(x,y)log⁡p(x,y)p(x)p(y)=∑p(x,y)log⁡p(x∣y)p(x)=H(X)−H(X∣Y). \small \begin{aligned} \sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } &=\sum p(x, y) \log \frac{ p(x | y)}{p(x) } \\ &= \mathbf{H} ( X ) - \mathbf{H} ( X \mid Y). \end{aligned} ∑p(x,y)logp(x)p(y)p(x,y)​​=∑p(x,y)logp(x)p(x∣y)​=H(X)−H(X∣Y).​

Similarly, we can prove that

∑p(x,y)log⁡p(x,y)p(x)p(y)=∑p(x,y)log⁡p(x∣y)p(x)=H(Y)−H(X∣Y). \small \begin{aligned} \sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } &=\sum p(x, y) \log \frac{ p(x | y)}{ p(x) } \\ &=\mathbf{H} ( Y ) - \mathbf{H} ( X \mid Y). \end{aligned} ∑p(x,y)logp(x)p(y)p(x,y)​​=∑p(x,y)logp(x)p(x∣y)​=H(Y)−H(X∣Y).​

Lemma.

I(X;Y)≥0. \mathbf{I}(X; Y) \ge 0. I(X;Y)≥0.

Proof.

I(X;Y)=D(p(x,y)∣p(x)p(y))≥0 \mathbf{I}(X; Y) = D( p(x, y) \mid p(x) p(y) ) \ge 0 I(X;Y)=D(p(x,y)∣p(x)p(y))≥0

Or

∑p(x,y)log⁡p(x,y)p(x)p(y)=−∑p(x,y)log⁡p(x)p(y)p(x,y)≥−log⁡∑p(x,y)p(x)p(y)p(x,y)=0\begin{aligned} \sum p(x, y) \log \frac{ p(x, y)}{p(x) p(y) } &= -\sum p(x, y) \log \frac{p(x) p(y) }{ p(x, y)} \\ &\ge - \log \sum p(x, y) \frac{p(x) p(y) }{ p(x, y)} \\ &=0 \end{aligned} ∑p(x,y)logp(x)p(y)p(x,y)​​=−∑p(x,y)logp(x,y)p(x)p(y)​≥−log∑p(x,y)p(x,y)p(x)p(y)​=0​

□\square□

Corollaries

H(Y)−H(X∣Y)=H(X)−H(X∣Y)≥0. \mathbf{H} ( Y ) - \mathbf{H} ( X \mid Y) = \mathbf{H} ( X ) - \mathbf{H} ( X \mid Y) \ge 0. H(Y)−H(X∣Y)=H(X)−H(X∣Y)≥0.

Chain Rules

Let X1,X2,…,XnX_1, X_2, \ldots, X_nX1​,X2​,…,Xn​ be random variables whose density function is p(x1,x2,…,xn)p(x_1, x_2, \ldots, x_n)p(x1​,x2​,…,xn​).
