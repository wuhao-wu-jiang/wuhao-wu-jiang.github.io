Lebesgue Measure Construction

Posted on 2020-10-07 Edited on 2022-09-18

Measure Space

Let \(\Omega\) be a set and \(2^\Omega\) be the power set of \(\Omega\).

\(\sigma\)-Algebra

Definition. A collection of subsets \(\mathcal{F} \subseteq 2^\Omega\) is a \(\sigma\)-algebra (or \(\sigma\)-field) if
* It is closed under countable union, intersection and complement.

Measure

Definition. Given a collection of subsets \(\mathcal{F} \subseteq 2^\Omega\), a function \(\mu : \mathcal{F} \rightarrow \mathbb{R}\) is a measure if it satisfies

  1. It is non-negative: \(\forall A \in \mathcal{F}, \mu(A) \ge 0\).

  2. The empty set is mapped to 0: \(\mu(\emptyset) = 0\).

  3. \(\sigma\)-additivity: if \(A_i \in \mathcal{F}\) is a countable sequence of disjoint sets, s.t., \(\cup_{i \ge 1} A_i \in \mathcal{F}\), then
    \[ \mu(\cup_{i \ge 1} A_i) = \sum_{i \ge 1} \mu(A_i ). \]

Measure Space

Definition. A measure space \((\Omega, \mathcal{F}, \mu)\) consists of three part:

  1. A set \(\Omega\), on which it is defined;

  2. A \(\sigma\)-field \(\mathcal{F}\) that contains all measurable sets;

  3. A measure \(\mu\) defined on \(\mathcal{F}\).

In general, it is not easy to construct a measure space on \(\Omega\) directly. Our indirect construction consists of three steps:

  1. Consider a simple set class \(S\) and a measure \(\mu\) defined on \(S\).
  2. We extend the domain of \(\mu\) from \(S\) to a larger one \(\Omega=2^\Omega\).
  3. We choose a subset from \(2^\Omega\), such that it is a \(\sigma\)-field and \(\mu^*\) is a measure defined on it.

In what follows, we illustrate the idea via constructing measures on \(\mathbb{R}^d\).

Semialgebra

Definition. A class of subsets \(S \subseteq 2^\Omega\) is a semialgebra if

  1. It is closed under intersection: \(\forall A, B \in S\), \(A \cap B \in S\).

  2. If \(A \in S\), then \(\bar A\) is a finite disjoint union of sets in \(S\).

Examples. For \(d \in \mathbb{Z}^+\), the collection of half-open intervals in \(\mathbb{R}^d\) plus the empty set \[ S_d \doteq \{ \times_{i \in [d] } (a_i, b_i] : -\infty \le a_i \lt b_i \le \infty, \forall i \in [d] \} \cup \{ \emptyset \} \]

Algebra

Definition. A class of subsets \(\mathcal{F} \subseteq 2^\Omega\) is an algebra (or field) if

  • It is closed under union, intersection and complement.

Remark. By De Morgan's laws, \(\overline{A \cap B } = \bar A \cup \bar B\) and \(\overline{A \cup B} = \bar A \cap \bar B\). If we know a collection is closed under intersection (union) and complement, so is union (intersection).

Theorem If \(S \subseteq \Omega\) is a semialgebra, then \[ \widetilde{S} \doteq \{ \text{finite disjoint union of sets in } S \} \] is an algebra.

Proof.

  1. \(\widetilde{S}\) is closed under intersection: Let \(A, B \in \widetilde{S}\) with \(A = +_i A_i\) and \(B = +_j B_j\), where \(+\) denotes disjoint union and \(A_i, B_j \in S\). Then \(A \cap B = +_{i, j} A_i \cap B_j\). As \(S\) is a semialgebra, it follows that \(A_i \cap B_j \in S\) and \(A \cap B \in \widetilde{S}\).

  2. \(\widetilde{S}\) is closed under complement. Let \(A \in \widetilde{S}\) with \(A = +_i A_i\), where \(A_i \in S\). Then \(\bar A = \cap_i \bar A_i\). As \(S\) is a semialgebra, \(\bar A_i\) is a finite disjoint union of sets in \(S\), which implies that \(\bar A_i \in \widetilde{S}\). Therefore \(\bar A\) is a finite intersection of sets in \(\widetilde{S}\). Since \(\widetilde{S}\) is closed under intersection, \(\bar A \in \widetilde{S}\).

\(\square\)

Measure Extension: Semialgebra \(\rightarrow\) Algebra

Theorem. Let \(S\) be a semialgebra and \(\mu\) be a measure on \(S\). Then \(\mu\) has a unique extension \(\bar \mu\) that is a measure on \(\widetilde S\), the algebra generated by \(S\).

Remark. The condition of \(\mu\) being a measure can be replaced by
1. Finite additivity. If \(A, A_i \in S\) with \(A = +_i A_i\), then \(\mu(A) = \sum_i \mu (A_i)\).
2. Subadditivity. If \(A, A_i \in S\) with \(A = +_{i \ge 1} A_i\), then \(\mu (A) \le \sum_{i \ge 1} \mu (A_i)\).

The subscript \(i \ge 1\) in the conditions indicates a countable union, while a plain subscript \(i\) or \(j\) indicates a finite union.

The relation between the condition stated in the Theorem and the relaxed ones is established as follows.

Exercise. A finitely additive nonnegative set function on an algebra \(\widetilde{S}\) is monotone and the following conditions are equivalent 1. \(\sigma\)-additivity. 2. Subadditivity.

The proof for the Theorem consists of two steps.

\(\blacktriangleright\) Defining \(\bar \mu\)

If \(\mu\) is finitely additive in \(S\), then it has a finitely additive extension \(\bar \mu\) in \(\widetilde{S}\).

  1. Extension. If \(A = +_i A_i \in \widetilde{S}\) with \(A_i \in S\), define \[ \bar \mu(A) = \sum_i \mu (A_i). \]

    We need to verify the value of \(\bar \mu(A)\) is unique. Suppose \(A\) has another decomposition: \(A = +_j B_j\) with \(B_j \in S\). Since \(S\) is a semialgebra, \(A_i \cap B_j \in S\). Using that \(\mu\) is a measure \[ \sum_i \mu (A_i) = \sum_{i, j} \mu(A_i \cap B_j) = \sum_j \mu(B_j). \]

    It follows that \(\bar \mu(A)\) does not change with the decomposition of \(A\) and hence is well-defined.

  2. Finite additivity. Let \(A = +_i A_i \in \widetilde{S}\) with \(A_i \in \widetilde{S}\) and suppose that \(A_i = +_j B_{i,j}\) with \(B_j \in S\). Then \(A = +_{i,j} B_{i,j}\). By definition of \(\bar \mu (A)\) and \(\bar \mu (A_i)\), \[ \bar \mu (A) = \sum_{i,j} \mu (B_{i,j}) = \sum_{i} \bar \mu ( A_i ). \]

We list two more properties about \(\bar \mu\) based on finite additivity.

  • Monotonicity. If \(A, B \in \widetilde{S}\) with \(A \subseteq B\), then \(\bar \mu (A) \le \bar \mu (B)\). As \(\widetilde{S}\) is an algebra, \(B - A \in \widetilde{S}\) and by finite additivity of \(\bar \mu\), \[ \bar \mu (A) \le \bar \mu (A) + \bar \mu (B - A) = \bar \mu (B). \]

  • Finite subadditivity. If \(A \subseteq \cup_{i \in [n] } A_i\) with \(A, A_i \in \widetilde{S}\), then \(\bar \mu(A) \le \sum_i \bar \mu (A_i)\). Let \(B_1 = A_1\) and \(B_i = A_i - A_{i - 1} - \ldots - A_1\) for \(i \ge 1\). As \(\widetilde{S}\) is an algebra, \(B_i \in \widetilde{S}\). By monotonicity and finite additivity of \(\bar \mu\), \[ \bar \mu (A) \le \bar \mu( +_i B_i ) = \sum_i \bar \mu (B_i) \le \sum_i \bar \mu (A_i). \]

\(\square\)

\(\blacktriangleright\) \(\sigma\)-Additivity of \(\bar \mu\)

If \(A, A_1, \ldots \in \widetilde S\) with \(A = +_{i \ge 1} A_i\), then \[\small \bar \mu (A) = \sum_{i \ge 1} \bar \mu (A_i). \]

Proof. Let \(A_i = +_j B_{i,j}\) with \(B_{i,j} \in S\). Then \(A = +_{i,j} B_{i,j}\) and \(\sum_{i \ge 1} \bar \mu(A_i) = \sum_{i \ge 1} \sum_j \mu ( B_{i,j} )\). WLOG, we assume that \(A_i\) are countable union of disjoint sets in \(S\).

Next, assume that \(A = +_k C_k\) with \(C_k \in S\). Using that \(\mu\) is a measure and \(S\) is closed under intersection, \[ \mu(C_k) = \sum_{i \ge 1} \mu (C_k \cap A_i). \]

Observing that nonnegative numbers can be summed in any order, \[ \small \begin{aligned} \bar \mu(A) = \sum_k \mu (C_k) = \sum_k \sum_{i \ge 1} \mu (C_k \cap A_i) = \sum_{i \ge 1} \sum_k \mu (C_k \cap A_i) = \sum_{i \ge 1} \mu (A_i). \end{aligned} \]

\(\square\)

Measure Extension: Algebra \(\rightarrow\) \(\sigma\)-Algebra

Definition. \(\bar \mu\) is said to be \(\sigma\)-finite if there is a sequence of sets \(A_i \in \widetilde{S}\) such that \(\mu(A_i) < \infty\) and \(\cup_{i \ge 1} A_i = \Omega\).

Caratheodory’s Extension Theorem. Let \(\bar \mu\) be a \(\sigma\)-finite measure on an algebra \(\widetilde S\). Then \(\bar \mu\) has a unique extension \(\mu^*\) (that is a measure) to \(\sigma(\widetilde S)\), the smallest \(\sigma\)-field containing \(\widetilde S\).

Remark: Observe that \(\sigma(\widetilde S) = \sigma(S)\).

The proof is included at the end of the article.

Lebesgue-Stieltjes Measure

In this section, we construct concrete measures on \(\mathbb{R}^d\).

Definition. A function \(F: \mathbb{R} \rightarrow \mathbb{R}\) is a Stieltjes measure function if

  1. \(F\) is non-decreasing.

  2. \(F\) is right continuous, i.e., \(\lim_{y \downarrow x} F(y) = F(x)\).

Definition (Borel Sets). Let \(\mathcal{R}^d = \sigma (S_d)\). It is the smallest \(\sigma\)-field containing all open sets in \(\mathbb{R}^d\).

Theorem 3. Associated with each Stieltjes measure function \(F\) there is a unique measure \(\mu\) on \((\mathbb{R},\mathcal{R}^1)\) with \[ \mu \big( (a, b] \big) = F(b) - F(a). \] When \(F(x) = x\), the measure \(\mu\) is called Lebesgue measure.

Proof. We just to verity that \(\mu\) is a measure defined on the following semi-algebra: \[ S_1 \doteq \{ (a, b] : -\infty \le a \lt b \le \infty \} \cup \{ \emptyset \}. \]

  1. \(\mu((a, b]) \ge 0\), since \(F\) is non-decreasing.

  2. Finite additivity. If \((a, b] = +_i (a_i, b_i]\), \(\mu \big( (a, b] \big) = \sum_i \mu \big( ( a_i, b_i ] \big)\). We can rearrange the \(\{ (a_i, b_i] \}\), such that \(a = a_1\), \(b_i = a_{i + 1}\) for all \(1 \le i < n\), and \(b_n = b\). Hence, \[ \mu \big( (a, b] \big) = F(b) - F(a) = \sum_i ( F(b_i) - F(a_i) ) = \sum_i \mu \big( ( a_i, b_i ] \big). \]

  3. Finite subadditivity. If \((a, b] \subseteq \cup_{i \in [n] } (a_i, b_i]\), \(\mu \big( (a, b] \big) \le \sum_{i \in [ n ] } \mu \big( ( a_i, b_i ] \big)\). Recall that the definition of \(\bar \mu\) only requires finite additivity of \(\mu\). Working with \(\widetilde{S_1}\) and \(\bar \mu\) significantly simplifies the proof. As for all \(A \in S_1\), it holds that \(\mu(A) = \bar \mu(A)\). It suffices to prove that \[ \bar \mu \big( (a, b] \big) \le \sum_{i \in [ n ] } \bar \mu \big( (a_i, b_i] \big). \] But this follows exactly from finite subadditivity of \(\bar \mu\).

  4. \(\sigma\)-additivity. If \((a, b] = +_{i \ge 1} (a_i, b_i]\), then \(\mu \big( (a, b] \big) = \sum_{i \ge 1} \mu \big( ( a_i, b_i ] \big)\). Using finite subadditivity of \(\mu\), for all \(n \in \mathbb{Z}^+\), \[ \mu \big( (a, b] \big) \ge \sum_{i \in [n] } \mu \big( ( a_i, b_i ] \big). \] Letting \(n \rightarrow \infty\), we get \(\mu \big( (a, b] \big) \ge \sum_{i \ge 1} \mu \big( ( a_i, b_i ] \big)\).

    It is left to prove the reverse direction of the inequality. If \(\mu \big( ( a_i, b_i ] \big) = \infty\) for some \(i\), then the inequality is trivial. WLOG, assume that \(-\infty < a_i < b_i < \infty\) for all \(i\).

    First, suppose that both \(a\) and \(b\) are finite. For each \(\epsilon > 0\), as \(F\) is right continuous, there exists some \(\delta > 0\), such that \[ \begin{aligned} F(a + \delta) &\le F(a) + \epsilon\,,\qquad \text{ and } \\ \mu \Big( (a, b] \Big) &\le \mu \Big( (a + \delta, b] \Big) + \epsilon. \end{aligned} \] For each \(i\), there exists some \(\delta_i > 0\), such that \[ \begin{aligned} F(b_i + \delta_i) &\le F(b_i) + \epsilon / 2^i\,,\qquad \text{ and } \\ \mu \Big( (a_i, b_i + \delta_i] \Big) &\le \mu \Big( (a_i, b_i] \Big) + \epsilon / 2^i. \end{aligned} \] Since \(\{ (a_i, b_i + \delta_i) \}\) is an open cover of the closed interval \([a + \delta, b]\), there is a finite sub-cover \(\{ (s_i, t_i) \}\). It follows that \(\{ (s_i, t_i] \}\) covers \((a + \delta, b]\). Therefore, \[ \mu \Big( (a + \delta, b] \Big) \le \sum_i \mu \Big( (s_i, t_i] \Big) \le \sum_{i \ge 1} \mu \Big( (a_i, b_i + \delta_i] \Big) \] It follows that \[ \mu \Big( (a, b] \Big) \le \sum_{i \ge 1} \mu \Big( (a_i, b_i] \Big) + 2 \epsilon. \] Letting \(\epsilon \rightarrow 0\) finishes the proof.

    If \(a = -\infty\) or \(b = \infty\), the inequality \(\small \mu \Big( (a', b'] \Big) \le \sum_{i \ge 1} \mu \Big( (a_i, b_i] \Big)\) holds for all finite subset \((a', b'] \subseteq (a, b]\). Hence it holds for \((a, b]\).

\(\square\)

Caratheodory’s Extension Theorem

Caratheodory’s Extension Theorem. Let \(\bar \mu\) be a \(\sigma\)-finite measure on an algebra \(\widetilde S\). Then \(\bar \mu\) has a unique extension \(\mu^*\) (that is a measure) to \(\sigma(\widetilde S)\), the smallest \(\sigma\)-field containing \(\widetilde S\).

\(\spadesuit\) Outer Measure

Define the function \(\mu^*: 2^\Omega \rightarrow \mathbb{R}_{\ge 0}\) as \(\forall A \subseteq \Omega\), \[ \mu^*(A) = \inf \left\{ \sum_{i \ge 1} \bar{ \mu }(A_i) : A \subseteq \cup_{i \ge 1} A_i \text{ and } A_i \in \widetilde{S}\,,\forall i \right\} \]

Intuitively, \(\mu^*\) is an upper bound on the measure of \(A\). The following properties follows immediately form the definition.

  1. \(\mu^*(\emptyset) \le \sum_{i = 1}^\infty \bar{ \mu }(\emptyset) = 0\).

  2. \(\mu^*\) is monotone: \(A \subseteq B\) implies that \(\mu^*(A) \le \mu^*(B)\).

  3. Subadditivity. If \(A \subseteq \cup_{i \ge 1} A_i\), then \(\mu^*(A) \le \sum_{i \ge 1} \mu^*(A_i)\). Given \(\epsilon > 0\), for each \(A_i\), there exists a countable sequence \(\{ B_{i, j} : j \in \mathbb{Z}^+ \}\) in \(\widetilde{S}\) such that it covers \(A_i\) and \[ \sum_{j \ge 1} \bar{ \mu }(B_{i, j}) \le \mu^*(A_i) + \epsilon / 2^i. \] The sequence \(\{ B_{i, j} : i, j \in \mathbb{Z}^+ \}\) is countable and covers \(A\). Further \[ \mu^*( A ) \le \sum_{i \ge 1} \sum_{j \ge 1} \bar{ \mu }(B_{i, j}) \le \sum_{i \ge 1} \mu^* (A_i) + \epsilon. \] The inequality holds for all \(\epsilon > 0\) hence it holds that \[ \mu^*( A ) \le \sum_{i \ge 1} \mu^* (A_i). \]

Now, \(\mu^*\) is almost a measure: it is an nonnegative set function and satisfies subadditivity. By previous analysis, we know that

Given a nonnegative, finitely additive set function defined on an algebra, the following conditions are equivalent * \(\sigma\)-additivity. * \(\sigma\)-subadditivity.

If we can prove that \(\mu^*\) is finitely additive, then it is a measure on the \(\sigma\)-algebra, \(2^\Omega\) (which is also an algebra). However, in general, finite additivity does not hold on \(2^\Omega\). Therefore, we restrict our discussion on a subset of \(2^\Omega\).

Measurable Set

Intuitively, the measurable sets are the ones on which finite additivity of \(\mu^*\) holds. Formally, we say that a set \(A \subseteq \Omega\) is called measurable if \[ \mu^* (B) = \mu^* (B \cap A) + \mu^* (B \cap \bar A),\qquad \forall B \subseteq \Omega. \]

Using subadditivity, to prove \(A\) is measurable, it is enough to show that \[ \mu^* (B) \ge \mu^* (B \cap A) + \mu^* (B \cap \bar A),\qquad \forall B \subseteq \Omega. \]

Definition. \(S^* =\) { the collection of all measurable sets }.

Theorem. \(S^*\) is a \(\sigma\)-field that contains \(\widetilde{S}\) and the restriction of \(\mu^*\) to \(S^*\) is a measure.

\(\blacktriangleright \quad S^*\) is a \(\sigma\)-field

  1. If \(A \in S^*\), then so is \(\bar A\).

    Proof. By definition.

  2. If \(A_1, A_2 \in S^*\), then \(A_1 \cup A_2 \in S^*\) and \(A_1 \cap A_2 \in S^*\).

    Proof. Observe that \[ A_1 \cup A_2 \subseteq (A_1 \cap \bar A_2) \cup (A_1 \cap A_2) \cup (\bar A_1 \cap A_2). \] For all \(B \subseteq \Omega\), it holds that \[ \begin{aligned} \mu^*(B) &= \mu^*(B \cap A_1) + \mu^* (B \cap \bar A_1) \\ &= \mu^*(B \cap A_1 \cap A_2 ) + \mu^*(B \cap A_1 \cap \bar A_2 ) \\ &\quad + \mu^* (B \cap \bar A_1 \cap A_2 ) + \mu^* (B \cap \bar A_1 \cap \bar A_2 ) \\ &\ge \mu^*(B \cap (A_1 \cup A_2) ) + + \mu^* (B \cap \bar A_1 \cap \bar A_2 ) \end{aligned} \] The last inequality follows form subadditivity of \(\mu^*\). Hence \(A_1 \cup A_2 \in S^*\). Finally, \(A_1 \cap A_2 = \overline{ \bar A_1 \cup \bar A_2} \in S^*\).

    \(\square\)

  3. If \(A_i \in S^*\), then \(A = \cup_{i \ge 1} A_i \in S^*\).

    Proof. We have proven that \(S^*\) is an algebra. Let \(B_1 = A_1\) and \(B_i = A_i - A_{i - 1} - \ldots - A_1\) for \(i \ge 2\). It holds that \(B_i \in S^*\) and \(\cup_i A_i = \cup B_i\). Replacing \(A_i\) with \(B_i\) we can assume without lose of generality that the \(A_i\) are disjoint. For all \(n \in \mathbb{Z}^+\), let \(C_n = +_{i \in [n] } A_i\). For all \(B \subseteq \Omega\) and \(n > 1\), \[ \begin{array}{ll} \mu^*(B) &= \mu^* (B \cap C_n ) + \mu^* (B \cap \bar C_n ) \\ &= \mu^* (B \cap C_n \cap A_n ) + \mu^* (B \cap C_n \cap \bar A_n ) + \mu^* (B \cap \bar C_n ) \\ &= \mu^* (B \cap A_n ) + \mu^* (B \cap C_{n - 1} ) + \mu^* (B \cap \bar C_n ) \\ \end{array} \] The last equality follows from that \(C_n \cap A_n = A_n\) and \(C_n \cap \bar A_n = C_{n - 1}\). By induction, and monotonicity of \(\mu^*\), we get \[ \begin{array}{ll} \mu^*(B) &= \sum_{i \in [n] } \mu^* (B \cap A_i ) + \mu^* (B \cap \bar C_n ) \\ &\ge \sum_{i \in [n] } \mu^* (B \cap A_i ) + \mu^* (B \cap \bar A ) \end{array} \] Letting \(n \rightarrow \infty\) finishes the proof.

    \(\square\)

\(\blacktriangleright \quad S^* \supseteq \widetilde{S}\)

Proof. For all \(B \subseteq \Omega\), and all \(\epsilon > 0\), there exists a sequence \(\{ B_i \}\) in \(\widetilde{S}\), such that \(B \subseteq \cup_i B_i\) and \[ \begin{array}{ll} \mu^* (B) + \epsilon \ge \sum_{i \ge 1} \bar \mu (B_i). \end{array} \] Let \(A \in \widetilde{S}\). Observe that \(B_i \cap A, B_i \cap \bar A \in \widetilde{S}\), and that \[ \begin{array}{ll} B \cap A \subseteq \cup_i (B_i \cap A)\,, \qquad B \cap \bar A \subseteq \cup_i (B_i \cap \bar A). \end{array} \] By the definition of \(\mu^*\) and finite additivity of \(\bar \mu\), it holds that \[ \small \begin{aligned} \mu^* (B \cap A) + \mu^* (B \cap \bar A) \le \sum_{i \ge 1} \bar \mu (B_i \cap A) + \sum_{i \ge 1} \bar \mu (B_i \cap \bar A) = \sum_{i \ge 1 } \bar \mu (B_i). \end{aligned} \]

Letting \(\epsilon \rightarrow 0\), it concludes that \[ \mu^*(B) \ge \mu^* (B \cap A) + \mu^* (B \cap \bar A). \]

\(\square\)

\(\blacktriangleright \quad \mu^*\) is a measure restricted to \(S^*\)

Sigma additivity of \(\mu^*\). Let \(A = +_{i \ge 1} A_i\), with \(A_i \in S^*\), then \[ \mu^*(A) = \sum_{i \ge 1 } \mu^*( A_i ). \]

Proof. As \(S^*\) is an algebra and \(\mu^*\) is \(\sigma\)-subadditive, it suffices to prove that \(\mu^*\) is finitely additive. Let \(A = +_{i \in [n] } A_i\). \[ \begin{aligned} \mu^*(A) &= \mu^*(A \cap A_n) + \mu^*(A \cap \bar A_n) \\ &= \mu^*(A_n) + \mu^*( +_{i \in [n - 1] } A_i ) \\ &\ldots\, \\ &= \sum_{ i \in [n] } \mu^* (A_i). \end{aligned} \]

\(\square\)

\(\spadesuit\) Uniqueness

\(\blacktriangleright\,\,\pi\)-system

Definition. \(\mathcal{P}\) is said to be a \(\pi\)-system if it is closed under intersection.

\(\blacktriangleright\,\,\lambda\)-system

Definition. \(\mathcal{L}\) is said to be a \(\lambda\)-system if
(1). \(\Omega \in \mathcal{L}\).
(2). If \(A \in \mathcal{L}\), then \(\bar A \in \mathcal{L}\).
(3). If \(A_1, A_2, \ldots \in \mathcal{L}\) with \(A_n \cap A_m = \emptyset\) for \(n \neq m\), then \(\cup_{n \ge 1} A_n \in \mathcal{L}\).

Note that a \(\lambda\)-system is almost a \(\sigma\)-algebra, expect that it is only closed under disjoint union.

Lemma. Replacing (2) in the definition with the following one results in equivalent definition
(2'). If \(A, B \in \mathcal{L}\) with \(A \subseteq B\), then \(B - A \in \mathcal{L}\).

Proof.

Assume that (1), (2), (3) hold. If \(A, B \in \mathcal{L}\) with \(A \subseteq B\), then \[ \bar B \in \mathcal{L} \implies A \cup \bar B \in \mathcal{L} \implies \overline{A \cup \bar B} = B - A \in \mathcal{L}. \] Hence (2') holds.

Assume that (1), (2'), (3) hold. Since \(\Omega \in \mathcal{L}\), if \(A \in \mathcal{L}\), then \[ \bar A = \Omega - A \in \mathcal{L}. \] Hence (2) holds.

\(\square\)

Lemma. If a \(\lambda\)-system \(\mathcal{L}\) is closed under intersection, then it is a \(\sigma\)-field.

Proof. Since it is closed under complement and intersection, it is an algebra. It is a \(\sigma\)-field as it satisfies monotonicity.

\(\square\)

\(\blacktriangleright\,\,\) \(\pi \text{-} \lambda\) Lemma

\(\pi \text{-} \lambda\) Lemma. Let \(\mathcal{P}\) be a \(\pi\) system and \(\mathcal{L}\) be a \(\lambda\)-system. If \(\mathcal{P} \subseteq \mathcal{L}\), then \(\sigma(\mathcal{P}) \subseteq \mathcal{L}\).

Proof. Let \(\lambda(\mathcal{P})\) the smallest \(\lambda\)-system that contains \(\mathcal{P}\). We will show that \(\lambda(\mathcal{P})\) is a \(\sigma\)-field. Then \[ \sigma(\mathcal{P}) \subseteq \lambda(\mathcal{P}) \subseteq \mathcal{L}. \]

It suffices to prove that \(\lambda(\mathcal{P})\) is a \(\pi\)-system.

​For each \(A \in \lambda (\mathcal{P})\), define \[ \Pi_A \doteq \{ B \in \lambda (\mathcal{P}) : B \cap A \in \lambda (\mathcal{P}) \}. \]

We will prove that \(\Pi_A\) is a \(\lambda\)-system and \(\Pi_A \supseteq \mathcal{P}\). It follows that \(\Pi_A \supseteq \lambda(\mathcal{P})\) and \(\lambda(\mathcal{P})\) is closed under intersection, and therefore a \(\pi\)-system.

\(\Pi_A\) is a \(\lambda\)-system:

  1. \(\Omega \in \Pi_A\).

  2. If \(B \in \Pi_A\), we need to prove that \(\bar B \in \Pi_A\). We need to prove that \(A \cap \bar B = A - \big( A \cap B \big) \in \lambda(\mathcal{P})\). This follows from definition (2') of \(\lambda\)-system and that \[ A, \big( A \cap B \big) \in \lambda(\mathcal{P})\, \wedge\, A \supseteq \big( A \cap B \big). \]

  3. If \(B_1, B_2, \ldots \in \Pi_A\) with \(B_n \cap B_m = \emptyset\), then \(\cup_{n \ge 1} B_n\in \Pi_A\), since \[ \big( \cup_{n \ge 1} B_n \big) \cap A = \cup_{n \ge 1} (B_n \cap A) \in \lambda (\mathcal{P}). \]

\(\Pi_A\) contains \(\mathcal{P}\):

If \(B \in \mathcal{P}\), denote \(\Pi_B = \{ A \subseteq \Omega : A \cap B \in \lambda (\mathcal{P}) \}\). Since \(\mathcal{P}\) is closed under intersection, \(\Pi_B \supseteq \mathcal{P}\). Since \(\Pi_B\) is a \(\lambda\)-system, \(\Pi_B \supseteq \lambda (\mathcal{P})\).

Hence, for all \(A \in \lambda (\mathcal{P}), B \in \mathcal{P}\), we have \(A \cap B \in \lambda (\mathcal{P})\). Therefore \(\Pi_A \supseteq \mathcal{P}\).

\(\square\)

Unique Extension Theorem

Theorem. Let \(\mathcal{P}\) be a \(\pi\)-system. If \(\nu_1\) and \(\nu_2\) are measures (on \(\sigma\)-fields \(\mathcal{F}_1\) and \(\mathcal{F}_2\)) that agree on \(\mathcal{P}\) and there is a sequence \(A_n \in \mathcal{P}\) with \(A_n \uparrow \Omega\) and \(\nu_i (A_n) < \infty\) for \(i = 1, 2\), then \(\nu_1\) and \(\nu_2\) agree on \(\sigma( \mathcal{P} )\).

Proof. Since \(\nu_i\) are measures on \(\sigma(\mathcal{P} )\), for all \(A \in \sigma(\mathcal{P})\), \[ \nu_i (A) = \lim_{n \rightarrow \infty} \nu_i (A \cap A_n). \] It suffices to prove that \(\nu_1 (A \cap A_n) = \nu_2 (A \cap A_n)\) holds for all \(n \in \mathbb{N}^+\).

For a fixed \(n\), now consider the set
\[ \mathcal{L} = \{ B \in \sigma(\mathcal{P} ) : \nu_1 (B \cap A_n) = \nu_2 (B \cap A_n) \} \subseteq \sigma( \mathcal{P} ). \]

\(\triangleright\) \(\mathcal{P} \subseteq \mathcal{L}:\) via assumption that \(\mathcal{P}\) is a \(\pi\)-system and that \(\nu_1\) and \(\nu_2\) agrees on \(\mathcal{P}\).

\(\triangleright\) \(\mathcal{L}\) is a \(\lambda\) system:

  • \(\Omega \in \mathcal{L}\).

  • If \(B \in \mathcal{L}\), then \(\bar B \in \mathcal{L}\) since
    \[ \begin{aligned} \nu_1 ( \bar B \cap A_n) &= \nu_1 ( A_n ) - \nu_1 (B \cap A_n) \\ &= \nu_2 ( A_n ) - \nu_2 (B \cap A_n) = \nu_2 ( \bar B \cap A_n ), \end{aligned} \] where the third equality follows from that \(\nu_1 ( A_n ) = \nu_2 ( A_n )\) and \(\nu_1 (B \cap A_n) = \nu_2 (B \cap A_n)\).

  • If \(B_j\) are disjoint sets in \(\mathcal{L}\), then \(\cup_{j \ge 1} B_j \in \mathcal{L}\), since \[ \begin{aligned} \nu_1 \Big( \big( \cup_{j \ge 1} B_j \big) \cap A_n \Big) &= \sum_{j} \nu_1 (B_j \cap A_n) \\ &= \sum_{j} \nu_2 (B_j \cap A_n) = \nu_2 \Big( \big( \cup_{j \ge 1} B_j \big) \cap A_n \Big). \end{aligned} \]

Using the \(\pi\)-\(\lambda\) lemma, we get \(\mathcal{L} \supseteq \sigma( \mathcal{P} )\), which finishes the proof.

\(\square\)

Median-of-Mean

Posted on 2020-06-21

Suppose that we have an algorithm \(A\) that estimates the mean \(E[Y]\) of a random variable \(Y\) within some specified range \(r\) with probability \(\rho > \frac{1}{2}\). We can derive a new algorithm \(A^*\) from \(A\) that boosts the successful probability to \(1 - \delta\), as follows:

  1. Repeat \(A\) for \(m = \frac{1}{2 (\rho - 0.5)^2 } \ln \frac{1}{\delta}\) times.
  2. Take the median of the \(m\) outputs.

To prove it, define the indicator random variable \[ X_i = \begin{cases} 1, \text{ if the } i \text{-th output of } A \text{ is within the range } r \\ 0, \text{otherwise} \end{cases} \]

for \(1 \le i \le m\). Then \(E[X_i] = \rho > 0.5\). Let \(\mu = E \left[\sum_{i = 1}^m X_i \right] = m\rho\).

By Hoeffding's inequality, for $> 0 $, \[ \Pr \left[ \left| \sum_{i = 1}^m X_i - \mu \right| \ge \lambda \right] \le \exp \left(- \frac{2\lambda^2}{m} \right) \]

Replacing \(\lambda\) with \(\mu - 0.5m\), and \(m\) with \(m = \frac{1}{2 (\rho - 0.5)^2 }\ln \frac{1}{\delta}\), we get \[ \begin{aligned} \Pr \left[ \sum_{i = 1}^m X_i \le 0.5m \right] &\le \Pr \left[ \left| \sum_{i = 1}^m X_i - \mu \right| \ge m(\rho - 0.5) \right] \\ &\le \exp \left(- 2m(\rho - 0.5)^2 \right) \\ &= \delta \end{aligned} \]

Note that if more than half of \(A\) outputs are correct, then the output of \(A^*\) is correct. This happens with probability at least \(1 - \delta\).

Entropy and Message Transmission

Posted on 2020-06-18 Edited on 2020-07-01

We discuss another application that sheds light on meaning of entropy, the message transmission problem. In this model, a sender transmits a message to the receiver through a channel. In real world, the channel could be an optical fiber, a wireless channel, a hard disk etc. In the final application, the computer that writes and reads information from the hard disk is both the sender and the receiver.

In real world the channel is not perfect and the message transmitted is distorted by noise. It is natural to ask whether the message can be transmitted accurately under the noise, i.e., whether the receiver can recover the noisy message.

To study the problem, we need a mathematical model for both the channel and the noise. We focus on the simplest binary channel with bits of 0 and 1. The noise causes the bits to flip and is modelled by the distribution on the bit-flips. We study the simplest one that flips each bit independently with some identical probability \(p < 0.5\). For a message with length \(n\), the number of bit flips follows a Binomial distribution \(B(n, p)\). We call such channel a binary symmetric channel.

To protect the message against noise, a naïve way is to send the each bit multiple times. For example, if the sender wants to send a bit 1, it sends \(10\) copies of \(1\) as \(1111111111\). The receiver decides that the bit sent is 1, if the majority of the \(10\) bits it receives is 1.

Could it be improved? Repeating each bit may not be the mot efficient way against the noise. In general, if the sender wants to send a message of \(k\) bits, it can convert it into a new message of \(n\) bits and sends the new one. The receiver considers the \(n\) bits received as a whole, and try to recover the \(k\)-bit message the sender wants to send.

We call the method used by the sender to convert the original message the encoding function, and the one used by the receiver to recover the message the decoding function.

Definition. A \((k, n)\) encoding function \(\text{Enc}: \{0, 1\}^k \rightarrow \{0, 1\}^n\) and a \((k, n)\) decoding function \(\text{Dec}: \{0, 1\}^n \rightarrow \{0, 1\}^k\), where \(n \ge k\).

Since the noise flips the bits randomly, it is impossible to have an encoding function and a decoding function without error (unless \(p = 0\)). Instead, we aim to control the probability of error within some specified threshold \(\delta\). To achieve this, we add redundancy to the message and encode a \(k\)-bit one into an \(n\)-bit one. Now, \(n - k\) is the amount of redundancy introduced. We would like to make \(n - k\) as small as possible. On the other hand, the value of \(n-k\) should positively related to \(p\). The larger \(p\) is, the noisier the channel is and the larger \(n - k\) should be.

For fixed \(\delta\) and \(p\), the Shannon's Theorem says that the smallest possible value of \(n - k\) we can achieve is roughly \(n\mathbf{H}(p)\). Intuitively, \(\mathbf{H}(p)\) is the amount of randomness the noise exerts on each bit. Therefore \(1 - \mathbf{H}(p)\) is the maximum amount of information we can transmit by one bit.

Before we proceed, we prove the following lemma.

Lemma 1
For \(q \in \mathbb{R}, q \le 1 / 2\) and \(n \in \mathbb{N}\), it hold that \[ \begin{aligned} \sum_{i = 0}^{\lfloor q n \rfloor} \binom{n}{i} &\le 2^{ n \mathbf{H} ( q) + \log \frac{n + 1}{2} } \end{aligned} \]

Proof.
\[ \begin{aligned} \sum_{i = 0}^{\lfloor q n \rfloor} \binom{n}{i} &\le \frac{n + 1}{2} \binom{n }{\lfloor q n \rfloor } \\ &\le \frac{n + 1}{2} 2^{ n \mathbf{H} \left( { \lfloor q n \rfloor } / { n } \right) } \\ &\le 2^{ n \mathbf{H} ( q) + \log \frac{n + 1}{2} } \end{aligned} \]

The first inequality follows from that \(\binom{n}{i}\) is increasing for \(i \le n / 2\) and that the summation is over at most \((n + 1) /2\) terms. The last one follows from that \(\mathbf{H}(x)\) is increasing for \(x \le 1 / 2\) and that \({ \lfloor q n \rfloor } / { n } \le { q n } / { n } = q < 1/2\).

\(\blacksquare\)

Shannon's Theorem
Given a binary symmetric channel with flip probability \(p < 1 / 2\) and for any any \(\epsilon, \delta > 0\), when \(n\) is large enough,

  1. if \(k \le n (1 - \mathbf{H}(p) - \epsilon)\), there exist \((k, n)\) encoding and decoding functions such that the receiver fails to obtain the correct message with probability at most \(2\delta\).
  2. \(\nexists (k, n)\) encoding and decoding functions with \(k \ge n (1 - \mathbf{H}(p) + \epsilon)\) such that the decoding is correctly is at most \(\delta\) for a \(k\)-bit input message chosen uniformly at random.

Proof of 1.

Intuition. When an \(n\)-bit message \(s\) is transmitted through the channel, the number of flipped bits is roughly \(np\). As \(p < {1} / {2}\), we can find a \(\lambda > 0\) such that \(p + \lambda < {1} / {2}\). Let \(R\) be the received \(n\)-bit message and define \(d(s, R)\) the number of different bits between \(s\) and \(R\), i.e., the Hamming distance between \(s\) and \(R\). Given \(s\) and \(p\), let \(\mathfrak{D}(s, p)\) be the distribution of \(R\) on \(\{0, 1\}^n\).

By Hoeffding's inequality, it holds that \[ \Pr_{R \sim \mathfrak{D}(s, p) } [ |d(s, R) - pn | \ge \lambda n] \le \exp(- 2 \lambda^2 n). \]

Define the \((p + \lambda)n\) ball centered \(s\) as \[ B_{s, (p + \lambda)n} \doteq \{ r \in \{0, 1\}^n : d(s, r) < (p + \lambda)n \}. \]

The the Hoeffding's inequality implies that for large enough \(n\), we have \[ \Pr_{R \sim \mathfrak{D}(s, p) } [ R \notin B_{s, (p + \lambda)n}] \le \delta. \]

That is, with probability at most \(\delta\), the received message fall outside the ball \(B_{s, (p + \lambda)n}\). This motivates to decode each message in \(B_{s, (p + \lambda)n}\) as the original message the sender wants to send.

Further, by Lemma 1, the size of \(B_{s, (p + \lambda) n}\) is bounded by \[ \begin{aligned} \sum_{i = 0}^{\lfloor (p + \lambda) n \rfloor} \binom{n}{i} &\le 2^{ n \mathbf{H} ( p + \lambda) + \log \frac{n + 1}{2} } \end{aligned} \]

Therefore, the maximum number of non-overlapped balls we can pack into the message space \(\{0, 1\}^n\) is
\[ \frac{2^n}{ 2^{ n \mathbf{H} ( p + \lambda) + \log \frac{n + 1}{2} } } = 2^{ n (1 - \mathbf{H} ( p + \lambda) - \frac{1}{n} \log \frac{n + 1}{2}) } \]

When \(n\) is large enough, this is at least \(2^{ n (1 - \mathbf{H} ( p ) - \epsilon) }\).

Remark: Hoeffding's inequality is an overkill. Chebyshev's inequality suffices for this proof.

Designing Encoding and Decoding Functions.

This motivates one to design of encoding and decoding function as follows: find a set of messages \(s_1, s_2, ..., s_{2^k} \in \{0, 1\}^n\), such that \[ B_{s_1, (p + \lambda)n}, B_{s_2, (p + \lambda)n}, ..., B_{s_{2^k} , (p + \lambda)n} \]

are disjoint balls, and assign the messages in \(\{0, 1\}^k\) to the ones in \(\{ s_i \}'s\) one by one. If the sending want to send \(i\), it sends \(s_i\). On receiving a message \(r\), the receiver determines which ball \(r\) belongs to. The probability of decoding error is at most \(\delta\).

Question to ponder: can we find such a set efficiently, such that
\[ B_{s_1, (p + \lambda)n}, B_{s_2, (p + \lambda)n}, ..., B_{s_{2^k} , (p + \lambda)n} \] are disjoint?

Finding the Codewords.

It is left to find a set of satisfactory \(s_1, s_2, ..., s_{2^k} \in \{0, 1\}^n\). The method we show here does not find a set of non-overlapped balls. Instead it aims to find a set such that

if \(s_i\) is transmitted, then the probability received message \(r\) falls into the another ball, i.e., \(r \in B_{s_{j} , (p + \lambda)n}\) (\(j \neq i\)), is less than \(\delta\).

The property implies that the overlap between \(B_{s_{i} , (p + \lambda)n}\) and \(\cup_{j \neq i} B_{s_{j} , (p + \lambda)n}\) is less than \(\delta\) (measured by probability). To formalize the statement, define the random variable \(S\) to be the message sent and \(R\) to be the one received. Given that \(S = s_i\) is sent, the conditional probability of receiving \(R = r\) is \[ \Pr_{R \sim \mathfrak{D}(s_i, p) } [R = r \mid S = s_i] = p^{d(s_i,r)} (1 - p)^{1 - d(s_i, r)} \]

To goal is to prove that it holds for all \(i\), \[ \Pr_{R \sim \mathfrak{D}(s_i, p) } [R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n } \mid S = s_i] = \sum_{r \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n} } p_r \le \delta \]

Then , by union bound, it holds that \[ \begin{aligned} \Pr_{R \sim \mathfrak{D}(s_i, p) } [ R \text{ is not recovered correctly} \mid S = s_i] &= \Pr_{R \sim \mathfrak{D}(s_i, p) } [ R \notin B_{s_i, (p + \lambda)n} \vee R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n} \mid S = s_i] \\ &\le \Pr_{R \sim \mathfrak{D}(s_i, p) } [ R \notin B_{s_i, (p + \lambda)n} \mid S = s_i] + \Pr[ R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n} \mid S = s_i] \\ &\le 2 \delta \end{aligned} \] That is, \(s_i\) is transmitted and decoded correctly with probability at least \(1 - 2 \delta\).

In what follows, we will prove the following claim:

Claim 1. If we generate \(2^{k + 1}\) codewords uniformly at random from \(\{0, 1\}^n\), then in expectation \[ \underset{ {s_1, s_2, ...,s_{2^{k + 1} } \sim \mathbf{U}(\{0, 1\}^n) } } {\mathbb{E} } \left[ \sum_{i = 1}^{2^{k + 1} } \Pr_{R \sim \mathfrak{D}(s_i, p) } [R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n } \mid S = s_i] \right] \le 2^k \delta \]

Claim 1 implies that there exits a set of codewords \(s_1, s_2, ..., s_{2^{k + 1} }\) with \[ \sum_{i = 1}^{2^{k + 1} } \Pr_{R \sim \mathfrak{D}(s_i, p) } [R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n } \mid S = s_i] \le 2^k \delta \]

If we send each \(s_i\) with probability \(1 / 2^k\), then the expected error probability is already \(\delta\). But we can have a stronger result: we can find a set of \(2^k\) codewords, such that for each \(i\), \[ \Pr_{R \sim \mathfrak{D}(s_i, p) } [R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n } \mid S = s_i] \le 2 \delta \]

W.L.O.G., suppose that \(s_1, s_2, ..., s_{2^k}\) are the ones with the smallest \(\Pr_{R \sim \mathfrak{D}(s_i, p) } [R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n } \mid S = s_i]\) 's, it must be that \[ \Pr_{R \sim \mathfrak{D}(s_i, p) } [R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n } \mid S = s_i] \le \frac{2^k \delta }{2^k } = \delta \]

Proof of Claim 1. By Lemma 1, for a fixed \(r\), the number of strings \(s \in \{0, 1\}^n\) such that \(d(s, r) < (p + \lambda) n\) is bounded by \[ \begin{aligned} \sum_{k = 0}^{\lfloor (p + \lambda) n \rfloor} \binom{n}{k} &\le \frac{n + 1}{2} 2^{ n \mathbf{H} ( p + \lambda) } \end{aligned} \]

If a message \(s\) is picked uniformly at random from \(\{0, 1\}^n\), the probability such that \[ \Pr_{s \sim \mathbf{U}(\{0, 1\}^n) } [d(s, r) < (p + \lambda) n ] \le \frac{n + 1}{2^{n + 1} } 2^{ n \mathbf{H} \left( p + \lambda \right) } \]

By union bound,
\[ \begin{aligned} \Pr_{s_1, s_2, ...,s_{2^{k + 1} } \sim \mathbf{U}(\{0, 1\}^n) } [ r \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n} ] &< 2^k \frac{n + 1}{2^{n + 1} } 2^{ n \mathbf{H}( p + \lambda) } \\ &= 2^{ k + \log (n + 1) + n \mathbf{H} ( p + \lambda) - n - 1 } \end{aligned} \]

As \(k \le n(1 - \mathbf{H}(p) - \epsilon)\), when \(n\) is sufficient large, it follows
\[ \begin{aligned} \Pr_{s_1, s_2, ...,s_{2^{k + 1} } \sim \mathbf{U}(\{0, 1\}^n) } [ r \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n} ] < 2^{ \log \frac{n + 1}{2} + n ( \mathbf{H} ( p + \lambda) - \mathbf{H} ( p) - \delta) } \le \frac{\delta}{2} \end{aligned} \]

Therefore, \[ \begin{aligned} &\underset{ {s_1, s_2, ...,s_{2^{k + 1} } \sim \mathbf{U}(\{0, 1\}^n) } } {\mathbb{E} } \left[ \Pr_{R \sim \mathfrak{D}(s_i, p) } [R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n } \mid S = s_i] \right] \\ &= \sum_{r \in \{0, 1\}^n } \Pr_{s_1, s_2, ...,s_{2^{k + 1} } \sim \mathbf{U}(\{0, 1\}^n) } [ r \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n} ] \cdot \Pr_{R \sim \mathfrak{D}(s_i, p) } [ R = r \mid S = s_i] \\ &\le \frac{\delta}{2} \end{aligned} \]

and \[ \underset{ {s_1, s_2, ...,s_{2^{k + 1} } \sim \mathbf{U}(\{0, 1\}^n) } } {\mathbb{E} } \left[ \sum_{i = 1}^{2^{k + 1} } \Pr_{R \sim \mathfrak{D}(s_i, p) } [R \in \cup_{j \neq i} B_{s_{j} , (p + \lambda)n } \mid S = s_i] \right] \le 2^{k + 1} \frac{\delta}{2} = 2^k \delta \]

\(\blacksquare\)

Proof of 2.

The difficulty here is to prove that the claim holds for arbitrary \((k, n)\) encoding and decoding functions. The proof relies on an important property:

the decoding function \(\text{Dec}\) is a function on \(\{0, 1\}^n\)

For any \(r \in \{0, 1\}^n\), there is a unique output \(\text{Dec}(r) \in \{0, 1\}^k\). The decoding function needs to decide the unique message \(\{0, 1\}^n\) the sender want to send.

Now, define \[ S_i \doteq \{ r \in \{0, 1\}^n : r \text{ is decoded correctly if } s_i \text{ is sent} \} \]

The \(S_i\)'s are non-overlapped and we have, \[ \cup_{i = 1}^{2^k} S_i \subset \{0, 1 \}^n \]

If \(s_i\) is sent, then by Hoeffding inequality, with probability at least \(1 - \exp(- 2\lambda^2 n)\), the received message \(R\) is likely to fall into a ring centered at \(s_i\): \[ \text{ring} (s_i) \doteq \{ r \in \{0, 1\}^n : |d(r, s_i) - pn |< \lambda n \} \]

Finally, if the messages \(s_1, s_2, ..., s_{2^k}\) are sent uniformly at random, i.e., \(i \sim \mathbf{U}[1, 2^k]\), then \[ \begin{aligned} \Pr[ R \text{ is decoded correctly}] &= \sum_{i = 1}^{2^k} \sum_{r \in S_i } \Pr_{R \sim \mathfrak{D}(s_i, p) } [R = r \mid S = s_i] \Pr_{i \sim \mathbf{U}[1, 2^k]} [ S = s_i] \\ &= \frac{1}{2^k} \sum_{i = 1}^{2^k} \sum_{r \in S_i } \Pr_{R \sim \mathfrak{D}(s_i, p) } [R = r \mid S = s_i] \end{aligned} \]

Observed that \(S_i \subset \text{ring} (s_i) \cup (S_i \cap \text{ring} (s_i) )\), we get

\[ \begin{aligned} &\Pr[ R \text{ is decoded correctly}] \\ &\le \frac{1}{2^k} \sum_{i = 1}^{2^k} \left( \sum_{r \notin \text{ring} (s_i) } \Pr_{R \sim \mathfrak{D}(s_i, p) } [R = r \mid S = s_i] +\sum_{r \in S_i \cap \text{ring} (s_i) } \Pr_{R \sim \mathfrak{D}(s_i, p) } [R = r \mid S = s_i] \right) \\ &\le \frac{1}{2^k} \sum_{i = 1}^{2^k} \left( \exp( -2\lambda^2n) +\sum_{r \in S_i \cap \text{ring} (s_i) } \Pr_{R \sim \mathfrak{D}(s_i, p) } [R = r \mid S = s_i] \right) \end{aligned} \]

Note that for \(r \in S_i \cap \text{ring} (s_i)\), as \(p < 1 / 2\), it holds that \[ \Pr_{R \sim \mathfrak{D}(s_i, p) } [R = r \mid S = s_i] \le p^{ \lceil(p - \lambda) n \rceil} (1 - p)^{ n - \lceil(p - \lambda) n \rceil } \]

We obtain

\[ \begin{aligned} &\Pr[ R \text{ is decoded correctly}] \\ &\le \exp( -2\lambda^2n) + \frac{1}{2^k} \sum_{i = 1}^{2^k} \sum_{r \in S_i \cap \text{ring} (s_i) } p^{ \lceil(p - \lambda) n \rceil} (1 - p)^{ n - \lceil(p - \lambda) n \rceil } \\ &\le \exp( -2\lambda^2n) + \frac{1}{2^k} \sum_{i = 1}^{2^k} \sum_{r \in S_i \cap \text{ring} (s_i) } p^{ (p - \lambda) n } (1 - p)^{ n - (p - \lambda) n } \\ &= \exp( -2\lambda^2n) + p^{ (p - \lambda) n } (1 - p)^{ n - (p - \lambda) n } \sum_{i = 1}^{2^k} \sum_{r \in S_i \cap \text{ring} (s_i) } \frac{1}{2^k} \\ &= \exp( -2\lambda^2n) + p^{ pn } (1 - p)^{ n - pn } \left(\frac{ 1 - p}{ p} \right)^{\lambda n} 2^{n - k}\\ &= \exp( -2\lambda^2n) + 2^{n - k - n \mathbf{H}(p)} \left(\frac{ 1 - p}{ p} \right)^{\lambda n} \end{aligned} \]

Conditioning on that \(k \ge n (1 - \mathbf{H}(p) + \delta)\), \[ \Pr[ R \text{ is decoded correctly}] \le \exp( -2\lambda^2n) + 2^{- n (\delta + \lambda \log \frac{ 1 - p}{ p} ) } \]

By choosing sufficient small \(\lambda\) and large enough \(n\), \(\Pr[ R \text{ is decoded correctly}]\) can be arbitrary small.

\(\blacksquare\)
