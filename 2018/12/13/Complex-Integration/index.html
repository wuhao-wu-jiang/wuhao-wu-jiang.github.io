Posted on 2018-12-13 Edited on 2021-10-26

We establish complex integral step by step in this article.
We first review the classical Riemann integral and see how this motivates complex integral.

Riemann Sums

Given an interval [a,b][a, b][a,b] and a function f:[a,b]→Rf : [a, b] \rightarrow \Rf:[a,b]→R, the Riemann integration divides [a,b][a, b][a,b] into many small intervals, and computes the sum of the products of the length of each interval and value of fff at an arbitrary chosen point on the interval.
Such a sum is well-defined if it converges when the chosen partition is “refined” enough.

DEFINITION. A finite sequence P={x0,x1,…,xn}P = \{x_0, x_1, \ldots, x_n \}P={x0​,x1​,…,xn​} is a partition of [a,b][a, b][a,b], if

a=x0<x1<…<xn=b. a = x_0 < x_1 < \ldots < x_n = b. a=x0​<x1​<…<xn​=b.

∥P∥≐max⁡i∈[n]Δi \Vert P \Vert \doteq \max_{i \in [n] } \Delta_i ∥P∥≐i∈[n]max​Δi​

where Δi≐xi−xi−1\Delta_i \doteq x_i - x_{i - 1}Δi​≐xi​−xi−1​ for each i∈[n]i \in [n]i∈[n].
