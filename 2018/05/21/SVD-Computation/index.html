Singular vector decomposition is one of the highlights of linear algebra.

Review of Linear Transformation

Before diving into the topic, we have a brief review of linear transformation. Given a matrix A∈Rm×nA \in \mathbb{R}^{m \times n}A∈Rm×n with rank rrr, we can view it as a linear transformation from f:Rn→Rmf: \mathbb{R}^n \rightarrow \mathbb{R}^mf:Rn→Rm defined as

f(x)=Ax,∀x∈Rn. f(x) = Ax, \forall x \in \mathbb{R}^n. f(x)=Ax,∀x∈Rn.

It maps a vector in the row space, i.e., the subspace spanned by row vectors of AAA, to a vector in the column space, the subspace spanned by column vectors of AAA.

Define

  1. S(AT)={∑i=1nλiA[i,:]:λi∈R,∀i∈[n]}S(A^T) = \{ \sum_{i = 1}^n \lambda_i A[i,:] : \lambda_i \in \mathbb{R}, \forall i \in [n] \}S(AT)={∑i=1n​λi​A[i,:]:λi​∈R,∀i∈[n]}, which we call the row space of AAA.

  2. N(AT)={x∈Rm,s.t.,Ax=0}N(A^T) = \{ x \in \mathbb{R}^m, s.t., Ax = 0\}N(AT)={x∈Rm,s.t.,Ax=0}, the subspace perpendicular to S(AT)S(A^T)S(AT).

  3. S(A)={∑i=1mλiA[:,i]:λi∈R,∀i∈[m]}S(A) = \{ \sum_{i = 1}^m \lambda_i A[:,i] : \lambda_i \in \mathbb{R}, \forall i \in [m] \}S(A)={∑i=1m​λi​A[:,i]:λi​∈R,∀i∈[m]}, the columns space of AAA.

  4. N(A)N(A)N(A) the space perpendicular to S(A)S(A)S(A).

Note that S(AT)S(A^T)S(AT) and S(A)S(A)S(A) has the same dimension rrr.
