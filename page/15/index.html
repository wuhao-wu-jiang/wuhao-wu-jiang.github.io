Power Series--Radius of Convergence

Posted on 2021-07-17 Edited on 2022-11-13

Theorem (Hadamard's Formula). Let \(f : \mathbb{C} \rightarrow \mathbb{C}\) be a power series defined as \(f(z) = \sum_{n \ge 1} a_n z^n\). The radius of convergence of \(f\), denoted as \(R \in [0, \infty]\), is given by \[ 1 / R = L \doteq \limsup_{k \ge 1} |a_k|^{1 / k}. \] where we adopt the convention that \(1 / 0 = \infty\) and \(1 / \infty = 0\).

It is guaranteed that 1. If \(|z| < R\), then \(f\) converges absolutely. 2. If \(|z| > R\), then \(f\) diverges.

Remark: The region \(|z| < R\) is called the disc of convergence of \(f\).

Proof.
If \(|z| < R\), then \(L \cdot |z| < 1\). Hence, \[ \exists \epsilon > 0, \, r < 1, \, \text{ s.t. } \, (L + \epsilon) \cdot |z| < r. \]

Since \(\sup_{k \ge n} | a_k |^{1 / k} \rightarrow L\), for large enough \(n\), we have \[ L + \epsilon \ge \sup_{k \ge n} | a_k |^{1 / k} \ge |a_n|^{1 / n}. \]

Therefore, \[ | a_n z^n | = \big( |a_n|^{1 / n} \cdot |z| \big)^n \le \big( (L + \epsilon) \cdot |z| \big)^n \le r^n. \]

It follows that \(f\) converges absolutely.

If \(|z| > R\), then \(L \cdot |z| > 1\). Hence, \[ \exists \epsilon > 0, \, r > 1, \, \text{ s.t. } \, (L - \epsilon) \cdot |z| > r. \]

Since \(\sup_{k \ge n} | a_k |^{1 / k}\) decreases to \(L\), for each \(n \in \mathbb{N}^+\), \[ \exists k \ge n, \, \text{ s.t. } \, |a_k|^{1 / k} \ge (L - \epsilon). \]

For such \(k\), \[ | a_k z^k | = \big( |a_k|^{1 / k} \cdot |z| \big)^k \ge \big( (L - \epsilon) \cdot |z| \big)^k \ge r^k. \]

Since the terms of \(f\) are unbounded, the series is consequently divergent.

\(\square\)

Theorem. Let \(f : \mathbb{C} \rightarrow \mathbb{C}\) be a power series defined as \(f(z) = \sum_{n \ge 1} a_n z^n\). Then it is holomorphic in its disc of convergence. Its derivative is given by \[ f'(z) = \sum_{n \ge 1} n a_n z_n^{n - 1}. \] Moreover, \(f'(z)\) has the same disc of convergence.

Proof. Let \(b_n = n a_n\). Define \[ g(z) = \sum_{n \ge 1} b_n z_n^{n - 1} = \sum_{n \ge 1} n a_n z_n^{n - 1}. \] Observe that \[ \begin{aligned} \limsup_n | b_n |^{ 1 / n} &= \limsup_n | n a_n |^{ 1 / n} \\ &= \lim_n | n |^{ 1 / n} \cdot \limsup_n | a_n |^{ 1 / n} \\ &= \limsup_n | a_n |^{ 1 / n}. \end{aligned} \]

It follows that \(g(z)\) has the same disc of convergence as \(f(z)\).

It is left to prove that \(g(z)\) is the derivative of \(f(z)\). For \(k \in \mathbb{N}^+\), write \(f(z) = p_k (z) + r_k(z)\), where \[ p_k(z) = \sum_{n \in [k] } a_n z^n,\quad r_k(z) = \sum_{n > k} a_n z^n. \]

Denote \(R\) the radius of convergence of \(f(z)\). and suppose \(0 \le |z_0| < r < R\). Then for \(h \in \mathbb{C}\), such that \(|z_0 + h| < r\), we have \[ \begin{aligned} \left| \frac{ f(z_0 + h) - f(z_0) }{ h } - g(z_0) \right| &= \left| \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } + \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } - g(z_0) \right| \\ &= \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } + \left( \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } - p_k'(z_0) \right) + p_k'(z_0) - g(z_0 ) \right| \\ &\le \left| \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } - p_k'(z_0) \right| + \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| + \left| p_k'(z_0) - g(z_0 ) \right| . \end{aligned}. \]

Claim. \(\forall \epsilon > 0\), \(\exists k \in \mathbb{N}^+\), s.t., \[ \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| \le \epsilon, \quad \left| p_k'(z_0) - g(z_0 ) \right| \le \epsilon. \] Further, for such integer \(k\), \(\exists h > 0\), s.t., \[ \left| \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } - p_k'(z_0) \right| \le \epsilon. \]

Case One \(\left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| \le \epsilon\): since \(a^n - b^n = (a - b) \left( a^{n - 1} + a^{n - 2} b^2 + \ldots + b^{n - 1} \right),\) and \(|z_0 + h| < r\), \(|z_0 | < r\), \[ \begin{aligned} \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| &= \left| \frac{ \sum_{n > k} a_n \Big[ (z_0 + h)^n - z ^n \Big] }{ h } \right| &\le \sum_{n > k} a_n n r^{n - 1}. \end{aligned} \]

Since \(g(r)\) converges absolutely, for large enough \(k\), \[ \left| \frac{ r_k(z_0 + h) - r_k(z_0) }{ h } \right| \le \sum_{n > k} a_n n r^{n - 1} \le \epsilon. \]

Case Two \(\left| p_k'(z_0) - g(z_0 ) \right| \le \epsilon\): since \(g(z_0)\) converges absolutely, for large enough \(k\), \[ \left| p_k'(z_0) - g(z_0 ) \right| \le \epsilon. \]

Case Three \(\left| \frac{ p_k(z_0 + h) - p_k(z_0) }{ h } - p_k'(z_0) \right| \le \epsilon\): following the definition of derivative, this is true for small enough \(h\).

\(\square\)

Reference

[1] L. V. Ahlfors, Complex analysis: an introduction to the theory of analytic functions of one complex variable, 3d ed. New York: McGraw-Hill, 1979.

Complex Mapping

Posted on 2021-07-16 Edited on 2021-07-23

Suppose we have complex mapping \(f : X \rightarrow Y\), where \(X, Y \subset \mathbb{C}\). How do we visualize it? As, we can't visualize four dimension data in a single plot, we split it into two, one for the domain and one for the codomain.

For example, consider the unit cycle centered at \(2\) \[ S_1 \doteq \{ z \in \mathbb{C} : | z - 2 | < 1 \}. \]

Let \(f(z) \doteq \frac{2}{z - 1}\). Then \(S_1\) is mapped to the set \[ S_2 \doteq \{ z \in \mathbb{C} : \text{Re}(z) > 1 \}. \]

This is shown by the picture below.

We verify this algebraically.

Proof 1. Let \(w = \frac{2}{z - 1}\). As \(w \neq 0\), we have \[ {\small z - 1 = \frac{2}{w} \implies z = \frac{2}{w} + 1. } \] Hence, \[ \begin{aligned} \small |z - 2 | < 1 &\implies \left| \frac{2}{w} - 1 \right| < 1 \\ &\implies \left| \frac{2 - w}{w} \right|^2 < 1 \\ &\implies \left| 2 - w \right|^2 < |w|^2 \\ &\implies \Vert (2 - x, y) \Vert_2^2 < \Vert (x, y) \Vert_2^2 &\text{let } w = (x, y) \\ &\implies 4 - 4 x < 0 \\ &\implies 1 < x. \end{aligned} \] \(\square\)

Proof 2. Denote the border of \(S_1\) as \(z = 2 + e^{i \theta}\), where \(\theta \in [-\pi, \pi)\). Then \[ \begin{aligned} w &= \frac{2}{z - 1} \\ &= \frac{2}{1 + e^{i \theta } } \\ &= \frac{2}{1 + e^{i \theta } } \cdot \frac{e^{- i\theta / 2 } }{ e^{ - i \theta / 2 } } \\ &= \frac{2 e^{ - i \theta / 2} }{ e^{ - i \theta / 2 } + e^{i \theta / 2} } \\ &= 2 \cdot \frac{ \cos (\theta / 2) + i \sin (\theta / 2) }{ 2 \cos (\theta / 2) } \\ &= 1 + i \tan (\theta / 2). \end{aligned} \]

\(\square\)

Fano’s Inequality

Posted on 2021-07-14 Edited on 2021-07-25

Suppose that Alice wants to transmit a random variable \(X\) to Bob via a noisy channel. The noisy channel is modelled by a stochastic mapping \(f\), and Bob receives a randomized version of \(X\), denoted as \(Y = f(X)\). Bob post-processes \(Y\) by another stochastic mapping \(g\) to produce an estimate \(\hat X\) of \(X\). The mapping \(g\) depends only on \(Y\) and therefore \(\hat X\) and \(X\) are conditionally independent given \(Y\). The random variables \(X\), \(Y\) and \(\hat X\) forms a Markov chain, denoted as \[ X \rightarrow Y \rightarrow \hat X. \]

Denote \(p_e = \Pr[ \hat X \neq X]\) the probability of estimation error. If the channel is noiseless, then \(H(X \mid Y ) = 0\), and the identity mapping \(g(Y) \doteq Y\) has error probability \(p_e = 0\). As the noisy of the channel increases, \(H(X \mid Y)\) increases, so as \(p_e\). \(H(X \mid Y)\) can be viewed as the information lost incurred by the channel. The information lost is small, only if when \(p_e\) is small, which is established by the following theorem.

Theorem (Fano's Inequality). Suppose that the random variables \(X \rightarrow Y \rightarrow \hat X\) forms a Markov chain. Let \(p_e \doteq \Pr[ X \neq \hat X]\). The information lost of \(H(X \mid Y)\) is bounded by: \[ H(X \mid Y) \le H ( X \mid \hat X) \le H_2(p_e) + p_e \log ( | \Omega | - 1), \] where \(H_2(p) \doteq p \cdot \big( \log \frac{1}{p} \big) + (1 - p) \cdot \log \frac{1}{1 - p}\).

Remark. Note that \[ \lim_{p_e \rightarrow 0 } H_2 (p_2) + p_e \log ( | \Omega | - 1 ) = 0, \] which complies with our discussion before.

Proof. Let \(E\) be an indicator random variable for the event \(X \neq \hat X\).

Via chain rule for entropy, we expand \(H(E, X \mid \hat X)\) in two different ways \[ \begin{aligned} H(E, X \mid \hat X) &= H( X \mid \hat X ) + H( E \mid X, \hat X) \\ &= H( E \mid \hat X ) + H( X \mid E, \hat X). \end{aligned} \]

We see that * \(H(E \mid X, \hat X) = 0\), since \(E\) is a function of \(X\) and \(\hat X\), the conditional entropy.

  • \(H( E \mid \hat X ) \le H(E) = H_2 (p_e)\), since conditioning reduces entropy and \(E\) is a binary random variable.

  • \(H(X \mid E, \hat X) \le p_e \log ( | \Omega | - 1)\), which is obtained as follows:
    \[ \begin{aligned} H(X \mid E, \hat X) &= p_e H( X \mid \hat X, E = 1) + (1 - p_e) H(X \mid \hat X, E = 0) \\ &\le p_e \log ( | \Omega | - 1). \end{aligned} \]

    The last inequality holds, since given \(\hat X\),

    1. when \(E = 0\), there is no randomness about \(X\); and

    2. when \(E = 1\), \(X\) has \(|\Omega - 1|\) possible outcomes and its randomness is less than the one of uniform distribution over the possible outcomes.

Combining these results, we get \[ \begin{aligned} H(E, X \mid \hat X) = H( X \mid \hat X ) \le H_2 ( p_2 ) + p_e \log ( | \Omega | - 1). \end{aligned} \]

It is left to prove that \[ H(X \mid Y) \le H( X \mid \hat X). \]

Via the data-processing inequality for the Markov chain \(X \rightarrow Y \rightarrow \hat X\): \[ I(X ; Y) \ge I(X ; \hat X), \]

which implies \[ H( X \mid \hat X) = H(X) - I(X ; \hat X) \ge H(X) - I(X ; Y) = H(X \mid Y). \]

\(\square\)

Reference

[1] T. M. Cover and J. A. Thomas, “ELEMENTS OF INFORMATION THEORY,” p. 774.
