Convergence Tests

Posted on 2019-02-27 Edited on 2022-04-09

Ration Test

Given a series \(s_n = \sum_{k \in [n]} a_n\) with \(a_n \neq 0\), the Ratio Test states that if \((a_n)\) satisfies \[ \lim \left| \frac{a_{n + 1} }{ a_n } \right| = r \le 1 \] then \((s_n)\) converges absolutely.

Proof. The condition implies that \[ \exists N > 0, s.t., \exists r \le r' < 1, \forall n \ge N, a_{n + 1} \le r' a_n \]

It follows that for \(n > m \ge N\), \[ \begin{aligned} |s_n - s_m| &= | a_{m + 1} + a_{m + 1} + ... + a_n| \\ &\le |a_N| \sum_{k = 0}^{n - m - 1} (r')^{m - N + k} \\ &\le |a_N| \sum_{k = 0}^{ \infty } (r')^{m - N + k} \\ &\le \frac{|a_N| }{1 - (r') } (r')^{m - N} \end{aligned} \]

If \(m\) is sufficient large, then \(|s_n - s_m|\) is arbitrary small. It concludes that \((s_n)\) is Cauchy sequence.

\(\square\)

Abel's Test

If the series \(s_n = \sum_{k \in [n]} a_n\) converges and if \((b_n)\) is a sequence satisfying \[ b_1 \ge b_2 \ge b_3 \ge ... \ge 0 \] then the series \(\sum_{k \ge 1} a_k b_k\) converges.

Lemma: Partial Sum

Define \(s_0 = 0\), then \[ \begin{aligned} \sum_{k \in [n]} x_k y_k &= \sum_{k \in [n]} (s_k - s_{k - 1}) b_k \\ &= \sum_{k \in [n]} s_k b_k - \sum_{k \in [n-1]} s_k b_{k + 1} \\ &= s_n b_n + \sum_{k \in [n-1]} s_k (b_k - b_{k + 1}). \end{aligned} \]

Proof. \((b_n)\) is monotone and bounded, therefore \((b_n)\) converges. Since both \((s_n)\) and \((b_n)\) converges, \((s_n b_n)\) converges.

It remains to prove \(\sum_{k \in [n-1]} s_k (b_k - b_{k + 1})\) converges. We show it is a Cauchy sequence. We know the convergent sequence \((s_n)\) is bounded. Let \(| s_n | \le M\). Thus, for \(n > m\), \[ \left| \sum_{k = m}^{n - 1} s_k (b_k - b_{k + 1}) \right| \le M \sum_{k = m}^{n - 1} (b_k - b_{k + 1} ) = M (b_m - b_n) \]

The fact \((b_n)\) converges implies that \(b_m - b_n \rightarrow 0\) for large \(m\) and so, \(|\sum_{k = m}^{n - 1} s_k (b_k - b_{k + 1})| \rightarrow 0\).

\(\square\)

Dirichlet's Test

If the series \(s_n = \sum_{k \in [n]} a_n\) is bounded (but not necessary convergent), and if \((b_k)\) is a sequence satisfying \[ \begin{cases} b_1 \ge b_2 \ge b_3 \ge ... \ge 0 \quad and \\ \lim b_k = 0 \end{cases} \] then the series \(\sum_{k \ge 1} a_k b_k\) converges.

Proof. Let \(|s_n| \le M\). As \((b_n) \rightarrow 0\), \((s_n b_n) \rightarrow 0\). The proof for the convergence of \(\sum_{k \in [n-1]} s_k (b_k - b_{k + 1})\) is exactly the same as that of Abel's test.

\(\square\)

Reference

[1]. Stephen Abbott, Understanding Analysis, Second Edition.
