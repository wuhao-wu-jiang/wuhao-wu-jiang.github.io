Less Communication

Posted on 2019-09-03

A set of kkk servers are receiving requests and they are required to report to a coordinator the numbers of requests they process. An event is trigerrd by the coordinator if the culmulative requests received by the serves reaches a threshold τ\tauτ. The goal is to minimized the communicaiton between the coordinator and the servers.

A naive approach asks the servers to send a notificiation to the coordinator each time they process a request, resulting in O(τ)O(\tau)O(τ) bits communication in total.

An alternative approach achieves O(hlog⁡τ)O(h \log \tau)O(hlogτ) words communication cost. Instead of response to the coordinator every a new request comes, the server reports only after some requests reaches. However, we don’t want the coordinator to miss the τ\tauτ threshold, i.e., it must triger the event when exactly τ\tauτ requests come in total.

Here is the scheme.

  1. If τ≤5⋅h\tau \le 5 \cdot hτ≤5⋅h, the server report each time a request comes.
  2. The coordinator sends the threshold ⌊τ2h⌋\lfloor \frac{\tau}{ 2h} \rfloor⌊2hτ​⌋ to each server. This requires hhh words.
  3. Each server sends a report to the coordinator every ⌊τ2h⌋\lfloor \frac{\tau}{ 2h} \rfloor⌊2hτ​⌋ requests comes.
  4. When the coordinator learns that, the servers has processed ⌊τ/2⌋\lfloor \tau / 2 \rfloor⌊τ/2⌋ requests (at most 2h2h2h words communication needed), it collects how many requests are proceed in each servers which are not reported to it (at most 2h2h2h words communication needed). The number of requests comes is at most

    ⌊τ/2⌋+(h−1)(⌊τ/2h⌋−1)<τ/2+τ/2=τ \lfloor \tau / 2 \rfloor + (h - 1) (\lfloor \tau / 2h \rfloor - 1) < \tau /2 + \tau / 2 = \tau ⌊τ/2⌋+(h−1)(⌊τ/2h⌋−1)<τ/2+τ/2=τ

  5. The number of requests remains to reach the τ\tauτ threshold is at most τ′≐⌈τ/2⌉\tau' \doteq \lceil \tau / 2 \rceilτ′≐⌈τ/2⌉. We set τ′\tau'τ′ as the new threshold and go to step 1.
  6. The overall communication cost is given by

    T(τ)=T(⌈τ/2⌉)+5⋅h T(\tau) = T(\lceil \tau / 2 \rceil) + 5 \cdot h T(τ)=T(⌈τ/2⌉)+5⋅h

    for some constant kkk.
  7. There are at most 2log⁡τ2 \log \tau2logτ recursions. Each recursion requires kkk words communication, resulting in a total communication cost O(hlog⁡τ)O(h \log \tau)O(hlogτ) words.
