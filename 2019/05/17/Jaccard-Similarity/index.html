Jaccard Similarity

Posted on 2019-05-17 Edited on 2019-05-18

The Jaccard similarity measures the fraction of shared elements between set. In particular, given set AAA and BBB, the Jaccard similarity between them is defined as

J(A,B)=∣A∩B∣∣A∪B∣J(A, B) = \frac{|A \cap B|}{|A \cup B|} J(A,B)=∣A∪B∣∣A∩B∣​
